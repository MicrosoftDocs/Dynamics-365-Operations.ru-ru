# <a name="analyze-store-performance"></a>Анализ эффективности магазина

В этой статье объясняется, как можно использовать аналитики в памяти и в режиме реального времени для получения доступа, изучения и получить представление о быстродействии банка сообщений, на основе данных Microsoft Dynamics 365 для данных операций. 

Как часть Dynamics 365 для операций, пользователям оценить производительность магазина в режиме реального времени на разных уровнях иерархии организации за выбранный период, открыв-встроенных **Сводка канала** отчет из следующих мест:

-   **Retail store управления** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**каналов**&gt;**управления Retail store**&gt;**отчетов**&gt;**канале сводный отчет**
-   **Retail store Финансы** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**каналов**&gt;**Финансы Retail store**&gt;**отчетов**&gt;**канале сводный отчет**
-   **Запросы и отчеты** разделе Dynamics 365 для операций &gt;**розничной торговли и электронной коммерции**&gt;**запросов и отчетов**&gt;**отчетов продажи**&gt;**канале сводный отчет**

Этот отчет содержит снимок следующих сводок об эффективности работы магазина.

-   Сводка по валовым продажам
-   Сводка по типу платежного средства
-   Сводка по налогам
-   Сводка переопределения цены
-   Сводка по скидке
