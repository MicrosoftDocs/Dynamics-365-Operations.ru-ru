# <a name="analyze-store-performance"></a>Анализ эффективности магазина

[!include[banner](includes/banner.md)]


В этой статье описывается, как можно использовать размещенные в памяти аналитики и аналитики реального времени для получения доступа, изучения и анализа эффективности магазина на основе данных Microsoft Dynamics 365 for Operations. 

В рамках использования Microsoft Dynamics 365 for Operations пользователи могут проанализировать эффективность работы магазина в реальном времени на разных уровнях иерархии организации за выбранный период, открыв готовый отчет **Сводка по каналам** в любом из следующих расположений.

-   Рабочая область Dynamics 365 for Operations **Управление розничными магазинами** &gt; **Розничная торговля и коммерция** &gt; **Каналы** &gt; **Управление розничными магазинами** &gt; **Отчеты** &gt; **Сводный отчет по каналу**
-   Рабочая область Dynamics 365 for Operations **Финансовая информация розничного магазина** &gt; **Розничная торговля и коммерция** &gt; **Каналы** &gt; **Финансовая информация розничного магазина** &gt; **Отчеты** &gt; **Сводный отчет по каналу**
-   Раздел Dynamics 365 for Operations **Запросы и отчеты** &gt; **Розничная торговля и коммерция** &gt; **Запросы и отчеты** &gt; **Отчеты о продажах** &gt; **Сводный отчет по каналу**

Этот отчет содержит снимок следующих сводок об эффективности работы магазина.

-   Сводка по валовым продажам
-   Сводка по типу платежного средства
-   Сводка по налогам
-   Сводка переопределения цены
-   Сводка по скидке
