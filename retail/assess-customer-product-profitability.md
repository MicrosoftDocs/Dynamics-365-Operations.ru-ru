# <a name="assess-customer-and-product-profitability"></a>Оценка рентабельности по клиентам и продуктам

В этой статье объясняется, как можно использовать аналитики в памяти и в режиме реального времени для получения доступа, изучения и получить представление о клиентах и рентабельности продуктов из данных Microsoft Dynamics 365 для данных операций. 

Как часть Dynamics 365 для операций пользователей можно изучить рентабельности для самых клиентов (10-100) на разных уровнях иерархии организации, на основе одного из следующих условий:

-   Сумма продаж
-   Количество
-   Маржа для валовой прибыли
-   Процент маржи

Для этой оценки можно использовать готовый отчет **Ведущие клиенты**, который можно открыть в любом из следующих расположений.

-   **Retail store управления** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**каналов**&gt;**управления Retail store**&gt;**отчеты**&gt;**лучших клиентов отчета**
-   **Запросы и отчеты** разделе Dynamics 365 для операций &gt;**розничной торговли и электронной коммерции**&gt;**запросов и отчетов**&gt;**отчетов продажи**&gt;**лучших клиентов отчета**

Аналогично пользователи могут анализировать прибыльность ключевых продуктов (от 10 до 100) на разных уровнях иерархии организации с учетом одного из следующих критериев.

-   Сумма продаж
-   Количество
-   Маржа для валовой прибыли
-   Процент маржи

Для этой оценки можно использовать готовый отчет **Ведущие продукты**, который можно открыть в любом из следующих расположений.

-   **Retail store управления** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**каналов**&gt;**управления Retail store**&gt;**отчеты**&gt;**лучших продуктов отчета**
-   **Категории и процесса управления продуктом** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**продукты и категории**&gt;**управления Retail store**&gt;**отчетов**&gt;**лучших продуктов отчета**
-   **Запросы и отчеты** разделе Dynamics 365 для операций &gt;**розничной торговли и электронной коммерции**&gt;**запросов и отчетов**&gt;**отчетов продажи**&gt;**лучших продуктов отчета**
