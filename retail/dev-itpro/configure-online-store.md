# <a name="configure-an-online-store"></a>Настройка интернет-магазина

В этой статье приводятся ссылки на разделы, которые помогут централизованно настройки и управления и интерактивного магазина из Microsoft Dynamics 365 для операций.

Разделы, перечисленные в следующей таблице помогут настроить Dynamics 365 для операций - компонентов розничной торговли и Интернет-магазина Retail в Dynamics 365 для операций клиента.

## <a name="configure-an-online-store"></a>Настройка интернет-магазина
| Задача                                                | Подробности                                                                                                                                                                                                                                                                                                                                                   | Разделы                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Настройка розничных компонентов.                        | Настройка и ведение сведений для розничных операций. Эта информация включает магазины, налоги, продукты, подарочные карты, специальные акции и скидки.                                                                                                                                                                                                          | [Настройка и поддержка модуля Retail](https://technet.microsoft.com/en-us/library/hh597201.aspx) (TechNet содержимого для Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                          |
| Настройте иерархию навигации канала розничной торговли.    | Создайте иерархию навигационных категорий канала розничной торговли, которая может быть использована чтобы настроить структуру категорий для продуктов, которые вы предлагаете через Интернет-магазин. Вы определяете иерархию категорий и назначаете продукты, группы атрибутов продукта и значения атрибутов категориям. Назначьте иерархии категорий для Интернет-магазина.                            | [Настройка розничной иерархии](https://technet.microsoft.com/en-us/library/hh580593.aspx) (TechNet содержимого для Microsoft Dynamics AX 2012) [настроить атрибуты и типы атрибутов](https://technet.microsoft.com/en-us/library/hh227548.aspx) (TechNet содержимого для Microsoft Dynamics AX 2012) [Настройка розничных групп атрибутов](https://technet.microsoft.com/en-us/library/jj728713.aspx) (TechNet содержимого для Microsoft Dynamics AX 2012) |

Как минимум интернет-магазин необходимо назначить организационной иерархии, которая включает ассортименты продуктов. | [Настройка Интернет-магазина](https://technet.microsoft.com/en-us/library/jj682095.aspx) (TechNet содержимого для Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |

| Сопоставление атрибутов и добавление метаданных.                   | Выберите параметры, которые определяют, как атрибуты для каждой категории или продукта канала должны вести себя в Интернет-магазине на сайте Microsoft SharePoint.                                                                                                                                                                                              | [Настройка Интернет-магазина](https://technet.microsoft.com/en-us/library/jj682095.aspx) (TechNet содержимого для Microsoft Dynamics AX 2012)                                                                                                                                                                                                                                                                                                     |
