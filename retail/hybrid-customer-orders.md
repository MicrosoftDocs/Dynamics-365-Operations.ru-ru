# <a name="hybrid-customer-orders"></a>Комбинированный заказов клиентов

Заказ клиента гибридную представляет один заказ, который содержит продукты, которые могут быть выполнены из хранилища по клиентам, а также продуктов, которые будут выбраны или стал позже поставляться.

В Microsoft Dynamics 365 для операций - Розничная продажа, можно выбрать либо выполнения всех продуктов или выполнять выбранные продукты для заказа клиента. Строки, помеченные как выполнять автоматически выставляются после создания заказа на продукт, аналогично это то же самое для заказа, который должен быть скомплектованы в большую сторону после создания заказа. Суммы, подлежащей оплате на гибридную заказов определяется Добавление Процент депозита по комплектации и отгрузки товары с использованием полного объема выполняемые строк. Для заказов гибридную для переключения системы между заказа клиента и Продажа без доставки денежных режимов следующим образом:

-   Если все товары в корзине установлены как **выполнения поставки**, заказ будет обрабатываться как проводка Продажа без доставки при оплате наличными.
-   Если все строки в корзине устанавливаются либо **выберите** или **отгрузки поставки**, заказ будет обрабатываться как проводка заказа клиента.

Если строка корзины для покупок и **отобрать выбранное**, **поставки выбранного**, или **выполнения выбранного** будет установлен, только строки для покупок определенной устанавливается с помощью этого способа поставки. В этом случае нисходящего потока операции продолжается в обычном режиме. Тем не менее если **отобрать выбранное**, **поставки выбранного**, или **выполнения выбранного** установлен без строки корзины выбора, новый откроется страница, которые отображаются все строки покупок. В этом окне можно выбрать несколько строк за один раз Задание способа поставки. При использовании этого метода для выбора строки будут переопределены любого предыдущего метода доставки, назначенной данной строке.
