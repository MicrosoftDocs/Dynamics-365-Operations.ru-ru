# <a name="track-commissions-in-pos-using-sales-groups"></a>Отслеживание комиссионные в POS-терминала, с помощью групп продаж

Принято розничной торговли для отслеживания продаж, сотрудник, работавший с клиентом — Предоставление помощи, увеличивая продажи и кросс-продажи контента данной проводке.

Отслеживание продаж по менеджерам — это мера партнеров, продающих возможностями, во время продажи по кассир — это мера скорости и эффективности. Под контролем торгового представителя продаж также часто используются для расчета комиссионных или других поощрения.

## <a name="configuring-a-worker-to-be-a-sales-representative-in-pos"></a>Настройка работника, который будет торгового представителя в POS
Когда работник добавляется группа продавцов, они станут доступны для комиссии и может быть указан в качестве торгового представителя в системе. Работник, который не входит в группу продаж не подходит для комиссии и не будет отображаться в качестве торгового представителя в точке продаж (POS) приложения. В POS список торговых представителей извлекается из всех групп продаж, которые содержат по крайней мере один работник, назначенный магазину. Список отображается в POS, как сочетание продажи код группы и имя (ИД: имя). Группа продавцов по умолчанию может быть назначены работникам для поддержки сценариев, где выбирает розничной торговли для автоматической установки торгового представителя в строках POS. Пользователи могут выбирать любой Группа продавцов, работник входит в состав.

## <a name="functionality-profile-settings"></a>Настройки профиля функциональности
Существует несколько параметров профиля функциональности для магазина, который определяет потока и процесса в POS, включающих торговых представителей.

| **Default to cashier when available** | Если этот параметр включен, строк проводок с группой продажи по умолчанию текущего кассир будут автоматически заполнены в POS. Если кассир не имеет указанной группы продажи по умолчанию, не будет иметь значение. Пользователь может вручную задать группы продаж с помощью кнопки сетки кнопок POS.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Prompt for sales representative**   | Этот параметр имеет три возможных значения: ** нет ** — Если выбран этот параметр, пользователю не будет предложено выбрать группу продавцов. Это значение по-прежнему может быть установлена с помощью группы продаж по умолчанию кассир или вручную с помощью POS кнопки сетки кнопок. **Начало транзакции** — Если выбран этот параметр и либо **по умолчанию используется значение кассир** не включен параметр или текущего кассир не содержит группу продаж по умолчанию, пользователю будет предложено выбрать группу продавцов в начале каждой транзакции. Выбор группы продаж из этой командной строки по умолчанию все последующие строки в выбранную группу продаж. Пользователь может вручную задать группы продаж с помощью кнопки сетки кнопок POS. **Для каждой строки** — Если выбран этот параметр и либо **по умолчанию используется значение кассир** не включен параметр или текущего кассир не содержит группу продаж по умолчанию, пользователю будет предложено выбрать группу продавцов после добавления каждой строки. Пользователь может вручную задать Группа продаж с помощью кнопки сетки кнопок POS. |

После добавления строки пользователя с достаточными разрешениями может удалить из строки группы продаж. ## <a name="displaying-the-sales-representative-information-on-the-pos-transactions-screen"></a>Отображение на экране проводок POS сведения представителя продаж
Выходную форму экрана POS проводки и содержимое можно настроить, используя макеты экрана конструктора и назначенный макет экрана магазины, ККМ или работников. **Торговый представитель** поля могут быть добавлены на вкладку строки в области чека.  При этом отобразится на экране проводки идентификатор указанной группы продаж для каждой строки.

## <a name="adding-sales-representative-operations-to-pos-button-grids"></a>Добавление продаж представитель операциях в POS кнопку сеток
Настройка сетки кнопок, которые включены в макеты экрана, чтобы предоставить доступ к операциям на POS POS пользователи. Кнопки сетки кнопок, которые относятся к Торговый представитель может относиться следующие операции в POS.

|                                           |                                                                                                                                                                                                                                                                                              |
|-------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Операция**                             | **Description**                                                                                                                                                                                                                                                                              |
| Задать торгового представителя в строке          | Эта операция POS отображает список подходящих групп продаж (ИД: имя) для магазина. Выбор Группа продаж из этого списка установит значение для текущей строки операции.                                                                                                            |
| Удалить торгового представителя в строке        | Эта операция POS удаляет текущее значение группы продаж из текущей строки операции.                                                                                                                                                                                                  |
| Набор торгового представителя в проводке   | Эта операция POS отображает список подходящих групп продаж (ИД: имя) для магазина. Выбор группы продаж из этого списка установит значение по умолчанию для текущей проводки. Все существующие строки без назначенного Группа продавцов будет набора, а также любые позднее добавленные строки. |
| Снимите флажок Представитель по проводке | Эта операция POS удаляет текущее значение группы продаж по умолчанию из текущей проводки. Она не влияет на все строки, уже существующие в проводке.                                                                                                                             |

## <a name="calculating-commissions"></a>Расчета комиссионных
Комиссия вычисляется для работников в указанной групп продаж во время разноски журнала операций и разноски заказа на продажу. Сумма комиссии, определяется по Распределение комиссии работника, как указано в группе продаж и параметры расчета комиссии связанных клиента и/или продуктов по проводке.
