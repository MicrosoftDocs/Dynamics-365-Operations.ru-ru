# <a name="add-a-recommendations-control-to-the-transaction-page-on-a-pos-device"></a>Добавьте элемент управления рекомендации на странице проводки на устройстве POS

В этом разделе описывается, как добавить элемент управления рекомендации на экране проводки для точки продажи (POS) устройства, используя конструктор макета экрана в Microsoft Dynamics 365 для операций.

Рекомендации по продукции можно отображать на устройстве POS при использовании Microsoft Dynamics 365 для операции. *Рекомендации по* это номенклатуры, которые ваш клиент может быть заинтересован в на основе их История приобретений, номенклатур в список планируемых покупок и номенклатуры, приобретенные остальных клиентов в интерактивном режиме, а также и физическими магазинами. Чтобы отобразить рекомендации по продукции, необходимо добавить элемент управления на экран проводки, используя конструктор макета экрана.

## <a name="open-layout-designer"></a>Откройте конструктор макета
1.  Последовательно выберите пункты **розничной торговли и электронной коммерции**&gt;**настройку канала**&gt;**настройки POS**&gt;**POS**&gt;**макеты экрана**.
2.  Используйте экспресс-фильтра, чтобы найти на экран, который требуется добавить элемент управления. Например, фильтрацию **код выходной формы экрана** поле, используя значение «F2CP16:9M».
3.  В списке найдите и выберите требуемую запись. Например, выберите "имя: код выходной формы экрана F2CP16:9M: F2CP16:9M".
4.  Щелкните **конструктора макета**.
5.  Следуйте инструкциям на экране для запуска конструктора макета. В ответ на запрос учетных данных, следует ввести те же учетные данные, которые были при запуске конструктора макета из **макеты экрана** страницы.
6.  При входе в отображаться страница с похожими на рисунке ниже. Формат будет отличаться в зависимости от настроек, которые были внесены в хранилище.

[![screenlayout-pic-1](./media/screenlayout-pic-1.png)](./media/screenlayout-pic-1.png) выберите вариант отображения
-----------------------

Существует два варианта конфигурации. Выберите параметр, который лучше всего подходит для вашего хранилища и следуйте инструкциям, чтобы завершить настройку элемента управления. Два параметра являются:
-   Рекомендации отображаются всегда.
-   A **рекомендации** вкладка отображается в сетке в правой части экрана.

#### <a name="to-make-recommendations-always-visible"></a>Для создания рекомендаций всегда видимым

1.  Уменьшите высоту области подробных сведений строки проводки, чтобы он стал высоте панели клиента слева. [](./media/pic-2.png)[![screenlayout-pic-2](./media/screenlayout-pic-2.png)](./media/screenlayout-pic-2.png)
2.  Выберите в меню слева перетаскивание между область сведений строки проводки и сетки кнопок в нижней части экрана проводки центра управления рекомендации. Размер элемента управления, чтобы его длина удовлетворяла в этой области. [](./media/pic-3.png)[![screenlayout-pic-3](./media/screenlayout-pic-3.png)](./media/screenlayout-pic-3.png)
3.  Щелкните **X** чтобы сохранить и выйти из конструктора макета.
4.  В Dynamics 365 for Operations, перейдите к **розничной торговли и электронной коммерции**&gt;**Retail ИТ**&gt;**графики распределения**.
5.  В списке выберите **регистрирует 1090**.
6.  Щелкните **запустить сейчас**.

#### <a name="to-add-a-recommendations-tab-to-the-button-grid-on-the-right-side-of-the-screen"></a>Чтобы добавить вкладку рекомендации для сетки кнопок в правой части экрана

1.  Щелкните правой кнопкой мыши пустое пространство под вкладкой последней в сетке кнопок, находящуюся в правой части страницы.
2.  Щелкните **Настройка**. [![pic-5](./media/pic-5.png)](./media/pic-5.png)
3.  Щелкните **новой вкладке**.
4.  Найти новую вкладку, который был добавлен. Может потребоваться выполнить прокрутку вниз.
5.  В **содержимого** раскрывающегося списка, выберите **рекомендуется продукты**. [![Рис 6](./media/pic-6.png)](./media/pic-6.png)
6.  В **метка** введите имя для вкладки "рекомендации". Например введите «Рекомендуемые продукты».
7.  В **изображения** выберите изображение, которое отображается на вкладке.
8.  Click **OK**. Новая вкладка отображается в сетке кнопок.
9.  Щелкните **X** чтобы сохранить и выйти из конструктора макета.
10. В Dynamics 365 for Operations, перейдите к **розничной торговли и электронной коммерции**&gt;**Retail ИТ**&gt;**графики распределения**.
11. В списке выберите **регистрирует 1090**.
12. Щелкните **запустить сейчас**.
