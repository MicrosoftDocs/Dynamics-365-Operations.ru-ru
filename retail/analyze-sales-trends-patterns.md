# <a name="analyze-sales-trends-and-patterns"></a>Анализ тенденций и моделей продаж

Вы также можете изучить тенденций изменения объемов продаж и шаблонов в режиме реального времени в Microsoft Dynamics 365 для операций - розничной торговли.

В рамках Dynamics 365 для операций, пользователей можно изучить тенденций изменения объемов продаж и шаблонов в режиме реального времени на разных уровнях иерархии организации в течение года с помощью «out» **канале продаж по годам** отчета. Этот отчет можно открыть из любого из следующих местонахождений:

-   **Retail store управления** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**каналов**&gt;**управления Retail store**&gt;**отчеты**&gt;**канале продаж по отчету года**
-   **Финансы Retail store** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**каналов**&gt;**Retail store Финансы**&gt;**отчеты**&gt;**канале продаж по отчету года**
-   **Запросы и отчеты** разделе Dynamics 365 для операций &gt;**розничной торговли и электронной коммерции**&gt;**запросов и отчетов**&gt;**отчетов продажи**&gt;**канале продаж по отчету года**

Пользователи могут анализировать тенденции и модели продаж по часам на разных уровнях иерархии организации за выбранный период с помощью готового отчета **Продажи каналов по часам**. Этот отчет можно открыть из любого из следующих местонахождений:

-   **Управление Retail store** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**каналов**&gt;**управления Retail store**&gt;**отчетов**&gt;**канале продаж по отчету час**
-   **Финансы Retail store** Dynamics 365 рабочее пространство для операций &gt;**розничной торговли и электронной коммерции**&gt;**каналов**&gt;**Retail store Финансы**&gt;**отчеты**&gt;**канале продаж по часам отчета**
-   **Запросы и отчеты** разделе Dynamics 365 для операций &gt;**розничной торговли и электронной коммерции**&gt;**запросов и отчетов**&gt;**отчетов продажи**&gt;**канале продаж по отчету час**
