# <a name="bom-designer-functionality"></a>Функции конструктора спецификаций

Эта статья описывает, как вы можете использовать страницу Конструктор спецификаций для разработки и работы с древовидными структурами для спецификаций (BOMs). Можно щелкнуть "Настройка", чтобы выбрать различных конфигураций и определить, какие сведения отображать на ветках дерева.

Когда вы открываете страницу **Конструктора спецификаций** на странице **Выпущенные продукты**, на ней показывается иерархия спецификаций, которые активны и одобрены для выбранной номенклатуры, сайт заказа по умолчанию и фактическая дата.  

Щелкните **Фильтр**, чтобы изменить изначальный выбор в представлении. Установив принцип отображения как **Выбрано/активно или выбрано**, вы можете выбрать отдельную спецификацию или версии маршрута для использования в представлении. Вы можете выбрать неодобренные и неактивные версии спецификаций, чтобы показывать или вести конструктор спецификаций.  

**Примечание.** Если вы откроете конструктор спецификаций на странице со списком **Спецификации**, в нем не будет показаны сведения по маршруту. В настоящее время вариант выбора спецификации или версии маршрута является свойством спецификации и версии маршрута и применяется ко всем экземплярам конструктора спецификаций.  

Следующие разделы описывают функциональность, которая доступна на различных вкладках конструктора спецификаций.

## <a name="analyzing-a-bom-structure-by-using-the-bom-designer"></a>Анализ структуры спецификации с помощью конструктора спецификаций
Конструктор спецификаций имеет два раздела:

-   Древовидное представление структуры спецификации.
-   Раздел сведений, в котором показаны сведения выбранных данных. Когда вы выберете узел в древовидном представлении, экспресс-вкладки в разделе сведений обновляются в соответствии с этим узлом:
    -   **Сведения о строке спецификации** — сведения о строке спецификации, выбранной в древовидном представлении.
    -   **Данные номенклатуры** — сведения о главной номенклатуре или номенклатуре, которая используется в выбранном узле. Вы можете щелкнуть **Изменить выпущенный продукт** для ведения выбранной номенклатуры.
    -   **Спецификация** — заголовок спецификации, который относится к выбранному узлу.
    -   **Маршрут** — заголовок маршрута, который относится к выбранному узлу.
    -   **Операции маршрута** — предварительный просмотр операций для маршрута. Когда выбрана строка спецификации, назначенная для конкретной операции, операция помечается как **Компонент, необходимый для операций**.

Когда вы используете конструктор спецификаций для того, чтобы просмотреть или редактировать спецификации, которые состоят из одного или нескольких уровней виртуализации, маршрут, который связан с верхней номенклатурой, обычно распространяется на всю иерархию спецификации. Чтобы упростить Обзор, можно заблокировать маршрут верхнего уровня на экране, щелкнув **представление**&gt;**Блокир.**. Чтобы разблокировать маршрут, щелкните **представление**&gt;**маршрут разблокировать**.

## <a name="adding-and-editing-boms-and-bom-lines"></a>Добавление и изменение спецификаций и строк спецификаций
Используйте ** строки Спецификации ** или **Спецификации** функции для изменения строк Спецификации или Спецификации. Когда вы выбираете узел в дереве, тип узла определяет, какие функции доступны.

| Функция                            | описание                                                                                               | Тип и условия узла                                                                                                                                                                                                                                                                       |
|-------------------------------------|-----------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Строки Спецификации &gt;изменение                 | Раскройте диалоговое окно, где вы можете редактировать атрибуты строк спецификации.                                             | Эта функция доступна, когда выбран узел строки спецификации.                                                                                                                                                                                                                                   |
| Строки Спецификации &gt;удаления               | Удалить строку спецификации из выбранной спецификации.                                                                  | Эта функция доступна, когда строка спецификации выбрана и спецификация не заблокирована для редактирования.                                                                                                                                                                                             |
| Строки Спецификации &gt;добавить перед строкой      | Откройте диалоговое окно, где вы можете выбрать вариант продукта для того, чтобы включить перед выбранной строкой спецификации.         | Эта функция доступна, когда выбран узел строки спецификации.                                                                                                                                                                                                                                   |
| Строки Спецификации &gt;добавить компонент Спецификации | Откройте диалоговое окно, где вы можете выбрать вариант продукта для того, чтобы включить в конце выбранной строки спецификации.       | Эта функция доступна, когда у выбранного узла есть выбранная спецификация. Если эта функция недоступна, то версия спецификации может отсутствовать для выбранного варианта номенклатуры. Таким образом, можно нажать кнопку **Спецификации**&gt;**создать версия** для создания отсутствующих версии для выбранного узла. |
| Строки Спецификации &gt;добавить после строки       | Откройте диалоговое окно, где вы можете выбрать вариант продукта для того, чтобы включить после выбранной строки спецификации.          | Эта функция доступна, когда выбран узел строки спецификации.                                                                                                                                                                                                                                   |
| Спецификации &gt;Создание версии             | Создайте новую версию спецификации или спецификацию для варианта продукта выбранного узла.                             | Эта функция доступна, когда выбранная строка спецификации связана с номенклатурой с типом продукции **Спецификация** или **Формула**.                                                                                                                                                  |
| Спецификации &gt;расчета                | Раскройте диалоговое окно, где вы можете выполнить расчет стоимости или цены продажи для выбранного варианта продукта. | Эта функция доступна, когда выбранный узел связан с версией спецификации.                                                                                                                                                                                                         |
| Спецификации &gt;проверка                      | Утвердите и проверьте выбранную спецификацию.                                                                      | Эта функция доступна, когда выбранный узел связан с версией спецификации.                                                                                                                                                                                                         |

## <a name="configuring-the-tree-view"></a>Настройка древовидного представления
Щелкните **Настройка** для настройки информации, отображаемой в древовидном представлении конструктора спецификаций.

| Группа полей | Описание                                                                                                                                                  |
|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Спецификация         | Используйте флажки, чтобы выбрать критерии, которые показываются в древовидной структуре. Конструктор спецификаций отображает выбранные критерии в нижней части обеих вкладок. |
| Маршрут       | Используйте флажки, чтобы выбрать критерии, которые показываются для маршрутов.                                                                                    |
