# <a name="sales-orders-mobile-workspace-for-microsoft-dynamics-365-for-operations-app"></a>Заказы на продажу мобильное рабочее пространство для Microsoft Dynamics 365 для операций приложения

С рабочей области мобильных заказов на продажу, вы может отслеживайте новости в заказы на продажу в любом месте и в любое время.

-------------

| Необходимые условия                                                         | описание                                                                                                                                                                   |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| Dynamics 365 для операций                                          | Убедитесь, что используется в среде, имеющей Microsoft Dynamics 365 для операций версии 1611 и Microsoft Dynamics for платформы операций обновления 3 (ноябрь 2016). |

## <a name="overview"></a>Обзор
Это мобильное рабочее пространство обращается к Dynamics 365 для операций приложения и позволяет просмотреть подробные сведения о каждой заказа на продажу, например, статус заказа, контактных сведений о клиенте и контактной информации принимающая заказ. Мобильное рабочее пространство обеспечивает Просмотр заказов на продажу. Можно просмотреть заказы на продажу по клиенту, или просмотреть все заказы на продажу или просмотра сведений о конкретном заказе на продажу. Мобильное рабочее пространство содержит два представления, чтобы помочь проанализировать заказов продажи в глубину.

### <a name="view-all-sales-orders"></a>Просмотр всех заказов на продажу

В этом представлении перечисляются все заказы на продажу.

-   Используйте один из следующих фильтров для выбора заказов на продажу, которые нужно просмотреть.
    -   Поиск по заказам на продажу
    -   Поиск по счету клиента
    -   Поиск по имени клиента
    -   Поиск по статусу
    -   Поиск по состоянию на запуск в производство
    -   Поиск, Дата и время создания

<!-- -->

-   После выбора заказов на продажу, можно просмотреть сведения по отдельным заказам. В частности можно просмотреть:
    -   Имя и адрес, сведения о клиенте
    -   Даты другой заказ на продажу, например Запрошенная дата отгрузки и подтвержденная дата отгрузки
    -   Контактные данные принимающая заказ
    -   Контактные данные клиента
    -   Строки заказа
    -   Отгрузок, которые показывают, как и когда было отгружено заказа на продажу

### <a name="view-orders-for-a-customer-"></a>Просмотр заказов клиентов ** **

Это представление содержит заказов на продажу по клиенту.

-   Используйте один из следующих фильтров для просмотра заказов для клиента.
    -   Поиск по имени
    -   Поиск по счетам

<!-- -->

-   После установки клиента, можно просматривать:
    -   Имя и группу клиента
    -   Контактные данные клиента
    -   Заказы на продажу клиента и подробные сведения о заказах на покупку:
        -   Имя и адрес, сведения о клиенте
        -   Даты в другой заказ на продажу
        -   Контактные данные принимающая заказ
        -   Контактные данные клиента
        -   Строки заказа
        -   Отгрузок, которые показывают, как и когда было отгружено заказов на продажу

## <a name="get-started"></a>Начать
Выполните следующие действия, чтобы начать работу с заказы на продажу мобильного рабочего пространства на мобильном устройстве.

1.  Загрузите и установите Microsoft Dynamics 365 для операций приложения на хранилище мобильных приложений.
2.  Запустите приложение на устройстве.
3.  Введите URL-адрес Dynamics 365.
4.  Введите компании для входа в систему. Например, введите **USMF**.
5.  При первом входе, появится запрос ввести имя пользователя и пароль для данных Microsoft Dynamics 365 для счета операций. Введите свои учетные данные. После входа см. рабочие области доступны для вашей компании.

Чтобы просмотреть рабочие пространства на мобильного приложения, необходимо опубликовать нужных рабочих областей Dynamics 365 для операций приложения.

1.  Запустите Dynamics 365 для операций.
2.  Последовательно выберите пункты **администрирование системы**&gt;**установки**&gt;**параметров системы**.
3.  Выберите **управление мобильного приложения**.
4.  Выберите рабочую область, чтобы опубликовать мобильных платформ.
5.  Выберите **публикация рабочей области**.
6.  Обновление устройства для просмотра опубликованных рабочих областей.

## <a name="view-information-about-sales-orders-for-a-customer"></a>Просмотр сведений о заказах на продажу для клиента
1.  На мобильном устройстве, выберите **заказы на продажу** рабочей области.
2.  Выберите **Просмотр заказов для клиента**.
3.  Используйте ** счета ** или ** имя клиента ** сведения для поиска нужного клиента.
4.  Выберите клиента.
5.  Выберите **контактная информация** или **заказы на продажу**.
6.  Если **заказы на продажу** — флажок установлен, будет отображен список заказов на продажу для клиента.
7.  Выберите **заказ на продажу**.
8.  Здесь можно просмотреть сведения о строк заказа на продажу, отгрузок, контактных сведений о клиенте и контактной информации принимающая заказ.
