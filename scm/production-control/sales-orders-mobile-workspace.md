# <a name="sales-orders-mobile-workspace-for-microsoft-dynamics-365-for-operations-app"></a>Мобильная рабочая область заказов на продажу для приложения Microsoft Dynamics 365 for Operations

С мобильной рабочей областью заказов на продажу вы может отслеживать актуальные данные о заказах на продажу в любом месте и в любое время.

| Необходимые условия                                                         | описание                                                                                                                                                                   |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Сведения о мобильной платформе Microsoft Dynamics 365 for Operations | [Мобильная платформа Dynamics 365 for Operations](/dynamics365/operations/dev-itpro/mobile-apps/mobile-platform)                                                              |
| Dynamics 365 for Operations                                          | Убедитесь, что используется среда с установленными Microsoft Dynamics 365 for Operations версии 1611 и обновлением 3 платформы Microsoft Dynamics for Operations (ноябрь 2016). |

| Мобильное устройство, на которое установлено приложение Dynamics 365 for Operations | Загрузите приложение Dynamics 365 for Operations из магазина мобильных приложений.                                                                                                      |

## <a name="overview"></a>Обзор
Эта мобильная рабочая область обращается к приложению Dynamics 365 for Operations и позволяет просматривать подробные сведения о каждом заказе на продажу, например статус заказа, контактные сведения клиента и контактные сведения получателя заказа. Мобильная рабочая область обеспечивает мгновенный просмотр заказов на продажу. Можно просматривать заказы на продажу по клиентам, просмотреть все заказы на продажу или просмотреть сведения о конкретном заказе на продажу. Мобильная рабочая область содержит два представления, чтобы помочь глубже анализировать заказы продажу.

### <a name="view-all-sales-orders"></a>Просмотр всех заказов на продажу

В этом представлении перечисляются все заказы на продажу.

-   Используйте один из следующих фильтров для выбора заказов на продажу, которые нужно просмотреть.
    -   Поиск по заказу на продажу
    -   Поиск по счету клиента
    -   Поиск по имени клиента
    -   Поиск по статусу
    -   Поиск по статусу запуска в производство
    -   Поиск по дате и времени создания

<!-- -->

-   После выбора заказов на продажу можно просматривать сведения для конкретных заказов. В частности, можно просмотреть:
    -   Имя клиента и сведения об адресе
    -   Различные даты заказа на продажу, например запрошенная дата отгрузки и подтвержденная дата отгрузки
    -   Контактная информация получателя заказа
    -   Контактные данные клиента
    -   Строки заказа
    -   Отгрузки, которые показывают, как и когда был отгружен заказ на продажу

### <a name="view-orders-for-a-customer-"></a>Просмотр заказов для клиента** **

В этом представлении перечислены заказы на продажу по клиенту.

-   Используйте один из следующих фильтров для просмотра заказов для клиента.
    -   Поиск по имени
    -   Поиск по счету

<!-- -->

-   После выбора клиента можно просматривать:
    -   Имя и группа клиентов
    -   Контактные данные клиента
    -   Заказы на продажу для клиента и подробные сведения о заказах на покупку:
        -   Имя клиента и сведения об адресе
        -   Различные даты заказа на продажу
        -   Контактная информация получателя заказа
        -   Контактные данные клиента
        -   Строки заказа
        -   Отгрузки, которые показывают, как и когда были отгружены заказы на продажу

## <a name="get-started"></a>Начать
Выполните следующие действия, чтобы приступить к работе с помощью мобильной рабочей области заказов на продажу на мобильном устройстве.

1.  Загрузите приложение Microsoft Dynamics 365 for Operations из магазина мобильных приложений и установите его.
2.  Запустите приложение на устройстве.
3.  Введите URL-адрес Dynamics 365.
4.  Введите компанию для входа в систему. Например, введите **USMF**.
5.  При первом входе появится запрос ввести имя пользователя и пароль для учетной записи Microsoft Dynamics 365 for Operations. Введите свои учетные данные. После входа вы увидите доступные рабочие области для вашей компании.

Чтобы просмотреть рабочие области в мобильном приложении, необходимо сначала опубликовать нужные рабочие области для приложения Dynamics 365 for Operations.

1.  Запустите Dynamics 365 for Operations.
2.  Перейдите в раздел **Администрирование системы** &gt; **Настройка** &gt; **параметры системы**.
3.  Выберите **Управление мобильным приложением**.
4.  Выберите рабочую область, чтобы опубликовать ее для мобильной платформы.
5.  Выберите **Публикация рабочей области**.
6.  Обновите устройство для просмотра опубликованных рабочих областей.

## <a name="view-information-about-sales-orders-for-a-customer"></a>Просмотр сведений о заказах на продажу для определенного клиента
1.  На мобильном устройстве выберите рабочую область **Заказы на продажу**.
2.  Выберите **Просмотр заказов для клиента**.
3.  Используйте сведения **Счет **или **Имя клиента ** для поиска нужного клиента.
4.  Выберите клиента.
5.  Выберите **Контактная информация** или **Заказы на продажу**.
6.  Если выбран пункт **Заказы на продажу**, отображается список заказов на продажу для клиента.
7.  Выберите **Заказ на продажу**.
8.  Здесь можно просмотреть сведения о строках заказа на продажу, отгрузки, контактные данные клиента и контактные данные получателя заказа.
