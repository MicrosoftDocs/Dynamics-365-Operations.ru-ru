# <a name="create-working-time-scheduling-properties"></a>Создание свойств графика рабочего времени

[!include[task guide banner](../../includes/task-guide-banner.md)]

Свойства графика рабочего времени используются для классификации интервалов рабочего времени и определения настроек, которые должны учитывать операторы. Следующая процедура используется для определения свойства графика рабочего времени и связанных значений свойства. Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USMF или собственные данные.

1. Перейдите в раздел "Все рабочие области" > "Управление жизненным циклом ресурса".
2. Щелкните Свойства.

## <a name="create-property"></a>Создание свойства
1. Щелкните "Создать".
2. В поле "Свойство" введите значение.
3. В поле "Имя" введите значение.

## <a name="create-property-values"></a>Создание значений свойства
1. В списке пометьте выбранную строку.
2. В поле "Имя" введите значение.
3. В поле "Значение" введите значение.
4. Нажмите кнопку "Сохранить".
