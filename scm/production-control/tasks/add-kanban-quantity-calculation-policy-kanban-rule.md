# <a name="add-a-kanban-quantity-calculation-policy-to-a-kanban-rule"></a>Добавление политики расчета количества канбанов в правило канбана

[!include[task guide banner](../../includes/task-guide-banner.md)]

Эта процедура заключается в создании политики вычисления количества канбанов и добавлении ее к правилу канбана для оптимизации размера и количества канбанов. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Эта процедура предназначена для менеджера потока создания ценности. Эта процедура является предварительным условием для создания процедуры "Расчет предложений количества канбанов". 


## <a name="create-a-kanban-quantity-calculation-policy"></a>Создание политики расчета количества канбанов
1. Перейдите в раздел "Управление производством" > "Периодические задачи" > "Расчет количества канбанов" > "Политики расчета количества канбанов".
2. Щелкните "Создать".
3. В поле "Имя" введите значение.
    * Например, введите Speaker2016.  
4. В поле "Сводный план" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
5. В списке найдите и выберите требуемую запись.
    * Выберите StaticPlan, чтобы рассчитать спрос.  
6. В списке перейдите по ссылке в выбранной строке.
7. Нажмите кнопку "Сохранить".
8. В поле "Минимальное количество канбанов" введите "1".
    * Это дополнительное количество канбанов, включаемое в расчет количества канбанов.  
9. Установите коэффициент безопасности равным 1.
    * Это коэффициент, используемый для расчета дополнительного количества резервного запаса.  
10. В поле "Дни перед" введите "30".
    * Это число дней перед датой расчета количества канбанов, включаемое в расчет спроса.  
11. В поле "Дни до" введите "30".
    * Это число дней после даты расчета количества канбанов, включаемое в расчет спроса.  Формула, используемая для расчета, показана с фактическими значениями. Например, количество канбанов = ((средний ежедневный спрос x время упреждения x 2,00) / количество продукта на единицу обработки) + 1  
12. Закройте страницу.

## <a name="add-the-kanban-quantity-calculation-policy-to-a-kanban-rule"></a>Добавление политики расчета количества канбанов к правилу канбана
1. Перейдите в раздел "Управление сведениями о продуктах" > "Бережливое производство" > "Правила канбана".
2. В списке найдите и выберите требуемую запись.
    * Выберите для этой процедуры правило канбана 000020.  
3. В списке перейдите по ссылке в выбранной строке.
4. Переключите развертывание раздела "Политики расчета количества канбанов".
5. Нажмите кнопку Добавить.
    * Добавьте политику расчета количества канбанов, созданную в предыдущей подзадаче.  
6. В списке пометьте выбранную строку.
7. В поле "Имя" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
8. В списке перейдите по ссылке в выбранной строке.
    * Выберите политику Speaker2016, созданную в предыдущей подзадаче.
