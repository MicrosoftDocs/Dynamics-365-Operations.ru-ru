# <a name="report-progress-on-a-mobile-job-device"></a>Проверка хода выполнения на мобильном устройстве задания

[!include[task guide banner](../../includes/task-guide-banner.md)]

В этой процедуры показано, как запустить производственное задание в регистрационной форме устройства задания и как сообщать о ходе выполнения задания.



Для выполнения этой процедуры необходимо иметь роль администратора системы или оператора станка, связанную с учетной записью пользователя.

1. Перейдите в раздел "Управление производством" > "Управление производством" > "Карта задания — устройство".
2. В поле WorkerTextField введите значок работника. В демонстрационных данных USMF для пользователя Christina Portra введите "123".
3. Щелкните "Войти в систему".
4. Нажмите кнопку "Фильтр".
5. Установите или снимите флажок "Применить фильтр конфигурации". Если установить фильтр, можно использовать производственное подразделение 110 в USMF.
6. В поле "Производственное подразделение" выберите группу ресурсов, над производственными заданиями которой может работать работник.
7. В списке перейдите по ссылке в выбранной строке.
8. Нажмите кнопку "OК".
9. Нажмите кнопку "Запустить задание".
10. Нажмите кнопку "OК".
11. Нажмите кнопку "Проверить ход выполнения".
12. Нажмите кнопку "OК".
13. Нажмите кнопку "Следующее задание".
14. Нажмите "Распределено" для просмотра обзора всех производственных заданий.
15. Закройте страницу.
16. Нажмите кнопку "Прерывание".
17. В списке найдите и выберите требуемую запись.
18. Нажмите кнопку "OК".
19. Нажмите кнопку "Уход".
20. Выберите команду выхода из системы.
21. Нажмите кнопку "OК".
22. В поле WorkerTextField снова войдите в систему. Можно выбрать работника "123" в демонстрационных данных USMF.
23. Щелкните "Войти в систему".
24. Нажмите "Остановить прерывание".
25. Нажмите кнопку "Действие".
26. Щелкните "Отмена".
27. Нажмите кнопку "Уход".
28. Выберите команду времени ухода.
29. Нажмите кнопку "OК".
30. Выберите причину, по которой вы уходите раньше.
