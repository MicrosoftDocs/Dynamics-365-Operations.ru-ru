# <a name="inventory-tag-counting"></a>Учет маркеров запасов

Эта статья предоставляет информацию о Меченной инвентаризации, которые используются для сравнения фактического содержимого склада с запасами в наличии. 

Путем создания строк в **учет маркеров** страницы, установите номер маркера на каждой номенклатуры запасов, такие как число от 1 до 500. Во время пересчета вводится код номенклатуры и количество на соответствующем маркере. Затем этот маркер можно использовать как основу для ввода данных в журнале учета маркеров. После разноски журнала учета маркеров создается новый журнал учета на странице **Инвентаризация**. Новый журнал основан на строках созданного журнала учета маркеров. Чтобы учесть маркеры номенклатур по определенной складской аналитике, выберите аналитику на странице **Отобразить аналитики**, которая открывается при создании журнала учета маркеров. Например, чтобы учесть номенклатуры на определенном складе, установите флажок **Склад**. Если выбран ползунок **Блокир. номенкл. при инвентаризации** на странице **Параметры модуля "Управление запасами и складами"**, номенклатуры невозможно физически обновить во время инвентаризации. Однако номенклатуры в журналах учета маркеров не блокируются во время инвентаризации. Складские проводки не создаются, пока строки учета маркеров не будут разнесены и перенесены в журнал учета. Если маркеры вводятся случайным образом и требуется идентифицировать отсутствующие маркеры, щелкните заголовок столбца **Маркер**, чтобы отсортировать строки по маркеру.
