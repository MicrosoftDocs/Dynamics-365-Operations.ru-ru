# <a name="packing-materials-and-fees"></a>Упаковочные материалы и сборы

[!include[banner](../includes/banner.md)]


Сборы за упаковочные материалы выплачиваются компании утилизации с определенными интервалами. Для каждого материала, из которого состоит единица упаковки, должна выплачиваться сумма за единицу веса. Сборы за упаковочные материалы рассчитываются и учитываются в отчетах, но никакие проводки ГК не разносятся, так как сборы не рассматриваются в качестве налогов, которые необходимо платить налоговому органу.

Вес и сборы для упаковочных материалов рассчитываются как для строк заказов на продажу, так и для строк заказов на покупку.

Можно определить одну или несколько единиц упаковки для номенклатуры, для группы упаковки номенклатур или для всех номенклатур. Единица упаковки состоит из различных упаковочных материалов и их веса, а также количества номенклатур, входящих в единицу упаковки. Каждому определенному типу упаковочных материалов назначается код упаковочных материалов. На основе кода упаковочных материалов можно указать цену за заданный период. Сборы за упаковочные материалы рассчитываются на основе этих сведений.

| **Примечание**                                                                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------------------|
| Даже если компания не оплачивает сборы за упаковочный материал, эту функцию можно использовать, чтобы рассчитать статистику для веса упаковочных материалов. |

## <a name="setup-requirements-for-packing-material-fees"></a>Настройка требований к сборам за упаковочные материалы
Перед расчетом веса или сборов для упаковочных материалов или обоих значений, необходимо создать следующие базовые данные.

-   Группы упаковки. Создайте группы упаковки для назначения номенклатурам.
-   Коды упаковочных материалов. Создайте коды упаковочных материалов для каждого определенного типа упаковочных материалов.
-   Единицы упаковки. Определите единицу упаковки для номенклатуры, группы упаковки или для всех номенклатур. Для каждой единицы упаковки определите включаемые упаковочные материалы, назначьте вес и в поле "Коэффициент единицы упаковки" введите коэффициент преобразования из единиц складского учета.
-   Сборы за упаковочные материалы. Определите сборы за упаковочный материал для кода упаковочного материала. Для каждого типа материала определите период действия, цену за материал, валюту и единицу.

## <a name="packing-units-on-sales-order-lines"></a>Единицы упаковки в строках заказа на продажу
При создании строки заказа на продажу система проверяет, были ли определены единицы упаковки для номенклатуры. Если единицы упаковки были указаны, в строке заказа на продажу обновляются указанная единица упаковки и количество единиц упаковки. Количество единиц упаковки рассчитывается путем деления заказанного количества на число номенклатур в выбранной единице упаковки. Если единица упаковки не была определена, можно вручную ввести единицу упаковки и количество единиц упаковки в строке заказа на продажу. Единицу упаковки и количество единиц упаковки также можно изменить при разноске строки заказа на продажу. Это имеет значение, если доставка или выписывание накладной для строки заказа на продажу выполнены только частично. Проводки по упаковочному материалу создаются при выставлении накладной по строке заказа на продажу. Проводки по упаковочным материалам содержат вес упаковочных материалов для строки продажи. После выставления накладных проводки можно отредактировать, а затем создать новые.

## <a name="packing-units-on-purchase-order-lines"></a>Единицы упаковки в строках заказа на покупку
Система не создает проводки по упаковочному материалу для строк заказа на покупку. Проводки для строк заказа на покупку, но которым выставлена накладная, создаются вручную на странице **Проводки по упаковочному материалу**.

## <a name="set-up-customer-packagingmaterialfee-license-numbers"></a>Настройка номера лицензий сборов за упаковочный материал клиента
Если клиенты уплачивают сбор за упаковочные материалы, укажите номера лицензий сборов за упаковочный материал для таких клиентов на странице **Клиенты**. После назначения клиенту номера лицензии сборы за упаковочный материал рассчитываются автоматически при выставлении счета к заказу на продажу. После выставления накладной флажок **Рассчитать сбор** на странице **Проводки по упаковочному материалу** сбрасывается, поскольку нет необходимости готовить и распечатывать отчет. Предусмотрена возможность печати веса упаковочных материалов в накладной и уведомления клиента о необходимости уплаты сборов. 

Если сбор за упаковочный материал уплачивает ваша компания, не вводите номера лицензий клиентов. После выставления накладной флажок **Рассчитать сбор** устанавливается на странице **Проводки по упаковочному материалу**. Это означает, что расчет сборов производится при создании отчета. Можно распечатать вес на накладной и указать, что ваша компания уплачивает сборы.

## <a name="print-packaging-material-weights-on-invoices"></a>Распечатывание веса упаковочных материалов на накладных
Можно распечатать вес упаковочных материалов в накладной и указать, кто уплачивает сборы за упаковочные материалы. Вес суммируется по упаковочному коду.
