# <a name="piece-picking-confirmation"></a>Поштучное подтверждение комплектации

[!include[banner](../includes/banner.md)]

Поштучная комплектация позволяет подтверждать каждую штуку запасов посредством работ по комплектации или инвентаризации на мобильном устройстве При комплектации можно подтвердить количество работы, подлежащей обработке, вплоть до количества, указанного в работе для комплектации. Для работы по инвентаризации вы можете сканировать запасы, которые подсчитываете, и отслеживать общую сумму.

При включении поштучной комплектации подтверждение продукта устанавливается автоматически. Для работ типа "комплектация" включается максимальное число штук. Это позволяет задать максимальное число штук, которые должны быть подтверждены во время рабочего процесса. Максимальное количество основывается на текущей единице измерения работы, которая обрабатывается. Для работ типа "инвентаризация" ввод максимума не предусмотрен.

Можно также использовать количество и единицу измерения, связанные со сканируемым штрих-кодом. Это будет работать для оформления прихода входящих потоков, включая получение грузомест со смешанными номенклатурами, номенклатуры заказа на покупку, номенклатуры заказа на перемещение и номенклатуры загрузки. Это также работает для поштучной комплектации, где сканирование штрих-кода будет добавлять количество к общему количеству подтвержденных штук, с преобразованием между единицей измерения в штрих-коде и единицей измерения работы. Если при подсчете единицы измерения в штрих-коде подтверждается, что количество разрешено для подсчета в группе упорядочения, количество будет добавлено к общему количеству.

## <a name="where-it-applies"></a>Где это применяется

Поштучная комплектация подходит для всех работ по инвентаризации и для первоначальной комплектации для любого типа работы. Поштучная комплектация не применяется, если номенклатура контролируется по серийным номерам или если это комплектация для производства или комплектация по канбану с грузоместа, и номенклатура перемещается в промежуточное местонахождение.

## <a name="set-up-piece-picking"></a>Настройка поштучной комплектации

1.  В меню мобильного устройства откройте форму настройки для подтверждения работы: Управление складом> **Управление складом** > **Настройка** > **Мобильное устройство** > **Пункты меню мобильного устройства**. 
2. В пунктах меню мобильного устройства откройте "Настройка подтверждения работы".

Следующие параметры становятся доступны для выбора, когда тип работы — "комплектация" или "инвентаризация".

| Параметр        | описание   | 
| ------------- | ------------- |
| Поштучное подтверждение комплектации   | Доступно для типов работы "комплектация" и "инвентаризация". Подтверждение продукта устанавливается автоматически. Позволяет подтверждать каждую штуку запасов с мобильного устройства. | 
| Максимальное число штук     | Доступно для типа работы "комплектация", если включено подтверждение комплектации. Устанавливает ограничение на количество штук, которые необходимо подтвердить. |
