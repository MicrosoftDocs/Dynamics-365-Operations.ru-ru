# <a name="modify-a-demand-forecast-manually"></a>Изменение прогноза спроса вручную

[!include[task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как внести изменения в прогноз для номенклатуры. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Эта запись предназначена для планировщика производства. 


## <a name="modify-the-forecast-for-an-item"></a>Изменение прогноза для номенклатуры
1. Щелкните "Управление сведениями о продукте" > "Продукты" > "Выпущенные продукты".
2. В списке найдите и выберите требуемую запись.
    * Выберите номенклатуру, прогноз для которой требуется изменить. Например, можно выбрать номенклатуру D0001.  
3. В области действий щелкните "План".
4. Щелкните "Прогноз спроса".
5. В списке пометьте выбранную строку.
    * Если строк прогноза нет, создайте новую строку, щелкнув "Создать" на панели приложения.  
6. В поле "Продаваемое количество" введите число.
    * Это число представляет собой прогнозное количество для номенклатуры.  
7. Нажмите кнопку "Сохранить".

## <a name="modify-the-forecast-in-excel"></a>Изменение прогноза в Excel
1. Щелкните "Открыть в Microsoft Office".
2. Щелкните "Редактировать прогноза спроса в Excel".
    * В Excel можно добавлять, удалять и изменять строки прогноза спроса. Если вы не видите данные в Excel, необходимо войти в Microsoft Dynamics 365 for Finance and Operations, Enterprise edition, установив флажок "Оставаться в системе"; кроме того, необходимо доверять приложению для подключения к данным.
