# <a name="remove-outliers-from-historical-transaction-data-when-calculating-a-demand-forecast"></a>Удаление выбросов из исторических транзакционных данных при расчете прогноза спроса

В этой статье описывается, как удалить выбросы из исторических данных, которые использовались для расчета прогноза спроса. Путем исключения выбросов, вы можете повысить точность прогноза.

Можно исключить выбросы, чтобы повысить точность прогноза. Эта задача не является обязательной. Обзор процесса:

1.  Щелкните **сводное планирование**&gt;**установки**&gt;**прогнозирование спроса**&gt;**удаления выбросов** чтобы открыть **удаления выбросов** страницу, где можно использовать запрос для выбора проводок, чтобы исключить.
2.  Выберите компанию, к которой применяется запрос, а затем введите имя и описание. В поле **Дата запроса** автоматически задается текущая дата.
3.  Установите флажок **Активно**, чтобы исключить проводки, найденные в результате запроса, из исторических данных. Эта настройка вступит в силу при создании базового прогноза.
4.  На странице **Запрос удаления выбросов** можно добавить, удалить и выбрать критерии, определяющие проводки, которые будут исключены при расчете базового прогноза. Например, выберите определенную номенклатуру или проводку по заказу, которую необходимо исключить.
5.  Щелкните **Отобразить проводки**. На странице **Проводки-выбросы** содержится список проводок, которые отвечают критериям, указанным в запросе, и которые будут исключены из исторических данных при расчете прогноза спроса.

**Примечание.** Вы можете также создать запрос, который основан на существующем запросе. Выберите запрос для копирования и после этого щелкните **Дублировать**. В поле **Дата запроса** указывается версия. Вы можете использовать запрос "как есть" или можете щелкнуть **Изменить запрос**, чтобы изменить критерии. Вы можете также изменить имя и описание нового запроса.
