# <a name="coverage-settings"></a>Параметры покрытия

При сводном планировании для вычисления потребностей в номенклатуре используются параметры покрытия. 

Параметры покрытия можно задать несколькими способами:

-   Определение параметров покрытия для группы покрытия. Можно создать группу покрытия, которая содержит настройки для всех продуктов, связанных с этой группой покрытия. Щелкните **сводное планирование &gt;установки &gt;покрытия &gt;группы покрытия**, чтобы создать группу покрытия. Затем можно связать группу покрытия с продуктом. Если связь относится к конкретному месту, складу или аналитике продукта, используйте поле **Группа покрытия** на странице **Покрытие номенклатуры**. Если связь является общей и не зависит от аналитик продукта, используйте **Группу покрытия** на экспресс-вкладке **План** на странице **Сведения о продукте**. Если группа покрытия не связана с продуктом, то в сводном планировании используется **Общая группа покрытия**, которая указана на странице **Параметры сводного планирования** в качестве параметра по умолчанию.

-   Определение настроек покрытия для продукта. Вы можете создавать настройки покрытия для определенного продукта. Щелкните **Управление сведениями о продукте &gt;продукты &gt;выпущенные продукты**. Выбор продукта, на **панели операций**на **план** вкладке в **группы покрытия**, щелкните **покрытие номенклатуры** чтобы открыть **покрытие номенклатуры** страницы. Если продукт уже связан с группой покрытия, вы можете переопределить настройки группы покрытия, используя поле **Переопределить**. Настройки покрытия на странице **Покрытие номенклатуры** имеют приоритет над настройками на странице **Группа покрытия**.

<!-- -->

-   Определение настроек покрытия для продукта с помощью мастера. Мастер представляет собой пошаговое руководство по настройке основных параметров покрытия номенклатуры. На странице **Покрытие номенклатуры** щелкните **Мастер**, чтобы открыть **Мастер покрытия номенклатуры**.

<!-- -->

-   Определение параметров покрытия для группы аналитик. Щелкните **Управление сведениями о продукте &gt;обычные &gt;выпущенные продукты**. На ** запущено со сведениями о продукции ** на странице **Общие** вкладке **Администрирование** щелкните **группу аналитик хранения** ссылок. На странице **Группа аналитик хранения** выберите поле **План покрытия по аналитикам**, чтобы создать настройки покрытия для аналитики в группе аналитик хранения. Должны иметь все аналитики продукта, такие как конфигурация, цвет, размер, стиль, **план покрытия по аналитикам** выбранного поля.
