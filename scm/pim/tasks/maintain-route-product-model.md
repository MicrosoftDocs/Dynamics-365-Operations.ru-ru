# <a name="maintain-a-route-for-a-product-model"></a>Ведение маршрута для модели продукта

[!include[task guide banner](../../includes/task-guide-banner.md)]

Выполнение этой процедуры требует наличия модели конфигурации продукта. Эта процедура использует модель динамика класса Hi-End в демонстрационной компании USMF для описания процесса.


## <a name="add-a-route-operation"></a>Добавление операции маршрута
1. Щелкните "Определение модели вариантов продукта".
2. Щелкните "Модели конфигурации продукта".
3. В списке найдите и выберите требуемую запись.
    * Выберите модель динамика класса Hi--End для этого упражнения.  
4. В списке перейдите по ссылке в выбранной строке.
5. Разверните раздел "Операции маршрута".
6. Нажмите кнопку Добавить.
7. В поле "Имя" введите значение.
8. В поле "Описание" введите значение.
9. Нажмите кнопку "Сохранить".

## <a name="enter-route-operation-details"></a>Ввод сведений об операции маршрута
1. Щелкните "Сведения об операции маршрута".
2. В поле "Операция" введите или выберите значение.
3. В поле Операция № п/п введите число.
    * Номера операции определяют последовательность маршрута.  
    * Каждое свойство операции маршрута может получить статическое значение или быть сопоставленным с атрибутом. Сопоставление с атрибутом приведет к тому, что значение будет задано в рамках конфигурации.  
4. В поле "Маршрутная группа" введите или выберите значение.
    * Маршрутная группа определяет важное поведение для учета затрат, потребления и настройки.  
5. Перейдите на вкладку "Настройка".
6. Перейдите на вкладку "Время".
7. В поле "К-во процесса" введите число.
    * Укажите количество, которое будет обработано в течение одной операции.  
8. В поле "Часы/время" введите число.
    * Введите коэффициент времени.  
9. Установите флажок "Задать".
10. В поле "Время выполнения" введите число.
    * Определите время обработки для указанного количества.  
11. Нажмите на вкладку "Потребности ресурса".
12. Нажмите кнопку Добавить.
13. В списке пометьте выбранную строку.
14. В поле "Тип потребности" выберите вариант.
    * Укажите при необходимости определенные ресурсы или возможности, которыми они должны обладать.  
15. В поле "Потребность" введите или выберите значение.
16. Нажмите кнопку "OК".
