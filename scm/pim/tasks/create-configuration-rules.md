# <a name="create-configuration-rules"></a>Создание правил конфигурации

[!include[task guide banner](../../includes/task-guide-banner.md)]

Эта процедура заключается в создании правил конфигурации, которые можно использовать для конфигурации на основе аналитик, чтобы принудительно устанавливать или запрещать определенные сочетания строк спецификации. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Это седьмая из восьми процедур, в которых поясняется, как строить сочетания для конфигурации на основе аналитик.

1. Перейдите в раздел "Управление сведениями о продукте" > "Спецификации и формулы" > "Спецификации".
2. В списке найдите и выберите требуемую запись.
    * Найдите и выберите спецификацию для конфигурации на основе аналитик.  
3. В области действий щелкните "Параметры".
4. Щелкните "Изменить режим просмотра".
5. Щелкните Вид заголовка.
    * Откройте представление заголовка для доступа к экспресс-вкладке "Конфигурационный маршрут".  
6. Разверните или сверните раздел "Конфигурационный маршрут".
    * Экспресс-вкладка "Конфигурационный маршрут" должна находиться в развернутом режиме.  
7. Щелкните "Правила конфигурации".
8. Щелкните "Создать".
9. В списке пометьте выбранную строку.
10. В поле "Код номенклатуры" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
    * Отображаются номенклатуры в текущей конфигурационной группе. Выберите одну, представляющую условие в правиле.  
11. В списке перейдите по ссылке в выбранной строке.
12. В поле "Метод" выберите вариант.
    * Можно принудительно выбирать или снимать выбор с номенклатур из другой конфигурационной группы.  
13. В поле "Производная группа" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
14. В списке найдите и выберите требуемую запись.
15. В списке перейдите по ссылке в выбранной строке.
    * Выберите требуемую конфигурационную группу.  
16. В поле "Код производной номенклатуры" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
17. В списке перейдите по ссылке в выбранной строке.
    * Выберите номер номенклатуры, которая будет выбираться или выбор с которой будет сниматься в зависимости от выбранного метода.  
18. Закройте страницу.
