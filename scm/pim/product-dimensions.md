# <a name="product-dimensions"></a>Аналитики продуктов

[!include[banner](../includes/banner.md)]


Предусмотрено четыре аналитики продукта — цвет, конфигурация, размер и стиль. Аналитики продукта объединяются в группы аналитик, а группы аналитик назначаются шаблонам продуктов. Комбинации аналитик продукта определяют, как определяются варианты продуктов.

Аналитики продукта — это характеристики, которые позволяют определить вариант продукта. Вы можете использовать комбинации аналитик продукта для определения вариантов продуктов. Необходимо определить хотя бы одну аналитику продукта для шаблона продукта, чтобы создать вариант продукта.
Варианты продуктов
----------------

Варианты продуктов также называются номенклатурой. Номенклатура — это материальный продукт в отличие от услуги. Можно также определить шаблон продукта типа "Услуга". Используя тип "Услуга", можно указать варианты продукта, включающие услуги. Например, можно указать шаблон продукта для консультаций и варианты продуктов для работы, которая выполняется старшими и младшими консультантами.

## <a name="product-dimensions"></a>Аналитики продуктов
Доступны следующие аналитики продукта: конфигурация, цвет, размер и стиль. Вариант продукта можно создать на основе значений аналитик продукта.

Значения аналитик продукта, такие как размер, цвет и стиль, можно создать на страницах **Размер**, **Цвет** и **Стиль**, которые можно открыть из следующих расположений: **Управление сведениями о продукте** &gt; **Настройка** &gt; **Группы аналитик и вариантов** &gt; **Размеры/Цвета/Стили**. Значения аналитик продуктов для аналитики "Конфигурация" обычно создаются с помощью конфигуратора продукции или конфигуратора на основе аналитик. Аналитики продуктов можно также создавать и поддерживать на странице **Аналитики продуктов**, которую можно открыть из следующих местоположений:
-   Щелкните **Управление сведениями о продукте** &gt; **Продукты** &gt; **Шаблоны продукта**. В **области действий** щелкните **Аналитики продуктов**.
-   Щелкните **Управление сведениями о продукте** &gt; **Продукты** &gt; **Все продукты и шаблоны продукции**. Выберите шаблон продукта. В **области действий** щелкните **Аналитики продуктов**.
-   Щелкните **Управление сведениями о продукте** &gt; **Выпущенные продукты**. Выберите шаблон продукта. В **области действий**, щелкните **Продукт**. В группе **Шаблон продукта** щелкните **Аналитики продуктов**.

Число вариантов, которые можно создать для номенклатуры, ограничено числом возможных комбинаций аналитик продукта.
| **Совет**                                                                                                                                              |
|------------------------------------------------------------------------------------------------------------------------------------------------------|
| При использовании продукта, например, в строке заказа, вы выбираете аналитики продукта для указания варианта продукта, с которым требуется работать. |

## <a name="example"></a>Пример
Компания продает джинсы из денима. Для номенклатуры "Джинсы" используются аналитики "Цвет" и "Размер". Джинсы продаются в трех различных цветовых решениях и могут быть шести различных размеров. Цвета: Голубой, Черный, Коричневый Размеры: XS, S, M, L, XL, XXL Не все размеры доступны во всех трех цветах. Если бы все комбинации были доступны, будет создано 18 разных типов джинсов. В этом примере создаются только следующие девять комбинаций вариантов продукта.

| Цвет | Размер |
|-------|------|
| Голубой  | XS   |
| Голубой  | П    |
| Голубой  | Н    |
| Черный | Н    |
| Черный | М    |
| Черный | XL   |
| Коричневый | М    |
| Коричневый | XL   |
| Коричневый | XXL  |
