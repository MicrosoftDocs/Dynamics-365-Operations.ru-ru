# <a name="set-up-an-appointment-for-a-load"></a>Настройка назначения для загрузки

[!include[task guide banner](../../includes/task-guide-banner.md)]

Эта процедура показывает порядок настройки и планирования встречи для дебаркадера для погрузки. Обычно это делает координатор транспортировки. Эту процедуру можно выполнить, используя компанию с демонстрационными данными USMF.


## <a name="select-the-load"></a>Выберите груз.
1. Перейдите в раздел "Управление транспортировкой" > "Планирование" > "Рабочее место планирования загрузки".
2. Снимите флажок "Скрыть отгруженные и полученные".
3. В списке выберите груз со статусом "Отгружено".
4. Щелкните "Транспортировка".
5. Щелкните "Планирование отправок-прибытий".

## <a name="create-an-appointment"></a>Создание встречи
1. Щелкните "Создать".
2. В поле "Правило встречи" введите или выберите значение.
3. Нажмите кнопку "Сохранить".
4. Щелкните "Изменить статус".
5. Щелкните "Утверждение".
6. Нажмите кнопку "Сохранить".
7. Закройте страницу.
