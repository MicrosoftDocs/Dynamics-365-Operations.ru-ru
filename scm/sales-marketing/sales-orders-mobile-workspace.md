# <a name="sales-orders-mobile-workspace"></a>Мобильная рабочая область заказов на продажу

[!include[banner](../includes/banner.md)]


В этом разделе проводятся сведения о мобильной рабочей области заказов на продажу, которая доступна для мобильного приложения Microsoft Dynamics 365 for Operations. Эта рабочая область помогает отслеживать заказы на продажу в любом месте и в любое время. 

<a name="overview-of-the-sales-orders-mobile-workspace"></a>Общие сведения о мобильной рабочей области заказов на продажу
---------------------------------------------

Мобильная рабочая область **Заказы на продажу** получает доступ к Microsoft Dynamics 365 for Operations и позволяет просмотреть подробные сведения по каждому заказу на продажу. Эти сведения включают статус заказа, контактную информацию для клиента и контактную информацию для приемщика заказа. Мобильная рабочая область **Заказы на продажу** обеспечивает мгновенный просмотр заказов на продажу. Можно просматривать все заказы на продажу, заказы на продажу по клиентам или сведения о конкретном заказе на продажу. 

Мобильная рабочая область содержит два представления, чтобы помочь глубже анализировать заказы продажу.

### <a name="view-all-sales-orders"></a>Просмотр всех заказов на продажу

В этом представлении перечисляются все заказы на продажу.

-   Используйте один из следующих фильтров для выбора заказов на продажу, которые нужно просмотреть:
    -   Поиск по заказу на продажу
    -   Поиск по счету клиента
    -   Поиск по имени клиента
    -   Поиск по статусу
    -   Поиск по статусу запуска в производство
    -   Поиск по дате и времени создания
    
-   После выбора заказов на продажу можно просматривать сведения для конкретных заказов. Конкретно, можно просмотреть следующую информацию:
    -   Имя клиента и сведения об адресе
    -   Различные даты для заказа на продажу, такие как запрошенная дата отгрузки и подтвержденная дата отгрузки.
    -   Контактная информация для приемщика заказа
    -   Контактные данные клиента
    -   Строки заказа
    -   Отгрузки, которые показывают, как и когда был отгружен заказ на продажу

### <a name="view-orders-for-a-customer"></a>Просмотр заказов для клиента

В этом представлении перечислены заказы на продажу по клиенту.

-   Используйте один из следующих фильтров для просмотра заказов для клиента:
    -   Поиск по имени
    -   Поиск по счету

-   После выбора клиента можно просматривать следующую информацию:
    -   Имя и группа клиентов
    -   Контактные данные клиента
    -   Заказы на продажу для клиента и подробные сведения об этих заказах на продажу:
        -   Имя клиента и сведения об адресе
        -   Различные даты заказа на продажу
        -   Контактная информация для приемщика заказа
        -   Контактные данные клиента
        -   Строки заказа
        -   Отгрузки, которые показывают, как и когда был отгружен заказ на продажу

## <a name="download-and-install-the-dynamics-365-for-operations-mobile-app"></a>Загрузка и установка мобильного приложения Dynamics 365 for Operations
Загрузите и установите мобильное приложение Dynamics 365 for Operations из магазина мобильных приложений.

-   Для Android: [Dynamics 365 for Operations в магазине Google Play](https://play.google.com/store/apps/details?id=com.microsoft.dynamics365.operations.mobile)
-   Для iPhone: [Dynamics 365 for Operations в магазине приложений iTunes](https://itunes.apple.com/us/app/dynamics-365-for-operations/id1180836730?mt=8)

## <a name="sign-in-to-the-dynamics-365-for-operations-mobile-app"></a>Вход в мобильное приложение Dynamics 365 for Operations
1.  Запустите приложение на мобильном устройстве.
2.  Введите свой URL-адрес Dynamics 365 for Operations.
3.  Введите компанию для входа в систему. Например, введите **USMF**.
4.  При первом входе появится запрос ввести имя пользователя и пароль для учетной записи Dynamics 365 for Operations. Введите свои учетные данные.
5.  После входа вы увидите доступные рабочие области для вашей компании. Обратите внимание, что если позже системный администратор опубликует новую рабочую область, вы может потянуть список, чтобы обновить список мобильных рабочих областей. 

    [![Потянуть для обновления](./media/pull-to-refresh-list-of-workspaces-183x300.png)](./media/pull-to-refresh-list-of-workspaces.png)

## <a name="view-information-about-sales-orders-for-a-customer-by-using-the-mobile-workspace"></a>Просмотр сведений о заказах на продажу для клиента с помощью мобильной рабочей области
1.  На мобильном устройстве выберите рабочую область **Заказы на продажу**.
2.  Выберите **Просмотр заказов для клиента**.
3.  Используйте сведения о счете или имени клиента для поиска нужного клиента.
4.  Выберите клиента.
5.  Выберите **Контактная информация** или **Заказы на продажу**. Если выбран пункт **Заказы на продажу**, отображается список заказов на продажу для клиента.
6.  Выберите **Заказ на продажу**. Теперь можно просмотреть сведения о строках заказа на продажу, сведения об отгрузках, контактные данные клиента и контактные данные для приемщика заказа.
