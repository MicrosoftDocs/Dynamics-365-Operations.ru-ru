# <a name="list-of-er-functions-in-the-mathematical-category"></a>Список функций ER в математической категории

[!include [banner](../includes/banner.md)]

Математические функции электронной отчетности (ER) могут быть использованы для многих общих математических вычислений. В данной статье приводится краткое описание этих функций.

## <a name="list-of-supported-functions"></a>Список поддерживаемых функций

| Функция | Описание |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Эта функция возвращает абсолютное значение (модуль) указанного числа в качестве *вещественного* значения. Другими словами, возвращает число без знака. |
| [Мощность](er-functions-mathematical-power.md)         | Эта функция возвращает *вещественное* значение, которое представляет собой результат увеличения указанного положительного числа до указанной степени. |
| [Round](er-functions-mathematical-round.md)         | Эта функция возвращает указанное число в виде *вещественного* значения после его округления до указанного числа десятичных знаков. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Эта функция возвращает указанное число в виде *вещественного* значения после его округления вниз до указанного числа десятичных знаков. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Эта функция возвращает указанное число в виде *вещественного* значения после его округления вверх до указанного числа десятичных знаков. |
