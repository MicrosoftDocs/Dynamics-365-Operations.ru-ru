# <a name="learning-power-bi-content"></a>Содержимое Power BI "Обучение"

[!include [banner](../includes/banner.md)]

В этом разделе описывается содержимое **Обучение** для Microsoft Power BI.

## <a name="reports-that-are-included-in-the-power-bi-content"></a>Отчеты, включенные в содержимое Power BI

Отчеты, включенные в содержимое Power BI **Обучение**, включают диаграммы и таблицы, которые содержат дополнительные сведения. В следующей таблице приводится описание отчетов.

| Отчет                | Содержание |
|-----------------------|----------|
| Обзор "Обучение"     | Сводка по другим отчетам |
| Анализ курса       | Регистрация по местоположению, участник по статусу, курсы по типу в компании и посещение курсов по заданию |
| Анализ регистрации | Список регистрации |
| Типы курсов          | Типы курсов по навыку |
| Анализ лектора   | Отношение курсов к лекторам, число инструкторов, курсы по лектору, курсы на лектора, планом курса по лектору |
| Предлагаемые курсы       | Список курсов |
| Тип курса        | План курса |

Можно фильтровать диаграммы и плитки в этих отчетах, а также закреплять диаграммы и плитки на панели мониторинга. Дополнительные сведения о том, как отфильтровать и закрепить элементы в Power BI, см. в разделе [Создание и настройка панели мониторинга](https://powerbi.microsoft.com/guided-learning/powerbi-learning-4-2-create-configure-dashboards).

## <a name="understanding-the-data-model-and-entities"></a>Понимание модели данных и объектов

Следующая информация используется для заполнения отчетов содержимом Power BI **Обучение**. В этой таблице показаны объекты, на которых основано содержимое.

| Объект           | Содержание                                                         | Связи с другими объектами |
|------------------|------------------------------------------------------------------|-----------------------------------|
| Смещение календаря  | Календарь корреспондирует с отчетами по секторам                                | План курса, участники курса |
| Организация          | Компании, по которым требуется фильтровать отчеты                                   | План курса, участники курса |
| Курс           | Курс, описание, имя лектора, расположение, комната и статус | План курса, участники курса, навыки курса |
| План курса    | Программа, курс и время начала и окончания                          | Компания, смещение календаря, курс |
| Участники курса | Имя, статус, должность и дата регистрации                         | Компания, смещение календаря, дата, курсе, демографические данные, занятость, курс, имя сотрудника, должность сотрудника, задание, должность |
| Навык курса     | Навык, типа навыка и уровень                                     | Курс |
| Дата             | Дни, недели, месяцы и годы                                   | План курса, участники курса |
| Демографические данные     | Дата рождения, пол, этническое происхождение и семейное положение         | План курса, участники курса |
| Занятость       | Дата начала, дата окончания и дата перехода                        | План курса, участники курса |
| Задание              | Функция, тип и должность                                        | План курса, участники курса |
| Занимаемая должность         | Должность, название и эквивалент полной занятости (FTE)                  | План курса, участники курса |
| Имя сотрудника    | Имя, фамилия и ФИО                             | Участники курса |
| Должность сотрудника   | Должность и дата трудового стажа                                         | Участники курса |
