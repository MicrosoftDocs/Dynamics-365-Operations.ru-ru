# <a name="file-destination"></a>Место назначения файла

[!include [banner](../includes/banner.md)]

Можно настроить место назначения файла для каждого компонента ПАПКА или ФАЙЛ формата электронной отчетности (ER), настроенного для создания исходящих документов. В зависимости от настройки места назначения созданный документ доступен для загрузки из веб-браузера.

На странице **Параметры места назначения** если для параметра **Включено** задано значение **Да**, после завершения выполнения конфигурации появляется диалоговое окно открытия или сохранения.
