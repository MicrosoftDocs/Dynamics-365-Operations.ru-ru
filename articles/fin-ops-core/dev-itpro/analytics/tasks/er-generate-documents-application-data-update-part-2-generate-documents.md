# <a name="design-configurations-to-generate-documents-that-have-application-data"></a>Разработка конфигураций для формирования документов, имеющих данные приложений

[!include [banner](../../includes/banner.md)]

Для выполнения действий в этой процедуре необходимо сначала выполнить процедуру "Электронная отчетность — Формирование документов с обновлением данных приложения (Часть 1. Импорт конфигураций)".



В этой процедуре поясняется разработка конфигураций электронной отчетности для формирования электронного документа. В этой процедуре вам предстоит запустить импортированную конфигурацию формата электронной отчетности, созданную для демонстрационной компании Litware, Inc., для формирования электронных документов.



Эта процедура предназначена для пользователей, которым назначена роль "Системный администратор" или "Разработчик электронной отчетности". Действия можно выполнять с использованием набора данных DEMF. 



Прежде чем начать, измените контекст страны для компании DEMF с DEU (Германии) на BEL (Бельгия). Выберите "Управление организацией" > "Организации" > "Юридические лица", чтобы обновить код страны в основном адресе юридического лица DEMF. Перезапустите приложение.


## <a name="run-imported-er-format"></a>Запуск импортированного формата электронной отчетности
1. Перейдите в раздел "Управление организацией" > "Электронная отчетность" > "Конфигурации".
2. В дереве разверните узел "Intrastat (model)".
3. В дереве выберите "Intrastat (model)\Intrastat (format)".
4. Щелкните "Выполнить".
    * Запустите черновую версию конфигурацию формата электронной отчетности, чтобы сформировать отчет Интрастат.  
5. В поле "Введите имя файла" введите "intrastat.xml".
    * Укажите имя файла.  
6. Нажмите кнопку "OК".
    * Просмотрите сформированный файл XML.  
7. Закройте страницу.
8. Перейдите в раздел "Налог" > "Декларации" > "Внешняя торговля" > "Интрастат".
    * Откройте эту форму для просмотра проводок Интрастат, включенных в сформированный электронный документ.  
9. Щелкните "Архив Интрастат".
    * Поскольку выполненный формат электронной отчетности не содержит никаких параметров для обновления данных приложения, сведения о завершенном отчете Интрастат архивированы не были.  
10. Закройте страницу.
11. Закройте страницу.
