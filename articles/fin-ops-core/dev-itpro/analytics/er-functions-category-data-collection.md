# <a name="list-of-er-functions-in-the-data-collection-category"></a>Список функций ER в категории сбора данных

[!include [banner](../includes/banner.md)]

Функции сбора данных электронной отчетности (ER) используются для подсчета и подведения итогов в формате ER, который выполняется, на основе данных вывода, которые уже были созданы в формате **Текст** или **Xml**. Этот подход используется для повышения производительности формата ER, который работает, для ввода значений итогов выполнения в создаваемых документах и для других целей. В данной статье приводится краткое описание этих функций.

## <a name="list-of-supported-functions"></a>Список поддерживаемых функций

| Функция | Описание |
|----------|-------------|
| [CollectedList](er-functions-datacollection-collectedlist.md) | Эта функция возвращает значение *Список записей*, содержащее список значений, которые были возвращены свойством **Значение ключа собранных данных** формата элементов и собраны, когда элементы формата использовались для создания исходящего документа во время выполнения формата, и что удовлетворяет указанным условиям. Каждое условие состоит из диапазона ключей и значения ключа. |
| [CountIF](er-functions-datacollection-countif.md) | Эта функция возвращает *целочисленное* значение, представляющее количество элементов формата, которые были собраны при использовании элементов формата для создания исходящего документа во время выполнения формата, и удовлетворяющее указанному условию. Условие состоит из диапазона ключей и значения ключа. |
| [CountIFs](er-functions-datacollection-countifs.md) | Эта функция возвращает *целочисленное* значение, представляющее количество элементов формата, которые были собраны при использовании элементов формата для создания исходящего документа во время выполнения формата, и удовлетворяющее указанным условиям. Каждое условие состоит из диапазона ключей и значения ключа. |
| [FormatElementName](er-functions-datacollection-formatelementname.md) | Эта функция возвращает значение *строка*, представляющее имя элемента текущего формата ER. |
| [SumIF](er-functions-datacollection-sumif.md) | Эта функция возвращает *вещественное* значение, представляющее сумму значений, которые были возвращены привязками элементов формата и собраны во время использования для создания исходящего документа во время выполнения формата, и удовлетворяющее указанному условию. Условие состоит из диапазона ключей и значения ключа. |
| [SumIFs](er-functions-datacollection-sumifs.md) | Эта функция возвращает *вещественное* значение, представляющее сумму значений, которые были возвращены привязками элементов формата и собраны во время использования для создания исходящего документа во время выполнения формата, и удовлетворяющее указанным условиям. Каждое условие состоит из диапазона ключей и значения ключа. |
