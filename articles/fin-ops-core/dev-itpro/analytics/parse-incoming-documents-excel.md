# <a name="parse-incoming-documents-in-excel-format"></a>Разбор входящих документов в формате Excel

[!include[banner](../includes/banner.md)]

Можно разработать форматы электронной отчетности (ER) для разбора входящих файлы Microsoft Excel, представляющих данные в книгах Microsoft Excel (файлы в формате XLSX). Затем можно использовать содержимое из этих файлов для обновления данных приложения. Это полезно, если вы:

- Создаете новую модель и формат и хотите проверить их во время выполнения. В этом случае Excel будет моделировать реальные данные приложения.
- Управляете данными за пределами приложения в Excel и хотите импортировать эти данные для отправки конкретного отчета.

Для получения дополнительных сведений о данной функции, выполните проводник по задаче **Импортировать данные ER из файла Microsoft Excel (часть 1: формат разработки)** и **Импортировать данные ER из файла Microsoft Excel (часть 2: Импорт данных)** (части бизнес-процесса 7.5.4.3 Приобретение/разработка компонентов ИТ-услуг и решений (10677)). Эти проводники по задаче описывают, как входящий файл Excel можно проанализировать с помощью формата ER для импорта информации из входящих документов и обновления данных приложения. Файлы проводника по задаче можно загрузить из [центра загрузки Майкрософт](https://go.microsoft.com/fwlink/?linkid=874684).

Загрузите следующие файлы для завершения проводников по задаче.

| Описание содержания                         | Хранилище файлов                                                                       |
|---------------------------------------------|----------------------------------------------------------------------------|
| Входящий файл в формате XLSX - шаблон    | [1099import template.xlsx](https://go.microsoft.com/fwlink/?linkid=862266) |
| Входящий файл в формате .XLSX - образец данных | [1099import data.xlsx](https://go.microsoft.com/fwlink/?linkid=862266)     |

Если вы еще не выполняли следующий проводник по задаче, [ER создать конфигурации, необходимые для импорта данных из внешнего файла](./tasks/er-required-configurations-import-data.md) в текущем приложении Finance and Operations, загрузите следующий файл.

| Описание содержания    | Хранилище файлов                                                            |
|------------------------|-----------------------------------------------------------------|
| Конфигурация модели ER | [1099model.xml](https://go.microsoft.com/fwlink/?linkid=862266) |
