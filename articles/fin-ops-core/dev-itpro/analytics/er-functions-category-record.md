# <a name="list-of-er-functions-in-the-record-category"></a>Список функций ER в категории записей

[!include [banner](../includes/banner.md)]

Функции записей электронной отчетности (ER) могут использоваться для извлечения информации и выполнения операций по источникам данных типа данных *Контейнер (запись)*. В этой теме приводится краткое изложение этих функций.

## <a name="list-of-supported-functions"></a>Список поддерживаемых функций

| Функция | Описание |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Эта функция возвращает нулевое значение *Контейнер (запись)*, у которого такая же структура, что и у указанного списка записей или записи. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Эта функция возвращает нулевое значение *Контейнер (запись)*, у которого такая же структура, что и у указанного списка записей или записи. |
