# <a name="access-to-finance-and-tax-reference-data"></a>Доступ к финансовым и налоговым ссылочным данным

[!include [banner](../../includes/banner.md)]

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Каждое предприятие работает с базовым набором финансовых данных, таких как год финансового календаря, валюта, в которой работает компания, счета, на которые поступают и с которых снимаются денежные средства для работы компании, налоговые ставки и переводы. Эти данные хранятся в приложениях Finance and Operations. Однако они доступны для Dataverse, чтобы приложения для взаимодействия с клиентами могли иметь единый источник для финансовых и налоговых данных. Таким образом, данные являются едиными для всей экосистемы бизнеса.

Финансовые и налоговые данные интегрируются с использованием следующих сопоставлений:

+ [Интегрированная книга учета](ledger-mapping.md)
+ [Интегрированный налоговый справочник](tax-mapping.md)
