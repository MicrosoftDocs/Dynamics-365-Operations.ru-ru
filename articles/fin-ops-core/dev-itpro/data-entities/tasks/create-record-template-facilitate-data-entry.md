# <a name="create-a-record-template-to-facilitate-data-entry"></a>Создание шаблона записи для облегчения ввода данных

[!include [banner](../../includes/banner.md)]

В этой статье демонстрируется, как создать шаблон записи, чтобы значения полей, которые используются часто, не требовалось вводить явно для каждой новой записи. В этой процедуре предстоит создать новую запись для новых ноутбуков, которые должны быть добавлены к ОС. В данной процедуре используется демонстрационная компания USMF.

1. В области перехода, перейдите к **Модули > Фиксированные активы > Фиксированные активы > Фиксированные активы**.
2. Выберите **Создать**.
3. В поле **Группа ОС** введите или выберите значение.
4. В поле **Имя** введите значение. Например, введите **Ноутбук корпоративного интереса**.  
5. В поле **Краткое наименование** введите значение. Например, введите **ноутбук**.  
6. Разверните раздел **Техническая информация**.
7. В полях **Марка**, **Модель** и **Модельный год** введите значения.
8. В области панели операций выберите **Параметры**.
9. Выберите **Сведения о записи**.
10. Выберите **Шаблон пользователя**.
11. В поле **Имя** введите значение.
12. В поле **Описание** введите значение.
13. Нажмите **ОК**.
14. Выберите **Закрыть**.
