# <a name="removed-or-deprecated-features-for-finance-and-operations"></a>Удаленные или устаревшие функции для Finance and Operations

[!include [banner](../includes/banner.md)]

В этой теме описываются функции, которые были удалены или устарели для Finance and Operations.

- *Удаленная* функция больше недоступна в продукте.
- *Устаревшая* функция не находится в активной разработке и может быть удалена в следующем обновлении.

Этот список поможет вам учитывать эти удаления и устаревания при своем собственном планировании. 

> [!NOTE]
> Начиная с выпуска Finance and Operations от июля 2017 г. с обновлением платформы Platform update 8, типы развертывания указаны для каждой удаленной или устаревшей функции. Все предыдущие выпуски, упоминаемые в этом разделе, поддерживают только облачные развертывания.

Подробные сведения об объектах в Finance and Operations можно найти в документе [Технический справочник по отчетам](https://mbs.microsoft.com/customersource/northamerica/AX/downloads/reports/axtechrefrep). Можно сравнить различные версии этих отчетов, чтобы получить сведения об объектах, которые были изменены или были исключены в каждой версии Finance and Operations.

|   |  |

Следующие методы станут устаревшими после удаления этого параметра: **IsSalesOrderAllowedForMultipleFundingSources** с файле таблицы **ProjTable**, метод **IsAllowSalesOrdersForMultipleFundingSourcesParamEnabled** в файле таблицы **ProjTable**, поле данных **AllowSalesOrdersForMultipleFundingSources** в форме **ProjParameters** и файлах **ProjParameterEntity**, закрытый метод **IsAssociatedToMultipleFundingSourcesContract** в файле таблицы **ProjTable**. |
| **Вариант развертывания**              | Все  |

### <a name="aif-axd-and-axbc-integrations"></a>Интеграции AIF, AxD и AxBC

Dynamics AX включает службы, которые основаны на документах и .NET Business Connector (AxBC). Документ создается с использованием XML. XML включает данные заголовка, добавляемого для создания *сообщения*, которое можно переносить в или из Dynamics AX. Примеры документов включают заказы на продажу и заказы на покупку. Однако почти любой объект, например клиент, может быть представлен документом. В службах, которые основаны на документах, используются классы **Axd \<Document\>**.

### <a name="virtual-company-accounts"></a>Виртуальная компания

Функция виртуальных компаний больше не поддерживается в Dynamics AX. Функция виртуальных компаний позволяла пользователям настроить таблицы, которые могут быть общими для набора компаний. Описание функции здесь: [Счета компаний и счета виртуальных компаний](https://msdn.microsoft.com/library/aa834382(v=ax.10).aspx). Функция работает путем группирования таблиц в коллекции, которые назначаются виртуальным компаниям, которые представляют собой группы существующих "реальных" компаний. Запросы создаются так, что все компании в виртуальной компании могут получить доступ к данным в таблицах связанных коллекций таблиц.

|   |  | 
|------------|--------------------|
| **Причина устаревания/удаления** | - Виртуальные компании необходимо настроить до сохранения данных в таблицах. Встроить виртуальные компании в существующую реализацию очень трудно.<br><br>- Поскольку в текущей версии Dynamics AX широко используется нормализация данных, выяснить, что требуется добавить в коллекции таблиц (например, выяснить, какие таблицы должны быть общими) стало очень сложно. Например, сложно определить, какие таблицы, которые будут использоваться совместно. Все таблицы, на которые имеются ссылки в таблицах, которые находятся в виртуальной компании, также необходимо добавить. Из-за нормализации таблицы даже простой справочник, включенный во множество таблиц, должен быть частью виртуальной компании. Любая ошибка, совершенная здесь, приведет к функциональным проблемам.<br><br>- Когда таблица является частью виртуальной компании, она теряет информацию об источнике данных, и записывается только виртуальная компания.   |
| **Заменена другой функцией?** | Для создания таблиц, доступных всем компаниям, могут использоваться глобальные таблицы. В данный момент, нет никакой замены. |   
| **Затрагиваемые области продукта**       | Все модули |   
| **Статус**                       | Удалено в Dynamics AX 7.0.   |
