# <a name="supported-web-browsers-for-lifecycle-services"></a>Браузеры, поддерживаемые для Lifecycle Services 

[!include[banner](../includes/banner.md)]

Пользователи могут получить доступ к Microsoft Dynamics Lifecyle Services (LCS), используя самые последние версии этих популярных браузеров: 

- Microsoft Edge (рекомендуется: [Microsoft Edge на основе Chromium](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Чтобы найти последний выпуск для каждого веб-браузера, перейдите на веб-сайт производителя программного обеспечения. Для оптимальной производительности и оптимального интерфейса рекомендуется использовать последнюю версию современного браузера, особенно Microsoft Edge.
