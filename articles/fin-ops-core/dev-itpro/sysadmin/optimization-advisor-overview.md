<span data-ttu-id="0522a-106">Неправильная конфигурация и настройка модуля может негативно повлиять на доступность функциональных возможностей в приложении, производительность системы и бесперебойную работу бизнес-процессов.</span><span class="sxs-lookup"><span data-stu-id="0522a-106">Incorrect configuration and setup of a module can adversely affect the availability of application features, system performance, and the smooth operation of business processes.</span></span> <span data-ttu-id="0522a-107">Качество бизнес-данных (например, их правильность, полнота и чистота) также влияет на производительность системы и возможности по принятию решений, производительность и т. д. в организации.</span><span class="sxs-lookup"><span data-stu-id="0522a-107">The quality of business data (for example, the correctness, completeness, and cleanliness of the data) also affects system performance, and an organization's decision-making capabilities, productivity, and so on.</span></span>

<span data-ttu-id="0522a-113">При обнаружении нарушения правила создается возможность оптимизации, которая появляется в рабочей области **Помощник по оптимизации**.</span><span class="sxs-lookup"><span data-stu-id="0522a-113">When a violation of a rule is detected, an optimization opportunity is generated and appears in the **Optimization advisor** workspace.</span></span> <span data-ttu-id="0522a-114">Пользователь может выполнить соответствующие исправления непосредственно из рабочей области **Помощник по оптимизации**.</span><span class="sxs-lookup"><span data-stu-id="0522a-114">A user can take appropriate corrective action directly from the **Optimization advisor** workspace.</span></span>

<span data-ttu-id="0522a-115">Возможности могут быть связаны с конкретной компанией или относиться ко всем компаниям, в зависимости от типа настройки и данных, которые нарушаются.</span><span class="sxs-lookup"><span data-stu-id="0522a-115">Opportunities can be company-specific or cross-company, depending on the type of setup and data that is being validated.</span></span> <span data-ttu-id="0522a-116">Возможности для всех компаний можно просмотреть из всех компаний.</span><span class="sxs-lookup"><span data-stu-id="0522a-116">Cross-company opportunities can be viewed from all companies.</span></span> <span data-ttu-id="0522a-117">Для просмотра возможностей для конкретной компании необходимо сначала выбрать компанию.</span><span class="sxs-lookup"><span data-stu-id="0522a-117">To view the opportunities for a specific company, you must first select the company.</span></span>

<span data-ttu-id="0522a-120">При выполнении действия по некоторой возможности оптимизации система рассчитывает влияние возможности с точки зрения сокращения времени выполнения бизнес-процессов.</span><span class="sxs-lookup"><span data-stu-id="0522a-120">When you take action on some optimization opportunities, the system calculates the impact of the opportunity in terms of the reduction in the runtime of business processes.</span></span> <span data-ttu-id="0522a-121">К сожалению эта функция доступна не для всех возможностей оптимизации.</span><span class="sxs-lookup"><span data-stu-id="0522a-121">Unfortunately, this feature isn't available for all optimization opportunities.</span></span>
