# <a name="create-an-operating-unit"></a><span data-ttu-id="5cedf-103">Создание операционной единицы</span><span class="sxs-lookup"><span data-stu-id="5cedf-103">Create an operating unit</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="5cedf-104">Операционная единица — это организация, используемая для разделения управления экономическими ресурсами и операционными процессами в бизнесе.</span><span class="sxs-lookup"><span data-stu-id="5cedf-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="5cedf-105">Сотрудники операционной единицы обязаны использовать ограниченные ресурсы максимально эффективно, оптимизировать процессы и отчитываться за свою деятельность.</span><span class="sxs-lookup"><span data-stu-id="5cedf-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="5cedf-106">Ттипы операционных единиц включают центры затрат, бизнес-единицы, подразделения и потоки создания ценности.</span><span class="sxs-lookup"><span data-stu-id="5cedf-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="5cedf-107">Для создания операционной единицы используется следующая процедура.</span><span class="sxs-lookup"><span data-stu-id="5cedf-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="5cedf-108">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="5cedf-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="5cedf-109">Перейдите в раздел **Область переходов > Модули > Управление организацией > Организации > Операционные единицы**.</span><span class="sxs-lookup"><span data-stu-id="5cedf-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="5cedf-110">Щелкните **Создать**, чтобы открыть раскрывающееся диалоговое окно.</span><span class="sxs-lookup"><span data-stu-id="5cedf-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="5cedf-111">В списке найдите и выберите требуемую запись.</span><span class="sxs-lookup"><span data-stu-id="5cedf-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="5cedf-112">Выберите тип операционной единицы, которую необходимо создать.</span><span class="sxs-lookup"><span data-stu-id="5cedf-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="5cedf-113">В списке перейдите по ссылке в выбранной строке.</span><span class="sxs-lookup"><span data-stu-id="5cedf-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="5cedf-114">В поле **Имя** введите значение.</span><span class="sxs-lookup"><span data-stu-id="5cedf-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="5cedf-115">Разверните раздел **Общее**, если необходимо.</span><span class="sxs-lookup"><span data-stu-id="5cedf-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="5cedf-116">Введите общие сведения о операционной единице, например идентификационный номер, номер DUNS и менеджер.</span><span class="sxs-lookup"><span data-stu-id="5cedf-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="5cedf-117">Разверните раздел **Адреса**, если необходимо.</span><span class="sxs-lookup"><span data-stu-id="5cedf-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="5cedf-118">Введите информацию об адресе, например улицу и номер дома, почтовый индекс и город.</span><span class="sxs-lookup"><span data-stu-id="5cedf-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="5cedf-119">Щелкните **Добавить**, чтобы ввести новую запись адреса, или "Изменить", чтобы изменить существующую запись адреса.</span><span class="sxs-lookup"><span data-stu-id="5cedf-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="5cedf-120">При необходимости разверните раздел **Контактная информация**.</span><span class="sxs-lookup"><span data-stu-id="5cedf-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="5cedf-121">Введите сведения о способах связи, например адреса электронной почты, URL-адреса и номера телефона.</span><span class="sxs-lookup"><span data-stu-id="5cedf-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="5cedf-122">Для ввода новой записи щелкните "Создать".</span><span class="sxs-lookup"><span data-stu-id="5cedf-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="5cedf-123">Для изменения существующей записи щелкните **Дополнительные параметры > Дополнительно**.</span><span class="sxs-lookup"><span data-stu-id="5cedf-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="5cedf-124">При необходимости измените **Номер операционной единицы**.</span><span class="sxs-lookup"><span data-stu-id="5cedf-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="5cedf-125">Обратите внимание на то, что этот номер является уникальным идентификатором для записи **Субъект** и не может совпадать с любой другой операционной единицей.</span><span class="sxs-lookup"><span data-stu-id="5cedf-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="5cedf-126">Нажмите **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="5cedf-126">Select **Save**.</span></span>
