# <a name="create-a-legal-entity"></a><span data-ttu-id="27f32-103">Создание юридического лица</span><span class="sxs-lookup"><span data-stu-id="27f32-103">Create a legal entity</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="27f32-104">Юридическое лицо - это организация, которая определяется посредством регистрации в органе власти.</span><span class="sxs-lookup"><span data-stu-id="27f32-104">A legal entity is an organization that is identified through registration with a legal authority.</span></span> <span data-ttu-id="27f32-105">Юридические лица могут заключать контракты и обязаны подготавливать отчетность по результатам своей деятельности.</span><span class="sxs-lookup"><span data-stu-id="27f32-105">Legal entities can enter into contracts and are required to prepare statements that report on their performance.</span></span> <span data-ttu-id="27f32-106">В следующей процедуре показано, как создать юридическое лицо.</span><span class="sxs-lookup"><span data-stu-id="27f32-106">The following procedure explains how to create a legal entity.</span></span> <span data-ttu-id="27f32-107">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="27f32-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="27f32-108">Перейдите в раздел **Область переходов > Модули > Управление организацией > Организации > Юридические лица**.</span><span class="sxs-lookup"><span data-stu-id="27f32-108">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="27f32-109">Нажмите кнопку **Создать**.</span><span class="sxs-lookup"><span data-stu-id="27f32-109">Click **New**.</span></span>
3. <span data-ttu-id="27f32-110">В поле **Имя** введите значение.</span><span class="sxs-lookup"><span data-stu-id="27f32-110">In the **Name** field, type a value.</span></span>
4. <span data-ttu-id="27f32-111">В поле **Компания** введите значение.</span><span class="sxs-lookup"><span data-stu-id="27f32-111">In the **Company** field, type a value.</span></span>
5. <span data-ttu-id="27f32-112">В поле **Страна/регион** введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="27f32-112">In the **Country/region** field, enter or select a value.</span></span>
6. <span data-ttu-id="27f32-113">Щелкните **OK**.</span><span class="sxs-lookup"><span data-stu-id="27f32-113">Click **OK**.</span></span> <span data-ttu-id="27f32-114">В разделе **Общее** укажите следующие общие сведения о юридическом лице: Введите краткое наименование, если имя поиска необходимо.</span><span class="sxs-lookup"><span data-stu-id="27f32-114">In the **General** section, provide the following general information about the legal entity: Enter a search name, if a search name is required.</span></span> <span data-ttu-id="27f32-115">Краткое наименование — это альтернативное имя, которое можно использовать для поиска этого юридического лица.</span><span class="sxs-lookup"><span data-stu-id="27f32-115">A search name is an alternate name that can be used to search for this legal entity.</span></span> <span data-ttu-id="27f32-116">Выберите используется ли это юридическое лицо как консолидированная компания.</span><span class="sxs-lookup"><span data-stu-id="27f32-116">Select whether this legal entity is being used as a consolidation company.</span></span> <span data-ttu-id="27f32-117">Выберите используется ли это юридическое лицо как Компания элиминирования.</span><span class="sxs-lookup"><span data-stu-id="27f32-117">Select whether this legal entity is being used as an elimination company.</span></span> 
7. <span data-ttu-id="27f32-118">Разверните раздел **Адреса**.</span><span class="sxs-lookup"><span data-stu-id="27f32-118">Expand the **Addresses** section.</span></span> <span data-ttu-id="27f32-119">В разделе **Адреса** щелкните **Изменить**, чтобы ввести информацию об адресе, например название улицы, номер дома, почтовый индекс и название города.</span><span class="sxs-lookup"><span data-stu-id="27f32-119">In the **Addresses** section, click **Edit** to enter address information, such as the street name and number, postal code, and city.</span></span>
8. <span data-ttu-id="27f32-120">Разверните раздел **Контактная информация**.</span><span class="sxs-lookup"><span data-stu-id="27f32-120">Expand the **Contact information** section.</span></span> <span data-ttu-id="27f32-121">В разделе **Контактная информация** введите сведения о методах связи, например адреса электронной почты, URL-адреса и номера телефонов.</span><span class="sxs-lookup"><span data-stu-id="27f32-121">In the **Contact information** section, enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> 
9. <span data-ttu-id="27f32-122">Разверните раздел **Предусмотренная отчетность**.</span><span class="sxs-lookup"><span data-stu-id="27f32-122">Expand the **Statutory reporting** section.</span></span> <span data-ttu-id="27f32-123">В разделе **Предусмотренная отчетность** введите регистрационные номера, которые используются для предусмотренной отчетности.</span><span class="sxs-lookup"><span data-stu-id="27f32-123">In the **Statutory reporting** section, enter the registration numbers that are used for statutory reporting.</span></span>
10. <span data-ttu-id="27f32-124">Разверните раздел **Регистрационные номера**.</span><span class="sxs-lookup"><span data-stu-id="27f32-124">Expand the **Registration numbers** section.</span></span> <span data-ttu-id="27f32-125">В разделе **Регистрационные номера** введите любую информацию, требуемую юридическим лицом.</span><span class="sxs-lookup"><span data-stu-id="27f32-125">In the **Registration numbers** section, enter any information required by the legal entity.</span></span>  
11. <span data-ttu-id="27f32-126">Разверните раздел **Сведения о банковском счете**.</span><span class="sxs-lookup"><span data-stu-id="27f32-126">Expand the **Bank account information** section.</span></span> <span data-ttu-id="27f32-127">В разделе **Сведения о банковском счете** введите банковские счета и внутренние номера для юридического лица.</span><span class="sxs-lookup"><span data-stu-id="27f32-127">In the **Bank account information** section, enter bank accounts and routing numbers for the legal entity.</span></span>
12. <span data-ttu-id="27f32-128">Разверните раздел **Внешняя торговля и логистика**.</span><span class="sxs-lookup"><span data-stu-id="27f32-128">Expand the **Foreign trade and logistics** section.</span></span> <span data-ttu-id="27f32-129">В разделе **Внешняя торговля и логистика** введите сведения об отгрузке для юридического лица.</span><span class="sxs-lookup"><span data-stu-id="27f32-129">In the **Foreign trade and logistics** section, enter shipping information for the legal entity.</span></span>  
13. <span data-ttu-id="27f32-130">Разверните раздел **Номерные серии**.</span><span class="sxs-lookup"><span data-stu-id="27f32-130">Expand the **Number sequences** section.</span></span> <span data-ttu-id="27f32-131">В разделе **Номерные серии** можно просмотреть номерные серии, которые связаны с юридическим лицом.</span><span class="sxs-lookup"><span data-stu-id="27f32-131">In the **Number sequences** section, you can view the number sequences that are associated with the legal entity.</span></span>  
14. <span data-ttu-id="27f32-132">Разверните раздел **Изображения**.</span><span class="sxs-lookup"><span data-stu-id="27f32-132">Expand the **Images** section.</span></span> <span data-ttu-id="27f32-133">В разделе **Изображения** просмотрите или измените изображение эмблемы или панели мониторинга, связанной с юридическим лицом.</span><span class="sxs-lookup"><span data-stu-id="27f32-133">In the **Images** section, view or change the logo and/or dashboard image that are associated with the legal entity.</span></span>  
15. <span data-ttu-id="27f32-134">Разверните раздел **Налоговая регистрация**.</span><span class="sxs-lookup"><span data-stu-id="27f32-134">Expand the **Tax registration** section.</span></span> <span data-ttu-id="27f32-135">В разделе **Налоговая регистрация** введите регистрационные номера, которые используются для отчета в налоговые органы.</span><span class="sxs-lookup"><span data-stu-id="27f32-135">In the **Tax registration** section, enter the registration numbers that are used to report to tax authorities.</span></span>
16. <span data-ttu-id="27f32-136">Разверните раздел **Налог по форме 1099**.</span><span class="sxs-lookup"><span data-stu-id="27f32-136">Expand the **Tax 1099** section.</span></span> <span data-ttu-id="27f32-137">В разделе **Налог по форме 1099** введите сведения 1099 для юридического лица.</span><span class="sxs-lookup"><span data-stu-id="27f32-137">In the **Tax 1099** section, enter 1099 information for the legal entity.</span></span>  
17. <span data-ttu-id="27f32-138">Нажмите кнопку **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="27f32-138">Click **Save**.</span></span>
