# <a name="configure-the-global-address-book"></a>Настройка глобальной адресной книги

[!include [banner](../../includes/banner.md)]

Эта процедура используется для настройки значений по умолчанию и политик безопасности для глобальной адресной книги. 

В качестве компании с демонстрационными данными для создания этой задачи используется USMF. Эта задача назначена для группы планирования и конфигурации.

1. В области переходов перейдите в раздел **Модули > Администрирование организации > Глобальная адресная книга > Параметры глобальной адресной книги**.
2. В поле **Последовательность имя/фамилия** выберите способ отображения имен и фамилий.
3. В поле **Удалить субъекты без ролей** выберите, требуется ли удалять субъектов, которым не была назначена никакая роль.
4. В поле **Использовать проверку на дубликаты** выберите, требуется ли проверять наличие дубликатов записей.
5. В поле **Отображать DUNS-номер в адресах** выберите, отображать ли номер DUNS в адресах.
6. В поле **Проверка уникальности номера DUNS** выберите, требуется ли проверять уникальность номеров DUNS.
7. В поле **Субъект** выберите один из вариантов.
8. В поле **Клиент** выберите один из вариантов.
9. В поле **Поставщик** выберите один из вариантов.
10. В поле **Перспективный клиент** выберите один из вариантов.
11. В поле **Конкурент** выберите один из вариантов.
12. Перейдите на вкладку **Безопасность частного местоположения**.
13. В списке найдите и выберите требуемую запись. Нажмите клавишу SHIFT, чтобы выбрать несколько ролей для добавления в область **Выбранные роли**, а затем щелкните стрелку, чтобы добавить из к выбранным ролям.  
14. Нажмите кнопку **Сохранить**.
