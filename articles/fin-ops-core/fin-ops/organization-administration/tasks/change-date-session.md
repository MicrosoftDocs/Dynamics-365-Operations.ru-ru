# <a name="change-the-date-for-a-session"></a>Изменение даты для сеанса

[!include [banner](../../includes/banner.md)]


[!INCLUDE [PEAP](../../../../includes/peap-1.md)]

В этой статье объясняется, как изменить дату для сеанса. По умолчанию текущая дата используется при вводе и разноске записей журналов или документов-источников. Можно изменить дату для текущего сеанса. Используйте эту функцию для пометки задним числом записей журналов или документов-источников, если необходимо.

1. В области переходов выберите **Модули > Общие > Общие > Дата и время сеанса**.
2. В поле **Дата** введите дату.
3. Нажмите **ОК**.
