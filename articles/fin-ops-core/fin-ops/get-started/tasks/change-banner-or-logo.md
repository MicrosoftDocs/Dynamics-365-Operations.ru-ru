# <a name="change-the-banner-or-logo"></a><span data-ttu-id="ae802-103">Изменение баннера или эмблемы</span><span class="sxs-lookup"><span data-stu-id="ae802-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="ae802-104">Следующая процедура перечисляет этапы, которые системные администраторы могут использовать для обновления изображения баннера или логотипа, отображаемого для юридического лица.</span><span class="sxs-lookup"><span data-stu-id="ae802-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="ae802-105">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="ae802-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="ae802-106">Перейдите в раздел **Область переходов > Модули > Управление организацией > Организации > Юридические лица**.</span><span class="sxs-lookup"><span data-stu-id="ae802-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="ae802-107">В списке в верхнем левом углу выберите юридическое лицо, для которого необходимо обновить баннер или эмблему.</span><span class="sxs-lookup"><span data-stu-id="ae802-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="ae802-108">Если оно уже выбрано, перейдите к следующему шагу.</span><span class="sxs-lookup"><span data-stu-id="ae802-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="ae802-109">Разверните вкладку **Изображение на панели мониторинга**.</span><span class="sxs-lookup"><span data-stu-id="ae802-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="ae802-110">Выберите **Изменить**.</span><span class="sxs-lookup"><span data-stu-id="ae802-110">Select **Change**.</span></span>
    
    <span data-ttu-id="ae802-111">Идеальное разрешение изображения баннера составляет 1920 x 281 пикселей.</span><span class="sxs-lookup"><span data-stu-id="ae802-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="ae802-112">Идеальная ширина изображения баннера составляет 350 пикселей.</span><span class="sxs-lookup"><span data-stu-id="ae802-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="ae802-113">Нажмите **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="ae802-113">Select **Save**.</span></span>
6. <span data-ttu-id="ae802-114">Перейдите в раздел **Область переходов > Модули > Общие > Общая > Панель мониторинга по умолчанию**.</span><span class="sxs-lookup"><span data-stu-id="ae802-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="ae802-115">Вы должны видеть новый баннер и изображение эмблемы на панели мониторинга.</span><span class="sxs-lookup"><span data-stu-id="ae802-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="ae802-116">Вам может потребоваться изменить свою компанию, используя элемент управления выбора компании, на ту, для которой вы загрузили баннер.</span><span class="sxs-lookup"><span data-stu-id="ae802-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
