# <a name="deliver-employee-benefits-program"></a>Реализация программы льгот сотрудника

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой задаче показано, как создать элементы льгот, которые будут использоваться при создании новой льготы. В качестве компании с демонстрационными данными для создания этой задачи используется USMF. Эта задача предназначена для менеджера по компенсациям и льготам.


## <a name="create-benefit-elements"></a>Создание элементов льготы
1. Эта задача начинается со страницы списка "Льготы ". Запустите задачу, открыв эту страницу или выполнив поиск страницы списка "Льготы".
2. Нажмите Создать.
3. В поле "Тип" введите имя создаваемого типа льготы.
4. В поле "Описание" введите значение.
5. В поле "Одновременная регистрация" выберите вариант.
    * Чтобы ограничить возможность сотрудников регистрироваться в нескольких медицинских планах, установите флажок "Одна регистрация для каждого типа".  
6. В поле "Категория по зарплате" выберите вариант.
7. Перейдите на вкладку "Планы".
8. Нажмите Создать.
9. В поле "План" введите значение.
10. В поле "Описание" введите значение.
11. В поле "Тип" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
12. Поиск и выбор требуемой записи в списке.
13. В списке перейдите по ссылке в выбранной строке.
14. В поле "Влияние на зарплату" выберите вариант.
15. Перейдите на вкладку "Параметры".
16. Нажмите Создать.
17. В поле "Параметр" введите значение.
18. В поле "Описание" введите значение.

## <a name="create-a-benefit"></a>Создание льготы
1. Закройте страницу.
2. Перейдите в раздел "Управление персоналом" > "Льготы" > "Льготы".
3. Щелкните "Создать", чтобы открыть раскрывающееся диалоговое окно.
4. В поле "План" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
5. В списке найдите и выберите требуемую запись.
6. В списке перейдите по ссылке в выбранной строке.
7. В поле "Параметр" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
8. Поиск и выбор требуемой записи в списке.
9. В списке перейдите по ссылке в выбранной строке.
10. В поле "Действует" введите дату и время.
11. Щелкните "Создать льготу".
12. Переключите развертывание раздела "Сведения по зарплате".
13. В поле "Частота" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
14. В списке найдите и выберите требуемую запись.
15. В списке перейдите по ссылке в выбранной строке.
16. В поле "Основание" выберите вариант.
17. В поле "Сумма или ставка" введите число.
