# <a name="hiring-candidate-through-recruiting"></a>Прием кандидата через набор сотрудников

[!include [banner](../../includes/banner.md)]

Эта процедура позволяет специалисту по найму принять на работу кандидата, подавшего заявление через конкретный проект по набору сотрудников. При приеме кандидата на работу через проект по набору сотрудников будет создана новая запись работника, и статус записи кандидата изменится на "Принят на работу". В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Чтобы выполнить эту процедуру, перейдите в раздел "Управление персоналом" > "Набор сотрудников" > "Заявления" > "Заявления" 

1. Выберите заявление для внешнего кандидата.
2. В списке перейдите по ссылке в выбранной строке.
3. Щелкните "Статус заявления".
4. Щелкните "Нанять нового работника".
5. В поле "Дата приема сотрудника на работу" введите дату и время.
6. В поле "Должность" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
7. В списке перейдите по ссылке в выбранной строке.
8. В поле "Начало назначения" введите дату и время.
9. Щелкните "Нанять нового работника".
10. Разверните информационное поле "История занятости".
11. Разверните информационное поле "Текущие должности".
12. Разверните информационное поле "История занятости".
13. Разверните информационное поле "Текущие должности".
14. Разверните или сверните раздел "Адреса".
15. Разверните или сверните раздел "Контактная информация".
16. Разверните или сверните раздел "Личные данные".
