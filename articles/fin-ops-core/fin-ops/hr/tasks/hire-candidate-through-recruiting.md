# <a name="hiring-candidate-through-recruiting"></a>Прием кандидата через набор сотрудников

[!include [banner](../../includes/banner.md)]

> [!IMPORTANT]
> Функции набора персонала в этом разделе будут называться проектами по набору сотрудников и фокусированы на соискателях, приложениях и проектах по набору сотрудников.  

Эта процедура позволяет специалисту по найму принять на работу кандидата, подавшего заявление через конкретный проект по набору сотрудников. При приеме кандидата на работу через проект по набору сотрудников создается новая запись работника, и статус записи кандидата изменяется на **Принят на работу**. Компания **USMF** с демонстрационными данными используется для создания этой процедуры.

1. Перейдите в раздел **Управление персоналом \> Набор сотрудников \> Заявления \> Заявления**. 
2. Выберите заявление для внешнего кандидата.
3. В списке выберите ссылку в выбранной строке.
4. Выберите пункт **Состояние заявления**.
5. Выберите **Нанять нового работника**.
6. В поле **Дата приема сотрудника на работу** введите дату и время.
7. В поле **Должность** выберите стрелку раскрывающегося списка, чтобы открыть подстановку.
8. В списке выберите ссылку в выбранной строке.
9. В поле **Начало назначения** введите дату и время.
10. Выберите **Нанять нового работника**.
11. Разверните информационное поле **История занятости**.
12. Разверните информационное поле **Текущие должности**.
13. Разверните информационное поле **История занятости**.
14. Разверните информационное поле **Текущие должности**.
15. Разверните или сверните раздел **Адреса**.
16. Разверните или сверните раздел **Контактная информация**.
17. Разверните или сверните раздел **Личные данные**.
