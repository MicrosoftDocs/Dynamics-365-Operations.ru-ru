# <a name="develop-and-open-job-requisition"></a>Разработка и открытие заявки на должность

[!include [banner](../../includes/banner.md)]

> [!IMPORTANT]
> Функции, перечисленные в этом разделе, в настоящее время доступны для клиентов Human Resources в инфраструктуре Finance.  


Проекты по набору сотрудников помогают управлять процессом найма. Для каждого проекта по набору сотрудников можно настроить сведения, например должность, для которой производится набор, имя агента по найму, статус проекта и отдел, в котором будет находиться должность. После создания проекта по набору сотрудников можно написать объявление о найме для него, опубликовать его на странице **Самообслуживание сотрудников**, связать заявления о приеме на работу с проектом и отслеживать действия для проекта. Компания **USMF** с демонстрационными данными используется для создания этой процедуры.

1. Перейдите в раздел **Управление персоналом \> Набор сотрудников \> Проекты по набору сотрудников \> Проекты по набору сотрудников**.
2. Выберите **Создать**.
3. В поле **Проект по набору сотрудников** введите значение.
4. В поле **Описание** введите значение.
5. В поле **Специалист по найму** выберите стрелку раскрывающегося списка, чтобы открыть подстановку.
6. В списке найдите и выберите требуемую запись.
7. В списке выберите ссылку в выбранной строке.
8. Выберите **Выбрать**.
9. В поле **Подразделение** выберите стрелку раскрывающегося списка, чтобы открыть поиск.
10. В списке выберите ссылку в выбранной строке.
11. В поле **Должность** выберите стрелку раскрывающегося списка, чтобы открыть подстановку.
12. В списке найдите и выберите требуемую запись.
13. В списке выберите ссылку в выбранной строке.
14. В поле **Количество вакансий** введите число.
15. В поле **Специалист по комплектации штата** нажмите стрелку раскрывающегося списка, чтобы открыть поиск.
16. В списке найдите и выберите требуемую запись.
17. В списке выберите ссылку в выбранной строке.
18. Выберите **Выбрать**.
19. В поле **Крайний срок подачи заявления** введите дату.
20. Выберите **Носитель**.

    Проекты по набору сотрудников включают параметр для определения медиаплощадок, которые будут использоваться для рекламы открытых позиций.

21. Выберите **Создать**.
22. В поле **Носитель** выберите стрелку раскрывающегося списка, чтобы открыть подстановку.
23. В списке выберите ссылку в выбранной строке.
24. В поле **Дата начала** введите дату.
25. В поле **Дата окончания** введите дату.
26. Нажмите **Сохранить**.
27. Закройте страницу.
28. Выберите **Объявления о приеме на работу**.
29. Нажмите **Сохранить**.
30. Закройте страницу.
31. Выберите или снимите флажок **Отобразить на портале самообслуживания сотрудников**.

    Выберите этот флажок, чтобы сделать проект по набору сотрудников видимым для сотрудников на их странице **Самообслуживание сотрудников**.

32. Выберите статус **Проект по набору сотрудников**.
33. Выберите **Пуск**.

    Статус **Начато** указывает, что проект готов получать заявления.

34. Нажмите **ОК**.
