# <a name="enter-project-timesheets"></a>Ввод табелей учета рабочего времени проекта

Эта процедура позволят создать табель учета рабочего времени с помощью пустой страницы такого табеля. Новый табель учета рабочего времени можно создать на основе сведений из предыдущего табеля или из назначений проекта и действий на странице **Избранное**. По умолчанию на странице списка **Все табели учета рабочего времени** показаны все табели учета рабочего времени за текущий период. Можно использовать поле **Показать** на странице **Мои табели учета рабочего времени**, чтобы отфильтровать список табелей по периоду времени или проекту или для просмотра табелей, которые были созданы от имени других работников. Компания **USSI** с демонстрационными данными используется для создания этой процедуры.

1. Перейдите в раздел **Управление и учет по проектам \> Табели учета рабочего времени \> Мои табели учета рабочего времени**.
2. Выберите **Создать**, чтобы ввести новый табель учета рабочего времени.

    По умолчанию в поле **Ресурс** отображается работник, который назначен текущему пользователю.

    Если пользователь определен как представитель, будут перечислены имена, чтобы пользователь мог ввести табель от их имени.

3. В поле **Дата** введите дату. Если выбран этот параметр, новые строки табеля будут создаваться с использованием параметров табелей, которые были установлены как избранное.
4. Нажмите **ОК**.
5. Выберите **Создать строку**.
6. В списке пометьте выбранную строку. По умолчанию в поле **Юридическое лицо** отображается текущее юридическое лицо.
7. В поле **Проект** нажмите стрелку раскрывающегося списка, чтобы открыть подстановку.
8. В списке найдите и выберите требуемую запись.
9. В списке выберите ссылку в выбранной строке.
10. В поле **Номер действия** нажмите стрелку раскрывающегося списка, чтобы открыть подстановку.
11. В списке найдите и выберите требуемую запись.
12. В списке выберите ссылку в выбранной строке.
13. В поле **Категория** нажмите стрелку раскрывающегося списка, чтобы открыть подстановку.
14. В списке найдите и выберите требуемую запись.
15. В списке выберите ссылку в выбранной строке.
16. Введите количество часов, отработанных за каждый день. Вводите количество часов в десятичном формате. Например, если вы проработали два часа и пятнадцать минут, введите значение **2,25**.

    В разделе **Сведения по строке** доступны следующие параметры:

    - Добавьте информацию о налогах и финансовых аналитиках на вкладках **Общие** и **Финансовые аналитики**.
    - Добавьте комментарии к строке табеля на вкладке **Комментарии**.

17. В области действий выберите **Рабочий процесс**, чтобы открыть раскрывающееся диалоговое окно.
18. Выберите **Отправить**.
19. Выберите **Отправить**.
