# <a name="enter-project-timesheets"></a>Ввод табелей учета рабочего времени проекта

Эта процедура позволят создать табель учета рабочего времени с помощью пустой формы такого табеля. Новый табель учета рабочего времени можно создать на основе сведений из предыдущего табеля или из назначений проекта и действий на странице **Избранное**. По умолчанию на странице списка **Все табели учета рабочего времени** показаны все табели учета рабочего времени за текущий период. Можно использовать раскрывающийся список в поле **Показать** на странице **Мои табели учета рабочего времени**, чтобы отфильтровать список табелей по периоду времени или проекту или для просмотра табелей, которые были созданы от имени других работников. В качестве компании с демонстрационными данными для создания этой процедуры используется USSI.  

1. В **области переходов** выберите **Модули > Управление и учет по проектам > Табели учета рабочего времени > Мои табели учета рабочего времени**.
2. Чтобы ввести новый табель учета рабочего времени, щелкните **Создать**.
    - В раскрывающемся списке "Ресурса" отображается работник, по умолчанию назначенный текущему пользователю.  
    - Если пользователь определен как представитель, здесь будут перечислены имена, чтобы пользователь мог ввести табель от их имени.  
3. В поле **Дата** введите дату. Если выбран этот параметр, новые строки табеля будут создаваться с использованием параметров табелей, которые были установлены как избранное.  
4. Щелкните **OK**.
5. Щелкните **Новая строка**.
6. В списке пометьте выбранную строку. В поле **Юридическое лицо** отображается текущее юридическое лицо по умолчанию.   
7. В поле **Проект** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
8. Поиск и выбор требуемой записи в списке.
9. В списке перейдите по ссылке в выбранной строке.
10. В поле **Номер мероприятия** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
11. В списке найдите и выберите требуемую запись.
12. В списке перейдите по ссылке в выбранной строке.
13. В поле **Категория** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
14. Поиск и выбор требуемой записи в списке.
15. В списке перейдите по ссылке в выбранной строке.
16. Введите количество часов, отработанных в каждый из дней. Вводите количество часов в десятичном формате. Например, если вы проработали два часа и пятнадцать минут, введите значение 2,25.   
17. В поле **Сведения по строке** доступны следующие параметры:
    - Добавьте информацию о налогах и финансовых аналитиках на вкладке **Общие** и **Финансовые аналитики**.
    - Добавьте комментарии к строке табеля на вкладке **Комментарии**.
20. На **Панели операций** щелкните **Рабочий процесс**, чтобы открыть раскрывающееся диалоговое окно.
21. Щелкните **Отправить**.
22. Щелкните **Отправить**.
