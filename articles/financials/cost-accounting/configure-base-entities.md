# <a name="configure-base-entities"></a>Настройка базовых объектов 

[!include[banner](../includes/banner.md)]


Учет затрат — это независимый модуль, для работы которого требуется правильная настройка следующих базовых объектов в Dynamics 365 for Finance and Operations, Enterprise Edition.

-  Денежный
-  Валютный курс
-  Фискальный календарь  

Прежде чем использовать модуль "Учет затрат", вы должны определить для своей компании валюту, валютный курс и финансовых календарь. 

Чтобы узнать больше об этих объектах, пройдите курс обучения электронного обучения [Модуль 2. Настройка валюты и валютного курса](https://mbspartner.microsoft.com/AX/CourseModules/1215) (требуется  учетная запись CustomerSource).
