[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="c5e30-104">Книга учета затрат представляет единицу отчетности в целом.</span><span class="sxs-lookup"><span data-stu-id="c5e30-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="c5e30-105">Она определяется аналитикой элементов затрат, статистической аналитикой, финансовым календарем и валютой.</span><span class="sxs-lookup"><span data-stu-id="c5e30-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="c5e30-106">От концепции юридических лиц она не зависит.</span><span class="sxs-lookup"><span data-stu-id="c5e30-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="c5e30-107">Юридическое лицо и его данные могут быть связаны с множеством книг учета затрат.</span><span class="sxs-lookup"><span data-stu-id="c5e30-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="c5e30-108">В этой записи используется компания с демонстрационными данными USP2.</span><span class="sxs-lookup"><span data-stu-id="c5e30-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="c5e30-109">Перейдите в раздел "Учет затрат" > "Настройка главной книги" > "Книги учета затрат".</span><span class="sxs-lookup"><span data-stu-id="c5e30-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="c5e30-110">Щелкните "Создать".</span><span class="sxs-lookup"><span data-stu-id="c5e30-110">Click New.</span></span>
3. <span data-ttu-id="c5e30-111">В поле "Имя" введите значение.</span><span class="sxs-lookup"><span data-stu-id="c5e30-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="c5e30-112">В поле "Аналитика элемента затрат" введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="c5e30-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="c5e30-113">В поле "Финансовый календарь" введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="c5e30-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="c5e30-114">В поле "Валюта учета" введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="c5e30-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="c5e30-115">В поле "Тип валютного курса" введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="c5e30-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="c5e30-116">В поле "Статистическая аналитика" введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="c5e30-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="c5e30-117">Нажмите кнопку "Сохранить".</span><span class="sxs-lookup"><span data-stu-id="c5e30-117">Click Save.</span></span>
