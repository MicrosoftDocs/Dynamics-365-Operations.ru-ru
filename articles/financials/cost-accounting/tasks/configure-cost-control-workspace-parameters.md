# <a name="configure-cost-control-workspace-parameters"></a>Настройка параметров рабочей области управления затратами

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Эта процедура используется для настройки рабочей области "Управление затратами" таким образом, чтобы менеджеры на различных уровнях организации могли получать представление о своих объектах затрат, например центрах затрат и группах продуктов. Для создания этой записи использовалась демонстрационная компания USP2.

1. Перейдите в раздел "Учет затрат" > "Настройка" > "Конфигурация рабочей области управления затратами".
2. Щелкните "Создать".
3. В поле "Имя" введите значение.
4. В поле "Описание" введите значение.
5. Выберите значение "Да" в поле "Опубликовано".
    * При выборе варианта «Да» просматривать отчет в рабочей области "Управление затратами" могут пользователи, которым назначена одна из следующих ролей: диспетчер учета затрат, бухгалтер по учету затрат, младший бухгалтер по учету затрат или контроллер объектов затрат. При выборе варианта «Нет» просматривать отчет в рабочей области "Управление затратами" могут только пользователи, которым назначена одна из следующих ролей: диспетчер учета затрат, бухгалтер по учету затрат, младший бухгалтер по учету затрат или контроллер объектов затрат.  
6. Разверните раздел "Фильтрация данных".
7. В поле "Единица управления затратами" введите или выберите значение.
8. В поле "Исходная версия бюджета" введите или выберите значение.
9. В поле "Иерархия аналитик элементов затрат" введите или выберите значение.
10. В поле "Иерархия аналитик объектов затрат" введите или выберите значение.
11. Разверните раздел "Назначить записи расчета".
12. Щелкните "Создать".
13. В списке пометьте выбранную строку.
14. В поле "Период финансового календаря" введите или выберите значение.
15. В поле "Фактическая версия" введите или выберите значение.
16. Разверните раздел "Финансовые периоды по столбцам".
17. Выберите "Да" в поле "Текущий период".
18. Разверните раздел "Столбцы для отображения затрат".
19. Выберите "Да" в поле "Постоянные затраты".
20. Выберите "Да" в поле "Переменные затраты".
21. Выберите "Да" в поле "Общие затраты".
22. Нажмите кнопку "Сохранить".
23. Закройте страницу.
24. Перейдите в раздел "Учет затрат" > "Рабочие области" > "Управление затратами".
25. Выберите отчет для просмотра постоянных, переменных, общих и неклассифицированных затрат для выбранных финансовых периодов.
26. В поле "Период финансового календаря" введите или выберите значение.
27. В поле "Узел иерархии аналитик объектов затрат" введите или выберите значение.
    * Выбрав иерархию аналитик объектов затрат, разверните иерархию аналитик элементов затрат для просмотра желаемых значений затрат. Например, можно развернуть иерархию до производственных накладных расходов, чтобы увидеть значение.
