# <a name="process-and-trace-source-data"></a>Обработка и трассировка исходных данных

[!include [task guide banner](../../includes/task-guide-banner.md)]

Вся обработка данных выполняется посредством заданий. Для каждого задания и поставщика данных создается журнал для документирования того, что процесс был выполнен, и что были обработаны записи в текущем задании. Эта процедура используется для настройки источника данных с последующим отслеживанием основания конкретной записи затрат. В этой записи используется компания с демонстрационными данными USP2. Перед выполнением этой задачи не забудьте воспроизвести следующие проводники по задачам: "Создание книги учета затрат", "Определение единиц управления затратами" и "Управление источником данных для книги учета затрат".

1. Перейдите в раздел "Учет затрат" > "Настройка главной книги" > "Книги учета затрат".
2. В списке найдите и выберите требуемую запись.
    * Выберите ранее созданную книгу учета затрат.  
3. Щелкните "Фактические версии".
4. На панели операций щелкните "Обработка исходный данных".
5. Щелкните "Журналы переноса записей главной книги".
6. В списке найдите и выберите требуемую запись.
7. Щелкните "Записи журнала".
8. В списке пометьте выбранную строку.
9. Щелкните "Записи затрат".
10. Щелкните "Исходная запись".
11. На панели операций щелкните "Обработка исходный данных".
12. Щелкните "Главная книга".
13. В поле "Период финансового календаря" введите или выберите значение.
    * В этом примере выберите "Fiscal 2017 Period 9".  
14. Нажмите кнопку "OК".
