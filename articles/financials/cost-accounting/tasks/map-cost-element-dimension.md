# <a name="map-a-cost-element-dimension"></a>Сопоставление аналитики элемента затрат

[!include [task guide banner](../../includes/task-guide-banner.md)]

Контроллер затрат может использовать эту процедуру для сопоставления аналитики элемента затрат аналитике элемента затрат в юридическом лице MXMF. В этой записи используется компания с демонстрационными данными USP2.

1. Перейдите в раздел "Учет затрат" > "Аналитики" > "Аналитики элемента затрат".
2. В списке найдите и выберите требуемую запись.
    * В этом примере выберите "Элементы затрат".  
3. Щелкните "Сопоставления аналитик".
4. Щелкните "Настроить сопоставления из этой аналитики".
5. Щелкните "Создать".
6. В поле "В аналитику" введите или выберите значение.
    * В этом примере выберите "Элементы затрат MXMF".  
7. Щелкните "Создать".
8. В списке пометьте выбранную строку.
9. В поле "Из элемента аналитики" введите или выберите значение.
    * В этом примере выберите элемент аналитики 606400, "Расходы на телефон и факс".  
10. В поле "В элемент аналитики" введите или выберите значение.
    * В этом примере выберите элемент аналитики 6001004 "Telefono".  
11. Нажмите кнопку "Сохранить".
