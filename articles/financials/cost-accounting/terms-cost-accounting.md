<span data-ttu-id="7a0aa-104">В этой разделе определяются ключевые термины, используемые в модуле учета затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-104">This topic defines the key terms that are used in Cost accounting.</span></span>

<span data-ttu-id="7a0aa-105">**База распределения**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-105">**Allocation base**</span></span>

<span data-ttu-id="7a0aa-106">База распределения используется для количественного измерения операций, например используемого машинного времени, потребленных киловатт-часов или занимаемой площади.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-106">The allocation base is used to measure and quantify activities, such as machine hours that are used, kilowatt hours that are consumed, or square footage that is occupied.</span></span> <span data-ttu-id="7a0aa-107">Она используется в качестве базы для распределения затрат между одним или несколькими объектами затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-107">It's used as basis for allocating costs to one or more cost objects</span></span>

<span data-ttu-id="7a0aa-108">**Учет затрат**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-108">**Cost accounting**</span></span>

<span data-ttu-id="7a0aa-109">Учет затрат позволяет вам собирать данные из различных источников, таких как ГК, вспомогательные книги, бюджеты и статистическая информация.</span><span the source data.</span></span> <span data-ttu-id="7a0aa-113">Однако при сборе данных по затратам из различных источников и особенно при импорте счетов ГК из главной книги в Microsoft Dynamics 365 for Finance and Operations в качестве элементов затрат, возникает дублирование данных, поскольку одни и те же данные существуют как в главной книге, так и в модуле учета затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-113">However, when you collect cost data from various sources, and especially when you import the main accounts from General ledger in Microsoft Dynamics дублирование необходимо, поскольку вы используете финансовое управление для внешней отчетности, а учет затрат — для внутренней отчетности.</span><span class="sxs-lookup"><span <span data-ttu-id="7a0aa-115">**Книга учета затрат**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-115">**Cost accounting ledger**</span></span>

<span data-ttu-id="7a0aa-117">**Запись себестоимости**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-117">**Cost entry**</span></span>

<span data-ttu-id="7a0aa-118">Записи цены являются результатом перемещения через соединители данных из записей ГК, распределений затрат и разнесенных записей затрат в журналах затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-118">Cost entries are the result of a transfer via data connectors from general ledger entries, cost allocations, and posted cost entries in cost journals.</span></span>

<span data-ttu-id="7a0aa-120">Объект любого типа, выбранный для управления затратами.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-120">Any type of object that is selected for cost control.</span></span> <span data-ttu-id="7a0aa-121">Затраты или выручка разносятся либо непосредственно, либо распределяются по объектам затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-121">Costs or revenues are either directly posted on or allocated to cost objects.</span></span> <span data-ttu-id="7a0aa-122">Некоторые типичные объекты затрат:</span><span class="sxs-lookup"><span data-stu-id="7a0aa-122">Some typical cost objects are:</span></span>

-  <span data-ttu-id="7a0aa-123">Товары</span><span class="sxs-lookup"><span data-stu-id="7a0aa-123">Products</span></span>
-  <span data-ttu-id="7a0aa-124">Проекты</span><span class="sxs-lookup"><span data-stu-id="7a0aa-124">Projects</span></span>
-  <span data-ttu-id="7a0aa-125">Отделы</span><span class="sxs-lookup"><span data-stu-id="7a0aa-125">Departments</span></span>
-  <span data-ttu-id="7a0aa-126">Места возникновения затрат</span><span class="sxs-lookup"><span data-stu-id="7a0aa-126">Cost centers</span></span>

<span data-ttu-id="7a0aa-127">Руководство использует объекты затраты для определения количественной меры затрат, но также и для проведения анализа прибыльности.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-127">Management uses cost objects to quantify costs, but also to drive profitability analysis.</span></span>

<span data-ttu-id="7a0aa-128">**Элемент затрат**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-128">**Cost element**</span></span>

<span data-ttu-id="7a0aa-129">Используется в качестве функции для отслеживания и классификации затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-129">Used as a function to track and categorize costs.</span></span> <span data-ttu-id="7a0aa-130">Имеется два типа элементов затрат: первичные и вторичные.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-130">There are two types of cost elements: primary and secondary.</span></span>

<span data-ttu-id="7a0aa-131">Элементы первичных затрат представляют поток затрат из финансового учета в модуль "Учет затрат".</span><span class="sxs-lookup"><span data-stu-id="7a0aa-131">Primary cost elements represent the cost flow from financial accounting to Cost accounting.</span></span> <span data-ttu-id="7a0aa-132">Структура элемента затрат обычно соответствует структуре счета прибылей и убытков в главной книге, где элемент затрат может соответствовать счету ГК.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-132">The structure typically corresponds to the profit and loss account structure in the general ledger where a cost element can correspond to a main account.</span></span> <span data-ttu-id="7a0aa-133">Не все счета ГК необходимо представлять в виде элементов затрат, в зависимости от требований компании.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-133">Not all main accounts must be represented as cost elements, depending on business requirements.</span></span> 

<span data-ttu-id="7a0aa-134">Элементы вторичных затрат представляют внутренний поток затрат, поскольку эти затраты используются только в модуле "Учет затрат".</span><span class="sxs-lookup"><span data-stu-id="7a0aa-134">Secondary cost elements represent the internal cost flow because these costs are only used in Cost accounting.</span></span> <span data-ttu-id="7a0aa-135">Они используются в правилах свертки затрат для агрегирования затрат в содержательные интервалы, используемые при расчете накладных расходов.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-135">They are used in cost roll-up rules to aggregate costs into meaningful buckets used by overhead calculation.</span></span> 

<span data-ttu-id="7a0aa-136">**Классификация затрат**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-136">**Cost classification**</span></span>

-   <span data-ttu-id="7a0aa-139">**По элементам** — материалы, труд и накладные расходы.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-139">**By elements** – Materials, labor, and expenses.</span></span>
-   <span data-ttu-id="7a0aa-140">**По возможности отслеживания** — прямые затраты и косвенные затраты.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-140">**By traceability** – Direct costs and indirect costs.</span></span> <span data-ttu-id="7a0aa-141">Прямые затраты назначаются непосредственно объектам затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-141">Direct costs are assigned directly to cost objects.</span></span> <span data-ttu-id="7a0aa-142">Косвенные затраты невозможно напрямую проследить до объектов затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-142">Indirect costs aren't directly traceable to cost objects.</span></span> <span data-ttu-id="7a0aa-143">Косвенные затраты распределяются объектам затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-143">Indirect costs are allocated to cost objects.</span></span>
-   <span data-ttu-id="7a0aa-144">**По поведению** — постоянные, переменные и полупеременные.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-144">**By behavior** – Fixed, variable, and semi-variable.</span></span>

<span data-ttu-id="7a0aa-145">**Поведение затрат**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-145">**Cost behavior**</span></span>

<span data-ttu-id="7a0aa-146">Поведение затрат классифицирует затраты согласно их поведению по отношению к изменениям в ключевые бизнес-мероприятиях.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-146">Cost behavior classifies costs according to their behavior in relation to changes in key business activities.</span></span> <span data-ttu-id="7a0aa-147">Чтобы эффективно контролировать затраты, руководство должно понимать поведение затрат.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-147">To control costs effectively, management must understand the cost behavior.</span></span> <span data-ttu-id="7a0aa-148">Существует три типа поведения затрат: постоянные, переменные и полупеременной.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-148">There are three types of cost behavior pattern: fixed, variable, and semi-variable.</span></span>

- <span data-ttu-id="7a0aa-149">**Постоянные затраты** — Постоянные затраты представляют собой затраты, которые не изменяются в краткосрочной перспективе, независимо от изменений в уровне активности.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-149">**Fixed cost** - A fixed cost is a cost that doesn't vary in the short term, regardless of changes in activity level.</span></span> <span data-ttu-id="7a0aa-150">Например, постоянные затраты могут быть основными эксплуатационными расходами компании, такими как аренда, которая не изменится, даже если уровень активности увеличивается или уменьшается.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-150">For example, a fixed cost can be a basic operating expense of a business, such as rent, that won't be affected even if the activity level increases or decreases.</span></span>

- <span data-ttu-id="7a0aa-154">**Полупеременные затраты** — Полупеременные затраты частично постоянные и частично переменные.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-154">**Semi-variable cost** - Semi-variable costs are partly fixed and partly variable costs.</span></span> <span data-ttu-id="7a0aa-155">Например, плата за доступ в Интернет включает в себя стандартный ежемесячный платеж за доступ и плату за широкополосный трафик.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-155">For example, an Internet access fee includes a standard monthly access fee and a broadband usage fee.</span></span> <span data-ttu-id="7a0aa-156">Стандартный ежемесячный платеж является постоянными затратами, тогда как плата за широкополосный трафик является переменными затратами.</span><span class="sxs-lookup"><span data-stu-id="7a0aa-156">The standard monthly access fee is a fixed cost, whereas the broadband usage fee is a variable cost.</span></span>

<span data-ttu-id="7a0aa-157">**Единица управления затратами**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-157">**Cost control unit**</span></span>

<span data-ttu-id="7a0aa-160">**Распространение затрат**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-160">**Cost distribution**</span></span>

<span data-ttu-id="7a0aa-171">**Политика распределения затрат**</span><span class="sxs-lookup"><span data-stu-id="7a0aa-171">**Cost allocation policy**</span></span>
