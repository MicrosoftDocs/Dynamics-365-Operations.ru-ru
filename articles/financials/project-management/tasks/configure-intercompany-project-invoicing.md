# <a name="configure-intercompany-project-invoicing"></a>Настройка внутрихолдингового выставления накладных по проектам

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой теме показано, как настроить выставление накладных по проекту между двумя компаниями в организации. В этой задаче используется набор данных компании USSI.

1. В области переходов выберите **Модули > Расчеты с поставщиками > Поставщики > Все поставщики**.
2. В списке **Все поставщики** найдите и выберите требуемую запись.
3. На панели операций выберите **Общее**.
4. Выберите **Внутрихолдинговый**.
5. Установите для параметра **Активно** значение **Да**, чтобы включить внутрихолдинговые торговые отношения.
6. В поле **Компания-клиент** введите или выберите значение.
7. В поле **Мой счет** введите или выберите значение.
8. Нажмите **Сохранить**.
9. Закройте страницы и вернитесь на домашнюю страницу.
10. В области переходов выберите **Модули > Управление и учет по проектам > Настройка > Параметры модуля "Управление и учет по проектам**.
11. Выберите вкладку **Внутрихолдинговый**.
12. Переместите ползунок в положение **Да**, чтобы включить планирование и графики внутрихолдинговых ресурсов.
13. В списке пометьте выбранную строку.
14. Выберите **Создать**.
15. В поле **Получающее юридическое лицо** введите или выберите значение.
16. Установите флажок **Начислить выручку**.
17. В поле **Категория табеля учета рабочего времени по умолчанию** введите или выберите значение.
18. В поле **Категория расходов по умолчанию** введите или выберите значение.
19. Нажмите **Сохранить**.
20. Закройте страницу.
21. В области переходов выберите **Модули > Управление и учет по проектам > Настройка > Разноска > Настройка разноски ГК**.
22. В поле **Типы счетов ГК** выберите параметр.
23. Выберите **Создать**.
24. В поле **Счет ГК** в новой строке укажите требуемые значения.
25. Нажмите **Сохранить**.
26. Закройте страницу.
27. В области переходов выберите **Модули > Управление и учет по проектам > Настройка > Цены > Трансферная цена**.
28. Выберите **Создать**.
29. В поле **Действует с** введите дату.
30. В поле **Получающее юридическое лицо** введите или выберите значение.
31. В поле **Модель трансферных цен** выберите параметр.
32. В поле **Ценообразование** введите число.
33. Нажмите **Сохранить**.
