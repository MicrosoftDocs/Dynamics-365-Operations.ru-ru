# <a name="configure-intercompany-project-invoicing"></a>Настройка внутрихолдингового выставления накладных по проектам

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как настроить выставление накладных по проекту между двумя компаниями в организации. В этой задаче используется набор данных компании USSI.

1. Перейдите в раздел "Расчеты с поставщиками" > "Поставщики" > "Все поставщики".
2. В списке найдите и выберите требуемую запись.
3. В области действий щелкните "Общее".
4. Щелкните "Внутрихолдинговый".
5. Установите для параметра "Активно" значение "Да", чтобы включить внутрихолдинговые торговые отношения.
6. В поле "Компания-клиент" введите или выберите значение.
7. В поле "Мой счет" введите или выберите значение.
8. Нажмите кнопку "Сохранить".
9. Закройте страницу.
10. Закройте страницу.
11. Перейдите в раздел "Управление и учет по проектам" > "Настройка" > «Параметры модуля "Управление и учет по проектам"».
12. Перейдите на вкладку "Внутрихолдинговый".
13. Переместите ползунок в положение "Да", чтобы включить планирование и графики внутрихолдинговых ресурсов.
14. В списке пометьте выбранную строку.
15. Щелкните "Создать".
16. В списке пометьте выбранную строку.
17. В поле "Получающее юридическое лицо" введите или выберите значение.
18. Установите флажок "Начислить выручку".
19. В поле "Категория табеля учета рабочего времени по умолчанию" введите или выберите значение.
20. В поле "Категория расходов по умолчанию" введите или выберите значение.
21. Нажмите кнопку "Сохранить".
22. Закройте страницу.
23. Перейдите в раздел "Управление и учет по проектам" > "Настройка" > "Разноска" > "Настройка разноски ГК".
24. В поле "Типы счетов ГК" выберите параметр.
25. Щелкните "Создать".
26. В списке пометьте выбранную строку.
27. В списке пометьте выбранную строку.
28. В поле "Счет ГК" укажите требуемые значения.
29. Нажмите кнопку "Сохранить".
30. Закройте страницу.
31. Перейдите в раздел "Управление и учет по проектам" > "Настройка" > "Цены" > "Трансферная цена".
32. Щелкните "Создать".
33. В поле "Действует с" введите дату.
34. В поле "Получающее юридическое лицо" введите или выберите значение.
35. В поле "Модель трансферных цен" выберите параметр.
36. В поле "Ценообразование" введите число.
37. Нажмите кнопку "Сохранить".
