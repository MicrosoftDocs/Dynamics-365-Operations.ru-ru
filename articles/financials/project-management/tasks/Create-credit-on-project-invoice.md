# <a name="create-a-credit-note-on-project-invoices"></a>Создание кредит-нот для накладных по проекту

[!include[task guide banner](../../includes/task-guide-banner.md)]

1. Перейдите в раздел "Управление и учет по проектам" > "Проекты" > "Все проекты". 
2. В списке найдите и выберите требуемую запись. 
3. Выберите код проекта, который используется для кредит-ноты. В списке перейдите по ссылке в выбранной строке. 
4. В области действий щелкните "Управлять". 
5. Щелкните "Журналы накладных". 
6. Выберите проводку для кредит-ноты. В списке пометьте выбранную строку. 
7. Щелкните Функции. 
8. Щелкните "Выбрать" для кредит-ноты. 
9. Установите флажок, чтобы выбрать проводку для кредит-ноты.
10. Нажмите кнопку "OК". 

После этого этапа кредит-нота будет доступна для предложения по накладной.
