1.  <span data-ttu-id="655e5-118">Запишите URL-адрес Dynamics 365 для страницы **Управление и учет по проектам** \> **Табели учета рабочего времени** \> **Мои табели учета рабочего времени (оптимизировано для мобильных устройств)**.</span><span class="sxs-lookup"><span data-stu-id="655e5-118">Note the Dynamics 365 URL for the **Project management and accounting** \> **Timesheets** \> **My timesheets(Optimized for mobile)** page.</span></span>

3.  <span data-ttu-id="655e5-120">Выберите **Управление и учет по проектам** \> **Табели учета рабочего времени** \> **Мои табели учета рабочего времени (оптимизировано для мобильных устройств)**.</span><span class="sxs-lookup"><span data-stu-id="655e5-120">Select **Project management and Accounting** \> **Timesheets** \> **My timesheets(Optimized for mobile)**.</span></span>

5.  <span data-ttu-id="655e5-122">Выберите ресурс.</span><span class="sxs-lookup"><span data-stu-id="655e5-122">Select a resource.</span></span>

6.  <span data-ttu-id="655e5-123">Отображаются календарные даты для текущей недели.</span><span class="sxs-lookup"><span data-stu-id="655e5-123">The calendar dates for the current week are shown.</span></span>
