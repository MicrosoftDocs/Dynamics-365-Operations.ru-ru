# <a name="view-a-related-intercompany-voucher-from-a-journal"></a>Просмотр связанных внутрихолдинговых ваучеров из журнала

[!include [task guide banner](../../includes/task-guide-banner.md)]

В окне связанного ваучера отображается ваучер в корреспондирующей компании при разноске внутрихолдинговой проводки из общего журнала.


## <a name="post-an-intercompany-journal"></a>Разноска внутрихолдингового журнала
1. Перейдите в "Общие журналы".
2. Щелкните "Создать".
3. В списке пометьте выбранную строку.
4. В поле "Имя" введите или выберите имя внутрихолдингового журнала.
5. Выберите Строки.
6. В списке пометьте выбранную строку.
7. В поле "Счет" укажите требуемые значения.
8. В поле "Описание" введите или выберите значение.
9. В поле "Описание" введите значение.
10. Закройте страницу.
11. В поле "Дебет" введите число.
12. В поле "Корреспондирующая компания" введите или выберите корреспондирующую компанию.
13. В поле "Корреспондирующая компания" введите или выберите значение.
14. Закройте страницу.
15. В поле "Корр.счет" укажите требуемые значения.
16. Щелкните "Разнести".

## <a name="view-related-intercompany-voucher"></a>Просмотр связанного внутрихолдингового ваучера
1. Щелкните "Операция".
2. Щелкните "Связанные ваучеры".
3. В списке пометьте выбранную строку.
4. Щелкните "Операция".
