3. <span data-ttu-id="be451-107">Щелкните для перехода по ссылке в поле "Номер партии журнала".</span><span class="sxs-lookup"><span data-stu-id="be451-107">Click to follow the link in the Journal batch number field.</span></span>
4. <span data-ttu-id="be451-108">В списке пометьте выбранную строку.</span><span class="sxs-lookup"><span data-stu-id="be451-108">In the list, mark the selected row.</span></span>
5. <span data-ttu-id="be451-109">В поле "Счет" укажите требуемые значения.</span><span class="sxs-lookup"><span data-stu-id="be451-109">In the Account field, specify the desired values.</span></span>
    * <span data-ttu-id="be451-110">В этом примере определяются расходы для страховки.</span><span class="sxs-lookup"><span data-stu-id="be451-110">In this example, we are defining the expense for the insurance.</span></span> <span data-ttu-id="be451-111">Они станут периодической суммой расходов.</span><span class="sxs-lookup"><span data-stu-id="be451-111">It will become a periodic expense amount.</span></span>  
6. <span data-ttu-id="be451-112">В поле "Описание" введите значение.</span><span class="sxs-lookup"><span data-stu-id="be451-112">In the Description field, type a value.</span></span>
7. <span data-ttu-id="be451-113">В поле "Дебет" введите число.</span><span class="sxs-lookup"><span data-stu-id="be451-113">In the Debit field, enter a number.</span></span>

9. <span data-ttu-id="be451-115">Щелкните Функции.</span><span class="sxs-lookup"><span data-stu-id="be451-115">Click Functions.</span></span>
10. <span data-ttu-id="be451-116">Щелкните "Начисления ГК".</span><span class="sxs-lookup"><span data-stu-id="be451-116">Click Ledger accruals.</span></span>

14. <span data-ttu-id="be451-120">В поле "Дата начала" введите дату.</span><span class="sxs-lookup"><span data-stu-id="be451-120">In the Start date field, enter a date.</span></span>

16. <span data-ttu-id="be451-122">Закройте страницу.</span><span class="sxs-lookup"><span data-stu-id="be451-122">Close the page.</span></span>
17. <span data-ttu-id="be451-123">Нажмите кнопку "OК".</span><span class="sxs-lookup"><span data-stu-id="be451-123">Click OK.</span></span>
18. <span data-ttu-id="be451-124">Щелкните "Разнести".</span><span class="sxs-lookup"><span data-stu-id="be451-124">Click Post.</span></span>
