# <a name="create-and-assign-advanced-rule-structures"></a>Создание и назначение структур дополнительных правил

[!include [task guide banner](../../includes/task-guide-banner.md)]

Это руководство описывает создание и назначение структуры дополнительного правила в структуре счета. В данном руководстве используется демонстрационная компания USMF.


## <a name="create-an-advanced-rule-structure"></a>Создание новой структуры дополнительного правила
1. Перейдите в раздел "Главная книга" > "План счетов" > "Структуры" > "Структуры дополнительного правила".
2. Щелкните "Создать", чтобы открыть раскрывающееся диалоговое окно.
3. В поле "Структура дополнительного правила" введите имя для описания структуры правила.
4. В поле "Описание" введите значение для описания структуры.
5. Нажмите кнопку OK.
6. Нажмите "Добавить сегмент".
7. В списке сегментов выберите финансовую аналитику.
    * Например, хранилище.  
8. Нажмите "Добавить сегмент".
9. В списке нажмите на ссылку структуры дополнительного правила для ее просмотра.
10. Нажмите кнопку Активировать.
11. Нажмите кнопку Активировать.

## <a name="apply-an-advanced-rule-structure-to-an-account-structure"></a>Применение структуры дополнительного правила к структуре счета
1. Закройте форму.
2. Закройте страницу.
3. Перейдите в раздел "Главная книга" > "План счетов" > "Структуры" > "Настройка структур счета".
4. В списке найдите и выберите структуру счета, к которой необходимо применить дополнительное правило.
5. Нажмите имя структуры счета, чтобы ее открыть.
6. Щелкните "Изменить".
    * Можно также нажать "Дополнительные правила" и будет предложено перевести структуру счета в режим черновика.  
7. Щелкните "Дополнительные правила".
8. Щелкните "Создать", чтобы открыть раскрывающееся диалоговое окно.
9. В поле "Дополнительное правило" введите значение.
10. В поле "Имя" введите значение.
11. Щелкните Создать.
12. Щелкните "Добавить новые критерии".
13. В поле "Где" выберите счет ГК или финансовую аналитику.
14. В поле "Оператор" выберите параметр, например "находится в диапазоне и включает".
15. В поле "Значение" введите значение.
16. В поле введите значение.
17. Щелкните "Добавить", чтобы открыть раскрывающееся диалоговое окно.
18. В списке найдите структуру дополнительного правила, которая должна использоваться при соответствии указанных критериев.
19. Нажмите кнопку Добавить.
20. Закройте страницу.
21. Нажмите кнопку Активировать.
22. Нажмите кнопку Активировать.
