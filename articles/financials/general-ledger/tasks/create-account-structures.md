# <a name="create-account-structures"></a>Создание структур счетов

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этом руководства рассказывается о создании структуры счета. Для выполнения шагов используется компания с демонстрационными данными USMF.

1. Перейдите в раздел "Главная книга" > "План счетов" > "Структуры" > "Настройка структур счета".
2. Щелкните "Создать", чтобы открыть раскрывающееся диалоговое окно.
3. В поле "Структура счета" введите имя для описания назначения структуры счета.
4. В поле "Описание" введите описание для указания назначения структуры счета.
5. Щелкните Создать.
6. Нажмите "Добавить сегмент".
7. В списке "Аналитики" выберите аналитику для добавления в структуру счета.
8. Нажмите "Добавить сегмент".
9. Нажмите "Добавить сегмент".
10. В списке "Аналитики" выберите аналитику для добавления в структуру счета.
11. Нажмите "Добавить сегмент".
12. Нажмите "Добавить сегмент".
13. В списке "Аналитики" выберите аналитику для добавления в структуру счета.
14. Нажмите "Добавить сегмент".
15. В сетке выберите сегмент для редактирования допустимых значений.
    * Например, щелкните "Счет ГК".  
16. В поле "Оператор" выберите параметр, например "находится в диапазоне и включает".
17. В поле "Значение" введите значение.
    * Например, 600000.  
18. В поле введите значение.
    * Например, 699999.  
19. Нажмите кнопку Применить.
20. В сетке выберите сегмент для редактирования допустимых значений.
    * Например, "Отдел".  
21. В поле "Оператор" выберите параметр, например "находится в диапазоне и включает".
22. В поле "Значение" введите значение.
    * Например, 022.  
23. В поле введите значение.
    * Например, 031.  
24. Щелкните "Добавить новые критерии".
25. В поле "Оператор" выберите параметр, например "находится в диапазоне и включает".
26. В поле "Значение" введите значение.
    * Например, 033.  
27. В поле введите значение.
    * Например, 034.  
28. Нажмите кнопку Применить.
29. В сетке выберите сегмент для редактирования допустимых значений.
    * Например, "Место возникновения затрат".  
30. В поле CostCenter введите значение.
    * Например, 007..021.  
31. Нажмите кнопку Добавить.
32. В поле MainAccount введите значение.
    * Например, 600000..699999  
33. В сетке выберите сегмент для редактирования допустимых значений.
    * Например, "Отдел".  
34. В поле "Подразделение" введите значение.
    * Например, 032.  
35. В поле CostCenter введите значение.
    * Например, 086.  
36. Щелкните "Проверить".
37. Нажмите кнопку Активировать.
38. Нажмите кнопку Активировать.
