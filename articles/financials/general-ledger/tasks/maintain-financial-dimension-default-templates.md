# <a name="maintain-financial-dimension-default-templates"></a>Ведение шаблонов финансовой аналитики по умолчанию

[!include[task guide banner](../../includes/task-guide-banner.md)]

Эта процедура показывает порядок настройки шаблонов финансовых аналитик по умолчанию. Шаблоны можно использовать, чтобы быстро распределить суммы по исходным документам.

1. Перейдите в раздел "Главная книга" > "План счетов" > "Аналитики" > "Шаблоны финансовой аналитики по умолчанию".
2. Щелкните "Создать".
3. В поле "Код шаблона" введите код шаблона.
4. Ввод процент для первых значений аналитики.
5. Введите значения финансовой аналитики, которые соответствуют введенному проценту.
6. Нажмите кнопку Добавить.
7. Введите другой процент для следующих значений аналитики.
    * В этом примере введите процент, чтобы все еще было назначено менее 100%.  
8. Введите значения финансовой аналитики, которые соответствуют введенному проценту.
9. Нажмите кнопку Добавить.
10. Введите другой процент, который доведет сумму процентных значений до 100%.
11. Введите значения финансовой аналитики, которые соответствуют введенному проценту.
12. Нажмите кнопку "Сохранить".
