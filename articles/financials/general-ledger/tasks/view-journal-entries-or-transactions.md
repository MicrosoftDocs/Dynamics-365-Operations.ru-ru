# <a name="view-journal-entries-or-transactions"></a>Просмотр записей или проводок в журнале

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как использовать запрос проводок по ваучеру для поиска записей журнала или проводок.

1. Перейдите в раздел "Главная книга" > "Запросы и отчеты" > "Проводки по ваучеру".
2. Выберите поле для которого требуется определить критерии фильтра.
3. Введите свои критерии фильтра для выбранного поля.
    * Можно фильтровать по одному значению или по диапазону. При определении диапазона убедитесь, что используется правильный синтаксис. Значения должны быть разделены двумя точками (..).  
4. Щелкните вкладку "Соединения" для добавления дополнительных таблиц, из которых будет выполняться фильтрация.
5. В дереве выберите "Таблицы/Запись в общем журнале".
6. Щелкните "Добавить соединение таблицы".
7. Нажмите кнопку "Отмена", если вы решили не добавлять дополнительную таблицу.
8. Перейдите на вкладку "Диапазон".
9. Для выполнения запроса нажмите кнопку OK.
10. Щелкните "Источник проводки".
    * Различные кнопки рядом с сеткой можно использовать, чтобы исследовать дополнительные сведения о выбранной записи ваучера. Некоторые кнопки могут быть недоступны в зависимости от типа проводки и свойств проводки.  
11. Закройте страницу.
12. Щелкните "Исходный документ".
13. Закройте страницу.
