# <a name="general-ledger-account-balances"></a>Сальдо счета главной книги

[!include[banner](../includes/banner.md)]


Эта статья описывает 2 метода для просмотра сальдо счета ГК — страница списка оборотно-сальдовой ведомости и финансовые отчеты. Она также рассматривает, как обновлять сальдо набора аналитик.

Существует несколько способов просмотра сальдо в главной книге. Вот некоторые из самых популярных способов:

-   Оборотно-сальдовая ведомость
-   Финансовые отчеты
-   Проводки ваучера
-   Отчеты по главной книге

Самыми популярными способами являются страница списка пробного баланса и финансовые отчеты.

## <a name="trial-balance"></a>Оборотно-сальдовая ведомость
Пробный баланс — это страница списка, на которой отображаются сальдо счета и аналитики за определенный период времени. При открытии пробного баланса в первый раз он обновляется для отображения сальдо для дат и свойств, заданных на странице "Параметры". Свойства, которые можно изменить на странице "Параметры", — это даты, слой разноски, способ отображения входящих сальдо и то, какие типы закрывающих проводок отображать. 

Когда пользователь меняет параметры, сальдо обновляются. Пользователь также может выбрать набор аналитик, для которого следует просмотреть сальдо, и указать, следует ли отображать каждую аналитику в отдельных столбцах. 

Пользователи могут выполнить детализацию сальдо для просмотра проводок, которые составляют сальдо.    

Дополнительные сведения см. в разделе [Просмотр финансовых отчетов](view-financial-reports.md).
