<span data-ttu-id="1392c-106">Эта процедура была создана с использованием демонстрационных данных компании DEMF, но можно использовать любую компанию с демонстрационными данными для этих целей.</span><span class="sxs-lookup"><span data-stu-id="1392c-106">This procedure was created using the demo data company DEMF but you can use any demo data company for this purpose.</span></span>
