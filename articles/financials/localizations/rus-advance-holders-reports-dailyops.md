# <a name="advance-reports-with-budget-control"></a>Авансовые отчеты с бюджетным контролем

[!include [banner](../includes/banner.md)]

## <a name="accounting-distributions-and-subledger-journals"></a>Распределения по бухгалтерским счетам и журналам субкниги
Строки журнала субкниги — это записи ГК, разнесенные в главную книгу с помощью общего журнала. Можно создать субкниги из документов-источников, таких как накладные, отборочные накладные и отгрузочные накладные для клиентов и поставщиков. Перед разноской сведения о поставщике в главной книге можно просмотреть или изменить журналы субкниги с помощью метода распределения. Этот метод позволяет распределить суммы разнесения между несколькими финансовыми аналитиками и изменяет номер счета ГК или значения финансовой аналитики по умолчанию на основе разрешений пользователя. Распределения служат в качестве интерфейса в журналы субкниги и содержат только одну сторону учетной записи.

Авансовый отчет используется для сообщения о путевых расходах, понесенных сотрудником во время командировки. Можно распределить сумму расходов между аналитиками учета. Можно просмотреть созданные записи ГК и проверить распределение суммы расходов. Разноска авансового отчета поддерживает функции строк ГК, корректировки курса и авансовые корректировки. Если проводки авансового отчета разделены по распределению, отмененный авансовый отчет поддерживает создание учетной проводки со сторно для учета распределения.

## <a name="generate-and-post-advance-report-lines-manually"></a>Создание и разноска строк расширенного отчета вручную
Авансовый отчет используется для сообщения о путевых расходах, понесенных сотрудником во время командировки. Эта процедура используется для создания и разноски строк авансового отчета вручную.

Можно распределить сумму расходов между несколькими аналитиками учета. Можно просмотреть созданные записи ГК и проверить распределение суммы расходов.

1.  Щелкните **Расчеты с поставщиками** \> **Общее** \> **Подотчетные лица** \> **Авансовые отчеты**.
2.  Нажмите CTRL+N, чтобы создать новый авансовый отчет. Для получения дополнительных сведений см.
3.  Щелкните экспресс-вкладку **Строки авансового отчета**, затем щелкните **Добавить строку** для создания новой строки.
4.  В поле **Дата документа по расходу** введите дату документа, описывающего расход.
5.  В поле **Номер документа** введите номер подтверждающего документа.
6.  В поле **Наименование документа** введите наименование подтверждающего документа.
7.  В поле **Валюта** выберите валюту, которая используется для проводки.
8.  В поле **Сумма** введите потраченную сумму проводки.
9.  В поле **Подтвержденная сумма авансового отчета** введите подтвержденный расход для авансового отчета.
10. В поле **Счет учета** выберите счет ГК, к которому относится расход.
11. Щелкните **Распределить суммы**.
12. В поле **Принцип распределения** выберите, следует ли распределять суммы без учета скидок и наценок или с процентом скидки.
13. Можно создать распределения следующими способами:
    
      - Чтобы создать несколько распределений с одинаковым количеством, процентом или распределением суммы, щелкните **Разбиение** для каждого распределения. Выберите счета ГК для каждого распределения и щелкните **Распределить поровну**.
    
      - Для создания распределения по одному щелкните **Разбиение**. Выберите счет ГК для распределения строки накладной и введите количество, процент или сумму для распределения. Например, если вы выбрали **Процент** в поле **Принцип распределения**, введите процент в поле **Процент**.
    
    Повторите действия до тех пор, пока не будет завершено создание распределений.

14. Закройте форму.
15. Перейдите на вкладку **Настройка**.
16. В поле **Налоговая группа** выберите код налоговой группы для авансового отчета.
17. В поле **Налоговая группа номенклатур** выберите код налоговой группы номенклатур для авансового отчета.
18. Установите флажок **Цены включают налог**, чтобы указать, что сумма в авансовом отчете включает налог.
19. В области **Панель операций** щелкните **Разнести** для разноски авансового отчета. Статус авансового отчета меняется на **Разнесено**.

## <a name="generate-an-expense-account-line-using-the-copy-from-sources-function"></a>Создание строки счета расходов, используя функцию "Копировать из источника"

1. Щелкните **Расчеты с поставщиками** \> **Общее** \> **Подотчетные лица** \> **Авансовые отчеты**.
2. Нажмите клавиши CTRL+N и введите сведения в требуемые поля.
3. В нижней области щелкните вкладку **Строки**.
4. Выберите **Функции**, затем выберите **Копировать из источников**, чтобы открыть форму **Создание строк - авансовый отчет**. В форме отображается список кредитных транзакций, разнесенных подотчетным лицом.
    
    > [!NOTE]
    > <P>В верхней области в поле <STRONG>Тип источника</STRONG> отображается тип идентификации исходного документа. Источником для типа <STRONG>Фактура</STRONG> служит заказ на покупку, а для типа <STRONG>Кредиторская задолженность</STRONG> – транзакция поставщика. Поля <STRONG>Валюта суммы</STRONG>, <STRONG>Валюта</STRONG> и <STRONG>Сумма</STRONG> указывают общую сумму во вторичной валюте транзакции, код валюты и сумму в стандартной валюте компании. В нижней области в поле <STRONG>Номер</STRONG> отображается номер счета-фактуры поставщика. В поле <STRONG>Описание</STRONG> отображается имя элемента. В полях <STRONG>Валюта</STRONG>, <STRONG>Валюта суммы</STRONG> и <STRONG>Сумма налога в валюте</STRONG> отображаются соответственно валюта ваучера, сумма по строке и сумма налога.</P>

5. В верхней области установите флажок **Пометить**, чтобы подтвердить копирование документов.
6. Нажмите кнопку **OK**, чтобы перенести данные расходов в авансовый отчет.
    
    В поле **Номер документа** авансового отчета копируется номер исходного счета-фактуры, если поле **Тип строки** авансового отчета установлено в значение **Фактура**. Если же поле **Тип строки** имеет значение **Поставщик**, в него копируется номер транзакции.
    
    > [!NOTE]
    > <P>Изменять поля <STRONG>Сумма</STRONG>, <STRONG>Валюта</STRONG> и <STRONG>Счет главной книги</STRONG> невозможно.</P>

7.  Нажмите кнопку **Разнести**, чтобы разнести отчет.
8.  Нажмите клавишу CTRL+S, чтобы закрыть форму.

## <a name="generate-advance-report-lines-using-the-copy-from-expenses-function"></a>Создание строк авансового отчета с помощью функции "Копирование из расходов"

1.  Щелкните **Расчеты с поставщиками** \> **Общее** \> **Подотчетные лица** \> **Авансовые отчеты**.
2.  Нажмите клавиши CTRL+N, затем введите требуемые сведения.
3.  В нижней области щелкните вкладку **Строки**.
4.  Введите требуемую информацию.
5.  Щелкните **Функции** и выберите **Копирование из расходов**.
6.  Выберите строку расхода.
7.  Введите дату проводки в поле **Дата**.
8.  В поле **Номер документа** введите номер ваучера подтверждающего расходы. 

    > [!NOTE]
    > <P>Записи в полях <STRONG>Имя документа</STRONG>, <STRONG>Курс</STRONG> и <STRONG>Валюта</STRONG> автоматически отображаются из книги справки ставки расхода.</P>

9.  В поле **Количество дней** введите количество дней командировки.
10. В поле **В день недели** введите сумму суточных расхода.

    > [!NOTE]
    > <P>Поле <STRONG>Итоговая сумма</STRONG> отображает итоговую сумму расхода. Поля <STRONG>По норме</STRONG> и <STRONG>Сверх нормы</STRONG> отображают сведения о типе расходов.</P>

11. Щелкните **Сохранить** для перемещения записей из формы **Создать строки** в форму **Авансовый отчет**.  

    > [!NOTE]
    > <P>Запись в поле <STRONG>Счет ГК</STRONG> отображается в соответствии с настройкой разноски ставки расхода. Если фактические расходы превышают установленные ставки, создается дополнительная строка для сверхнормальной суммы, когда расходы переносятся в авансовый отчет. В этом случае флажок <STRONG>Сверх нормы</STRONG> устанавливается автоматически. Поле <STRONG>Тип строки</STRONG> авансового отчета, которое создается с помощью функции <STRONG>Копирование из расходов</STRONG>, отображает значение <STRONG>Расход</STRONG>.</P>

12. Нажмите кнопку **Разнести**, чтобы разнести отчет.
13. Нажмите клавишу CTRL+S, чтобы закрыть форму.

## <a name="generate-and-post-a-facture-for-an-advance-report"></a>Создание и разноска счет-фактуры для авансового отчета

Функция создания счета-фактуры по счету расходов может использоваться для отражения НДС в книге покупок для личные расходов подотчетных лиц, такие как стандартная часть суточных расходов.

1.  Щелкните **Расчеты с поставщиками** \> **Общее** \> **Подотчетные лица** \> **Авансовые отчеты**. Щелкните вкладку **Обновление счета-фактуры**, чтобы открыть форму **Обновление счета-фактуры**.
2.  Щелкните вкладку **Авансовый отчет**, чтобы просмотреть сведения об авансовом отчете.
3.  Выберите флажок **В фактуру**, чтобы включить в счет-фактуру авансовые отчеты.
4.  Щелкните вкладку **Строки авансового отчета**.
5.  Снимите флажок **В фактуру**, чтобы исключить строки авансового отчета, не включенные в счет-фактуру.
6.  Щелкните вкладку **Выбранные строки**, чтобы отобразить включенные строки отчета **Авансовый отчет**.
7.  В верхней области щелкните на вкладке **Обзор**.
8.  В поле **Дата регистрации** введите дату регистрации счета-фактуры.
9.  В поле **Дата фактуры** введите дату счета-фактуры.
    
    > [!NOTE]
    > <P>Необходимо снять флажок <STRONG>Та же</STRONG> для ввода значения в поле <STRONG>Дата фактуры</STRONG>.</P>

10. Щелкните **Разноска**, затем щелкните **Обновление и печать**, чтобы разнести документ.
11. Нажмите клавиши CTRL+S или закройте форму **Обновление счета-фактуры**.
12. Щелкните **Запросы**, затем щелкните **Счет-фактура по авансовому отчету** для просмотра созданных счетов-фактур.
13. Нажмите клавиши CTRL+S или закройте форму **Авансовые отчеты**.

## <a name="void-an-expense-report"></a>Аннулирование отчета по расходам

Можно аннулировать разнесенные отчеты о расходах и отчеты, для которых статус расширенного отчета — **Разнесено**. Эти отчеты не должны были быть сопоставлены. Фактуры для этих отчетов не создаются.

1.  Щелкните **Расчеты с поставщиками** \> **Общее** \> **Подотчетные лица** \> **Авансовые отчеты**.
2.  Выберите авансовый отчет для аннулирования.
3.  Щелкните **Отклонение**. Для авансового отчета устанавливается статус **Отклонено**.

    > [!NOTE]
    > <P>При отклонении авансового отчета сторнированные проводки создаются для подотчетного лица. Флажок <STRONG>Сторнировано</STRONG> автоматически выбирается в форме <STRONG>Проводки по подотч. лицам</STRONG>. Проводки сопоставляются и сторнируются автоматически.</P>

4.  Закройте форму.
5.  Щелкните **Расчеты с поставщиками** \> **Общее** \> **Подотчетные лица** \> **Подотчетные лица**. Щелкните **Проводки**, чтобы открыть форму **Проводки по подотч. лицам**.
6.  Щелкните **Ваучер** и убедитесь, что флажок **Коррекция** установлен.
    
    Можно проверить распределение расходов для реверсированной проводки. Реверсированные проводки или учетные проводки со сторно распределяются, как исходные. Однако распределения для сторнированных проводок или учетных проводок со сторно имеют отрицательный знак.
    
## <a name="budget-control-for-advance-reports"></a>Бюджетный контроль авансовых отчетов

Использование бюджетного контроля позволяет проверять наличие достаточных средств для запланированных или фактических покупок. После настройки базового бюджетирования можно настроить бюджетный контроль. Бюджетный контроль можно использовать для подтверждения доступности бюджетных средств по документам-источникам, таким как заказы на покупку, отчеты по расходам, авансовые отчеты и журналы учета. Если бюджетные средства превышают установленное ограничение бюджета, можно запретить дополнительную обработку документа.

Авансовый отчет в основном используется для учета командировочных расходов, понесенных сотрудником во время командировки. Если понесенные по авансовому отчету расходы превышают заданный бюджет, бюджетный контроль укажет на недоступность средств.

Чтобы активировать бюджетный контроль для авансовых отчетов, выполните следующие шаги.

  - В форме **Конфигурация бюджетного контроля** в области **Выберите документы-источники** выберите авансовые отчеты в качестве документа-источника для бюджетного контроля и активируйте проверки бюджета для строк авансового отчета.

  - В форме **Конфигурация бюджетного контроля** в области **Доступные бюджетные средства** задайте процедуру расчета для определения доступности бюджетных средств для авансовых отчетов.

### <a name="budget-calculation"></a>Расчет бюджета

При задании процедуры расчета бюджета для авансовых отчетов необходимо учитывать фактические расходы и неразнесенные фактические расходы.

**Пример**

Доступные бюджетные средства = (**Исходный бюджет** + **Изменения бюджета** + **Переносы бюджета**) – (**Фактические расходы** + **Неразнесенные фактические расходы**)

Фактические расходы и неразнесенные фактические расходы для каждой строки авансового отчета можно посмотреть в форме **Фактические расходы**.

### <a name="budget-checking"></a>Проверка бюджета

Проверка бюджета для авансовых отчетов позволяет проверять наличие бюджетных средств для запланированных или будущих расходов. Процесс проверки бюджета запускается автоматически при сохранении или разноске авансового отчета.
