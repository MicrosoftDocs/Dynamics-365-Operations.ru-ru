# <a name="regulatory-updates"></a>Нормативные обновления

[!include [banner](../includes/banner.md)]

В этом разделе перечислены нормативные обновления, которые запланированы для выпуска в поддерживаемых локализациях Microsoft Dynamics 365 for Finance and Operations. График поставки может измениться, и планируемые функции могут быть изменены или не выпущены. Дополнительные сведения см. в [политике корпорации Майкрософт](https://go.microsoft.com/fwlink/p/?linkid=2007332). 

Нормативные обновления отличаются от нормативных функций. Сведения о планируемых или выпущенных нормативных функциях см. в следующих источниках.

- [Замечания к выпуску Dynamics 365 и платформы Power](https://docs.microsoft.com/business-applications-release-notes/index)
- [Что нового для Microsoft Dynamics 365 for Finance and Operations](../../fin-and-ops/get-started/whats-new-changed.md)

При использовании месяца в столбце даты нормативные обновление планируется поставить в течение этого месяца. Дата поставки может быть любым днем в пределах этого месяца, а не только первый день месяца.
