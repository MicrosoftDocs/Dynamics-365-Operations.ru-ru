# <a name="create-and-post-budget-journals-for-fixed-asset-acquisitions-russia"></a><span data-ttu-id="a4342-103">Создание и разноска журналов бюджета для приобретений ОС (Россия)</span><span class="sxs-lookup"><span data-stu-id="a4342-103">Create and post budget journals for fixed asset acquisitions (Russia)</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a4342-104">В модуле **Основные средства** можно создавать финансовые планы и текущие бюджеты с помощью бюджетных моделей.</span><span class="sxs-lookup"><span data-stu-id="a4342-104">You can create financial plans and current budgets in the **Fixed assets** module by using budget models.</span></span> <span data-ttu-id="a4342-105">Модель бюджета представляет собой коллекцию планируемых оборотов для конкретных счетов и периодов.</span><span class="sxs-lookup"><span data-stu-id="a4342-105">A budget model represents a collection of planned turnovers for specific accounts and periods.</span></span>

1. <span data-ttu-id="a4342-106">Выберите **Бюджетирование** \> **Настройка** \> **Основное бюджетирование** \> **Бюджетные модели**.</span><span class="sxs-lookup"><span data-stu-id="a4342-106">Select **Budgeting** \> **Setup** \> **Basic budgeting** \> **Budget models**.</span></span>
2. <span data-ttu-id="a4342-107">Выберите **Создать**, чтобы создать бюджетную модель.</span><span class="sxs-lookup"><span data-stu-id="a4342-107">Select **New** to create a budget model.</span></span>
3. <span data-ttu-id="a4342-108">В поле **Бюджетная модель** введите код для бюджетной модели.</span><span class="sxs-lookup"><span data-stu-id="a4342-108">In the **Budget model** field, enter a code for the budget model.</span></span> <span data-ttu-id="a4342-109">В поле **Имя** введите имя для модели бюджета.</span><span class="sxs-lookup"><span data-stu-id="a4342-109">In the **Name** field, enter a name for the budget model.</span></span>
4. <span data-ttu-id="a4342-110">Закройте страницу **Бюджетные модели**.</span><span class="sxs-lookup"><span data-stu-id="a4342-110">Close the **Budget model** page.</span></span>
5. <span data-ttu-id="a4342-111">Выберите **Основные средства (Россия)** \> **Журналы** \> **Журнал бюджетов ОС**.</span><span class="sxs-lookup"><span data-stu-id="a4342-111">Select **Fixed assets (Russia)** \> **Journals** \> **FA budget journal**.</span></span>
6. <span data-ttu-id="a4342-112">Выберите вкладку **Список** и выберите **Создать**, чтобы создать журнал.</span><span class="sxs-lookup"><span data-stu-id="a4342-112">Select the **List** tab, and then select **New** to create a journal.</span></span>
7. <span data-ttu-id="a4342-113">В поле **Имя** выберите имя журнала.</span><span class="sxs-lookup"><span data-stu-id="a4342-113">In the **Name** field, select a journal name.</span></span>
8. <span data-ttu-id="a4342-114">Выберите **Строки**, чтобы открыть страницу **Ваучер журнала**, на которой можно вводить проводки по бюджету основных средств.</span><span class="sxs-lookup"><span data-stu-id="a4342-114">Select **Lines** to open the **Journal voucher** page, where you can enter fixed asset budget transactions.</span></span>
9. <span data-ttu-id="a4342-115">Выберите вкладку **Список** и выберите **Создать**, чтобы создать проводку бюджета.</span><span class="sxs-lookup"><span data-stu-id="a4342-115">Select the **List** tab, and then select **New** to create a line for a budget transaction.</span></span>
10. <span data-ttu-id="a4342-116">В поле **Модель бюджета** выберите модель бюджета.</span><span class="sxs-lookup"><span data-stu-id="a4342-116">In the **Budget model** field, select the budget model.</span></span>
11. <span data-ttu-id="a4342-117">В поле **Учет** можно изменить модель стоимости основных средств.</span><span class="sxs-lookup"><span data-stu-id="a4342-117">In the **Accounting** field, you can change the fixed asset value model.</span></span>
12. <span data-ttu-id="a4342-118">В поле **Тип проводки** выберите **Ввод в действие**.</span><span class="sxs-lookup"><span data-stu-id="a4342-118">In the **Transaction type** field, select **Putting into operation**.</span></span>
13. <span data-ttu-id="a4342-119">В поле **Дата** можно изменить дату проводки.</span><span class="sxs-lookup"><span data-stu-id="a4342-119">In the **Date** field, you can change the transaction date.</span></span>
14. <span data-ttu-id="a4342-120">В поле **Тип счета** можно изменить тип счета.</span><span class="sxs-lookup"><span data-stu-id="a4342-120">In the **Account type** field, you can change the type of account.</span></span>
15. <span data-ttu-id="a4342-121">В поле **Счет** выберите номер основного средства.</span><span class="sxs-lookup"><span data-stu-id="a4342-121">In the **Account** field, select a fixed asset number.</span></span>
16. <span data-ttu-id="a4342-122">В поле **Текст проводки** выберите текст проводки.</span><span class="sxs-lookup"><span data-stu-id="a4342-122">In the **Transaction text** field, select a transaction text.</span></span>
17. <span data-ttu-id="a4342-123">В поле **Дебет** или **Кредит** введите сумму транзакции.</span><span class="sxs-lookup"><span data-stu-id="a4342-123">In the **Debit** or **Credit** field, enter a transaction amount.</span></span>
18. <span data-ttu-id="a4342-124">В поле **Тип корр. счета** выберите тип счета ГК.</span><span class="sxs-lookup"><span data-stu-id="a4342-124">In the **Offset account type** field, select the type of ledger account.</span></span>
19. <span data-ttu-id="a4342-125">В поле **Корр. счет** выберите номер счета ГК.</span><span class="sxs-lookup"><span data-stu-id="a4342-125">In the **Offset account** field, select the ledger account number.</span></span>

    > [!NOTE]
    > <span data-ttu-id="a4342-126">Если для этой проводки установлен профиль разноски, в поле **Корр. счет** автоматически отображается номер счета.</span><span class="sxs-lookup"><span data-stu-id="a4342-126">If the posting profile is set up for this transaction, the **Offset account** field automatically shows the account number.</span></span>

20. <span data-ttu-id="a4342-127">Повторите шаги с 9 по 19 для каждого дополнительного типа проводки по основным средствам.</span><span class="sxs-lookup"><span data-stu-id="a4342-127">Repeat steps 9 through 19 for every additional type of fixed asset transaction.</span></span>

    > [!NOTE]
    > <span data-ttu-id="a4342-128">Чтобы создать проводки группового приобретения или группового расчета амортизации, выберите **Ввод в эксплуатацию** или **Амортизация**.</span><span class="sxs-lookup"><span data-stu-id="a4342-128">To create transactions for group acquisitions or group depreciation calculations, select **Putting into operation** or **Depreciation**.</span></span> <span data-ttu-id="a4342-129">При создании проводки приобретения укажите дату проводки и бюджетную модель.</span><span class="sxs-lookup"><span data-stu-id="a4342-129">When you create an acquisition transaction, specify the transaction date and budget model.</span></span> <span data-ttu-id="a4342-130">При создании проводки для расчета амортизации укажите дату проводки, бюджетную модель и модель стоимости для проводки.</span><span class="sxs-lookup"><span data-stu-id="a4342-130">When you create a transaction for a depreciation calculation, specify the transaction date, budget model, and value model for the transaction.</span></span>

21. <span data-ttu-id="a4342-131">Выберите **Проверить** \> **Проверить**, чтобы проверить в журнале сведения об основном средстве.</span><span class="sxs-lookup"><span data-stu-id="a4342-131">Select **Validate** \> **Validate** to verify the asset information in the journal.</span></span>
22. <span data-ttu-id="a4342-132">Выберите **Разнести** \> **Перенос в бюджет ОС**, чтобы перенести проводки в бюджет основных средств.</span><span class="sxs-lookup"><span data-stu-id="a4342-132">Select **Post** \> **Transfer to fixed asset budget** to transfer the transactions to the fixed asset budget.</span></span>
23. <span data-ttu-id="a4342-133">Выберите **Разнести** \> **Перенос в бюджет ОС и в бюджет ГК**, чтобы перенести проводки в бюджет основных средств и ГК.</span><span class="sxs-lookup"><span data-stu-id="a4342-133">Select **Post** \> **Transfer to fixed asset and ledger budget** to transfer the transactions to the fixed asset and general ledger budget.</span></span>
