# <a name="sales-books-purchase-books-and-invoice-factures-journals"></a>Книги продаж, книги покупок и журналы счетов-фактур

[!include [banner](../includes/banner.md)]

## <a name="generate-a-sales-book-purchase-book-and-additional-sheets"></a>Создание книги продаж, книги покупок и дополнительных листов

Книги продаж и книги покупок являются устаревшими документами, которые необходимо подготовить и хранить в течение налогового периода. Они должны быть подаваться в налоговые органы периодически и по запросу. Книги налога на добавленную стоимость (НДС) содержат счета-фактуры, которые были выданы и получены.

### <a name="set-up-parameters-for-sales-and-purchase-books"></a>Настройка параметров книг продаж и покупок

1. Перейдите в раздел **Расчеты с клиентами \> Настройка \> Параметры модуля расчетов с клиентами**.
2. На странице **Параметры модуля расчетов с клиентами** на вкладке **Номерные серии** определите номерную серию для ссылки **Книга продаж**.
3. Перейдите в раздел **Расчеты с поставщиками \> Настройка \> Параметры модуля расчетов с клиентами**.
4. На странице **Параметры модуля расчетов с клиентами** на вкладке **Номерные серии** определите номерную серию для ссылки **Книга покупок**.
5. Перейдите в раздел **Налог \> Косвенные налоги \> Налог \> Периоды сопоставления налогов**.
6. На странице **Периоды сопоставления налогов** создайте налоговый период.
7. Перейдите в раздел **Главная книга \> Настройка главной книги \> Параметры главной книги**.
8. На странице **Параметры главной книги** на вкладке **Налог** на экспресс-вкладке **Параметры налога** в поле **Разделитель дат и чисел для книг продаж/покупок** определите разделитель, используемый между датой и номером счетов-фактур в формат печати книг продаж и книг покупок.
9. На странице **Параметры модуля расчетов с клиентами** на вкладке **Главная книга и налог** на экспресс-вкладке **Книга продаж** выберите форматы электронной отчетности (ER) для книги продаж и дополнительного листа книги продаж в формате XML.
10. На странице **Параметры модуля расчетов с поставщиками** на вкладке **Главная книга и налог** на экспресс-вкладке **Книга покупок** выберите форматы электронной отчетности (ER) для книги покупок и дополнительного листа книги покупок в формате XML.

### <a name="generate-and-print-a-sales-book"></a>Создание и печать книги продаж

#### <a name="generate-a-sales-book"></a>Создание книги продаж

1. Выберите **Расчеты с клиентами \> Периодические задачи \> Книга продаж \> Журнал книг продаж**.
2. На странице **Журнал книг продаж** создайте журнал книг продаж. Книгу можно создать только в том случае, если все книги, перечисленные на странице, закрыты.
3. В поле **Имя** введите имя книги.
4. В области действий выберите **Функции \> Обновить** для создания строк в журнале книг продаж.
5. В диалоговом окне **Книга продаж** задайте для параметра **Закрыть книгу** значение **Да**, чтобы закрыть книгу после обновления. После закрытия книги нельзя создать новые счета-фактуры в закрытом периоде.

    Поля **Начальная дата** и **Конечная дата** устанавливаются автоматически в зависимости от периода сопоставления налогов.

6. Нажмите **ОК**. Строки книги продаж создаются, и дата закрытия вводится в поле **Дата закрытия** книги журнала книг продаж.

#### <a name="commands-on-the-sales-books-journal-page"></a>Команды на странице журнала книг продаж

Следующие команды доступны в области действий страницы **Журнал книг продаж**.

| Команда | описание |
|---------|-------------|
| Позиции | Откройте страницу **Строки книги продаж**, где можно проверить информацию о строках книги продаж. |
| Итоговые значения | Откройте диалоговое окно **Итоги**, где можно просмотреть общую базу НДС и суммы НДС для книги продаж. Итоговые значения отображаются отдельно для внутреннего НДС, экспортного НДС и восстановления НДС. |
| Печать \> Книга продаж | Печать отчета по книге продаж. |
| Печать \> Печать доп. листа | Печать отчета по дополнительному листу книги продаж. |

#### <a name="print-a-sales-book"></a>Печать книги продаж

1. На странице **Журнал книг продаж** выберите строку журнала книг продаж, затем выберите **Печать \> Книга продаж**.
2. В диалоговом окне **Книга продаж в Microsoft Office Excel** задайте следующие параметры:

    - **Дата начала** и **Дата окончания** — проверьте даты в этих полях. Эти поля определяют отчетный период книги продаж.
    - **Сгруппировать по фактурам** — задайте для этого параметра значение **Да** для группировки счетов-фактур с одинаковым номером счета-фактуры на одной строке, содержащей общую сумму.
    - **Исключить сторно** — задайте для этого параметра значение **Да** для исключения из отчета за указанный период исходных проводок и проводок сторно.
    - **Создать XML-файл** — задайте для этого параметра значение **Да** для создания электронного отчета в формате прежних версий, в дополнение к отчету Microsoft Excel.

3. Выберите **OK** для печати книги продаж.

#### <a name="print-a-sales-book-additional-sheet"></a>Печать дополнительного листа книги продаж

На странице **Журнал книг продаж** выберите строку журнала книг продаж, затем выберите **Печать \> Печать доп. листа**.

Дополнительные листы создаются и печатаются для всех периодов сопоставления налога, которые были исправлены в течение выбранного периода для книги продаж.

В настоящее время при создании дополнительных листов рассматриваются только корректирующие и измененные счета-фактуры, уменьшающие сумму НДС, которую во время скорректированного периода необходимо заплатить налоговым органам. Корректирующие и измененные счета-фактур, увеличивающие сумму НДС, которая должна быть выплачена, включаются в книгу продаж за текущий период.

## <a name="print-an-issued-and-received-factures-journal"></a>Печать журнала выписанных и полученных счетов-фактур

Журнал учета счетов-фактур содержит список выписанных и полученных счетов-фактур, связанных с действиями в пользу другого объекта. Примеры этих действий включают проводки и комиссионные агента.

### <a name="set-up-parameters-for-a-factures-journal"></a>Настройка параметров для журнала счетов-фактур

На странице **Параметры главной книги** на вкладке **Налог** на экспресс-вкладке **Параметры налога** определите следующие параметры:

   - **Разделитель для кодов операций со счетами-фактурами** — выберите разделитель, используемый, если одному счету-фактуре назначено несколько кодов типов операции.
   - **Сопоставление формата журнала учета счетов-фактур** — выберите формат электронной отчетности для журнала счетов-фактур в формате XML.

### <a name="print-a-facture-accounting-journal"></a>Печать журнала учета счетов-фактур

1. Выберите **Главная книга \> Запросы и отчеты \> Отчеты журналов \> Журнал учета счетов-фактур**.
2. В диалоговом окне **Журнал учета счетов-фактур** в поле **Код интервала дат** выберите отчетный период. Можно также указать собственный диапазон дат с помощью полей **Начальная дата** и **Конечная дата**.
3. В поле **Исходящие** выберите критерий включения счетов-фактур по сделкам с посредником в журнал счетов-фактур:

    - **Дата регистрации** — журнал выписанных счетов-фактур содержит все счета-фактуры по сделкам с посредником, даже если они не были подтверждены принципалом. (Сведения по сделкам с посредником для этих счетов-фактур могут быть пустыми.)
    - **Дата подтверждения** — журнал выписанных счетов-фактур содержит только подтвержденные счета-фактуры по сделкам с посредником. (Сведений по сделкам с посредником для этих счетов-фактур заполняются счетами-фактурами, которые были получены от принципала.)

4. Задайте для параметра **Дата отчетности** значение **Да** для фильтрации счетов-фактур по дате отчетности, а не по дате счета-фактуры.
5. Задайте для параметра **Создать XML-файл** значение **Да** для создания электронного отчета XML в дополнение к отчету Excel.
6. Выберите **ОК**, чтобы напечатать журналы учета счетов-фактур для выбранного периода.
