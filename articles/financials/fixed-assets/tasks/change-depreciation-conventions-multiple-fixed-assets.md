# <a name="change-depreciation-conventions-for-multiple-fixed-assets"></a>Изменение соглашений по амортизации для нескольких основных средств

[!include [task guide banner](../../includes/task-guide-banner.md)]

В результате этой задачи будет обновлено соглашение по амортизации для указанной группы основных средств. В данном руководстве по задаче используется демонстрационная компания USMF.

1. Перейдите в раздел "Основные средства" > "Периодические задачи" > "Массовое обновление"
2. В поле "Журнал амортизации" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
3. В списке перейдите по ссылке в выбранной строке.
4. В поле "Начало размещения на обслуживание" введите дату.
5. В поле "Завершение размещения на обслуживание" введите дату.
    * Будут обновлены только основные средства, являющиеся частью выбранного журнала амортизации и помещенные на сервис между этими датами.  
6. В поле "Текущее соглашение по амортизации" выберите вариант.
    * Будут обновлены только основные средства, которые имеют текущее соглашение по амортизации.  
7. В поле "Новое соглашение по амортизации" выберите вариант.
    * Убедитесь, что отчет будет распечатан в нужном месте.  
8. Разверните раздел "Записи для добавления".
9. Щелкните "Фильтр".
10. В списке выберите группу ОС.
11. В поле "Критерии" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
12. Выберите требуемую группу ОС.
13. В списке перейдите по ссылке в выбранной строке.
14. Нажмите кнопку "OК".
15. Нажмите кнопку "OК".
    *  Результаты процесса отображаются в отчете "Массовое обновление".
