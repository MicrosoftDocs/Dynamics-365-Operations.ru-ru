# <a name="calculate-fixed-asset-depreciation-across-legal-entities"></a>Расчет амортизации ОС в разных юридических лицах

[!include[task guide banner](../../includes/task-guide-banner.md)]

Используйте эту процедуру для изменения группы ОС, которой назначено основное средство. Основные средства должны быть назначены правильной группе основных средств. Группа ОС используется при создании запросов и отчетов, настройке новых основных средств и интеграции книг и разноски проводок с основными средствами на соответствующие счета книги учета.

В данной записи используется демонстрационная компания USMF.

1. Перейдите в раздел "Основные средства" > "Основные средства" > "Основные средства".
2. Выберите основное средство, для которого требуется изменить группу ОС.
3. Щелкните "Изменение группы основных средств".
4. В поле "Новая группа" введите или выберите значение.
5. Задайте для параметра "Новый инвентарный номер ОС" значение "Да", чтобы назначить инвентарный номер ОС выбранному основному средству.
    * Поле "Инв. номер ОС" становится доступным, если для параметра "Новый инвентарный номер ОС" установлено значение "Да".   Если для основных средств настроена автоматическая нумерация, в этом поле отображается следующий доступный номер ОС. Этот номер можно изменить.   Если для основных средств настроена нумерация вручную, поле остается пустым, и следует ввести новый номер ОС.  
6. Нажмите кнопку "OК".
7. Щелкните Да.
