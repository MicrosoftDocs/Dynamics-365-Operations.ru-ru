# <a name="customer-transaction-list-page"></a>Страница списка "Проводка по клиенту"

[!include [banner](../includes/banner.md)]

## <a name="view-settlements"></a>Просмотр сопоставлений

Вкладка **Просмотр сопоставлений** в области действий обеспечивает быстрый доступ к истории сопоставления и дополнительной информации о проводке сопоставления в целом. Также можно отображать дополнительные проводки, связанные с выбранной проводкой — либо потому что они входят в то же сопоставление, либо потому что они представляют собой платежи, созданные в том же журнале платежей.

1. Выберите **Расчеты с клиентами \> Клиенты**.
2. Выберите клиента, по которому есть проводки, а затем выберите **Клиент \> Проводки**.
3. Выберите проводку, которую вы хотите изучить.
4. Выберите вкладку **Просмотр сопоставлений** в области действий.

    Появится диалоговое окно **Просмотр сопоставлений** с выбранной проводкой и всеми документами, которые с ней сопоставлены. Это диалоговое окно напоминает представление истории сопоставления, однако включает в себя все связанные документы. 

5. В этом диалоговом окне можно выполнять различные задачи. Выберите один или несколько ваучеров, а затем выберите один из следующих пунктов меню:

   - **Просмотр учета** — отображаются все ваучеры, связанные с выбранными документами. Выберите **Закрыть**, чтобы закрыть диалоговое окно.
   - **Экспорт** — экспорт выбранных ваучеров в Microsoft Excel.
   - **Просмотр связанных платежей** — отображаются все проводки журнала платежей, созданные в журнале платежей, связанном с выбранным документом. Кроме того, отображаются все сопоставления, связанные с этими платежами. Метка этого пункта меню также меняется на **Просмотр сопоставлений**. Выберите **Просмотр сопоставлений**, чтобы отобразить только те проводки, которые отображались при первом открытии диалогового окна **Просмотр сопоставлений**.
    - **Сопоставление проводок** — этот пункт меню отображается, если исходный выбранный документ не был полностью сопоставлен. Если выбрать его, появится диалоговое окно **Сопоставление проводок**, где можно выбрать проводки для сопоставления.
    - **Отменить сопоставление** — этот пункт меню отображается, если исходный выбранный документ был полностью сопоставлен. Если выбрать его, появится диалоговое окно **Отменить сопоставление**, где можно отменить сопоставления, сделанные для этого документа.
