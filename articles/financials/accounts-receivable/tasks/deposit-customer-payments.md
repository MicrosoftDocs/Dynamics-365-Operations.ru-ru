# <a name="deposit-customer-payments"></a>Внесение платежей клиентов

[!include[task guide banner](../../includes/task-guide-banner.md)]

Депонируйте платежи клиента. В этой задаче используется демонстрационная компания USMF.

1. Перейдите в раздел "Расчеты с клиентами" > "Настройка платежей" > "Журнал платежей".
2. Щелкните "Создать".
3. В поле "Имя" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. Выберите журнал платежей. 
5. Выберите Строки.
6. В поле "Счет" выберите клиента, для которого записывается платеж.
7. В поле "Кредит" введите сумму платежа.
    * Можно оставить сумму незаполненной, чтобы система рассчитывала ее путем выбора оплаченных накладных.  
8. В поле "Ссылка на платеж" введите значение.
    * Ссылкой на платеж может быть номер чека для вводимого платежа. Ссылка на платеж требуется для включения платежа в бланк депозита.  
9. Установите флажок "Использовать бланк депозита".
    * Если платеж должен быть включен в депозит, измените значение этого параметра на "Да".  
10. Щелкните "Создать".
11. В поле "Счет" выберите клиента для следующего платежа.
12. В поле "Кредит" введите сумму платежа.
13. В поле "Ссылка на платеж" введите значение.
14. Установите флажок "Использовать бланк депозита".
15. Щелкните "Разнести".
    * Платежи необходимо разнести до создания бланка депозита. Это гарантирует, что платежи не изменятся после создания бланка депозита.  
16. Щелкните Функции.
17. Щелкните "Бланк депозита".
18. Нажмите кнопку "OК".
    * Первая страница используется для создания бланка депозита.  
19. Нажмите кнопку "OК".
    * Второй этап — печать бланка депозита, но этот шаг не является обязательным.
