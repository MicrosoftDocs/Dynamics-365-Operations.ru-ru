# <a name="key-invoice-data-into-the-ap-system-using-invoice-pool"></a>Ввод данных счета в модуль расчетов с поставщиками с использованием кластера накладных

[!include[task guide banner](../../includes/task-guide-banner.md)]

В этом руководстве по задаче показано, как использовать регистр накладных для создания накладных.  Затем используйте кластер накладных для сопоставления накладной с заказом на покупку и завершения оформления расходов на странице накладной поставщика.


## <a name="create-a-purchase-order"></a>Создание заказа на покупку
1. Перейдите в раздел "Расчеты с поставщиками" > "Заказы на покупку" > "Заказы на покупку".
2. Щелкните "Создать", чтобы создать заказ на покупку.
3. В поле "Счет поставщика" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. Выберите поставщика в списке. Например, выберите поставщика 1001.
5. Нажмите кнопку OK.
6. В поле "Код номенклатуры" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
7. Найдите код номенклатуры обслуживания в списке. Например, выберите S0001.
8. Щелкните код номенклатуры, чтобы выбрать его.
    * Чистая сумма равна 75,00.  Это сумма, которая ожидается в накладной.  
9. В области действий щелкните "Покупка".
10. Нажмите кнопку "Подтвердить".

## <a name="create-and-post-and-invoice"></a>Создание и разноска накладной
1. Перейдите в раздел "Расчеты с поставщиками" > "Накладные" > "Регистр накладных".
2. Щелкните "Создать".
3. Откройте поиск, чтобы выбрать имя регистра накладных для использования.
4. Выберите имя регистра накладных для использования.
5. Щелкните "Строки", чтобы открыть регистр и ввести строки расходов.
6. В окне поиска выберите поставщика. Например, выберите поставщика 1001.
7. В поле Накладная введите номер накладной.
8. В поле "Описание" введите значение.
9. В поле "Кредит" введите число.
10. В поле "Заказ на покупку" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
11. Выберите заказ на покупку, созданный ранее.
12. В поле "Кем утверждено" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
13. Выделите утверждающее лицо и нажмите кнопку "Выбрать", чтобы выбрать его.
14. Щелкните "Разнести".
15. Закройте форму.
16. Закройте форму.

## <a name="open-an-invoice-from-the-pool-and-match-it-to-a-purchase-order-to-complete-the-invoice-process"></a>Открытие накладной из кластера и сопоставление накладной с заказом на покупку для завершения процесса накладной
1. Перейдите в раздел "Расчеты с поставщиками" > "Накладные" > "Кластер накладных".
2. Щелкните "Заказ на покупку", чтобы создать накладную поставщика из накладной в кластере.
3. Выберите накладную, которую требуется просмотреть.
4. Щелкните "Обновить статус сопоставления", чтобы завершить сопоставление.
5. В области действий щелкните "Параметры".
6. Щелкните "Изменить режим просмотра".
7. Щелкните "Представление сетки".
8. Щелкните "Разнести".
9. Закройте форму.
10. Перейдите в раздел "Расчеты с поставщиками" > "Поставщики" > "Поставщики".
11. Выберите поставщика, указанного в заказе на покупку. Например, выберите поставщика 1001.
12. В списке перейдите по ссылке в выбранной строке.
13. В области действий щелкните "Поставщик".
14. Щелкните "Проводки".
15. Выберите созданную накладную.
    * Начисление регистра накладных было реверсировано и разнесено на соответствующий счет расходов.
