# <a name="take-a-cash-discount-outside-the-cash-discount-period"></a>Принимать скидку по оплате вне периода скидки по оплате

[!include[banner](../includes/banner.md)]


В этой статье приводятся два сценария, которые показывают, как можно использовать скидку по оплате, даже если платеж осуществляется вне периода скидки по оплате.

28 июня Эйприл создает накладную на 2 000,00 для поставщика 3052. По этой накладной применяется скидка на оплату 1 %, если накладная оплачена в течение 14 дней.

## <a name="use-cash-discount-option--always"></a>Параметр "Использовать скидку по оплате" = "Всегда"
Эйприл создает платеж 1 июля, то есть после даты скидки. Эйприл открывает страницу **Сопоставление проводок** для просмотра проводок, которые можно сопоставить. 

Эйприл отмечает накладную для оплаты. Скидка на оплату не действует, потому что платеж выполнен после даты скидки. Однако поставщик разрешил Эйприл все равно применить скидку по оплате. Поэтому Эйприл изменяет значение в поле **Использовать скидку по оплате** на **Всегда**.

| Пометка     | Использовать скидку по оплате | Операция   | Учетная запись | Дата скидки по оплате | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|----------|-------------------|-----------|---------|--------------------|-----------|---------|--------------------------------|----------|------------------|
| Выбрано | Всегда            | Inv-10030 | 3052    | 28.06.2015          | 12.07.2015 | 10030   | -2 000,00                      | американский доллар      | -1 980,00        |

Данные по скидкам появляются внизу страницы **Сопоставление проводок**.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 12.07.2015 |
| Сумма скидки по оплате         | -20,00    |
| Использовать скидку по оплате            | Всегда    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | -20,00    |

## <a name="date-to-use-for-calculating-discounts--selected-date"></a>Дата, используемая для расчета скидок = Выбранная дата
Если накладная и оплата были разнесены, то скидку по оплате все равно можно применить при сопоставлении проводок на странице **Сопоставление проводок**. Эйприл изменяет значение в поле **Дата, используемая для расчета скидок** на **Выбранная дата**. Затем она вводит дату, 28 июля, которая входит в период предоставления скидки по оплате для данного счета. Дата используется для расчета скидки на оплату по этой проводке. На странице **Сопоставление открытых проводок** Эйприл видит что, по умолчанию, отображается полная скидка 20,00. Строка счета показывает, что необходимо сопоставить сумму 1 980,00.

| Пометка                     | Использовать скидку по оплате | Операция   | Учетная запись | Дата скидки по оплате | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|--------------------------|-------------------|-----------|---------|--------------------|-----------|---------|--------------------------------|----------|------------------|
| Выбранный и выделенный | Обычная            | Inv-10030 | 3052    | 28.06.2015          | 12.07.2015 | 10030   | -2 000,00                      | американский доллар      | -1 980,00        |
| Установлен                 | Обычная            | APP-10030 | 3052    | 15.07.2015          | 15.07.2015 |         | 500,00                         | американский доллар      | 500,00           |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**. Сумма применяемой скидки — 20,00, потому что сумма для сопоставления по счету — это сумма по умолчанию, 1 980,00.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 12.07.2015 |
| Сумма скидки по оплате         | -20,00    |
| Использовать скидку по оплате            | Обычная    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | -20,00    |

Эйприл обновляет значение в поле **Сумма сопоставления** на **500,00**. Значение в поле **Сумма скидки по оплате** вычисляется как **5,05**.

| Пометка                     | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|--------------------------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
| Выбранный и выделенный | Обычная            | Inv-10030 | 3052    | 28.06.2015 | 12.07.2015 | 10030   | 2,000.00                       | американский доллар      | -500,00          |
| Установлен                 | Обычная            | APP-10030 | 3052    | 15.07.2015 | 15.07.2015 |         | 500,00                         | американский доллар      | 500,00           |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**. Значение в поле **Сумма скидки по оплате** — **5,05**, потому что сумма для сопоставления по счету была изменена на сумму платежа, 500,00.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 12.07.2015 |
| Сумма скидки по оплате         | -20,00    |
| Использовать скидку по оплате            | Обычная    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | -5,05     |
