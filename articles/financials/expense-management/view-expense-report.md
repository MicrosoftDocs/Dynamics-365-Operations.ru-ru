# <a name="view-an-expense-report"></a><span data-ttu-id="07778-103">Просмотр отчета о расходах</span><span class="sxs-lookup"><span data-stu-id="07778-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="07778-104">При выполнении запроса на просмотр проводок в главной книге или модуле "Закупки и источники" запрос может возвращать слишком много информации.</span><span class="sxs-lookup"><span data-stu-id="07778-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="07778-105">Поэтому может потребоваться уточнить поиск.</span><span class="sxs-lookup"><span data-stu-id="07778-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="07778-106">Чтобы получить необходимые сведения, можно перейти в проводку, чтобы просмотреть в ней отчет по расходам.</span><span class="sxs-lookup"><span data-stu-id="07778-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="07778-107">Для отчетов по расходам можно перейти от проводки к исходному отчету, который хранится в управлении расходами.</span><span class="sxs-lookup"><span data-stu-id="07778-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="07778-108">Помимо общей суммы расходов, можно просмотреть детализированные проводки, проводки, подлежащие возмещению НДС, и чеки, которые приложены к расходам.</span><span class="sxs-lookup"><span data-stu-id="07778-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="07778-109">Чтобы перейти к исходному отчету о расходах, на котором основана проводка, выберите проводку, затем выберите **Исходный документ**.</span><span class="sxs-lookup"><span data-stu-id="07778-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="07778-110">При просмотре отдельных проводок можно также просмотреть любые дополнительные данные, относящиеся к проводке.</span><span class="sxs-lookup"><span data-stu-id="07778-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="07778-111">Например, для классифицированных расходов на гостиницу можно просмотреть подробный список расходов, таких как доступ в Интернет, телефонные звонки и гостиничное обслуживание.</span><span class="sxs-lookup"><span data-stu-id="07778-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
