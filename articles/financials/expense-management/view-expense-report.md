# <a name="view-an-expense-report"></a>Просмотр отчета о расходах

[!include[banner](../includes/banner.md)]

При выполнении запроса на просмотр проводок в главной книге или модуле "Закупки и источники" запрос может возвращать слишком много информации. Поэтому может потребоваться уточнить поиск. Чтобы получить необходимые сведения, можно перейти в проводку, чтобы просмотреть в ней отчет по расходам.

Для отчетов по расходам можно перейти от проводки к исходному отчету, который хранится в управлении расходами. Помимо общей суммы расходов, можно просмотреть детализированные проводки, проводки, подлежащие возмещению НДС, и чеки, которые приложены к расходам.

Чтобы перейти к исходному отчету о расходах, на котором основана проводка, выберите проводку, затем выберите **Исходный документ**.

При просмотре отдельных проводок можно также просмотреть любые дополнительные данные, относящиеся к проводке. Например, для классифицированных расходов на гостиницу можно просмотреть подробный список расходов, таких как доступ в Интернет, телефонные звонки и гостиничное обслуживание.
