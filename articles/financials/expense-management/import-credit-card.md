# <a name="import-and-maintain-credit-card-transactions"></a>Импорт и ведение проводок по кредитным картам

[!include[banner](../includes/banner.md)]

Связанные с расходами проводки по кредитным картам можно настроить так, чтобы они автоматически импортировались по графику. Кроме того, проводки можно импортировать вручную по мере необходимости. Проводки по кредитным картам импортируются посредством информационного объекта "Проводки кредитной карты".

Более подробную информацию об информационных объектах см. в теме [Информационные объекты](../../dev-itpro/data-entities/data-entities.md).

## <a name="import-credit-card-transactions"></a>Импорт проводок по кредитной карте

1. На странице **Проводки кредитной карты** выберите **Импорт транзакций**. Если вы впервые открываете управление данными, система должна будет обновить список информационных объектов, прежде чем вы сможете продолжить.
2. В поле **Имя** введите уникальное описание задания импорта.
3. В поле **Формат исходных данных** выберите формат файла, который содержит проводки по кредитным картам для импорта.
4. Выберите **Отправить**, а затем найдите и выберите файл для импорта.
5. После отправки файла проверьте сопоставление файла проводок по кредитным картам и столбцов информационного объекта "Проводки кредитной карты", выбрав ссылку **Просмотр карты** на плитке. При наличии ошибок сопоставления или если сопоставление необходимо изменить, внесите изменения в сопоставление на вкладке **Визуализация сопоставления** или **Сведения о сопоставлении**.
6. Чтобы автоматизировать проводки по кредитным картам, выберите **Создать повторяющееся задание данных**. После этого вы можете настроить повторение, которое определяет, как часто должны импортироваться проводки по кредитным картам. Закончив, выберите **OK**.
7. Чтобы импортировать выбранный файл прямо сейчас, выберите **Импорт**.
8. При возникновении ошибок во время импорта можно просмотреть журнал выполнения или промежуточные данные, чтобы увидеть ошибки, которые необходимо исправить для того, чтобы импорт прошел успешно.

> [!NOTE]
> Если необходимо импортировать файлы нескольких форматов, необходимо создать отдельные задания импорта для каждого формата.

## <a name="reassign-the-credit-card-transactions-for-terminated-employees"></a>Переназначение проводок по кредитным картам для уволенных сотрудников

После прекращения действия записи сотрудника учетная запись сотрудника в доменных службах Active Directory (AD DS) отключается. Однако при этом могут быть активные проводки по кредитной карте, которые все равно нужно учесть как расходы и возместить. На странице **Проводки кредитной карты** можно переназначить сотрудника для любой проводки по кредитной карте, когда связанный сотрудник был уволен.

Выберите одну или несколько проводок по кредитной карте, а затем выберите **Переназначить проводки**. После этого можно выбрать другого сотрудника для назначения ему проводок по кредитной карте. После переназначения проводок по кредитной карте эти проводки можно выбрать для отчета о расходах и оплатить посредством обычного процесса возмещения отчета о расходах.
