# <a name="import-and-export-ratings-and-reviews"></a>Импорт и экспорт оценок и отзывов

[!include [banner](includes/banner.md)]

В этой теме описывается, как импортировать и экспортировать оценки и отзывы продуктов в Microsoft Dynamics 365 Commerce.

Dynamics 365 Commerce предлагает [оценки и отзывы](ratings-reviews-overview.md) как многоканальное решение. При переходе к решению оценок и отзывов в Dynamics 365 Commerce может возникнуть необходимость переместить существующие данные оценок и отзывов на платформу Commerce. Кроме того, можно также экспортировать данные оценок и отзывов из Commerce в зависимости от требований бизнеса. Соединитель Power Automate позволяет импортировать оценки и отзывы в Commerce и экспортировать их из Commerce.

> [!NOTE]
> Сведения о начале работы с логическими потоками в Power Automate см. в разделе [Создание облачного потока в Power Automate](/power-automate/get-started-logic-flow).

## <a name="prerequisites"></a>Необходимые условия

Для импорта и экспорта оценок и отзывов предварительно необходимо выполнить следующие необходимые условия:

- Для сайта электронной коммерции на платформе Commerce необходимо активировать решение оценки и отзывов. Дополнительные сведения см. в разделе [Соглашение на использование оценок и отзывов](opt-in-ratings-reviews.md).
- Соединитель приложения Power App оценок и отзывов Dynamics 365 должен быть настроен таким образом, чтобы включить действия "отправить отзывы" или "экспортировать отзывы" в Power Automate. Дополнительные сведения см. в разделе [Dynamics 365 Commerce — оценки и отзывы (предварительная версия)](/connectors/dynamics365ratingsre/).
- Проверка подлинности между службами (S2S) должна быть настроена для безопасного вызова прикладного интерфейса программирования (API) оценок и отзывов извне Commerce. Дополнительные сведения см. в разделе [Настройка проверки подлинности между службами](service-to-service-auth.md).

## <a name="import-ratings-and-reviews"></a>Импорт оценок и отзывов

Чтобы импортировать данные оценок и отзывов из существующей системы в Commerce, необходимо добавить соединитель оценок и отзывов Dynamics 365 Power Automate в существующий или новый поток Power Automate. Дополнительные сведения см. в разделе [Dynamics 365 Commerce — оценки и отзывы (предварительная версия)](/connectors/dynamics365ratingsre/).

> [!IMPORTANT]
> Перед тем как завершить эту процедуру, необходимо [настроить проверку подлинности S2S](service-to-service-auth.md).

Чтобы импортировать оценки и отзывы в Commerce с помощью соединителя оценок и отзывов Dynamics 365 Power Automate, выполните следующие действия.

1. Выберите действие **Отправить отзыв пользователя**.
1. Создайте подключение с помощью сведений о приложении Azure Active Directory (Azure AD), которые были созданы при настройке проверки подлинности S2S. Дополнительные сведения см. в разделе [Настройка проверки подлинности между службами](service-to-service-auth.md).
1. Действие **Отправить отзыв пользователя** принимает по одному отзыву за раз. Таким образом, повторите действие. Используйте исходные отзывы в качестве списка, чтобы массово отправлять отзывы.
    
## <a name="export-ratings-and-reviews"></a>Экспорт оценок и отзывов

Чтобы экспортировать данные оценок и отзывов из Commerce, необходимо добавить соединитель оценок и отзывов Dynamics 365 Power Automate в существующий или новый поток Power Automate. Дополнительные сведения см. в разделе [Dynamics 365 Commerce — оценки и отзывы (предварительная версия)](/connectors/dynamics365ratingsre/).

Чтобы экспортировать оценки и отзывы из Commerce с помощью соединителя оценок и отзывов Dynamics 365 Power Automate, выполните следующие действия.

1. Выберите действие **Экспортировать все отзывы**.
1. Завершите действие.
