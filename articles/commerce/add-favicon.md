# <a name="add-a-favicon"></a><span data-ttu-id="57936-103">Добавление значка сайта</span><span class="sxs-lookup"><span data-stu-id="57936-103">Add a favicon</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="57936-104">В этой теме объясняется, как добавить значок сайта к сайту.</span><span class="sxs-lookup"><span data-stu-id="57936-104">This topic explains how to add a favicon to your site.</span></span>

## <a name="overview"></a><span data-ttu-id="57936-105">Обзор</span><span class="sxs-lookup"><span data-stu-id="57936-105">Overview</span></span>

<span data-ttu-id="57936-106">Значок сайта — это небольшой графический файл, который отображается на вкладке веб-браузера, в адресной строке, в журнале просмотра и в закладках или избранном, а также в других местах.</span><span class="sxs-lookup"><span data-stu-id="57936-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="57936-107">Рекомендуется добавить значок сайта к сайту, так как он представляет и подкрепляет вашу торговую марку и помогает отличать веб-сайт от других сайтов, которые посещает клиент.</span><span class="sxs-lookup"><span data-stu-id="57936-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="57936-108">Хотя на сайт можно добавлять несколько значков сайта различных размеров и типов файлов, в этом разделе показано, как добавить один значок сайта.</span><span class="sxs-lookup"><span data-stu-id="57936-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="57936-109">Однако этот же процесс и место используются для добавления дополнительных значков сайта.</span><span class="sxs-lookup"><span data-stu-id="57936-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="57936-110">Отправка значка сайта в коллекцию активов вашего сайта</span><span class="sxs-lookup"><span data-stu-id="57936-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="57936-111">Чтобы отправить значок сайта в коллекцию активов вашего сайта, выполните следующие действия.</span><span class="sxs-lookup"><span data-stu-id="57936-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="57936-112">В левой области переходов выберите **Библиотека мультимедиа**.</span><span class="sxs-lookup"><span data-stu-id="57936-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="57936-113">На панели команд выберите пункт **Отправить \> Отправить элементы мультимедиа**.</span><span class="sxs-lookup"><span data-stu-id="57936-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="57936-114">В окне проводника перейдите к файлу изображения значка сайта, который необходимо загрузить, выберите его и нажмите **Открыть**.</span><span class="sxs-lookup"><span data-stu-id="57936-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>
1. <span data-ttu-id="57936-115">В диалоговом окне **Отправить элемент мультимедиа** введите требуемый заголовок и альтернативный текст.</span><span class="sxs-lookup"><span data-stu-id="57936-115">In the **Upload Media Item** dialog box, enter the required title and alt text.</span></span>
1. <span data-ttu-id="57936-116">Если требуется опубликовать изображение сразу после отправки, установите флажок **Опубликовать элементы мультимедиа после отправки**.</span><span class="sxs-lookup"><span data-stu-id="57936-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="57936-117">Если не установлен флажок **Опубликовать элементы мультимедиа после отправки**, необходимо вернуться на страницу **Элементы мультимедиа** и вручную опубликовать значок сайта позже.</span><span class="sxs-lookup"><span data-stu-id="57936-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="57936-118">Нажмите **ОК**.</span><span class="sxs-lookup"><span data-stu-id="57936-118">Select **OK**.</span></span>
1. <span data-ttu-id="57936-119">На панели свойств в правой части скопируйте общедоступный URL-адрес значка сайта.</span><span class="sxs-lookup"><span data-stu-id="57936-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="57936-120">Этот URL-адрес будет использоваться позже.</span><span class="sxs-lookup"><span data-stu-id="57936-120">You will use this URL later.</span></span>

## <a name="create-a-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="57936-124">Создание фрагмента, содержащего метатег для значка сайта</span><span class="sxs-lookup"><span data-stu-id="57936-124">Create a fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="57936-125">Чтобы создать фрагмент, содержащий метатег для значка сайта, выполните следующие действия.</span><span class="sxs-lookup"><span data-stu-id="57936-125">To create a fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="57936-126">Перейдите к разделу **Фрагменты**, и выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="57936-126">Go to **Fragments**, and select **New**.</span></span>
1. <span data-ttu-id="57936-127">В диалоговом окне **Создание фрагмента** выберите **Метатеги** в качестве модуля, на котором основан фрагмент.</span><span class="sxs-lookup"><span data-stu-id="57936-127">In the **New fragment** dialog box, select **Metatags** as the module that the fragment is based on.</span></span>
1. <span data-ttu-id="57936-128">Введите имя для фрагмента, затем выберите **ОК**.</span><span class="sxs-lookup"><span data-stu-id="57936-128">Enter a name for the fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="57936-129">В дереве иерархии фрагментов выберите дочерний элемент **Метатеги по умолчанию**.</span><span class="sxs-lookup"><span data-stu-id="57936-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="57936-130">В правой области в разделе **Метатеги** выберите **Добавить**, а затем введите HTML-строку, созданную ранее для значка сайта.</span><span class="sxs-lookup"><span data-stu-id="57936-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span> 
1. <span data-ttu-id="57936-131">Выберите **Завершить редактирование**, а затем нажмите кнопку **Опубликовать**, чтобы опубликовать фрагмент.</span><span class="sxs-lookup"><span data-stu-id="57936-131">Select **Finish editing**, and then select **Publish** to publish the fragment.</span></span>

## <a name="add-the-metatag-fragment-to-the-html-head-section-of-your-pages"></a><span data-ttu-id="57936-132">Добавьте фрагмент метатега в раздел HTML head страниц</span><span class="sxs-lookup"><span data-stu-id="57936-132">Add the metatag fragment to the HTML head section of your pages</span></span>

<span data-ttu-id="57936-133">Чтобы добавить фрагмент метатега в раздел HTML **head** страниц, выполните следующие действия.</span><span class="sxs-lookup"><span data-stu-id="57936-133">To add the metatag fragment to the HTML **head** section of your pages, follow these steps.</span></span>

1. <span data-ttu-id="57936-134">Перейдите к разделу **Шаблоны** и откройте шаблон для страниц, к которым необходимо добавить значок сайта, а затем выберите **Изменить**.</span><span class="sxs-lookup"><span data-stu-id="57936-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="57936-135">В дереве иерархии шаблонов нажмите кнопку с многоточием (**...**) справа от контейнера **HTML head**, а затем выберите **Добавить фрагмент**.</span><span class="sxs-lookup"><span data-stu-id="57936-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add fragment**.</span></span>
1. <span data-ttu-id="57936-136">В диалоговом окне **Выбор фрагмента** выберите созданный ранее фрагмент метатега и нажмите **ОК**.</span><span class="sxs-lookup"><span data-stu-id="57936-136">In the **Select fragment** dialog box, select the metatag fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="57936-137">Выберите **Завершить редактирование**, а затем нажмите кнопку **Опубликовать**, чтобы опубликовать шаблон.</span><span class="sxs-lookup"><span data-stu-id="57936-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="57936-138">Если сайт использует более одного шаблона, необходимо добавить фрагмент метатега во все эти шаблоны.</span><span class="sxs-lookup"><span data-stu-id="57936-138">If your site uses more than one template, you must add the metatags fragment to all of them.</span></span>

<span data-ttu-id="57936-139">При предварительном просмотре страниц, которые основаны на шаблоне, к которому был добавлен фрагмент метатега, вы должны будете увидеть значок сайта на вкладке браузера.</span><span class="sxs-lookup"><span data-stu-id="57936-139">When you preview pages that are based on the template that you added the metatags fragment to, you should now see the favicon on the browser tab.</span></span>
