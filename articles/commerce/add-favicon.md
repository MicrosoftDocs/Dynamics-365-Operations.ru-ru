# <a name="add-a-favicon"></a>Добавление значка сайта

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

В этой теме объясняется, как добавить значок сайта к сайту.

## <a name="overview"></a>Обзор

Значок сайта — это небольшой графический файл, который отображается на вкладке веб-браузера, в адресной строке, в журнале просмотра и в закладках или избранном, а также в других местах. Рекомендуется добавить значок сайта к сайту, так как он представляет и подкрепляет вашу торговую марку и помогает отличать веб-сайт от других сайтов, которые посещает клиент.

Хотя на сайт можно добавлять несколько значков сайта различных размеров и типов файлов, в этом разделе показано, как добавить один значок сайта. Однако этот же процесс и место используются для добавления дополнительных значков сайта.

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a>Отправка значка сайта в коллекцию активов вашего сайта

Чтобы отправить значок сайта в коллекцию активов вашего сайта, выполните следующие действия.

1. Перейти в раздел **Активы \> Отправить \> Отправить активы**.
1. Найдите и выберите значок сайта в локальной файловой системе.
1. Введите заголовок, затем выберите **ОК**. 
1. На панели свойств в правой части скопируйте общедоступный URL-адрес значка сайта.

> [!NOTE]
> Если не выбран параметр **Опубликовать активы после отправки**, необходимо вернуться в страницу **Активы** и вручную опубликовать значок сайта позже.

## <a name="create-the-html-for-the-favicon"></a>Создание кода HTML для значка сайта

Чтобы создать код HTML для значка сайта, используйте следующий фрагмент HTML. Для атрибута **href** замените **"Public\_URL\_for\_your\_favicon"** на общедоступный URL-адрес, который был скопирован ранее.

`<link rel="shortcut icon" href="Public_URL_for_your_favicon">`

## <a name="add-the-html-for-the-favicon-to-the-head-element-of-your-pages"></a>Добавление HTML-кода для значка сайта в элемент \<головной\> ваших страниц

Чтобы добавить значок сайта на сайт, используйте ту же процедуру, которая используется для добавления любого типа HTML или сценариев в элемент **\<head\>** страниц сайта.
