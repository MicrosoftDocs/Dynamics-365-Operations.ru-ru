# <a name="add-a-favicon"></a><span data-ttu-id="69375-103">Добавление значка сайта</span><span class="sxs-lookup"><span data-stu-id="69375-103">Add a favicon</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="69375-104">В этой теме объясняется, как добавить значок сайта к сайту.</span><span class="sxs-lookup"><span data-stu-id="69375-104">This topic explains how to add a favicon to your site.</span></span>

## <a name="overview"></a><span data-ttu-id="69375-105">Обзор</span><span class="sxs-lookup"><span data-stu-id="69375-105">Overview</span></span>

<span data-ttu-id="69375-106">Значок сайта — это небольшой графический файл, который отображается на вкладке веб-браузера, в адресной строке, в журнале просмотра и в закладках или избранном, а также в других местах.</span><span class="sxs-lookup"><span data-stu-id="69375-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="69375-107">Рекомендуется добавить значок сайта к сайту, так как он представляет и подкрепляет вашу торговую марку и помогает отличать веб-сайт от других сайтов, которые посещает клиент.</span><span class="sxs-lookup"><span data-stu-id="69375-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="69375-108">Хотя на сайт можно добавлять несколько значков сайта различных размеров и типов файлов, в этом разделе показано, как добавить один значок сайта.</span><span class="sxs-lookup"><span data-stu-id="69375-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="69375-109">Однако этот же процесс и место используются для добавления дополнительных значков сайта.</span><span class="sxs-lookup"><span data-stu-id="69375-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="69375-110">Отправка значка сайта в коллекцию активов вашего сайта</span><span class="sxs-lookup"><span data-stu-id="69375-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="69375-111">Чтобы отправить значок сайта в коллекцию активов вашего сайта, выполните следующие действия.</span><span class="sxs-lookup"><span data-stu-id="69375-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="69375-112">В левой области переходов выберите **Библиотека мультимедиа**.</span><span class="sxs-lookup"><span data-stu-id="69375-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="69375-113">На панели команд выберите пункт **Отправить \> Отправить элементы мультимедиа**.</span><span class="sxs-lookup"><span data-stu-id="69375-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="69375-114">В окне проводника перейдите к файлу изображения значка сайта, который необходимо загрузить, выберите его и нажмите **Открыть**.</span><span class="sxs-lookup"><span data-stu-id="69375-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>
1. <span data-ttu-id="69375-115">В диалоговом окне **Отправить элемент мультимедиа** введите требуемый заголовок и альтернативный текст.</span><span class="sxs-lookup"><span data-stu-id="69375-115">In the **Upload Media Item** dialog box, enter the required title and alt text.</span></span>
1. <span data-ttu-id="69375-116">Если требуется опубликовать изображение сразу после отправки, установите флажок **Опубликовать элементы мультимедиа после отправки**.</span><span class="sxs-lookup"><span data-stu-id="69375-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="69375-117">Если не установлен флажок **Опубликовать элементы мультимедиа после отправки**, необходимо вернуться на страницу **Элементы мультимедиа** и вручную опубликовать значок сайта позже.</span><span class="sxs-lookup"><span data-stu-id="69375-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="69375-118">Нажмите **ОК**.</span><span class="sxs-lookup"><span data-stu-id="69375-118">Select **OK**.</span></span>
1. <span data-ttu-id="69375-119">На панели свойств в правой части скопируйте общедоступный URL-адрес значка сайта.</span><span class="sxs-lookup"><span data-stu-id="69375-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="69375-120">Этот URL-адрес будет использоваться позже.</span><span class="sxs-lookup"><span data-stu-id="69375-120">You will use this URL later.</span></span>

<span data-ttu-id="69375-122">Чтобы создать код HTML для значка сайта, используйте следующая строка HTML.</span><span class="sxs-lookup"><span data-stu-id="69375-122">To create the HTML for the favicon, use the following HTML string.</span></span> <span data-ttu-id="69375-123">Для атрибута **href** замените **Public\_URL\_for\_your\_favicon** на общедоступный URL-адрес, который был скопирован ранее.</span><span class="sxs-lookup"><span data-stu-id="69375-123">For the **href** attribute, replace **Public\_URL\_for\_your\_favicon** with the public URL that you copied earlier.</span></span>

## <a name="create-a-page-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="69375-124">Создание фрагмента страницы, содержащего метатег для значка сайта</span><span class="sxs-lookup"><span data-stu-id="69375-124">Create a page fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="69375-125">Чтобы создать фрагмент страницы, содержащий метатег для значка сайта, выполните следующие действия.</span><span class="sxs-lookup"><span data-stu-id="69375-125">To create a page fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="69375-126">Перейдите к разделу **Фрагменты**, и выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="69375-126">Go to **Fragments**, and select **New**.</span></span>
1. <span data-ttu-id="69375-127">В диалоговом окне **Создание фрагмента страницы** выберите **Метатеги** в качестве модуля, на котором основан фрагмент страницы.</span><span class="sxs-lookup"><span data-stu-id="69375-127">In the **New page fragment** dialog box, select **Metatags** as the module that the page fragment is based on.</span></span>
1. <span data-ttu-id="69375-128">Введите имя для фрагмента страницы, затем выберите **ОК**.</span><span class="sxs-lookup"><span data-stu-id="69375-128">Enter a name for the page fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="69375-129">В дереве иерархии фрагментов выберите дочерний элемент **Метатеги по умолчанию**.</span><span class="sxs-lookup"><span data-stu-id="69375-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="69375-130">В правой области в разделе **Метатеги** выберите **Добавить**, а затем введите HTML-строку, созданную ранее для значка сайта.</span><span class="sxs-lookup"><span data-stu-id="69375-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span> 
1. <span data-ttu-id="69375-131">Выберите **Завершить редактирование**, а затем нажмите кнопку **Опубликовать**, чтобы опубликовать фрагмент страницы.</span><span class="sxs-lookup"><span data-stu-id="69375-131">Select **Finish editing**, and then select **Publish** to publish the page fragment.</span></span>

## <a name="add-the-metatag-page-fragment-to-the-html-head-section-of-your-pages"></a><span data-ttu-id="69375-132">Добавьте фрагмент страницы метатега в раздел HTML head страниц</span><span class="sxs-lookup"><span data-stu-id="69375-132">Add the metatag page fragment to the HTML head section of your pages</span></span>

<span data-ttu-id="69375-133">Чтобы добавить фрагмент страницы метатега в раздел HTML **head** страниц, выполните следующие действия.</span><span class="sxs-lookup"><span data-stu-id="69375-133">To add the metatag page fragment to the HTML **head** section of your pages, follow these steps.</span></span>

1. <span data-ttu-id="69375-134">Перейдите к разделу **Шаблоны** и откройте шаблон для страниц, к которым необходимо добавить значок сайта, а затем выберите **Изменить**.</span><span class="sxs-lookup"><span data-stu-id="69375-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="69375-135">В дереве иерархии шаблонов нажмите кнопку с многоточием (**...**) справа от контейнера **HTML head**, а затем выберите **Добавить фрагмент страницы**.</span><span class="sxs-lookup"><span data-stu-id="69375-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add page fragment**.</span></span>
1. <span data-ttu-id="69375-136">В диалоговом окне **Выбор фрагмента страницы** выберите созданный ранее фрагмент страницы метатега и нажмите **ОК**.</span><span class="sxs-lookup"><span data-stu-id="69375-136">In the **Select page fragment** dialog box, select the metatag page fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="69375-137">Выберите **Завершить редактирование**, а затем нажмите кнопку **Опубликовать**, чтобы опубликовать шаблон.</span><span class="sxs-lookup"><span data-stu-id="69375-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="69375-138">Если сайт использует более одного шаблона, необходимо добавить фрагмент страницы метатега во все эти шаблоны.</span><span class="sxs-lookup"><span data-stu-id="69375-138">If your site uses more than one template, you must add the metatags page fragment to all of them.</span></span>

<span data-ttu-id="69375-139">При предварительном просмотре страниц, которые основаны на шаблоне, к которому был добавлен фрагмент страницы метатега, вы должны будете увидеть значок сайта на вкладке браузера.</span><span class="sxs-lookup"><span data-stu-id="69375-139">When you preview pages that are based on the template that you added the metatags page fragment to, you should now see the favicon on the browser tab.</span></span>
