# <a name="enable-shop-similar-description-recommendations"></a>Включение рекомендаций "купить похожие по описанию"

[!include [banner](includes/banner.md)]

В этом разделе описывается, как включить рекомендации по продуктам "купить похожие по описанию" в Microsoft Dynamics 365 Commerce.

Функция рекомендаций "купить похожие по описанию" в Dynamics 365 Commerce использует возможности искусственного интеллекта и машинного обучения (ИИ-ML) для предоставления пользователям рекомендаций по продуктам, описания которых похожи на то, что ищет клиент. Благодаря созданию рекомендаций "купить похожие по описанию" для всех каналов розничной Commerce предприятия розничной торговли могут помочь клиентам легко находить то, что им нужно.

Функция рекомендаций "купить похожие по описанию" использует название и описание начальных продуктов, чтобы найти и рекомендовать схожие продукты в каталоге продуктов предприятия розничной торговли.

Рекомендации "купить похожие по описанию" доступны как в POS-приложениях, так и в электронной коммерции.

## <a name="example-scenarios"></a>Пример сценариев

В следующих примерах сценариев показаны типы рекомендаций, которые может предоставить функция "купить похожие по описанию":

- Клиент просматривает пару очков в роговой оправе в стиле ретро и получает набор рекомендаций для других очков, имеющих подобный дизайн, в контексте отрасли розничного продавца.
- Клиент использует рекомендации "купить похожие по описанию" для обнаружения разновидностей кофе, аналогичных разновидностям, которые ранее были приобретены у продавца.

## <a name="set-up-shop-similar-description-recommendations"></a>Настройка рекомендаций "купить похожие по описанию"

Рекомендации продуктов поддерживаются только для пользователей Commerce, перенесших свои хранилища в Azure Data Lake Storage Gen2.

### <a name="prerequisites"></a>Необходимые условия

Перед тем как можно будет отобразить рекомендации "купить похожие по описанию" для клиентов, необходимо выполнить следующие необходимые условия:

- [Включение рекомендаций по продукту](enable-product-recommendations.md) в Commerce headquarters.
- Убедитесь, что сервер мультимедиа поддерживает вызовы HTTPS.

### <a name="turn-on-the-shop-similar-description-recommendations-feature"></a>Включение функции рекомендаций "купить похожие по описанию"

Чтобы включить функцию рекомендаций "купить похожие по описанию" в Commerce headquarters, выполните следующие действия.

1. В рабочей области **Управление функциями** в списке доступных функций найдите и выберите **Купить похожие по описанию**.
1. В правой области выберите **Включить**.

> [!NOTE]
> Когда функция включена, система начинает создавать списки рекомендаций продуктов. Чтобы эти списки стали доступны и видны и Интернете и POS-терминалах, может потребоваться до одного дня.
>
> Если выключить эту функцию, это не повлияет на другие типы рекомендаций по продуктам. Дополнительные сведения о рекомендациях продуктов см. в разделе [Обзор рекомендаций продуктов](product-recommendations.md).

## <a name="add-a-shop-similar-description-button-to-product-details-pages"></a>Добавление кнопки "купить похожие по описанию" на страницы сведений о продуктах

После включения функции рекомендаций "купить похожие по описанию" в Commerce headquarters вы можете добавить кнопку **купить похожие по описанию** в поле покупки на любой странице сведений о продукте (PDP). Клиент, который выбрал эту кнопку, переводится на специальную страницу **Купить похожие по описанию**, которая показывает визуально похожие продукты. Клиент затем может использовать селекторы для дальнейшей фильтрации продуктов.

Чтобы добавить кнопку **Купить похожие по описанию** к страницам сведений о продукте с помощью конфигуратора сайта Commerce, выполните следующие действия.

1. Откройте существующую страницу построителя сайтов, которая содержит модуль поля покупки.
1. В области переходов слева выберите модуль поля покупки.
1. В правой области перехода установите флажок **Включить ссылку "купить похожие по описанию"**.
1. Нажмите **Сохранить**.
1. Выберите **Завершить редактирование** для возврата страницы, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее. После публикации страницы сведения о продукте включают кнопку **Купить похожие по описанию**.

На следующем рисунке показан флажок **Включить ссылку "Купить похожие по описанию"** и кнопка **Купить похожие по описанию** на примере страницы PDP в конструкторе сайтов.
