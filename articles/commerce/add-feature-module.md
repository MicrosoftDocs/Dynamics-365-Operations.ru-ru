# <a name="feature-module"></a>Модуль компонента 

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

В этом разделе описываются модули компонентов, а также описывается, как добавлять их к страницам сайта в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Модуль компонента используется для рекламы продуктов или рекламных акций с помощью комбинации изображений и текста. Например, розничный продавец может добавить модуль компонента к странице сведений о продукте, чтобы выделить особенности продукта.

Модуль компонента управляется данными из системы управления содержимым (CMS). Он является самостоятельным модулем, не зависящим от других модулей на странице. Модуль компонента может быть размещен на любой странице сайта, где розничная сеть хочет выполнить маркетинг или рекламировать что-либо (например, продукты, продажи или особенности).

## <a name="examples-of-feature-modules-in-e-commerce"></a>Примеры модулей компонентов в электронной коммерции

Модуль компонента может использоваться на следующих страницах:

- Страница сведения о продукте для демонстрации особенностей продукта
- Домашняя страница, чтобы рекламировать продукты
- Страница категории, чтобы выделить категорию продуктов

## <a name="feature-module-properties"></a>Свойства модуля компонента

| Имя свойства     | Значения | Описание |
|-------------------|--------|-------------|
| Изображение             | Файл изображения | Изображение может использоваться для продвижения продукта или акции. Можно отправить изображение в коллекцию изображений или использовать существующее изображение. |
| Заголовок           | Текст заголовка и метка заголовка (**H1**, **H2**, **H3**, **H4**, **H5** или **H6**) | Каждый модуль компонента может иметь заголовок. По умолчанию для заголовка используется тег заголовков **H2**. Однако тег можно изменить для соответствия требованиям к специальным возможностям. |
| Абзац         | Текст абзаца | Модули компонентов поддерживают текст абзаца в формате RTF. Поддерживаются некоторые основные возможности форматированного текста, такие как жирный шрифт, подчеркнутый текст, курсив и гиперссылки. Некоторые из этих возможностей могут быть переопределены темой страницы, применяемой к модулю. |
| Ссылка              | Текст ссылки, URL-адрес ссылки, метка доступных полнофункциональных интернет-приложений (ARIA) и **Открыть ссылку в новой вкладке** | Модули компонентов поддерживают одну или несколько ссылок "призыва к действию". При добавлении ссылки требуются текст ссылки, URL-адрес и метка ARIA. Метки ARIA должны быть описательными для соответствия требованиям специальных возможностей. Ссылки можно настроить таким образом, чтобы они открывались на новой вкладке. |
| Размещение изображения   | **Справа**, **Слева**, **Сверху** или **Снизу** | Это свойство определяет положение изображения относительно текста. Например, если выбрано значение **Справа**, изображение отображается справа от текста. |
| Размер столбца изображения | Значение от **1** до **12** | Это свойство определяет размер изображения относительно текста. Размер выражается как число столбцов на странице. На странице имеется 12 столбцов. По умолчанию изображение и текст имеют одинаковый размер (по шесть столбцов каждый). Однако при необходимости размер можно скорректировать. |

## <a name="add-a-feature-module-to-a-new-page"></a>Добавление модуля компонента на новую страницу 

Чтобы добавить модуль компонента на новую страницу и задать необходимые свойства, выполните следующие действия.

1. Перейдите к пункту **Шаблоны** и создайте шаблон страницы с именем **Шаблон компонента**.
1. В слоте **Главный** страницы по умолчанию добавьте модуль компонента.
1. Верните шаблон и опубликуйте его.
1. Используйте только что созданный шаблон компонента для создания страницы с именем **страница компонента**.
1. В слоте **Главный** страницы по умолчанию выберите кнопку с многоточием (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавление модуля** в списке **Выберите модули** выберите модуль компонента, затем выберите **ОК**.
1. В структуре дерева слева выберите модуль компонента.
1. В области свойств справа выберите **Добавить изображение**. Затем выберите существующее изображение или отправьте новое изображение.
1. Выберите **Заголовок**.
1. В диалоговом окне **Заголовок** добавьте текст заголовка, выберите уровень заголовка, затем выберите **ОК**.
1. В разделе **Форматированный текст** добавьте нужный текст.
1. Выберите **Добавить ссылку действия**.
1. В диалоговом окне **Ссылка действия** добавьте текст ссылки, URL-адрес ссылки и метку ARIA для ссылки, затем выберите **ОК**.
1. Сохраните страницу и просмотрите изменения.
1. Верните страницу и опубликуйте его.
