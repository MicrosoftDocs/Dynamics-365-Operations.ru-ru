# <a name="product-recommendations-overview"></a>Обзор рекомендаций по продуктам

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Commerce может использоваться для демонстрации рекомендаций по продуктам на веб-сайте электронной коммерции и на POS-терминале. Рекомендации продуктов — это номенклатуры, которые могут заинтересовать клиента. Рекомендации основаны на тенденциях покупки других клиентов в интернет-магазинах и обычных магазинах.

Рекомендации продуктов позволяют пользователям легко и быстро находить нужные им продукты, в то время как они имеют опыт, который хорошо им помогает. Перекрестные продажи и дополнительные продажи могут быть даже использованы, чтобы помочь клиентам найти другие продукты, которые они изначально не планировали покупать. При использовании рекомендаций для улучшения обнаружения продуктов они могут создавать дополнительные возможности конверсии, повышать доходы от продаж и даже улучшать степень удовлетворенности и удержания клиентов.

В электронной коммерции рекомендации по продуктам производятся на основе технологий машинного обучения рекомендациям корпорации Microsoft в больших масштабах.

## <a name="recommendation-service"></a>Служба рекомендаций

Служба рекомендаций продуктов использует технологии искусственного интеллекта и машинного обучения (AI-ML) следующим образом:

- Данные в формате, требуемом службой рекомендаций, извлекаются из операционной базы данных Commerce и отправляются в Azure Data Lake Storage или хранилище объектов.
- Служба рекомендаций использует сохраненные данные для обучения моделей рекомендаций для списков **Людям также нравятся**, **Часто покупаются вместе**, **Новое**, **Лидеры продаж** и **Популярное**.

## <a name="scenarios"></a>Сценарии

Рекомендации по продукции доступны в следующих сценариях:

- **На любой странице магазина для обзора или целевой странице электронной коммерции**: если клиенты или продавцы магазина посетят страницу магазина, подсистема рекомендаций может предлагать продукты в списках **Новые**, **Лидеры продаж** и **Популярные**.
- **На странице сведений о продукте**: если клиенты или продавцы магазина посещают страницу **Сведения о продукте**, подсистема рекомендаций предлагает дополнительные номенклатуры, которые также могут быть приобретены с высокой вероятностью. Эти номенклатуры отображаются в списке **Людям также нравится**.
- **На странице проводки или на странице оформления заказа:** механизм рекомендаций предлагает номенклатуры, основываясь на полном списке номенклатур в корзине. Эти номенклатуры отображаются в списке **Вместе с этими товарами часто покупают**.
- **Персонализированные рекомендации**: продавцы могут предоставлять пользователям, вошедшим в систему, персонализированный список **Для вас**, а также новые функции, которые позволяют персонализировать существующие сценарии со списками на основе этого клиента. Дополнительные сведения см. в разделе [Включение персонализированных рекомендаций](personalized-recommendations.md).

### <a name="types-of-product-recommendations"></a>Типы рекомендаций продуктов

В следующей таблице описываются различные типы автоматизированных рекомендаций продуктов, которые могут быть реализованы в решении Dynamics 365 Commerce в розничной торговли с помощью [модуля коллекции продуктов](product-collection-module-overview.md). Розничные торговцы могут также отображать персонализированные результаты для пользователя, выполнившего вход, если автор сайта выбирает этот вариант.

| Модуль семейства продуктов  | Вид | описание |
|----------------------------|------|-------------|
| Сoздать                        | Алгоритмический | Этот модуль отображает список новейших продуктов, которые были недавно добавлены в каналы и каталоги. |
| Лидеры продаж               | Алгоритмический | В этом модуле отображается список продуктов, которые ранжированы по наибольшему количеству продаж. |
| Популярные                   | Алгоритмический | Этот модуль отображает список самых эффективных продуктов для данного периода, отсортированных по наибольшему количеству продаж.  |
| Часто покупают вместе | Искусственный интеллект – машинное обучение | Этот модуль рекомендует список продуктов, которые обычно приобретают вместе с содержимым текущей корзины клиентов. |
| Людям также нравится           | Искусственный интеллект – машинное обучение | Этот модуль рекомендует продукты для данного начального продукта на основе шаблонов покупок клиента. |
| Для вас              | Искусственный интеллект – машинное обучение | Этот модуль рекомендует персонализированный список продуктов, основанный на шаблонах покупок вошедшего пользователя. Для гостевого пользователя этот список будет свернут. |
