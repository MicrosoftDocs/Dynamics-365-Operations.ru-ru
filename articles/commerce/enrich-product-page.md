# <a name="enrich-a-product-page"></a>Расширение возможностей страницы продукта

[!include [banner](includes/banner.md)]

В этом разделе описывается, как расширить возможности страницы продукта в Microsoft Dynamics 365 Commerce.

По умолчанию сайт использует универсальную страницу для отображения данных о продукте. На этой странице содержатся основные сведения о продукте и об элементах управления, которые необходимы для продажи продукта. Однако можно дополнить сведения, поступающие из Commerce Scale Unit, с дополнительными изображениями или текстом для конкретного продукта. Этот процесс называется обогащением страницы продукта.

Во многих случаях потребуется использовать специфическое дополнительное содержимое для ваших продуктов. При переходе в модуль **Retail и Commerce** в инструменте разработки будет отображен список продуктов из канала, назначенного сайту. В этом списке столбец **Обогащенная** указывает, является ли страница продукта для продукта обогащенной. Если в столбце имеется галочка, для продукта существует обогащенная страница продукта. Если не отображается галочка, для продукта используются страница продукта и содержимое по умолчанию. Можно просмотреть как обогащенные, так и необогащенные страницы продукта, выбрав название продукта в списке.

## <a name="enrich-a-product-page"></a>Расширение возможностей страницы продукта

Чтобы обогатить страницу продукта, выполните следующие действия.

1. В области **Сайты** выберите **Fabrikam** (или имя вашего сайта).
1. В области переходов слева выберите **Продукты**.
1. Выберите любой продукт, у которого нет обогащенной страницы продукта.
1. В области панели операций выберите **Обогатить страницу продукта**.
1. Выберите **Шаблон PDP**, затем выберите **ОК**.
1. В структуре дерева страниц слева разверните слот **Главный**.
1. Нажмите кнопку с многоточием (**...**) для слота **Главный**, затем выберите **Добавить модуль**.
1. Выберите **Контейнер 2**, затем выберите **ОК**.
1. Нажмите кнопку с многоточием для модуля **Контейнер 2**, затем выберите **Добавить модуль**.
1. Выберите **Компонент**, затем выберите **ОК**.
1. В области свойств справа в поле **Форматированный текст** введите обновленное описание продукта.
1. В поле **Заголовок** введите текст заголовка, затем выберите **ОК**.
1. Выберите **Сохранить**, затем выберите **Завершить правку**.
1. В поле **Комментарии** введите **Обогащен продукт**, затем выберите **ОК**.
1. Выберите **Предварительный просмотр**, чтобы просмотреть обогащенную страницу продукта. По завершении закройте вкладку предварительного просмотра, чтобы вернуться к инструменту разработки.
1. Выберите **Опубликовать**.
