# <a name="video-player-module"></a>Модуль видеопроигрывателя

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

В этом разделе описываются модули видеопроигрывателя, а также описывается, как добавлять их к страницам сайта в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Для поддержки воспроизведения видео используются модули видеопроигрывателя. В базовом наборе магазина имеется два модуля видеопроигрывателя: модуль внутреннего видеопроигрывателя и модуль видеопроигрывателя. Оба проигрывателя поддерживают тип файла .MP4.

## <a name="ambient-video-player-module"></a>Модуль внутреннего видеопроигрывателя

Модуль внутреннего видеопроигрывателя поддерживает краткие информационные видеоматериалы. Он должен использоваться для видеороликов длительностью менее пяти секунд. Этот проигрыватель поддерживает ограниченные возможности воспроизведения видео, например воспроизведение и пауза.

### <a name="examples-of-ambient-video-player-modules-in-e-commerce"></a>Примеры модулей внутреннего видеопроигрывателя в электронной коммерции

- Краткий информационный видеоролик, который выделяет новые продукты на домашней странице
- Краткие информационные видеоматериалы, выделяющие характеристики продуктов на странице сведений о продукте

### <a name="ambient-video-player-module-properties"></a>Свойства модуля внутреннего видеопроигрывателя

| Имя свойства     | Стоимость                 | Описание |
|-------------------|-----------------------|-------------|
| Автозапуск          | **True** или **False** | Когда установлено значение **True**, видео автоматически воспроизводится. |
| Выключить звук              | **True** или **False** | Когда установлено значение **True**, звук видео автоматически отключается. Для этого проигрывателя значение по умолчанию — **True**. В браузере Google Chrome звук автоматически воспроизводимого видео по умолчанию отключен, и воспроизведение звука выполняется только в том случае, если видеозапись воспроизводится вручную пользователем. |
| Цикл              | **True** или **False** | Когда установлено значение **True**, видео повторяется циклически. |
| Источник             |  Имя и путь видеофайла | Видеофайл, воспроизводимый видеопроигрывателем. |
| Элементы управления воспроизведением | **True** или **False** | Когда установлено значение **True**, на видео отображается кнопка воспроизведения/паузы. Если установлено значение **False**, кнопка воспроизведения/паузы не отображается, но пользователи все равно могут приостанавливать и возобновлять воспроизведение с помощью клавиатуры. |

## <a name="video-player-module"></a>Модуль видеопроигрывателя

Модуль видеопроигрывателя может использоваться для демонстрации видеороликов на сайте электронной коммерции. Он поддерживает все возможности воспроизведения, такие как воспроизведение, приостановка, режим полного размера и скрытые субтитры. Модуль видеопроигрывателя также поддерживает настройку скрытых субтитров для соответствия стандартам специальных возможностей Microsoft. Например, можно настроить размер шрифта и цвет фона.

Модуль видеопроигрывателя также поддерживает второстепенные звуковые дорожки. При отправке видео также может быть загружена второстепенная аудиодорожка. Модуль видеопроигрывателя может воспроизводить второстепенную звуковую дорожку, если пользователь ее выбирает.

### <a name="examples-of-video-player-modules-in-e-commerce"></a>Примеры модулей видеопроигрывателя в электронной коммерции

- Обучающие видеоматериалы на страницах с подробными сведениями о продуктах или на маркетинговых страницах
- Рекламные видеоматериалы или видеозаписи о политиках на любой странице маркетинга
- Маркетинговые видеоматериалы, отражающие характеристики продукции на страницах с подробными сведениями о продуктах или на маркетинговых страницах

### <a name="video-player-module-properties"></a>Свойства модуля видеопроигрывателя

| Имя свойства         | Стоимость                               | Описание |
|-----------------------|-------------------------------------|-------------|
| Автозапуск             | **True** или **False**               | Когда установлено значение **True**, видео автоматически воспроизводится. |
| Выключить звук                  | **True** или **False**               | Когда установлено значение **True**, звук видео автоматически отключается. Для этого проигрывателя значение по умолчанию — **False**. В браузере Chrome звук автоматически воспроизводимого видео по умолчанию отключен, и воспроизведение звука выполняется только в том случае, если видеозапись воспроизводится вручную пользователем. |
| Цикл                  | **True** или **False**               | Когда установлено значение **True**, видео повторяется циклически. |
| Источник                 | Имя и путь видеофайла | Видеофайл, воспроизводимый видеопроигрывателем. |
| Элементы управления воспроизведением     | **True** или **False**               | Когда установлено значение **True**, на видео отображается кнопка воспроизведения/паузы. Если установлено значение **False**, кнопка воспроизведения/паузы не отображается, но пользователи все равно могут приостанавливать и возобновлять воспроизведение с помощью клавиатуры. |
| Воспроизведение в полноэкранном режиме       | **True** или **False**               | Когда установлено значение **True**, видео воспроизводится в полноэкранном режиме. |
| Элементы управления              | **True** или **False**               | Когда установлено значение **True**, отображаются все элементы управления. К этим элементам управления относятся кнопки воспроизведения и паузы, индикатор хода выполнения и скрытые субтитры. |
| Скрыть рамку афиши     | **True** или **False**               | Видео может иметь рамку афиши. Когда для этого свойства установлено значение **True**, рамка афиши скрывается. |
| Уровень максирования            | Число от **0** до **100** | Маска, применяемая к видео для стилизации. |
| Стиль значка полноэкранного режима | **Квадрат** или **Стрелка**             | Стиль значка полноэкранного изображения, который отображается в видеопроигрывателе. |

## <a name="add-a-video-player-module-to-a-page"></a>Добавление модуля видеопроигрывателя на страницу

Чтобы добавить модуль видеопроигрывателя на новую страницу и задать необходимые свойства, выполните следующие действия.

1. Создайте шаблон страницы с именем **шаблон видеопроигрывателя**.
1. В слоте **Главный** страницы по умолчанию добавьте модуль-контейнер.
1. В модуле-контейнере добавьте модули видеопроигрывателя и внутреннего видеопроигрывателя.
1. Верните шаблон и опубликуйте его.
1. Используйте только что созданный шаблон видеопроигрывателя для создания страницы с именем **страница видеопроигрывателя**.
1. В слоте **Главный** новой страницы добавьте модуль внутреннего видеопроигрывателя.
1. В настройках модуля внутреннего видеопроигрывателя перейдите к пункту **Мультимедиа**и отправьте видеофайл. При необходимости можно изменить свойства **Автозапуск**, **Цикл** и другие свойства.
1. Сохраните и просмотрите страницу.
1. В слоте **Главный** новой страницы добавьте модуль видеопроигрывателя.
1. В настройках модуля видеопроигрывателя перейдите к пункту **Мультимедиа**и отправьте видеофайл.
1. Сохраните и просмотрите страницу. На странице должны отобразиться оба модуля видео. Можно изменить дополнительные настройки, чтобы настроить поведение каждого модуля.
1. Верните страницу и опубликуйте его.
