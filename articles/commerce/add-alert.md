# <a name="alert-module"></a>Модуль оповещения

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

В этом разделе описываются модули оповещений, а также описывается, как добавлять их к страницам сайта в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Модуль оповещения используется для отображения на странице встроенных информационных сообщений. Модули оповещения поддерживают текстовое сообщение и ссылку. Они могут использоваться для отображения рекламных акций на уровне всего сайта, которые появляются на всех страницах сайта электронной коммерции. 

Модули оповещения управляются данными из системы управления контентом (CMS) и их можно разместить на любой странице.

## <a name="examples-of-alert-modules-in-e-commerce"></a>Примеры модулей оповещений в электронной коммерции

Модули оповещения могут использоваться в заголовке сайта, чтобы указать рекламные акции или сообщения на уровне всего сайта. Далее приводятся некоторые примеры.

"Ежегодная распродажа заканчивается через 10 дней"

"Большая экономия на распродаже в связи с началом учебного года. Покупайте сейчас."

## <a name="alert-module-properties"></a>Свойства модуля оповещения

| Имя свойства  | Стоимость                              | Описание |
|----------------|------------------------------------|-------------|
| Текст           | Текст                               | Текстовое сообщение, отображаемое в модуле оповещения. |
| Выравнивание текста | **По правому краю**, **По левому краю** или **По центру** | Значение, определяющее выравнивание текста в модуле оповещения. |
| Закрыть оповещение  | **True** или **False**              | Если установлено значение **True**, клиент может закрыть оповещение. |
| Ссылка           | URL-адрес                                | URL-адрес для необязательной ссылки. |

## <a name="add-an-alert-module-to-a-page"></a>Добавление модуля оповещения на страницу 

Чтобы добавить модуль оповещения на страницу и задать необходимые свойства, выполните следующие действия.

1. Создайте шаблон страницы с именем **шаблон оповещения**.
1. В слоте **Главный** страницы по умолчанию добавьте модуль оповещения.
1. Верните шаблон и опубликуйте его. 
1. Используйте только что созданный шаблон оповещения для создания страницы с именем **страница оповещения**. 
1. В слоте **Главный** новой страницы добавьте модуль оповещения.
1. В настройках модуля оповещения введите текст оповещения. Если требуется дополнительно настроить модуль оповещений, можно изменить другие свойства.
1. Сохраните и просмотрите страницу. В верхней части страницы должно отобразиться оповещение с добавленным текстом.
1. Верните страницу и опубликуйте его.
