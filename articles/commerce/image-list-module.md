# <a name="image-list-module"></a>Модуль списка изображений

[!include [banner](includes/banner.md)]

В этой теме описываются модули списка изображений, а также описывается, как добавлять их к страницам сайта в Microsoft Dynamics 365 Commerce.

Модуль списка изображений может использоваться для простого добавления коллекции (массива) изображений на страницы сайта. Каждое изображение в массиве может быть настроено с использованием текста абзаца и ссылок на URL-адреса. Модуль списка изображений лучше всего использовать для отображения логотипов торгового знака или списка, содержащего логотипы.

> [!IMPORTANT]
> - Модуль списка изображений доступен в библиотеке модуля Commerce в версии Dynamics 365 Commerce 10.0.20.
> - Модуль списка изображений демонстрируется в теме Adventure Works.

## <a name="image-list-module-properties"></a>Свойства модуля списка изображений

| Имя свойства | Значения | описание |
|---------------|--------|-------------|
| Заголовок       | Текст заголовка и метка заголовка (**H1**, **H2**, **H3**, **H4**, **H5** или **H6**) | Заголовок текста для модуля списка изображений. |
| Список изображений    | Изображения, текст и URL-адреса | Каждый элемент в массиве представляет собой изображение, которое сопровождается текстом абзаца и URL-адресом. |

## <a name="add-an-image-list-module-to-a-new-page"></a>Добавление модуля списка изображений на новую страницу

Чтобы добавить модуль списка изображений на новую страницу и задать необходимые свойства в конструкторе сайтов Commerce, выполните следующие действия.

1. Перейдите к **Шаблоны** и откройте шаблон маркетинга для домашней страницы сайта (или создайте новый маркетинговый шаблон).
1. В слоте **Главный** страницы по умолчанию выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Список изображений**, затем выберите **ОК**.
1. Выберите **Сохранить**, выберите **Завершить редактирование** для возврата шаблона, затем нажмите кнопку **Опубликовать**, чтобы опубликовать его.
1. Перейдите к **Страницы** и откройте домашнюю страницу сайта (или создайте новую домашнюю страницу с помощью маркетингового шаблона).
1. В слоте **Главный** страницы по умолчанию выберите кнопку с многоточием (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите **Список изображений**, затем выберите **ОК**.
1. В области свойств модуля списка изображений добавьте заголовок (например, **Наши торговые марки**).
1. Добавьте элемент списка изображений и укажите изображение, некоторый текст абзаца и URL-адреса перенаправления.
1. Добавьте и настройте необходимые модули списка изображений по мере необходимости.
1. Выберите **Сохранить**, затем выберите **Предварительный просмотр**, чтобы просмотреть страницу.
1. Выберите **Завершить редактирование** для возврата шаблона, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.
