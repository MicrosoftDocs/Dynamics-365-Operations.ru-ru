# <a name="task-management-overview"></a><span data-ttu-id="af2b7-103">Обзор управления задачами</span><span class="sxs-lookup"><span data-stu-id="af2b7-103">Task management overview</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="af2b7-104">В этом разделе представлен обзор управления задачами для руководителей и работников в Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="af2b7-104">This topic provides an overview of task management for managers and workers in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="af2b7-105">В розничной среде всегда сложно обеспечить выполнение задач требуемым лицом в нужное время.</span><span class="sxs-lookup"><span data-stu-id="af2b7-105">In a retail environment, it's always difficult to make sure that tasks are performed by the right person at the right time.</span></span> <span data-ttu-id="af2b7-106">Продавцы должны иметь возможность уведомлять работников о предстоящих задачах и обеспечивать связанный контекст, чтобы задачи могли выполняться правильно и вовремя.</span><span class="sxs-lookup"><span data-stu-id="af2b7-106">Retailers must be able to notify workers about upcoming tasks and provide related business context, so that the tasks can be completed correctly and on time.</span></span>

<span data-ttu-id="af2b7-107">Управление задачами — это функция повышения производительности в Dynamics 365 Commerce, которая позволяет руководителям и работникам создавать списки задач, управлять критериями назначений, отслеживать статус задач и интегрировать эти операции между бэк-офисом Commerce и приложениями POS.</span><span class="sxs-lookup"><span data-stu-id="af2b7-107">Task management is a productivity feature in Dynamics 365 Commerce that lets managers and workers create task lists, manage assignment criteria, track task status, and integrate these operations between Commerce back office and point of sale (POS) applications.</span></span>

<span data-ttu-id="af2b7-108">Пользователи Headquarters могут использовать управление задачами для создания списков задач для розничных магазинов и для отслеживания статуса по магазинам или работникам.</span><span class="sxs-lookup"><span data-stu-id="af2b7-108">Headquarters personas can use task management to create task lists for retail stores, and to track status by store or worker.</span></span> <span data-ttu-id="af2b7-109">Они также могут создавать повторяющиеся задачи (например, "Контрольный список для закрытия на ночь в четверг").</span><span class="sxs-lookup"><span data-stu-id="af2b7-109">They can also create recurrent tasks (for example, "Thursday night closing checklist").</span></span>

<span data-ttu-id="af2b7-110">Руководители магазинов могут использовать управление задачами, чтобы назначать задачи отдельным работникам, отправлять уведомления о предстоящих задачах или о просроченных заданиях, обновлять статус задачи и создавать одноцелевые задачи в приложении POS.</span><span class="sxs-lookup"><span data-stu-id="af2b7-110">Store managers can use task management to assign tasks to individual workers, send notifications about upcoming tasks or tasks that are past due, update task status, and create single-purpose tasks in the POS application.</span></span> <span data-ttu-id="af2b7-111">Работники могут затем просматривать уведомления, просматривать сведения о задачах и обновлять статус задачи в POS.</span><span class="sxs-lookup"><span data-stu-id="af2b7-111">Workers can then see notifications, view task details, and update task status at the POS.</span></span>

<span data-ttu-id="af2b7-112">На следующем рисунке показана концептуальная архитектура управления задачами в Commerce.</span><span class="sxs-lookup"><span data-stu-id="af2b7-112">The following illustration shows the conceptual architecture of task management in Commerce.</span></span>
