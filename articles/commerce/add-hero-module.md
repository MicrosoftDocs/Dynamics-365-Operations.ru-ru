# <a name="hero-module"></a>Модуль главного имиджевого баннера

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

В этом разделе описываются модули главного имиджевого баннера, а также описывается, как добавлять их к страницам сайта в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Модуль главного имиджевого баннера используется для рекламы продуктов или рекламных акций с помощью комбинации изображений и текста. Например, розничный продавец может добавить модуль главного имиджевого баннера на домашнюю страницу сайта электронной коммерции для продвижения нового продукта и привлечения внимания клиентов.

Модуль главного имиджевого баннера управляется данными из системы управления содержимым (CMS). Он является самостоятельным модулем, не зависящим от других модулей на странице. Модуль главного имиджевого баннера может быть размещен на любой странице сайта, где розничная сеть хочет выполнить маркетинг или рекламировать что-либо (например, продукты, продажи или особенности).

## <a name="examples-of-hero-module-in-e-commerce"></a>Примеры модуля главного имиджевого баннера в электронной коммерции

- Модуль главного имиджевого баннера может использоваться на домашней странице сайта электронной коммерции для выделения рекламных акций и новых продуктов.
- Модуль главного имиджевого баннера может использоваться на странице сведений о продукте для демонстрации информации о продукте.
- Для выделения нескольких продуктов или рекламных акций в модуле карусели могут быть помещены несколько модулей главного имиджевого баннера.

## <a name="hero-module-properties"></a>Свойства модуля главного имиджевого баннера

| Имя свойства  | Значения | Описание |
|----------------|--------|-------------|
| Изображение          | Файл изображения | Изображение может использоваться для демонстрации продукта или рекламной акции. Можно отправить изображение в коллекцию изображений или использовать существующее изображение. |
| Заголовок        | Текст заголовка и метка заголовка (**H1**, **H2**, **H3**, **H4**, **H5** или **H6**) | Каждый модуль главного имиджевого баннера может иметь заголовок. По умолчанию для заголовка используется тег заголовков **H2**. Однако тег можно изменить для соответствия требованиям к специальным возможностям. |
| Абзац      | Текст абзаца | Модули главного имиджевого баннера поддерживают текст абзаца в формате RTF. Поддерживаются некоторые основные возможности форматированного текста, такие как жирный шрифт, подчеркнутый текст, курсив и гиперссылки. Некоторые из этих возможностей могут быть переопределены темой страницы, применяемой к модулю. |
| Ссылка           | Текст ссылки, URL-адрес ссылки, метка доступных полнофункциональных интернет-приложений (ARIA) и **Открыть ссылку в новой вкладке** | Модули главного имиджевого баннера поддерживают одну или несколько ссылок "призыва к действию". При добавлении ссылки требуются текст ссылки, URL-адрес и метка ARIA. Метки ARIA должны быть описательными для соответствия требованиям специальных возможностей. Ссылки можно настроить таким образом, чтобы они открывались на новой вкладке. |
| Размещение текста | **Левая сторона верхней части**, **Правая сторона верхней части**, **Верхний центр**, **Левая сторона нижней части**, **Правая сторона нижней части**, **Нижний центр**, **Левая сторона центральной части**, **Правая сторона центральной части** или **Центр центральной части** | Это свойство определяет положение изображения относительно текста. Например, если выбрано значение **Справа**, изображение отображается справа от текста. |
| Тема текста     | **Светлая** или **Темная** | Цветовую схему можно определить для текста на основе фонового изображения. Например, если изображение имеет темный фон, можно применить световую тему, чтобы текст был лучше виден, а также для целей обеспечения соответствия требованиям к коэффициентам контрастности цветов для специальных возможностей. |
| Градиент       | **True** или **False** | Градиент может быть применен к изображению для соответствия коэффициентов контрастности цветов для целей специальных возможностей. |

## <a name="add-a-hero-module-to-a-new-page"></a>Добавление модуля главного имиджевого баннера на новую страницу

Чтобы добавить модуль главного имиджевого баннера на новую страницу и задать необходимые свойства, выполните следующие действия.

1. Перейдите к пункту **Шаблоны** и создайте шаблон страницы с именем **Шаблон главного имиджевого баннера**.
1. В слоте **Главный** страницы по умолчанию добавьте модуль главного имиджевого баннера.
1. Верните шаблон и опубликуйте его.
1. Используйте только что созданный шаблон главного имиджевого баннера для создания страницы с именем **страница главного имиджевого баннера**.
1. В слоте **Главный** страницы по умолчанию выберите кнопку с многоточием (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавление модуля** в списке **Выберите модули** выберите модуль главного имиджевого баннера, затем выберите **ОК**.
1. В структуре дерева слева выберите модуль главного имиджевого баннера.
1. В области свойств справа выберите **Добавить изображение**. Затем выберите существующее изображение или отправьте новое изображение.
1. Выберите **Заголовок**.
1. В диалоговом окне **Заголовок** добавьте текст заголовка, выберите уровень заголовка, затем выберите **ОК**.
1. В разделе **Форматированный текст** добавьте нужный текст.
1. Выберите **Добавить ссылку действия**.
1. В диалоговом окне **Ссылка действия** добавьте текст ссылки, URL-адрес ссылки и метку ARIA для ссылки, затем выберите **ОК**.
1. Сохраните страницу и просмотрите изменения.
1. Верните страницу и опубликуйте его.
