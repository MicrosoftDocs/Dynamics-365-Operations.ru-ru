# <a name="define-channel-attributes"></a>Определение атрибутов канала

[!include [banner](../includes/banner.md)]

Обогатите канал, добавив атрибуты канала и продуктов. Необходимо добавить канал в организационную иерархию по умолчанию, прежде чем можно будет задать атрибуты.


## <a name="define-channel-product-attribute"></a>Определите атрибут продукта канала.
1. Перейдите в раздел "Retail и Commerce" > "Настройка канала" > "Категории каналов и атрибуты продуктов".
2. В дереве выберите "Contoso Retail\Fashion\Houston".
3. Щелкните "Изменить".
4. В поле "Иерархия категорий" введите или выберите значение.
    * Навигационная иерархия будет использоваться для перемещения продукта внутри ваших каналов.  
5. Разверните раздел "Группа атрибутов".
6. Нажмите кнопку Добавить.
    * Группа атрибутов будет отображаться в сведениях о продукте, а также поможет при использовании фильтра продуктов.  
7. В поле "Имя" введите или выберите значение.
8. Щелкните "Просмотр атрибутов".
9. Разверните раздел "Атрибуты".
10. Нажмите кнопку Добавить.
11. В списке пометьте выбранную строку.
12. Щелкните AddBtn.
13. Нажмите кнопку "OК".
14. Нажмите кнопку "Сохранить".
15. Закройте страницу.
16. Разверните раздел "Продукты".
17. Нажмите кнопку Добавить.
18. В списке пометьте выбранную строку.
19. Нажмите кнопку Добавить.
    * Аналогично можно добавить несколько продуктов.  
20. Нажмите кнопку "OК".
21. Щелкните "Атрибуты".
22. Разверните раздел "Канал СМИ".
23. Щелкните "Изменить".
24. В поле "Канал" выберите вариант.
25. Установите флажок "По умолчанию".
26. Нажмите кнопку OK.
27. Разверните раздел "Атрибуты продуктов канала".
28. В поле "Марка" введите значение.
    * Аналогично можно настроить другие атрибуты.  
29. Нажмите кнопку "Сохранить".
30. Закройте страницу.
31. Нажмите кнопку "Сохранить".
32. Щелкните "Опубликовать обновления канала".
    * После публикация данные будут готовы к синхронизации с таблицами канала. Если вы не выполнили публикацию, измененные данные не будут синхронизованы с таблицами канала.  
33. Нажмите кнопку "OК".
34. Нажмите кнопку "Сохранить".
