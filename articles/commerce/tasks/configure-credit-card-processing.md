# <a name="configure-credit-card-processing"></a>Настройка обработки кредитных карт

[!include [banner](../includes/banner.md)]

Эта процедура содержит инструкции по просмотру списка поставщиков платежей и настройке счета платежа для расчетов с клиентами. Эта процедура использует компанию USRT с демонстрационными данными и предназначена для администраторов и ИТ-специалистов.


## <a name="view-a-list-of-payment-providers"></a>Просмотр списка поставщиков платежей
1. Перейдите в раздел "Расчеты с клиентами" > "Настройка платежей" > "Службы платежей".
2. Щелкните "Просмотр доступных поставщиков".

## <a name="configure-payment-account"></a>Настройка счета платежа
1. Щелкните "Создать".
2. В поле "Служба платежей" введите значение.
3. В поле "Соединитель платежей" выберите вариант.
4. Переключите развертывание раздела "Учетная запись службы платежей".
5. В поле "Среда:" введите "PROD".
6. Щелкните "Типы кредитных карт".
7. В поле "Журнал платежей" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
8. В списке перейдите по ссылке в выбранной строке.
9. Нажмите кнопку Добавить.
10. В поле "Валюта" введите значение.
11. В списке найдите и выберите требуемую запись.
12. В поле "Журнал платежей" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
13. В списке перейдите по ссылке в выбранной строке.
14. Нажмите кнопку Добавить.
15. В поле "Валюта" введите значение.
16. В списке найдите и выберите требуемую запись.
    * Можно повторить эти шаги для произвольного количества типов карт.  
17. В поле "Журнал платежей" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
18. В списке перейдите по ссылке в выбранной строке.
19. Нажмите кнопку Добавить.
20. В поле "Валюта" введите значение.
21. Нажмите кнопку "Сохранить".
22. Закройте страницу.
23. Щелкните "Проверить".
24. Установите флажок "Процессор по умолчанию для новых кредитных карт".
25. Нажмите кнопку "Сохранить".
