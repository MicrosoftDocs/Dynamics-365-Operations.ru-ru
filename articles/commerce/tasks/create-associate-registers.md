# <a name="create-and-associate-registers"></a>Создание и связывание регистраторов

[!include [banner](../includes/banner.md)]

В этой процедуре показано, как создать регистр POS. В этой процедуре используется компания с демонстрационными данными USRT.

1. Перейдите в раздел "Retail и Commerce" > "Настройка канала" > "Настройка POS" > "Кассы".
2. Нажмите Создать.
3. В поле "Номер регистра" введите код нового регистра.
    * Код регистра обычно включает коды, которые помогают сопоставить регистр с магазином, к которому он принадлежит, и местонахождением внутри магазина.  
4. В поле "Имя" введите описательное имя регистра.
5. В поле "Номер магазина" введите или выберите значение.
6. В поле "Профиль оборудования" введите или выберите значение.
    * Профили оборудования используются для определения периферийных устройств, которые будут подключены к регистру, например кассовый лоток и принтер чеков.  
7. В поле "Визуальный профиль" введите или выберите значение.
    * Визуальные профили используются для определения изображений, используемых для фона и страницы входа POS, а также тем POS.  
8. В поле "Номер POS для электронных платежей" введите значение.
    * Номер регистра POS для электронных платежей используется для уведомления процессора платежа, которому терминал платежей отправляет запросы на авторизацию. Это значение часто называется "Код терминала" или "TID". TID обычно указывается на наклейке на платежном терминале.  
9. Нажмите кнопку "Сохранить".
