# <a name="create-call-center-orders"></a>Создание заказов центра обработки вызовов

[!include [banner](../includes/banner.md)]

Эта процедура демонстрирует поиск клиента, создание нового заказа, поиск продукта и получение платежа от клиента. Эта процедура использует демонстрационные данные компании USRT и предназначена для сотрудника, занимающегося заказами на продажу. Предварительные условия: пользователь, выполняющий эту процедуру, настроен как пользователь центра обработки вызовов, и полугодовой каталог Fabrikam опубликован и содержит по крайней мере один исходный код.

1. Перейдите в раздел **Retail и Commerce \> Клиенты \> Обслуживание клиентов**.
2. Для **SearchText** введите условия поиска для поиска клиента.
    * Для этой демонстрационной процедуры введите "Karen" и выберите **TAB**.  
3. Выберите Поиск.
    * Поскольку в демонстрационных данных есть только один клиент с именем Karen, результат будет выбран автоматически.  
4. Выберите **Создать заказ на продажу**.
5. Разверните или сверните раздел заголовка **Заказ на продажу**.
6. Выберите код источника для каталога
    * Если нет активных кодов источника, можно пропустить этот шаг.  
7. Выберите **Добавить строку**.
8. Для **Код номенклатуры** введите поисковый запрос для номенклатуры.
    * Для этой демонстрационной процедуры введите частичный код номенклатуры "8111" и нажмите TAB. Это действие открывает окно поиска номенклатуры.  
9. Выберите продукта для добавления в заказ на продажу.
10. Введите количество продаваемого продукта.
11. Выберите **Создать**.
12. Выберите **Завершить**, чтобы зафиксировать платеж клиента.
13. Выберите **Добавить**.
    * Команда "Добавить ссылку" находится на вкладке "Платежи". Разверните вкладку "Платежи", если она свернута.  
14. Выберите способ оплаты.
    * Для этой процедуры выберите способ оплаты наличными.  
15. Закройте страницу.
16. Введите сумму.
    * Для этой процедуры введите сумму, равную сальдо заказа, которую можно видеть на странице сводке заказ на продажу слева от поля суммы. Это действие позволит вам завершить заказ как полностью оплаченный.  
17. Нажмите **ОК**.
18. Выберите **Отправить**.
