# <a name="set-up-rules-and-parameters-for-cross-docking-and-buyers-push"></a>Настройка правил и параметров для централизованного распределения и кросс-докинга

[!include[task guide banner](../includes/task-guide-banner.md)]

Эта процедура демонстрирует шаги по созданию правил пополнения. Правила пополнения можно использовать для управления тем, как продукты распределяются по магазинам при использовании кросс-докинга и централизованного распределения. Правила пополнения можно настраивать для магазинов или групп магазинов. Вес, указанный для каждой строки в правиле, будет определять, как количества продуктов будут распределяться между магазинами при использовании правил пополнения в качестве метода распределения для кросс-докинга или централизованного распределения. В данной процедуре используется демонстрационная компания USRT.

1. Перейдите в раздел "Правила пополнения".
2. Щелкните "Создать".
3. В поле "Правило пополнения" введите значение.
4. В поле "Описание" введите значение.
5. Нажмите кнопку "Сохранить".
6. Нажмите кнопку Добавить.
7. В списке пометьте выбранную строку.
    * Можно выбрать в качестве типа "Иерархия пополнения" или "Канал". Значение определяет, будет в поле "Имя" выбрана иерархия каналов или конкретный канал.  В этом примере оставьте значение "Иерархия пополнения".  
8. В поле "Имя" выберите значение.
    * Значение веса по умолчанию подставляется из веса, определенного на складе.  Этот вес можно использовать для правила пополнения или можно ввести в поле "Вес" новый вес.  
9. В поле "Вес" введите число.
10. Нажмите кнопку Добавить.
11. В списке пометьте выбранную строку.
12. В поле "Тип" выберите "Канал".
13. В поле "Имя" введите или выберите значение.
14. В поле "Вес" введите число.
15. Нажмите кнопку "Сохранить".
