# <a name="configure-a-worker"></a>Настройка работника

[!include [banner](../includes/banner.md)]

Эта процедура демонстрирует порядок настройки работника в качестве торгового представителя, который имеет право на комиссионные по продажам в POS. В этой процедуре используется компания с демонстрационными данными USRT.


## <a name="create-a-commission-sales-group-for-the-worker"></a>Создание группы комиссий по продажам для работника
1. Перейдите в раздел "Продажи и маркетинг" > "Комиссии" > "Группы продаж".
    * Работники и могут быть назначены в одну или несколько групп продаж. В POS можно выбрать любую группу продаж, содержащую работников из адресной книги магазина.  
2. Щелкните "Создать".
3. В поле "Группа" введите значение.
4. В поле "Имя" введите значение.
5. Нажмите кнопку "Сохранить".
6. В области действий щелкните "Общие".
7. Щелкните "Торг. представитель".
    * Группа продаж может содержать несколько работников. Комиссии можно разделить между работниками на основе того, как определена доля в комиссии.  
8. В поле "Имя" введите или выберите значение.
9. В поле "Распределение комиссии" введите число.
10. Нажмите кнопку "Сохранить".
11. Закройте страницу.
12. Закройте страницу.

## <a name="assign-the-workers-default-sales-group"></a>Назначение группы продаж по умолчанию для работников
1. Перейдите в раздел "Retail и Commerce" > "Сотрудники" > "Работники".
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.
4. Щелкните вкладку Commerce.
    * Работник может быть назначен группе продаж по умолчанию. Группа продаж по умолчанию будет автоматически добавлена в строки продаж в POS, если этот параметр включен в профиле функции для магазина.  
5. Щелкните "Изменить".
6. В поле "Группа по умолчанию" введите или выберите значение.
7. Нажмите кнопку "Сохранить".
