# <a name="add-fields-to-an-excel-workbook-to-edit-retail-transactions"></a>Добавление полей в книгу Excel для редактирования проводок розничной торговли

[!include [banner](../includes/banner.md)]

В этой статье описывается, как добавить поля в книгу Microsoft Excel, чтобы можно было редактировать проводки розничной торговли в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

При создании файла Excel для редактирования проводок розничной торговли файл заполняется данными из некоторых полей по умолчанию. Если поле, которое должно быть обновлено, не отображается по умолчанию в созданном файле Excel, его можно добавить.

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a>Добавление полей в лист в книге Excel

Чтобы добавить поля в книгу Excel для редактирования проводок розничной торговли, выполните следующие действия.

1. Загрузите и откройте файл Excel на странице **Журналы операций**, странице **Проводки розничной торговли** или плитке **Ошибки проверки проводок** в рабочей области **Финансовая информация магазина**.
1. Выберите **Разработка**.
1. Выберите символ карандаша для нужной таблицы, затем в списке доступных полей найдите и выберите поле, которое требуется добавить.
1. Выберите **Добавить** и нажмите **Обновить**. Можно изменить порядок полей.
1. По завершении обновления выберите **Обновить**, чтобы получить данные для нового столбца.

Теперь новое поле и данные для него должны быть доступны для редактирования в Excel.
