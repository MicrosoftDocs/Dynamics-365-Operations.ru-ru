<span data-ttu-id="56942-106">При создании файла Excel для редактирования проводок розничной торговли файл заполняется данными из некоторых полей по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="56942-106">When you generate an Excel file so that you can edit retail transactions, the file is filled with some default fields.</span></span> <span data-ttu-id="56942-107">Если поле, которое должно быть обновлено, не отображается по умолчанию в созданном файле Excel, его можно добавить.</span><span class="sxs-lookup"><span data-stu-id="56942-107">If a field that must be updated isn't visible by default in the generated Excel file, you can add it.</span></span>

<span data-ttu-id="56942-109">Чтобы добавить поля в книгу Excel для редактирования проводок розничной торговли, выполните следующие действия.</span><span class="sxs-lookup"><span data-stu-id="56942-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="56942-110">Загрузите и откройте файл Excel на странице **Журналы операций**, странице **Проводки розничной торговли** или плитке **Ошибки проверки проводок** в рабочей области **Финансовая информация магазина**.</span><span class="sxs-lookup"><span data-stu-id="56942-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="56942-111">Выберите **Разработка**.</span><span class="sxs-lookup"><span data-stu-id="56942-111">Select **Design**.</span></span>
1. <span data-ttu-id="56942-112">Выберите символ карандаша для нужной таблицы, затем в списке доступных полей найдите и выберите поле, которое требуется добавить.</span><span class="sxs-lookup"><span data-stu-id="56942-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
