<span data-ttu-id="8215d-114">Для отображения скидок система выполняет поиск всех скидок, соответствующих одному или нескольким из следующих условий:</span><span class="sxs-lookup"><span data-stu-id="8215d-114">To show discounts, the system looks for all the discounts that match one or more of the following conditions:</span></span>

- <span data-ttu-id="8215d-117">Ценовая группа скидки сопоставляется с каталогом, связанным с магазином.</span><span class="sxs-lookup"><span data-stu-id="8215d-117">The price group of the discount is mapped to a catalog that is associated with the store.</span></span>

<span data-ttu-id="8215d-127">На странице **Все скидки** продавцы могут также искать скидки по ключевым словам.</span><span class="sxs-lookup"><span data-stu-id="8215d-127">On the **All discounts** page, sales associates can also search discounts by keywords.</span></span> <span data-ttu-id="8215d-128">При поиске по ключевому слову выполняется поиск в полях, содержащих название и описание скидки.</span><span class="sxs-lookup"><span data-stu-id="8215d-128">The keyword search looks in the fields that hold the discount name and discount description.</span></span> <span data-ttu-id="8215d-129">Продавцы могут также фильтровать скидки в зависимости от того, требуется ли для скидки код купона.</span><span class="sxs-lookup"><span data-stu-id="8215d-129">Sales associates can also filter discounts based on whether a discount requires a coupon code.</span></span>

## <a name="cross-sell-and-upsell-by-using-discounts"></a><span data-ttu-id="8215d-130">Перекрестные продажи и расширенные продажи с использованием скидок</span><span class="sxs-lookup"><span data-stu-id="8215d-130">Cross-sell and upsell by using discounts</span></span>

<span data-ttu-id="8215d-131">Многострочные скидки, такие как скидки за количество, скидки за комплект и пороговые скидки, являются хорошим способом мотивации клиентов на приобретение дополнительных продуктов для получения больших скидок.</span><span class="sxs-lookup"><span data-stu-id="8215d-131">Multiline discounts, such as quantity discounts, mix-and-match discounts, and threshold discounts, are a great way to motivate customers to buy more products to get larger discounts.</span></span> <span data-ttu-id="8215d-132">Поэтому они также помогают увеличить размер корзины клиента и доход розничного торговца.</span><span class="sxs-lookup"><span data-stu-id="8215d-132">Therefore, they also help increase the size of a customer's cart and retailer revenue.</span></span> <span data-ttu-id="8215d-133">Эти скидки могут быть общедоступными на веб-сайтах электронной коммерции, в социальных сетях и на баннерах в магазине.</span><span class="sxs-lookup"><span data-stu-id="8215d-133">These discounts can be publicized on e-commerce websites, on social media, and on banners in the store.</span></span>

<span data-ttu-id="8215d-134">Однако даже в том случае, когда используются все эти методы оповещения, клиенты могут пропускать возможность воспользоваться преимуществами рекламных акций.</span><span class="sxs-lookup"><span data-stu-id="8215d-134">However, even when all these publicity methods are used, customers might miss the opportunity to take advantage of promotions.</span></span> <span data-ttu-id="8215d-135">Для облегчения продавцам получения сведений о том, какие рекламные акции применимы к выбранной строке или даже ко всей корзине, розничные продавцы могут добавить кнопку для операции "Просмотр всех скидок" для любой сетки кнопок в POS-терминале.</span><span class="sxs-lookup"><span data-stu-id="8215d-135">To make it easy for sales associates to learn what promotions are applicable to a selected line, or even to the whole cart, retailers can add the button for the "View all discounts" operation to any button grid in POS.</span></span> <span data-ttu-id="8215d-136">Рекомендуется добавить кнопку в сетку кнопок для страницы **Проводки**.</span><span class="sxs-lookup"><span data-stu-id="8215d-136">We recommend that the button be added to the button grid for the **Transaction** page.</span></span> <span data-ttu-id="8215d-137">Таким образом, продавец можно выбрать строку проводки, затем выбрать эту кнопку, чтобы отобразить все скидки, доступные для выбранной строки.</span><span class="sxs-lookup"><span data-stu-id="8215d-137">In that way, a sales associate can select a transaction line and then select the button to show all the discounts that are available for the selected line.</span></span> <span data-ttu-id="8215d-138">Продавец также может выбрать другую вкладку для отображения скидок, которые применяются для всей проводки.</span><span class="sxs-lookup"><span data-stu-id="8215d-138">The sales associate can also select another tab to show discounts that apply to the whole transaction.</span></span>

<span data-ttu-id="8215d-139">На странице **Все скидки**, которая была упомянута выше, отображаются только те скидки, которые не конкурируют ни с какими из примененных скидок.</span><span class="sxs-lookup"><span data-stu-id="8215d-139">The **All discounts** page that was mentioned earlier shows only discounts that don't compete with any of the applied discounts.</span></span> <span data-ttu-id="8215d-140">Это позволяет гарантировать, что если продавец информирует клиента о скидке и клиент выполняет необходимое действие (например, клиент покупает еще одну номенклатуру для получения скидки 10 процентов), скидка применяется к проводке.</span><span class="sxs-lookup"><span data-stu-id="8215d-140">This behavior helps ensure that, if a sales associate informs a customer about a discount, and the customer takes the required action (for example, the customer buys one more item to get 10 percent off), the discount is applied to the transaction.</span></span> <span data-ttu-id="8215d-141">Как указано ранее, скидки на основе купонов отображаются только в том случае, если параметр **Применить без кода купона** включен.</span><span class="sxs-lookup"><span data-stu-id="8215d-141">As was mentioned earlier, coupon-based discounts are shown only when the **Apply without a coupon code** option is turned on.</span></span>
