# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Требования к браузеру для сайтов Dynamics 365 Commerce

[!include [banner](includes/banner.md)]

В этой теме рассматриваются требования к браузеру для сайтов Microsoft Dynamics 365 Commerce.

Сайты Dynamics 365 Commerce лучше всего поддерживаются следующими веб-браузерами. Чтобы найти последний выпуск для каждого веб-браузера, перейдите на веб-сайт производителя программного обеспечения.

- [Microsoft Edge](https://www.microsoft.com/edge) (до текущей версии минус 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (до текущей версии минус 1)
- [Google Chrome](https://www.google.com/chrome) (до текущей версии минус 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (до текущей версии минус 1)

> [!NOTE]
> Internet Explorer не поддерживается. Если вы используете Internet Explorer, некоторые функции могут не работать, и у пользователей может быть неполный опыт работы.
