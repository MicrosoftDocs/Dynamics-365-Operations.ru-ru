# <a name="cart-icon-module"></a>Модуль значка корзины

[!include [banner](includes/banner.md)]

В этом разделе описывается модуль значка корзины, а также описывается, как добавить его на страницы сайта в Microsoft Dynamics 365 Commerce.

Модуль значка корзины представляет корзину в модуле заголовка страницы и отображает количество номенклатур в корзине. В модуле значка корзины также отображается сводка по корзине (также называется мини-корзиной), когда на значок корзины наводится указатель. Мини-корзина предоставляет пользователю сводку номенклатур в корзине без перехода на страницу корзины. Кроме того, она позволяет пользователю непосредственно перейти на страницу оформления заказа, если его устраивает сводка. Это позволяет сократить число переходов между страницами и ускорить оформление.

## <a name="module-properties"></a>Свойства модуля

- **Показать мини- корзину** — если для этого свойства задано значение **True**, будет показана сводка корзины (мини-корзина), когда указатель наводит курсор на значок корзины. Эта функциональная возможность поддерживается только для портов просмотра рабочего стола.
- **Разрешить анонимное оформление** — если это свойство имеет значение **True**, мини-корзина позволяет пользователям, не вошедшим в систему, выполнять оформление гостя. Это свойство доступно в Commerce версии 10.0.21 в составе пакета библиотеки модулей Commerce.
- **Порядок номенклатур** — это свойство управляет порядком, в котором номенклатуры отображаются в мини-корзине. Когда выбран параметр **Новые номенклатуры добавляются в начало списка**, новые номенклатуры, добавленные в корзину, появляются в верхней части списка элементов мини-корзины. Когда выбран параметр по умолчанию **Новые номенклатуры добавляются в низ списка**, новые номенклатуры, добавленные в корзину, появляются в нижней части списка элементов мини-корзины. Это свойство доступно в Commerce версии 10.0.21 в составе пакета библиотеки модулей Commerce.

> [!IMPORTANT]
> Свойства **Разрешить анонимное оформление** и **Порядок номенклатур** доступны в Commerce версии 10.0.21. Они требуют установки пакета библиотеки модулей Commerce версии 9.31.

## <a name="module-properties-and-slots-in-the-adventure-works-theme"></a>Свойства модуля и слоты в теме Adventure Works

В теме Adventure Works модуль значка корзины включает в себя две дополнительные ячейки для мини-корзины. Эти ячейки включаются в качестве расширения определения модуля.

- **Рекламные материалы в пустой корзине** — эта ячейка принимает модуль блока содержимого. Если корзина пуста, отображается указанный модуль блока содержимого. Модуль блока содержимого может использоваться для рекламы, маркетинга и ссылок на страницы категорий, чтобы помочь пользователям продолжить свои покупки.
- **Рекламное содержимое** — эта ячейка может использоваться для демонстрации рекламных акций, таких как "Бесплатная доставка по заказам на сумму больше 100 долларов США". В ячейке рекламных материалов можно использовать модули блока содержимого, блока текста и списка изображений.

На следующем рисунке показан пример модуля "Корзина" в теме Adventure Works, который отображает рекламный контент в мини-корзине.

> [!IMPORTANT]
> Ячейки темы Adventure Works доступны в выпуске Dynamics 365 Commerce версии 10.0.20.
