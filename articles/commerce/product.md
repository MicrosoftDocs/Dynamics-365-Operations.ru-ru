# <a name="add-product-recommendations-on-pos"></a>Добавление рекомендаций по продуктам в POS

[!include [banner](includes/banner.md)]

По своей сути, рекомендации продуктов представляют собой трансформируемое бизнес-приложение, охватывающее все сферы коммерции для создания богатого, привлекательного и специализированного процесса обнаружения продукции. Чтобы реализовать эту функцию на POS-терминале, следуйте [указаниям по добавлению рекомендаций в POS-устройства](add-recommendations-control-pos-screen.md). 

Дополнительные сведения о функциях рекомендаций продуктов см. в разделе [Обзор рекомендаций продуктов](../commerce/product-recommendations.md). 

## <a name="scenarios"></a>Сценарии

Рекомендации по продукции включены в следующих сценариях POS. Они доступны в Cloud POS или Modern POS (MPOS).

1. На странице **Сведения о продукте**:

    - Если сотрудник магазина открывает страницу **Сведения о продукте** при просмотре предыдущих проводок в разных каналах, служба рекомендаций предлагает дополнительные номенклатуры, которые часто приобретаются вместе. В зависимости от надстроек для службы, розничные продавцы могут показывать для продуктов рекомендации **Купить похожие продукты** и **Купить продукты с похожим описанием**, а также персонализированные рекомендации для пользователей, имеющих историю предыдущих покупок.

    [![Рекомендации на странице "Сведения о продукте".](./media/proddetails.png)](./media/proddetails.png)

2. На странице **Проводка**:

    - Механизм рекомендаций предлагает номенклатуры на основе всего списка номенклатур в корзине, которые часто приобретаются вместе.

    > [!NOTE]
    > Для отображения рекомендаций на странице **Проводка** предприятие розничной торговли должно обновить макет экрана в Dynamics 365 Commerce. Элемент управления **Рекомендации** необходимо перетащить на страницу **Проводка**.

    [![Рекомендации на странице "Проводки".](./media/transactionscreenmultipleproductslargemessengersbag-5.jpg)](./media/transactionscreenmultipleproductslargemessengersbag-5.jpg)

## <a name="configure-commerce-to-enable-pos-recommendations"></a>Настройка Commerce для включения рекомендаций POS 

Чтобы настроить рекомендации по продукту, убедитесь, что вы выполнили процедуру подготовки для рекомендаций по продуктам Commerce, следуя указаниям в разделе [Включение рекомендаций по продукту](../commerce/enable-product-recommendations.md). По умолчанию рекомендации выводятся на странице **Сведения о продукте** и на странице **Сведения о клиенте** после выполнения шагов подготовки и успешной обработки данных в файлах cookies. 

## <a name="add-recommendations-to-the-transaction-screen"></a>Добавление рекомендаций на экран проводок

1. Чтобы добавить рекомендации на экран транзакции, выполните шаги, описанные в разделе [Добавление рекомендаций на экран транзакции](add-recommendations-control-pos-screen.md).
1. Чтобы отразить изменения, сделанные в конструкторе макета экрана POS, выполните задание настройки канала **1070** в Commerce Headquarters.

> [!NOTE] 
> Если необходимо включить рекомендации POS с помощью файла RecoMock с разделителями-запятыми (CSV), перед настройкой диспетчера макета необходимо развернуть файл CSV в библиотеке ресурсов жизненного цикла Microsoft Dynamics Lifecycle Services (LCS). При использовании CSV-файла RecoMock не требуется включать рекомендации. Файл CSV доступен только для демонстрационных целей. Это рекомендуется пользователям и архитекторам решений, желающим смоделировать внешний вид списков рекомендаций для демонстрационных целей без приобретения дополнительного модуля единиц складского хранения (SKU).
