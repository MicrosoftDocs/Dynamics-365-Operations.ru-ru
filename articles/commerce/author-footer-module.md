# <a name="footer-module"></a>Модуль нижнего колонтитула  

[!include [banner](includes/banner.md)]

В этом разделе описываются модули нижнего колонтитула, а также описывается, как создавать их в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Модуль нижнего колонтитула — это специальный контейнер, который используется для размещения модулей, появляющихся в нижнем колонтитуле страницы. Например, он может содержать ссылки на различные страницы сайта, такие как , такие как страницы **Свяжитесь с нами** и **Политики магазина**.

На следующем рисунке показан пример модуля нижнего колонтитула на странице сайта.

![Пример модуля нижнего колонтитула](./media/ecommerce-footer.PNG)

## <a name="footer-module-properties"></a>Свойства модуля нижнего колонтитула 

Как и большинство контейнеров, модуль нижнего колонтитула поддерживает свойства для заголовка и ширины. Он также поддерживает добавление нескольких модулей категории нижних колонтитулов. Каждый добавляемый модуль категорий нижнего колонтитула отображается как столбец в модуле нижнего колонтитула.

## <a name="modules-available-in-a-footer-module"></a>Модули, доступные в модуле нижнего колонтитула

**Элементы нижнего колонтитула** — модуль элементов нижнего колонтитула может содержать заголовок, изображение и ссылку. Заголовок может использоваться как отдельно, так и в сочетании с изображением и ссылкой. Каждую ссылку в нижнем колонтитуле можно настроить таким образом, чтобы она состояла только из текста (например, ссылки "Свяжитесь с нами" и "Конфиденциальность") или имела и текст, и изображение (например, ссылки на социальные сети).

**К началу** — модуль "К началу" содержит ссылку для быстрого перехода к верхней части страницы. Требуется место назначения. По умолчанию значением места назначения является \#, что переводит пользователя в верхнюю часть страницы.

## <a name="create-a-footer-module"></a>Создание модуля нижнего колонтитула

1. Перейдите к разделу **Фрагменты страниц**, выберите **Создать**, чтобы создать фрагмент.
1. В диалоговом окне **Создание фрагмента страницы** выберите модуль **Контейнер**, введите имя для фрагмента страницы, затем выберите **ОК**.
1. В ячейке **Контейнер по умолчанию** выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Категория нижнего колонтитула**, затем выберите **ОК**.
1. В ячейке **Категория нижнего колонтитула** выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Элемент нижнего колонтитула**, затем выберите **ОК**.
1. Выберите ячейку **Элемент нижнего колонтитула**, затем в области свойств справа настройте требуемые заголовок, ссылку и текст ссылки, а также изображение.
1. Повторите шаги 5–7, чтобы добавить другие элементы нижнего колонтитула.
1. Чтобы добавить в нижний колонтитул ссылку "К началу", выберите многоточие (**…**) в ячейке **Категория нижнего колонтитула**, затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **К началу**, затем выберите **ОК**.
1. Выберите ячейку **К началу**, затем в области свойств справа настройте требуемые текст и другие свойства модуля.
1. Выберите **Завершить редактирование** для возврата фрагмента, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.

Чтобы гарантировать, что заголовок появляется на каждой странице, выполните следующие действия в каждом шаблоне страниц, созданном для сайта.

1. В слоте **Нижний колонтитул** модуля **Страница по умолчанию** добавьте созданный фрагмент нижнего колонтитула.
1. Выберите **Завершить редактирование** для возврата шаблона, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.

Добавление фрагмента страницы в шаблоны страниц помогает гарантировать отображение нижнего колонтитула на каждой странице.
