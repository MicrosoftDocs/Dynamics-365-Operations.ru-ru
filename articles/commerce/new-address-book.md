# <a name="create-new-address-book"></a>Создание адресной книги

[!include [banner](includes/banner.md)]

В этом разделе описывается, как создать адресную книгу в Microsoft Dynamics 365 Commerce.

Адресные книги используются в Commerce по различным причинам, включая хранение списков клиентов и списков сотрудников для канала. Адресные книги могут использоваться для одного канала или для совместного использования каналами.

## <a name="create-a-new-address-book"></a>Создание адресной книги

Чтобы создать адресную книгу, выполните следующие действия.
 
1. В области переходов выберите **Модули \> Настройка канала \> Адресные книги**.
1. В области действий выберите **Создать**.
1. Введите сведения об имени и описание.
1. В области действий выберите **Сохранить**.

На следующем рисунке показано создание адресной книги сотрудников для розничного магазина.
