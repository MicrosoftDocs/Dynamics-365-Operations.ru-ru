<span data-ttu-id="72b1b-106">Модуль размещения содержимого — это особый контейнер, в который могут быть помещены другие модули в конкретном макете.</span><span class="sxs-lookup"><span data-stu-id="72b1b-106">A content placement module is a special container that other modules can be put inside in a specific layout.</span></span> <span data-ttu-id="72b1b-107">Модули размещения содержимого поддерживают следующие типы модулей как дочерние модули: элемент размещения содержимого, элемент приветствия учетной записи, элемент заказа учетной записи, элемент списка желаний учетной записи и элемент профиля учетной записи.</span><span class="sxs-lookup"><span data-stu-id="72b1b-107">Content placement modules support the following types of modules as child modules: content placement item, account welcome item, account order item, account wish list item, and account profile item.</span></span>

<span data-ttu-id="72b1b-108">Модули размещения содержимого получают данные из дочерних модулей, которые они поддерживают.</span><span class="sxs-lookup"><span data-stu-id="72b1b-108">Content placement modules derive data from the child modules that they support.</span></span> <span data-ttu-id="72b1b-109">Таким образом, модули размещения содержимого могут использоваться различными способами, в зависимости от того, какие модули были добавлены в него.</span><span class="sxs-lookup"><span data-stu-id="72b1b-109">Therefore, content placement modules can be used in various ways, depending on the modules that are added to it.</span></span>

<span data-ttu-id="72b1b-110">Модули элементов размещения содержимого используют как изображения, так и текст для демонстрации рекламных акций, политик и характеристик продуктов.</span><span class="sxs-lookup"><span data-stu-id="72b1b-110">Content placement item modules use both images and text to showcase promotions, policies, and product features.</span></span> <span data-ttu-id="72b1b-111">Например, модуль элемента размещения содержимого может использоваться на домашней странице для отображения политик магазина или сведений об отгрузке.</span><span class="sxs-lookup"><span data-stu-id="72b1b-111">For example, a content placement item module can be used on a home page to show store policies or shipping information.</span></span> <span data-ttu-id="72b1b-112">Модули элементов размещения содержимого управляются данными из системы управления контентом (CMS) и могут быть размещены на любой странице.</span><span class="sxs-lookup"><span data-stu-id="72b1b-112">Content placement item modules are driven by data from the content management system (CMS) and can be put on any page.</span></span> <span data-ttu-id="72b1b-113">Однако они могут использоваться только внутри модуля размещения содержимого.</span><span class="sxs-lookup"><span data-stu-id="72b1b-113">However, they can be used only inside a content placement module.</span></span>

## <a name="examples-of-content-placement-modules-in-e-commerce"></a><span data-ttu-id="72b1b-114">Примеры модулей размещения содержимого в электронной коммерции</span><span class="sxs-lookup"><span data-stu-id="72b1b-114">Examples of content placement modules in e-Commerce</span></span>
