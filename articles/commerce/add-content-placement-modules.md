# <a name="content-placement-module"></a>Модуль размещения содержимого

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

В этом разделе описываются модули размещения содержимого и элементы размещения содержимого, а также описывается, как добавлять их к страницам сайта в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Модуль размещения содержимого — это особый контейнер, в который могут быть помещены другие модули в конкретном макете. Модули размещения содержимого поддерживают следующие типы модулей как дочерние модули: элемент размещения содержимого, элемент приветствия учетной записи, элемент заказа учетной записи, элемент списка желаний учетной записи и элемент профиля учетной записи.

Модули размещения содержимого получают данные из дочерних модулей, которые они поддерживают. Таким образом, модули размещения содержимого могут использоваться различными способами, в зависимости от того, какие модули были добавлены в него.

Модули элементов размещения содержимого используют как изображения, так и текст для демонстрации рекламных акций, политик и характеристик продуктов. Например, модуль элемента размещения содержимого может использоваться на домашней странице для отображения политик магазина или сведений об отгрузке. Модули элементов размещения содержимого управляются данными из системы управления контентом (CMS) и могут быть размещены на любой странице. Однако они могут использоваться только внутри модуля размещения содержимого.

## <a name="examples-of-content-placement-modules-in-e-commerce"></a>Примеры модулей размещения содержимого в электронной коммерции

* Модули размещения содержимого, содержащие модули элементов размещения содержимого, могут использоваться на домашней странице или на страницах маркетинга для демонстрации характеристик продукта, рекламных акций, политик магазина, информации об отгрузке и другой информации с помощью изображений и текста.
* Для демонстрации характеристик продукта можно использовать модули размещения содержимого, содержащие модули элементов размещения содержимого на страницах со сведениями о продукте.
* Модули размещения содержимого, содержащие элемент приветствия учетной записи или модули элемента заказов учетной записи, могут использоваться на страницах управления учетными записями.

## <a name="content-placement-module-properties"></a>Свойства модуля размещения содержимого

| Имя свойства | Стоимость | Описание |
|---------------|-------|-------------|
| Ширина         | **Вписать в контейнер** или **Заполнить экран** | Если значение равно **Вписать в контейнер**, элементы внутри модуля размещения содержимого ограничиваются шириной модуля размещения содержимого. Если установлено значение **Заполнить экран**, элементы не ограничиваются шириной модуля размещения содержимого, но могут быть переведены в полноэкранный режим. |

## <a name="content-placement-item-module-properties"></a>Свойства модуля элемента размещения содержимого

| Имя свойства | Стоимость | Описание |
|---------------|-------|-------------|
| Заголовок       | Текст заголовка и теги заголовка (**H1**, **H2**, **H3**, **H4**, **H5** или **H6**) | Каждый модуль элемента размещения содержимого может иметь заголовок. Свойство **Заголовок** поддерживает теги заголовка. По умолчанию используется тег заголовка **H2**, но для удовлетворения требованиям специальных возможностей его можно изменить. |
| Абзац     | Текст абзаца | Модули элементов размещения содержимого поддерживают текст абзаца в формате RTF. Поддерживаются некоторые основные возможности форматированного текста, такие как жирный шрифт, подчеркнутый текст, курсив и гиперссылки. Некоторые из этих возможностей могут быть переопределены темой страницы, применяемой к модулю. |
| Изображение         | Файл изображения | Изображение может быть связано с текстом. |
| Ссылка          | URL-адрес ссылки и текст ссылки | К тексту можно добавить ссылку, чтобы перенаправлять пользователей на определенную страницу. |
| Размер плитки     | Число от **1** до **12** | Для каждого элемента размещения содержимого это свойство определяет число гнезд, которые элемент должен заполнять в модуле размещения содержимого. Максимальный размер модуля размещения содержимого составляет 12 гнезд. Если общий размер плитки для первых *n* элементов в модуле размещения содержимого превышает 12 слотов, они переносятся на следующую строку. |

## <a name="add-a-content-placement-module-that-contains-a-content-placement-item-module-to-a-page"></a>Добавление модуля размещения содержимого, содержащего модуль элемента размещения содержимого, на страницу

Чтобы добавить модуль размещения содержимого, содержащий модуль элемента размещения содержимого, на новую страницу и задать необходимые свойства, выполните следующие действия.

1. Создайте шаблон с именем **шаблон размещения содержимого**.
1. В слоте **Главный** страницы по умолчанию добавьте модуль размещения содержимого.
1. В модуле размещения содержимого добавьте модуль элемента размещения содержимого.
1. Верните шаблон и опубликуйте его.
1. Используйте только что созданный шаблон размещения содержимого для создания страницы с именем **Страница размещения содержимого**.
1. В слоте **Главный** новой страницы добавьте модуль размещения содержимого.
1. В модуле размещения содержимого добавьте модуль элемента размещения содержимого.
1. На панели свойств модуля элемента размещения содержимого выберите изображение, добавьте заголовок, добавьте абзац, добавьте ссылку, задайте URL-адрес ссылки и установите размер плитки равным **6**.
1. Повторите шаги 7 и 8, чтобы добавить другой модуль элемента размещения содержимого с тем же размером плитки.
1. Сохраните и просмотрите страницу. Вы должны увидеть два элемента размещения содержимого, которые должны отображаться рядом друг с другом. Можно настроить свойство **Размер плитки** каждого модуля элемент размещения содержимого или добавить дополнительные модули для достижения нужного макета.
1. Верните страницу и опубликуйте его.
