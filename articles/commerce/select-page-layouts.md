# <a name="select-page-layouts"></a>Выбор макета страницы


[!include [banner](includes/banner.md)]

В этом разделе объясняется, как создавать и выбирать макеты страниц в Microsoft Dynamics 365 Commerce.

## <a name="create-layouts-for-an-existing-page"></a>Создание макетов для существующей страницы

> [!NOTE]
> Макеты можно создавать для существующих страниц только в том случае, если на этой странице имеется по крайней мере два модуля в основном слоте.

Чтобы создать макет для существующей страницы, выполните следующие действия.

1. Перейдите в раздел **Страницы** и найдите существующую страницу в списке. При необходимости воспользуйтесь функцией поиска.
1. Выберите страницу, выберите **Правка**, чтобы извлечь ее, затем выберите имя страницы, чтобы открыть ее. Запишите порядок модулей.
1. Выберите **Сохранить как новый макет**.
1. Введите имя для макета, затем выберите **ОК**.
1. Выберите **Преобразовать во внедренный макет**.
1. Измените порядок модулей так, как требуется, и запишите новый порядок.
1. Выберите **Сохранить как новый макет**.
1. Введите имя для макета, затем выберите **ОК**.
1. Выберите **Изменить макет**, выберите первый созданный макет, затем выберите **ОК**. Запишите порядок модулей. Измените его так, чтобы он соответствовал порядку модулей, сохраненному в макете.
1. Выберите **Завершить редактирование** для возврата страницы, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее. 

## <a name="select-a-different-layout-for-an-existing-page"></a>Выбор другого макета для существующей страницы

> [!NOTE]
> Можно выбрать другой макет для существующей страницы только в том случае, если шаблон, который использовался для создания этой страницы, имеет несколько макетов.

Чтобы выбрать другой макет для существующей страницы, выполните следующие действия.

1. Перейдите в раздел **Страницы** и найдите существующую страницу в списке. При необходимости воспользуйтесь функцией поиска.
1. Выберите страницу, выберите **Правка**, чтобы извлечь ее, затем выберите имя страницы, чтобы открыть ее.
1. Выберите **Изменить макет**.
1. Выберите новый макет для страницы, затем выберите **ОК**. Редактор страниц обновляется для отображения нового макета.
1. Выберите **Завершить редактирование** для возврата страницы, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.
