# <a name="subscribe-module"></a>Модуль подписки

[!include [banner](includes/banner.md)]

В этой статье описываются модули подписки, а также описывается, как добавлять их на страницы сайта в Microsoft Dynamics 365 Commerce.

Модули подписки могут использоваться на страницах сайта для сбора сведений о клиенте для информационных бюллетеней или рекламных акций.

На следующем рисунке показан пример модуля подписки в нижнем колонтитуле на странице сайта Adventure Works.

![Пример модуля подписки в нижнем колонтитуле страницы сайта Adventure Works](./media/Subscribe.PNG)

> [!IMPORTANT]
> - Модуль подписки доступен в библиотеке модуля Commerce в версии Dynamics 365 Commerce 10.0.20.
> - Модуль подписки демонстрируется в теме Adventure Works.
> - Для модуля подписки требуется расширение действий с данными для работы с некоторыми поставщиками электронной почты, чтобы бюллетени или рекламные письма могли быть отправлены после записи информации о клиенте.

## <a name="subscribe-module-properties"></a>Свойства модуля подписки

| Имя свойства | Значения | описание |
|---------------|--------|-------------|
| Заголовок       | Текст заголовка и метка заголовка (**H1**, **H2**, **H3**, **H4**, **H5** или **H6**) | Текстовый заголовок модуля подписки, например **Подписка на бюллетень** или **Регистрация со скидкой 10%**. |
| Параграф     | Текст параграфа | Модуль подписки поддерживает текст в виде абзаца в формате RTF, чтобы предоставить дополнительные сведения для призыва к действию в заголовке. |

## <a name="add-a-subscribe-module-to-a-new-page"></a>Добавление модуля подписки на новую страницу

Чтобы добавить модуль подписки на новую страницу и задать необходимые свойства в конструкторе сайтов Commerce, выполните следующие действия.

1. Перейдите к **Шаблоны** и откройте шаблон маркетинга для домашней страницы сайта (или создайте новый маркетинговый шаблон).
1. В слоте **Главный** страницы по умолчанию выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Выбрать модуль** выберите модуль **Подписаться**, затем выберите **ОК**.
1. Выберите **Сохранить**, выберите **Завершить редактирование** для возврата шаблона, затем нажмите кнопку **Опубликовать**, чтобы опубликовать его.
1. Перейдите к **Страницы** и откройте домашнюю страницу сайта (или создайте новую домашнюю страницу с помощью маркетингового шаблона).
1. В слоте **Главный** страницы по умолчанию выберите кнопку с многоточием (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Выбрать модуль** выберите модуль **Подписаться**, затем выберите **ОК**.
1. В области свойств модуля подписки добавьте заголовок, например **Подписка**.
1. Добавление текста абзаца, например, **Последние тенденции, стили и рекламные акции. Участвуете? Подпишитесь и получите новейшие современные предложения!**
1. Выберите **Сохранить**, затем выберите **Предварительный просмотр**, чтобы просмотреть страницу.
1. Выберите **Завершить редактирование** для возврата шаблона, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.
