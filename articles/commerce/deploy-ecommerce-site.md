# <a name="deploy-a-new-e-commerce-tenant"></a>Развертывание нового клиента электронной коммерции

[!include [banner](includes/banner.md)]

В этой статье описывается, как развернуть новый сайт электронной коммерции Dynamics 365 Commerce, используя службы Microsoft Dynamics Lifecycle Services (LCS).

Microsoft Dynamics Lifecycle Services (LCS) — это рабочая область совместной работы на основе облака, которую партнеры и клиенты могут использовать для управления проектами и средами, просмотра последних сведений о продуктах и функциях Microsoft Dynamics, а также для создания, отслеживания и просмотра обращений в службу технической поддержки. Функции управления электронной коммерцией интегрированы в LCS.

Дополнительные сведения о LCS см. в [Руководстве пользователя Lifecycle Services](/dynamics365/unified-operations/dev-itpro/lifecycle-services/lcs-user-guide).
    
## <a name="get-started"></a>Начать

Прежде чем можно будет инициализировать электронную коммерцию, необходимо инициализировать проект, среду и Retail Cloud Scale Unit (RCSU). Чтобы выполнить инициализацию в LCS, необходимо иметь разрешения для роли владельца проекта или руководителя среды. Поддерживаются топологии производственной среды и песочницы.

Дополнительные сведения о средах см. в разделе [Планирование среды](/dynamics365/unified-operations/fin-and-ops/imp-lifecycle/environment-planning). Дополнительные сведения о RCSU см. в разделе [Инициализация Retail Cloud Scale Unit](/dynamics365/unified-operations/dev-itpro/deployment/initialize-retail-channels).

## <a name="initialize-e-commerce"></a>Инициализация электронной коммерции

Эта процедура используется для инициализации функции электронной коммерции в существующей среде.

Прежде чем начать, убедитесь, что имеются следующие требуемые сведения:

- RCSU, который будет использован.
- Группа безопасности Microsoft Azure Active Directory, которая будет использоваться для системных администраторов электронной коммерции.
- Группа безопасности Microsoft Azure Active Directory, которая будет использоваться для модераторов оценок и отзывов.
- Домены, которые будут связаны со средой.

Кроме того, можно собрать следующие дополнительные сведения:

- Сведения Azure AD B2C:

    - Имя владельца.
    - Код клиента.
    - Пользовательский домен для входа.
    - URL-адрес для ответа.
    - Идентификатор политики регистрации или входа.
    - Код политики сброса пароля.
    - Идентификатор политики изменения профиля.

> [!NOTE]
> Эти сведения могут быть добавлены позже через запрос на обслуживание.

После сбора необходимых сведений выполните следующие действия для инициализации электронной коммерции.

1. Войдите в [LCS](https://lcs.dynamics.com).
1. Откройте проект, содержащий среду, в которой необходимо инициализировать электронную коммерцию.
1. В разделе **Среды** выберите среду.
1. В разделе **Компоненты среды** выберите ссылку **Управление розничной торговлей**.
1. На вкладке **Электронная коммерция** выберите **Настройка**. Появится диалоговое окно, в котором необходимо ввести сведения, необходимые для подготовки.
1. Введите требуемые сведения, затем перейдите к следующей странице.
1. На следующей странице заполните требуемые сведения, затем отправьте форму. Вы возвращаетесь на вкладку **Электронная коммерция**, где вы должны видеть, что инициализация запущена.
1. Чтобы просмотреть статус инициализации, выберите команду **Обновить** или вернитесь на вкладку **Электронная коммерция** позднее.
    
При инициализации электронной коммерции из LCS система подготавливает несколько компонентов, необходимых для электронной коммерции, и связывает их со средой. После завершения подготовки вкладка **Электронная коммерция** на странице **Управление розничной торговлей** обновляется, чтобы отразить подготовку. На странице отображаются последние развертывания настроек и статус всех остальных текущих развертываний. Она также содержит ссылки на сайт электронной коммерции и построитель сайтов электронной коммерции, где создаются сайты.

## <a name="access-commerce-site-builder"></a>Доступ к построителю сайтов Commerce

Для доступа к построителю сайтов Commerce перейдите на вкладку **Электронная коммерция** на странице **Управление розничной торговлей** в LCS и выберите ссылку **Средство управления сайтом электронной коммерции**. Страница построителя сайтов отображает представление на уровне клиента. На этой странице можно выполнить следующие действия.

- Изменение настроек на уровне клиента.
- Перейти на любой созданный сайт, для которого есть разрешение на просмотр. 
- Получить доступ к таким функциям, как модерация и отчетность.
- Создать новый сайт. Дополнительные сведения о создании и развертывание сайта см. в разделе [Создание сайта электронной коммерции](create-ecommerce-site.md).
