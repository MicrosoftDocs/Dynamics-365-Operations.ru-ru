# <a name="pickup-information-module"></a>Информационный модуль отправки

[!include [banner](includes/banner.md)]

В этом разделе описывается модуль сведений о самовывозе, а также описывается, как добавить его на страницы оформления заказа в Microsoft Dynamics 365 Commerce.

Модуль сведений о самовывозе может использоваться в модуле оформления заказа для отображения сведений о самовывозе заказа. Клиенты могут просматривать доступные даты и временные интервалы самовывоза, а затем выбирать подходящее время для получения их заказа. Например, клиент может выбрать забрать заказ в 15:00 21 марта из магазина в Сан-Франциско.

Временные интервалы получения для соответствующих магазинов необходимо настроить в Commerce Headquarters. Дополнительные сведения см. в разделе [Создание и обновление временных интервалов для самовывоза клиентом](dev-itpro/pickup-timeslots.md).

Если модуль сведений о самовывозе создается на странице оформления заказа, но для магазина, выбранного для получения, не определен ни один из временных интервалов, то в модуле будут показаны сведения, но пользователь не сможет выбрать какие-либо временные интервалы. Временные интервалы являются необязательными и не требуются для размещения заказа.

Если для получения в нескольких магазинах выбрано несколько номенклатур, в модуле сведений о самовывозе будет разрешено пользователю выбрать временной интервал для каждого магазина, при условии, что для него доступны временные интервалы.

> [!NOTE]
> Поддержка временных интервалов и модуля информации о самовывозе при оформлении заказа доступна в Dynamics 365 Commerce версии 10.0.15 и выше.

## <a name="module-properties"></a>Свойства модуля

- **Заголовок** — введите заголовок для модуля.

## <a name="show-time-slot-information-after-an-order-is-placed"></a>Отображение сведений о временном интервале после размещения заказа

После размещения заказа сведения о выбранном временном интервале можно просмотреть в [модуле подтверждения заказа](order-confirmation-module.md) и в [модуле сведений о заказе](account-management.md#order-details-page). Оба этих модуля имеют свойство **Показать сведения о временном интервале**. Чтобы можно было показать выбранный временной интервал во время процесса заказа, этому свойству должно быть присвоено значение **True**.
