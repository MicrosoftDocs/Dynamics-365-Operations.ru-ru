# <a name="store-commerce-app-capabilities"></a>Возможности приложения Store Commerce

[!include [banner](includes/banner.md)]

Приложение Store Commerce является современным интерфейсом POS-терминала для Microsoft Dynamics 365 Commerce. Оно позволяет компаниям обрабатывать проводки в магазине и управлять операциями серверной части, такими как запасы и обработка заказов. Приложение также позволяет предприятиям управлять взаимоотношениями с клиентами с помощью программы лояльности и отношений с клиентами. 

Приложение Store Commerce, построенное на основе Commerce Scale Unit (CSU), предоставляет полное многоканальное решение. Например, клиент может купить продукт по интернету и забрать его в ближайшем магазине, тем самым продолжая свой цикл взаимодействия покупки по нескольким каналам без потери данных. 

В этой статье представлен обзор возможностей приложения Store Commerce.
