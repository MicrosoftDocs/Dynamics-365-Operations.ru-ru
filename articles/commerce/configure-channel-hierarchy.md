# <a name="configure-a-channel-to-use-a-channel-navigation-hierarchy"></a>Настройка канала для использования навигационной иерархии канала


[!include [banner](includes/banner.md)]

В этом разделе описывается, как настроить канал для использования навигационной иерархии канала в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Навигационные иерархии канала объединяют продукты в категории, чтобы продукты могли просматриваться на веб-узле электронной коммерции или в POS-терминалах. Розничные и интернет-каналы должны быть сконфигурированы с помощью навигационной иерархии канала.

## <a name="configure-the-channel"></a>Настройка канала

Чтобы настроить канал для использования навигационной иерархии канала, выполните следующие действия.

1. В области переходов выберите **Модули \> Розничная торговля и коммерция \> Настройка канала \> Категории канала и атрибуты продуктов**.
1. Выберите канал для настройки.
1. В области действий выберите **Задать метаданные атрибута**.
1. В раскрывающемся списке **Иерархия категорий** выберите соответствующую навигационную иерархию канала.
1. На панели операций выберите **Сохранить**.
1. В **Группа атрибутов** добавьте любые группы атрибутов, которые будут глобальными атрибутами для всех узлов.

На следующем рисунке показано, как настроить канал для использования навигационной иерархии канала.

![Пример конфигурации канала.](media/configure-channel-hierarchy-1.png)

## <a name="set-attribute-metadata"></a>Задать метаданные атрибута

Задание метаданных атрибута разрешит настройку атрибутов на каждом узле.

Чтобы задать метаданные атрибутов, сделайте следующее.

1. В области действий выберите **Задать метаданные атрибута**.
1. Для каждого узла выберите **Атрибуты продукта канала**.
1. Установите для **Показать атрибут для канала** значение **Да** и для **Возможно уточнение** значение **Да**, чтобы включить уточнения для этого канала.
1. После настройки каждого узла в **области действий** нажмите кнопку **Сохранить** для сохранения.
1. Выберите **X** в верхнем правом углу, чтобы закрыть этот экран и вернуться назад на страницу **Категории каналов и атрибуты продукта**.

На следующем рисунке показан пример набора атрибутов продукта канала, настроенных в узле категории канала.

![Атрибуты канала в узле категории каналов.](media/configure-channel-hierarchy-2.png)

## <a name="publish-changes"></a>Публикация изменений

Чтобы изменения вступили в силу, необходимо опубликовать изменения.

Чтобы опубликовать изменения, выполните следующие действия.

1. В области действий выберите **Опубликовать обновления канала**.
1. В области **Опубликовать обновления канала** выберите **ОК**.

На следующем рисунке показано, как опубликовать обновления канала.

![Опубликовать обновления канала.](media/configure-channel-hierarchy-3.png)
