# <a name="create-and-update-time-slots-for-customer-pickup"></a>Создание и обновление временных интервалов для самовывоза клиентами

[!include [banner](../../includes/banner.md)]

В этом разделе описывается создание, настройка и обновление временных интервалов самовывоза клиентами в Commerce Headquarters.

Функция временного интервала дает розничным торговцам способ определить временной интервал для номенклатур, для которых включен режим самовывоза клиентами. Временные интервалы позволяют розничным торговцам определить дни и время, когда заказы можно забрать из магазина. Розничные торговцы также могут определить количество заказов, которые можно забрать в течение определенного периода. Таким образом, розничные торговцы могут ограничить количество заказов, которые можно забрать в данный день и в данный момент времени. Результатом является более качественный опыт для их клиентов.

> [!NOTE]
> Функция временного интервала доступна в Microsoft Dynamics 365 Commerce версии 10.0.15 и позже.

Следующая иллюстрация показывает пример выбора временного интервала во время оформления заказа электронной коммерции.

## <a name="time-slot-properties"></a>Свойства временного интервала

Временной интервал — это определенный интервал, когда клиент может выбрать получение заказа из определенного магазина или местоположения. Функция управления временным интервалом доступна только при настройке режима доставки "самовывоз клиентом" в Dynamics 365 Commerce.

Временной интервал определяется с помощью следующих свойств:

- **Режим доставки** — укажите способ доставки "самовывоз", на который распространяется временной интервал.
- **Минимальные дни** и **Максимальные дни** — укажите самые ранние и поздние даты, которые могут быть выбраны для самовывоза относительно даты при размещении заказа. 

    Свойство **Минимальные дни** гарантирует, что у продавца будет достаточно времени для обработки заказа, прежде чем он будет готов к самовывозу. Свойство **Максимальные дни** гарантирует, что пользователь не сможет выбрать дату, которая будет слишком далеко в будущем. Например, если минимальное значение установлено на **1**, и заказ размещен 20 сентября, самый ранний день, когда заказ будет доступен для самовывоза будет на следующий день (21 сентября). Аналогичным образом, установив максимальное значение, можно определить максимальное количество дней, в течение которых можно получить заказ. Когда минимальные и максимальные значения определены, пользователи сайта могут видеть и выбирать только определенный набор дней во время оформления заказа.

    Минимальное значение можно установить на десятичное значение, которое меньше 1. Например, если самовывоз доступен через четыре часа после размещения заказа, установите минимальное значение **0,17** (= 4 ÷ 24, округленное до двух десятичных знаков). Однако если вы установите минимальное значение на десятичное значение, которое больше 1, оно всегда округляется вверх до ближайшего целого числа. Например, значение **1,2** будет округлено до **2**. Аналогично, если вы установите максимальное значение на десятичное значение, оно всегда округляется вверх до ближайшего целого числа. 

- **Дата начала** и **Дата окончания** — укажите даты начала и окончания временного интервала. Каждая запись временного интервала имеет дату начала и дату окончания. Таким образом, у вас есть гибкость, чтобы добавить различные сроки в течение года (например, самовывоз в праздничные часы). Если начало и даты временного интервала изменены после размещения заказа, изменения не будут применяться к этому заказу. При определении даты начала и окончания необходимо учитывать даты закрытия магазинов (например, на Рождество) и гарантировать, что временные интервалы для этих дней не определены.
- **Активные часы получения** — укажите период, когда разрешен самовывоз. Например, время самовывоза может быть между 14:00 и 17:00 каждый день. Это свойство позволяет задавать время самовывоза независимо от часов работы магазина. Таким образом, розничный торговец может настроить время самовывоза, которое соответствует его конкретным бизнес-требованиям. При определении активных часов самовывоза необходимо учитывать часы работы магазина и гарантировать, что время самовывоза не определено для времени, когда магазин закрыт.

    > [!NOTE]
    > Часы для самовывоза из магазина должны быть определены в часовом поясе соответствующего магазина.

- **Интервал временного интервала** — укажите продолжительность, которая может быть выделена каждому временной интервалу. Например, продолжительность каждого временного интервала может быть с шагом в 15 минут, 30 минут или один час. Если значение интервала времени равно 0, интервал времени будет доступен для всей продолжительности между временем начала и окончания.
- **Временных интервалов за интервал** — укажите количество клиентов или заказов, которые могут быть обслужены для самовывоза в течение каждого интервала временного интервала. Например, введите **1**, **2**, **3** или любое другое целое число.
- **Активные дни** — укажите дни недели, когда активны интервалы времени самовывоза. Это свойство позволяет розничному торговцу определить дни, когда он хочет поддерживать самовывоз заказов.
- **Розничные каналы** — укажите розничные каналы. Каждый временной интервал может быть связан с одним или несколькими розничными магазинами. В зависимости от часов работы каждого магазина, можно создать одну или несколько записей временного интервала и связать их с каналом. 

<!-- ![HQ Timeslot overview.](../dev-itpro/media/Curbside_timeslot_Settings_overview.PNG) -->

Для каждого канала можно настроить только один шаблон временного интервала. Эти каналы включают в себя физические магазины, центры обработки звонков, мобильные устройства и сайты электронной коммерции.

## <a name="configure-the-time-slot-feature-in-commerce-headquarters"></a>Настройка функции временного интервала в Commerce Headquarters

Временные интервалы должны быть определены для каждого способа доставки в Commerce Headquarters, чтобы на них могли быть созданы ссылки на POS-терминале и в каналах электронной коммерции.

- С каждым магазином или каналом можно связать только один шаблон временных интервалов.
- Каждый созданный временной интервал должен быть уникальным для каждого способа поставки в каждом шаблоне.
- После настройки функции временного интервала календарь временных интервалов будет доступен для выбранных магазинов или групп магазинов. Он также будет отображаться на POS-терминале для справки.

Чтобы настроить функцию временного интервала в Commerce Headquarters, выполните следующие шаги.

1. Перейдите в раздел **Commerce** \> **Настройка каналов** \> **Временной интервал получения в магазине**.
1. Выберите **Создать** для создания нового шаблона временных интервалов. Для использования существующего шаблона выберите шаблон в левой области.
1. Заполните поля **Код временного интервала** и **Описание**.
1. На экспресс-вкладке **Выдача заказов — параметры времени** выберите **Добавить**.
1. В диалоговом окне **Выдача заказов — параметры времени** определите диапазон дат, способ поставки, активные часы доставки, активные дни, интервал временного слота, число слотов на интервал и другие настройки.

    Если временные интервалы будут статическими для ожидаемого будущего, задайте в поле **Дата окончания** значение **Никогда**.

    > [!NOTE]
    > Можно создать несколько шаблонов, но только один из них может быть связан с одним каналом или магазином.

    ![Диалоговое окно "Выдача заказов — параметры времени".](../dev-itpro/media/Curbside_timeslot_Settings_Page.PNG)

1. Закончив, выберите **OK**.
1. Если временные интервалы в течение дня будут различаться, создайте дополнительные записи на экспресс-вкладке **Получение заказов — параметры времени**, чтобы убедиться, что даты и время не перекрываются.
1. На экспресс-вкладке **Розничные каналы** выберите **Добавить**, чтобы связать шаблон временного интервала с магазинами или каналами, в которых он будет использоваться.
1. В диалоговом окне **Выбор узлов организации** кнопками со стрелками выберите (или отмените выбор) магазины, регионы и организации, с которыми должен быть связан шаблон.

    <!-- ![HQ Timeslot overview.](../dev-itpro/media/Curbside_timeslot_Settings_overview.PNG) -->

1. Закончив, выберите **OK**.
1. На странице **График распределения** запустите задания **1070** и **1135** для синхронизации данных с каналами.

## <a name="time-slot-selection-for-pos-orders"></a>Выбор временного интервала для заказов на POS-терминале

В POS-терминале, когда заказ или строка заказа определены для самовывоза, кассир может выбрать магазин или местоположение для получения, а также дату и временной интервал. Если клиент имеет заказ с получением в другом магазине, кассир может выбрать даты, когда товары можно забрать в том магазине. Поиск магазинов обеспечит ссылку на даты и время работы магазина.

Если страница оформления заказа не содержит модуля сведений о самовывозе, заказы будут размещены без предоставления пользователям возможности указывать или просматривать сведения об интервале времени.
