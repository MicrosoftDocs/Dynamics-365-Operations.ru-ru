# <a name="quick-view-module"></a>Модуль быстрого просмотра

[!include [banner](includes/banner.md)]
[!include [banner](includes/preview-banner.md)]

В этом разделе описываются модули быстрого просмотра, а также описывается, как добавлять их к страницам сайта в Microsoft Dynamics 365 Commerce.

Модуль быстрого просмотра позволяет пользователям быстро просматривать сведения о продукте при просмотре продуктов на странице списка и добавлять один или несколько продуктов в корзину со страницы списка без перехода на страницу сведений о продукте (PDP). В модуле быстрого просмотра содержится обзор сведений о продукте, которые требуются пользователям для принятия решения "добавить в корзину". Он также предоставляет ссылку на страницу PDP, чтобы пользователи могли просмотреть дополнительные сведения о продукте и параметры покупки.

Модуль быстрого просмотра поддерживается модулями [коллекции продуктов](product-collection-module-overview.md) и [результатов поиска](search-result-module.md).

> [!IMPORTANT]
> Модуль быстрого просмотра доступен в выпуске Commerce версии 10.0.17.

## <a name="module-properties"></a>Свойства модуля

Модуль быстрого просмотра поддерживает те же функции, что и модуль поля покупки. Таким образом, свойства модуля быстрого просмотра похожи на свойства модуля поля покупки.

| Свойство | Значения | описание |
|----------------|--------|-------------|
| Тег заголовка | **H1**, **H2**, **H3**, **H4**, **H5** или **H6** | Это свойство определяет тег заголовка для названия продукта. Если модуль быстрого просмотра находится вверху страницы, это свойство должно иметь значение **H1** для соответствия стандартам специальных возможностей. |
| Разрешить произвольную цену | **True** или **False** | Если это свойство имеет значение **True**, пользователь может ввести пользовательскую цену. |
| Минимальная цена | Целое число | Это свойство применимо только в том случае, если свойство **Разрешить нестандартную цену** имеет значение **True**. Оно определяет минимальную цену, которую пользователь может ввести (например, 1 доллар США). |
| Максимальная цена | Целое число | Это свойство применимо только в том случае, если свойство **Разрешить нестандартную цену** имеет значение **True**. Оно определяет максимальную цену, которую пользователь может ввести (например, 1 000 долларов США). |

## <a name="commerce-site-builder-settings"></a>Параметры конструктора сайтов Commerce

Подобно модулю поля покупки, модуль быстрого просмотра учитывает настройки **Параметры сайта \> Расширения \> Добавить продукт в корзину**. Однако параметр **Перейти к странице корзины** игнорируется, поскольку он не согласуется с целью модуля быстрого просмотра, который должен позволять пользователям просматривать несколько продуктов на странице списка и добавлять их в корзину без ухода со страницы списка.

## <a name="add-a-quick-view-module-to-a-product-collection-module"></a>Добавление модуля быстрого просмотра на страницу коллекции продуктов

Модуль быстрого просмотра может быть добавлен в модули коллекции продуктов и результатов поиска.

Чтобы добавить модуль быстрого просмотра в модуль коллекции продуктов в конструкторе сайтов Commerce, выполните следующие действия.

1. Перейдите в раздел **Страницы**, затем выберите главную страницу сайта Fabrikam.
1. Перейдите к любому модулю **Коллекция продуктов** на главной странице, выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Быстрый просмотр**, затем выберите **ОК**.
1. В области свойств модуля **Быстрый просмотр** выберите **Заголовок**. В диалоговом окне **Заголовок** задайте в поле **Уровень заголовка** значение **H2**, затем выберите **ОК**.
1. Выберите **Сохранить**, выберите **Завершить редактирование** для возврата страницы, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.
