# <a name="order-details-module"></a>Модуль сведений о заказе


[!include [banner](includes/banner.md)]

В этом разделе описываются модули сведений о заказе, а также описывается, как использовать их в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

После размещения заказа можно использовать модуль сведений о заказе для отображения подробных сведений подтверждения заказа. Здесь отображается код подтверждения заказа, контактная информация заказа и другие сведения о заказе, такие как купленные номенклатуры, сведения о платежах и методе отгрузки.

## <a name="order-details-module-properties"></a>Свойства модуля сведений о заказе

| Имя свойства  | Значения | описание |
|----------------|--------|-------------|
| Заголовок        | Текст заголовка и метка заголовка (**H1**, **H2**, **H3**, **H4**, **H5** или **H6**) | Модуль сведений о заказе может иметь заголовок. По умолчанию для заголовка используется тег заголовков **H2**. Однако тег можно изменить для соответствия требованиям к специальным возможностям. |
| Контактный номер | Текст | Для вопросов, касающихся заказов, может быть указан номер контакта. |

## <a name="modules-that-can-be-used-on-an-order-details-page"></a>Модули, которые могут использоваться на странице сведений о заказе

При создании страницы сведений о заказе в дополнение к модулю сведений о заказе можно добавлять другие соответствующие модули. Далее приводятся некоторые примеры.

- **Модуль рекомендаций** — модуль рекомендаций можно добавить на страницу сведений о заказе, чтобы предложить другие продукты для данного клиента.
- **Модули маркетинга** — любой модуль маркетинга можно добавить на страницу сведений о заказах для отображения маркетингового содержимого.

## <a name="add-an-order-details-module-to-a-page"></a>Добавление модуля сведений о заказе на страницу

Чтобы добавить модуль сведений о заказе на новую страницу и задать необходимые свойства, выполните следующие действия.

1. Перейдите к пункту **Шаблоны**, выберите **Создать**, чтобы создать шаблон.
1. В диалоговом окне **Создать шаблон** в разделе **Имя шаблона** введите имя **Шаблон сведений о заказе**, затем выберите **ОК**.
1. В ячейке **Содержание** выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Страница по умолчанию**, затем выберите **ОК**.
1. В слоте **Главный** модуля **Страница по умолчанию** выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Сведения о заказе**, затем выберите **ОК**.
1. Выберите **Сохранить**, затем выберите **Предварительный просмотр**, чтобы просмотреть шаблон. Модуль сведений о заказе не должен обрабатываться, поскольку ему требуется контекст номера подтверждения заказа.
1. Выберите **Завершить редактирование** для возврата шаблона, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.
1. Перейдите к разделу **Страницы**, выберите **Создать**, чтобы создать страницу.
1. В диалоговом окне **Выбор шаблона** выберите **Шаблон сведений о заказе**. В поле **Имя страницы** введите **Страница сведений о заказе**, затем выберите **ОК**.
1. В слоте **Главный** модуля **Страница по умолчанию** выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Сведения о заказе**, затем выберите **ОК**.
1. На панели свойств для модуля сведений о заказе выберите **Заголовок** рядом с символом карандаша.
1. В поле **Текст заголовка** в диалоговом окне **Заголовок** введите текст заголовка **Сведения о заказе**, затем выберите **ОК**.
1. Выберите **Сохранить**, затем выберите **Предварительный просмотр**, чтобы просмотреть страницу.
1. Выберите **Завершить редактирование** для возврата страницы, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.
