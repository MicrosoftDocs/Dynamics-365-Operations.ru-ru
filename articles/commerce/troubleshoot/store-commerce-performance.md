# <a name="troubleshoot-store-commerce-performance-issues"></a>Устранение проблем с производительностью Store Commerce

[!include [banner](../includes/banner.md)]

В этой статье объясняется, как устранять проблемы с производительностью в приложении Microsoft Dynamics 365 Commerce Store Commerce.

- При устранении неполадок, связанных с производительностью Store Commerce, избегайте использования удаленного входа. Вместо этого выполните вход непосредственно на устройстве Store Commerce.
- Если приложение Store Commerce работает медленно, проверьте использование ЦП, памяти, сети и диска с помощью диспетчера задач или монитора ресурсов. Убедитесь, что нет значительного потребления ресурсов устройств.
- Проверьте состояние подключения к сети и убедитесь, что запросы и ответы HTTPS не длятся более нескольких секунд.
