# <a name="product-collection-modules"></a>Модули семейства продуктов


[!include [banner](includes/banner.md)]

В этом разделе представлен обзор модулей семейства продуктов в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Обнаружение продукта — это основное средство, используемое розничными компаниями для работы с клиентами на веб-сайте электронной коммерции. Модули семейства продуктов помогают предприятиям розничной торговли создавать привлекательные возможности для покупок, представляя интуитивно понятный визуальный интерфейс, который может использоваться для быстрой разработки семейства продуктов.

Модули семейства продуктов представляют физические продукты и услуги на веб-сайте. Модуль семейства продуктов обычно связан со страницей сведений, где клиенты могут купить продукт или услугу, или получить дополнительные сведения о ней. 

Источниками для семейств продуктов могут быть списки следующих четырех типов:

- Редакционные списки продуктов, которые вручную определяются в Dynamics 365 Commerce в качестве связанных продуктов для продукта или списков продуктов
- Алгоритмические списки, такие как списки новых, лучше всего продаваемых их трендовых продуктов
- Списки рекомендаций на основе машинного обучения
- Списки персонализации, которые поддерживают персонализированные результаты для клиента. Клиенты должны войти на сайт электронной коммерции, чтобы увидеть персонализированные результаты. Гостевые пользователи не видят персонализированные результаты. Клиенты могут отказаться от персонализации на [странице управления учетной записью](account-management.md).

На следующем рисунке показаны различные типы семейств продуктов, используемых на сайте электронной коммерции.

## <a name="product-collection-modules-and-types"></a>Модули и типы семейства продуктов

В следующей таблице описываются различные типы модулей семейств продуктов в Dynamics 365 Commerce.

| Модуль семейства продуктов  | Вид | Описание |
|----------------------------|------|-------------|
| Категория                   | Категория | Этот модуль показывает список продуктов в категории, определяемый иерархией категории навигации, созданной розничным продавцом для канала. |
| Связанные продукты           | Редакция | В этом типе модуля семейства продуктов отображается список продуктов, которые менеджер по продажам настроил в качестве связанных продуктов в Commerce, для типа отношений, выбранного автором. |
| Результаты поиска             | Поисковый запрос | В этом типе модуля семейства продуктов отображается список продуктов, которые лучше всего соответствуют поисковому запросу, введенному клиентом. |
| Списки подобранных продуктов      | Редакция | Этот модуль показывает настраиваемые списки, созданные менеджерами по продажам и редакторами в Commerce. |
| Сoздать                        | Алгоритмический | Этот модуль отображает список новейших продуктов, которые были добавлены в каналы и каталоги. Этот список может отображать персонализированные результаты для пользователя, выполнившего вход, если автор сайта выбирает этот вариант. |
| Лидеры продаж               | Алгоритмический | В этом модуле отображается список продуктов, которые ранжированы по наибольшему количеству продаж. Этот список может отображать персонализированные результаты для пользователя, выполнившего вход, если автор сайта выбирает этот вариант. |
| Популярные                   | Алгоритмический | В этом модуле показан список наиболее производительных продуктов для данного периода. Этот список может отображать персонализированные результаты для пользователя, выполнившего вход, если автор сайта выбирает этот вариант. |
| Часто покупают вместе | Искусственный интеллект или машинное обучение | Этот модуль использует машинное обучение для анализа шаблонов покупок потребителей и рекомендует связанные номенклатуры, которые часто приобретаются вместе с данным продуктом. Этот список может отображать персонализированные результаты для пользователя, выполнившего вход, если автор сайта выбирает этот вариант. |
| Людям также нравится           | Искусственный интеллект или машинное обучение | Этот модуль использует машинное обучение для анализа шаблонов покупок потребителей и рекомендует номенклатуры, связанные с данным продуктом. Этот список может отображать персонализированные результаты для пользователя, выполнившего вход, если автор сайта выбирает этот вариант. |
| Для вас              | Искусственный интеллект или машинное обучение | Этот модуль использует машинное обучение для анализа шаблонов покупок пользователя, выполнившего вход, и предоставления персонализированных рекомендаций, основанных на этих шаблонах покупок. Для гостевого пользователя этот список будет свернут. |

## <a name="add-a-product-collection-module-to-a-category-page"></a>Добавление модуля семейства продуктов на страницу категории

Чтобы добавить модуль семейства продуктов на страницу категории, выполните следующие действия.

1. Перейдите к разделу **Страницы**, выберите **Создать**, чтобы создать страницу.
1. В диалоговом окне **Выбор шаблона** выберите тот же шаблон, что используется страницей категории по умолчанию. В поле **Имя страницы** введите соответствующее имя, затем выберите **ОК**.
1. В ячейке **Вложенный нижний колонтитул** выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Контейнер**, затем выберите **ОК**.
1. В ячейке **Контейнер** выберите многоточие (**...**), затем выберите **Добавить модуль**.
1. В диалоговом окне **Добавить модуль** выберите модуль **Семейство продуктов**, затем выберите **ОК**.  
1. В области свойства для модуля семейства продуктов выберите **Добавить список продуктов**.
1. В диалоговом окне **Выбор конфигурации списка продуктов** выберите тип списка, источник списка и введите число номенклатур. Настройте любые другие параметры, доступные для данного типа списка. Дополнительные сведения о типах списков см. в приведенной ниже таблице. 
1. Нажмите **ОК**.
1. Выберите **Сохранить**, затем выберите **Предварительный просмотр**, чтобы просмотреть страницу.
1. Выберите **Завершить редактирование** для возврата страницы, затем нажмите кнопку **Опубликовать**, чтобы опубликовать ее.

В следующей таблице перечислены типы списков, доступные для выбора в диалоговом окне **Выбор конфигурации списка продуктов**.

|----------------------------|-------------|-------|--------------|------------------|-----------------|
