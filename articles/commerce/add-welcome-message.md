# <a name="add-a-welcome-message"></a>Добавление приветственного сообщения

[!include [banner](includes/banner.md)]

В этом разделе описывается, как добавить приветственное сообщение на веб-сайт Microsoft Dynamics 365 Commerce.

Приветственное сообщение на веб-сайте электронной коммерции может информировать посетителей о текущих распродажах, обновлениях сайта или доступности сезонных коллекций. Приветственное сообщение устанавливается с помощью модуля оповещения.

Необходимо добавить модуль оповещения в слот **Сообщения об ошибках/информационные сообщения** в фрагменте заголовка. Модуль оповещений позволяет указать отображаемый текст, цвет текста и выравнивание. Он также позволяет указать, могут ли посетители сайта закрыть сообщение.

Когда приветственное сообщение добавляется в фрагмент общего заголовка, оно будет отображаться на каждой странице, использующей шаблон, в котором используется этот общий фрагмент заголовка.

Чтобы добавить приветственное сообщение на сайт, выполните следующие действия.

1. В конфигураторе сайта Commerce перейдите на свой сайт.
1. Выберите **Фрагменты**.
1. Выберите фрагмент заголовка, в который добавляется сообщение.
1. В дереве структуры разверните узел **Сообщения об ошибках/информационные сообщения**.
1. Выберите модуль оповещений, затем выберите **ОК**. Если модуль оповещения еще не существует, сначала выберите кнопку с многоточием (**...**) рядом с пунктом **Сообщения об ошибках/информационные сообщения**, затем выберите **Добавить модуль**.
1. На панели свойств справа на вкладке **Данные** выберите **Добавить источник данных**, затем выберите **Содержимое**.
1. В поле **Введите текст** введите текст приветственного сообщения.
1. Выберите **Сохранить**, выберите **Завершить редактирование** для возврата фрагмента заголовка, затем нажмите кнопку **Опубликовать**, чтобы опубликовать его. 

Приветственное сообщение теперь появится в верхней части каждой страницы сайта, использующей выбранный фрагмент заголовка.
