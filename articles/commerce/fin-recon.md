[!include [banner](includes/banner.md)]

<span data-ttu-id="9906b-105">Например, они могут выполнять декларации платежных средств, закрывать смены вслепую, выверять проводки смены и закрывать смены.</span><span class="sxs-lookup"><span ## <a name="set-up-financial-reconciliation"></a><span data-ttu-id="9906b-119">Настройка финансовой выверки</span><span class="sxs-lookup"><span data-stu-id="9906b-119">Set up financial reconciliation</span></span>

<span data-ttu-id="9906b-120">Выполните следующие действия, чтобы использовать функцию финансовой выверки.</span><span class="sxs-lookup"><span data-stu-id="9906b-120">Follow these steps to use the financial reconciliation functionality.</span></span>

## <a name="more-information-about-financial-reconciliation"></a><span data-ttu-id="9906b-123">Дополнительные сведения о финансовой выверке</span><span class="sxs-lookup"><span data-stu-id="9906b-123">More information about financial reconciliation</span></span>

<span data-ttu-id="9906b-124">При включении функции финансовой выверки некоторые параметры, определенные на экспресс-вкладке **Операция/закрытие** страницы **Розничные магазины** в центральном офисе Commerce, синхронизируются с базой данных канала.</span><span class="sxs-lookup"><span data-stu-id="9906b-124">When the financial reconciliation functionality is turned on, some of the parameters that are defined on the **Statement/closing** FastTab of the **Retail stores** page in Commerce headquarters are synced to the channel database.</span></span> <span data-ttu-id="9906b-125">Применяется тот же набор критериев расчета и пороговых значений сумм, которые используются для журналов операций розничной торговли.</span><span class="sxs-lookup"><span data-stu-id="9906b-125">The same set of calculation criteria and amount thresholds that is used for retail statements is enforced.</span></span>
