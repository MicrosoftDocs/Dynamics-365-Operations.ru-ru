# <a name="crop-images"></a>Кадрирование изображений

[!include [banner](includes/banner.md)]

В этой статье описывается, как кадрировать изображения в конструктор сайта Microsoft Dynamics 365 Commerce.

Библиотека мультимедиа конфигуратора сайтов Commerce позволяет обрезать изображения для оптимизации их для различных типов модулей и окон просмотра.

## <a name="crop-an-image"></a>Кадрирование изображения

Чтобы кадрировать изображение в конфигураторе сайта, выполните следующие действия.

1. В левой области навигации конфигуратора сайта Commerce выберите **Библиотека мультимедиа**.
1. В главном окне выберите изображение, которое необходимо изменить.
1. На панели команд выберите **Правка**.
1. Выберите изображение, чтобы перейти в **Режим правки**.
1. В пункте **Режим правки** выберите **Редактировать представление по модулям**.
1. В раскрывающемся меню **Модуль** выберите тип модуля.
1. В раскрывающемся меню **Тип представления** выберите тип представления.
1. В раскрывающемся меню **Размещение** выберите размещение изображения.
1. В раскрывающемся меню **Окно просмотра** выберите размер окна просмотра.
1. Изображение накладывается на область, представляющую область кадрирования. При необходимости переместите и измените размер области кадрирования. Соотношение сторон будет поддерживаться автоматически.
1. Когда закончите, на панели команд выберите **Сохранить**, затем выберите **Завершить редактирование**. 

После завершения пользовательской обрезки изменения изображения вступят в силу почти немедленно.
