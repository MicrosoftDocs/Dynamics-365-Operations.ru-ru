<span data-ttu-id="639aa-106">Библиотека мультимедиа конфигуратора сайтов Commerce позволяет обрезать изображения для оптимизации их для различных типов модулей и окон просмотра.</span><span class="sxs-lookup"><span data-stu-id="639aa-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="639aa-107">Кадрирование изображения</span><span class="sxs-lookup"><span data-stu-id="639aa-107">Crop an image</span></span>

<span data-ttu-id="639aa-108">Чтобы кадрировать изображение в конфигураторе сайта, выполните следующие действия.</span><span class="sxs-lookup"><span data-stu-id="639aa-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="639aa-109">В левой области навигации конфигуратора сайта Commerce выберите **Библиотека мультимедиа**.</span><span class="sxs-lookup"><span data-stu-id="639aa-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="639aa-110">В главном окне выберите изображение, которое необходимо изменить.</span><span class="sxs-lookup"><span data-stu-id="639aa-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="639aa-111">На панели команд выберите **Правка**.</span><span class="sxs-lookup"><span data-stu-id="639aa-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="639aa-112">Выберите изображение, чтобы перейти в **Режим правки**.</span><span class="sxs-lookup"><span data-stu-id="639aa-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="639aa-113">В пункте **Режим правки** выберите **Редактировать представление по модулям**.</span><span class="sxs-lookup"><span data-stu-id="639aa-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="639aa-114">В раскрывающемся меню **Модуль** выберите тип модуля.</span><span class="sxs-lookup"><span data-stu-id="639aa-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="639aa-115">В раскрывающемся меню **Тип представления** выберите тип представления.</span><span class="sxs-lookup"><span data-stu-id="639aa-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="639aa-116">В раскрывающемся меню **Размещение** выберите размещение изображения.</span><span class="sxs-lookup"><span data-stu-id="639aa-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="639aa-117">В раскрывающемся меню **Окно просмотра** выберите размер окна просмотра.</span><span class="sxs-lookup"><span data-stu-id="639aa-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="639aa-118">Изображение накладывается на область, представляющую область кадрирования.</span><span class="sxs-lookup"><span data-stu-id="639aa-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="639aa-119">При необходимости переместите и измените размер области кадрирования.</span><span class="sxs-lookup"><span data-stu-id="639aa-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="639aa-120">Соотношение сторон будет поддерживаться автоматически.</span><span class="sxs-lookup"><span data-stu-id="639aa-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="639aa-121">Когда закончите, на панели команд выберите **Сохранить**, затем выберите **Завершить редактирование**.</span><span class="sxs-lookup"><span data-stu-id="639aa-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="639aa-122">После завершения пользовательской обрезки изменения изображения вступят в силу почти немедленно.</span><span class="sxs-lookup"><span data-stu-id="639aa-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
