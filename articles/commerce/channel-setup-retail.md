# <a name="set-up-a-retail-channel"></a>Настройка канала розничной торговли


[!include [banner](includes/banner.md)]

В этом разделе описывается, как создать канал розничной торговли в Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Обзор

Dynamics 365 Commerce поддерживает несколько розничных каналов. Эти розничные каналы включают в себя интернет-магазины, центры обработки вызовов и розничные магазины (также называются физическими магазинами). Каждый канал розничного магазина может иметь свои собственные способы оплаты, группы цен, контрольно-кассовые машины POS, счета выручки и расходов и сотрудников. Необходимо настроить все эти элементы перед тем, как можно создать канал розничной торговли. 

Перед созданием канала розничной торговли убедитесь, что выполнены [необходимые условия для канала](channels-prerequisites.md).

## <a name="create-and-configure-a-new-retail-channel"></a>Создание и настройка нового канала розничной торговли

1. В области переходов перейдите **Модули \> Каналы \> Магазины \> Все магазины**.
1. В области действий выберите **Создать**.
1. Вставьте имя для нового канала в поле **Имя**.
1. В поле **Номер магазина** выберите уникальный номер магазина. Это число может состоять из букв и цифр и не больше 10 символов.
1. В раскрывающемся списке **Юридическое лицо** введите соответствующее юридическое лицо.
1. В раскрывающемся списке **Склад** введите соответствующий склад.
1. В поле **Часовой пояс магазина** выберите нужный часовой пояс.
1. В раскрывающемся списке **Налоговая группа** выберите подходящую налоговую группу для магазина.
1. В поле **Валюта** выберите подходящую валюту.
1. В поле **Адресная книга клиентов** укажите допустимую адресную книгу.
1. В поле **Клиент по умолчанию** укажите допустимый клиент по умолчанию.
1. В поле **Профиль функциональности** выберите профиль функциональности, если это применимо.
1. В поле **Профиль уведомлений по электронной почте** укажите допустимый профиль уведомлений по электронной почте.
1. На панели операций выберите **Сохранить**.

## <a name="additional-channel-set-up"></a>Дополнительная настройка канала

Имеются дополнительные элементы, которые необходимо настроить для канала, который можно найти на **панели операций** в разделе **Настройка**.

Дополнительные задачи, необходимые для настройки интернет-канала, включают настройку способов оплаты, декларацию наличных денег, способов поставки, приходный/расходный счет, разделы, назначения группы выполнения и безопасности.

На следующем рисунке показаны дополнительные параметры настройки канала розничной торговли на вкладке **Настройка**.

![Настройка канала](media/channel-setup-retail-4.png)

### <a name="set-up-payment-methods"></a>Настройка способов оплаты

Для настройки способов оплаты для каждого типа платежа, поддерживаемого в этом канале, выполните следующие действия.

1. На панели операций выберите вкладку **Настройка**, а затем выберите **Способы оплаты**.
1. В области действий выберите **Создать**.
1. В области переходов выберите нужный способ оплаты.
1. В разделе **Общие** введите **Имя операции** и настройте другие необходимые элементы.
1. Настройте дополнительные параметры, необходимые для типа оплаты.
1. На панели операций выберите **Сохранить**.

На следующем рисунке показан пример способы наличной оплаты.

![Пример способов оплаты](media/channel-setup-retail-5.png)

### <a name="set-up-cash-declaration"></a>Настройка декларирования наличности

1. На панели операций выберите вкладку **Настройка**, а затем выберите **Декларирование наличности**.
1. В области действий выберите **Создать**, а затем создайте все номиналы **Монета** и **Банкнота**, которые используются.

На следующем рисунке показан пример способы декларирования наличности.

![Настройка декларирования наличности](media/channel-setup-retail-6.png)

### <a name="set-up-modes-of-delivery"></a>Настройка способов поставки

Настроенные способы поставки можно просмотреть, выбрав **Способы поставки** на вкладке **Настройка** на **панели операций**.  

Чтобы изменить или добавить способ поставки, выполните следующие действия.

1. В области переходов перейдите **Модули \> Управление запасами \> Режимы поставки**.
1. В области действий выберите **Создать**, чтобы создать новый способ поставки, или выберите существующий режим.
1. В разделе **Каналы розничной торговли** выберите **Добавить строку**, чтобы добавить канал. Добавление каналов с помощью узлов организации вместо добавления каждого канала по отдельности может ускорить добавление каналов.

На следующем рисунке показан пример режима поставки.

![Настройка способов поставки](media/channel-setup-retail-7.png)

### <a name="set-up-incomeexpense-account"></a>Настройка приходного/расходного счета

Чтобы настроить приходный/расходный счет, сделайте следующее.

1. На панели операций выберите вкладку **Настройка**, а затем выберите **Приходный/расходный счет**.
1. В области действий выберите **Создать**.
1. В поле **Имя** введите имя.
1. В области **Краткое наименование** введите краткое наименование.
1. В **Тип счета** введите тип счета.
1. Введите текст для **Строка сообщения 1**, **Строка сообщения 2**, **Текст квитанции 1** и **Текст квитанции 2**, как необходимо.
1. В области **Разноска** введите сведения о разноске.
1. На панели операций выберите **Сохранить**.

На следующем рисунке показан пример счета приходного/расходного счета.

![Настройка приходных/расходных счетов](media/channel-setup-retail-8.png)

### <a name="set-up-sections"></a>Настройка разделов

Чтобы настроить разделы, выполните следующие шаги.

1. На панели операций выберите вкладку **Настройка**, а затем выберите **Разделы**.
1. В области действий выберите **Создать**.
1. В поле **Номер раздела** введите номер раздела.
1. В **Описание** введите описание.
1. В поле **Размер раздела** введите размер раздела.
1. При необходимости настройте дополнительные параметры для **Общее** и **Статистика по продажам**.
1. На панели операций выберите **Сохранить**.

### <a name="set-up-a-fulfillment-group-assignment"></a>Настройка назначения группы выполнения

Чтобы настроить назначение группы выполнения, выполните следующие действия.

1. На панели операций выберите вкладку **Настройка**, а затем выберите **Назначение группы выполнения**.
1. В области действий выберите **Создать**.
1. В раскрывающемся списке **Группа выполнения** выберите группу выполнения.
1. В **Описание** введите описание.
1. На панели операций выберите **Сохранить**

На следующем рисунке показан пример настройки назначения группы выполнения.

![Настройка назначений группы выполнения](media/channel-setup-retail-9.png)

### <a name="set-up-safes"></a>Настройка сейфов

Чтобы настроить сейфы, выполните следующие шаги.

1. На панели операций выберите вкладку **Настройка**, а затем выберите **Сейфы**.
1. В области действий выберите **Создать**.
1. Введите имя сейфа.
1. На панели операций выберите **Сохранить**.
