# <a name="employee-development-power-bi-content"></a>Содержимое Power BI "Развитие сотрудников"

[!include [banner](../includes/banner.md)]

В этом разделе описывается содержимое Microsoft Power BI **Развитие сотрудников**

## <a name="reports-that-are-included-in-the-power-bi-content"></a>Отчеты, включенные в содержимое Power BI
Отчеты, включенные в пакет содержимое Power BI **Развитие сотрудников**, включают диаграммы и таблицы, которые содержат дополнительные сведения. В следующей таблице приводится описание отчетов.

| Отчет                        | Содержание |
|-------------------------------|----------|
| Обзор развития сотрудников | Сводка по другим отчетам |
| Анализ навыков сотрудника       | Типы навыков сотрудников и навыки сотрудников по типу |
| Анализ уровня навыков сотрудника | Уровни навыков сотрудников по подразделению, сотрудники по уровню и типу навыков, низшие и высшие уровни по навыку |
| Профиль навыков                 | Профиль навыков для выбранного сотрудника |
| Анализ навыков                | Навыки по типам и рейтингу |
| Анализ оценки производительности   | Сотрудники по низшей и высшей оценке по заданию, рейтинги сотрудника по подразделению, сотрудники по рейтингу и типу должности, максимальный и минимальный рейтинг по должности |
| Анализ производительности сотрудника | Рейтинг сотрудника для выбранного менеджером рейтинга |

Можно фильтровать диаграммы и плитки в этих отчетах, а также закреплять диаграммы и плитки на панели мониторинга. Дополнительные сведения о том, как отфильтровать и закрепить элементы в Power BI, см. в разделе [Создание и настройка панели мониторинга](https://powerbi.microsoft.com/en-us/guided-learning/powerbi-learning-4-2-create-configure-dashboards).
