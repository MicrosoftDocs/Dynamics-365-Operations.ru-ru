# <a name="generate-er-reports-by-adding-content-as-raw-xml"></a>Формирование отчетов ER путем добавления содержимого в виде исходного XML-документа

[!include[banner](../includes/banner.md)]

Можно использовать новый элемент формата **ИСХОДНОГО XML-документа** для разработки форматов электронной отчетности (ER), создающих исходящие документы в формате XML. В некоторых случаях требуется добавить исходные данные XML к этим отчетам по одной или нескольким из следующих причин:

- Удобнее использовать исходный XML для оригинального дизайна и текущего обслуживания отчета, поскольку структура XML может быть автоматически создана с помощью выполнения выражения для среды выполнения. Таким образом не требуется определять несколько привязок для нескольких элементов формата во время разработки. Это возможно, когда используемые источники данных содержат сведения, которые можно использовать для XML-элементов, в то время, как формируется отчет. 
- Ни один из других методов не может использоваться для заполнения отчета с XML-содержимым, которое ранее было получено и сохранено в системе. Например XML-ответ, который создается, может потребоваться с содержимым ранее отправленного запроса XML.
- Никакой другой метод не может использоваться для вставки символов в создаваемый документ на основании их числовых кодов. Для некоторых языков и символов коды этого типа не существуют. Примеры включают греческие буквы Ро (ρ) и коды объектов HTML, такие как \&eacute; для *e*, которая имеет знак ударения (é).

> [!NOTE]
> Имейте в виду, что структура не контролирует правильность содержимого XML, помещенного в созданный документ, с помощью элемент формата **ИСХОДНОГО XML**.

Для получения дополнительных сведений о данной функции, воспроизведите проводники по задаче **ER использует исходные данные XML для создания отчетов XML (часть 1: разработка модели данных)** и **ER использует исходные данные XML для создания отчетов XML (часть 2: разработка и выполнение отчета)** которые являются частью бизнес-процесса **7.5.4.3 Приобретение/разработка компонентов ИТ-услуг и решений (10677)** и могут быть загружены из [центра загрузки Майкрософт](https://go.microsoft.com/fwlink/?linkid=874684). Эти задачи проводники по задаче описывают процесс настройки формата ER для вставки исходных данных XML в созданные файлы.
