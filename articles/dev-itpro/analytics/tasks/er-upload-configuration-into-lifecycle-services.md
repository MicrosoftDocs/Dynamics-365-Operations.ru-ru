# <a name="er-upload-a-configuration-into-lifecycle-services"></a>Электронная отчетность — Отправка конфигурации в Lifecycle Services

[!include [task guide banner](../../includes/task-guide-banner.md)]

В следующих шагах поясняется, как пользователь с ролью "Системный администратор" или "Разработчик электронной отчетности" может создать новую конфигурацию электронной отчетности и отправить ее в Microsoft Lifecycle Services (LCS).

В этом примере вам предстоит создать конфигурацию и отправить ее в LCS для демонстрационной компании Litware, Inc. Эти шаги можно выполнить в любой компании, поскольку конфигурации электронной отчетности являются общими для всех компаний. Для выполнения этих шагов необходимо сначала выполнить шаги в процедуре "Создание поставщика конфигурации и установка его в качестве активного". Доступ к LCS также является обязательным условием для выполнения следующих шагов.

1. Перейдите в раздел "Управление организацией" > "Рабочие области" > "Электронная отчетность".
2. Выберите "Litware, Inc." и установите этого поставщика в качестве активного.
3. Щелкните "Конфигурации".

## <a name="create-a-new-data-model-configuration"></a>Создать новой конфигурации модели данных
1. Щелкните "Создать конфигурацию", чтобы открыть ниспадающее диалоговое окно.
    * Вам предстоит создать конфигурацию, которая содержит пример модели данных для электронных документов. Эта конфигурация модели данных будет отправлена в LCS позже.  
2. В поле "Имя" введите "Пример конфигурации модели".
    * Пример конфигурации модели  
3. В поле "Описание" введите "Пример конфигурации модели".
    * Пример конфигурации модели  
4. Нажмите Создать конфигурацию.
5. Щелкните "Конструктор моделей".
6. Щелкните "Создать".
7. В поле "Имя" введите "Точка входа".
    * Точка входа  
8. Нажмите кнопку Добавить.
9. Нажмите кнопку "Сохранить".
10. Закройте страницу.
11. Щелкните "Изменить статус".
12. Щелкните "Завершить".
13. Нажмите кнопку "OК".

## <a name="register-a-new--repository"></a>Регистрация нового репозитория
1. Закройте страницу.
2. Щелкните "Репозитории".
    * Это позволяет открыть список репозиториев для поставщика конфигурации Litware, Inc.  
3. Щелкните "Добавить", чтобы открыть раскрывающееся диалоговое окно.
    * Это позволяет добавить новый репозиторий.  
4. В поле "Тип репозитория конфигурации" выберите "LCS".
5. Щелкните "Создать репозиторий".
6. В поле "Проект" введите или выберите значение.
    * Выберите требуемый проект LCS. Вы должны иметь доступ к проекту.  
7. Нажмите кнопку "OК".
    * Заполните новую запись репозитория.  
8. В списке пометьте выбранную строку.
    * Выберите запись "Репозиторий LCS".  
    * Обратите внимание, что зарегистрированный репозиторий помечен текущим поставщиком, что значит, что только конфигурации, которыми владеет этот поставщик, можно поместить в этот репозиторий и, следовательно, отправить в выбранный проект LCS.  
9. Щелкните Открыть.
    * Откройте репозиторий для просмотра списка конфигураций электронной отчетности. Он будет пустым, если этот проект еще не использовался для совместного использования конфигураций электронной отчетности.  
10. Закройте страницу.
11. Закройте страницу.

## <a name="upload-configuration-into-lcs"></a>Отправка конфигурации в LCS
1. Щелкните "Конфигурации".
2. В дереве выберите "Пример конфигурации модели".
    * Выберите созданную конфигурацию, которая уже завершена.  
3. В списке найдите и выберите требуемую запись.
    * Выберите версию выбранной конфигурации со статусом "Завершено".  
4. Щелкните "Изменить статус".
5. Щелкните "Поделиться".
    * Статус конфигурации изменится с "Завершено" на "Общее" после публикации в LCS.  
6. Нажмите кнопку "OК".
7. В списке найдите и выберите требуемую запись.
    * Выберите версию конфигурации со статусом "Общие".  
    * Обратите внимание, что статус выбранной версии изменился с "Завершено" на "Общее".  
8. Закройте страницу.
9. Щелкните "Репозитории".
    * Это позволяет открыть список репозиториев для поставщика конфигурации Litware, Inc.  
10. Щелкните Открыть.
    * Выберите репозиторий LCS и откройте его.  
    * Обратите внимание, что выбранная конфигурация отображается как актив выбранного проекта LCS.  
    * Откройте LCS с помощью https://lcs.dynamics.com. Откройте проект, который использовался ранее для регистрации репозитория, откройте библиотеку активов данного проекта и разверните содержимое типа актива "Конфигурация GER". Отправленная конфигурация электронной отчетности станет доступна. Обратите внимание, что отправленную конфигурацию LCS можно импортировать в другой экземпляр Microsoft Dynamics 365 for Finance and Operations, Enterprise edition, если поставщики имеют доступ к этому проекту LCS.
