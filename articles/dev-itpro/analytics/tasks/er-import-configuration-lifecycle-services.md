# <a name="import-a-configuration-from-lifecycle-services-for-electronic-reporting-er"></a>Импорт конфигурации из Lifecycle Services для электронной отчетности (ER)

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

В следующих шагах поясняется, как пользователь с ролью "Системный администратор" или "Разработчик электронной отчетности" может импортировать новую конфигурацию электронной отчетности из Microsoft Lifecycle Services (LCS).

В этом примере вам предстоит выбрать желаемую версию конфигурации электронной отчетности для демонстрационной компании Litware, Inc. Эти шаги можно выполнить в любой компании, поскольку конфигурации электронной отчетности являются общими для всех компаний. Для выполнения этих шагов необходимо сначала выполнить шаги в процедуре "Отправка конфигурации электронной отчетности в Lifecycle Services". Доступ к LCS также является обязательным условием для выполнения следующих шагов.

1. Перейдите в раздел "Управление организацией" > "Рабочие области" > "Электронная отчетность".
2. Щелкните "Конфигурации".

## <a name="delete-a-shared-version-of-data-model-configuration"></a>Удаление общей версии конфигурации модели данных
1. В дереве выберите "Пример конфигурации модели".
    * Первая версия конфигурации примера модели данных создана и опубликована в LCS во время выполнения процедуры "Отправка конфигурации электронной отчетности в Lifecycle Services". В этой процедуре мы удалим эту версию конфигурации электронной отчетности. Эта версия примера конфигурации модели данных будет позднее импортирована из LCS.  
2. В списке найдите и выберите требуемую запись.
    * Выберите версию этой конфигурации со статусом "Общие". Этот статус указывает, что конфигурация была опубликована в LCS.  
3. Щелкните "Изменить статус".
4. Щелкните "Отменить".
    * Измените статус выбранной версии с "Общее" на "Отменено", чтобы сделать ее доступной для удаления.  
5. Нажмите кнопку "OК".
6. В списке найдите и выберите требуемую запись.
    * Выберите версию этой конфигурации со статусом "Отменено".  
7. Нажмите кнопку Удалить.
8. Щелкните Да.
    * Обратите внимание, что доступна только черновая версия 2 выбранной конфигурации модели данных.  
9. Закройте страницу.

## <a name="import-a-shared-version-of-data-model-configuration-from-lcs"></a>Импорт общей версии конфигурации модели данных из LCS
1. В списке пометьте выбранную строку.
    * Откройте список репозиториев для Litware, Inc. Litware, Inc.  
2. Щелкните "Репозитории".
3. Щелкните Открыть.
    * Выберите репозиторий LCS и откройте его.  
4. В списке пометьте выбранную строку.
    * Выберите первую версию примере конфигурации модели в списке версий.  
5. Нажмите кнопку Импорт.
6. Щелкните Да.
    * Подтвердите импорт выбранной версии из LCS.  
    * Обратите внимание, что информационное сообщение (над формой) подтверждает успешное завершение импорта выбранной версии.  
7. Закройте страницу.
8. Закройте страницу.
9. Щелкните "Конфигурации".
10. В дереве выберите "Пример конфигурации модели".
11. В списке найдите и выберите требуемую запись.
    * Выберите версию этой конфигурации со статусом "Общее".  
    * Обратите внимание, что общая версия 1 выбранной конфигурации модели данных теперь также доступна.
