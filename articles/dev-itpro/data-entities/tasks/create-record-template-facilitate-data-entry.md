# <a name="create-a-record-template-to-facilitate-data-entry"></a>Создание шаблона записи для облегчения ввода данных

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре демонстрируется, как создать шаблон записи, чтобы значения полей, которые используются часто, не требовалось вводить явно для каждой новой записи. В этой процедуре предстоит создать новую запись для новых ноутбуков, которые должны быть добавлены к ОС. В данной процедуре используется демонстрационная компания USMF.

1. Перейдите в раздел "Основные средства" > "Основные средства" > "Основные средства".
2. Щелкните "Создать".
3. В поле "Группа ОС" введите или выберите значение.
4. В поле "Имя" введите значение.
    * Например, введите "Ноутбук корпоративного интереса".  
5. В поле "Краткое наименование" введите значение.
    * Например, введите "ноутбук".  
6. Разверните раздел "Техническая информация".
7. В поле "Марка" введите значение.
8. В поле "Модель" введите значение.
9. В поле "Год модели" введите значение.
10. В области действий щелкните "Параметры".
11. Щелкните "Сведения о записи".
12. Щелкните "Шаблон пользователя".
13. В поле "Имя" введите значение.
    * Например, введите "Корпоративный ноутбук".  
14. В поле "Описание" введите значение.
    * Например, введите "Корпоративный ноутбук".  
15. Нажмите кнопку "OК".
16. Щелкните "Закрыть".
