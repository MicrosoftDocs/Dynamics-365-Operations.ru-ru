<span data-ttu-id="c280a-104">Microsoft Dynamics 365 Talent: Onboard позволяет обмениваться руководствами и шаблонами по адаптации с другими участниками, чтобы они могли просматривать документы и вносить изменения.</span><span class="sxs-lookup"><span data-stu-id="c280a-104">Microsoft Dynamics 365 Talent: Onboard lets you share onboarding guides and templates with other contributors so that they can review the documents and make changes.</span></span>

## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="c280a-105">Предоставление общего доступа к руководству по адаптации другим участникам</span><span class="sxs-lookup"><span data-stu-id="c280a-105">Share an onboarding guide with other contributors</span></span>

1. <span data-ttu-id="c280a-106">В левом меню выберите **Руководства**, затем выберите руководство, которым вы хотите поделиться.</span><span class="sxs-lookup"><span data-stu-id="c280a-106">On the left menu, select **Guides**, and then select the guide that you want to share.</span></span>
2. <span data-ttu-id="c280a-107">Выберите кнопку **Дополнительно** (три вертикальные точки) в верхнем правом углу страницы, затем выберите **Поделиться**.</span><span class="sxs-lookup"><span data-stu-id="c280a-107">Select the **More** button (the three vertical dots) in the upper-right corner of the page, and then select **Share**.</span></span>

    ![[<span data-ttu-id="c280a-108">Предоставление общего доступа к руководству по адаптации другим участникам</span><span class="sxs-lookup"><span data-stu-id="c280a-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="c280a-109">В разделе **Поделиться** если получатели должны быть в состоянии пересмотреть руководство, установите для параметра **Эти пользователи могут вносить изменения?** значение **Вкл.**.</span><span class="sxs-lookup"><span data-stu-id="c280a-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="c280a-110">Если получатели должны быть уведомлены по электронной почте, что вы поделились руководством с ними, установите для параметра **Отправить уведомление по электронной почте получателям** значение **Вкл.**.</span><span class="sxs-lookup"><span data-stu-id="c280a-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="c280a-111">Задание параметров доступа и электронной почты для руководства по адаптации</span><span class="sxs-lookup"><span data-stu-id="c280a-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="c280a-112">Введите имя или адрес электронной почты и выберите нужного получателя в раскрывающемся списке.</span><span class="sxs-lookup"><span data-stu-id="c280a-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="c280a-113">На вкладке **Управление доступом** можно просмотреть или изменить то, что получатели могут сделать с руководством.</span><span class="sxs-lookup"><span data-stu-id="c280a-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="c280a-114">Чтобы изменить доступ получателя, в раскрывающемся списке рядом с именем получателя, выберите **Могут просматривать**, **Могут изменять** или **Удалить** по мере необходимости.</span><span class="sxs-lookup"><span data-stu-id="c280a-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="c280a-115">После завершения добавления или изменения получателей выберите **Обновить**.</span><span class="sxs-lookup"><span data-stu-id="c280a-115">When you've finished adding or modifying recipients, select **Update**.</span></span>

    <span data-ttu-id="c280a-119">[![Предоставление общего доступа к шаблону другим участникам](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span><span class="sxs-lookup"><span data-stu-id="c280a-119">[![Sharing a template with other contributors](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span></span>

    <span data-ttu-id="c280a-122">[![Задание параметров доступа и электронной почты для шаблона](./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span><span class="sxs-lookup"><span data-stu-id="c280a-122">[![Setting access and email preferences for a template](./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span></span>

4. <span data-ttu-id="c280a-123">Введите имя или адрес электронной почты и выберите нужного получателя в раскрывающемся списке.</span><span class="sxs-lookup"><span data-stu-id="c280a-123">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="c280a-124">На вкладке **Управление доступом** можно просмотреть или изменить то, что получатели могут сделать с шаблоном.</span><span class="sxs-lookup"><span data-stu-id="c280a-124">On the **Manage access** tab, you can review or change what recipients can do with the template.</span></span> <span data-ttu-id="c280a-125">Чтобы изменить доступ получателя, в раскрывающемся списке рядом с именем получателя, выберите **Могут просматривать**, **Могут изменять** или **Удалить** по мере необходимости.</span><span class="sxs-lookup"><span data-stu-id="c280a-125">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
