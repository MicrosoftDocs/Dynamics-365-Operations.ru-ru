# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-october-19-2018"></a>Что нового и что изменилось в Dynamics 365 for Talent Core HR (19 октября 2018 г.)

[!include[banner](includes/banner.md)]

**Сборка 8.1.1067**

В этой теме описываются новые и измененные компоненты Core HR.

## <a name="allow-managers-to-update-time-off-requests"></a>Разрешить руководителям обновлять запросы на отгулы

Запросы на отгулы сотрудников могут требовать обновления после их утверждения через workflow-процесс. Во многих случаях менеджер делает эти обновления от имени сотрудника. Эта новая функция позволяет менеджерам обновлять отгул или отменять запросы отгулов от имени своих сотрудников. Эта возможность управляется параметром **Работать от имени**, который можно установить на странице **Параметры управления персоналом**. 
 
## <a name="allow-hr-to-update-time-off-requests"></a>Разрешить отделу кадров обновлять запросы на отгулы

Аналогично функции выше, запросы сотрудников на отгул может потребоваться обновить в отделе кадров после того как они были утверждены ранее через workflow-процесс. Эта возможность позволяет пользователям отдела кадров обновлять запросы отгулов. Возможность включается в рабочей области **Люди** и на странице **Работник** с помощью нового параметра **Просмотреть отгул**. На этих страницах пользователи отдела кадров могут просматривать запросы и обновлять проводки отгулов, подобно тому как менеджеры могут выполнять эти действия.

Полномочия безопасности, которые контролируют доступ к этой функции, следующие:
- Полномочия: ведение процессов отпусков и отсутствия для определенных работников.
- Привилегия: ведение запросов на отпуск и отсутствие для всех работников.

## <a name="other-changes"></a>Другие изменения
В данном выпуске были сделаны следующие обновления:
- Изменения в действия приема на работу работника, чтобы они больше не "застревали" в состоянии **Workflow-процесс завершен**.
- Теперь можно создать запись о приеме на работу, не указывая дату приема на работу.
- Дата регистрации Dynamics 365 Talent для участия в курсах, отображается функцией самообслуживания сотрудников, теперь учитывает смещение часового пояса в этой дате.
- Файлы Excel можно импортировать несколько раз без ошибок с помощью **Сущности сотрудников**.

## <a name="known-issue"></a>Известная проблема

- **Проблема**: при добавлении нового вложения для работника кнопки **Создать** и **Изменить** неактивны. 
- **Временное решение:** перед открытием страницы вложения убедитесь, что информационные панели на странице **Рабочий** закрыты. Если информационные панели закрыты при загрузке страницы **Работник**, кнопки вложений будут активны. (Эта проблема будет исправлена в следующем обновлении платформы.)
