# <a name="create-a-hiring-team"></a>Создание группы комплектации штата

[!include [banner](includes/banner.md)]

В Microsoft Dynamics 365 Talent: Onboard можно создавать группы комплектации штата. Затем вы можете назначить руководства и шаблоны по адаптации для каждой группы.

## <a name="create-a-hiring-team"></a>Создание группы комплектации штата

1. В левом меню выберите **Группы**.
2. В поле **Группы** выберите плитку **Добавить** (знак плюс \[**+**\]).
3. В диалоговом окне **Создать новую группу** в поле **Имя группы** введите имя группы комплектации штата.

    ![[Создание новой группы в Onboard](./media/onboard-create-team.png)](./media/onboard-create-team.png)

4. В разделе **Выберите участников группы** введите имя или адрес электронной почты каждого участника группы.

    Чтобы удалить участников группы, выберите **X**, который отображается рядом с их именем под полем.

5. Чтобы отправить сообщение электронной почты новым участникам группы, включите параметр **Отправлять сообщение электронной почты новым участникам**.
6. Выберите **Создать**.
7. Если вы указали, что хотите отправить сообщение электронной почты новым участникам группы, измените сообщение электронной почты в следующем диалоговом окне, затем выберите **Готово**.

## <a name="assign-onboarding-guides-to-a-hiring-team"></a>Назначение руководств по адаптации группе комплектации штата

1. В левом меню выберите **Группы**.
2. Выберите группу.
3. На вкладке **Руководства** выберите **Добавить руководства**.

    ![[Добавление руководств по адаптации в группу](./media/onboard-add-guides-to-team.png)](./media/onboard-add-guides-to-team.png)

4. Установите флажок для каждого руководства по адаптации, которое вы хотите назначить группе, затем выберите **Добавить**.

    ![[Выбор руководств по адаптации для добавления в группу](./media/onboard-select-guides.png)](./media/onboard-select-guides.png)

## <a name="assign-onboarding-templates-to-a-hiring-team"></a>Назначение шаблонов по адаптации группе комплектации штата

1. В левом меню выберите **Группы**.
2. Выберите группу.
3. На вкладке **Шаблоны** выберите **Добавить шаблон**.

    ![[Добавление шаблонов в группу](./media/onboard-add-templates-to-team.png)](./media/onboard-add-templates-to-team.png)

4. Установите флажок для каждого шаблона, который вы хотите назначить группе, затем выберите **Добавить**.

    ![[Выбор шаблонов для добавления в группу](./media/onboard-select-templates.png)](./media/onboard-select-templates.png)
