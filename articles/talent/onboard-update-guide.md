# <a name="update-onboarding-guides-in-dynamics-365-for-talent-onboard"></a>Обновление руководства по адаптации в Dynamics 365 for Talent: Onboard

[!include [banner](includes/banner.md)]

Если необходимо внести изменения в руководства по адаптации в Microsoft Dynamics 365 for Talent: Onboard, вы можете обновить их и передать изменения, даже если вы уже отправили руководства. У вас есть два варианта обновления руководства по адаптации:

- Измените руководство по адаптации напрямую и сохраните изменения.
- Измените шаблон по адаптации, затем передайте изменения на все руководства по адаптации, которые были созданы из него.

## <a name="edit-an-onboarding-guide-directly"></a>Изменение руководства по адаптации напрямую

1. В левом меню выберите **Руководства**.
2. Выберите руководство, который хотите изменить.
3. Внесите все необходимые изменения, затем выберите кнопку **Сохранить** (символ диска).

    ![[Сохранение изменений в руководстве по адаптации](./media/onboard-save.png)](./media/onboard-save.png)

Onboard автоматически отправит новому сотруднику сообщение электронной почты со сведениями об изменениях. Для легкой идентификации рядом с каждым изменением появится красный тег **Новое**.

## <a name="update-multiple-guides-by-editing-the-onboarding-template"></a>Обновление нескольких руководств путем редактирования шаблона по адаптации

1. В левом меню выберите **Шаблоны**.
2. В разделе **Мои шаблоны** выберите шаблон, который требуется изменить.
3. Внесите все необходимые изменения, затем выберите кнопку **Сохранить** (символ диска).
4. Чтобы отправить изменения во все руководства, основанные на шаблоне, выберите **Отправить эти изменения**.

    ![[Отправка изменений шаблона по адаптации во все руководства, которые были созданы на его основе](./media/onboard-push-changes.png)](./media/onboard-push-changes.png)

Изменения будут видны новым сотрудникам, открывающим руководства по адаптации. Однако приложение Onboard не будет отправлять оповещения по электронной почте новым сотрудникам, чтобы сообщить им об изменении их руководств по адаптации. Для легкой идентификации рядом с каждым изменением появится красный тег **Новое**.
