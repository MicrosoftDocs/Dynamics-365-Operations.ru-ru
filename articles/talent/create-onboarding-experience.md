# <a name="get-started-with-onboard"></a>Начало работы с Onboard

[!include [banner](includes/banner.md)]

С помощью Microsoft Dynamics 365 for Talent: Onboard вы можете помочь новым сотрудникам работать более эффективно и получить чувство принадлежности к их новым ролям. Эффективный процесс адаптации помогает новым сотрудникам понять свои новые роли. Это также дает им фору в установлении ценных отношений, которые помогут обеспечить их успех.

Onboard позволяет быстро создать руководство по адаптации, которое будет приветствовать ваших новых сотрудников, чтобы они были рады присоединиться к вашей организации.
