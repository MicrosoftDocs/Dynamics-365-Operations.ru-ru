# <a name="manage-preview-features"></a>Управление предварительными версиями функций

[!include[banner](../includes/banner.md)]

В рамках нашего непрерывного процесса развертывания возможностей управление человеческим капиталом (HCM) для Microsoft Dynamics 365 Talent мы хотим предоставить клиентам возможность испытать новые функции. Администраторы могут видеть и использовать функции предварительного просмотра в средах. Общедоступная версия этих функций почти готова, и они прошли множество тестов. Мы хотели бы получить окончательные отзывы клиентов и выполнить проверку до выпуска общедоступной версии.

В этом разделе описывается, как можно включить функции предварительного просмотра, и перечисляются функции, которые в настоящее время доступны для предварительного просмотра. Этот список будет обновляться по мере выпуска общедоступной версии функций и выпуска новых функций для предварительного просмотра. Уведомление не отправляется при выпуске новых функций для предварительного просмотра. Функции просто отобразятся для пользователей. Дополнительные сведения о новых возможностях в Talent см. в разделах [Что нового и что изменилось в Dynamics 365 Talent](./whats-new.md) и [Заметки о выпуске Dynamics 365 и Power Platform](https://docs.microsoft.com/business-applications-release-notes).

## <a name="enable-or-disable-preview-features"></a>Включение и отключение функций предварительного просмотра

Чтобы получить доступ к предварительным версиям функций, необходимо сначала включить их в своей среде. Включение или отключение функций предварительного просмотра зависят от среды.

> [!IMPORTANT]
> При включении параметра **Функции предварительного просмотра** вы включаете функции предварительного просмотра для всех пользователей в организации, имеющихся в данной среде. При отключении параметра вы отключаете функции предварительного просмотра и делаете их недоступными для пользователей. Функции предварительного просмотра имеют ограниченную поддержку в Talent. К ним могут применяться меньше требований к соблюдению конфиденциальности и безопасности, и они не включаются в соглашение об уровне обслуживания (SLA) Talent. Не следует использовать функции предварительного просмотра для обработки личных данных (то есть любой информации, которая может идентифицировать пользователя) или для обработки других данных, на которые действуют юридические или нормативные требования.

### <a name="attract"></a>Attract

1. Войдите в Microsoft Dynamics 365 Talent: Attract.
2. В меню **Настройка** (символ шестеренки) в правом верхнем углу выберите **Центр администрирования**.
3. На вкладке **Управление функциями** выберите вариант рядом с параметром **Функции предварительного просмотра**, чтобы он стал синим и содержат текст **Вкл.**.

    ![Включение предварительных версий функций в Attract](./media/attract-enable-preview-features.png)

4. Выберите или отмените выбор отдельных функций предварительного просмотра. Если ничего не сделать, включаются все доступные функции предварительного просмотра.
5. Обновите браузер, чтобы отобразить новые функции. Все пользователи, которые уже выполнили вход в систему, увидят функции при следующем входе. Также они могут обновить браузер, чтобы отобразить функции немедленно.

> [!NOTE]
> Некоторые функции предварительного просмотра могут потребовать дополнительной настройки. Для завершения настройки щелкните ссылки рядом с функцией для предварительного просмотра.

### <a name="core-hr"></a>Core HR

1. Войдите в Talent.
2. Выберите **Администрирование системы**, затем перейдите на вкладку **Ссылки**.
3. На странице **Администрирование системы** в разделе **Настройка** выберите пункт **Системные параметры**.
4. На странице **Системные параметры** выберите вкладку **Предварительные версии функций**.
5. Установите для параметра **Включить режим предварительного просмотра для всех пользователей** значение **Да**, чтобы сделать доступными предварительные версии функций.

    ![Включение предварительных версий функций в Core HR](./media/corehr-enable-preview-features.png)

> [!NOTE]
> Чтобы отключить предварительные версии функций, выполните те же шаги, но установите для параметра **Включить режим предварительного просмотра для всех пользователей** значение **Нет**. При отключении функций предварительного просмотра они становятся недоступными для пользователей, и могут произойти ошибки в процессах, которые связаны с функциями.

### <a name="onboard"></a>Адаптация

В настоящее время предварительные версии функций недоступны для Microsoft Dynamics 365 Talent: Onboard.

## <a name="features-that-are-currently-in-preview"></a>Функции, которые в данный момент находятся в режиме предварительного просмотра

### <a name="attract"></a>Attract

- [Рекомендация кандидата](./intelligent-recommendations.md#candidate-recommendations) — если имеется более десяти кандидатов, для которых имеются резюме или полные профили, кандидаты, наиболее точно соответствующие требованиям для должности, отображаются в разделе **Кандидаты для рассмотрения** на странице этой должности.
- [Рекомендация вакансии](./intelligent-recommendations.md#job-recommendations) — если на сайте карьеры опубликовано более десяти вакансий, Attract предоставляет рекомендации по вакансиям для кандидатов.
- [Интеграция Broadbean](./posting-jobs-external.md#post-jobs-to-broadbean) — можно публиковать вакансии из Attract на Broadbean, внешнем сайте для публикации вакансий. После включения этой предварительной версии функции необходимо завершить настройку, введя имя пользователя Broadbean, идентификатор клиента и маркер шифрования.
- [Аналитика](./analytic-reports.md) —в Центре аналитики группы найма могут просматривать ключевые метрики для одной должности плюс агрегированные показатели по всем должностям.
- [EEO](./activities-attract.md) — новые типы действий позволяют использовать предопределенную форму для сбора у кандидатов данных по формам Equal Employment Opportunity (EEO) и Office of Federal Contract Compliance Program (OFCCP). Изменение предварительно определенной формы невозможно.
- [Рекомендация по соискателям](./intelligent-recommendations.md#prospect-recommendations) — Attract проверяет прошлых кандидатов и текущих кандидатов, чтобы предоставить список перспективных кандидатов, которые хорошо подходят для вашей должности.
- [Поиск по релевантности](./attract-talent-pools.md#search-and-view-candidate-profiles) — можно выполнить поиск по отдельным навыкам, названиям или уровням образования во всей базе данных кандидатов. Attract производит поиск по всему профилю и выделяет все найденные совпадения. Attract также выполняет поиск во всех документах, которые доступны для кандидата, и разумно ранжирует результаты поиска.
- [Аудитория мероприятия](./whats-new-talent-march-20.md#setting-the-audience-on-activities) — можно настроить аудиторию для мероприятий (таких как собеседование, график или обратная связь), задав **Все кандидаты**, **Внешние кандидаты** или **Внутренние кандидаты**. Действия заказчиков, такие как видео YouTube, веб-содержимое и Microsoft Forms, могут доставляться всем кандидатам, только внутренним кандидатам, только внешним кандидатам или группе по найму на работу.
- [Подать заявление с помощью LinkedIn](./career-site.md#enable-applying-for-jobs-with-linkedin-profiles) — можно настроить параметр на сайте карьеры Attract, чтобы кандидаты могли подавать заявления на должность с помощью LinkedIn. Эта функция упрощает процесс подачи заявления для кандидатов, позволяя им использовать свой профиль LinkedIn для автоматического заполнения своих заявлений на сайте карьеры.
- [Отслеживание источника](./source-tracking.md) — Attract отслеживает источник заявлений кандидатов с целью предоставления ценных сведений, которые могут помочь направить ваши усилия по найму работников. Можно также выбрать источник заявления, когда вручную добавляется кандидат на должность или в кадровый пул.
- [Серебряный медалист](./whats-new-talent-march-20.md#designate-silver-medalists-to-assign-high-value-applicants-for-future-positions) — если какие-либо кандидаты являются отличными для вашей организации, но предложение на них не было расширено в соответствии с текущей должностью, их можно назначить как серебряных медалистов. Эта функция позволяет сократить время найма в следующий раз, когда имеется аналогичная вакансия.

### <a name="core-hr"></a>Core HR

- [Проверка данных иерархии должностей](./whats-new-talent-may-13-2019.md#new-page-to-validate-position-hierarchy-data) — можно проверять иерархию управления на наличие любых циклических ссылок, которые были случайно импортированы.
- [Указание кодов причин для типов отпусков](./whats-new-talent-may-13-2019.md#specify-reason-codes-on-leave-types) — можно указать коды причин для типов отпусков.
- [Требовать коды причин для запросов отгулов](./whats-new-talent-may-13-2019.md#require-reason-codes-for-specific-leave-types-on-time-off-requests) — в дополнение к указаниям кодов причин для типов отпусков можно потребовать коды причин для запросов отгулов.
- [Предоставление отпуска и списка проводок отсутствия для отдела кадров](./whats-new-talent-may-13-2019.md#provide-a-leave-and-absence-transaction-list-for-hr) — можно просмотреть список проводок отпусков и отсутствия, чтобы понять баланс времени отгулов.

### <a name="onboard"></a>Адаптация
