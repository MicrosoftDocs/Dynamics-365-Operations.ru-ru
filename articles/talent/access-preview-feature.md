# <a name="access-preview-features-in-talent"></a>Доступ к функциям предварительного просмотра в Talent

[!include[banner](../includes/banner.md)]

В рамках нашего непрерывного процесса выгрузки возможностей продукта мы хотим предоставить клиентам возможность испытать новые функции. Администраторы могут видеть и использовать функции предварительного просмотра в средах. Общедоступная версия этих функций почти готова, и они прошли множество тестов. Мы хотели бы получить окончательные отзывы клиентов и выполнить проверку до выпуска общедоступной версии.

В этом разделе описывается, как администратор может включить функции предварительного просмотра, и перечисляются функции, которые в настоящее время доступны для предварительного просмотра. Этот список будет обновляться по мере выпуска общедоступной версии функций и выпуска новых функций для предварительного просмотра. Уведомление не отправляется при выпуске новых функций для предварительного просмотра. Функции просто отобразятся для пользователей.

## <a name="enable-or-disable-preview-features"></a>Включение и отключение функций предварительного просмотра

Можно использовать параметр **Функции предварительного просмотра** в центре администрирования Microsoft Dynamics 365 for Talent, чтобы включить или отключить функции предварительного просмотра. По умолчанию они отключены. Действие включения или отключения функций предварительного просмотра зависят от среды.

> [!IMPORTANT]
> При включении параметра **Функции предварительного просмотра** вы включаете функции предварительного просмотра для всех пользователей в организации, имеющихся в данной среде. Отключив параметр, вы отключаете функции предварительного просмотра и делаете их недоступными для пользователей. Функции предварительного просмотра имеют ограниченную поддержку в Talent. К ним могут применяться меньше требований к соблюдению конфиденциальности и безопасности, и они не включаются в соглашение об уровне обслуживания Talent. Не следует использовать функции предварительного просмотра для обработки личных данных (то есть любой информации, которая может идентифицировать пользователя) или для обработки других данных, на которые действуют юридические или нормативные требования.

### <a name="enable-or-disable-preview-features-for-your-organization"></a>Включение или отключение функций предварительного просмотра для организации

#### <a name="attract"></a>Привлечение

1. Войдите в Microsoft Dynamics 365 for Talent: Attract.
2. В меню **Настройка** (символ шестеренки) в правом верхнем углу выберите **Параметры администрирования**.
3. На вкладке **Управление функциями** выберите вариант рядом с параметром **Функции предварительного просмотра**, чтобы он стал синим.
4. При необходимости можно управлять отдельными функциями, включая и отключая определенные функции на этой странице.
5. Обновите браузер, чтобы отобразить новые функции. (Все пользователи, которые уже выполнили вход в систему, увидят функции при следующем входе. Также они могут обновить браузер, чтобы отобразить функции немедленно.)

#### <a name="core-hr"></a>Core HR

1. Войдите в Talent. Откроется рабочая область core HR, в которой следует выполнить оставшиеся шаги. 
2. Выберите **Администрирование системы \> Ссылки Системные параметры**.
3. На странице **Системные параметры**, на вкладке **Функции предварительного просмотра** задайте для параметра **Включить режим предварительного просмотра для всех пользователей** значение **Да**, чтобы предоставить доступ к функциям предварительного просмотра.

> [!NOTE]
> Чтобы отключить функции предварительного просмотра, выполните те же базовые шаги. При отключении функций предварительного просмотра они становятся недоступными для пользователей, и могут произойти ошибки в процессах, которые связаны с функциями.

## <a name="features-that-are-currently-in-preview"></a>Функции, которые в данный момент находятся в режиме предварительного просмотра

### <a name="attract"></a>Привлечение

- **Соответствующие кандидаты в должности** — агенты по найму кадров и менеджеры по найму могут легко выяснить, какие кандидаты могут быть наиболее подходящими для должности из всех кандидатов. Отображаются 5 лучших кандидатов на основе соответствия их резюме или профиля описанию должности.
- **Соответствующие должности** — кандидаты теперь видят список других должностей, имеющих отношение к ним на основе их резюме или профиля и описания вакансий.  В настоящее время это отображается для кандидатов при подаче заявления в качестве предложения для других возможностей.
- **Поддержка EEO/OFCCP** — новые типы действий позволяют использовать предопределенную форму для сбора у кандидатов данных по формам Equal Employment Opportunity (EEO) и Office of Federal Contract Compliance Program (OFCCP).  Это стандартная форма, которую невозможно изменить.

    > [!NOTE]
    > Размещенные объявления о приеме на работу отображаются только для клиентов, которые подписаны на один или несколько продуктов объявлений о вакансии LinkedIn. В противном случае клиенты увидят должность, только если они выполнят явный поиск должности. При публикации объявлений о приеме на работу в LinkedIn возникает задержка. После публикации должности из Attract ее отображение может занять до нескольких часов.

- **Подача заявления о приеме на работу кандидатом** — внутренние и внешние кандидаты теперь могут подать заявление о приеме на работу на странице должностей на сайте карьеры.
- **Управление предложениями** — пользователи теперь могут создавать письма с предложением из шаблонов, содержащих заполнители. Когда кандидаты переходят на стадию предложения, специалисты по найму и специалисты по комплектации штата могут использовать средство Offer для подготовки формального предложения кандидата с помощью шаблонов, отправки предложения на внутреннее утверждение и, наконец, отправки предложения кандидату на подпись. Со временем в Offer будет добавлено много новых возможностей, и функция предварительного просмотра будет обновлена для включения этих возможностей по мере их готовности к выпуску для предварительного просмотра.

### <a name="core-hr"></a>Core HR

- **Открытая регистрация** — благодаря открытой регистрации сотрудники могут самостоятельно и легко выбрать льготы. Администраторы модуля "Управление персоналом" могут настроить процесс открытой регистрации льгот для организации и возможность регистрации для сотрудников с помощью простого управляемого решения.
