# <a name="extend-the-functionality-of-microsoft-dynamics-365-for-talent"></a>Расширение функциональных возможностей Microsoft Dynamics 365 for Talent

[!include[banner](includes/banner.md)]

Если вы создали какие-либо приложения Microsoft PowerApps, вы можете запускать эти приложения с помощью ссылок в Microsoft Dynamics 365 for Talent. Чтобы настроить доступ к приложениям, необходимо настроить некоторую информацию на странице конфигурации Talent, которую можно открыть из рабочей области **Администрирование системы**.

## <a name="configuring-embedded-powerapps-within-talent"></a>Настройка внедренных приложений PowerApps в Talent
Используйте страницу **Настройка встроенных приложений Microsoft PowerApps**, чтобы настроить страницы Talent для запуска приложений PowerApps. Чтобы открыть страницу **Настройка встроенных приложений Microsoft PowerApps**, откройте рабочую область **Администрирование системы**, затем откройте вкладку **Ссылки**. Выберите **Microsoft PowerApps** в группе **Настройка**. 

На этой странице вводится или задается следующая информация: 

> - Описательное имя или идентификатор для каждого приложения PowerApps.
> - Уникальный идентификатор (GUID) для каждого приложения, которое добавляется на страницу Talent. Идентификатор приложения доступен на сайте PowerApps, [powerapps.com](http://powerapps.com/). 
> - Страница, с которой пользователи могут открыть приложение или отчет. Не все страницы Talent поддерживают внедренные приложения PowerApps и отчеты Power BI. 

 > [!NOTE]
 >  Вводите внутреннее имя страницы, а не отображаемое имя, которое отображается в верхней части страницы. Чтобы найти внутреннее имя, откройте страницу, внутреннее имя которой необходимо определить, и щелкните правой кнопкой мыши в любом месте на странице. В открывшемся меню наведите указатель мыши на пункт **Сведения о форме**. Внутреннее имя формы отображается рядом с пунктом **Сведения о форме** в меню.
 
> - Укажите элемент управления формы, из которого приложение может получить контекстные данные. Например, приложение может использовать данные о работнике. Если ввести страницу **Работник** в поле **Контекст**, при запуске приложения будет открываться страница **Работник**. Заполнять **Поле контекста** не обязательно. 
> - Задайте размер диалогового окна, в котором будет выполняться приложение PowerApps. Диалоговые окна обозначаются как "небольшое" или "большое", чтобы оптимизировать интерфейс пользователя, когда приложение выполняется на телефоне или крупном устройстве, соответственно. 

Можно также указать, каким юридическим лицам будет доступно приложение, или можно сделать приложение доступным для всех юридических лиц. По умолчанию ваши приложения PowerApps доступны для всех юридических лиц.

## <a name="opening-an-application"></a>Открытие приложения
После настройки встроенных приложений PowerApps ссылки на указанные вами приложения добавляются на страницы в Talent. Щелкните ссылку, чтобы открыть приложение.
