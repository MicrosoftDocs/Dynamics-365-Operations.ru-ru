# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Что нового и что изменилось в Dynamics 365 Talent (17 декабря 2019 г.)

В этой статье описываются новые и измененные компоненты в Dynamics 365 Talent.

## <a name="changes-in-attract"></a>Изменения в Attract

Этот выпуск содержит исправления незначительных ошибок для Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Изменения в Onboard

Этот выпуск содержит исправления незначительных ошибок для Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Изменения в Core HR

Изменения, описанные в этом разделе, относятся к сборке номер 8.1.2674. Числа в скобках в некоторых заголовках относятся к номерам поддержки в службе Microsoft Dynamics Lifecycle Services (LCS).

### <a name="print-performance-reviews"></a>Печать оценок производительности

Печать обзоров производительности теперь доступна во всех средах. Для включения этой функциональности используйте рабочее пространство **управления функциями**. Дополнительные сведения об управлении функциями см. в разделе [Обзор управления функциями](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Для получения дополнительной информации об обзорах см. [Оценки производительности](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Позиция выбытия с прямыми отчетами "плавает" на организационной схеме как открытая позиция (385436)

В выпуске этой недели исправлено, как позиции выбытия с прямыми отчетами отображаются в иерархии отчетов.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>Система не в состоянии создавать календарные дни (381195) 

Изменения на этой неделе исправляют проблему, в которой календарные дни не могут быть созданы.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>Текущий отчет о балансе в отпуске PBI удваивает баланс сотрудников (385581)

Это изменение исправляет текущий отчет о балансе в этом сценарии.
