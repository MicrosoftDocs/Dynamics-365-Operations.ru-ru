# <a name="create-a-working-time-calendar"></a>Создание календаря рабочего времени


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Календарь рабочего времени в Dynamics 365 Human Resources отображает дни и часы, которые сотрудники работают в вашей организации. Когда сотрудник отправит запрос об отсутствии, ему не придется беспокоиться о праздниках и нерабочих днях.

Чтобы упростить запросы отсутствия, настройте эти элементы для организации:

- Календарь рабочего времени
- Праздники и нерабочие дни
- Нерабочее время

Можно добавить последние два элемента во время настройки календаря рабочего времени. Можно также настраивать и обновлять их отдельно.

## <a name="set-up-a-working-time-calendar"></a>Настройка календаря рабочего времени

Настройте по крайней мере один календарь рабочего времени, который показывает дни и часы работы. Если имеются местоположения в нескольких странах и регионах, может потребоваться настроить календарь рабочего времени для каждой области.

1. На странице **Администрирование организации** щелкните **Календари**.

2. Выберите **Создать**, а затем введите название и описание календаря.

3. В **Параметры создания** выберите рабочие дни для организации и введите рабочее время. 
   - Чтобы добавить праздник или нерабочий день, нажмите кнопку **Добавить** рядом **Праздники и нерабочие дни**.
   - Чтобы добавить время, не являющееся рабочим временем, например обеды или перерывы, выберите **Добавить** в **НЕРАБОЧЕЕ ВРЕМЯ** и введите имя и диапазон времени.

4. В **Дни** выберите **Создать**, чтобы сформировать дни в вашем календаре. Введите диапазон дат для календаря, а затем выберите **Создать дни**.

5. Чтобы добавить рабочие расписания в **График работы**, выберите **Добавить**, а затем введите значения времени для каждого графика работы.

## <a name="configure-holidays-and-closures"></a>Настройка праздников и нерабочих дней

Праздники и нерабочие дни можно добавлять или изменять отдельно от календаря рабочего времени.

1. На странице **Администрирование организации** щелкните **Праздники и нерабочие дни**.

2. Выберите **Создать**, а затем введите название и дату праздника или нерабочего дня.

## <a name="configure-non-work-time"></a>Настройка нерабочего времени

Нерабочее время можно добавлять или изменять отдельно от календаря рабочего времени.

1. На странице **Администрирование организации** выберите **НЕРАБОЧЕЕ ВРЕМЯ**.

2. Выберите **Создать** и введите имя и диапазон времени для нерабочего времени.

Если вы включили функцию предварительного просмотра корректировок для государственных праздников для отпуска и отсутствия, Human Resources даты праздников и нерабочих дней для определения количества дней для корректировки для сотрудников, зарегистрированных в календаре.
