# <a name="configure-skills"></a>Настройка навыков

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Можно отслеживать навыки работника в Dynamics 365 Human Resources. Также можно указывать навыки, необходимые для определенной должности.

Примеры навыков, которые можно отслеживать:

- Способность контролировать - Умение контролировать работу других.
- Лидерство -Умение возглавить сотрудников и деловые домены.
- Планирование — умение предвидеть, формулировать визуальные утверждения и принимать решения на их основании.
- HTML — Возможность создавать код HTML.

Если типы навыков и модели рейтинга еще не настроены, перед созданием навыков необходимо добавить некоторые.

Следующие сотрудники могут ввести навыки для сотрудника:

- Работники могут вводить навыки для себя в системе самообслуживания сотрудников. Эти навыки требуют утверждения руководителем.
- Менеджеры могут вводить навыки для своих работников. Можно создать рабочий процесс, который автоматически утверждает эти навыки.

## <a name="create-a-skill-type"></a>Создание типа навыков

Типы навыков представляют собой категории, под которые попадают индивидуальные навыки, например, администрирование или продажи.

1. В рабочей области **Развитие сотрудников** выберите **Ссылки**.

2. В области **Настройка компетенции** выберите **Типы навыков**.

3. Выберите **Создать**.

4. Заполните следующие поля:

   - **Тип навыка**: введите название для типа навыка.
   - **Описание**: введите описание для типа навыков.

5. Нажмите **Сохранить**.

## <a name="create-a-rating-model"></a>Создание модели рейтинга

Модели рейтинга помогают оценить фактического уровня навыков лица, уровень, для которого они должны работать для достижения, или уровень навыка, необходимого для должности. Каждому уровню в модели рейтинга назначается коэффициент.

1. В рабочей области **Развитие сотрудников** выберите **Ссылки**.

2. В области **Настройка компетенции** выберите **Модели рейтинга**.

3. Выберите **Создать**.

4. Заполните следующие поля:

   - **Рейтинг**: введите имя модели рейтинга, например **Навыки**.
   - **Описание**: введите описание для модели рейтинга, например **Рейтинги навыков**.

5. В разделе **Уровни** выберите **Создать**. Для каждого уровня, который требуется добавить, заполните следующие поля:

   - **Уровень**: введите имя для уровня.
   - **Описание**: введите описание для уровня.
   - **Коэффициент**: введите коэффициент, равный 0–9. Коэффициенты помогают нормировать оценки навыков, использующих разные модели рейтингов. Каждый уровень должен иметь уникальный коэффициент. Уровни с более высокими значениями коэффициента имеют больший вес в модели рейтинга.

   Продолжайте добавлять уровни по мере необходимости. Вы можете ввести до 10 уровней для каждой модели рейтингов.

6. Нажмите **Сохранить**.

## <a name="create-a-skill"></a>Создание навыка

Перед назначением навыка или созданием поиска сопоставления навыков или профиля навыков, необходимо ввести информацию о навыках на странице **Навыки**. Для каждого навыка можно выбрать тип навыков и модель рейтинга.

1. В рабочей области **Развитие сотрудников** выберите **Ссылки**.

2. В области **Настройка компетенции** выберите **Навыки**.

3. Выберите **Создать**.

4. Заполните следующие поля:

   - **Навык**: введите название для навыка.
   - **Описание**: введите описание для навыка.
   - **Рейтинг**: введите модель рейтинга, которую хотите использовать для этого навыка.
   - **Тип навыка**: выберите из списка типов навыков.

5. Нажмите **Сохранить**.
