# <a name="custom-fields"></a>Настраиваемые поля

К страницам в Dynamics 365 Human Resources можно добавлять настраиваемые поля. Дополнительные сведения о способе создания настраиваемых полей на странице см. в разделе [Создание настраиваемых полей и работа с ними](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Следующие объекты Common Data Service теперь поддерживают настраиваемые поля, созданные в Human Resources:

- Рабочий
- Этническое происхождение
- Статус ветерана
- Код языка
- Должность
- Тип вакансии
- Должностные функции
- Занимаемая должность
- Тип позиции

Убедитесь, что все созданные настраиваемые поля активированы, чтобы их можно было использовать в Common Data Service. Дополнительные сведения см. в разделе [Предоставление настраиваемых полей для информационных объектов](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
