# <a name="custom-fields"></a>Настраиваемые поля

К страницам в Dynamics 365 Human Resources можно добавлять настраиваемые поля. Дополнительные сведения о создании настраиваемых полей см. в разделе [Создание настраиваемых полей и работа с ними](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Большинство объектов Common Data Service поддерживают настраиваемые поля. Полный список объектов Common Data Service см. в разделе [Объекты Common Data Service](https://docs.microsoft.com/dynamics365/human-resources/hr-developer-entities). 

Убедитесь, что все созданные настраиваемые поля активированы, чтобы их можно было использовать в Common Data Service. Дополнительные сведения см. в разделе [Предоставление настраиваемых полей для информационных объектов](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
