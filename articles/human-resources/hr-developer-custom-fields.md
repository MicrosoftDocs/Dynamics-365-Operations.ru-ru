# <a name="custom-fields"></a>Настраиваемые поля

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

К страницам в Dynamics 365 Human Resources можно добавлять настраиваемые поля. Дополнительные сведения о создании настраиваемых полей см. в разделе [Создание настраиваемых полей и работа с ними](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Большинство таблиц Dataverse поддерживают пользовательские поля. Полный список таблиц Dataverse см. в разделе [Таблицы Dataverse](https://docs.microsoft.com/dynamics365/human-resources/hr-developer-entities). 

Убедитесь, что все созданные настраиваемые поля активированы, чтобы их можно было использовать в Dataverse. Дополнительные сведения см. в разделе [Предоставление настраиваемых полей для информационных объектов](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
