# <a name="skill-level-type"></a>Тип уровня навыка


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этой статье описывается набор параметров "Тип уровня навыка" для Dynamics 365 Human Resources.

Физическое имя: mshr_hrmskillleveltype

Это перечисление предоставляет набор параметров со значениями, которые классифицируют уровень, назначенный навыку кандидата.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Фактический | Фактический уровень навыка. |
| 200000001 | Цель | Целевое значение для уровня навыка. |
