# <a name="create-a-new-benefit"></a>Создать новую льготу


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этой задаче показано, как создать элементы льгот, которые будут использоваться при создании новой льготы. В качестве компании с демонстрационными данными для создания этой задачи используется USMF. Эта задача предназначена для менеджера по компенсациям и льготам.


## <a name="create-benefit-elements"></a>Создание элементов льготы

1. Перейдите в раздел **Управление персоналом \> Льготы \> Настройка \> Элементы льготы**.
2. Выберите **Создать**.
3. В поле **Тип** введите имя создаваемого типа льготы.
4. В поле **Описание** введите значение.
5. В поле **Одновременная регистрация** выберите вариант.

    Чтобы ограничить возможность сотрудников регистрироваться в нескольких медицинских планах, установите флажок **Одна регистрация для каждого типа**.

6. В поле **Категория по зарплате** выберите вариант.
7. На вкладке **Планы** выберите **Создать**.
8. В поле **План** введите значение.
9. В поле **Описание** введите значение.
10. В поле **Тип** введите или выберите значение.
11. В поле **Влияние на зарплату** выберите вариант.
12. Нажмите **Сохранить**.

## <a name="create-a-benefit"></a>Создание льготы

1. Перейдите в раздел **Управление персоналом \> Льготы \> Льготы**.
2. Выберите **Создать**.
3. В раскрывающемся диалоговом окне в поле **План** введите или выберите значение.
4. В поле **Параметр** введите или выберите значение.
5. В поле **Действует** введите дату и время.
6. Выберите **Создать льготу**.
