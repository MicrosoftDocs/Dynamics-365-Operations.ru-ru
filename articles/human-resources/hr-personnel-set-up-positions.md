# <a name="set-up-positions"></a>Настройка должностей


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



Должности являются важным элементом нижнего уровня организационной иерархии. Позиция — это индивидуальный экземпляр должности. Например, должность «Менеджер по продажам (восток)» — это одна из должностей, связанных с общей должностью, «Менеджер по продажам». Должность существует в подразделении, и ей можно назначить только одного работника. В этой задаче рассматриваются шаги, которые необходимо выполнить для создания должности. Эта процедура предназначена для специалиста по управлению персоналом.

1. Выберите **Управление трудовыми ресурсами**.
2. Выберите **Открытые позиции**.
3. Выберите **Создать**, чтобы открыть раскрывающееся диалоговое окно.
4. В поле **Должность** введите или выберите значение.

    Поля **Должностные обязанности**, **Должность** и **Коэффициент занятости эквивалента полной занятости** автоматически копируются из выбранной должности в позицию.

5. Выберите **Создать позицию**.
6. В поле **Подразделение** введите или выберите значение.
7. В поле **Тип должности** введите или выберите значение.
8. В поле **Регион компенсации** введите или выберите значение.

    Поле **Регион компенсации** определяет правила приемлемости компенсации и бюджеты фиксированных увеличений, которые применяются к сотруднику на данной должности.

9. В поле **Доступна для назначения** введите дату и время.
10. Разверните раздел **Период должности**.

    Период должности вводится по умолчанию на основании дат активации и выбытия, которые были введены ранее.

11. Разверните раздел **Отчитывается перед должностью**.

    При назначении работника на должность, подотчетную другой должности, создается связь непосредственного подчиненного между работниками, назначенными на две должности.

12. Выберите **Создать**, чтобы открыть раскрывающееся диалоговое окно.
13. В поле **Отчитывается перед** введите или выберите значение.
14. Выберите **Создать**.
15. Разверните раздел **Назначение работника**.
16. Разверните раздел **Отношения**.

    Если ваша организация использует матричную иерархию или другую специальную иерархию, можно настроить типы иерархии должностей и затем добавить отношения подотчетности для должностей для каждого настроенного типа иерархии.

17. Выберите **Добавить**.
18. В списке пометьте выбранную строку.
19. В поле **Имя иерархии** введите или выберите значение.
20. В поле **Отчитывается перед должностью** введите или выберите значение.
21. Разверните раздел **Заработная плата**.
22. В поле **Цикл оплаты** введите или выберите значение.
23. В поле **Кем оплачивается** введите или выберите значение.
24. В поле **Обычные часы за год** введите число.

    Введенное значение является количеством регулярно оплачиваемых часов, в течение которых работник на данной должности должен работать каждый год.

25. Разверните раздел **Профсоюз**.
26. Сверните раздел **Профсоюз**.
27. Разверните раздел **Финансовые аналитики**.
28. В поле **Шаблон распределения** введите или выберите значение.
29. В поле **Подразделение** введите или выберите значение.
30. Нажмите **Сохранить**.
