# <a name="reset-stuck-batch-jobs"></a>Сброс застрявших пакетных заданий

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

## <a name="issue"></a>Расход

Microsoft Dynamics 365 Human Resources может столкнуться с проблемами с пакетными заданиями, которые застряли в состоянии **Выполнение** или **Отмена** и не завершаются.

## <a name="resolution"></a>Приказ

Когда пакетное задание зависает в состоянии **Выполняется** или **Отмена**, можно сбросить статус, принудительно отменив задание. После отмены пакетного задания его можно сбросить, установив для него статус **Ожидание**. Затем оно будет снова отобрано для выполнения в следующем запланированном пакетном запуске.

1. В рабочей области **Администрирование системы** выберите страницу **Ссылки**, затем выберите **Пакетные задания**.

2. На странице списка **Пакетное задание** выберите задание, которое необходимо сбросить.

3. На ленте действий выберите **Принудительная отмена** и подтвердите действие.

   > [!NOTE]
   > Действие **Принудительная отмена** доступно только в том случае, если выбранное пакетное задание имеет статус **Выполнение** или **Отмена**, но для данного задания не запущены процессы пакетного выполнения или отмены.

4. На ленте действий выберите **Изменить статус**.

5. На странице **Выбор нового состояния** выберите **Ожидание**, затем выберите **ОК**.
