# <a name="request-a-leave-of-absence"></a>Запрос на отсутствие

>[!Important]
>Функции, перечисленные в этой статье, в настоящее время доступны для клиентов в отдельной версии Dynamics 365 Human Resources. Некоторые или все функции будут доступны в составе будущего выпуска в инфраструктуре Finance после выпуска Finance 10.0.26.


[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Можно отправить запрос об отсутствии и просмотреть статус своих запросов на отпуск в Dynamics 365 Human Resources.

## <a name="request-a-leave-of-absence"></a>Запрос на отсутствие

1. В рабочей области **Самообслуживания сотрудников** выберите **Еще** (...) на плитке **Время отсутствия**.

2. Для отправки запроса об отсутствии выберите **Запрос об отсутствии**.

3. Введите сведения для **Тип отпуска**, **Дата начала** и **Дата окончания**.

4. Если необходимо отправить какую-либо подтверждающую документацию, выберите **Отправить** в **Вложения**.

5. При необходимости введите данные **Комментарий**.

6. Выберите **Отправить**, когда будете готовы отправить запрос. В противном случае выберите **Сохранить черновик**.


## <a name="view-leave-of-absence-request-status"></a>Просмотр статуса запроса на отсутствие

1. В рабочей области **Самообслуживания сотрудников** выберите **Еще** (...) на плитке **Время отсутствия**.

2. Для просмотра своих запросов об отсутствии выберите **Просмотр отсутствия**.

## <a name="update-a-leave-of-absence-request"></a>Обновление запроса на отпуск

1. В рабочей области **Самообслуживания сотрудников** на плитке **Отпуск** выберите **Еще (...)**.
2. Выберите запрос на отпуск для обновления, а затем выберите **Обновить отпуск**.
3. В поле **Дата окончания** обновите значение, если это необходимо, чтобы увеличить или сократить отпуск.
4. Если конечная дата подтверждена, установите для параметра **Подтверждать дату окончания** значение **Да**.
5. После того как параметру **Подтверждать дату окончания** задано значение **Да**, можно загрузить уведомление о возврате на работу. Затем установите флажок, чтобы подтвердить, что уведомление о возврате на работу было отправлено.
6. Выберите **Отправить**, чтобы обновить запрос на отпуск.

## <a name="cancel-a-leave-of-absence-request"></a>Отмена запроса на отпуск

1. В рабочей области **Самообслуживания сотрудников** на плитке **Отпуск** выберите **Еще (...)**.
2. Выберите запрос на отпуск для отмены, а затем выберите **Обновить отпуск**.
3. Установите параметру **Отменить отпуск** значение **Да**.
4. Выберите **Отправить**, чтобы отменить запрос на отпуск.

## <a name="importing-leave-requests-from-other-systems-or-older-systems"></a>Импорт запросов на отпуск из других систем или старых систем

Чтобы импортировать запросы на отпуск из другой системы, необходимо пройти обычный рабочий процесс, чтобы создать соответствующие проводки отпусков. В качестве альтернативы можно импортировать банковские проводки отпусков и запросы отпусков в завершенном состоянии. Обратите внимание, что банковские проводки отпусков не создаются автоматически, если импортируются только запросы на отпуск.
