# <a name="develop-a-succession-plan"></a>Разработка плана преемственности


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

По мере роста организации следует начать планирование преемственности. При планировании преемственности может потребоваться найти человека, который имеет навыки, аналогичные навыкам другого лица. Модуль подбора персонала позволяет проанализировать существующих сотрудников и кандидатов, чтобы узнать, обладают ли они похожим набором навыков с ценным сотрудником. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Перейдите в раздел **Управление персоналом > Компетенции > Анализ навыков > Профили подбора персонала**.
2. Выберите **Создать**.
3. В поле **Подбор персонала** введите имя подбора персонала. Пример: "Сотрудник".
4. В поле **Описание** введите значение.
5. В поле **Дата** введите дату.
6. Выберите **Выбор профиля**.
7. Выберите **Физическое лицо**.
8. В поле **Физическое лицо** введите имя или откройте раскрывающийся список.
9. Нажмите **ОК**.
10. Разверните экспресс-вкладку **Сертификаты**, чтобы просмотреть или изменить сертификаты, включенные в подбор персонала.
11. Разверните экспресс-вкладку **Навыки**, чтобы просмотреть или изменить навыки, которые требуется включить.
12. В списке пометьте первую строку. Пример: "Учет".
13. Выберите флажок **Необязательно**.
14. В поле **Важность** выберите вариант. Если навык отмечен как необязательный, необходимо выбрать уровень важности навыка.  
15. В списке выберите строку 2.
16. Выберите флажок **Необязательно**.
17. В поле **Важность** выберите вариант.
18. В списке выберите строку 3.
19. Выберите флажок **Необязательно**.
20. В поле **Важность** выберите вариант.
21. В списке выберите строку 4.
22. Выберите флажок **Необязательно**.
23. В поле **Важность** выберите вариант.
24. Разверните экспресс-вкладку **Образование**, чтобы просмотреть или изменить компетенции образования, которые необходимо включить в подбор персонала.
25. Выберите **Выполнить**.
26. Нажмите **ОК**.
27. Выберите **Результат**.
28. Выберите **Отчет**. Лучшие совпадения будут перечислены вверху списка в отчете. В списке также указывается элемент несоответствия. Несоответствие — это разница между уровнем сопоставлений навыков и уровнем навыков сотрудника.
