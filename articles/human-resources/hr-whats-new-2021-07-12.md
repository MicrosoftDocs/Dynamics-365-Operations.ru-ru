# <a name="whats-new-or-changed-in-dynamics-365-human-resources-july-12-2021"></a>Что нового и что изменилось в Dynamics 365 Human Resources 12 июля 2021 г.

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этой статье описываются новые, измененные или ожидаемые в ближайшее время функции в Dynamics 365 Human Resources.

Дополнительные сведения о нашем процессе обновления и графике см. в разделе [Процесс обновления](hr-admin-setup-update-process.md).

Дополнительные сведения о новых функциях и ожидаемых датах общей доступности см. в разделе [Обзор Dynamics 365 Human Resources 2021](/dynamics365-release-plan/2021wave1/human-resources/dynamics365-human-resources/).

## <a name="in-this-release"></a>В данном выпуске

Этот выпуск включает следующие новые функции и исправления ошибок. Изменения применяются для номера сборки 8.1.4353.

### <a name="new-features"></a>Новые возможности

В этой версии следующие функции стали общедоступными.

| Функция | План выпуска | Документация |
| --- | --- | --- |
|  Переключатель отображения стажа | |Эта функция позволяет использовать разное число дат для расчета стажа, который отображается на странице **Упрощенный вход сотрудников** и на странице **Сотрудники**.  Он будет доступен в [параметрах Human Resources](/dynamics365/human-resources/hr-setup-parameters). |


### <a name="bug-fixes"></a>Исправления ошибок

Этот выпуск содержит следующие исправления ошибок.

> [!NOTE]
> Наша цель — предоставить эту информацию вам как можно скорее. Мы можем обновить эту статью для включения в нее исправлений ошибок, которые были сделаны в сборке после первоначальной публикации этой статьи.

| Номер проблемы | Проблема |  Описание |
| --- | --- | --- |
| 595871 | Панель "Сведения" в модуле Human Resources имеет неверную терминологию Dataverse | После изменения бренда Common Data Service на Dataverse была обновлена терминология на информационной панели Microsoft Dynamics 365 Human Resources (**Справка и поддержка > Сведения**). |
| 598676 | Задача переопределения упрощенной формы записи сотрудников может вызывать ошибку при использовании с сохраненным представлением| На странице **Работник** если включена функция "Упрощенная запись сотрудника", в приложении могут возникнуть сбои, если для сохраненного представления установлено значение **Всегда открывать для редактирования**. |
| 592344 |Раздел компенсации работников на должности должен быть доступен только для чтения, если активировано управление льготами | Сведения о компенсации сотрудников создаются с помощью старых функциональных возможностей льгот.  Если включено управление льготами, поля будут доступны только для чтения. Когда включено управление льготами и для параметра **Скрыть старые формы льгот** задано значение **Да** в общих параметрах HR, вкладка **Компенсация работникам** будет скрыта. |
| 598617 | Вкладка активации формы **HcmDiscussion** вызывает бесконечный цикл при применении личных настроек | Если в обоих представлениях сетки и сведений включен параметр **Всегда открывать для редактирования**, код активации вкладки в переопределенном методе задачи конфликтует с персонализацией в том, какой элемент управления должен иметь фокус, и создает бесконечный цикл. |
| 593553 | Поле даты журнала в журнале производительности отображается в формате UTC | Поле **Дата журнала** для журналов производительности отображается в часовом поясе UTC вместо часового пояса, определенного в настройке системной даты, что приводит к неправильному заданию даты для некоторых часовых поясов. |
| 586930 | При открытии действий для целей производительности открывается совершенно другая запись | Когда функция "Представление расширенных отчетов журнала производительности" включена в модуле управления функциями, выбор целей производительности для расширенных отчетов на вкладке **Моя рабочая группа** в службе **Самообслуживание сотрудников** открывает цели для некоторого сотрудника, а не для выбранного сотрудника. |
| 569959 |  Объект HcmPositionWorkerAssignmentV2 не назначает сотрудника на должность | Пользователи получали сообщение об ошибке при добавлении записи назначения должности через сущность, и публикация заканчивалась сбоем. |
| 582259 | В отчете VETS 4212 используется форма 2017 вместо формы 2020 | Обновлено до формата 2020.  Чтобы загрузить новый формат, перейдите в **Настройки отчетов** и удалите отчет VETS-4212 в левом столбце. Выберите **Электронная отчетность - Репозитории - Ресурсы HR** и выберите **Открыть**.  Выберите **PDF-распечатка VETS-4212**, затем выберите **Импорт**.|
