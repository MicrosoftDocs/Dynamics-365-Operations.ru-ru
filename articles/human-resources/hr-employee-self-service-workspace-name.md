# <a name="change-employee-self-service-workspace-name"></a>Изменение имени рабочей области самообслуживания сотрудников

Если имеются добровольцы или другие лица, не являющиеся сотрудниками, вам может потребоваться изменить имя рабочей области **Самообслуживание сотрудников**. Имя этой рабочей области можно изменить на **Самообслуживание**.

> [!NOTE]
> При изменении имени рабочей области **Самообслуживание сотрудников** также изменяется пункт меню, используемый для внутренних целей Dynamics 365 Human Resources. Если ранее настройки безопасности были применены к пункту меню **HcmEmployeeSelfServiceWorkspace**, мы рекомендуем применить те же изменения в **HcmSelfServiceWorkspace** для сохранения единства.

1. В модуле Human Resources выберите **Управление персоналом**, выберите **Ссылки**, затем выберите **Параметры Human Resources**.

2. Выберите вкладку **Самообслуживание сотрудников**.

3. В поле **Отображаемое имя** выберите **Самообслуживание**.

   ![Изменение имени рабочей области самообслуживания сотрудников на самообслуживание](./media/hr-employee-self-service-workspace-name.png)

4. Нажмите **Сохранить**.
