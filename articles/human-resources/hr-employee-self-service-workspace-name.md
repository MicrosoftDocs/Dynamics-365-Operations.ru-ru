# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="b5d36-103">Изменение имени рабочей области самообслуживания сотрудников</span><span class="sxs-lookup"><span data-stu-id="b5d36-103">Change Employee self service workspace name</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="b5d36-104">Если имеются добровольцы или другие лица, не являющиеся сотрудниками, вам может потребоваться изменить имя рабочей области **Самообслуживание сотрудников**.</span><span class="sxs-lookup"><span data-stu-id="b5d36-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="b5d36-105">Имя этой рабочей области можно изменить на **Самообслуживание**.</span><span class="sxs-lookup"><span data-stu-id="b5d36-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="b5d36-106">При изменении имени рабочей области **Самообслуживание сотрудников** также изменяется пункт меню, используемый для внутренних целей Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="b5d36-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="b5d36-107">Если ранее настройки безопасности были применены к пункту меню **HcmEmployeeSelfServiceWorkspace**, мы рекомендуем применить те же изменения в **HcmSelfServiceWorkspace** для сохранения единства.</span><span class="sxs-lookup"><span data-stu-id="b5d36-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="b5d36-108">В модуле Human Resources выберите **Управление персоналом**, выберите **Ссылки**, затем выберите **Параметры Human Resources**.</span><span class="sxs-lookup"><span data-stu-id="b5d36-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

2. <span data-ttu-id="b5d36-109">Выберите вкладку **Самообслуживание сотрудников**.</span><span class="sxs-lookup"><span data-stu-id="b5d36-109">Select the **Employee self-service** tab.</span></span>

3. <span data-ttu-id="b5d36-110">В поле **Отображаемое имя** выберите **Самообслуживание**.</span><span class="sxs-lookup"><span data-stu-id="b5d36-110">Under **Display name**, select **Self service**.</span></span>

   ![Изменение имени рабочей области самообслуживания сотрудников на самообслуживание](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="b5d36-112">Нажмите **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="b5d36-112">Select **Save**.</span></span>
