В этой статье описывается набор параметров "Тип трудоустройства" для Dynamics 365 Human Resources.

## <a name="option-set-employment-type"></a>Набор параметров: Тип трудоустройства

Это перечисление предоставляет набор параметров типа трудоустройства для сотрудника. Оно доступно в наборе параметров mshr_hcmemploymenttype.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Код сотрудника | Код сотрудника |
| 200000001 | Подрядчик | Подрядчик |
