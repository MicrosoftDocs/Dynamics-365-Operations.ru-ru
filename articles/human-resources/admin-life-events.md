# <a name="set-administrator-rights-for-life-events"></a>Установка прав администратора для жизненных событий

[!include [banner](../includes/preview-banner.md)]

Администраторы могут обновлять параметры жизненных событий с учетом настроек конфигурации. На странице **Параметры управления персоналом** можно настроить параметры, позволяющие администраторам вносить изменения в выбранные планы. Имеется также возможность полностью запретить администраторам вносить изменения.

На странице **Параметры управления персоналом** можно настроить ограничения на изменение плана для утвержденных планов и параметров жизненных событий.

Если выбран параметр **Подтвержденные планы**, изменения можно вносить только в том случае, если период регистрации активен. (Примерами периодов регистрации могут служить открытые периоды регистрации, периоды регистрации нового набора сотрудников и периоды регистрации жизненных событий.) Если этот параметр не выбран, изменения могут выполняться в любой момент.

Если выбран параметр **Параметры жизненного события**, изменения плана во время жизненного события ограничиваются параметрами жизненного события, определенными для данного типа плана. Если этот параметр не выбран, выбранные планы можно изменять во время жизненного события.

## <a name="set-plan-change-restrictions"></a>Установка ограничений на изменение плана

На странице **Параметры управления персоналом** на вкладке **Управление льготами** имеются перечисленные ниже поля.

| Поле | Описание |
|-------|-------------|
| Подтвержденные планы | Выберите этот параметр, если изменения в плане разрешены только в том случае, если период регистрации активен. Если этот параметр не выбран, изменения могут вноситься в любой момент. |
| Параметры жизненного события | Выберите этот параметр, если изменения плана во время жизненного события ограничиваются параметрами жизненного события, определенными для данного типа плана. Если этот параметр не выбран, выбранные планы можно изменять во время жизненного события. |
