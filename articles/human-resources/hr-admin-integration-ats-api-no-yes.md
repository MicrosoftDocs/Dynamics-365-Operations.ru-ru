## <a name="option-set-no-yes"></a>Набор параметров: Нет Да

Физическое имя: mshr_noyes

Это перечисление предоставляет набор параметров для типовых логических свойств в виртуальных сущностях Human Resources.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Нет | № п/п |
| 200000001 | Да | Да. |
