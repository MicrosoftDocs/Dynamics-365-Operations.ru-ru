# <a name="whats-new-or-changed-in-dynamics-365-human-resources-march-08-2021"></a>Что нового и что изменилось в Dynamics 365 Human Resources 08 марта 2021 г.

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этой теме описываются новые, измененные и ожидающиеся компоненты в Dynamics 365 Human Resources.

Дополнительные сведения о нашем процессе обновления и графике см. в разделе [Процесс обновления](hr-admin-setup-update-process.md).

Дополнительные сведения о новых функциях и ожидаемых датах общей доступности см. в разделе [Обзор Dynamics 365 Human Resources 2021](/dynamics365-release-plan/2021wave1/human-resources/dynamics365-human-resources/).

## <a name="in-this-release"></a>В данном выпуске

Этот выпуск включает следующие новые функции и исправления ошибок. Изменения применяются для номера сборки 8.1.4017.

### <a name="new-features"></a>Новые возможности

В этой версии следующие функции стали общедоступными.

| Функция | План выпуска | Документация |
| --- | --- | --- |
| Представление отпусков в нескольких компаниях для менеджеров | [Представление отпусков сотрудников в нескольких компаниях для менеджеров](/dynamics365-release-plan/2020wave2/human-resources/dynamics365-human-resources/cross-company-view-employee-leave-managers) | [Настройка параметров отпусков и отсутствий](./hr-leave-and-absence-parameters.md) |

### <a name="bug-fixes"></a>Исправления ошибок

Этот выпуск содержит следующие исправления ошибок.

> [!NOTE]
> Наша цель — предоставить эту информацию вам как можно скорее. Мы может обновить этот раздел для включения исправлений ошибок, которые были сделаны в сборке после первоначальной публикации этого раздела.

| Номер проблемы | Расход |  описание |
| --- | --- | --- |
| 486611 | Отпуск отображается в календаре отпусков, если включено **Отключить отпуск для всех календарей** | Если **Отключить отпуск для всех календарей** включено, сведения об отпуске не отображаются, если включены усовершенствования календаря отпусков и отсутствия.|
| 508972 | В банковской проводки отпуска и отсутствия отсутствует проверка регистрации | Когда используется объект "Банковская проводка отпуска и отсутствия ", сотрудники, не зарегистрированные в плане, больше не могут быть импортированы. |
| 554854 | Обновление календаря в объекте "Занятость" приводит к ошибке, если в параметрах пользователя по умолчанию используется другое юридическое лицо | Использование объекта занятости для обновления календаря для сотрудника больше не выдает сообщение об ошибке, если указано другое юридическое лицо по умолчанию в параметрах пользователя. |
| 558347 | При загрузке начальной страницы появляется "Не удается создать запись в конфигурациях форм (FormRunConfiguration)". | Персонализации вызывают ошибку при загрузке начальной страницы: "Не удается создать запись в конфигурациях форм (FormRunConfiguration)". |
| 557327 | Рабочая область управления льготами: пробел над сеткой. | Исправлена проблема в пользовательском интерфейсе, когда есть непредусмотренный проблем в границах сетки таблицы в рабочей области "Льготы". |
| 557334 | Рабочая область управления льготами: раскрывающийся список выбора **Период** должен отображаться только на вкладке **Сводка** | Раскрывающееся меню выбора **Период** льгот появляется теперь только в том случае , если вкладка **Сводка** активна в рабочей области льгот, а не в разделах **Результаты обработки** и **Ссылки**. |
| 557336 | Рабочая область управления льготами: текст **Открытая регистрация с извлеченными планами** обрезается в представлении плитки | Текст в представлении плитки изменен на **Извлеченные планы... открытая регистрация**, чтобы избежать усечения необходимого контекста. |
