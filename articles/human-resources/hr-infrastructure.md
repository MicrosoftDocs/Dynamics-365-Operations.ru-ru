# <a name="dynamics-365-human-resources-infrastructure-merge---release-10025-update"></a>Слияние инфраструктуры Dynamics 365 Human Resources — обновление выпуска 10.0.25

Выпуск 10.0.25 привносит первую волну возможностей при объединении инфраструктуры. Во время объединения инфраструктуры приложение Microsoft Dynamics 365 Human Resources будет объединено с инфраструктурой для финансов и операций. Однако оно будет по-прежнему лицензироваться как независимое приложение, как Dynamics 365 Finance и Dynamics 365 Supply Chain Management. Дополнительные сведения о слиянии инфраструктуры см. в разделе [Вопросы и ответы по слиянию инфраструктуры Dynamics 365 Human Resources](../human-resources/hr-infrastructure-merge-faq.md).

Слияние обеспечит согласованность для пользователей Human Resources следующими способами:

- [Управление средой и их интеграция согласованы между приложениями Human Resources и приложениями для финансов и операций](/dynamics365-release-plan/2021wave2/human-resources/dynamics365-human-resources/consistent-environment-management-integrations-between-human-resources-finance-operations-apps)

    - Управление средами в Microsoft Dynamics Lifecycle Services, поиск проблем и Regression Suite Automation Tool.
    - Имеется единая база кода, в которой новая функциональность приложения Human Resources выпускается в рамках регулярного процесса обновления одной версии.
    - Способ, которым обновления и исправления применяются к средам, согласован.

- [Улучшены параметры расширяемости.](/dynamics365-release-plan/2021wave2/human-resources/dynamics365-human-resources/improve-extensibility-options)

    - В случае необходимости можно продолжить использование инструментов Microsoft Power Platform Tools для расширения.
    - Функциональность можно расширить, используя формы, таблицы, методы и интерфейсы прикладного программирования (API).
    - Можно создавать и расширять сущности.

    Дополнительные сведения о доступных параметрах расширения см. в разделе [Обзор расширения в Dynamics 365](../fin-ops-core/dev-itpro/extensibility/extensibility-home-page.md).

- [Создание одного набора возможностей Human Resources в Dynamics 365.](/dynamics365-release-plan/2021wave2/human-resources/dynamics365-human-resources/create-one-set-human-resources-capabilities-within-dynamics-365)

    В выпуске 10.0.25 функциональные возможности, существовавшие только в приложении Human Resources, стали доступны в инфраструктуре для финансов и операций. Чтобы клиенты могли воспользоваться преимуществами этих возможностей в средах для финансов и операций, в управлении функциями должны быть включены следующие функции Human Resources.

    | Имя функции | Обзор | Статус использования | 
    |--------------|----------|----------------| 
    | Расчет количества лет на службе | Параметр настройки позволяет выбрать дату, которая будет использоваться для расчета значения параметра **Стаж**. | Общедоступно | 
    | Улучшения в работе workflow-процессов | Эта функция позволяет улучшить взаимодействие с пользователем для отправок и утверждений рабочих процессов. | Общедоступно | 
    | Печать оценок производительности | Можно печатать обзоры производительности в формате PDF. | Общедоступно | 
    | Пользовательские ссылки на **Портале самообслуживания менеджеров** | Можно создать пользовательские ссылки, которые появляются в разделе **Связанные ссылки** на **Портале самообслуживания менеджеров**. | Общедоступно | 
    | Представление межфирменной компенсации | Пользователи могут просматривать планы компенсаций на **Портале самообслуживания менеджеров** по всем юридическим лицам без необходимости смены компаний. | Общедоступно | 
    | Настройка нескольких уровней компенсации по должности\*&dagger; | Должности теперь поддерживают уровней компенсации. | Предварительный просмотр | 
    | Управление задачами\* | Можно создавать контрольные списки и задачи для процесса адаптации, увольнения и перевода. | Предварительный просмотр | 
    | Упрощенный ввод данных о сотрудниках | Эта функция обеспечивает обновленный интерфейс пользователя на существующей странице **Работник**. | Предварительный просмотр | 
    | Улучшения пользовательского интерфейса управления персоналом | См. таблицу в следующем разделе.  | Предварительный просмотр | 

\* Эта функция должна быть включена до функции **Улучшения пользовательского интерфейса Human Resources**.

&dagger; После включения этой функции отключить ее невозможно.

## <a name="human-resource-user-experience-enhancements"></a>Улучшения пользовательского интерфейса Human Resources

| Имя функции | Обзор | 
|--------------|----------| 
| Удалить трудоустройство | Можно удалить занятость сотрудника. | 
| Группы должностей | Можно отслеживать группу должностей, которая включает аналогичные работы, которые требуют аналогичных обучения, навыков, знаний и опыта. | 
| Дополнительные поля занятости | Добавлены следующие поля: **Категория трудоустройства**, **Тип трудоустройства** и **Статус сотрудника**. | 
| Страница **Работники без трудоустройства** | На странице отображается список сотрудников, не имеющих записи о трудоустройстве. | 
| Обновление интерфейса пользователей аналитики должности | Имеется улучшенный интерфейс пользователя для назначения аналитик должности по юридическим лицам. | 
| Изменения адресов в рабочей области **Управление персоналом** | Эта функция предоставляет количество всех изменений адресов, произошедших в течение указанного количества дней, определенного на странице **Параметры Human Resources**. | 
| Записи с истекшим сроком действия в рабочей области **Управление персоналом** | Эта функция предоставляет список элементов, срок действия которых истек или истечет для сертификатов, идентификации, испытаний, фильтров или проверок. | 
| Страница **Проверка иерархии должностей** | Проверка выполняется на наличие циклических ссылок в иерархии строк должностей. | 
| Сведения о заработной плате, специфичные для страны/региона | Дополнительные поля заработной платы доступны на странице **Прием на работу работника**, в зависимости от страны или региона юридического лица, в котором работают работники. | 
| Усовершенствования в отчетности по соответствию требованиям | Доступны дополнительные возможности отчетности для EEO-1, Vets 4212 и OSHA300a. | 
| Обновления рабочей области **Управление персоналом** | Были сделаны обновления для отслеживания изменений адресов и записей с истекающим сроком действия. Кроме того, на новых вкладках перечислены действия сотрудника и должности. |
