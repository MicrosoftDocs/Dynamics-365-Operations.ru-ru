# <a name="create-an-open-ended-leave-of-absence"></a>Создание отпуска с открытой датой

> [!IMPORTANT]
> Функции, описанные в этой статье, будут доступны в составе будущего выпуска в инфраструктуре Finance после выпуска Microsoft Dynamics 365 Finance 10.0.31.

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Можно отправить запросы на отпуск с открытой датой и просмотреть статус своих запросов на отпуск в Dynamics 365 Human Resources.

## <a name="prerequisites"></a>Необходимые условия

1. В разделе **Управление функциями** убедитесь, что включена функция **Отпуск с открытой датой**.

    > [!IMPORTANT]
    > Эту функцию невозможно отключить после ее включения.

2. Создайте отпуск типа "отпуск".
3. Введите такие сведения, как тип отпуска, описание и идентификатор рабочего процесса.
4. В поле **Тип запроса** выберите **Отпуск**.
5. В разделе сведений для отпусков с открытой датой установите для параметра **С открытой датой** значение **Да**.
6. Установите для параметра **Уведомление о возврате на работу** значение **Да** или **Нет**.
7. Уведомление о возврате на работу может дополнительно потребоваться для запросов на отпуск с открытой датой.

> [!NOTE]
> После включения этой функции будет включена функция **Требуется вложение**.

## <a name="request-an-open-ended-leave-of-absence"></a>Запрос отпуска с открытой датой

1. В рабочей области **Самообслуживания сотрудников** на плитке **Время отсутствия** выберите **Еще (...)**.
2. Для отправки запроса об отсутствии выберите **Запрос об отсутствии**.
3. Введите сведения для полей **Тип отпуска** и **Дата начала**. В поле **Дата окончания** введите ориентировочную дату окончания.
4. Если необходимо отправить какую-либо подтверждающую документацию, в разделе **Вложения** выберите **Отправить**.
5. Если вы готовы отправить запрос, выберите **Отправить**. В противном случае выберите **Сохранить черновик**.
6. Чтобы обновить запрос на отпуск с открытой датой, выберите запрос, введите конечную дату в поле **Дата окончания**, установите для параметра **Подтверждать дату окончания** значение **Да**, а затем отправьте документацию.
7. Если для параметра **Уведомление о возврате на работу** установлено значение **Да**, выберите **Отправить**, а затем установите флажок, чтобы подтвердить, что действительное уведомление о возврате на работу было отправлено.
8. После того как все данные введены, нажмите **Отправить**.
