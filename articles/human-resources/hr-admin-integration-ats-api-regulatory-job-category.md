# <a name="regulatory-job-category"></a>Категория нормативных должностей


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этой статье описывается набор параметров "Категория нормативных должностей" для Dynamics 365 Human Resources.

Физическое имя: mshr_hcmregulatoryjobcategory

Это перечисление определяет значения набора параметров для параметра категорий нормативных должностей. Это предусмотрено в существующем наборе параметров mshr_hcmregulatoryjobcategory.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Не допускается | Нет. |
| 200000001 | Руководитель | Должностные лица и менеджеры высшего и старшего уровня. |
| 200000002 | Менеджер | Должностные лица и менеджеры первого/среднего уровня. |
| 200000003 | Профессиональные | Профессионалы. |
| 200000004 | Техник | Техники. |
| 200000005 | SalesWorker | Работники отдела продаж. |
| 200000006 | Административный | Работники административной поддержки. |
| 200000007 | CraftWorker | Высококвалифицированные рабочие. |
| 200000008 | Operative | Оперативные работники. |
| 200000009 | Laborer | Чернорабочие/помощники. |
| 200000010 | ServiceWorker | Работники сферы услуг. |
