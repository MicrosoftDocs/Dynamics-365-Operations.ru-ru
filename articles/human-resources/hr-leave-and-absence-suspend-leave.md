# <a name="suspend-leave"></a><span data-ttu-id="5f438-103">Приостановить отпуск</span><span class="sxs-lookup"><span data-stu-id="5f438-103">Suspend leave</span></span>

[!include [banner](includes/preview-feature.md)]

<span data-ttu-id="5f438-104">Можно приостановить отпуск сотрудника, чтобы остановить обработку начислений отпусков для выбранных типов отпусков.</span><span class="sxs-lookup"><span data-stu-id="5f438-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="5f438-105">Приостановка отпусков и отсутствия для сотрудника</span><span class="sxs-lookup"><span data-stu-id="5f438-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="5f438-106">В записи сотрудника выберите **Отпуск**.</span><span class="sxs-lookup"><span data-stu-id="5f438-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="5f438-107">Выберите **Приостановка отпуска**.</span><span class="sxs-lookup"><span data-stu-id="5f438-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="5f438-108">Выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="5f438-108">Select **New**.</span></span>

4. <span data-ttu-id="5f438-109">В диалоговом окне **Приостановка начисления отпуска** выберите **Тип отпуска**, а также значения **Начальная дата** и **Конечная дата** для приостановки.</span><span class="sxs-lookup"><span data-stu-id="5f438-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="5f438-110">При необходимости можно добавить **Комментарий** к приостановке.</span><span class="sxs-lookup"><span data-stu-id="5f438-110">Optionally, you can add a **Comment** for the suspension.</span></span> 

<span data-ttu-id="5f438-111">Если начисления обрабатываются, когда отпуск сотрудника приостановлен, начисления не будут сделаны для приостановленных типов отпусков.</span><span class="sxs-lookup"><span data-stu-id="5f438-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
