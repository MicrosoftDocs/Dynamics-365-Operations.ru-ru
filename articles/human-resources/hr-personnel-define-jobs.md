# <a name="define-new-jobs"></a>Определение новых заданий

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



Эта задача рассматривает создание нового задания. Она будет выполнена с помощью ввода вручную и копирования из функциональных возможностей шаблонов. В качестве компании с демонстрационными данными для создания этой задачи используется USMF. Эта задача предназначена для специалиста по управлению персоналом.

1. Перейдите в раздел **Управление персоналом \> Задания \> Задания**.
2. Выберите **Создать**.
3. В поле **Задание** введите значение.
4. В поле **Описание** введите значение.
5. Выберите стандартный заголовок, который будет использоваться для задания.

    Этот заголовок будет использоваться как значение по умолчанию для новых штатных единиц, назначенных должности.

6. В списке найдите и выберите требуемую запись.
7. В списке выберите ссылку в выбранной строке.
8. В группе **Максимальное количество штатных единиц** выберите **Максимальное число штатных единиц**.
9. Введите максимальное количество штатных единиц. 

    Назначения должности, которые превышают это число, не будут разрешены.

10. В разделе **Описание** в поле **Примечание** введите **Введите стандартное описание задания**.

    Это описание применяется ко всем должностям, назначенным определяемому вами заданию.

11. В разделе **Классификация задания** выберите функциональные обязанности, которые будут использоваться в стандартной отчетности в модуле "Управление персоналом".
12. В списке найдите и выберите требуемую запись.
13. В списке выберите ссылку в выбранной строке.
14. В поле **Тип задания** выберите тип задания для дальнейшей классификации заданий в организации.

    Типы заданий можно использовать в отчетности в любой части модуля "Управление персоналом".

15. В списке найдите и выберите требуемую запись.
16. В списке выберите ссылку в выбранной строке.
17. В разделе **Компенсация** выберите уровень компенсации.

    Уровень компенсации определяет минимальную и максимальную компенсацию, которую может получить сотрудника для задания. Правила компенсации определяют, может ли выходить компенсация сотрудников за пределы диапазона.

18. В списке найдите и выберите требуемую запись.
19. В списке выберите ссылку в выбранной строке.
20. В разделе **Навыки** обратите внимание, что поля на экспресс-вкладках **Навыки**, **Образование** и **Сертификаты** пусты. После копирования информации из шаблона должности сведения, указанные в этих полях, будут введены одновременно.
21. Нажмите **Сохранить**.
22. Выберите **Копировать из**.
23. Выберите **Копировать из шаблона**.
24. В поле **Шаблон должности** нажмите стрелку раскрывающегося списка, чтобы открыть подстановку.
25. В списке найдите и выберите требуемую запись.
26. В списке выберите ссылку в выбранной строке.
27. Выберите вариант **Примечание**.

    Только выбранные пункты будут скопированы в выбранное задание.

28. Нажмите **ОК**.
29. Разверните раздел **Сертификаты**.
30. Разверните раздел **Образование**.
31. Нажмите **Сохранить**.
