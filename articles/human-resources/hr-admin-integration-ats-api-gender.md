# <a name="gender"></a>Пол


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этой статье описывается набор параметров "Пол" для Dynamics 365 Human Resources.

Физическое имя: mshr_hcmpersongender

Это перечисление предоставляет набор параметров поля для кандидата. Оно доступно в наборе параметров mshr_hcmpersongender.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Не допускается | Пол не указан. |
| 200000001 | Мужской | Мужской. |
| 200000002 | Женский | Женский. |
| 200000003 | Не указан | Выбор без определенного пола. |
