# <a name="set-up-tier-codes"></a>Настройка кодов уровней

Dynamics 365 Human Resources использует коды уровней со ставками льгот, оплаченными сотрудниками и работодателем. Можно определить неограниченное число кодов уровней и настроить расчеты ставок для удовлетворения потребностей компании.

1. В рабочей области **Управление льготами** в разделе **Настройка** выберите **Коды уровней**.

2. Выберите **Создать**.

3. Укажите значения для следующих полей:

   | Поле | Описание |
   | --- | --- |
   | **Код уровня** | Уникальный код уровня льготы или число для идентификации кода уровня. |
   | **Описание** | Описание кода уровня льготы. |

4. Нажмите **Сохранить**.
