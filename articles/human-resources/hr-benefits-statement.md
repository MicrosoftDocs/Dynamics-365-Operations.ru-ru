# <a name="benefit-statement"></a>Выписка по льготам


[!INCLUDE [PEAP](../includes/peap-2.md)]

В отчете **Выписка по льготам** предоставляется выписка по льготам, в которых сотрудник в настоящее время зарегистрирован. Доступ к отчету может получить только сам сотрудник или администратор льгот. **Выписка по льготам** представляет собой список зарегистрированных льгот сотрудника, параметров покрытия, затрат и всех зарегистрированных иждивенцев или бенефициаров. Отчет может быть распечатан для одного сотрудника или нескольких сотрудников.

> [!NOTE]
Функция **Выписка по льготам** должна быть включена в рабочей области **Управление функциями**. Для этого в управлении функциями должна быть включена функция **Управление льготами**. 


## <a name="importing-the-benefit-statement"></a>Импорт выписки по льготам 

Необходимо импортировать **Выписку по льготе**, используя конфигурацию отчета, прежде чем будет доступна **Выписка по льготам**. Для этого необходимо выполнить следующие шаги:

1.  Перейдите в рабочую область **Администрирование системы**.

2.  Выберите плитку **Электронная отчетность**.

3.  Перейдите в раздел **Поставщики конфигурации** и выберите **Репозитории**.

  ![Выбор репозиториев.](https://user-images.githubusercontent.com/26801678/134203290-7faf7245-ed08-44e9-95a1-a7ba278c42c6.png)

4.  Выберите **Ресурсы HR** из списка, затем выберите **Открыть**.

    ![Репозитории конфигураций.](https://user-images.githubusercontent.com/26801678/134203619-b3fd087d-1fe9-45ef-a588-1afedfe38dfd.png)

5.  Выберите **Модель выписки по льготам** в левой области и разверните ее, чтобы отобразить пункт **Формат выписки по льготам**.

6.  В левой области выберите **Формат выписок по льготам**.

7.  В правой части экрана будет расположена экспресс-вкладка **Версии**. Выберите **Импорт**.

    ![Экспресс-вкладка версий.](https://user-images.githubusercontent.com/26801678/134203763-f12ef549-e326-400d-ac69-b25fc94af47b.png)

## <a name="generate-the-benefit-statement-as-a-pdf-file"></a>Создание выписки по льготам в виде PDF-файла

По умолчанию **Выписка по льготам** будет распечатана в виде документа Microsoft Word. Для печати в формате PDF необходимо настроить параметр PDF в пункте **Место назначения электронной отчетности**. 

1. Для этого перейдите в рабочую область **Администрирование системы** > **Электронная отчетность** > **Связанные ссылки** > **Место назначения электронной отчетности**.

1.  Выберите **Создать**.

2.  В поле **Ссылка** выберите **Формат выписки по льготам**.

3.  На экспресс-вкладке **Место назначения файла** выберите **Создать**.

4.  В поле **Имя** введите **Выписка по льготам (PDF)**.

5.  В поле **Имя компонента файла** выберите **Выписка по льготам**.

6.  Установите флажок **Преобразовать в PDF**.

7.  Выберите **Параметры** из пункта меню. 

    ![Место назначения файла.](https://user-images.githubusercontent.com/26801678/134203881-a3f1ebc3-d816-485d-a53b-026cc29cae64.png)

8.  Перейдите экспресс-вкладку **Файл**, затем выберите **Включено**.

9.  Нажмите **ОК**.
   
> [!NOTE]
> Функция управления льготами находится в состоянии предварительной версии. Имеется известная проблема при использовании настройки адресата электронной почты в отчете **Место назначения электронной отчетности**. Может быть выведено сообщение об ошибке, и сообщение электронной почты не будет отправлено.

**Выписка по льготам** может быть создана со следующих страниц:

-   **Рабочая область управления льготами** > **Ссылки** > **Отчеты** > **Выписка по льготам**

-   **Сотрудники (устаревшая форма)** > **вкладка «Личные данные»** > **Выписка по льготам**

-   **Упрощенный вход сотрудников** > **Отчеты о льготах** > **Выписка по льготам**

-   **Самообслуживание сотрудников** > **Льготы** > **Просмотр выписки по льготам**

> [!NOTE]
>  Доступ к **Выписке по льготам** в пункте **Самообслуживание сотрудников** контролируется привилегией **Просмотр выписки по льготам**. Она находится под обязанностью **Льготы самообслуживания сотрудников**. Эта привилегия предоставляется сотрудникам по умолчанию.

## <a name="report-contents"></a>Содержимое отчета

В отчете **Выписка по льготам** печатаются подтвержденные выбранные параметры планов сотрудника, включая все отмененные планы. На следующем рисунке показан пример этого отчета. 

![Отчет выписки по льготам.](https://user-images.githubusercontent.com/26801678/134204058-61baa318-fede-4795-a256-acdf3217f9f9.png)

В отчете отображаются **План**, **Параметр покрытия**, **Расходы сотрудника** и **Расходы работодателя**. В отчете также будут перечислены все покрываемые иждивенцы. Если план имеет связанных с ним бенефициаров, будут отображены бенефициары и их приоритет и процент распределения.

Отчет **Выписка по льготам** может быть распечатан для нескольких сотрудников одновременно с использованием экспресс-вкладки **Включаемые записи** на странице **Выписка по льготам**.
