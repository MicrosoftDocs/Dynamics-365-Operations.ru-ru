# <a name="months-of-year"></a>Месяцы года

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этом разделе описывается набор параметров месяцев года для Dynamics 365 Human Resources.

Физическое имя: mshr_monthsofyear

Это перечисление предоставляет набор параметров для месяцев года.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Не допускается | Значение не было выбрано. |
| 200000001 | января | Январь. |
| 200000002 | февраля | Февраль. |
| 200000003 | марта | Март. |
| 200000004 | апреля | Апрель. |
| 200000005 | мая | Май. |
| 200000006 | июня | Июнь. |
| 200000007 | июля | Июль. |
| 200000008 | августа | Август. |
| 200000009 | сентября | Сентябрь. |
| 200000010 | октября | Октябрь. |
| 200000011 | ноября | Ноябрь. |
| 200000012 | декабря | Декабрь. |
