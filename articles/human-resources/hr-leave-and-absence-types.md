# <a name="configure-leave-and-absence-types"></a>Настройка типов отпусков и отсутствий

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Типы отпусков в Dynamics 365 Human Resources определяют типы отсутствий, о которых могут сообщать сотрудники. Типы отпусков можно настроить в соответствии с потребностями организации. Примеры типов отпуска:

- Оплачиваемое отсутствие (PTO)
- Неоплачиваемый отпуск
- Оплачиваемый отпуск
- Больничный
- Отпуск по медицинским причинам
- Отпуск по семейным обстоятельствам
- Кратковременная нетрудоспособность
- Отпуск в связи со смертью

## <a name="add-a-leave-type"></a>Добавление типа отпуска

1. На странице **Отпуск и отсутствия** выберите вкладку **Ссылки**.

2. В **Настройка** выберите **Типы отпуска и отсутствия**.

3. Выберите **Создать**.

4. Введите имя для типа отпуска в **Тип**, выберите workflow-процесс в **Код workflow-процесса** и введите описание в **Описание**.

5. В **Общее** выберите **Нет**, **Запланировано** или **Незапланированно** в раскрывающемся списке **Категория**.

6. Выбор кода дохода из раскрывающегося списка **Код дохода**.

7. В **Требуется код основания** необходимо указать, нужно ли требовать код причины. Если требуется указать коды причины, возможно, потребуется их добавить. В **Коды причин** выберите **Добавить**, выберите код причины, а затем установите флажок **Включено** рядом с ним.

8. В **Ограничить доступ выбранными ролями** выберите, следует ли ограничить доступ. Затем выберите роли безопасности в **Роли безопасности для данного типа отпуска**. Роли безопасности определяются в workflow-процессе, выбранном в **Код workflow-процесса** ранее в этой процедуре.

9. В разделе **Цвет календаря** выберите цвет для показа в календарях отпусков и отсутствия для данного типа отпуска. 

10. В области **Отношения приостановки** выберите, если хотите, чтобы этот тип отпуска был приостановлен, либо приостановлен другим типом отпуска. При отправке запроса об отсутствии для приостанавливаемого типа отпуска, приостановка отпуска будет автоматически создана для типа приостановленного отпуска. 

10. Нажмите **Сохранить**.

## <a name="configure-leave-type-rules"></a>Настройка правил типов отпусков

1. Настройка параметров округления для типа отпуска. Возможны следующие варианты: **Нет**, **Вверх**, **Вниз** и **Ближайшее**. Можно также задать точность округления для типа отпуска.

2. Настройте **Корректировка праздников** для типа отпуска. При выборе этого параметра число праздников, которые выпадают на рабочий день, будет использоваться, чтобы определить, как начислять время отсутствия для типа отпуска. Например, если Рождество приходится на понедельник, при обработке начислений модуль Human Resources вычтет один день из типа отпуска.

   Можно установить праздники в календаре рабочего времени. Дополнительные сведения см. в [Создание календаря рабочего времени](hr-leave-and-absence-working-time-calendar.md).
   
 3. Задайте **Перенести тип отпуска** для типа отпуска. При выборе этого параметра все перенесенные балансы будут перенесены в указанный тип отпуска. Тип переноса также должен быть включен в план отпусков и отсутствия. 
 
4. Задайте **Правила истечения срока действия** для типа отпуска. При настройке этого параметра можно выбрать единицу измерения дней или месяцев и установить срок действия. Дата вступления в силу правила истечения срока действия используется, чтобы определить время начала выполнения пакетного задания, которое обрабатывает истечение срока действия отпуска, или дату, в которую это правило вступает в силу. Срок действия всегда будет иметь значение в дату начала периода начисления. Например, если дата начала периода начисления — 3 августа, 2021, а для правила окончания срока действия было задано 6 месяцев, правило будет обработано на основе смещения окончания срока действия от даты начала периода начисления, так что будет выполнено 3 февраля 2022 года. Любые балансы отпусков, существующие на момент окончания срока, будут вычитаться из типа отпуска и будут отражены в балансе отпусков.
 
## <a name="configure-the-required-attachment-per-leave-type"></a>Настройка требуемого вложения для типа отпуска

> [!NOTE]
> Чтобы использовать поле **Требуется вложение**, необходимо сначала включить функцию **Настройка обязательного вложения для запросов на отпуск** в управлении функциями. Дополнительные сведения о порядке включения функций см. в разделе [Управление функциями](hr-admin-manage-features.md).

1. На странице **Отпуск и отсутствие** на вкладке **Ссылки** в области **Настройка** выберите **Типы отпусков и отсутствия**.

2. Выберите в списке тип отпуска и отсутствия. Затем в разделе **Общие** используйте поле **Требуется вложение**, чтобы указать, должно ли вложение быть отправлено, когда сотрудник отправляет новый запрос отпуска для выбранного типа отпуска. 

Сотрудникам потребуется отправить вложение при подаче нового запроса отпуска, который имеет тип отпуска, в котором включено поле **Требуется вложение**. Чтобы просмотреть вложение, отправленное в рамках запроса на отпуск, утверждающие запросов отпусков могут использовать параметр **Вложения** для назначенных им рабочих элементов. Если для доступа к запросу отпуска используется приложение Human Resources в Microsoft Teams, для запроса отпуска можно использовать параметр **Просмотр сведений**, чтобы просмотреть подробные сведения и любые вложения.

## <a name="configure-leave-units-hoursdays-per-leave-type"></a>Настройка единиц измерения отпуска (часы/дни) для каждого типа отпуска

> [!NOTE]
> Чтобы использовать функцию единиц измерения отпуска для типов отпусков, необходимо сначала включить функцию **Настройка единиц изменения отпуска для типа отпуска** в управлении функциями. Дополнительные сведения о порядке включения функций см. в разделе [Управление функциями](hr-admin-manage-features.md).

> [!IMPORTANT]
> По умолчанию типы отпусков в юридическом лице используют единицы измерения отпусков из настройки параметров отпусков на уровне юридического лица.
> 
> Единица измерения отпуска и тип отсутствия могут быть изменены только в том случае, если для данного типа отпуска отсутствуют проводки отпусков.
> 
> Эту функцию невозможно отключить после ее включения.

1. На странице **Отпуск и отсутствие** на вкладке **Ссылки** в области **Настройка** выберите **Типы отпусков и отсутствия**.

2. Выберите в списке тип отпуска и отсутствия. Затем в разделе **Общие** в поле **Единица измерения** выберите единицу измерения отпусков. Можно выбрать **Часы** или **Дни**.

3. Необязательно: если выбрано значение **Часы** в поле **Единицы измерения**, можно использовать поле **Включить определение половины дня**, чтобы указать, могут ли сотрудники выбирать отгул в первую половину дня или во вторую половину дня, если они запрашивают отгул на половину дня.

Сотрудники, отправляющие новый запрос отпуска, могут выбирать различные типы отпусков для создания своего запроса на отпуск. Однако все типы отпусков, выбранные как часть одного запроса отпуска, должны иметь одинаковые единицы измерения отпуска. Сотрудники могут просматривать единицу измерения отпуска для каждого типа отпуска в форме **Запрос отгула**.
