# <a name="loan-item-to-a-worker"></a>Выдача номенклатуры во временное пользование работнику

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]



В этой процедуре показано, как выдать номенклатуру во временное пользование работнику и зарегистрировать возврат этой номенклатуры работником. Работники могут также запрашивать выдачу номенклатур во временное пользование на странице **Самообслуживание сотрудников**. Компания **USMF** с демонстрационными данными используется для создания этой процедуры.


## <a name="loan-an-item-to-a-worker"></a>Выдача номенклатуры во временное пользование работнику

1. Перейдите в раздел **Управление персоналом \> Работники \> Одалживаемые номенклатуры \> Одолженное оборудование**.
2. Выберите **Создать**.
3. В поле **Человек** введите или выберите значение.
4. В поле **Одалживаемая номенклатура** введите или выберите значение.
5. В поле **Планируемый возврат** введите дату, когда сотрудник должен вернуть переданную во временное пользование номенклатуру.
6. Нажмите **Сохранить**.
7. Закройте страницу.

## <a name="return-a-loan-item"></a>Возврат выданной во временное пользование номенклатуры

1. Перейдите в раздел **Управление персоналом \> Работники \> Одалживаемые номенклатуры \> Одолженное оборудование**.
2. Выберите **Правка**.
3. Введите дату в поле **Фактический возврат**.
