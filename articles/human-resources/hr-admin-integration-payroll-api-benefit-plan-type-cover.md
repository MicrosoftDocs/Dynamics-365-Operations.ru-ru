# <a name="benefit-plan-type-cover"></a>Покрытие типа плана льгот

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этой теме описывается набор параметров покрытия типа плана льгот для Dynamics 365 Human Resources.

## <a name="option-set-process-type"></a>Набор параметров: тип процесса

Физическое имя: mshr_plantypecode

Это перечисление предоставляет набор параметров для типа процесса в виртуальных сущностях Human Resources.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Медицинское обслуживание | Медицинское обслуживание. |
| 200000001 | Пособие на зубоврачебное обслуживание | Пособие на зубоврачебное обслуживание. |
| 200000002 | Зрение | Зрение. |
| 200000003 | Сбережения | Сбережения. |
| 200000004 | Сберегательный счет на случай непредвиденных затрат | Сберегательный счет на случай непредвиденных затрат. |
| 200000005 | Страхование жизни | Страхование жизни. |
| 200000006 | AccidentalDeathDismemberment | AccidentalDeathDismemberment. |
| 200000007 | Другая | Другое.|
| 200000008 | LongTermDisability | LongTermDisability. |
| 200000009 | ShortTermDisability | ShortTermDisability. |
