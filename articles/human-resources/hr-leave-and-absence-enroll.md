# <a name="assign-workers-to-a-leave-plan"></a>Назначение работников плану отпусков

> [!Important]
> Функции, перечисленные в этой статье, в настоящее время доступны для клиентов в отдельной версии Dynamics 365 Human Resources. Некоторые или все функции будут доступны в составе будущего выпуска в инфраструктуре Finance после выпуска Finance 10.0.26.


[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

После создания плана отпуска и отсутствия в Dynamics 365 Human Resources необходимо назначить его работникам.

## <a name="assign-a-plan-to-a-group-of-employees"></a>Назначение плана для группы сотрудников

1. На странице **Отпуск и отсутствие** выберите план, который требуется назначить.

2. Выберите **Назначить сотрудникам**.

3. При необходимости настройте следующие параметры:

   - **Регистрация в дату начала трудоустройства**
   - **Дата регистрации**
   - **Дата завершения**
   - **Основа уровня**

4. В **Работники для регистрации** выберите все работы, которые будут назначены плану.

5. Выберите **Назначить план** в нижней части страницы.

## <a name="assign-a-plan-to-an-employee"></a>Назначение плана сотруднику

1. В записи сотрудника выберите **Отпуск**.

2. Выберите **Отпуск и отсутствие**.

3. Выберите **Создать**.

4. Введите дату регистрации и выберите план отпуска.
