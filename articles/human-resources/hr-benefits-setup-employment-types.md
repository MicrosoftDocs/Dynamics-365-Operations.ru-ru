# <a name="set-up-employment-types"></a>Настройка типов занятости

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Настройте типы занятости в Dynamics 365 Human Resources, чтобы они соответствовали сотрудникам.

1. В рабочей области **Управление персоналом** в разделе **Настройка** выберите **Типы занятости**.

2. Выберите **Создать**.

3. Укажите значения для следующих полей:

   | Поле | Описание |
   | --- | --- |
   | **Тип трудоустройства** | Уникальная идентификация типа занятости. |
   | **Описание** | Описание типа занятости. |

4. Нажмите **Сохранить**.
