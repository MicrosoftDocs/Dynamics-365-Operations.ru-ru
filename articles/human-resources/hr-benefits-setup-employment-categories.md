# <a name="create-employment-categories"></a>Создание категорий занятости


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Настройте категории занятости в Dynamics 365 Human Resources, чтобы они соответствовали сотрудникам.

1. В рабочей области **Управление льготами** в разделе **Настройка** выберите **Категории занятости**.

2. Выберите **Создать**.

3. Укажите значения для **Тип сотрудника**, **Категория занятости** и **Описание**.

4. Нажмите **Сохранить**.
