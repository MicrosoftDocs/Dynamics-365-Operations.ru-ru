# <a name="create-and-open-job-requisition"></a>Создание и открытие заявки на должность

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Проекты по набору сотрудников в можно использовать для управления процессом найма. После создания проекта по набору сотрудников можно выполнить следующие задачи:

- Написание объявления о найме на работу для проекта.
- Публикация объявления на страницах самообслуживания сотрудников.
- Связывание заявлений о приеме на работу с проектом.
- Отслеживание действий для этого проекта. 

1. Перейдите в раздел **Управление персоналом > Набор сотрудников > Проекты по набору сотрудников > Проекты по набору сотрудников**.
2. Выберите **Создать**.
3. В поле **Проект по набору сотрудников** введите значение.
4. В поле **Описание** введите значение.
5. В поле **Специалист по найму** выберите кнопку раскрывающегося списка, чтобы открыть подстановку.
6. В списке найдите и выберите вашу запись.
7. В списке выберите ссылку в выбранной строке.
8. Выберите **Выбрать**.
9. В поле **Подразделение** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
10. В списке выберите ссылку в выбранной строке.
11. В поле **Должность** выберите кнопку раскрывающегося списка, чтобы открыть подстановку.
12. В списке найдите и выберите вашу запись.
13. В списке выберите ссылку в выбранной строке.
14. В поле **Количество вакансий** введите число.
15. В поле **Специалист по комплектации штата** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
16. В списке найдите и выберите вашу запись.
17. В списке выберите ссылку в выбранной строке.
18. Выберите **Выбрать**.
19. В поле **Крайний срок подачи заявления** введите дату.
20. Выберите **Носитель**. Проекты по набору сотрудников включают параметр для определения медиаплощадок, которые требуется использовать для рекламы открытых позиций.  
21. Выберите **Создать**.
22. В поле **Носитель** выберите кнопку раскрывающегося списка, чтобы открыть подстановку.
23. В списке выберите ссылку в выбранной строке.
24. В поле **Дата начала** введите дату.
25. В поле **Дата окончания** введите дату.
26. Нажмите **Сохранить**.
27. Закройте страницу.
28. Выберите **Объявления о приеме на работу**.
29. Нажмите **Сохранить**.
30. Закройте страницу.
31. Установите или снимите флажок **Отобразить на портале самообслуживания сотрудников**. Выберите флажок **Отобразить на портале самообслуживания сотрудников**. Этот выбор позволяет сделать проект по набору сотрудников видимым для сотрудников на их странице **Самообслуживание сотрудников**.
32. Выберите **Статус проекта по набору сотрудников**.
33. Выберите **Пуск**. Статус **Начато** означает, что проект готов получать заявления.  
34. Нажмите **ОК**.
