# <a name="set-up-rounding-rules"></a>Настройка правил округления


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Можно настроить правила округления в Dynamics 365 Human Resources для соответствия с потребностям организации.

1. В рабочей области **Управление льготами** в разделе **Настройка** выберите **Правила округления**.

2. Выберите **Создать**.

3. Укажите значения для **Округление до ближайшего**, **Сценарий** и **Способ округления**.

4. Нажмите **Сохранить**.
