# <a name="map-skills"></a>Подбор персонала

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Можно создать поиск для подбора персонала, чтобы найти квалифицированного сотрудника в Dynamics 365 Human Resources. Поиск для подбора персонала возвращает результаты для введенного вами критерия путем поиска следующей информации:

- Навыки
- Образование
- Сертификаты
- Позиции
- Проектный опыт

Например, можно найти людей в организации, которые получили свои CPA.

Профили подбора персонала позволяют найти текущих сотрудников или кандидатов с квалификациями, которые прямо соответствуют бизнес-потребностям.

> [!NOTE]
> Только сотрудники, кандидаты и контактные лица, которые выбраны для включения в поиск подбора персонала, будут отображаться в списке результатов подбора персонала или включаться в профиль навыков. Для того, чтобы включить пользователя в поиски по подбору персонала, установите для параметра **Включить в подбор персонала** значение **Да** на следующих страницах:<br>
> - Рабочий<br>
> - Код сотрудника<br>
> - Кандидат<br>
> - Контактные лица<br>

Чтобы создать подбор персонала, перейдите на страницу **Развитие сотрудников > Ссылки > Подбор персонала**. Чтобы создать профиль подбора персонала, перейдите на страницу **Развитие сотрудников > Ссылки > Профили подбора персонала**.

## <a name="skill-gap-analysis-and-skill-profile-analysis"></a>Анализ несоответствия навыков и анализ профилей навыков

Можно создать анализ профиля навыков для просмотра списка компетенций сотрудника. Можно создать анализ несоответствия навыков для сравнения навыков лица с навыками, которые требуются для задания.

Чтобы создать анализ несоответствия, перейдите на страницу **Развитие сотрудников > Ссылки > Сравнительный анализ квалификационных навыков сотрудника заданию**. Чтобы создать анализ профиля навыков, перейдите на страницу **Развитие сотрудников > Ссылки > Анализ профиля навыков**.
