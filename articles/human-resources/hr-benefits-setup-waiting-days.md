# <a name="configure-waiting-days"></a>Настройка дней ожидания

В Microsoft Dynamics 365 Human Resources дни ожидания определяют месячный этап для использования в планах льгот. Например, день ожидания 15 означает 15-й день каждого месяца. 

1. В рабочей области **Управление льготами** в разделе **Настройка** выберите **Дни ожидания**.

2. Выберите **Создать**.

3. Укажите значения для следующих полей:

   | Поле | Описание |
   | --- | --- |
   | **День ожидания** | Уникальный идентификатор для дня ожидания. |
   | **Описание** | Описание дня ожидания. |
   | **День** | День месяца, в который система должна установить дату периода ожидания. |
   
4. Нажмите **Сохранить**.
