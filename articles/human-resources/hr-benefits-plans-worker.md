# <a name="create-worker-benefit-plans"></a>Создание планов льгот работников

Можно создать планы льгот работника в Microsoft Dynamics 365 Human Resources, чтобы выбрать планы льгот для сотрудников и подтвердить выбор плана льгот. Обычно сотрудники самостоятельно выбирают планы льгот, используя дистанционное обслуживание сотрудников, а затем администратор льгот подтверждает выбор. 

1. В рабочей области **Управления льготами** в **Планы** выберите **Планы льгот работников**.

2. Выберите **Создать**.

3. Укажите значения для следующих полей:

   | Поле | Описание |
   | --- | --- |
   | Период | Указывает период льгот, который будет использоваться для фильтрации планов на вкладке планов. Отфильтруйте планы, чтобы помочь выбрать подмножество всех записей плана, чтобы можно было подтвердить подмножество. Например, выберите созданный период, который называется "2015", чтобы подтвердить все льготные варианты регистрации для "2015". |
   | Рабочий | Указывает работника, который будет использоваться для фильтрации планов на вкладке планов. Отфильтруйте планы, чтобы помочь выбрать подмножество всех записей плана, чтобы можно было подтвердить подмножество. |
   | Информация о компании | Указывает юридическое лицо, которое будет использоваться для фильтрации планов на вкладке планов. Отфильтруйте планы, чтобы помочь выбрать подмножество всех записей плана, чтобы можно было подтвердить подмножество. |
   | Параметр покрытия | Указывает параметр покрытия, который будет использоваться для фильтрации планов на вкладке планов. Отфильтруйте планы, чтобы помочь выбрать подмножество всех записей плана, чтобы можно было подтвердить подмножество. |
   | По умолчанию | Фильтрует планы льгот в зависимости от того, являются ли они планом по умолчанию. Отфильтруйте планы, чтобы помочь выбрать подмножество всех записей плана, чтобы можно было подтвердить подмножество. |
   | Состояние | Фильтры планов льгот по их статусу. Отфильтруйте планы, чтобы помочь выбрать подмножество всех записей плана, чтобы можно было подтвердить подмножество. |
   | Подтверждение | Указывает статус подтверждения, который будет использоваться для фильтрации планов на вкладке планов. Отфильтруйте планы, чтобы помочь выбрать подмножество всех записей плана, чтобы можно было подтвердить подмножество. |
   | Отмена | Указывает статус отмены, который будет использоваться для фильтрации планов на вкладке планов. Отфильтруйте планы, чтобы помочь выбрать подмножество всех записей плана, чтобы можно было подтвердить подмножество. |
   | Фильтр дат вступления в силу | Фильтрует планы в зависимости от того, просрочены они, активны или будут активными в будущем. Установите флажки, соответствующие планам, которые необходимо увидеть на экспресс-вкладке планов. |
   | Планы | Экспресс-вкладка "Планы" содержит планы, отвечающие заданным критериям фильтрации. Соответствующие параметры конфигурации, которые были установлены сотрудниками отдела кадров, и выбранные сотрудниками регистрации, включаются в каждую строку. Поле квалификации указывает, имеется ли конфликт проверки с выбранным планом. |

4. Нажмите **Сохранить**.
