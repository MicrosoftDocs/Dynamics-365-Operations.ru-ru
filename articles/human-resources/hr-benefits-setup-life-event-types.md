# <a name="configure-life-event-types"></a>Настройка типов событий жизненного цикла

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dynamics 365 Human Resources использует типы жизненных событий для определения событий, когда допускается обновление регистрации льгот для сотрудников, например заключение брака или появление ребенка. Каждый код типа жизненного события может быть связан только с одним типом жизненного события. Например, если создается код жизненного события "Изменение адреса", который связан с типом жизненного события "Изменение адреса сотрудника", невозможно создать другой код с меткой "Изменение адреса сотрудника" и связать его с типом жизненного события "Изменение адреса сотрудника". Если тип жизненного события не связан с типом плана, тип жизненного события не будет вызывать жизненное событие. Дополнительные сведения см. в разделе [Создание типов планов](hr-benefits-setup-plan-types.md).

## <a name="create-a-life-event-type"></a>Создание типа жизненного события

1. В рабочей области **Управление льготами** в разделе **Настройка** выберите **Типы жизненных событий**.

2. Выберите **Создать**.

3. Укажите значения для следующих полей:

   | Поле | Описание |
   | --- | --- |
   | **Идентификатор типа жизненного события** | Уникальный код типа жизненного события. |
   | **Описание** | Описание типа жизненного события. |
   | **Тип жизненного события** | Катализатор для обновления регистрации льгот сотрудника. Список жизненных событий см. в разделе [Жизненные события](hr-benefits-setup-payment-frequencies.md?life-events). |

4. Нажмите **Сохранить**.

## <a name="view-attached-plans"></a>Просмотр присоединенных планов

Можно просмотреть список планов, присоединенных к выбранному типу жизненного события. Жизненные события присоединяются к типу плана, а типы планов связаны с планом.

1. В рабочей области **Управление льготами** в разделе **Настройка** выберите **Типы жизненных событий**.

2. Выберите **Действия**.

3. Выберите **Присоединенные планы**.

Личный контакт **Дата вступления в силу отпуска** обновлена.  |

| **Изменение в покрытии (зарплата)** | Работник > Компенсация > Фиксированный план<br>Работник > Личные сведения > Годовая зарплата для льгот | Если Управление льготами > Общие параметры Human Resources > Льготы > Годовая зарплата для льгот" не включено, обновление "Работник > Компенсация > Фиксированный план" создаст жизненное событие.
