# <a name="period-unit"></a>Единица измерения периода

В этом разделе описывается набор параметров еденицы измерения периода для Dynamics 365 Human Resources.

Физическое имя: mshr_periodunit

Это перечисление предоставляет набор параметров единиц измерения для периодов времени.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Дни | День. |
| 200000001 | Месяцы | Месяц. |
| 200000002 | Годы | Год. |
