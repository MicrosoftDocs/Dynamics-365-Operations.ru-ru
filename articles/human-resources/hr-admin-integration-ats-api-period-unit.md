# <a name="period-unit"></a>Единица измерения периода


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

В этой статье описывается набор параметров "Единица измерения периода" для Dynamics 365 Human Resources.

Физическое имя: mshr_periodunit

Это перечисление предоставляет набор параметров единиц измерения для периодов времени.

| значение | Этикетка | описание |
| --- | --- | --- |
| 200000000 | Дни | День. |
| 200000001 | Месяцы | Месяц. |
| 200000002 | Годы | Год. |
