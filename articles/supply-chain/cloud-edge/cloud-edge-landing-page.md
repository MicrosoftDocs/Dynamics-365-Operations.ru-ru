# <a name="scale-unit-capability-is-currently-paused-for-new-customers"></a>Возможность использования единиц масштабирования для новых клиентов в настоящее время приостановлена

[!include [banner](../includes/banner.md)]

> [!IMPORTANT]
> Возможность масштабирования в Microsoft Dynamics 365 Supply Chain Management приостановлена для новых клиентов, поскольку архитектура проходит усовершенствование. В настоящее время мы не предпринимает никаких новых клиентов для данного решения, но активно работает над составной, гибкой и удобной в применении архитектурой для нашей сферы деятельности цепочки поставок, что еще более улучшит нашу способность предоставлять для наших клиентов универсальное решение единицы масштабирования.
>
> Мы работаем над улучшением Dynamics 365 Supply Chain Management, чтобы обеспечить дополнительную сочетаемость между конкретными модулями, такими как управление складом. Мы будем продолжать планировать и разрабатывать новые предложения для облачных и пограничных единиц масштабирования для удовлетворения потребностей наших клиентов. Для получения дополнительных сведений обратитесь по адресу [d365scmcneext@microsoft.com](mailto:d365scmcneext@microsoft.com).
