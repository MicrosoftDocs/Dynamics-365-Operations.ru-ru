# <a name="create-a-purchase-order-governed-by-budget"></a>Создание заказа на покупку в соответствии с бюджетом

[!include [banner](../../includes/banner.md)]

Эта процедура служит для создания заказа на покупку, который проверяется на наличие доступного бюджета. В этой записи используется компания с демонстрационными данными USMF.


## <a name="review-the-budget-control-configuration"></a>Просмотр конфигурации бюджетного контроля
1. Перейдите в раздел "Бюджетирование" > "Настройка" > "Бюджетный контроль" > "Конфигурация бюджетного контроля".
2. Перейдите на вкладку "Доступные бюджетные средства".
3. Перейдите на вкладку "Документы и журналы".
4. Перейдите на вкладку "Определение правил бюджетного контроля".
5. Перейдите на вкладку "Определить бюджетные группы".
6. Закройте страницу.

## <a name="create-the-purchase-order-header"></a>Создание заголовка заказа на покупку
1. Перейдите в раздел "Закупки и источники" > "Заказы на покупку" > "Все заказы на покупку".
2. Щелкните "Создать".
3. В поле "Счет поставщика" введите или выберите значение.
4. Разверните раздел "Общие".
5. В поле "Финансовая дата" установите дату "01.01.2016".
6. Нажмите кнопку "OК".

## <a name="add-a-purchase-order-line"></a>Добавление строки заказа на покупку
1. В поле "Категория закупаемой продукции " введите или выберите значение.
2. Для параметра "Количество" задайте значение "2".
3. В поле "Единица измерения" введите или выберите значение.
4. В поле "Цена за ед. изм." введите "10 000".
5. Щелкните "Статистика".
6. Щелкните "Распределить суммы".
7. В поле "Счет ГК" укажите значение "601300-001-023--".
8. Закройте страницу.

## <a name="perform-budget-checking"></a>Выполнить проверку бюджета
1. Щелкните "Статистика".
2. Щелкните "Выполнить проверку бюджета".
3. Щелкните "Статистика".
4. Щелкните "Ошибки или предупреждения бюджетных проверок".
5. Щелкните "Закрыть".
