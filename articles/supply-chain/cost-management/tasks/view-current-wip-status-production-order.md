# <a name="view-current-wip-status-on-a-production-order"></a>Просмотр текущего статуса НЗП по производственному заказу

[!include [banner](../../includes/banner.md)]

Следующая процедура показывает просмотр отчета по НЗП конкретного производственного заказа. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Эта процедура предназначена для контролера затрат.

1. Щелкните "Администрирование затрат".
2. Щелкните "Производственные заказы".
3. Используйте экспресс-фильтр для фильтрации поля "Производство" со значением "p000153".
4. В области действий щелкните "Управление затратами".
5. Щелкните "Производственный отчет по НЗП".
6. В поле "Начальная дата" установите для даты значение "01.12.2012".
7. В поле "Конечная дата" установите для даты значение "31.12.2012".
