# <a name="compare-active-estimated-and-realized-costs-on-a-production-order"></a>Сравнение активных, оцененных и реализованных затрат по производственному заказу

[!include [banner](../../includes/banner.md)]

Следующая процедура используется для просмотра причин высокого отклонения цены производства от себестоимости для производственного заказа. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Эта процедура предназначена для контролера затрат.

1. Щелкните "Администрирование затрат".
2. В поле "Дата" введите или выберите значение.
    * В данной процедуре используется финансовый год 2012. Можно для параметра "Дата начала" указать значение "1 января 2012 г.", а для параметра "Дата окончания" — значение "31 декабря 2012 г.".  
3. Перейдите на вкладку "Высокие отклонения цены производства от себестоимости".
4. Щелкните для перехода по ссылке в поле "Производство".
    * Щелкните P000116 для перехода по ссылке в поле "Производство".  
5. В области действий щелкните "Управление затратами".
6. Щелкните "Просмотр сравнения стоимости".
7. Щелкните "Закрыть".
