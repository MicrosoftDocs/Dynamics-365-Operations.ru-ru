# <a name="view-cost-entries-for-a-cost-object"></a>Просмотр записей затрат для объекта затрат

[!include [banner](../../includes/banner.md)]

Ниже описан порядок просмотра записей затрат для объекта затрат. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Эта процедура предназначена для контролера затрат.

1. Щелкните "Администрирование затрат".
2. Щелкните "Выпущенные продукты".
3. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле "Код номенклатуры" по значению "m0004".
4. В области действий щелкните "Управление затратами".
5. Щелкните "Объекты затрат".
6. Щелкните "Записи затрат".
7. С помощью экспресс-фильтра отфильтруйте поле "Номер" по значению "p000031".
    * Если записи затрат пустые, установите дату начала на 31-е января 2012 года, а дату окончания на 31-е декабря 2012 года.
