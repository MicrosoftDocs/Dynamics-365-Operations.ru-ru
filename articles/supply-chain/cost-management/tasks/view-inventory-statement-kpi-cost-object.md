# <a name="view-inventory-statement-and-kpi-by-cost-object"></a>Просмотр складской выписки и КПЭ по объектам затрат

[!include [banner](../../includes/banner.md)]

В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Эта процедура предназначена для контролера затрат.

1. Выберите **Администрирование затрат**.
2. Выберите **Выпущенные продукты**.
3. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле **Код номенклатуры** по значению *d0005*.
4. На панели операций выберите **Управление затратами**.
5. Выберите **Объекты затрат**.
6. Выберите **Выписка по запасам и ключевые показатели эффективности**.
7. В поле **Дата начала** введите дату.
8. В поле **Дата окончания** введите дату.
