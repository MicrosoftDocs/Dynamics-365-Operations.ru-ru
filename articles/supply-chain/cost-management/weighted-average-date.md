<span data-ttu-id="2b600-104">Взвешенное среднее по дате — это складская модель, основанная на принципе взвешенного среднего.</span><span class="sxs-lookup"><span data-stu-id="2b600-104">Weighted average date is an inventory model that is based on the weighted average principle.</span></span> <span data-ttu-id="2b600-105">Для принципа взвешенного среднего расходы со склада оцениваются по среднему значению номенклатур, полученных на склад за каждый день периода закрытия запасов.</span><span class="sxs-lookup"><span data-stu-id="2b600-105">For the weighted average principle, issues from inventory are valued at the average value of the items that are received into inventory for each day in the inventory closing period.</span></span> 

<span data-ttu-id="2b600-106">При выполнении закрытия склада с использованием взвешенного среднего по дате все ежедневные приходы сопоставляются с виртуальным расходом.</span><span class="sxs-lookup"><span data-stu-id="2b600-106">When you run an inventory closing by using weighted average date, all daily receipts are settled against a virtual issue.</span></span> <span data-ttu-id="2b600-107">Этот виртуальный расход содержит общее полученное количество за этот день.</span><span class="sxs-lookup"><span data-stu-id="2b600-107">This virtual issue holds the total received quantity and value for that day.</span></span> <span data-ttu-id="2b600-108">Виртуальный расход имеет соответствующий виртуальный приход, с которым будут сопоставлены расходы.</span><span class="sxs-lookup"><span data-stu-id="2b600-108">The virtual issue has a corresponding virtual receipt that the issues will be settled against.</span></span> <span data-ttu-id="2b600-109">Таким образом, все расходы получают одинаковую среднюю стоимость.</span><span class="sxs-lookup"><span data-stu-id="2b600-109">Therefore, all issues receive the same average cost.</span></span> <span data-ttu-id="2b600-110">Виртуальный расход и виртуальный приход можно рассматривать как виртуальное перемещение, называемое *Перемещение закрытия запасов для взвешенного среднего*.</span><span class="sxs-lookup"><span data-stu-id="2b600-110">The virtual issue and virtual receipt can be considered a virtual transfer that is known as the *weighted average inventory closing transfer*.</span></span> 

<span data-ttu-id="2b600-111">Если в указанную дату или до нее произошел только один приход, оценивать среднее не нужно.</span><span class="sxs-lookup"><span data-stu-id="2b600-111">If only one receipt has occurred on or before the date, you don't have to value the average.</span></span> <span data-ttu-id="2b600-112">Поскольку все расходы сопоставляются с этим приходом, виртуальное перемещение не создается.</span><span class="sxs-lookup"><span data-stu-id="2b600-112">Because all issues are settled from that receipt, the virtual transfer won't be created.</span></span> <span data-ttu-id="2b600-113">Аналогичным образом, если на эту дату имеются только расходы, нет приходов, из которых оценивается среднее, и в этом случае не будет создаваться виртуальное перемещение.</span><span class="sxs-lookup"><span data-stu-id="2b600-113">Likewise, if only issues occur on the date, there are no receipts to value the average from, and the virtual transfer won't be created.</span></span> <span data-ttu-id="2b600-114">При использовании взвешенного среднего по дате можно пометить складские проводки так, чтобы конкретный приход номенклатуры сопоставлялся с конкретным расходом.</span><span class="sxs-lookup"><span data-stu-id="2b600-114">When you use weighted average date, you can mark inventory transactions so that a specific item receipt is settled against a specific issue.</span></span> <span data-ttu-id="2b600-115">В этом случае правило взвешенного среднего по дате не используется.</span><span class="sxs-lookup"><span data-stu-id="2b600-115">In this case, you don't use the weighted average date rule.</span></span> <span data-ttu-id="2b600-116">При использовании складской модели взвешенного среднего по дате рекомендуется ежемесячно выполнять закрытие запасов.</span><span class="sxs-lookup"><span data-stu-id="2b600-116">We recommend a monthly inventory closing when you use the weighted average date inventory model.</span></span> 

<span data-ttu-id="2b600-117">Расчет себестоимости по методу взвешенного среднего по дате производится по следующей формуле:</span><span class="sxs-lookup"><span data-stu-id="2b600-117">The following formula is used to calculate the weighted average date costing method:</span></span>
