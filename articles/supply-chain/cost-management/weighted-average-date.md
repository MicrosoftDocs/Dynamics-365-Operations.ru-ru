# <a name="weighted-average-date-with-include-physical-value-and-marking"></a>Взвешенное среднее по дате с включенной физической стоимостью и маркировкой

[!include [banner](../includes/banner.md)]

*Взвешенное среднее по дате* — это складская модель, основанная на среднем значении, которое вычисляется умножением каждого компонента (проводки по номенклатуре) на коэффициент (себестоимость), отражающий его важность (количество) на каждый день в периоде. Другими словами, эта складская модель назначает затраты на проводки расхода на основе среднего значения всех складских запасов, полученных за каждый день плюс все запасы в наличии с предыдущего дня.

При выполнении закрытия запасов с использованием складской модели взвешенного среднего по дате можно создать сопоставление двумя методами. Обычно все приходы сопоставляются с виртуальным расходом, в котором учитываются полученное количество и стоимость. Этот виртуальный расход имеет соответствующий виртуальный приход, из которого выполняется сопоставление расходов. Таким образом, все расходы получают одинаковую среднюю стоимость каждый день. Виртуальный расход и виртуальный приход могут рассматриваться как виртуальные перемещение. Это виртуальное перемещение называется *перемещением закрытия запасов с взвешенным средним значением*. Таким образом, этот метод сопоставления называется *суммарным сопоставлением со взвешенным средним*. Если существует только один приход, все расходы могут быть сопоставлены с ним, и виртуальное перемещение не создается. Этот метод сопоставления называется *прямым сопоставлением*. Любые запасы в наличии после закрытия запасов оцениваются по взвешенному среднему из последнего дня предыдущего периода и включаются в расчет взвешенного среднего по дате в следующем периоде.

Можно переопределить принцип взвешенного среднего по дате, отметив проводки по запасам так, чтобы конкретный приход номенклатуры сопоставлялся с конкретным расходом. Периодическое закрытие запасов требуется при использовании складской модели взвешенного среднего по дате для создания сопоставлений и корректировки значения выпусков в соответствии с принципом взвешенного среднего по дате. До тех пор, пока не будет запущено закрытие запасов, проводки расхода оцениваются по скользящему среднему, когда были выполнены физические и финансовые обновления. Если не используется маркировка, скользящее среднее вычисляется, когда выполняется физическое или финансовое обновление.

Калькуляция в методе средневзвешенной по дате стоимости запасов производится с использованием приведенной формулы для каждого дня:

*Затраты* = \[(*Q*<sub>*b*</sub> × *P*<sub>*b*</sub>) + &#x2211;<sub>*n*</sub>(*Q*<sub>*n*</sub> × *P*<sub>*n*</sub>)\] ÷ (*Q*<sub>*b*</sub> + &#x2211;<sub>*n*</sub>*Q*<sub>*n*</sub>)

- *Q* = количество по проводке
- *P* = цена по проводке

Другими словами, взвешенные средние затраты равны значению начального количества, умноженному на начальную цену, плюс сумма всех произведений количества прихода на цену прихода, все это поделенное на начальное количество плюс сумму количеств приходов.

Во время закрытия запасов расчет выполняется ежедневно в течение периода закрытия.

> [!NOTE]
> Дополнительные сведения о сопоставлениях см. в разделе [Закрытие запасов](inventory-close.md).

Следующие примеры показывают эффект использования средневзвешенного значения по дате с пятью конфигурациями:

- Прямое сопоставление взвешенного среднего по дате, когда параметр **Включать физическую стоимость** не используется
- Суммарное сопоставление взвешенного среднего по дате, когда параметр **Включать физическую стоимость** не используется
- Прямое сопоставление взвешенного среднего по дате, когда параметр **Включать физическую стоимость** используется
- Суммарное сопоставление взвешенного среднего по дате, когда параметр **Включать физическую стоимость** используется
- Взвешенное среднее по дате, когда используется маркировка

## <a name="weighted-average-date-direct-settlement-when-the-include-physical-value-option-isnt-used"></a>Прямое сопоставление взвешенного среднего по дате, когда параметр "Включать физическую стоимость" не используется

Принцип прямого сопоставления создает сопоставления непосредственно между приходом и расходом без создания дополнительных складских проводок. Система использует принцип прямого сопоставления в следующих ситуациях:

- Один приход и один или более расходов были разнесены в данном периоде.
- В данном периоде были разнесены только расходы, и запасы содержат номенклатуры в наличии из предыдущего закрытия.

В этом примере снят флажок **Включать физическую стоимость** на странице **Группа моделей номенклатуры** для выпущенного продукта. Следующая диаграмма показывает эти проводки:

**День 1:**

- 1a. Физический приход запасов в количестве равном 10 при себестоимости каждой единицы 10,00 долларов США.
- 1b. Финансовый приход запасов в количестве равном 10 при себестоимости каждой единицы 10,00 долларов США.
- 2a. Физический приход запасов в количестве равном 10 при себестоимости каждой единицы 20,00 долларов США.
- 3a. Физический расход запасов в количестве равном 1 при себестоимости USD 10,00 (скользящее среднее финансово разнесенных проводок).
- 3b. Физический расход запасов в количестве равном 1 при себестоимости USD 10,00 (скользящее среднее финансово разнесенных проводок).

**День 2:**

- 4a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 25,00 долларов США.
- 5a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 5б. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 6a. Физический расход запасов в количестве равном 1 при себестоимости 20,00 долларов США каждый (скользящее среднее финансово разнесенных проводок).

**День 3:**

- 7. Выполняется закрытие запасов. В зависимости от метода взвешенного среднего по дате, система использует метод прямого сопоставления за 30 декабря (30.12), поскольку только одна приемка финансово обновляется 30.12. В этом примере создается одно сопоставление между проводками 1b и 3b. Выполняется корректировка на 10,00 долл. США, чтобы вернуть значение проводки 3b к 20,00. Корректировка или сопоставление не выполняются на 31 декабря (31.12), поскольку отсутствуют финансово обновленные расходы на 31.12.

Нижеследующая диаграмма демонстрирует серию проводок и влияние использования складской модели с средневзвешенным значением и принципа прямого сопоставления без параметра **Включать физическую стоимость**.

![Прямое сопоставление взвешенного среднего по дате без параметра "Включать физическую стоимость".](media/weighted-average-date-direct-settlement-without-include-physical-value.png)

**Ключ к схеме:**

- Складские проводки представлены вертикальными стрелками.
- Физические проводки представлены в виде укороченных серых стрелок.
- Финансовые проводки представлены в виде более длинных черных стрелок.
- Приходы на склад представлены вертикальными стрелками над осью.
- Расходы со склада представлены вертикальными стрелками под осью.
- Каждая новая проводка прихода или расхода обозначается новой меткой.
- Каждая вертикальная стрелка обозначается последовательным идентификатором, например *1a*. Идентификаторы определяют последовательность разноски проводок по запасам по оси времени.
- Даты разделяются тонкими черными вертикальными линиями. Даты отмечаются в нижней части диаграммы.
- Закрытия запасов представлены вертикальными красными пунктирными линиями.
- Сопоставления, которые выполняются путем закрытия склада, представлены красными стрелками, нанесенными пунктиром в диагональном направлении от прихода к расходу.

## <a name="weighted-average-date-summarized-settlement-when-the-include-physical-value-option-isnt-used"></a>Суммарное сопоставление взвешенного среднего по дате, когда параметр "Включать физическую стоимость" не используется

Когда в периоде есть несколько приходов, взвешенное среднее по дате основано на принципе суммарного сопоставления, в котором все приходы в течение одного дня суммируются в проводку, которая называется *закрытием запасов для взвешенного среднего*. Все приходы за этот день будут сопоставлены расходу вновь созданной проводки запасов. Все расходы за день будут сопоставлены приходу новой проводки запасов. Если есть значение оставшихся запасов в наличии после закрытия запасов, значение запасов в наличии включается в проводку прихода проводок закрытия запасов со взвешенным средним.

Следующая диаграмма показывает эти проводки:

**День 1:**

- 1a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 10,00 долларов США.
- 1b. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 10,00 долларов США.
- 2a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 20,00 долларов США.
- 2b. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 22,00 долларов США.
- 3a. Физический расход запасов в количестве равном 1 при себестоимости USD 16,00 (скользящее среднее финансово разнесенных проводок).
- 3b. Физический расход запасов в количестве равном 1 при себестоимости USD 16,00 (скользящее среднее финансово разнесенных проводок).

**День 2:**

- 4a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 25,00 долларов США.
- 5a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 5б. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 6a. Физический расход запасов в количестве равном 1 при себестоимости USD 23,00 (скользящее среднее финансово разнесенных проводок).

**День 3:**

- 7. Выполняется закрытие запасов.
- 7a. Создан финансовый расход проводки закрытия запасов по средневзвешенной стоимости для суммирования сопоставлений всех финансовых приходов на склад.

    - Проводка 1b сопоставляется для количества 1 с сопоставленной суммой 10,00 долл. США.
    - Проводка 2b сопоставляется для количества 1 с сопоставленной суммой 22,00 долл. США.
    - Проводка 7a создается для количества 2 с сопоставленной суммой 32,00 долл. США. Эта проводка засчитывает сумму двух проводок прихода, которые финансово обновляются за период.

- 7b. Создан финансовый приход "Проводка закрытия запасов по средневзвешенной стоимости" как зачет для финансово разнесенных расходов.

    - Проводка 3b сопоставляется для количества 1 с сопоставленной суммой 16,00 долл. США. Эта проводка не скорректирована, поскольку она является взвешенным средним финансово разнесенных проводок 1 декабря (01.12).
    - Проводка 7b создается для количества 2 с финансовой суммой 32,00 долл. США и сопоставленной суммы 16,00 долл. США с корреспондирующей проводкой 3b. Эта проводка засчитывает сумму одной проводки расхода, которые финансово обновляются за период. Проводка остается открытой, так как есть одна в наличии.

На следующей диаграмме показана эта серия проводок с результатами использования складской модели взвешенного среднего и принципа суммарного сопоставления, но без использования параметра **Включать физическую стоимость**.

![Суммарное сопоставление взвешенного среднего по дате без параметра "Включать физическую стоимость".](media/weighted-average-date-summarized-settlement-without-include-physical-value.png)

**Ключ к схеме:**

- Физические проводки представлены в виде укороченных серых стрелок.
- Финансовые проводки представлены в виде более длинных черных стрелок.
- Приходы на склад представлены вертикальными стрелками над осью.
- Расходы со склада представлены вертикальными стрелками под осью.
- Каждая новая проводка прихода или расхода обозначается новой меткой.
- Каждая вертикальная стрелка обозначается последовательным идентификатором, например *1a*. Идентификаторы определяют последовательность разноски проводок по запасам по оси времени.
- Даты разделяются тонкими черными вертикальными линиями. Даты отмечаются в нижней части диаграммы.

## <a name="weighted-average-date-direct-settlement-when-the-include-physical-value-option-is-used"></a>Прямое сопоставление взвешенного среднего по дате, когда параметр "Включать физическую стоимость" используется

В текущей версии продукта параметр **Включать физическую стоимость** в модели запасов по средневзвешенной стоимости по дате действует иначе, чем в предыдущих версиях. Если установить флажок **Включать физическую стоимость** для номенклатуры на странице **Группа номенклатурных моделей**, система использует физически обновленные приходы, затем вычисляет оцененную себестоимость расхода или скользящее среднее. В течение периода расходы будут разноситься на основе этой расчетной себестоимости. Во время закрытия запасов при расчете средневзвешенного значения будут учитываться только финансово обновленные приходы.

Следующая диаграмма показывает эти проводки:

**День 1:**

- 1a. Физический приход запасов в количестве равном 10 при себестоимости каждой единицы 10,00 долларов США.
- 1b. Финансовый приход запасов в количестве равном 10 при себестоимости каждой единицы 10,00 долларов США.
- 2a. Физический приход запасов в количестве равном 10 при себестоимости каждой единицы 20,00 долларов США.
- 3a. Физический расход запасов в количестве равном 1 при себестоимости USD 15,00 (скользящее среднее финансовых и физических разнесенных проводок).
- 3b. Финансовый расход запасов в количестве равном 1 при себестоимости USD 15,00 (скользящее среднее финансовых и физических разнесенных проводок).

**День 2:**

- 4a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 25,00 долларов США.
- 5a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 5б. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 6a. Физический расход запасов в количестве равном 1 при себестоимости 21,25 долларов каждый (скользящее среднее финансовых и физических разнесенных проводок).

**День 3:**

- 7. Выполняется закрытие запасов. В зависимости от метода взвешенного среднего по дате, система использует метод прямого сопоставления за 30 декабря (30.12), поскольку только одна приемка финансово обновляется 30.12. В этом примере создается одно сопоставление между проводками 1b и 3b. Корректировка для расхода от 30.12 не выполняется. Дополнительно, корректировка или сопоставление не выполняются на 31 декабря (31.12), поскольку отсутствуют финансово обновленные расходы на 31.12.

- Каждая вертикальная стрелка обозначается последовательным идентификатором, например *1a*. Идентификаторы определяют последовательность разноски проводок по запасам по оси времени.
- Даты разделяются тонкими черными вертикальными линиями. Даты отмечаются в нижней части диаграммы.
- Закрытия запасов представлены вертикальными красными пунктирными линиями.
- Сопоставления, которые выполняются путем закрытия склада, представлены красными стрелками, нанесенными пунктиром в диагональном направлении от прихода к расходу.

## <a name="weighted-average-date-summarized-settlement-when-the-include-physical-value-option-is-used"></a>Суммарное сопоставление взвешенного среднего по дате, когда параметр "Включать физическую стоимость" используется

В текущей версии продукта параметр **Включать физическую стоимость** работает по другому со средневзвешенной стоимостью, чем в предыдущих версиях. Когда установлен флажок **Включать физическую стоимость** для номенклатуры на странице **Группа номенклатурных моделей**, система будет использовать физически обновленные приходы, затем вычисляет оцененную себестоимость или скользящее среднее. В течение периода расходы будут разноситься на основе этой расчетной себестоимости. Во время закрытия запасов при расчете средневзвешенного значения будут учитываться только финансово обновленные приходы. Мы рекомендуем выполнять ежемесячное закрытие запасов при использовании складской модели средневзвешенного значения. В данном примере суммарного сопоставления средневзвешенной стоимости по дате для складской модели помечен параметр включения физической стоимости.

Следующая диаграмма показывает эти проводки:

**День 1:**

- 1a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 10,00 долларов США.
- 1b. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 10,00 долларов США.
- 2a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 20,00 долларов США.
- 2b. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 22,00 долларов США.
- 3a. Физический расход запасов в количестве равном 1 при себестоимости USD 16,00 (скользящее среднее финансовых и физических разнесенных проводок).
- 3b. Финансовый расход запасов в количестве равном 1 при себестоимости USD 16,00 (скользящее среднее финансовых и физических разнесенных проводок).

**День 2:**

- 4a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 25,00 долларов США.
- 5a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 5б. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 6a. Физический расход запасов в количестве равном 1 при себестоимости USD 23,67 (скользящее среднее финансовых и физических разнесенных проводок).

**День 3:**

- 7. Выполняется закрытие запасов.
- 7a. Создан финансовый расход проводки закрытия запасов по средневзвешенной стоимости для суммирования сопоставлений всех финансовых приходов на склад.

    - Проводка 1b сопоставляется для количества 1 с сопоставленной суммой 10,00 долл. США.
    - Проводка 2b сопоставляется для количества 1 с сопоставленной суммой 22,00 долл. США.
    - Проводка 7a создается для количества 2 с сопоставленной суммой 32,00 долл. США. Эта проводка засчитывает сумму двух проводок прихода, которые финансово обновляются за период.

- 7b. Создан финансовый приход "Проводка закрытия запасов по средневзвешенной стоимости" как зачет для финансово разнесенных расходов.

    - Проводка 3b сопоставляется для количества 1 с сопоставленной суммой 16,00 долл. США. Эта проводка не скорректирована, поскольку она является взвешенным средним финансово разнесенных проводок 1 декабря (01.12).
    - Проводка 7b создается для количества 2 с финансовой суммой 32,00 долл. США и сопоставленной суммы 16,00 долл. США с корреспондирующей проводкой 3b. Эта проводка засчитывает сумму одной проводки расхода, которые финансово обновляются за период. Проводка остается открытой, так как есть одна в наличии.

## <a name="weighted-average-date-when-marking-is-used"></a>Взвешенное среднее по дате, когда используется маркировка

Пометка — это процесс, который позволяет связать (или пометить) проводку расхода с проводкой прихода. Пометить проводку можно до или после того, как она будет разнесена. Можно использовать маркировку, когда требуется определить точную себестоимость запасов при разноске проводки или при выполнении закрытия запасов.

Например, ваш отдел обслуживания клиентов принял срочный заказ от важного клиента. Поскольку это срочный заказ, для выполнения запроса клиента за данную номенклатуру придется заплатить больше. Необходимо гарантировать, чтобы затраты на эту складскую номенклатуру были отражены в прибыли или себестоимости проданных товаров для данной накладной по заказу на продажу.

При разноске заказа на покупку запасы получены по себестоимости равной 120,00 долларов США. Если этому документу заказа на продажу до разноски отборочной накладной или накладной присвоить метку, связывающую его с заказом на покупку, то себестоимость реализованной продукции составит 120,00 долларов США вместо текущего скользящего среднего значения себестоимости для данной номенклатуры. Если маркировка производится после разноски отборочной накладной или накладной по данному заказу на продажу, то себестоимость реализованной продукции будет разнесена в соответствии со скользящим средним значением себестоимости.

До выполнения операции закрытия запасов этим двум проводкам все еще можно присвоить метки таким образом, чтобы связать их друг с другом.

Если проводка прихода отмечена для проводки расхода, выбранный для группы складской модели номенклатуры метод оценки не будет приниматься во внимание, и система будет сопоставлять эти проводки друг с другом.

Можно пометить проводку расхода для проводки прихода до разноски проводки. Вы можете сделать эту маркировку из строки заказа на продажу на странице **Сведения по заказам на продажу**. Открытые проводки прихода отображаются на странице **Маркировка**.

Можно также пометить проводку расхода для проводки прихода после разноски проводки. Вы можете сопоставить или пометить проводку расхода для открытой проводки прихода для инвентаризованной номенклатуры из журнала корректировки разнесенных запасов.

Следующая диаграмма показывает эти проводки:

**День 1:**

- 1a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 10,00 долларов США.
- 1b. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 10,00 долларов США.
- 2a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 20,00 долларов США.
- 2b. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 22,00 долларов США.
- 3a. Физический расход запасов в количестве равном 1 при себестоимости USD 16,00 (скользящее среднее финансово разнесенных проводок).
- 3b. Физический расход запасов в количестве равном 1 при себестоимости USD 16,00 (скользящее среднее финансово разнесенных проводок).
- 3c. Финансовый расход запасов для проводки 3b отмечен как финансовый расход из запасов для проводки 2b.

**День 2:**

- 4a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 25,00 долларов США.
- 5a. Физический приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 5б. Финансовый приход запасов в количестве равном 1 при себестоимости каждой единицы 30,00 долларов США.
- 6a. Физический расход запасов в количестве равном 1 при себестоимости USD 23,00 (скользящее среднее финансово разнесенных проводок).

**День 3:**

- 7. Выполняется закрытие запасов. В зависимости от принципа маркировки, использующего метод взвешенного среднего, помеченные проводки сопоставляются друг с другом. В этом примере проводка 3b сопоставляется с проводкой 2b, а корректировка для 6,00 долларов США разносится в проводке 3b, чтобы вернуть значение в 22,00 долларов США. В этом примере не выполняется никаких дополнительных сопоставлений, поскольку закрытие создает сопоставления только для финансово обновленных проводок.
