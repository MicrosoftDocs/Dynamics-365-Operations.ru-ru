# <a name="production-order-cost-estimation"></a>Оценка стоимости производственного заказа

[!include [banner](../includes/banner.md)]

В Этой статья представлена информация об оценке производственных затрат. Оценка стоимости производства обеспечивает вас информацией о предполагаемых затратах на потребление материалов и мощностей при производстве номенклатуры в том количестве, которое указано в спланированном производственном заказе. 

После создания производственного заказа, необходимо оценить производственных затрат. Назначение данной операции заключается в оценке потребления номенклатур и потребления на маршруте, связанного с производственным процессом, поскольку эти оценки формируют основу для последующих процессов планирования и производства.

## <a name="production-cost-estimation"></a>Оценка стоимости производства
Оценки производственных затрат основаны на следующей информации:

-   Количество в производственном заказе
-   Компоненты в производственных спецификациях (BOM)
-   Операции маршрутизации в Маршруте Производства
-   Косвенные затраты, которые применяются к компонентам и операциям
-   Данные по активным затратам на дату расчета

Если в производственных спецификациях присутствуют строки искусственных номенклатур, расчеты отражают компоненты и операции маршрута искусственной номенклатуры. Задача оценки может использоваться для повторного расчета оценочной стоимости, чтобы они отражали обновление информации. Например, обновленная информация может представлять собой изменения количества по производственному заказу, компонентов производственной спецификации, операций маршрутизации в составе производственного маршрута, косвенных затрат, которые относятся к данным компонентам и операциям, а также активных данных по затратам на дату повторного расчета. В рамках расчета оценочной стоимости также предлагается цена продажи производственной номенклатуры, рассчитанная по принципу "затраты плюс наценка". Вычисления сметных затрат могут дополнительно применяться в отношении базисных заказов, отражающих другие производственные заказы, которые связаны с данным производственным заказом.

## <a name="view-the-estimated-costs"></a>Просмотр оценочных стоимостей
После выполнения оценки можно просматривать результаты на странице **Расчет цены**. При проведении оценки вычисляются следующие значения:

-   **Производственные затраты** – производственные затраты представляют собой верхнюю строку оценки. Они отражают полные затраты на выполнение производственного заказа и итоговую цену продажи для этого производства. Данные затраты представляют собой сумму всех строк затрат, включенных в эту оценку.
-   **Затраты маршрута или ресурса** — Затраты маршрута или ресурса — затраты по производственным операциям. Они включают стоимость таких элементов, как время настройки, время выполнения и накладные расходы.
-   **Затраты на материалы** — Затраты на материалы представляют собой себестоимости и цены компонентов спецификации, необходимых для производства номенклатуры. Они определены и введены в систему заранее.

## <a name="other-uses-of-cost-estimation"></a>Другие способы применения оценки затрат
Оценка затрат также содержит следующую информацию:

-   Осмысленные предложения по ценам
-   Оценки прибыльности этого заказа
-   Оценки использования сырья
-   Сравнения с информацией о затратах на предыдущие производства
-   Бюджетную и прогнозную информацию
-   Оценки объемов производства, необходимых для достижения конкретной величины затрат
