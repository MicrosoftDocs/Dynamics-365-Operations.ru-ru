# <a name="track-running-average-cost-per-inventory-dimension"></a>Отслеживание средних затрат на складскую аналитику

[!include [banner](../includes/banner.md)]

[!include [retail name](../includes/retail-name.md)]

Группа складской аналитики назначается для каждой складской номенклатуры. Поэтому скользящая средняя себестоимость номенклатуры рассчитывается на основе выбранных складских аналитик, в отношении которых осуществляется финансовый контроль.

Существует три типа складских аналитик: по продукту, складу и отслеживанию. Аналитики продукта включают конфигурацию, размер и цвет. Аналитик номенклатур обязательно осуществляется финансовый контроль. Аналитики хранения и отслеживания включают место, склад, местонахождение, статус запасов, номерной знак, номер партии и серийный номер. Вы можете выбрать складские аналитики и аналитики отслеживания, которые будут отслеживаться по финансам. 

**Пример 1** 

Если осуществляется финансовый контроль группы аналитики хранения, закрепленной за номенклатурой, в разрезе склада, то скользящая средняя себестоимость будет рассчитана для каждого склада. По следующим заказам на покупки были выставлены накладные:

-   Для заказа на покупку в количестве 2 шт. по себестоимости USD 10,00 была создана накладная для склада GW.
-   Для заказа на покупку в количестве 3 шт. по себестоимости USD 12,00 была создана накладная для склада GW.
-   Для заказа на покупку в количестве 5 шт. по себестоимости USD 15,00 была создана накладная для склада MW.

Скользящая средняя себестоимость для склада GW равна USD 11,20, а скользящая средняя себестоимость для склада MW равна USD 15,00. Накладная заказа на продажу разносится для склада GW. Стоимость запасов и себестоимость проданных товаров (до закрытия склада и без маркировки) составляет USD 11,20. Другой заказ на продажу разносится для склада MW. Стоимость запасов и себестоимость проданных товаров (до закрытия склада и без маркировки) составляет USD 15,00. 

**Пример 2.** Если осуществляется финансовый контроль по складам для группы аналитики хранения, закрепленной за номенклатурой, а группа аналитик отслеживания финансово отслеживается по номеру партии, скользящая средняя себестоимость вычисляется для каждой партии. 

**Примечание.** Рекомендуется всегда просматривать себестоимость по всем контролируемым финансовым аналитикам. По следующим заказам на покупки были выставлены накладные:

-   Для заказа на покупку в количестве 2 шт. по себестоимости USD 10,00 создана накладная для склада GW, партия ААА.
-   Для заказа на покупку в количестве 3 шт. по себестоимости USD 12,00 создана накладная для склада GW, партия ААА.
-   Для заказа на покупку в количестве 2 шт. по себестоимости USD 15,00 создана накладная для склада GW, партия BBB.

Скользящая средняя себестоимость для склада GW и партии AAA равна USD 11,20, а скользящая средняя себестоимость для склада GW и партии BBB равна USD 15,00.
