# <a name="cost-objects"></a>Объекты затрат

[!include [banner](../includes/banner.md)]

В этой статье представлена информация об объектах затрат и описан порядок накопления затрат и количеств. Объект затрат — объект, для которого накапливаются затраты и количества. Объект объекта затрат может быть или продуктом или вариантами продукта, такими как варианты для стиля и цвета.  

## <a name="cost-objects"></a>Объекты затрат

На странице **Объекты затрат** перечислены все объекты затрат, зарегистрированные для продукта. Объекты затрат определяются данными из следующих источников:

-   Продукт
-   Группа аналитик продуктов
-   Группа аналитик хранения
-   Группа аналитик отслеживания

**Примечание.** Объект затрат представляет только элемент затрат типа **Основные материалы**. Объект затрат и объект запасов отличаются тем, что объект затрат определяется складскими аналитиками, выбранными для финансовых запасов. Например, номенклатура имеет следующую конфигурацию:

-   **Сайт:** физические запасы = да, финансовые запасы = да
-   **Склад:** физические запасы = да, финансовые запасы = нет
-   **Партия №:** физические запасы = да, финансовые запасы = нет

В следующей таблице показано, что такое объект затрат и что такое объект запасов.

| Тип объекта      | Номер номенклатуры | Место | Склад | Партия № |
|------------------|-------------|------|-----------|-----------|
| Объект затрат      | х           | х    |           |           |
| Объект запасов | х           | х    |  х        | х         |

## <a name="accumulation-of-costs-and-quantities"></a>Накопление затрат и количеств
-   Значение в поле **Стоимость** — это сумма следующих значений:
    -   Физ. сумма
    -   Фин. сумма
    -   Корректировки
-   Значение в поле **Количество** — это сумма следующих значений:
    -   Получено
    -   Отпущено
    -   Разнесенное количество
-   Поле **Средняя стоимость единицы** — это расчетное поле. Значение рассчитывается путем деления значения **Стоимость** на значение **Количество**.

**Примечание.** Параметр **Включать физическую стоимость** не влияет на предшествующие расчеты.
