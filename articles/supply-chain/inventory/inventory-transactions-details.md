# <a name="inventory-transaction-details"></a>Подробные сведения о складской проводке

Страница **Сведения о проводках** используется для просмотра подробных сведений о любой выбранной складской проводке.

## <a name="open-the-transaction-details-page"></a>Открытие страницы «Сведения о проводке»

Чтобы открыть страницу **Сведения о проводке**, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Запросы и отчеты \> Проводки**.
1. Выберите проводку для проверки.
1. В области действий выберите **Сведения о проводке**.

## <a name="fasttabs-overview"></a>Обзор экспресс-вкладок

Страница **Сведения о проводках** разделена на несколько экспресс-вкладок. В таблице ниже приведены цели каждой экспресс-вкладки.

| Экспресс-вкладка | Описание |
|---|---|
| Общие | На этой экспресс-вкладке отображаются основные сведения о выбранной складской проводке. В дополнение к полям, отображаемым на странице **Складские проводки**, она включает в себя дополнительные поля, которые описаны далее в этом разделе. |
| Обновления | На этой экспресс-вкладке отображаются сведения о физических, финансовых и сопоставленных аспектах выбранной складской проводки. В зависимости от текущего статуса проводки некоторые поля могут быть пустыми. Однако эти поля будут автоматически установлены при разноске проводок. В дополнение к полям, отображаемым на странице **Складские проводки**, эта экспресс-вкладка включает в себя дополнительные поля, которые описаны далее в этом разделе. |
| Разноски главной книги | На этой экспресс-вкладке показан тип разноски и счет ГК, которые используются для физического обновления, финансового обновления, физического дохода, физических расходов, финансового дохода и финансовых расходов, относящихся к выбранной складской проводке. |
| Ссылки | На этой экспресс-вкладке отображаются дополнительные сведения об исходной проводке, которая создала выбранную складскую проводку. Например, эти сведения могут включать сведения из заказа на покупку или заказа на продажу. |
| Связанные сведения | На этой экспресс-вкладке отображаются дополнительные сведения о выбранной складской проводке. Эти поля могут не быть настроены, если не используются некоторые функции, например категории закупаемой продукции или проекты. |
| Финансовые аналитики – физические | На этой экспресс-вкладке отображаются значения финансовой аналитики, которые использовались при разноске физического обновления. Если физическое обновление еще не разнесено, поля будут пустыми. |
| Финансовые аналитики – финансовые | На этой экспресс-вкладке отображаются значения финансовой аналитики, которые использовались при разноске финансового обновления. Если финансовое обновление еще не разнесено, поля будут пустыми. |
| Складские аналитики | На этой экспресс-вкладке отображаются значения складской аналитики, которые назначены выбранной складской проводке. Эти значения включают место, склад, размер, цвет, конфигурацию, местонахождение, номер партии, серийный номер, статус запасов, номерной знак и владельца. |

## <a name="fields-on-the-general-fasttab"></a>Поля на экспресс-вкладке «Общее»

В следующей таблице описаны поля на экспресс-вкладке **Общее**, которые не отображаются на странице **Складские проводки**.

| Поле | Описание |
|---|---|
| Субъект | Имя соответствующего субъекта для выбранной складской проводки. Например, проводка заказа на покупку показывает название поставщика в заказе на покупку, а проводка заказа на продажу показывает имя клиента. Это поле недоступно для всех типов складских проводок. |
| Складская ссылка | Если другая складская проводка связана с выбранной складской проводкой, это относится к типу другой проводки. Например, если заказ на покупку отмечается в сравнении с заказом на продажу, для поля **Складская ссылка** проводки заказа на покупку будет установлено значение *Заказ на продажу*. Маркировка выполняется автоматически для прямых заказов на поставку и внутрихолдинговых заказов. При использовании маркировки с использованием методов учета затрат, отличных от *стандартных затрат* и *скользящего среднего*, система создаст сопоставления и корректировки для точных проводок, которые помечены. Таким образом, можно добиться «фактического» учета затрат, переопределяющего логику «первым пришел — первым ушел» (ФИФО); «последним пришел, первым ушел» (ЛИФО); или взвешенного среднего. |
| Инвентарный номер | Специальная проводка, которая связана с выбранной складской проводкой. Например, если заказ на покупку отмечается в сравнении с заказом на продажу, для поля **Складской номер** проводки заказа на покупку будет установлен номер заказа на продажу. |
| Код проекта | Если выбранная складская проводка связана с проектом, в этом поле устанавливается номер проекта. |
| Номер лота | Уникальный идентификатор лота, назначенный системой для выбранной складской проводки. |
| Ссылка на лот | Если другая складская проводка связана с выбранной складской проводкой, это относится к идентификатору лота другой проводки. Например, если заказ на покупку отмечается в сравнении с заказом на продажу, для поля **Ссылка на лот** проводки заказа на покупку будет установлено значение **Идентификатор лота** строки заказа на продажу складской проводки. |
| Номер аналитики | Уникальный идентификатор, представляющий комбинацию всех значений складских аналитик, которые назначены выбранной складской проводке. |
| Открытое значение | Значение, указывающее, была ли выбранная складская проводка сопоставлена с процессом закрытия запасов. Если для этого поля установлено значение *Да*, проводка еще не была сопоставлена. |
| Ожидаемая дата | Для проводок по приходу указывается дата ожидаемого прихода запасов. Например, это поле может показывать ожидаемую дату прихода в заказе блокировки запасов или дату поставки в строке заказа на покупку. |
| Дата запасов | Это поле устанавливается, когда проводка регистрации была выполнена в заказе на приемку до разноски физической приемки. |
| Нефинансовый перенос | Значение, указывающее, является ли выбранная складская проводка нефинансовым перемещением. Нефинансовая разноска происходит в том случае, когда изменение складских аналитик не было проведено на странице **Группа номенклатурных моделей**. |
| Номер лота для переноса | Если выбранная складская проводка является нефинансовым перемещением, в этом поле задается значение **Идентификатор лота** другой складской проводки, с которой сопоставляется выбранная проводка. |

## <a name="fields-on-the-updates-fasttab"></a>Поля на экспресс-вкладке «Обновления»

В следующей таблице описаны поля на экспресс-вкладке **Обновления**, которые не отображаются на странице **Складские проводки**.

| Поле | Описание |
|---|---|
| Физический ваучер | Номер ваучера из главной книги, в которой была создана физическая разноска для выбранной складской проводки. |
| Маршрут | Маршрут, который связан с выбранной складской проводкой. Это поле устанавливается только для проводок по отгрузочной накладной производства, когда спецификация или строка формулы связаны с конкретной номенклатурой. |
| Отборочная накладная | Номер отборочной накладной, который был введен для проводки прихода продукта для заказа на покупку. |
| Физ. сумма | Сумма, которая была внесена в главную книгу для физического обновления. Для продукта стандартной себестоимости эта сумма представляет стандартную себестоимость. Для других методов учета затрат он представляет взвешенное среднее для продукта во время разноски. |
| Реализовано физически | Сумма отложенной выручки, которая была внесена в главную книгу для физического обновления. |
| Код загрузки | Если текущая проводка относится к загрузке в модуле «Управление транспортировкой», в этом поле отображается уникальный номер загрузки, в которую включался данный элемент номенклатуры. |
| Разнесено физически | Значение, указывающее, была ли физически разнесена выбранная складская проводка. Если текущая проводка разносится в главную книгу, также будет доступен физический ваучер. |
| Разнесено финансово | Значение, указывающее, была ли финансово разнесена выбранная складская проводка. Если текущая проводка разносится в главную книгу, также будет доступен финансовый ваучер. |
| Физические расходы (разнесенные) | Значение, указывающее, разнесены ли физические расходы, относящиеся к выбранной складской проводке. |
| Разнесенные финансовые расходы | Значение, указывающее, разнесены ли финансовые расходы, относящиеся к выбранной складской проводке. |
| Финансовый ваучер | Номер ваучера в главной книге, в которой была создана финансовая разноска. |
| Накладная | Номер накладной, которая была сформирована, например, для заказа на покупку или заказа на продажу. |
| Корректировка | Сумма, скорректированная для выбранной складской проводки из процесса закрытия и корректировки запасов. |
| Прибыли и убытки, разнесенная сумма | Сумма выбранной складской проводки, которая была разнесена в отчет о прибылях и убытках. |
| Неразнесенная накладная | Номер накладной для соответствующего заказа на покупку, который отображается на странице **Отложенная накладная поставщика**. |
| Финансовое закрытие | Дата окончательного сопоставления проводки. |
| Покрываемое количество в двух единицах измерения | Количество в двух единицах измерения, охватываемое сопоставлением. |
| Сопоставленное количество | Количество, которое было сопоставлено для выбранной складской проводки. |
| Сумма сопоставления | Значение, которое было сопоставлено для выбранной складской проводки. |
