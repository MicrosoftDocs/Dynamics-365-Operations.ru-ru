# <a name="charges-for-nonconformance-operations"></a>Накладные расходы на операции несоответствия

[!include [banner](../includes/banner.md)]

В этой статье описывается, как создавать накладные расходы контроля качества, которые могут использоваться с операциями для несоответствия.

Используйте страницу **Расходы по контролю качества**, чтобы определить типы расходов, которые могут быть добавлены для операций, связанных с несоответствиями. Эти расходы позволяют отслеживать сведения о сборах и накладных расходах, которые вы захотите получить у клиента для несоответствующих продуктов, или о том, что поставщик должен компенсировать га несоответствующие полученные продукты. Можно также использовать накладные расходы для отслеживания затрат, которые необходимы для выполнения операции внешними поставщиками или услугами.

## <a name="examples-of-quality-charges"></a>Примеры расходов по контролю качества

Вы работаете в производственной компании. Ваша компания имеет контракты с несколькими поставщиками. Эти контракты описывают стандарты для отгрузки по времени, ущерба и качества продукта для номенклатур. Аналогично, у нескольких клиентов имеются соглашения с компанией о возврате, убытках и качестве продукта.

Структура сборов определяется для каждой ситуации и указывается в контракте. Можно настроить расходы по контролю качества для расчета различных типов накладных расходов, таких как *Повреждения*, *Поздняя отгрузка* и *Качество*. Затем при создании несоответствия добавляется одна или несколько операций. Для каждой операции выбираются **Накладные расходы** для определения сведений о сборах.

## <a name="create-a-quality-charge"></a>Создание расходов по контролю качества

1. Перейдите в раздел **Управление запасами \> Настройка \> Управление качеством \> Расходы по контролю качества**.
1. На панели операций выберите **Создать**, чтобы добавить строку в сетку. Затем задайте следующие поля для новой строки:

    - **Код накладных расходов** — введите уникальный код или имя для накладного расхода.
    - **Описание** — введите подробное описание расходов по контролю качества.

1. Закройте страницу.
