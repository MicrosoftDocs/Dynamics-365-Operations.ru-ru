# <a name="create-a-new-warehouse-layout"></a>Создание нового макета склада

[!include [banner](../../includes/banner.md)]

В этой статье описывается, как настроить информацию о местонахождениях на складе. Она применяется только к складам, созданным с помощью функции базового управления складами в модуле "Управление запасами", и не применяется к складам, созданным в модуле "Управление складом". Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USMF или собственные данные.


## <a name="set-the-default-location-capacity"></a>Настройка вместимости местонахождения по умолчанию
1. В области перехода выберите **Модули > Управление запасами > Настройка > Параметры модуля Управление запасами и складами**.
2. Выберите вкладку **Ячейки**.
3. В поле **Стандартная ширина** введите число.
4. В поле **Стандартная глубина** введите число.
5. В поле **Стандартная высота** введите число.
6. Нажмите **Сохранить**.
7. Закройте страницу.

## <a name="define-the-location-name-format"></a>Определение формата имени местонахождения
1. В области перехода выберите **Модули > Управление запасами > Настройка > Разделение запасов > Склады**.
2. Выберите **Создать**.
3. В поле **Склад** введите значение.
4. В поле **Имя** введите значение.
5. В поле **Сайт** выберите требуемую запись в поле подстановки.
6. Переключите развертывание раздела **Имена местонахождений**. Параметры в этом разделе определяют формат по умолчанию для имен местонахождений. В нашем примере мы включим номер прохода, номер стеллажа и номер полки.  
7. Установите для параметра **Включать проход** значение **Да**.
8. Установите для параметра **Включать стеллаж** значение **Да**. 
9. В поле **Формат** введите значение для стеллажа.
10. Установите для параметра **Включать полку** значение **Да**.
11. В поле **Формат** введите значение для полки.

## <a name="define-warehouse-locations"></a>Определение местонахождений склада
1. На панели операций выберите **Склад**.
2. Выберите **Мастер ячеек хранения**.
3. Выберите **Далее**.
4. Отмена выбора параметра **Дебаркадеры отгрузки**
5. Отмена выбора параметра **Буферные ячейки**
6. Нажимайте **Далее**, пока не перейдете к параметру для выбора **Готово**.
7. Закройте страницу.
8. Обновите страницу.
