# <a name="create-a-new-warehouse-layout"></a>Создание нового макета склада

[!include[task guide banner](../../includes/task-guide-banner.md)]

Следующая процедура используется для настройки сведений о местонахождениях на складе. Она применяется только к складам, созданным с помощью функции базового управления складами в модуле "Управление запасами", и не применяется к складам, созданным в модуле "Управление складом". Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USMF или собственные данные.


## <a name="set-the-default-location-capacity"></a>Настройка вместимости местонахождения по умолчанию
1. Перейдите в раздел "Управление запасами" > "Настройка" > "Параметры управления запасами и складами".
2. Перейдите на вкладку "Местонахождения".
3. В поле "Стандартная ширина" введите число.
4. В поле "Стандартная глубина" введите число.
5. В поле "Стандартная высота" введите число.
6. Нажмите кнопку "Сохранить".
7. Закройте страницу.

## <a name="define-the-location-name-format"></a>Определение формата имени местонахождения
1. Перейдите Управление запасами > Настройка > Разделение запасов > Склады.
2. Щелкните "Создать".
3. В поле "Склад" введите значение.
4. В поле "Имя" введите значение.
5. В поле "Сайт" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. В списке найдите и выберите требуемую запись.
7. Переключите развертывание раздела "Имена местонахождений".
    * Параметры в этом разделе определяют формат по умолчанию для имен местонахождений. В нашем примере мы включим номер прохода, номер стеллажа и номер полки.  
8. Установите для параметра "Включать проход" значение "Да".
9. Установите для параметра "Включать стеллаж" значение "Да".
10. В поле "Формат" введите значение для стеллажа.
    * Пример: -##  
11. Установите для параметра "Включать полку" значение "Да".
12. В поле "Формат" введите значение для полки.
    * Пример: -##  

## <a name="define-warehouse-locations"></a>Определение местонахождений склада
1. В области действий щелкните "Склад".
2. Щелкните "Мастер местонахождений".
3. Щелкните Далее.
4. Отмена выбора параметра "Дебаркадеры отгрузки"
5. Отмена выбора параметра "Буферные ячейки"
6. Щелкните Далее.
7. Щелкните Далее.
8. Щелкните Далее.
9. Щелкните Далее.
10. Щелкните Далее.
11. Щелкните Далее.
12. Щелкните Далее.
    * Обратите внимание, что физические аналитики, отображаемые на данной странице, — это аналитики, настроенные в начале этой процедуры.  
13. Щелкните Далее.
14. Нажмите кнопку Готово.
15. Закройте страницу.
16. Обновите страницу.
