# <a name="create-and-process-nonconformances"></a>Создание и обработка несоответствий

[!include [banner](../../includes/banner.md)]

В этой теме рассматривается порядок управления несоответствием на основании существующего заказа контроля качества. Обычно управление несоответствиями выполняется сотрудником отдела контроля качества. В качестве обязательного условия должен быть доступен заказ для контроля качества. (Дополнительные сведения о порядке создания заказа для контроля качества см. в разделе [Контроль качества товаров](inspect-quality-goods.md).)

Прежде чем пользователь сможет обработать утверждение несоответствия, работнику необходимо назначить его в поле **Сотрудник** на странице **Пользователи**. Кроме того, прежде чем пользователь сможет использовать примечания к документу, параметр **Разрешить работу с документами** должен быть задан как *Да* в параметрах пользователя.

## <a name="create-a-nonconformance"></a>Создание несоответствия

Чтобы создать несоответствие, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В области действий выберите **Создать**.
1. В диалоговом окне **Создание несоответствия** в поле **Тип проблемы** выберите тип проблемы, которая была обнаружена во время процесса проверки.
1. Нажмите **ОК**.

## <a name="approve-or-reject-a-nonconformance"></a>Утверждение или отклонение несоответствия

Для утверждения или отклонения несоответствия выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В списке выберите несоответствие, которое необходимо обновить.

    > [!NOTE]
    > Исправление можно добавлять только для несоответствий, которые утверждены.

1. В области действий выберите **Функции \> Утвердить несоответствие** для утверждения несоответствия или **Функции \> Отклонить несоответствие** для отклонения. Утвержденные несоответствия можно связать с [соответствующими операциями](../quality-operations.md). Таким образом можно записать работу, которая выполняется как часть обработки несоответствия и обработку исправления.
1. Выводится запрос на подтверждение выбора. Выберите **Да** для продолжения.

## <a name="add-operations-and-other-details-to-nonconformances"></a>Добавьте операции и другие сведения в несоответствия

После создания несоответствия можно начать добавлять связанные операции и указывать дополнительные сведения об этих операциях. Связанные операции можно добавлять только для несоответствий, которые утверждены.

Кроме основной информации, в операцию можно добавить следующие сведения:

- **Номенклатуры** — можно создать список номенклатур, которые потребляются для выполнения исправления. Например, номенклатуры могут быть продуктами, необходимыми для ремонта оборудования или составляющими, которые необходимы для повторной работы готового продукта.
- **Расходы по контролю качества** — можно создать список накладных расходов, которые были понесены или выставляются во внешние источники.
- **Табель учета рабочего времени** — можно создать журнал времени, которое каждый работник тратит на операцию.

Остальные параметры необязательные. Затраты для каждой номенклатуры, расходы по контролю качества и табели учета рабочего времени суммируются и отображаются в операции. Невозможно непосредственно изменить поле **Затраты** для операции.

### <a name="create-an-operation-for-a-nonconformance"></a>Создание операции для несоответствия

Чтобы создать операцию для несоответствия, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В списке выберите несоответствие, которое необходимо обновить.

    > [!NOTE]
    > Операции можно добавлять или обновлять только для несоответствий, которые утверждены.

1. В области операций выберите **Связанные операции**.
1. На странице **Связанные операции** в области действий выберите **Создать**, чтобы добавить строку в сетку. Затем задайте следующие поля для новой строки:

    - **Операция** — выберите код операции, которая будет выполнена для несоответствия.
    - **Причина** — введите подробное описание, объясняющее, почему необходимо выполнить операцию.
    - **Заказ на продажу** — если выбран код операции, связанный с типом заказа на продажу, выберите заказ на продажу, с которым связана выполняемая операция.
    - **Заказ на покупку** — если выбран код операции, связанный с типом заказа на покупку, выберите заказ на покупку, с которым связана выполняемая операция.

1. Закройте страницы.

### <a name="add-items-to-an-operation"></a>Добавление номенклатур в операцию

Чтобы добавить номенклатуры в операцию, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В списке выберите несоответствие, которое необходимо обновить.

    > [!NOTE]
    > Операции можно добавлять или обновлять только для несоответствий, которые утверждены.

1. В области операций выберите **Связанные операции**.
1. На странице **Связанные операции** выберите операцию, в которую необходимо добавить номенклатуры.
1. В области действий выберите **Номенклатуры**.
1. На странице **Связанные операции** в области действий выберите **Создать**, чтобы добавить строку в сетку. Затем задайте следующие поля для новой строки:

    - **Код номенклатуры** — выберите продукт, который будет потреблен как часть выбранной операции.
    - **Количество** — введите количество номенклатур, которое будет потреблено.

    > [!NOTE]
    > Затраты для номенклатуры можно просмотреть в поле **Затраты** на вкладке **Общее**. Отображаемые затраты поступают из затрат, настроенных на странице **Выпущенный продукт**. Затраты не могут быть обновлены непосредственно на странице **Номенклатура связанной операции**. Затраты на все номенклатуры, добавленные на странице **Номенклатуры связанной операции**, автоматически добавляются в поле **Затраты** на странице **Связанные операции**.

1. Повторите предыдущий шаг для каждой дополнительной номенклатуры, которую необходимо добавить.
1. Закройте страницы.

### <a name="add-quality-charges-to-an-operation"></a>Добавление расходов по контролю качества в операцию

Чтобы добавить расходы по контролю качества в операцию, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В списке выберите несоответствие, которое необходимо обновить.

    > [!NOTE]
    > Операции можно добавлять или обновлять только для несоответствий, которые утверждены.

1. В области операций выберите **Связанные операции**.
1. На странице **Связанные операции** выберите операцию, в которую необходимо добавить расходы по контролю качества.
1. В области операций выберите **Расходы по контролю качества**.
1. На странице **Расходы связанной операции** в области действий выберите **Создать**, чтобы добавить строку в сетку. Затем задайте следующие поля для новой строки:

    - **Код накладных расходов** — выберите расходы по контролю качества, которые требуется добавить.
    - **Описание** — введите подробное описание накладных расходов.
    - **Значение расходов** — вводится сумма накладных расходов.

1. Повторите предыдущий шаг для каждого дополнительного накладного расхода, который необходимо добавить.
1. Закройте страницы.

> [!NOTE]
> Сумма в поле **Значение накладных расходов** автоматически суммируется для всех расходов по контролю качества и добавляется ко всем другим суммам в поле **Затраты** на странице **Связанные операции**.

### <a name="create-a-timesheet-on-an-operation"></a>Создание табеля учета рабочего времени для операции

Чтобы добавить табель учета рабочего времени в операцию, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В списке выберите несоответствие, которое необходимо обновить.

    > [!NOTE]
    > Операции можно добавлять или обновлять только для несоответствий, которые утверждены.

1. В области операций выберите **Связанные операции**.
1. На странице **Связанные операции** выберите операцию, в которую необходимо добавить табель учета рабочего времени.
1. В области действий выберите **Табель учета рабочего времени**.
1. На странице **Табели учета рабочего времени связанной операции** в области действий выберите **Создать**, чтобы добавить строку в сетку. Затем задайте следующие поля для новой строки:

    - **Дата** — укажите дату завершения работы. По умолчанию для этого поля задана текущая дата.
    - **Работник** — выберите работника, который выполнил работу. По умолчанию в этом поле задается работник, назначенный текущему пользователю.
    - **Часы работы** — ввод количества часов, отработанных по выбранной операции.
    - **Выставлено накладных** — этот флажок устанавливается, если по времени была выставлена оплата для клиента или поставщика по накладной.

    > [!NOTE]
    > Можно просмотреть затраты в поле **Затраты** на вкладке **Общее**. Затраты рассчитываются с использованием ставки, определенной на странице **Параметры управления запасами**.

1. Повторите предыдущий шаг для каждого дополнительного табеля учета рабочего времени, который необходимо добавить.
1. Закройте страницы.

> [!NOTE]
> Сумма в поле **Затраты** суммируется для всех строк табеля учета рабочего времени и добавляется ко всем другим суммам в поле **Затраты** на странице **Связанные операции**.

## <a name="add-a-correction-to-a-nonconformance"></a>Добавление исправления для несоответствия

Чтобы добавить исправление для несоответствия, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В списке выберите несоответствие, которое необходимо обновить.

    > [!NOTE]
    > Исправление можно добавлять только для несоответствий, которые утверждены.

1. В области действий выберите **Корректировки**.
1. На странице **Корректировки** на панели операций выберите **Создать**, чтобы добавить строку в сетку. Затем задайте следующие поля для новой строки:

    - **Диагностика** — выберите тип диагностики, определяющий тип создаваемой корректировки.
    - **Работник** — выберите лицо, несущее ответственность за корректировку.
    - **Приоритет корректировки** — выберите параметр, чтобы указать приоритет корректировки (*низкий*, *нормальный* или *высокий*).
    - **Запрошенная дата** — введите дату запроса корректирующего действия.
    - **Спланированная дата** — введите дату, в которую ожидается завершение корректировки.
    - **Краткосрочное решение** — установите этот флажок, если корректирующее действие исправляет несоответствие только на короткое время.

1. Закройте страницы.

## <a name="mark-a-correction-as-completed"></a>Отметка корректировки как завершенной

Чтобы отметить корректировку несоответствия как завершенную, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В списке выберите несоответствие, которое необходимо обновить.

    > [!NOTE]
    > Исправление можно добавлять только для несоответствий, которые утверждены.

1. В области действий выберите **Корректировки**.
1. На странице **Корректировки** в сетке выберите корректировку, которое необходимо отметить как завершенную.
1. На панели операций выберите **Отметить как завершенное**.
1. Выводится запрос на подтверждение выбора. Нажмите **ОК**, чтобы продолжить. В поле **Дата и время завершения** устанавливается значение текущих даты и времени, и выбран флажок **Завершено**.
1. Закройте страницу.

## <a name="reopen-a-completed-correction"></a>Повторное открытие завершенной корректировки

Чтобы поврано открыть завершенную корректировку, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Несоответствия**.
1. В списке выберите несоответствие, которое необходимо обновить.
1. В области действий выберите **Корректировки**.
1. На странице **Корректировки** в сетке выберите корректировку, которую необходимо снова открыть.
1. В области действий выберите **Открыть повторно**.
1. Выводится запрос на подтверждение выбора. Нажмите **ОК**, чтобы продолжить. Значение удаляется из поля **Дата и время завершения**, флажок **Завершено** снимается.
1. Закройте страницу.

Теперь можно внести дополнительные изменения или обновления для корректировки. После завершения можно снова отметить корректировку как завершенную.

## <a name="close-a-nonconformance"></a>Закрытие несоответствия

Чтобы закрыть несоответствие, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Управление качеством \> Заказы для контроля качества**.
1. Выберите заказа для контроля качества в сетке.
1. В области действий выберите **Запросы \> Несоответствия**.
1. На странице **Несоответствии** выберите целевое несоответствие в сетке.
1. В области действий выберите **Функции \> Закрыть несоответствия**.
1. Выводится запрос на подтверждение выбора. Выберите **Да** для продолжения.
1. Закройте страницы.
