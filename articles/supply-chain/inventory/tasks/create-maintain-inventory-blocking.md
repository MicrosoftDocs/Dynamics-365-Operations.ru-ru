# <a name="create-and-maintain-an-inventory-blocking"></a>Создание и ведение блокировки запасов

[!include [banner](../../includes/banner.md)]

В этой теме показано, как использовать блокировку запасов во избежание резервирования физических запасов в наличии другими исходящими документами-источниками. Перед началом процедуры в этом разделе необходимо получить номенклатуру с физическими запасами в наличии.

## <a name="block-inventory"></a>Блокирование запасов

Чтобы создать запись блокировки запасов для блокирования запасов, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Блокировка запасов**.
1. В области действий выберите **Создать**.
1. В заголовке новой записи блокировки установите поле **Код номенклатуры** для номенклатуры, которую необходимо заблокировать, и введите описание.
1. На экспресс-вкладке **Общие** в поле **Количество** введите количество блокируемых номенклатур.
1. На экспресс-вкладке **Складские аналитики** укажите сайт и склад, где в данный момент находятся номенклатуры, которые необходимо заблокировать.
1. На панели операций выберите **Сохранить**.

## <a name="update-the-conditions-of-the-inventory-blocking"></a>Обновление условий блокировки запасов

Чтобы обновить запись блокировки запасов, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Блокировка запасов**.
1. В области списка выберите соответствующую запись блокировки.
1. Измените запись требуемым образом. Например, можно изменить значение поля **Ожидаемая дата**, чтобы указать, что заблокированные запасы должны стать доступными для резервирования. Если выбран параметр **Ожидаемые приходы**, дата появится в ожидаемой проводке. (Параметр **Ожидаемые приходы** по умолчанию выбирается при создании записи блокировки вручную.)
1. На панели операций выберите **Сохранить**.

## <a name="unblock-inventory"></a>Разблокировка запасов

Чтобы удалить запись блокировки запасов для разблокирования запасов, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Периодические задачи \> Блокировка запасов**.
1. В области списка выберите соответствующую запись блокировки.
1. В области действий выберите **Удалить**.
1. Выводится запрос на подтверждение операции. Выберите **Да** для продолжения.
1. Закройте страницу.
