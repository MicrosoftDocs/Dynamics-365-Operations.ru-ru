# <a name="correct-inventory-tracking-information"></a>Исправление сведений об отслеживании запасов

[!include [task guide banner](../../includes/task-guide-banner.md)]

Эта процедура содержит пошаговые инструкции по созданию и разноске журнала перемещения запасов для исправления данных отслеживания запасов. В этом примере мы обновим информацию о номенклатуре, контролируемой по партиям, изменив неправильно зарегистрированную партию на другую партию. Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USPI или собственные данные. При использовании собственных данных необходимо иметь номенклатуру, для которой включено отслеживание по партиям, и которая не контролируется по местонахождению. Кроме того, необходимо настроить имя журнала запасов было для перемещения запасов. Эти задачи обычно выполняются работником склада.


## <a name="create-an-inventory-transfer-journal"></a>Создание журнала перемещения запасов
1. Перейдите к "Перемещение".
2. Щелкните "Создать".
3. В поле "Имя" введите или выберите значение.
4. Нажмите кнопку "OК".

## <a name="create-journal-lines"></a>Создать строки журнала
1. Щелкните "Создать".
2. В поле "Код номенклатуры" введите или выберите значение.
    * При использовании USPI выберите номенклатуру M5003.  
3. В поле "Количество" введите число.
4. Перейдите на вкладку "Складские аналитики".
5. В поле "Номер партии" введите или выберите значение.
6. В поле "Сайт" введите или выберите значение.
7. В поле "Склад" введите или выберите значение.
8. В поле "Номер партии" введите или выберите значение.

## <a name="post-the-journal"></a>Разноска журнала
1. Щелкните "Разнести".
2. Нажмите кнопку "OК".

## <a name="check-tracing-information"></a>Проверка сведений о трассировке
1. Щелкните запасы.
2. Щелкните "Трассировка".
3. Нажмите кнопку "OК".
    * Используя эти данные трассировки можно отследить исходную партию в запасах, которая была исправлена.  Для просмотра этих данных также можно использовать страницу "Трассировка номенклатур".  
4. Закройте страницу.

## <a name="check-inventory-transactions"></a>Проверка проводок по запасам
1. Щелкните запасы.
2. Щелкните "Проводки".
    * Здесь можно просмотреть проводки, созданные при разноске журнала.
