# <a name="item-quality-groups"></a>Группы контроля качества номенклатуры

[!include [banner](../includes/banner.md)]

Группа контроля качества определяет общие требования к проверке номенклатур. В этой теме описывается, как использовать и создавать группы контроля качества номенклатур, чтобы логически сгруппировать продукты, чтобы они могли быть назначены для сопоставлений контроля качества для автоматического создания заказов контроля качества.

Для настройки, изменения и просмотра номенклатур, назначенных группе контроля качества, или групп контроля качества, назначенных номенклатуре, перейдите **Управление запасами \> Настройка \> Группы контроля качества**. После определения требованиям к испытанию на странице **Группы проверки** вы можете определить правила для автоматического создания заказов для контроля качества. Для того, чтобы упростить процесс, вы не определяете правила для индивидуальных номенклатур. Вместо этого вы определяете правила для группы качества путем использования страницы **Сопоставления контроля качества**.

## <a name="example-of-an-item-quality-group"></a>Пример группы контроля качества номенклатуры

Производственная компания приобретает различные виды сырья, для которых действуют одинаковые требования по проверке во время входного контроля. Поэтому компания определяет группу качества, и после этого назначает коды номенклатур, которые связаны с сырьем, для этой группы. Позднее компания покупает новый тип сырья, которое имеет такие же требования к испытаниям. Вместо настройки новых требований к испытаниям для нового материала компания добавляет код номенклатуры для нового материала к существующей группе качества.

Эта же компания также производит номенклатуры, которые имеют одинаковые требования в отношении производственных испытаний и одинаковые требования в отношении проведения испытаний перед отгрузкой. Поэтому компания определяет две дополнительные группы контроля качества и назначает соответствующие коды номенклатур каждой группе.

## <a name="create-a-quality-group"></a>Создание группы контроля качества

Чтобы создать группу контроля качества, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Настройка \> Контроль качества \> Группы контроля качества**.
1. На панели операций выберите **Создать**, чтобы добавить строку в сетку. Затем задайте следующие поля для новой строки:

    - **Группа контроля качества** — введите уникальный код или имя группы контроля качества.
    - **Описание** — введите подробное описание группы контроля качества.

1. Закройте страницу.

## <a name="manually-add-items-to-a-quality-group"></a>Добавление номенклатур в группу контроля качества вручную

Чтобы вручную добавить номенклатуры в группу качества, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Настройка \> Контроль качества \> Группы контроля качества**.
1. Выберите группу контроля качества, в которую необходимо добавить номенклатуры.
1. В области действий выберите **Номенклатуры**.
1. На странице **Номенклатуры в группе контроля качества** в области действий выберите **Создать**, чтобы добавить строку в сетку. Затем для новой строки задайте в поле **Код номенклатуры** код номенклатуры, которую требуется добавить.
1. Повторите предыдущий шаг для каждой дополнительной номенклатуры, которую необходимо добавить.
1. Закройте страницы.

## <a name="add-multiple-items-to-a-quality-group"></a>Добавление нескольких номенклатур в группу контроля качества

Чтобы добавить несколько номенклатур в группу качества, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Настройка \> Контроль качества \> Группы контроля качества**.
1. Создайте или выберите группу контроля качества, в которую необходимо добавить номенклатуры.
1. В области действий выберите **Добавить номенклатуры**.
1. В диалоговом окне **Запрос** введите критерии фильтрации для добавляемых номенклатур. Например, можно отфильтровать все номенклатуры в определенной группе номенклатуры.
1. Нажмите **ОК**.
1. В диалоговом окне **Добавление номенклатур** в сетке отображаются все номенклатуры, соответствующие запросу. Проверка результатов.

    Если требуется внести изменения, выберите **Выбрать**, чтобы вернуться в диалоговое окно **Запрос**, и настройте запрос.

1. Когда в результаты включены все номенклатуры, которые требуется добавить, нажмите кнопку **ОК**.
1. Закройте страницу.

## <a name="manually-associate-an-item-with-a-quality-group"></a>Связывание номенклатуры с группой контроля качества вручную

Чтобы вручную связать номенклатуру с группой качества, выполните следующие действия.

1. Перейдите в раздел **Управление запасами \> Настройка \> Контроль качества \> Группы контроля качества номенклатур**.
1. На панели операций выберите **Создать**, чтобы добавить строку в сетку. Затем задайте следующие поля для новой строки:

    - **Код номенклатуры** — выберите код добавляемой номенклатуры.
    - **Группа контроля качества** — выберите группу контроля качества, которая будет назначена номенклатуре.

1. Повторите предыдущий шаг для всех дополнительных комбинаций номенклатуры и группы контроля качества, которые необходимо добавить.
1. Закройте страницы.

## <a name="manually-add-a-quality-group-from-the-released-products-page"></a>Добавление группы контроля качества со страницы выпущенных продуктов вручную

Чтобы вручную добавить группу контроля качества со страницы **Выпущенные продукты**, выполните следующие действия.

1. Выберите **Управление сведениями о продукте \> Продукты \> Выпущенные продукты**.
1. Выберите продукты, которым необходимо назначить группу контроля качества.
1. На панели действий на вкладке **Управление складскими запасами** в группе **Качество** выберите **Группы контроля качества номенклатур**.
1. На странице **Номенклатуры в группе контроля качества** в области действий выберите **Создать**, чтобы добавить строку в сетку. Затем для новой строки задайте поле **Группа контроля качества** для группы контроля качества, которую необходимо назначить продукту.
1. Повторите предыдущий шаг для каждой дополнительной группы контроля качества, которую необходимо назначить продукту.
1. Закройте страницы.
