<span data-ttu-id="11cc6-109">Для настройки статусов ретробонусов перейдите в раздел **Управление ретробонусами \> Настройка \> Статус \> Статус**.</span><span class="sxs-lookup"><span data-stu-id="11cc6-109">To set up rebate statuses, go to **Rebate management \> Setup \> Status \> Status**.</span></span> <span data-ttu-id="11cc6-110">Затем используйте кнопки на панели операций для добавления и удаления значений статусов по мере необходимости.</span><span class="sxs-lookup"><span data-stu-id="11cc6-110">Then use the buttons on the Action Pane to add and remove status values as required.</span></span> <span data-ttu-id="11cc6-111">Для каждого статуса установите следующие поля:</span><span class="sxs-lookup"><span data-stu-id="11cc6-111">For each status, set the following fields:</span></span>
