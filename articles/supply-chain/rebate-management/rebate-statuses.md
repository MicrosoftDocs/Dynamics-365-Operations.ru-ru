# <a name="rebate-statuses"></a>Статусы ретробонусов

[!include [banner](../includes/banner.md)]

Статусы ретробонусов помогают отслеживать статус каждой сделки управления ретробонусами. Их можно также использовать для целей отчетности и запросов. Можно настроить и задать имена для стольких значений статуса ретробонусов, сколько требуется для поддержки бизнес-процесса. 

Для настройки статусов ретробонусов перейдите в раздел **Управление ретробонусами \> Настройка \> Статус \> Статус**. Затем используйте кнопки на панели операций для добавления и удаления значений статусов по мере необходимости. Для каждого статуса установите следующие поля:

- **Статус ретробонуса** — введите имя для статуса (например, *Согласование*, *Согласовано* или *Завершено*).
- **Описание** — введите описание статуса, чтобы предоставить дополнительную информацию о том, как он должен использоваться.

На [странице **Параметры управления ретробонусами**](rebate-management-parameters.md) можно выбрать статус по умолчанию. Этот статус будет затем назначен всем новым создаваемым сделкам управления ретробонусами.
