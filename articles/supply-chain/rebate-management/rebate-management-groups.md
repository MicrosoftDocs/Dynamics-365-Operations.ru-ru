# <a name="rebate-management-groups"></a>Группы управления ретробонусами

[!include [banner](../includes/banner.md)]

Расчеты управления ретробонусами могут управляться группами. Группы управления ретробонусами могут создаваться для клиентов, поставщиков и номенклатур. Они могут быть привязаны к главной записи.

## <a name="rebate-management-customer-groups"></a>Группы клиентов для управления ретробонусами

Расчет для группы клиентов часто создается для цепочки компаний, например, сеть супермаркетов или франчайзинговая компания. Этот тип расчета обычно относится к ретробонусу.

Вычет часто рассчитывается без учета того, кому был продан соответствующий заказ. Однако есть исключения. Например, лицензиат может создать схему вычетов, в которой группа клиентов A получит 4 процента, но группа клиентов B получит только 3 процента.

### <a name="set-up-customer-groups"></a>Настройка групп клиентов

Для работы с группами клиентов по управлению ретробонусами перейдите в раздел **Управление ретробонусами \> Настройка групп управления ретробонусами \> Группы клиентов**. Можно использовать кнопки на панели операций для добавления и удаления групп по мере необходимости. Для каждой группы установите следующие поля:

- **Группа управления ретробонусами** — введите уникальное имя для группы.
- **Описание** — введите описание группы, чтобы предоставить дополнительную информацию о том, как она должна использоваться.

### <a name="manage-customer-group-assignments"></a>Управление назначениями групп клиентов

Каждый клиент может принадлежать любому числу групп управления ретробонусами. Для просмотра и назначения участников группы можно начать со списка группы или списка клиентов.

Чтобы просмотреть, добавить или удалить клиентов для выбранной группы, выполните следующие действия.

1. Перейдите в раздел **Управление ретробонусами \> Настройка групп управления ретробонусами \> Группы клиентов**.
1. Выберите группу для управления.
1. На панели операций выберите **Клиенты**. Отображается страница **Группы управления ретробонусами**, на которой отображается список клиентов, которые уже являются участниками выбранной группы.
1. Чтобы добавить нового клиента в группу, выберите **Создать** на панели действий, чтобы добавить новую строку в сетку. Затем задайте следующее поле для новой строки:

    - **Учетная запись клиента** — выберите идентификатор учетной записи клиента.

1. Чтобы удалить клиента из группы, выберите клиента, затем выберите **Удалить** на панели операций.

Чтобы просмотреть, добавить или удалить назначения группы для выбранного клиента, выполните следующие действия.

1. Перейдите в раздел **Расчеты с клиентами \> Клиенты \> Все клиенты**.
1. Выберите клиента, с которым необходимо работать.
1. В области действий на вкладке **Клиент** в группе **Управление ретробонусами** выберите **Группы управления ретробонусами**. Отображается страница **Группы управления ретробонусами**, на которой отображается список групп, в которые уже входит выбранный клиент.
1. Чтобы добавить клиента в новую группу, выберите **Создать** на панели действий, чтобы добавить новую строку в сетку. Затем задайте следующее поле для новой строки:

    - **Группа управления ретробонусами** — выберите группу, в которую требуется добавить клиента.

1. Чтобы удалить клиента из группы, выберите группу, затем выберите **Удалить** на панели операций.

## <a name="rebate-management-vendor-groups"></a>Группы поставщиков для управления ретробонусами

## <a name="item-rebate-groups"></a>Группы бонусов номенклатуры

Путем группировки номенклатур обеспечивается большая гибкость при расчете ретробонусов и вычетов. Такой подход часто является более эффективным способом настройки ретробонусов и вычетов для клиентов и поставщиков.

### <a name="set-up-item-groups"></a>Настройка групп номенклатуры

Для работы с группами номенклатур по управлению ретробонусами перейдите в раздел **Управление ретробонусами \> Настройка групп управления ретробонусами \> Группы номенклатур**. Можно использовать кнопки на панели операций для добавления и удаления групп по мере необходимости. Для каждой группы установите следующие поля:

- **Группа управления ретробонусами** — введите уникальное имя для группы.
- **Описание** — введите описание группы, чтобы предоставить дополнительную информацию о том, как она должна использоваться.

### <a name="manage-item-group-assignments"></a>Управление назначениями групп номенклатур

Каждая номенклатура может принадлежать любому числу групп управления ретробонусами. Для просмотра и назначения участников группы можно начать со списка группы или списка номенклатур. Можно также добавлять номенклатуры, используя иерархию категорий.

Чтобы просмотреть, добавить или удалить номенклатуры для выбранной группы, выполните следующие действия.

1. Перейдите в раздел **Управление ретробонусами \> Настройка групп управления ретробонусами \> Группы номенклатур**.
1. Выберите группу для управления.
1. В области действий выберите **Номенклатуры**. Отображается страница **Группы управления ретробонусами**, на которой отображается список номенклатур, которые уже являются участниками выбранной группы.
1. Чтобы добавить номенклатуру в группу, выберите **Создать** на панели действий, чтобы добавить новую строку в сетку. Затем задайте следующее поле для новой строки:

    - **Учетная запись номенклатуры** — выберите идентификатор учетной записи номенклатуры.

1. Чтобы удалить номенклатуру из группы, выберите номенклатуру, затем выберите **Удалить** на панели операций.

Чтобы просмотреть, добавить или удалить назначения группы для выбранной номенклатуры, выполните следующие действия.

1. Выберите **Управление сведениями о продукте \> Продукты \> Выпущенные продукты**.
1. Выберите номенклатуру, с которой необходимо работать.
1. В области действий на вкладке **Продукт** в группе **Управление ретробонусами** выберите **Группы управления ретробонусами**. Отображается страница **Группы управления ретробонусами**, на которой отображается список групп, в которые уже входит выбранная номенклатура.
1. Чтобы добавить номенклатуру в новую группу, выберите **Создать** на панели действий, чтобы добавить новую строку в сетку. Затем задайте следующее поле для новой строки:

    - **Группа управления ретробонусами** — выберите группу, в которую требуется добавить номенклатуру.

1. Чтобы удалить номенклатуру из группы, выберите группу, затем выберите **Удалить** на панели операций.

Чтобы добавить номенклатуры в группу на основе иерархии категорий, выполните следующие действия.

1. Перейдите в раздел **Управление ретробонусами \> Настройка групп управления ретробонусами \> Группы номенклатур**.
1. Выберите группу для управления.
1. В области действий выберите **Добавить номенклатуры**.
1. В диалоговом окне **Добавить номенклатуры** в поле **Иерархия категорий** выберите категорию верхнего уровня.
1. Иерархия номенклатур открывается в левой области. Выберите уровень иерархии, который вы ищете. 
1. Номенклатуры из выбранной иерархии и уровня иерархии отображаются в правой области. Для работы с панелью выполните следующие действия:

    - Установите флажок для каждой номенклатуры, которую следует добавить.
    - Установите флажок в заголовке сетки, чтобы выбрать все перечисленные номенклатуры.
    - Выберите кнопку **Показать выбранные**, чтобы выполнить фильтрацию сетки так, чтобы отображались только те номенклатуры, которые были выбраны на данный момент времени. Снова выберите эту кнопку, чтобы вернуться к полному списку.

1. Выберите **ОК**, чтобы применить выбранные номенклатуры к выбранной группе.
