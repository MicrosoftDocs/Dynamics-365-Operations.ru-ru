# <a name="rebate-management-deal-workflows"></a>Рабочие процессы сделок управления ретробонусами

[!include [banner](../includes/banner.md)]

Для утверждения сделок ретробонусов в управлении бонусами используется та же платформа рабочих процессов, что и в других приложениях Finance and Operations. С каждым рабочим процессом связаны для процесса заданий:

- Один элемент рабочего процесса утверждает сделку.
- Один элемент рабочего процесса активирует сделку, чтобы пользователь или рабочий процесс мог утвердить проводки.

Прежде чем можно будет использовать сделку ретробонусов, она должна быть активна в модуле **Управление ретробонусами**. Чтобы активировать сделку, необходимо сначала создать и настроить *Рабочий процесс управления ретробонусами*.

Пользователи не могут утверждать сделки вручную. Этот рабочий процесс должен использоваться всегда.

## <a name="create-and-manage-rebate-management-deal-workflows"></a>Создание рабочих процессов сделок управления ретробонусами и управление ими

Для работы с вашими рабочими процессами сделок управления ретробонусами перейдите в раздел **Управление ретробонусами \> Настройка \> Рабочие процессы управления ретробонусами**. Там можно просмотреть, создать и обновить рабочие процессы, если это необходимо. В каждый момент времени может быть активен только один рабочий процесс этого типа. Дополнительные сведения о рабочих процессах, работе со страницей **Рабочие процессы управления ретробонусами** и создании рабочих процессов см. в разделе [Обзор системы рабочих процессов](../../fin-ops-core/fin-ops/organization-administration/overview-workflow-system.md) и связанных с ним разделах.

## <a name="use-a-workflow-to-activate-a-deal"></a>Использование рабочего процесса для активации сделки

Чтобы активировать сделку через рабочий процесс, откройте эту сделку (например, на странице **Все сделки управления ретробонусами**). Затем на панели операций выберите **Рабочий процесс \> Отправить**. После того как новая сделка была обработана и утверждена в рамках рабочего процесса, она будет активна и готова к использованию.

После активации сделки изменить большую часть настроек невозможно. Если необходимо изменить активную сделку, ее сначала следует деактивировать, а затем создать новую сделку. Если новая сделка будет похожа на старую, ее можно создать путем копирования старой сделки.

Можно изменить следующие параметры для сделки после ее активации:

- Выверка по
- Кумулятивная гарантия
- Профиль разноски
- Профиль разноски для гарантии
- Примечания к документам
- Валютное
- С даты
- По дату

Кроме того, можно удалить строки ретробонусов.
