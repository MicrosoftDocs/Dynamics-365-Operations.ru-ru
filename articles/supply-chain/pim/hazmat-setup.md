# <a name="set-up-hazardous-materials"></a>Настройка опасных материалов

[!include [banner](../includes/banner.md)]

Чтобы использовать функцию опасных материалов, необходимо сначала настроить данные, необходимые для классификации номенклатур как опасные материалы. Затем при создании заказа на продажу, включающего номенклатуру, которая классифицируется как опасный материал, система создает документацию на опасные материалы для этого заказа на продажу, когда он отгружается.

## <a name="turn-on-the-hazardous-materials-feature-for-your-system"></a>Включение функции опасных материалов для вашей системы

Прежде чем использовать эту функцию, она должна быть включена в системе. Администраторы могут использовать параметры [управления компонентами](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) для проверки статуса функции и ее включения. В рабочей области **Управление функциями** эта функция перечисляется следующими способами:

- **Модуль:** *Управление сведениями о продукте*
- **Название компонента:** *Информация о продукте и документация по отгрузке опасных материалов*

## <a name="hazardous-material-regulations"></a>Правила для опасных материалов

Чтобы использовать процессы для опасных материалов необходимо сначала создать регламентирование. Нормативные правила определяют, как создается печатный текст отгрузки для номенклатуры. Они также определяют связанные режимы доставки.

Вот несколько общих нормативных правил:

- **ADR** — правила, которые относятся к международной перевозке опасных товаров по дорогам
- **CFR 49** — правила в США для перевозки опасных товаров
- **IMDG** — кодекс International Marine Dangerous Goods (IMDG)
- **IATA** — правила для опасных товаров международной ассоциации воздушного транспорта (IATA)

Эти правила помогают определить, какую информацию следует отображать при печати текста отгрузки для номенклатуры. Можно определить столько нормативных правил, сколько необходимо соблюдать.

> [!IMPORTANT]
> Функция настройки кодов информации, которые относятся к опасным материалам, не делают вашу компанию совместимой с нормативными правилами. Это только инструмент, который поможет вам создавать процессы для вашей компании.

Обычно регламентирование доступно для конкретного режима транспортировки, такого как перевозка морским, автомобильным или воздушным транспортом. Таким образом, каждое нормативное правило можно связать с режимом доставки. Режим доставки будет использоваться при печати определенных документов в модуле "Управление складом". В модуле "Управление складом" каждая отгрузка связана с перевозчиком и службой перевозчика, которые настроены в модуле **Транспортировка**. Способ доставки связан с перевозчиком и службой перевозчика. При выполнении отчета используется режим доставки, чтобы найти печатный текст отгрузки, связанной с выпущенной номенклатурой.

Эти данные настройки не являются специфичными для каждого юридического лица (компании). Таким образом, можно иметь общий набор сведений об опасных материалах, которые являются общими для всех ваших юридических лиц.

Для каждого нормативного правила можно определить список материалов и использовать его в качестве списка шаблонов, связанных с выпущенными номенклатурами. Для каждого нормативного правила можно также определить настройку печати. Настройка печати позволяет определить, как создается текст отгрузки для номенклатуры. Настройка печати используется для создания печатного текста отгрузки для выпущенной номенклатуры.

Для настройки правил для опасных материалов перейдите на страницу **Управление сведениями о продукте \> Настройка \> Документация по отгрузке опасных материалов \> Правила для опасных материалов**. На странице **Правила для опасных материалов** можно создать любое количество нормативных правил и настроить каждое из них с помощью полей, описанных в следующих подразделах.

В следующей таблице описываются поля, которые доступны на экспресс-вкладке **Способ поставки**.

| Способ поставки | В зависимости от выбранного юридического лица выберите способ поставки, который должен быть связан с данным нормативным правилом. |

### <a name="country-fasttab"></a>Экспресс-вкладка страны

Для справочных целей можно указать страны или регионы, для которых требуется получить сведения о требуемых нормах. Однако при выполнении отчетов по отгрузке для определения нормативных правил используется только способ поставки. При просмотре складской отгрузки нет прямой связи со способом поставки. Отгрузка может быть связана с перевозчиком и службой перевозчика. При определении службы перевозчика необходимо связать ее со способом поставки. Эта связь будет использоваться в отчетах по отгрузке, таких как транспортная накладная, чтобы найти печатный текст отгрузки для номенклатуры.

В следующей таблице описывается поле, которое доступно на экспресс-вкладке **Страна**.

| Поле | описание |
|---|---|
| Страна/регион | Выберите страну/регион, который требуется связать с нормативным правилом. |

## <a name="material-codes"></a><a name="hazmat-codes"></a>Коды материалов

Коды материалов определяют настройки, которые относятся к конкретному опасному компоненту, который может быть включен в выпущенный продукт. Каждый код материала принадлежит конкретному нормативному правилу для вредных материалов, и его определение должно соответствовать этому нормативному правилу. При применении кода материала к выпущенному продукту с использованием поля **Код материала** для этого продукта автоматически применяются все настройки кода материала для опасного материала. Таким образом, процесс настройки выпущенных продуктов быстрее и менее подвержен ошибкам.

Чтобы управлять определениями опасных материалов, выполните следующие действия.

1. Перейдите на страницу **Управление сведениями о продукте \> Настройка \> Документация по отгрузке опасных материалов \> Правила для опасных материалов**.
2. Выберите нормативное правило для которого требуется настроить определение опасного материала.
3. На панели операций откройте вкладку **Настройка** и выберите **Опасные материалы**.
4. В поле **Код материала** введите код материала для определения опасного материала. Это значение будет выбрано при применении кода материала к выпущенному продукту.

    Поле **Код нормативного правила** доступно только для чтения и отображает нормативное правило, выбранное на шаге 2.

5. Оставшиеся поля на этой странице используются для создания и настройки каждого опасного материала, который применяется к выбранному нормативному правилу. Доступные поля являются подмножеством полей опасных материалов, которые доступны для отдельных выпущенных продуктов. Дополнительные сведения см. в разделе [Опасные материалы в продуктах, заказах, отгрузках и загрузках](hazmat-items.md).

## <a name="hazardous-material-classification-groups"></a><a name="classification-groups"></a>Группы классификации опасных материалов

Каждая группа классификации опасных материалов определяет группу значений полей, которые определяют шаблон. Этот шаблон можно использовать позже при настройке сведений об опасных материалах для выпущенной номенклатуры.

При назначении кода для группы классификации опасных материалов для выпущенной номенклатуры информация, связанная с этой группой классификации, будет скопирована в соответствующие поля номенклатуры. Таким образом, можно упростить процессы настройки, установив набор связанных значений полей, которые часто используются вместе.

Эти данные настройки не являются специфичными для каждого юридического лица. Таким образом, можно иметь общий набор сведений об опасных материалах, которые являются общими для всех ваших юридических лиц.

Для настройки групп классификации опасных материалов перейдите на страницу **Управление сведениями о продукте \> Настройка \> Документация по отгрузке опасных материалов \> Группы классификации опасных материалов**. На странице **Группы классификации опасных материалов** можно создать любое количество групп и настроить каждое из них с помощью полей, описанных в следующей таблице.

| Поле | описание |
|---|---|
| Код группы | Введите код для идентификации группы. |
| описание | Введите описание группы. |
| Код класса | Свяжите [код класса](#classes) опасных материалов с группой. |
| Код структурного подразделения | Свяжите [код подразделения](#divisions) опасных материалов с группой. |
| Код группы упаковки | Свяжите [код группы упаковки](#packing-group) с группой. |
| Код категории транспортировки | Свяжите [код категории транспортировки](#transport-category) с группой. |
| Множитель | Введите множитель опасных материалов, который применяется к выбранному классу и подразделению опасных материалов в соответствии с применимым нормативным правилом. Этот множитель используется как часть формулы, которая вычисляет общий *балл опасных материалов*, включенных в загрузку или отгрузку. Дополнительные сведения о баллах опасных материалов и об этом множителе см. в разделе [Экспресс-вкладка "Управление материалами"](hazmat-items.md#material-management). |

## <a name="hazardous-material-classes"></a><a name="classes"></a>Классы опасных материалов

Класс опасных материалов обычно сопоставляется со списком классов, предоставляемых в нормативном правиле, которому вы должны соответствовать. Например, в США в нормативных правилах CFR 49 "класс 3" указан как горючие и взрывоопасные жидкости. Можно настроить классы, относящиеся к материалам, которые необходимо классифицировать.

Каждый класс будет назначен для настройки материала в списке материалов, который относится к данным нормам. Вы назначаете требуемый класс каждой выпущенной номенклатуре для описания опасной природы продукта.

Эти данные настройки не являются специфичными для каждого юридического лица. Таким образом, можно иметь общий набор сведений об опасных материалах, которые являются общими для всех ваших юридических лиц.

Классы опасных материалов используются совместно с подразделениями, группами и группами совместимости следующим образом:

- При назначении выпущенной номенклатуре класса опасных материалов необходимо также назначить [подразделение опасных материалов](#divisions).
- Можно использовать классы опасных материалов вместе с [группами классификации опасных материалов](#classification-groups), чтобы определить шаблон кодов для настройки номенклатур.
- Можно использовать [группы совместимости опасных материалов](#compatibility-groups), чтобы определить, какие классы и подразделения опасных материалов могут быть отгружены вместе.

Для настройки классов опасных материалов перейдите на страницу **Управление сведениями о продукте \> Настройка \> Документация по отгрузке опасных материалов \> Классы опасных материалов**. На странице **Класс опасных материалов** можно создать любое количество классов и настроить каждое из них с помощью полей, описанных в следующей таблице.

| Поле | описание |
|---|---|
| Код класса | Введите код для идентификации этого класса. Этот код определяется для данной номенклатуры. Затем оно будет использоваться в списках подстановки при назначении выпущенной номенклатуре класса опасных материалов. |
| описание | Введите описание класса. |

## <a name="hazardous-material-divisions"></a><a name="divisions"></a>Подразделения опасных материалов

Подразделение опасных материалов является подмножеством класса опасных материалов. Необходимо назначить как подразделение, так и класс для каждого продукта, который содержит опасные материалы.

Для классов, не имеющих подразделений, создайте подразделение с кодом *0*. Например, в правилах ИАТА в классе 7 радиоактивных материалов отсутствуют подразделы. В этом случае вы создадите подразделение *0*, которое можно связать с выпущенным продуктом при назначении класса.

Эти данные настройки не являются специфичными для каждого юридического лица. Таким образом, можно иметь общий набор сведений об опасных материалах, которые являются общими для всех ваших юридических лиц.

Подразделения опасных материалов используются совместно с классами, группами и группами совместимости следующими способами:

- При назначении выпущенной номенклатуре подразделения опасных материалов необходимо также назначить [класс опасных материалов](#classes).
- Можно использовать подразделения опасных материалов вместе с [группами классификации опасных материалов](#classification-groups), чтобы определить шаблон кодов для настройки номенклатур.
- Можно использовать [группы совместимости опасных материалов](#compatibility-groups), чтобы определить, какие классы и подразделения опасных материалов могут быть отгружены вместе.

Для настройки подразделений опасных материалов перейдите на страницу **Управление сведениями о продукте \> Настройка \> Документация по отгрузке опасных материалов \> Подразделение опасных материалов**. На странице **Подразделение опасных материалов** можно создать любое количество подразделений и настроить каждое из них с помощью полей, описанных в следующей таблице.

| Поле | описание |
|---|---|
| Подразделение | Введите код для использования в качестве номера ссылки для подразделения. |
| описание | Введите описание подразделения. |
| Класс | Найдите и назначьте класс, к которому принадлежит подразделение. |

## <a name="hazardous-material-compatibility-groups"></a><a name="compatibility-groups"></a>Группы совместимости опасных материалов

Когда операторы создают складские загрузки или отгрузки, они могут выполнить проверку совместимости, которая выдаст предупреждение, если загрузка или отгрузка включает номенклатуры, которые не все принадлежат к одной группе совместимости.

Эти данные настройки не являются специфичными для каждого юридического лица. Таким образом, можно иметь общий набор сведений об опасных материалах, которые являются общими для всех ваших юридических лиц.

Для настройки групп совместимости опасных материалов перейдите на страницу **Управление сведениями о продукте \> Настройка \> Документация по отгрузке опасных материалов \> Группы совместимости опасных материалов**. На странице **Группы совместимости опасных материалов** можно создать любое количество групп совместимости и настроить каждое из них с помощью полей, описанных в следующих подразделах.

## <a name="hazardous-material-specification-values"></a>Значения спецификации опасного материала

Microsoft Dynamics 365 Supply Chain Management предоставляет несколько типов спецификаций опасных материалов. Для каждого типа необходимо создать централизованный набор значений для каждого соответствующего поля. После этого пользователи могут выбирать эти значения при настройке определений опасных материалов или выпущенных продуктов. Supply Chain Management предоставляет набор страниц, позволяющих установить значения. Каждая страница выделена для одного типа спецификации. Описание каждой доступной спецификации и сведения о том, как определить доступные для нее значения, см. в следующих подразделах.

Одним из примеров спецификации опасного материала является _код укладки_, который определяет, как можно хранить данный материал во время транспортировки. Используя сведения, представленные в этом разделе, вы сможете создать централизованную коллекцию кодов укладки. Эта коллекция затем будет представлена пользователям в раскрывающемся списке, когда они задают код укладки для выпущенного продукта.

Эти значения спецификации опасного материала не являются специфичными для каждого юридического лица. Поэтому можно иметь набор общих значений, которые являются общими для всех ваших юридических лиц.

Вы будете использовать [коды материалов](#hazmat-codes) для создания общих коллекций параметров для каждой спецификации при применении к конкретному правилу. Затем можно при необходимости применить соответствующий код для каждого выпущенного продукта. Сведения о том, как применять код опасного материала к конкретному выпущенному продукту, и о порядке настройки отдельных параметров этого продукта для любой описываемой здесь спецификации см. в разделе [Опасные материалы в продуктах, заказах, отгрузках и загрузках](hazmat-items.md).

### <a name="hazardous-material-identification"></a><a name="identification"></a>Идентификация опасных материалов

Спецификация *Идентификация опасных материалов* определяет класс или природу опасного материала. Обычно это значение является кодом, основанным на стандарте Объединенных Наций (UN). Каждый класс определяется кодом и описанием, и он может устанавливать ограничения для методов транспортировки. Например, чтобы определить огнеопасную номенклатуру или материал, необходимо создать класс опасных материалов, использующий код *FL* и описание *огнеопасно*. Кроме того, необходимо указать, что для этого класса не допускается перевозка воздушным транспортом.

Для настройки значений для этой спецификации перейдите на страницу **Управление сведениями о продукте \> Настройка \> Документация по отгрузке опасных материалов \> Идентификация опасных материалов**. | Поле | описание |
|---|---|
| Идентификация | Введите код, который будет использоваться в качестве ссылочного номера, который определяет данный класс опасных материалов. |
| описание | Введите описание этого класса. |
| Запретить авиационную транспортировку | Установите этот флажок, чтобы указать, что данный класс опасных материалов не должен перевозиться воздушным транспортом. |
| Запретить морскую транспортировку | Установите этот флажок, чтобы указать, что данный класс опасных материалов не должен перевозиться морским транспортом. |
