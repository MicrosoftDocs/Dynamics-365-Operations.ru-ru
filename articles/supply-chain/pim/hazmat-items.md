# <a name="hazardous-materials-in-products-orders-shipments-and-loads"></a>Опасные материалы в продуктах, заказах, отгрузках и загрузках

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

В этой теме объясняется, как задать свойства опасных материалов для выпущенных продуктов, как разместить лимиты запасов для опасных номенклатур и как включать опасные материалы в заказ на продажу, отгрузку или загрузку.

## <a name="set-hazardous-material-specifications-for-products"></a>Задание спецификаций опасных материалов для продуктов

После определения нормативных правил для опасных материалов и настройки связанных кодов ссылки, как описано в разделе [Настройка опасных материалов](hazmat-setup.md), можно связать эту информацию с выпущенными номенклатурами. Текст отгрузки для документов отгрузки будет взят из сведений об опасных материалах для выпущенных номенклатур.

В процессе связывания выпущенной номенклатуры с опасными материалами необходимо указать код нормативных правил и материал. Различные коды нормативного регулирования могут быть связаны с номенклатурой, в зависимости от режима транспортировки, и с каждой номенклатурой можно связать несколько нормативных и материальных кодов.

Чтобы настроить выпущенный продукт в качестве опасного материала, выполните следующие действия.

1. Выберите **Управление сведениями о продукте \> Продукты \> Выпущенные продукты**.
1. Выберите или создайте продукт, чтобы открыть его страницу **Сведения о выпущенном продукте**.
1. На экспресс-вкладке **Управление запасами** установите для параметра **Опасный материал** значение **Да**. Этот параметр определяет номенклатуру как опасный товар и используется при печати документации по отгрузке.
1. На панели действий на вкладке **Управление складскими запасами** в группе **Соответствие** выберите **Опасные материалы номенклатуры**.
1. Заполните страницу **Опасные материалы номенклатуры** для выбранной номенклатуры, используя поля, которые описаны в следующих подразделах.

Нормативные правила определяют, как создается печатный текст отгрузки для номенклатуры и связанных режимов доставки. После назначения код нельзя изменить на этой странице. Однако можно назначить новый код нормативного регулирования, выбрав **Создать**. |
| Код материала | (Необязательно) Выберите код нормативных правил для опасных материалов, которые применяются к продукту. Однако сначала система предложит подтвердить заполнение данных о материалах номенклатуры из кода материала. |

Как и в случае поля **Код материала**, при выборе кода все его спецификации опасных материалов будут скопированы в текущий продукт. Однако сначала система предложит подтвердить заполнение данных о группе класса номенклатуры из кода группы. |

#### <a name="how-the-hazardous-material-score-is-calculated"></a>Способ расчета показателя опасного материала

Некоторые значения, указанные на экспресс-вкладке **Управление материалами** для продукта, используются для расчета *показателя опасного материала* для каждой строки загрузки, содержащей этот продукт. Показатель рассчитывается по следующей формуле:

Показатель опасного материала = *&lt;LineQty&gt;* × *&lt;HazmatQty&gt;* × *&lt;UnitConversion&gt;* × *&lt;Multiplier&gt;*

Вот ключ для формулы:

- *&lt;LineQty&gt;*  — это количество продукта, которое указано для строки загрузки.
- *&lt;HazmatQty&gt;*  — это количество опасного материала, которое указано в поле **Количество** на экспресс-вкладке **Управление материалами**.
- *&lt;UnitConversion&gt;* является коэффициентом преобразования между единицей измерения, используемой для количества в строке загрузки, и единицей измерения, указанной для продукта в поле **Единица измерения** на экспресс-вкладке **Управление материалами**.
- *&lt;Multiplier&gt;*  — это множитель, которое указан для продукта в поле **Множитель** на экспресс-вкладке **Управление материалами**.

Этот показатель выводится для каждой строки загрузки, содержащей продукт, в котором указаны эти значения. Дополнительные сведения см. в разделах [Отгрузки, содержащие опасные материалы](#hazmat-shipments) и [Загрузки, содержащие опасные материалы](#hazmat-loads) далее в этой теме.

#### <a name="how-the-hazardous-material-weight-is-calculated"></a>Способ расчета веса опасного материала

Загрузки и строки загрузки, содержащие продукты, для которых параметр **Ограниченное количество** на экспресс-вкладке **Управление материалами** имеет значение **Да**, будут показывать общий вес опасного материала, как описано в разделах [Отгрузки, содержащие опасные материалы](#hazmat-shipments) и [Загрузки, содержащие опасные материалы](#hazmat-loads) далее в этой теме. Вес опасного материала рассчитывается по следующей формуле:

Вес опасного материала = *&lt;LineQty&gt;* × *&lt;ProductWeight&gt;* × *&lt;UnitConversion&gt;*

Вот ключ для формулы:

- *&lt;LineQty&gt;*  — это количество продукта, которое указано для строки загрузки.
- *&lt;ProductWeight&gt;*  — это чистый вес, указанный для продукта в единицах складского учета, которые указаны для продукта.
- *&lt;UnitConversion&gt;* является коэффициентом преобразования между единицей измерения, используемой для количества в строке загрузки, и единицей складского учета, используемой для параметра *&lt;ProductWeight&gt;*.

## <a name="set-stock-limits-for-hazardous-products"></a><a name="stock-limits"></a>Задание лимитов запасов для опасных продуктов

По соображениям безопасности, возможно, придется ограничить общее количество данного продукта, которое можно хранить в одном местоположении. Чтобы задать пределы запасов для выпущенного продукта, выполните следующие действия.

1. Выберите **Управление сведениями о продукте \> Продукты \> Выпущенные продукты**.
1. Выберите продукт, чтобы открыть его страницу **Сведения о выпущенном продукте**.
1. На панели действий на вкладке **Управление складскими запасами** в группе **Соответствие** выберите **Сведения по отчетности**.
1. В полях **Предел опасных запасов** и **Предел предупреждения об опасных материалах** задайте соответствующие значения для выбранного продукта.

Отчет **Предел запасов опасных материалов** позволяет отслеживать уровни запасов опасных материалов в местоположениях склада, чтобы убедиться, что они остаются ниже безопасных пределов, установленных здесь. Дополнительные сведения см. в разделе [Отчет о пределах запасов опасных материалов](hazmat-reports.md#stock-limit-report).

## <a name="sales-order-transactions-that-include-hazardous-materials"></a>Проводки по заказу на продажу, которые включают опасные материалы

Чтобы включить продукт, который классифицируется как опасный материал по заказу на продажу, необходимо связать соответствующего перевозчика с заказом на продажу. Откройте заказ на продажу, затем на экспресс-вкладке **Доставка** задайте требуемые значения в полях **Перевозчик** и **Услуга перевозчика**.

Перевозчик также связан с режимом доставки. Поэтому необходимо убедиться, что эти сведения выровнены с нормативными правилами по опасным материалам. Другими словами, режим доставки, указанный в нормативных правилах для опасных материалов, должен соответствовать спецификациям в заголовке заказа на продажу. Таким образом, нормативные правила, перевозчик и услуга присоединяются к строкам отгрузки, используемым в заказе на продажу.

После завершения заказа на продажу и его готовности к отгрузке он может быть выпущен на склад, чтобы указать перемещение между операциями продажи и складскими операциями.

## <a name="shipments-that-include-hazardous-materials"></a><a name="hazmat-shipments"></a>Отгрузки, содержащие опасные материалы

### <a name="view-hazardous-material-scores-for-each-shipment-line"></a>Просмотр показателей опасных материалов для каждой строки отгрузки

На странице **Сведения об отгрузке** для отгрузки отображаются общий вес опасного материала и значения баллов, рассчитанные для каждой строки загрузки, включенной в эту отгрузку. Чтобы просмотреть баллы и веса, выполните следующие действия.

1. Выберите **Управление складом \> Отгрузки \> Все отгрузки**.
1. Выберите отгрузку, чтобы открыть его страницу **Сведения об отгрузке**.
1. На экспресс-вкладке **Строки загрузки** проверьте строки. Для каждой строки будут отображены расчеты опасных материалов в следующих полях:

    - **Баллы опасного материала** — в этом поле отображается показатель опасного материала для строки загрузки. Значение рассчитывается в соответствии с правилами и значениями, установленными для соответствующего нормативного правила и в настройке выпущенного продукта. Расчет учитывает количество в строке загрузки и ссылается на множитель в [настройке управления материалами](#material-management) для выпущенного продукта.
    - **Чистый вес ограниченного количества** — для продуктов, помеченных как продукты с ограничением количества из-за наличия содержимого опасного материала, в этом поле отображается общий чистый вес опасного содержимого, которое включено в строку загрузки. Расчет производится на основе продуктов, помеченных как опасные материалы в настройке выпущенного продукта. Если номенклатура помечена как номенклатура с ограничением количества, расчет учитывает количество по каждой строке загрузки и ссылается на вес в [настройке управления материалами](#material-management) для выпущенного продукта.

### <a name="check-for-compatibility-among-hazardous-materials-that-are-included-in-a-shipment"></a>Проверка совместимости среди опасных материалов, включенных в отгрузку

Система может оценить, могут ли все опасные материалы, включенные в отгрузку, быть отгружены вместе. Чтобы оценить совместимость, система проверяет группу совместимости, которая назначена каждому продукту, включенному в отгрузку. Дополнительные сведения см. в разделе [Группы совместимости опасных материалов](hazmat-setup.md#compatibility-groups).

Чтобы выполнить проверку совместимости, выполните следующие действия.

1. Выберите **Управление складом \> Отгрузки \> Все отгрузки**.
1. Выберите отгрузку, чтобы открыть его страницу **Сведения об отгрузке**.
1. В области действий на вкладке **Отгрузки** в группе **Действия** выберите **Проверка совместимости**.

Вы получите сообщение, которое информирует вас о результатах проверки.

## <a name="loads-that-include-hazardous-materials"></a><a name="hazmat-loads"></a>Загрузки, содержащие опасные материалы

### <a name="view-hazardous-material-scores-for-each-load-line"></a>Просмотр показателей опасных материалов для каждой строки загрузки

На странице **Сведения о загрузке** для загрузки отображаются общий вес опасного материала и значения баллов, рассчитанные для этой загрузки и каждой строки загрузки. Чтобы просмотреть баллы и веса, выполните следующие действия.

1. Выберите **Управление складом \> Отгрузки \> Все загрузки**.
1. Выберите загрузку, чтобы открыть его страницу **Сведения о загрузке**. (Можно также открыть сведения о загрузке, выбрав ссылку из соответствующей отгрузки.)
1. На экспресс-вкладке **Загрузка** можно просмотреть общее показатели и веса опасных материалов для всей загрузки, изучив следующие поля:

    - **Баллы опасного материала** — в этом поле отображается показатель опасного материала для загрузки. Значение рассчитывается в соответствии с правилами и значениями, установленными для соответствующего нормативного правила и в настройке выпущенного продукта. Расчет учитывает количество, включенное в загрузку, и ссылается на множитель в [настройке управления материалами](#material-management) для выпущенного продукта.
    - **Чистый вес ограниченного количества** — для продуктов, помеченных как продукты с ограничением количества из-за наличия содержимого опасного материала, в этом поле отображается общий чистый вес опасного содержимого, которое включено в загрузку. Расчет производится на основе продуктов, помеченных как опасные материалы в настройке выпущенного продукта. Если номенклатура помечена как номенклатура с ограничением количества, расчет учитывает количество в каждой загрузке и ссылается на вес в [настройке управления материалами](#material-management) для выпущенного продукта.

1. Для просмотра показателей и весов для отдельных строк перейдите на экспресс-вкладку **Строки загрузки**. Значения, предоставляемые для каждой строки, похожи на значения, предоставляемые для всей загрузки, как описано в предыдущем шаге.

### <a name="check-for-compatibility-among-hazardous-materials-that-are-included-in-a-load"></a>Проверка совместимости среди опасных материалов, включенных в загрузку

Дополнительные сведения см. в разделе [Группы совместимости опасных материалов](hazmat-setup.md#compatibility-groups).
