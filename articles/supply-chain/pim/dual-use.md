# <a name="dual-use-goods"></a>Товары двойного назначения

[!include [banner](../includes/banner.md)]

Товары двойного назначения обычно являются номенклатурами, которые имеют как гражданское, так и военное применение. Например, химическое вещество может использоваться как удобрение или как взрывчатое вещество. У многих стран есть особые нормы, которые применимы к экспорту, импорту и транспортировке товаров двойного назначения. Таким образом, важно, чтобы компании, которые участвует в международной торговле товарами двойного назначения, отслеживали различные политики и сертификаты.

Функция двойного назначения помогает компаниям отслеживать продукты, которые определены как товары двойного назначения, хранить номера сертификатов для каждого соответствующего продукта и страны получателя, а также печатать действительные номера сертификатов в соответствующих счетах, отборочных накладных и/или заказах на продажу. Это помогает гарантировать, что при отгрузке продуктов они всегда содержат самые последние сертификаты.

Рассмотрим следующий пример:

1. Страница **Настройка страны для товаров двойного применения** в системе показывает, что для Франции требуется сертификат.
2. Страница **Сведения о выпущенном продукте** для продукта X-100 указывает на то, что это товар двойного назначения. Вместе код, категория, группа и режим указывают классификацию контроля экспорта, к которой относится продукт.
3. Страница **Сертификаты двойного назначения** содержит сертификат для продукта X-100 при его отгрузке во Францию. Срок действия этого сертификата истекает 1 января 2020 г.
4. 17 июня 2020 г. вы создаете заказ на продажу для компании клиента, которая базируется во Франции, и заказ включает продукт X-100.
5. При сохранении заказа на продажу система определяет следующие сведения:

    1. Имеются ли в заказе продукты, которые являются товарами двойного назначения?
    2. Если заказ включает товары двойного назначения, требуется ли для страны назначения сертификаты двойного назначения?
    3. Если для страны требуется наличие сертификатов двойного назначения, имеется ли действующий сертификат для каждого товара двойного назначения для страны назначения?

6. Заказ включает продукт X-100, продукт отгружается во Францию, и для продукта существует сертификат для Франции. Однако срок действия сертификата истек. Поэтому появляется следующее предупреждающее сообщение: "Сертификаты двойного назначения для одного или нескольких товаров двойного назначения в заказе на продажу недействительны. Вы хотите продолжить подтверждение?"

В этой теме объясняется, как настроить все параметры, необходимые для настройки товаров двойного назначения и поддержки этого сценария.

## <a name="define-dual-use-requirements-for-each-relevant-country"></a>Определение требований к двойному назначению для каждой соответствующей страны

В разных странах имеются различающиеся требования для товаров двойного назначения. Вы используете страницу **Настройка страны двойного назначения** для отслеживания стран, для которых требуется и для которых не требуется сертификат. Введенные здесь сведения проверяются при создании заказов на продажу, и вам будет предоставлено напоминание о необходимых сертификатах.

Чтобы настроить сведения о требованиях к двойному назначению для разных стран, выполните следующие действия.

1. Перейдите на страницу **Управление сведениями о продукте \> Настройка \> Соответствие продуктов \> Продукты двойного назначения \> Настройка стран двойного назначения**.
2. Выберите существующую настройку страны для ее редактирования или выберите **Создать** на панели операций, чтобы создать новую настройку страны.
3. Задайте следующие значения для выбранной или новой настройки страны.

    | Поле | описание |
    |---|---|
    | Страна/регион | Выберите страну, для которой осуществляется отслеживание требований. |
    | Требуется сертификат | Установите этот флажок для тех стран, для которых требуется сертификация на товары двойного назначения. Снимите его для тех стран, для которых сертификат не требуется. |

## <a name="create-dual-use-categories"></a>Создание категорий двойного назначения

Часто товары двойного назначения должны классифицироваться в соответствии с их классификационным номером экспортного контроля (ECCN). ECCN — это алфавитно-цифровой код, который классифицирует номенклатуры на основе таких факторов, как товар и технология. Страница **Категории двойного назначения** помогает создать список категорий, используемых для целей отчетности.

Чтобы настроить категории двойного назначения, сделайте следующее.

1. Перейдите на страницу **Управление сведениями о продукте \> Настройка \> Соответствие продуктов \> Продукты двойного назначения \> Категории двойного назначения**.
2. Выберите существующую категорию для ее редактирования или выберите **Создать** на панели операций, чтобы создать новую категорию.
3. Задайте следующие значения для выбранной или новой категории.

    | Поля | описание |
    |---|---|
    | Код двойного назначения | Введите полный код ECCN (например, *3A001*).|
    | Категория двойного назначения | Введите часть кода ECCN, представляющий категорию списка коммерческого управления (CCL). Например, для кода ECCN *3A001* это значение равно *3*. |
    | Группа двойного назначения | Введите часть кода ECCN, обозначающую группу продуктов. Например, для кода ECCN *3A001* это значение равно *A*. |
    | Режим двойного назначения | Введите код режима для номенклатуры. Этот код определяет причину, по которой номенклатура классифицируется как товар двойного назначения. Например, для кода ECCN *3A001* это значение равно *001*. |

## <a name="apply-dual-use-categories-to-products"></a>Применение категорий двойного назначения к продуктам

Чтобы идентифицировать продукт как товар двойного назначения и применить к нему категорию двойного назначения, выполните следующие действия.

1. Выберите **Управление сведениями о продукте \> Продукты \> Выпущенные продукты**.
1. Выберите или создайте продукт, чтобы открыть его страницу **Сведения о выпущенном продукте**.
1. На экспресс-вкладке **Внешняя торговля** установите для параметра **Продукты двойного назначения** значение **Да**, чтобы обозначить текущий продукт как товар двойного назначения.
1. В поле **Код двойного назначения** задайте код, который применяется к текущему продукту. (Этот код определен на странице **Категории двойного назначения**.)

Эта настройка проверяется при создании заказа на продажу.

## <a name="set-up-dual-use-certificates"></a>Настройка сертификатов двойного назначения

Страница **Сертификаты двойного назначения** используется для настройки и управления необходимыми сертификатами двойного назначения для каждого продукта и страны. Можно отслеживать сведения о каждом сертификате, такие как страна и сроки действия. Кроме того, можно задать параметры, определяющие место печати этих сведений. Например, информация может быть распечатана в счете, отборочной накладной и/или заказе на продажу. Эта настройка проверяется при создании заказа на продажу.

1. Перейдите на страницу **Управление сведениями о продукте \> Настройка \> Соответствие продуктов \> Продукты двойного назначения \> Сертификаты двойного назначения**.
2. Выберите существующий сертификат для его редактирования или выберите **Создать** на панели операций, чтобы создать новый сертификат.
3. Задайте следующие значения для выбранного или нового сертификата.

    | Поле | описание |
    |---|---|
    | Код номенклатуры | Выберите код номенклатуры товара двойного назначения, к которому применяется данный сертификат. |
    | Страна/регион | Страна или регион назначения, для которых необходимо использовать этот сертификат. |
    | Номер сертификата | Номер, который отображается в сертификате, выданном поставщику или клиенту. |
    | Действует | Выберите первую дату, когда текущий сертификат действителен.|
    | Истечение срока действия | Выберите последнюю дату, когда текущий сертификат действителен. |
    | Печать на счете | Установите этот флажок для печати номера сертификата на счетах, адресованных в указанную страну в указанном диапазоне дат. |
    | Печать на отборочной накладной | Установите этот флажок для печати номера сертификата на отборочных накладных, адресованных в указанную страну в указанном диапазоне дат. |
    | Печать на заказе на продажу | Установите этот флажок для печати номера сертификата на заказах на продажу, адресованных в указанную страну в указанном диапазоне дат. |
