# <a name="approve-a-product-configuration-model"></a>Утверждение модели конфигурации продукта

[!include [banner](../../includes/banner.md)]

Выполнение этой процедуры требует наличия не менее одной модели конфигурации продукта. Эта процедура использует модель динамика класса Hi-End в компании USMF с демонстрационными данными. Обратите внимание, что эта модель уже была утверждена, но процедура описывает весь процесс.

1. Перейдите в раздел **Управление сведениями о продукте \> Продукты \> Модели конфигурации продукта**.
1. В списке найдите и выберите требуемую запись.
    * Выберите модель динамика класса Hi--End для этой процедуры.  
1. Выберите **Версии**.
1. Выберите **Создать**.
1. В поле **Номер продукта** введите или выберите значение.
    * Ссылка на продукт, представляет версию модели конфигурации продукта. Только шаблоны продуктов с технологией конфигурации на основе ограничений появятся в данном списке.  
1. В поле **Дата начала** введите дату.
    * Выберите, когда версия модели продукта будет доступна.  
1. В поле **Дата окончания** введите дату.
    * Выберите дату окончания, когда истечет срок действия данной версии модели продукта, или выберите "Никогда".  
1. Выберите **Утвердить**, чтобы открыть ниспадающее диалоговое окно.
1. В поле **Кем утверждено** введите или выберите значение.
    * Выберите лицо, ответственное за утверждение модели продукта для эксплуатации.  
1. Нажмите **ОК**.
1. В поле **Метод ценообразования** выберите один из вариантов.
    * Активируйте версию модели продукта. Возможен только один активный продукт для одной модели продукта одновременно.  
1. Закройте страницу.
