# <a name="create-dimension-based-configurations"></a>Создание конфигураций на основе аналитик

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как определить конфигурацию для продукта на основе аналитик. Это последняя процедура в этой серии, в которой поясняется, как строить сочетания для конфигурации на основе аналитик. Выполнение этой процедуры зависит от данных, созданных в ходе предыдущих семи записей. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="find-the-dimension-based-product-master"></a>Поиск шаблона продукта на основе аналитик
1. Щелкните "Обслуживание выпущенного продукта".
2. Щелкните "Выпущенные продукты".
3. В списке пометьте выбранную строку.
    * Выберите шаблон продукта на основе аналитик, созданный в первой записи в этой последовательности из 8 записей.  

## <a name="create-configurations"></a>Создание конфигураций
1. На панели действий "Инжиниринг" щелкните "Ведение конфигураций".
2. Щелкните Настроить.
3. В списке пометьте выбранную строку.
4. В поле "Код номенклатуры" введите или выберите значение.
    * Выберите любые номенклатуры в первой конфигурационной группе.  
5. В списке найдите и выберите требуемую запись.
6. В поле "Код номенклатуры" введите или выберите значение.
    * Выберите любую номенклатуру из второй конфигурационной группы.  
7. Нажмите кнопку "OК".
8. В списке пометьте выбранную строку.
9. В поле "Конфигурация" введите значение.
    * Введите имя конфигурации, с помощью которого проще идентифицировать конфигурацию.  
10. В поле "Описание" введите значение.
    * Введите описание конфигурации, чтобы объяснить ее содержимое.  
11. Нажмите кнопку "OК".
