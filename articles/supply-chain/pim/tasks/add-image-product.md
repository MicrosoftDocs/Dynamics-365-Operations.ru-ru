# <a name="add-an-image-to-a-product"></a>Добавление изображение к продукту

[!include [banner](../../includes/banner.md)]

Эта процедура демонстрирует порядок добавления изображение к продукту. Для создания этой задачи используются демонстрационные данные компании USMF.

1. Щелкните "Управление сведениями о продукте" > "Продукты" > "Выпущенные продукты".
2. Разверните информационное поле "Изображение продукта".
3. В списке найдите и выберите требуемую запись.
    * Например, выберите код номенклатуры C0002.  
4. Щелкните "Изменить изображение".
5. Щелкните "Создать".
    * Щелкните Обзор. Найдите изображение, которое необходимо использовать, выберите его, затем закройте диалоговое окно.
