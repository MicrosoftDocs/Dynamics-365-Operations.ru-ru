# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="652d9-103">Создание штрих-кода для продукта</span><span class="sxs-lookup"><span data-stu-id="652d9-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="652d9-104">В этой теме описан порядок создания вручную штрих-кода с использованием кода номенклатуры M0001 в качестве примера.</span><span class="sxs-lookup"><span data-stu-id="652d9-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="652d9-105">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="652d9-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="652d9-106">Выберите **Управление сведениями о продукте \> Продукты \> Выпущенные продукты**.</span><span class="sxs-lookup"><span data-stu-id="652d9-106">Go to **Product information management \> Products \> Released products**.</span></span>
1. <span data-ttu-id="652d9-107">В списке найдите и выберите требуемую запись.</span><span class="sxs-lookup"><span data-stu-id="652d9-107">In the list, find and select the desired record.</span></span> <span data-ttu-id="652d9-108">В этом примере выберите код номенклатуры **M0001**.</span><span class="sxs-lookup"><span data-stu-id="652d9-108">For this example, select item number **M0001**.</span></span>
1. <span data-ttu-id="652d9-109">На панели операций выберите **Управление запасами**.</span><span class="sxs-lookup"><span data-stu-id="652d9-109">On the Action Pane, select **Manage inventory**.</span></span>
1. <span data-ttu-id="652d9-110">Выберите **Штрих-коды**.</span><span class="sxs-lookup"><span data-stu-id="652d9-110">Select **Bar codes**.</span></span>
1. <span data-ttu-id="652d9-111">Выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="652d9-111">Select **New**.</span></span>
1. <span data-ttu-id="652d9-112">Отметьте выбранную строку, которая создается в списке ниже.</span><span class="sxs-lookup"><span data-stu-id="652d9-112">Mark the selected row that is created in the list below.</span></span>
1. <span data-ttu-id="652d9-113">В поле **Настройка штрих-кода** введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="652d9-113">In the **Barcode setup** field, enter or select a value.</span></span>
1. <span data-ttu-id="652d9-114">В поле **Штрих-код** введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="652d9-114">In the **Bar code** field, enter or select a value.</span></span>
1. <span data-ttu-id="652d9-115">В поле **Штрих-код** введите значение.</span><span class="sxs-lookup"><span data-stu-id="652d9-115">In the **Bar code** field, type a value.</span></span>  
1. <span data-ttu-id="652d9-116">В поле **Количество** введите число.</span><span class="sxs-lookup"><span data-stu-id="652d9-116">In the **Quantity** field, enter a number.</span></span>
1. <span data-ttu-id="652d9-117">Нажмите **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="652d9-117">Select **Save**.</span></span>
1. <span data-ttu-id="652d9-118">Закройте страницу.</span><span class="sxs-lookup"><span data-stu-id="652d9-118">Close the page.</span></span>
