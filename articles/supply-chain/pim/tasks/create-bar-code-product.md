# <a name="create-a-bar-code-for-a-product"></a>Создание штрих-кода для продукта

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре описан порядок создания вручную штрих-кода с использованием кода номенклатуры M0001 в качестве примера. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Щелкните "Обслуживание выпущенного продукта".
2. Щелкните "Выпущенные продукты".
3. В списке найдите и выберите требуемую запись.
4. В области действий щелкните "Управление запасами".
5. Щелкните "Штрих-коды".
6. Щелкните "Создать".
7. В списке пометьте выбранную строку.
8. В поле "Настройка штрих-кода" введите или выберите значение.
9. В поле "Штрих-код" введите или выберите значение.
10. В поле "Штрих-код" введите значение.
    * Нажмите клавишу TAB.  
11. Закройте страницу.
12. В поле "Количество" введите число.
13. Нажмите кнопку "Сохранить".
    * Когда вы нажмете "Сохранить", выполняется штрих-код, и в этом случае отображается ошибка, что ожидалась контрольная цифра 8, а найдена цифра 3. Вручную обновите номер штрих-кода, чтобы он заканчивался цифрой 8.  
14. В поле "Штрих-код" введите или выберите значение.
15. В поле "Штрих-код" введите значение.
    * Нажмите клавишу TAB.  
16. Закройте страницу.
17. Нажмите кнопку "Сохранить".
18. Закройте страницу.
