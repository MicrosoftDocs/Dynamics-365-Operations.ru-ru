# <a name="create-a-bar-code-for-a-product"></a>Создание штрих-кода для продукта

[!include [banner](../../includes/banner.md)]

В этой теме описан порядок создания вручную штрих-кода с использованием кода номенклатуры M0001 в качестве примера. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Выберите **Обслуживание выпущенного продукта** на главной странице.
2. Выберите **Продукты > Выпущенные продукты** в разделе **Ссылки**.
3. В списке найдите и выберите требуемую запись. В этом примере выберите код номенклатуры **M0001**.
4. На панели операций выберите **Управление запасами**.
5. Выберите **Штрих-коды**.
6. Выберите **Создать**.
7. Отметьте выбранную строку, которая создается в списке ниже.
8. В поле **Настройка штрих-кода** введите или выберите значение.
9. В поле **Штрих-код** введите или выберите значение.
10. В поле **Штрих-код** введите значение.  
11. В поле **Количество** введите число.
12. Нажмите **Сохранить**.
13. Закройте страницу.
