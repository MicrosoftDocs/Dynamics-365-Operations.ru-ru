# <a name="create-a-bar-code-for-a-product"></a>Создание штрих-кода для продукта

[!include [banner](../../includes/banner.md)]

В этой теме описан порядок создания вручную штрих-кода с использованием кода номенклатуры M0001 в качестве примера. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Выберите **Управление сведениями о продукте \> Продукты \> Выпущенные продукты**.
1. В списке найдите и выберите требуемую запись. В этом примере выберите код номенклатуры **M0001**.
1. На панели операций выберите **Управление запасами**.
1. Выберите **Штрих-коды**.
1. Выберите **Создать**.
1. Отметьте выбранную строку, которая создается в списке ниже.
1. В поле **Настройка штрих-кода** введите или выберите значение.
1. В поле **Штрих-код** введите или выберите значение.
1. В поле **Штрих-код** введите значение.  
1. В поле **Количество** введите число.
1. Нажмите **Сохранить**.
1. Закройте страницу.
