# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="dff62-103">Создание штрих-кода для продукта</span><span class="sxs-lookup"><span data-stu-id="dff62-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="dff62-104">В этой теме описан порядок создания вручную штрих-кода с использованием кода номенклатуры M0001 в качестве примера.</span><span class="sxs-lookup"><span data-stu-id="dff62-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="dff62-105">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="dff62-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="dff62-106">Выберите **Обслуживание выпущенного продукта** на главной странице.</span><span class="sxs-lookup"><span data-stu-id="dff62-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="dff62-107">Выберите **Продукты > Выпущенные продукты** в разделе **Ссылки**.</span><span class="sxs-lookup"><span data-stu-id="dff62-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="dff62-108">В списке найдите и выберите требуемую запись.</span><span class="sxs-lookup"><span data-stu-id="dff62-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="dff62-109">В этом примере выберите код номенклатуры **M0001**.</span><span class="sxs-lookup"><span data-stu-id="dff62-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="dff62-110">На панели операций выберите **Управление запасами**.</span><span class="sxs-lookup"><span data-stu-id="dff62-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="dff62-111">Выберите **Штрих-коды**.</span><span class="sxs-lookup"><span data-stu-id="dff62-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="dff62-112">Выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="dff62-112">Select **New**.</span></span>
7. <span data-ttu-id="dff62-113">Отметьте выбранную строку, которая создается в списке ниже.</span><span class="sxs-lookup"><span data-stu-id="dff62-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="dff62-114">В поле **Настройка штрих-кода** введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="dff62-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="dff62-115">В поле **Штрих-код** введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="dff62-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="dff62-116">В поле **Штрих-код** введите значение.</span><span class="sxs-lookup"><span data-stu-id="dff62-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="dff62-117">В поле **Количество** введите число.</span><span class="sxs-lookup"><span data-stu-id="dff62-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="dff62-118">Нажмите **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="dff62-118">Select **Save**.</span></span>
13. <span data-ttu-id="dff62-119">Закройте страницу.</span><span class="sxs-lookup"><span data-stu-id="dff62-119">Close the page.</span></span>
