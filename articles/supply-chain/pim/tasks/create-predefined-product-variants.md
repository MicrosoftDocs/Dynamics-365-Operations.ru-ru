# <a name="create-predefined-product-variants"></a>Создание заранее определенных вариантов продукта

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как создать варианты продукта для шаблона продукта с использованием комбинаций аналитик продукта. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="create-a-product-master"></a>Создание шаблона продукта
1. Перейдите в раздел "Управление сведениями о продукте" > "Продукты" > "Шаблоны продукта".
2. Щелкните "Создать".
3. В поле "Номер продукта" введите значение.
    * Вводить номер продукта вручную обязательно, только если для поля "Номер продукта" не настроена номерная серия. Другими словами, пропустите этот шаг, если для поля настроена номерная серия.  
4. В поле "Имя продукта" введите значение.
5. В поле "Группа аналитик продуктов" введите или выберите значение.
    * Выберите группу аналитик продуктов SizeCol (размер и цвет).  
6. Нажмите кнопку "OК".

## <a name="add-product-dimensions"></a>Добавление аналитик продукта
1. Щелкните "Аналитики продуктов".
    * В этом примере показано, как ввести аналитики продукта вручную. Можно также выбрать размер, цвет или группу стилей, включающую значения аналитик продукта, которые необходимо использовать.  
2. Щелкните "Создать".
3. В списке пометьте выбранную строку.
4. В поле "Размер" введите или выберите значение.
5. В поле "Имя" введите значение.
6. Нажмите Создать.
7. В списке пометьте выбранную строку.
8. В поле "Размер" введите или выберите значение.
9. В поле "Имя" введите значение.
10. Перейдите на вкладку "Цвета".
11. Щелкните "Создать".
12. В списке пометьте выбранную строку.
13. В поле "Цвет" введите или выберите значение.
14. В поле "Имя" введите значение.
15. Нажмите Создать.
16. В списке пометьте выбранную строку.
17. В поле "Цвет" введите или выберите значение.
18. В поле "Имя" введите значение.
19. Нажмите кнопку "Сохранить".
20. Закройте страницу.

## <a name="generate-product-variants"></a>Создание вариантов продукта
1. Щелкните "Варианты продуктов".
2. Щелкните "Предложения вариантов".
3. Щелкните "Выбрать все".
    * В этом примере выбраны все возможные варианты. Если для создания вариантов будет использоваться только подмножество возможных комбинаций аналитик продукта, можно выбрать отдельные записи.  
4. Щелкните "Создать".
    * Можно создавать описания для всех имеющихся вариантов на основе комбинаций значений аналитик продукта. Описания являются необязательными.  
5. Нажмите кнопку "Сохранить".
