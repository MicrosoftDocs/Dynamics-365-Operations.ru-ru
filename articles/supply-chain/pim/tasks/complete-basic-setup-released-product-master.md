# <a name="complete-basic-setup-of-a-released-product-master"></a>Завершение базовой настройки шаблона запущенных в производство продуктов

[!include [banner](../../includes/banner.md)]

В этой теме показано, как выполнить минимальную настройку, необходимую, прежде чем шаблон продукта можно будет использовать в версиях спецификаций.

Это третья процедура из восьми, в которых поясняется, как строить сочетания для конфигурации на основе аналитик. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Перейдите в раздел **Область переходов > Модули > Управление сведениями о продукте > Продукты > Выпущенные продукты**.
2. В списке найдите и выберите требуемую запись. Выберите шаблон продукта, который вы запустили в производство во второй процедуре. Этот шаблон продукта создан с использованием технологии конфигурации на основе аналитик.  
3. В области панели операций выберите **Продукт**.
4. Выберите **Группы аналитики**, чтобы открыть ниспадающее диалоговое окно.
5. В поле **Группа аналитик хранения** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
6. В списке найдите и выберите требуемую запись. Группа аналитик хранения определяет, какие аналитики хранения используются для проводки продукта. Для этой процедуры выберите **Сайт**.  
7. В поле **Группа аналитик отслеживания** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
8. В списке найдите и выберите требуемую запись. Группа аналитик отслеживания определяет, какие аналитики отслеживания используются для проводки продукта. Для этой процедуры выберите **Нет**.  
9. Щелкните **OK**.
10. Выберите **Изменить**.
11. В поле **Группа номенклатурных моделей** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
12. В списке найдите и выберите требуемую запись. В группе номенклатурных моделей содержатся настройки, определяющие порядок контроля и управления приходом и расходом номенклатуры. Они также определяют расчет потребления номенклатуры. Для этой процедуры выберите **ФИФО**.  
13. Разверните раздел **Управление затратами**.
14. В поле **Группа номенклатур** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
15. В списке найдите и выберите требуемую запись. Номенклатурные группы используются для управления запасами, распределенными по группам. Для этой процедуры выберите **CarAudio**.  
16. В области действий выберите **План**.
17. Выберите **Параметры заказа по умолчанию**.
18. В поле **Тип заказа по умолчанию** выберите один из вариантов. Выберите **Производство**, чтобы указать, что вариант поставки по умолчанию для данного шаблона продукта по умолчанию — это его производство.  
19. Нажмите **Сохранить**.
20. Закройте страницу.
21. Закройте форму **Сведения о выпущенном продукте**.
