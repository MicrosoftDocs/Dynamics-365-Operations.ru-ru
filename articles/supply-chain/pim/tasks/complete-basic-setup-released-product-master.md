# <a name="complete-basic-setup-of-a-released-product-master"></a>Завершение базовой настройки шаблона запущенных в производство продуктов

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как выполнить минимальную настройку, необходимую, прежде чем шаблон продукта можно будет использовать в версиях спецификаций.

Это третья процедура из восьми, в которых поясняется, как строить сочетания для конфигурации на основе аналитик. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Щелкните "Управление сведениями о продукте" > "Продукты" > "Выпущенные продукты".
2. В списке найдите и выберите требуемую запись.
    * Выберите шаблон продукта, который вы запустили в производство во второй процедуре. Этот шаблон продукта создан с использованием технологии конфигурации на основе аналитик.  
3. В области действий щелкните "Продукт".
4. Щелкните "Группы аналитики", чтобы открыть ниспадающее диалоговое окно.
5. В поле "Группа аналитик хранения" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. В списке найдите и выберите требуемую запись.
    * Группа аналитик хранения определяет, какие аналитики хранения используются для проводки продукта. Для этой процедуры выберите "Сайт".  
7. В списке перейдите по ссылке в выбранной строке.
8. В поле "Группа аналитик отслеживания" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
9. В списке найдите и выберите требуемую запись.
    * Группа аналитик отслеживания определяет, какие аналитики отслеживания используются для проводки продукта. Для этой процедуры выберите "Нет".  
10. В списке перейдите по ссылке в выбранной строке.
11. Нажмите кнопку "OК".
12. В списке перейдите по ссылке в выбранной строке.
13. Щелкните "Изменить".
    * Откройте форму "Сведения о выпущенном продукте", чтобы продолжить выполнение задачи настройки.  
14. В поле "Группа номенклатурных моделей" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
15. В списке найдите и выберите требуемую запись.
    * В группе номенклатурных моделей содержатся настройки, определяющие порядок контроля и управления приходом и расходом номенклатуры. Они также определяют расчет потребления номенклатуры. Для этой процедуры выберите "ФИФО".  
16. В списке перейдите по ссылке в выбранной строке.
17. Разверните или сверните раздел "Управление затратами".
18. В поле "Номенклатурная группа" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
19. В списке найдите и выберите требуемую запись.
    * Номенклатурные группы используются для управления запасами, распределенными по группам. Для этой процедуры выберите "CarAudio".  
20. В списке перейдите по ссылке в выбранной строке.
21. В области действий щелкните "План".
22. Щелкните "Параметры заказа по умолчанию".
23. В поле "Тип заказа по умолчанию" выберите один из вариантов.
    * Выберите "Производство", чтобы указать, что вариант поставки по умолчанию для данного шаблона продукта по умолчанию — это его производство.  
24. Закройте страницу.
25. Закройте форму "Сведения о выпущенном продукте".
