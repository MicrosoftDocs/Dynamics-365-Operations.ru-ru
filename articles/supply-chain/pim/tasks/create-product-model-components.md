[!include [banner](../../includes/banner.md)]

<span data-ttu-id="1efde-104">В моделях конфигурации на основе ограничений используются компоненты в качестве строительных блоков, чтобы настроить структуру модели продукта.</span><span class="sxs-lookup"><span data-stu-id="1efde-104">Constraint-based product configuration models use components as building blocks to establish the product model structure.</span></span> <span data-ttu-id="1efde-105">Перед созданием модели продукта необходимо создать компоненты.</span><span class="sxs-lookup"><span data-stu-id="1efde-105">The components must be created before a product model can be built.</span></span> <span data-ttu-id="1efde-106">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="1efde-106">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="1efde-107">Перейдите в раздел **Управление сведениями о продукте \> Продукты \> Модели конфигурации продукта**.</span><span class="sxs-lookup"><span data-stu-id="1efde-107">Go to **Product information management \> Products \> Product configuration models**.</span></span>
1. <span data-ttu-id="1efde-108">В области действий выберите **Модель**.</span><span class="sxs-lookup"><span data-stu-id="1efde-108">On the Action Pane, select **Model**.</span></span>

1. <span data-ttu-id="1efde-110">Выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="1efde-110">Select **New**.</span></span>
1. <span data-ttu-id="1efde-111">В поле **Имя** введите значение.</span><span class="sxs-lookup"><span data-stu-id="1efde-111">In the **Name** field, type a value.</span></span>
1. <span data-ttu-id="1efde-112">В поле **Описание** введите значение.</span><span class="sxs-lookup"><span data-stu-id="1efde-112">In the **Description** field, type a value.</span></span>
