# <a name="create-product-model-components"></a>Создайте компонентов модели продукта

[!include [banner](../../includes/banner.md)]

В моделях конфигурации на основе ограничений используются компоненты в качестве строительных блоков, чтобы настроить структуру модели продукта. Перед созданием модели продукта необходимо создать компоненты. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Щелкните "Определение модели вариантов продукта".
2. Щелкните "Модели конфигурации продукта".
3. В области действий щелкните "Модель".
4. Щелкните "Компоненты".
5. Щелкните "Создать".
6. В поле "Имя" введите значение.
7. В поле "Описание" введите значение.
8. Закройте страницу.
