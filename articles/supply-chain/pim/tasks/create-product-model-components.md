<span data-ttu-id="16b99-104">В моделях конфигурации на основе ограничений используются компоненты в качестве строительных блоков, чтобы настроить структуру модели продукта.</span><span class="sxs-lookup"><span data-stu-id="16b99-104">Constraint-based product configuration models use components as building blocks to establish the product model structure.</span></span> <span data-ttu-id="16b99-105">Перед созданием модели продукта необходимо создать компоненты.</span><span class="sxs-lookup"><span data-stu-id="16b99-105">The components must be created before a product model can be built.</span></span> <span data-ttu-id="16b99-106">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="16b99-106">The demo data company used to create this procedure is USMF.</span></span>
