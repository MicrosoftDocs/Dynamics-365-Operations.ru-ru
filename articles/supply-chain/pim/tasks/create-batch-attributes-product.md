# <a name="create-batch-attributes-for-a-product"></a>Создание атрибутов партии для продукта

[!include[task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как создать атрибут партии, назначить диапазоны значений по умолчанию и включить атрибут в группу. В качестве компании с демонстрационными данными для создания этой процедуры используется компания USP2.

1. Перейдите в раздел "Управление запасами" > "Партия" > "Атрибуты партии".
2. Щелкните "Создать".
3. В поле "Атрибут" введите значение.
4. В поле "Описание" введите значение.
5. В поле "Тип атрибута" выберите "Дробь".
    * В этой процедуре используется тип "Дробь" для получения десятичных значений. Можно выбрать другие типы атрибутов. При выборе типа "Перечисление" необходимо ввести значения в список перечисления, прежде чем можно будет ввести значение в поле "Цель".  
6. В поле "Минимум" введите число.
7. В поле "Максимум" введите число.
8. В поле "Шаг" введите число.
9. В поле "Цель" введите значение.
10. Нажмите кнопку "Сохранить".
11. Закройте страницу.
12. Перейдите в раздел "Управление запасами" > "Партия" > "Группы атрибутов партии".
13. Щелкните "Создать".
14. В поле "Группа атрибутов" введите значение.
15. В поле "Описание" введите значение.
16. Нажмите кнопку "Сохранить".
17. Щелкните Сгруппировать атрибуты.
18. Щелкните "Создать".
19. В поле "Атрибут" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
20. В списке найдите и выберите требуемую запись.
21. В списке перейдите по ссылке в выбранной строке.
    * Атрибут может входить в любую из групп.  
22. Нажмите кнопку "Сохранить".
23. Закройте страницу.
