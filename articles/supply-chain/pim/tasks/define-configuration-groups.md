# <a name="define-configuration-groups"></a>Определение конфигурационных групп

[!include[task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как определить конфигурационные группы продукта, путем создания двух групп в компании с демонстрационными данными USMF. Это четвертая процедура из восьми, в которых поясняется, как строить сочетания для конфигурации на основе аналитик.

1. Перейдите в раздел "Управление сведениями о продукте" > "Настройка" ? "Спецификации и формулы" > "Конфигурационные группы".
2. Щелкните "Создать".
3. В поле "Конфигурационная группа" введите значение.
    * Введите имя конфигурационной группы, которая будет использоваться для строк спецификации.  
4. В поле "Имя" введите значение.
5. Щелкните "Создать".
6. В поле "Конфигурационная группа" введите значение.
    * Введите имя конфигурационной группы, которая будет использоваться для строк спецификации.  
7. В поле "Имя" введите значение.
8. Закройте страницу.
