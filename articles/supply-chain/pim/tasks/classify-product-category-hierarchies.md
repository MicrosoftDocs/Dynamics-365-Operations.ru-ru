# <a name="classify-a-product-using-category-hierarchies"></a>Классификация продукта с помощью иерархии категорий

[!include [banner](../../includes/banner.md)]

В этой теме описывается классификация продукта с помощью иерархий категорий. Используйте иерархии категорий для классификации продуктов для экспорта, покупки, продажи или других целей. Обычно эта задача выполняется менеджером продуктов. В этом примере используется номер продукта M0009 в демонстрационных данных компании USMF.

1. Выберите **Обслуживание выпущенного продукта**.
2. Выберите **Все выпущенные продукты**.
3. В списке найдите и выберите требуемую запись.
4. В области панели операций выберите **Продукт**.
5. Выберите **Категории продуктов**.
6. Выберите **Создать**.
7. В поле **Иерархия категорий** введите или выберите значение.
8. В поле **Категория** введите или выберите значение.
9. Выберите **Создать**.
10. В поле **Иерархия категорий** введите или выберите значение.
11. В поле **Категория** введите или выберите значение.
12. Закройте страницу.
