# <a name="assign-a-product-lifecycle-state-to-a-released-product"></a>Назначение состояния жизненного цикла продукта выпущенному продукту

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как назначить состояние жизненного цикла продукта выпущенному продукту. Предварительное требование: сначала необходимо просмотреть проводник по задаче "Создание нового состояния жизненного цикла продукта", чтобы убедитесь, что создано хотя бы одно состояние жизненного цикла продукта, до просмотра этого проводника.


## <a name="find-a-released-product-that-you-want-to-assign-a-product-lifcycle-state"></a>Поиск выпущенного продукта, которому требуется назначить состояние жизненного цикла продукта
1. Щелкните "Управление сведениями о продукте" > "Продукты" > "Выпущенные продукты".
2. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле "Код номенклатуры" по значению "d".
3. В списке перейдите по ссылке в выбранной строке.
4. Щелкните "Изменить".
5. В поле "Состояние жизненного цикла продукта" введите или выберите значение.
6. Нажмите кнопку "Сохранить".
