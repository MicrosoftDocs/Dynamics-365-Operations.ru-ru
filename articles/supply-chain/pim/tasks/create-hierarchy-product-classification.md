# <a name="create-a-hierarchy-of-product-classification"></a>Создание иерархии классификации продуктов

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как создать новую иерархию категорий и назначить тип иерархии "товарный код". В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Эта процедура предназначена для менеджера категорий.


## <a name="create-the-new-category-hierarchy"></a>Создание новой иерархии категорий
1. Выберите **Область переходов > Модули > Управление сведениями о продукте > Настройка > Категории и атрибуты > Иерархии категорий**.
2. Нажмите кнопку **Создать**.
3. В поле **Имя** введите значение.
4. В поле **Описание** введите значение.
5. Щелкните **Создать**.

## <a name="build-the-hierarchy"></a>Построение иерархии
1. Щелкните **Создать** узел категории.
2. В поле **Имя** введите значение.
3. В поле **Код** введите значение.
4. В поле **Понятное имя** введите значение.
5. Щелкните **Создать** узел категории.
6. В поле **Имя** введите значение.
7. В поле **Код** введите значение.
8. В поле **Понятное имя** введите значение.
9. Щелкните **Создать** узел категории.
10. В поле **Имя** введите значение.
11. В поле **Код** введите значение.
12. В поле **Понятное имя** введите значение.
13. Щелкните **Создать** узел категории.
14. В поле **Имя** введите значение.
15. В поле **Код** введите значение.
16. В поле **Понятное имя** введите значение.
17. Закройте страницу.

## <a name="classify-the-hierarchy"></a>Классификация иерархии
1. В списке найдите и выберите требуемую запись.
2. В области **Панель операций** щелкните **Иерархия категорий**.
3. Щелкните **Связать тип иерархии**.
4. Нажмите кнопку **Создать**.
5. В поле **Тип иерархии категорий** выберите один из вариантов. Выберите **Тип иерархии категорий "Товарный код" для классификации продуктов**.  
6. В поле **Иерархия категорий** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
7. Поиск и выбор требуемой записи в списке.
8. В списке перейдите по ссылке в выбранной строке.
9. Закройте страницу.
