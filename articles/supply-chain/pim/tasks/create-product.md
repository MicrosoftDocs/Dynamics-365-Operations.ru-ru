# <a name="create-a-product"></a>Создание продукта

[!include [banner](../../includes/banner.md)]

В этой процедуре рассматривается создание нового продукта в иерархии продуктов Commerce. В качестве компании с демонстрационными данными для создания этой задачи использовалась компания USRT. Задача предназначена для директора по сбыту и конструктора продуктов.

1. Перейдите в раздел Retail и Commerce > .. > "Продукты по категориям".
2. Нажмите Создать.
3. В поле "Имя продукта" введите имя для своего продукта.
4. В поле категории нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
5. В дереве разверните узел "ВСЕ (ВСЕ)\Мода (Мода)".
6. В дереве выберите "ВСЕ (ВСЕ)\МОДА (МОДА)\Модные аксессуары (Модные аксессуары)".
7. Нажмите кнопку OK.
8. В поле "Номер продукта" введите значение.
9. Нажмите кнопку "OК".
