# <a name="manage-units-of-measure"></a>Управление единицами измерения

[!include [banner](../../includes/banner.md)]

В этой теме показано, как определить единицу измерения, ввести переводы для единицы измерения и ее описание, а также определить правила преобразования для связанных единиц измерения.

## <a name="open-the-units-page"></a>Откройте страницу единиц измерения

Чтобы создать единицы измерения, доступные в системе, и работать с ними, перейдите в раздел **Администрирование организации \> Настройка \> Единицы измерения \> Единицы измерения**.

В остальных разделах этой главы описываются возможные действия на странице **Единицы измерения**.

## <a name="create-standard-units-and-conversions"></a>Создание стандартных единиц измерения и преобразования

Если ваша система еще не включала наиболее часто используемые единицы измерения для системы показателей и/или пользовательской системы США (USCS), мастер настройки единиц измерения поможет быстро начать работу с определениями базовых единиц измерения и их преобразованиями. Чтобы завершить работу мастера, выберите **Мастер создания единиц измерения** на панели операций, а затем следуйте инструкциям на экране.

## <a name="create-or-edit-a-unit-of-measure"></a>Создание или изменение единицы измерения

Чтобы создать или изменить единицу измерения, выполните следующие действия.

1. Выполните одно из следующих действий.

    - Чтобы изменить существующую единицу измерения, выберите ее на панели списка.
    - Чтобы создать новую единицу измерения, на панели операций выберите **Создать**.

1. В заголовке записи задайте следующие поля:

    - **Единица измерения** — введите код или символ, который будет использоваться для ссылки на единицу измерения на системном языке. Этот код обычно является типичным сокращением для единицы измерения, например *ea* для "each" (каждый) или *cm* для сантиметра.
    - **Описание** — введите описательное имя для единицы измерения на системном языке. Обычно это имя является полным именем единицы измерения, например, *Каждый* или *Сантиметр*.

1. На экспресс-вкладке **Общие** настройте следующие поля:<!-- KFM: confirm this:    - **Fixed unit assignment** and **Fixed unit** – These fields have an effect only if you're using the Microsoft Retail Essentials product. If the current unit can be mapped to one of the fixed units that are used by Retail Essentials, set the **Fixed unit assignment** option to *Yes*. Then select the fixed unit in the **Fixed unit** field. -->

    - **Класс единиц измерения** — выберите свойство, которое измеряет единица измерения (например, длина, площадь, масса или количество).
    - **Система единиц измерения** — выберите систему измерений, к которой принадлежит единица измерения (*Метрические единицы измерения* или *Типичные единицы измерения США*).
    - **Базовая единица измерения** — установите для этого параметра значение *Да*, чтобы использовать текущую единицу измерения в качестве базовой единицы для класса единиц измерения. В этом случае необходимо указать коэффициент преобразования между базовой единицей измерения и каждой дополнительной единицей измерения в классе единиц измерения. Затем система может преобразовать все единицы измерения в этом классе единиц измерения. Таким образом, упрощается настройка преобразований.

        Например, если галлон является базовой единицей для класса единиц измерения *Объем*, необходимо настроить коэффициенты преобразования из кварты в галлон и из пинты в галлон. Затем система может также преобразовать из кварты в пинту.

        Для одного класса единицы измерения может быть только одна базовая единица измерения.

    - **Системная единица** — установите для этого параметра значение *Да*, чтобы использовать текущую единицу измерения в качестве предполагаемой единицы всех неуказанных изменений в классе единиц измерения. Например, если поле, используемое для ввода количества, не разрешает указание единицы измерения (если пользователь не выбирает единицу измерения), система использует единицу измерения, заданную в качестве системной единицы измерения для класса единиц измерения *Количество*. Для одного класса единицы измерения может быть только одна системная единица измерения.
    - **Точность в десятичных знаках** — указывается число десятичных разрядов, для которых задаются значения для текущей единицы измерения или до которых округляются при преобразовании.

1. На панели операций выберите **Сохранить**.

## <a name="define-unit-translations"></a>Определение переводов для единицы измерения

Чтобы определить переводы для идентификатора или символа и описание для единицы измерения, выполните следующие действия.

1. Создайте или выберите единицу измерения, для которой необходимо создать переводы.
1. В области действий выберите **Тексты единиц измерения**.

    Отображается страница **Тексты единиц измерения**. Эта страница используется, чтобы определить переводы для идентификатора или символа выбранной единицы измерения. Эти переводы затем могут быть использованы для внешних документов на языках, отличных от клиента или поставщика.

1. В области действий выберите **Создать**.
1. В поле **Язык** выберите язык, на который следует перевести идентификатор или символ единицы измерения.
1. В поле **Текст** введите перевод идентификатора или символа единицы измерения на выбранном языке.
1. На панели операций выберите **Сохранить**.
1. Закройте страницу.
1. На **Панели операций** выберите **Описания пересчитанных единиц измерения**.

    Будет открыта страница **Описания переведенных единиц измерения**. Эта страница используется для определения зависящих от языка описаний для выбранной единицы измерения.

1. В области действий выберите **Создать**.
1. В поле **Язык** выберите язык, на который следует перевести описание единицы измерения.
1. В поле **Описание** введите перевод описания единицы измерения на выбранном языке.
1. На панели операций выберите **Сохранить**.
1. Закройте страницу.

## <a name="define-unit-conversion-rules"></a>Определение правил пересчета единицы измерения

Чтобы определить правила преобразования между единицами измерения, выполните следующие действия.

1. Создайте или выберите единицу измерения, для которой необходимо задать правила преобразования.
1. В области действий выберите **Пересчет единиц измерения**.

    Откроется страница **Пересчеты единиц измерения**. На этой странице определите правила для пересчета выбранной единицы измерения в другие единицы измерения (и из них) в выбранном классе единиц измерения.

1. В зависимости от типа пересчета, которое нужно настроить, выберите одну из следующих вкладок:

    - **Стандартные преобразования** — настройка стандартных правил преобразования для всех продуктов.
    - **Преобразования внутри класса** — настройка правил преобразования для конкретного продукта для единиц в одном и том же классе единиц измерения.
    - **Преобразования между классами** — настройка правил преобразования для конкретного продукта для единиц в разных классах единиц измерения.

1. Выполните одно из следующих действий.

    - Чтобы создать новый пересчет, на панели инструментов выберите **Создать**.
    - Чтобы изменить существующее преобразование, выберите преобразование в сетке, а затем нажмите **Правка** на панели инструментов.

1. В раскрывшемся диалоговом окне задайте следующие поля:

    - **Продукт** — выберите конкретный продукт, к которому применяется преобразование. Это поле доступно только для преобразования внутри класса и между классами.
    - **Структура формулы** — оставьте в поле значение *Простая*, чтобы задать простое преобразование с одним множителем. Задайте *Расширенная*, чтобы настроить более сложное уравнение. Формат сложных уравнений изменяется в зависимости от класса единиц измерения.
    - **Из единицы** — в этом поле отображается выбранная единица измерения. Обычно изменять значение нельзя. (Если нужно изменить значение, необходимо открыть страницу **Пересчеты единиц измерения** для выбранной единицы измерения, чтобы просмотреть новое преобразование после его сохранения.)
    - **В единицу** — выберите единицу измерения, в которую которое необходимо выполнить пересчет.
    - **Округление** — выберите способ округления на основе значения **Десятичная точность** выбранной единицы измерения (*До ближайшего*, *Вверх* или *Вниз*).
    - **Формула преобразования** — воспользуйтесь оставшимися полями в верхней части диалогового окна, чтобы указать формулу для преобразования двух единиц измерения. Доступные поля изменяются в зависимости от выбранного класса единиц измерения и структуры формулы.

1. Нажмите **ОК**.
1. Закройте страницу.

> [!TIP]
> Можно также настроить пересчет единицы для варианта продукта. Дополнительные сведения см. в разделе [Преобразование единиц измерения для вариантов продукта](../uom-conversion-per-product-variant.md).
