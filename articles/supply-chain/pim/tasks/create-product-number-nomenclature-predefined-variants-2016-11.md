# <a name="create-a-product-number-nomenclature-for-predefined-product-variants"></a>Создание номенклатуры номера продукта для заранее определенных вариантов продукта

[!include [banner](../../includes/banner.md)]

В этом теме показано, как настроить номенклатуру номера продукта для предопределенных вариантов продукта, и как назначить ее соответствующей группе аналитик продукта. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Новая номенклатура номера продукта назначается группе аналитик продукта "Цвет" и "Размер". Эта задача обычно выполняется конструктором продуктов.


## <a name="create-a-product-number-nomenclature"></a>Создание номенклатуры номеров продуктов

1. Перейдите в раздел **Управление сведениями о продукте \> Настройка \> Номенклатура продукта**.
1. Выберите **Создать**.
1. В поле **Имя** введите имя номенклатуры, которое помогает идентифицировать целевую группу аналитик продукта, например `ColorSize`.
1. В поле **Описание** введите значение.
1. Выберите **Добавить**.
1. Выберите **Номер шаблона продукта**.
1. Выберите **Добавить**.
1. Выберите **Текстовая константа**.
1. В поле **Текст** введите значение.
1. Выберите **Добавить**.
1. Выберите **Цвет**.
1. Выберите **Добавить**.
1. Выберите **Текстовая константа**.
1. В поле **Текст** введите значение.
1. Выберите **Добавить**.
1. Выберите **Размер**.
1. Закройте страницу.

## <a name="assign-the-nomenclature-to-a-product-master"></a>Назначение номенклатуры шаблону продукта

1. Выберите **Группы аналитик продукта**.
2. Выберите группу **Аналитика продукта SizeCol**.
3. Выберите **Правка**.
4. Выберите **Да** в поле **Использовать номенклатуру**.
5. В поле **Номенклатура номеров вариантов продуктов** введите или выберите значение.
6. Закройте страницу.
