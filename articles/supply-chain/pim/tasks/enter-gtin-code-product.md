# <a name="enter-the-gtin-code-for-a-product"></a>Ввод кода GTIN для продукта

[!include [banner](../../includes/banner.md)]

Коды GTIN назначаются организацией GS1. В этом примере вы введете код для номера продукта M0001 в демонстрационных данных компании USMF.

1. Щелкните "Обслуживание выпущенного продукта".
2. Щелкните "Выпущенные продукты".
3. Разверните информационное поле "Изображение продукта".
4. В списке найдите и выберите требуемую запись.
5. В области действий щелкните "Управление запасами".
6. Щелкните "Коды GTIN".
7. Щелкните "Создать".
8. В списке пометьте выбранную строку.
9. В поле "Единица измерения" введите или выберите значение.
10. В поле "Код GTIN" выберите вариант.
11. В поле "GTIN" введите значение.
12. Закройте страницу.
