<span data-ttu-id="2dfb5-105">Формирование расширенной загрузки волны автоматически назначает отгрузки существующим волнам в ходе выполнения волн.</span><span class="sxs-lookup"><span data-stu-id="2dfb5-105">Advanced wave load building automatically assigns shipments to existing waves during wave execution.</span></span> <span data-ttu-id="2dfb5-106">Таким образом, можно создать осмысленные загрузки, представляющую грузовики, без необходимости использования рабочего места планирования загрузки.</span><span class="sxs-lookup"><span data-stu-id="2dfb5-106">Therefore, you can create meaningful loads that represent trucks without having to use the load planning workbench.</span></span> <span data-ttu-id="2dfb5-107">Эта возможность полезна для компаний, которые хотят использовать концепцию загрузок для отслеживания и планирования отгрузки товаров на грузовике или прицепе, но не хотят создавать каждую загрузку вручную каждый день.</span><span class="sxs-lookup"><span data-stu-id="2dfb5-107">This capability is useful for businesses that want to use the concept of loads to track and plan the shipment of goods in a truck/trailer, but that don't want to manually create those loads every day.</span></span>
