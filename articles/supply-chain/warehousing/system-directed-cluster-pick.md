# <a name="system-directed-cluster-picking"></a><span data-ttu-id="ebdc0-103">Комплектация кластера, управляемая системой</span><span class="sxs-lookup"><span data-stu-id="ebdc0-103">System-directed cluster picking</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="ebdc0-104">Комплектации кластера — это процесс выбора номенклатур, который позволяет комплектовать номенклатуры для нескольких заказов одновременно путем кластеризации их в кластеры комплектации.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-104">Cluster picking is a piece picking process that lets you pick items for multiple orders at the same time by clustering them into pick clusters.</span></span> <span data-ttu-id="ebdc0-105">Затем вы должны посетить местонахождение комплектации только один раз.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-105">You then have to visit the pick location only one time.</span></span> <span data-ttu-id="ebdc0-106">Как правило, эта функциональность используется при небольшой комплектации заказов или при количествах, которые меньше количества обращений.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-106">Typically, this functionality is used with small order picking or quantities that are less than case quantities.</span></span>

<span data-ttu-id="ebdc0-107">При настройке комплектации кластера под управлением системы можно создавать заголовки работ комплектации по кластеру на основе создаваемого системой кластера.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-107">When system-directed cluster picking is set up, you can cluster-pick work headers, based on a system-generated cluster.</span></span> <span data-ttu-id="ebdc0-108">Система осуществляет комплектацию по кластерам для заказов до числа позиций, указанных в профиле кластера.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-108">The system cluster-picks orders up to the number of positions that is specified in the cluster profile.</span></span> <span data-ttu-id="ebdc0-109">Таким образом, можно комплектовать несколько заказов одновременно без необходимости вручную создавать кластер.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-109">Therefore, you can pick multiple orders at the same time without having to manually create a cluster.</span></span>

<span data-ttu-id="ebdc0-110">Комплектация кластера под управлением системы является альтернативой ручного создания кластеров, где для создания кластера используется профиль кластеров.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-110">System-directed cluster picking offers an alternative to manual cluster building, where a cluster profile is used to create a cluster.</span></span> <span data-ttu-id="ebdc0-111">Перед использованием в профиле кластера следует определить некоторые связанные с настройкой строки:</span><span class="sxs-lookup"><span data-stu-id="ebdc0-111">Several setup-related lines must be defined in the cluster profile before it's used:</span></span>

- <span data-ttu-id="ebdc0-115">**Создать идентификатор кластера** управляет, будет код кластера создан системой или введен пользователем.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-115">**Generate cluster ID** controls whether the cluster ID will be generated by the system or entered by the user.</span></span>

<span data-ttu-id="ebdc0-117">Новый пункт меню мобильного устройства используется для комплектации кластеров под управлением системы.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-117">A new mobile device menu item is used for system-directed cluster picking.</span></span> <span data-ttu-id="ebdc0-118">Код профиля кластера должен быть указан для параметра **Кем управляется**.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-118">The cluster profile ID must be specified for the **Directed by** option.</span></span> <span data-ttu-id="ebdc0-119">Кроме того, заказ запросов под управлением системы может группировать заказы на основе конкретных критериев компании.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-119">Additionally, the system-directed query order can group orders, based on company-specific criteria.</span></span> <span data-ttu-id="ebdc0-120">Таким образом, можно дополнительно оптимизировать назначение заказов на работу, указав настроенные критерии сортировки в заказе запроса под управлением системы.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-120">Therefore, you can further optimize the assignment of work orders by specifying customized sorting criteria on the system-directed query order.</span></span>

<span data-ttu-id="ebdc0-121">Когда комплектация кластера под управлением системы завершена, работникам складов предоставляется кластер, в котором заказы комплектации уже заданы для позиций кластера.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-121">When system-directed cluster picking is done, warehouse workers are presented with a cluster where picking orders have been preassigned to cluster positions.</span></span> <span data-ttu-id="ebdc0-122">Таким образом, работники могут начать комплектацию номенклатуры для нескольких заказов на работу, посетив место комплектации только один раз.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-122">Therefore, workers can start to pick an item for multiple work orders by visiting the pick location only one time.</span></span> <span data-ttu-id="ebdc0-123">Процесс комплектации кластера под управлением системы такой же, как и процесс комплектации кластера под управлением пользователя.</span><span class="sxs-lookup"><span data-stu-id="ebdc0-123">The picking process for system-directed cluster picking is the same as the process for user-directed cluster picking.</span></span>
