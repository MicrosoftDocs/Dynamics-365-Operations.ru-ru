# <a name="system-grouping-on-an-open-work-list"></a>Системная группировка в списке открытых работ

[!include[banner](../includes/banner.md)]

Используя поле системной группировки, можно отфильтровать список открытых работ, не редактируя пункт меню мобильного устройства.
Там, где она применяется, системная группировка используется для фильтрации списка работ по одному полю в заголовке работы. Использовать системную группировку для фильтрации по полям уровня строки нельзя.

## <a name="set-up-system-grouping-on-an-open-work-list"></a>Настройка системной группировки в списке открытых работ
Для настройки системной группировки в списке открытых работ выполните следующие действия.

-   Из пункта меню на мобильном устройстве выберите **Режим: Косвенный** и **Код мероприятия: Отобразить список открытых работ**. Станут доступны следующие параметры. Эти параметры необходимы для системной группировки списка открытых работ. 

| Параметр        | описание   | 
| ------------- | ------------- |
| Разрешить системную группировку   | Включает системную группировку для выбранного пункта меню списка работ.| 
| Поле системной группировки   | Доступен, только если параметр **Разрешить системную группировку** установлен в значение **Да**. Выберите поле, которое определяет, как работа по комплектации будет группироваться для работников. Например, если выбрать поле **ShipmentId**, работник будет сканировать код отгрузки для группировки работы комплектации. Вся работа для отгрузки будет назначена работнику. Это поле требует, чтобы вы создали пункт меню для использования существующей работы, сгруппированной системой. Используйте поле **Метка системной группировки**, чтобы сообщить работнику, что следует сканировать. |
| Метка системной группировки   | Доступен, только если параметр **Разрешить системную группировку** установлен в значение **Да**. Введите информацию для работника о том, что следует сканировать, когда работа комплектации группируется. Например, если вы используете поле **ShipmentId** для группировки работ по комплектации по отгрузке, можно ввести в этом поле "Код отгрузки". Это поле требует, чтобы вы создали пункт меню для использования существующей работы, сгруппированной системой. Также необходимо выбрать поле, по которому будет выполняться группировка, в поле **Системная группировка**.|
