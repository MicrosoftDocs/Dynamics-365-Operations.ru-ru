[!include [banner](../includes/banner.md)]

<span data-ttu-id="c0ed7-104">Поскольку Android является проектом с открытым кодом, любой производитель оборудования для сканеров складских штрих-кодов может создать устройство, работающее под управлением операционной системы Android.</span><span class="sxs-lookup"><span data-stu-id="c0ed7-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="c0ed7-105">Устройство совместимо с Android только в том случае, если оно может выполнять приложения, написанные для среды выполнения Android.</span><span class="sxs-lookup"><span data-stu-id="c0ed7-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="c0ed7-106">Тем не менее поставщик оборудования может изменять и создавать оболочки для версии Android, которая работает на их оборудовании.</span><span class="sxs-lookup"><span data-stu-id="c0ed7-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="c0ed7-107">Корпорация Майкрософт не может принимать на себя ответственность и гарантировать, что мобильное приложение сканирования штрих-кодов для Android будет совместимо с оборудованием изготовителя для сканирования штрих-кодов и версией Android, работающей на устройстве изготовителя.</span><span class="sxs-lookup"><span data-stu-id="c0ed7-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span> 

<span data-ttu-id="c0ed7-108">Приложение управления запасами для Microsoft Dynamics 365 for Finance and Operations протестировано на ряде устройств для сканирования штрих-кодов, работающих под управлением Android.</span><span class="sxs-lookup"><span data-stu-id="c0ed7-108">The Warehousing app for Microsoft Dynamics 365 for Finance and Operations has been tested with a selection of Android powered devices for bar code scanning.</span></span> <span data-ttu-id="c0ed7-109">Эти тесты охватывают только образцы устройств, доступные на рынке.</span><span class="sxs-lookup"><span data-stu-id="c0ed7-109">These tests only cover a sample of the devices that are available on the market.</span></span>

<span data-ttu-id="c0ed7-110">Мы рекомендуем клиентам проверять мобильное приложение сканирования складских штрих-кодов на выбранном оборудовании, прежде чем принимать решение о закупке оборудования.</span><span class="sxs-lookup"><span data-stu-id="c0ed7-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
