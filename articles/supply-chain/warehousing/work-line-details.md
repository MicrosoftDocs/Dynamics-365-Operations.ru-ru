# <a name="work-line-details"></a>Сведения строки работы

[!include [banner](../includes/banner.md)]

В этой теме приводятся сведения о странице **Сведения строки работы** , на которой показан полный, сортируемый и фильтруемый список отдельных строк работы в системе. Она обеспечивает полный обзор работ, которые планируются и выполняются на складе. Можно легко переключаться между просмотром всех строк работ и просмотром только открытых строк работ. Сведения, предоставляемые для каждой строки, включают статус работы, код номенклатуры, местоположение, количество работы, код загрузки и код отгрузки.

## <a name="turn-on-the-work-line-details-feature"></a>Включение функции сведений о строках работ

Прежде чем использовать эту функцию, она должна быть включена в системе. Администраторы могут использовать параметры [управления компонентами](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) для проверки статуса функции и ее включения, если это требуется. В рабочей области **Управление функциями** эта функция перечисляется следующими способами:

- **Модуль:** *Управление складом*
- **Название компонента:** *Сведения о строках работ*

## <a name="open-and-use-the-work-line-details-page"></a>Открытие и использование страницы сведения о строках работ

Чтобы просмотреть список сведений о строках работ, перейдите в раздел **Управление складом \> Работа \> Сведения строки работы**. Отсюда можно выполнить следующие действия:

- Используйте поле **Фильтр** , чтобы выполнить поиск по строкам, которые имеют конкретное значение для любого доступного параметра. (Доступные параметры включают множество параметров, которые не отображаются как столбцы в сетке.)
- Используйте флажок **Показать закрытые** , чтобы показать или скрыть закрытые строки.
- Выберите **Отобразить аналитики** , чтобы открыть диалоговое окно **Отображение аналитик** , в котором можно выбрать отображение или скрытие различных столбцов аналитик в сетке.
- Выберите любой заголовок столбца, чтобы открыть меню, в котором можно выбрать сортировку или отфильтровать список по значениям в этом столбце.
- Выберите строку работы, затем выберите **Изменить местоположение** , чтобы открыть диалоговое окно, в котором можно изменить местоположение для этой строки работы. Указываемое местоположение будет переопределять настройку директивы местоположения.
- Выберите строку работы, затем выберите **Отменить строку работы** , чтобы открыть диалоговое окно, в котором можно частично или полностью уменьшить количество этой строки работы.
- Настройте количества.
- Просмотрите проводки для любой строки работы.

## <a name="try-out-the-feature"></a>Опробывание функции

В этом разделе содержится демонстрация из трех частей, в которой показано, как работать со сведениями о строках работ.

### <a name="make-sample-data-available"></a>Сделать образцы данных доступными

Для работы с этой демонстрацией с помощью образцов записей и значений, указанных здесь, необходимо использовать систему, в которой установлены стандартные [демонстрационные данные](../../fin-ops-core/dev-itpro/deployment/deploy-demo-environment.md). Дополнительно перед началом необходимо выбрать юридическое лицо **USMF**.

Эту демонстрацию можно также использовать в качестве руководства при работе в производственной системе. Однако необходимо подставить собственные значения, и у вас могут отсутствовать некоторые типы необходимых записей, предоставляемых стандартными демонстрационными данными.

### <a name="verify-that-the-scenario-setup-includes-enough-available-inventory"></a>Убедитесь, что в настройке сценария имеется достаточное количество доступных запасов

При работе с демонстрационными данными **USMF** необходимо убедиться, что ваша система настроена таким образом, что доступно достаточное количество запасов для каждого соответствующего местоположения комплектации. В этой демонстрации предполагается, что у вас имеются следующие доступные запасы:

- **Номенклатура M9200:** 45 шт. (или более)
- **Номенклатура M9202:** 10 шт. (или более)

Выполните следующие действия, чтобы проверить доступность достаточных запасов и выполнить необходимые корректировки.

1. Перейдите к пункту **Управление складом \> Настройка \> Директивы местоположения** и определите, какие ячейки комплектации используются для комплектации заказов на продажу на складе 51. (Дополнительные сведения см. в разделе [Контроль работы склада с использованием шаблонов работы и директив для мест хранения](control-warehouse-location-directives.md).)
1. Проверьте уровни запасов в соответствующих местоположениях.
1. При необходимости скорректируйте запасы. Можно создать ручные перемещения, использовать пополнение или применить любой другой поток для корректировки запасов.

### <a name="part-1-create-picking-work"></a>Часть 1. Создание работы комплектации

Прежде чем приступить к созданию работы, убедитесь, что склад настроен на реагирование на запросы работы ожидаемым образом.

Выполните следующие действия, чтобы создать некоторую работу комплектации.

1. Перейдите в раздел **Продажи и маркетинг \> Заказы на продажу \> Все заказы на продажу**.
1. Выберите **Создать** , чтобы открыть диалоговое окно **Создание заказа на продажу**.
1. В диалоговом окне **Создание заказа на продажу** установите следующие значения:

    - На экспресс-вкладке **Клиент** задайте для поля **Счет клиента** значение _US-001_.
    - На экспресс-вкладке **Общее** задайте в поле **Склад** значение _51_.

1. Выберите **ОК** , чтобы создать заказ на продажу и закрыть диалоговое окно.
1. Открывается ваш новый заказ на продажу. Он включает новую пустую строку в сетке **Строки заказа на продажу**. В этом строке заказа установите следующие значения:

    - **Код номенклатуры:** _M9200_
    - **Количество:** _20_
    - **Ед. изм:** _шт._

1. Выберите новую строку заказа, затем в меню **Запасы** выберите **Резервирование** , чтобы открыть страницу **Резервирование**.
1. На странице **Резервирование** выберите **Резервирование лота** , чтобы зарезервировать полное количество по выбранной строке на складе.
1. Закройте страницу **Резервирование** , чтобы вернуться в заказ на продажу.
1. На панели операций на вкладке **Склад** выберите вариант **Выпустить на склад**. Система создает отгрузку, добавляет ее в новую загрузку и создает требуемую работу.
1. Создайте второй заказ на продажу для того же счета клиента и склада, который использовался для первого заказа. Добавьте следующие две строки заказа в этот заказ:

    - **Строка 1:** задайте в поле **Код номенклатуры** значение _M9200_ , в поле **Количество** значение _25_ , и в поле **Единицы измерения** значение _шт._.
    - **Строка 2:** задайте в поле **Код номенклатуры** значение _M9202_ , в поле **Количество** значение _10_ , и в поле **Единицы измерения** значение _шт._.

1. Повторите шаги с 6 по 8, чтобы зарезервировать запасы для каждой строки заказа (по одной за раз), затем повторите шаг 9, чтобы выпустить заказ на склад.

### <a name="part-2-change-the-location-for-a-work-line"></a>Часть 2. Изменение местоположения для строки работы

1. Перейдите в раздел **Управление складом \> Работа \> Сведения о строке работы**.
1. Найдите и выберите одну из строк работы, которая была создана для этой демонстрации.
1. Выберите **Изменить местоположение** , чтобы открыть диалоговое окно **Выберите новое местонахождение**.
1. В диалоговом окне **Выберите новое местонахождение** в поле **Местонахождение** выберите новое местонахождение для строки работы.
1. Выберите **ОК** , чтобы применить изменение и закрыть диалоговое окно.

> [!IMPORTANT]
> Можно отправлять изменения местонахождения только в том случае, если для нового местонахождения имеется достаточное количество запасов (для комплектации) или если оно проходит проверку типа местонахождения (для размещения).

### <a name="part-3-change-the-quantity-of-a-work-line-or-cancel-a-work-line"></a>Часть 3. Изменение количества в строке работы или отмена строки работы

1. Перейдите в раздел **Управление складом \> Работа \> Сведения о строке работы**.
1. Найдите и выберите одну из строк работы, которая была создана для этой демонстрации. Обратите внимание, что можно отменить или изменить количества только для строк работы с типом работы _Комплектация_.
1. Выберите **Отменить строку работы** , чтобы открыть диалоговое окно **Отменяемое количество**.
1. В диалоговом окне **Отменяемое количество** измените значение в поле **Количество** , чтобы указать количество, которое должно *вычитаться из* количества, которое в данный момент указано для строки. По умолчанию в поле **Количество** отображается полное количество.

    - При отмене полного количества значение **Статус работы** будет изменено на _Отменено_ , но в поле **Количество работы** все равно будет отображаться исходное значение.
    - Если отменить только часть количества, поле **Количество работы** будет обновлено для отображения нового значения, но значение **Статус работы** не изменится.

1. Выберите **ОК** , чтобы применить изменение и закрыть диалоговое окно.

> [!IMPORTANT]
> Если отменяется только часть количества для строки работы, необходимо также удалить устаревшее количество из строки загрузки. В противном случае, если недопоставка настроена неправильно, для строки загрузки невозможно будет подтвердить отгрузку.
