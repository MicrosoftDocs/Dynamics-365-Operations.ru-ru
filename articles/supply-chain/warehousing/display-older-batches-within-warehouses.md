# <a name="configure-display-older-batches-within-warehouse-on-a-mobile-device"></a>Настройка отображения более старых партий на складе на мобильном устройстве

[!include [banner](../includes/banner.md)]

Конфигурация **Показать более старые партии на складе** позволяет отобразить список местонахождений с партиями старше, чем текущее местонахождение строки работы. Список отображаемых местонахождений включает сведения о более старых партиях в местонахождении с указанием даты окончания срока действия и физических запасов каждой партии. Можно выбрать комплектацию из нового местонахождения или продолжить комплектацию из текущего местонахождения. 
- Комплектация из нового местонахождения. Если выбрать новое местонахождение для комплектации, текущая строка работы будет обновлена для использования нового местонахождения, и работа будет продолжена в обычном режиме с использованием нового местонахождения. Чтобы новое местонахождение было действительным, оно должно иметь достаточное доступное количество для всей строки работы. Если требуемое количество отсутствует, строка работы не будет обновлена, и отобразится список. 
- Продолжить комплектацию из текущего местонахождения. Если вы продолжаете использовать текущее местонахождение строки работы, количества для строки работы будут продолжать комплектоваться из первоначального местонахождения.

## <a name="where-it-applies"></a>Где это применяется
Отображение более старых партий на складе настраивается в меню мобильного устройства и влияет на комплектацию следующих партий номенклатур.

## <a name="set-up-display-older-batches-within-warehouse"></a>Настройка отображения более старых партий на складе
Конфигурация **Показать более старые партии на складе** доступна в пунктах меню мобильного устройства, если для параметра **Выбрать самую старую партию** задать значение **Предупредить**.

- В разделе **Управление складом** > **Настройка** > **Мобильное устройство** > **Пункты меню мобильного устройства** задайте для параметра **Использовать существующую работу** значение **Да** для пункта меню и выберите **Предупредить** в поле **Выбрать самую старую партию**.
