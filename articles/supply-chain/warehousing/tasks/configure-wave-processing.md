# <a name="configure-wave-processing"></a><span data-ttu-id="23560-103">Настройка обработки волн</span><span class="sxs-lookup"><span data-stu-id="23560-103">Configure wave processing</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="23560-104">В этом руководстве описывается, как настроить критерии, определяющие работу, создаваемую для склада при обработке волны, и способ обработки волн (вручную или автоматически).</span><span class="sxs-lookup"><span data-stu-id="23560-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="23560-105">Чтобы определить критерии, настройте шаблоны и запросы волн, которые соответствуют волне с запущенными в производство строками в заказах на продажу, производственных заказах или заказах канбана.</span><span class="sxs-lookup"><span data-stu-id="23560-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="23560-106">Обработка волны используется на складах, использующих функции модуля "Управление складом", и не используется на складах, использующих функции модуля "Управление запасами".</span><span class="sxs-lookup"><span data-stu-id="23560-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="23560-107">Эту процедуру можно выполнить, используя компанию с демонстрационными данными USMF.</span><span class="sxs-lookup"><span data-stu-id="23560-107">You can run this procedure in demo data company USMF.</span></span>

6. <span data-ttu-id="23560-118">В поле **Склад** введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="23560-118">In the **Warehouse** field, enter or select a value.</span></span> <span data-ttu-id="23560-119">При использовании USMF можно выбрать склад 24.</span><span class="sxs-lookup"><span data-stu-id="23560-119">If you're using USMF, you can select warehouse 24.</span></span>

9. <span data-ttu-id="23560-124">Установите для параметра **Автоматический запуск волны** значение **Да**.</span><span class="sxs-lookup"><span data-stu-id="23560-124">Set the **Automate wave release option** to **Yes**.</span></span> <span data-ttu-id="23560-125">Установите этот флажок для автоматического запуска волны в производство.</span><span class="sxs-lookup"><span data-stu-id="23560-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="23560-126">Работа комплектации создается и становится доступна на мобильных устройствах.</span><span class="sxs-lookup"><span data-stu-id="23560-126">The picking work is created and made available on mobile devices.</span></span>
