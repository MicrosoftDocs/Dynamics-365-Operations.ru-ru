# <a name="set-up-short-picking-item-reallocation"></a>Настройка перераспределения номенклатур для недоукомплектованности

[!include [task guide banner](../../includes/task-guide-banner.md)]

На этой процедуры показано, как разрешить работникам склада быстро искать альтернативные местоположения, если нет достаточного количества запасов в ячейке, к которой они были направлены. Можно использовать автоматический процесс перераспределения, который использует директивы местоположения, чтобы получить товары, если они доступны в другом месте. Кроме того, если используется перераспределение вручную, список местоположений с доступным количеством отображается на мобильном устройстве, позволяя работнику склада выбрать, из какого местоположения использовать запасы. Эту процедуру можно выполнить, используя компанию с демонстрационными данными USMF. Эта процедура для функции, которая была добавлена в версии 1611 Dynamics 365 for Operations.


## <a name="set-up-work-exceptions"></a>Настройка исключений работ
1. Перейдите в раздел "Управление складом" > "Настройка" > "Работа" > "Исключения работы".
2. Щелкните "Создать".
    * Можно определить несколько исключений работы с различными политиками переразмещения номенклатур, чтобы рабочий склада мог выбрать одно из них на основе потребностей отгрузки, которую он обрабатывает.  
3. В поле "Код исключения работы" введите значение.
    * Задайте название исключения работы для указания его назначения. Например, "Руководство по недостаточной комплектации".  
4. В поле "Описание" введите значение.
5. В поле "Тип исключения" выберите "Недоукомплектовать".
6. Установите флажок "Корректировка складских запасов".
    * Этот параметр означает, что запасы будут автоматически скорректированы на 0 в местоположении с недостаточной комплектацией.  
7. В поле "Код типа корректировки по умолчанию" введите или выберите значение.
    * Например, в USMF можно выбрать "Удалить Res Adj Out".  
8. В поле "Перераспределение номенклатур" выберите "Вручную".
    * Если выбрано "Вручную" или "Автоматически и вручную", то работнику склада должно быть разрешено выполнять переразмещение вручную.  

## <a name="set-up-a-worker-to-use-manual-item-reallocation"></a>Настройте работника для использования ручного перераспределения номенклатуры
1. Закройте страницу.
2. Перейдите в раздел "Управление складом" > "Настройка" > "Рабочий".
3. Щелкните "Изменить".
4. В списке выберите работника 24.
5. Разверните раздел "Работа".
6. Выберите "Да" в поле "Разрешить перераспределение номенклатур вручную".
