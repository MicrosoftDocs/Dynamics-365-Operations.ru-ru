# <a name="configure-locations-in-a-wms-enabled-warehouse"></a>Настройка мест хранения на складе, на котором включена система управления складом

[!include [banner](../../includes/banner.md)]

Это руководство показывает, как настроить места хранения для нового склада с поддержкой системы управления складом (склада, на котором используются комплексные процессы управления складом). Процесс обычно выполняется менеджером склада. Запускать этот руководство можно в компании с демонстрационными данными USMF или с собственными данными. Предварительным условием является наличие хотя бы одного настроенного сайта.


## <a name="create-a-new-warehouse"></a>Создание нового склада
1. Выберите **Область переходов > Модули > Управление запасами > Настройка > Разделение запасов > Склады**.
2. Нажмите кнопку **Создать**.
3. В поле **Склад** введите значение.
4. В поле **Имя** введите значение.
5. В поле **Сайт** введите значение.
6. Разверните раздел **Склад**.
7. Установите параметр **Использовать процессы управления складом** в значение "Да". Эта настройка позволяет запускать сложные комплексные процессы управления складом с использованием складских работ и мобильных устройств.
8. Закройте страницу.

## <a name="define-a-location-format"></a>Определение формата местонахождения
1. Выберите **Область переходов > Модули > Управление складом > Настройка > Склад > Форматы места хранения**. Форматы места хранения — этот система именования, используемая для создания уникальных единообразных имен для различных позиций (ячеек хранения), используемых на складе. В формате места хранения имеет смысл использовать разделители, чтобы было легче идентифицировать компоненты места хранения, такие как номер прохода. В этом примере мы создадим имя из четырех компонентов. Например, это могут быть проход, стойка, полка и ячейка.
2. Нажмите кнопку **Создать**.
3. В поле **Формат местонахождения** введите значение.
4. В поле **Имя** введите значение.
5. В поле **Описание сегмента** введите значение. Здесь описывается, что представляет собой первый компонент имени места хранения. Например, это может быть "Проход".
6. В поле **Длина** введите число. Это поле определяет, сколько знаков должна содержать эта часть имени местонахождения. Обратите внимание, что в сумме все компонентов в имени, включая разделители, не может быть больше 10 символов.    
7. В поле **Разделитель** введите значение. Это поле определяет, какой символ используется между первым и вторым компонентом имени.  
8. В разделе **Сведения** щелкните **Создать**.
9. В поле **Описание сегмента** введите значение.
10. В поле **Длина** введите число.
11. В поле **Разделитель** введите значение.
12. В разделе **Сведения** щелкните **Создать**.
13. В поле **Описание сегмента** введите значение.
14. В поле **Длина** введите число.
15. В поле **Разделитель** введите значение.
16. В разделе **Сведения** щелкните **Создать**.
17. В поле **Описание сегмента** введите значение.
18. В поле **Длина** введите число.
19. Нажмите кнопку **Сохранить**.
20. Закройте страницу.

## <a name="define-location-types"></a>Определение типов местонахождений
1. Выберите **Область переходов > Модули > Управление складом > Настройка > Склад > Типы местонахождения**. Типы мест хранения можно использовать в качестве параметров фильтрации для управления различными процессами управления складом. Как минимум необходимо создать типы мест хранения для промежуточного хранения и отгрузки, чтобы определить исходящий процесс управления складом. 
2. Нажмите кнопку **Создать**.
3. В поле **Тип местоположения** введите значение.
4. В поле **Описание** введите значение.
5. Закройте страницу.

## <a name="define-location-profile"></a>Определение профиля местонахождения
1. Выберите **Область переходов > Модули > Управление складом > Настройка > Склад > Профили местонахождения**. Определение профилей мест хранения имеет большую важность. Это позволяет контролировать емкость сгруппированных мест хранения, а также управлять политиками, связанными с тем, какие запасы в них хранятся и каким образом. Профили мест хранения можно использовать в качестве параметров фильтрации для управления различными процессами управления складом. Как минимум необходимо создать пользовательский профиль места хранения, чтобы сделать возможными процессы управления складом.
2. Нажмите кнопку **Создать**.
3. В поле **Код профиля местонахождения** введите значение.
4. В поле **Имя** введите значение.
5. В поле **Формат местонахождения** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. Поиск и выбор требуемой записи в списке.
7. В списке перейдите по ссылке в выбранной строке.
8. В поле **Тип местоположения** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
9. В списке найдите и выберите требуемую запись.
10. В списке перейдите по ссылке в выбранной строке.
11. Установите или снимите флажок **Разрешить смешанные статусы запасов**. Включите этот параметр, если требуется разрешить смешанные значения статуса запасов в местах хранения, которые будут группироваться по этому профилю места хранения. 
12. Установите или снимите флажок **Переопределить правила для пакетных дней**. Включите этот параметр, чтобы переопределить правило о том, на сколько дней могут отличаться даты истечения срока годности партий запасов и, соответственно, разрешить смешивать партии запасов, которые не удовлетворяют этому правилу.  
13. Установите или снимите флажок **Разрешить подсчет циклов**. Включите этот параметр, чтобы разрешить обработку подсчета циклов во всех местах гранения, которые будут группироваться по этому профилю места хранения. 
14. Разверните или сверните раздел **Аналитики**. Вкладка "Аналитики" позволяет определить параметры и методы для включения точных расчетов емкости в рамках каждого из мест хранения.  
15. Закройте страницу.

## <a name="enable-warehouse-management-parameters"></a>Включение параметров управления складом
1. Перейдите в **Область перехода > Модули > Управление складом > Настройка > Параметры управления складом**. Для обработки работы склада необходимо задать параметры для профиля места хранения пользователя, типа места хранения "промежуточное хранение" и типа места хранения "конечное местонахождение отгрузки". Как только исходящий процесс заканчивается в определенном вами конечном местонахождении отгрузки, статус связанных с ним исходящих проводок меняется на "Скомплектовано".
2. Разверните или сверните раздел **Профили местонахождения**.
3. В поле **Местонахождение пользователя** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. В списке перейдите по ссылке в выбранной строке.
5. Разверните или сверните раздел **Типы местонахождения**.
6. В поле **Тип промежуточного местонахождения** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
7. В списке перейдите по ссылке в выбранной строке.
8. В поле **Тип последнего местонахождения отгрузки** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
9. В списке перейдите по ссылке в выбранной строке.
10. Закройте страницу.

## <a name="define-warehouse-zone-groups"></a>Определение групп зон склада
1. Перейдите в **Область перехода > Модули > Управление складом > Настройка > Склад > Группы зон склада**. Зоны склада можно использовать в качестве фильтров для управления различными процессами управления складом. Прежде чем можно будет определить зону, необходимо создать группу зон.   
2. Нажмите кнопку **Создать**.
3. В поле **Код группы зон** введите значение.
4. В поле **Имя группы зоны** введите значение.
5. Закройте страницу.

## <a name="define-warehouse-zones"></a>Определение зон склада
1. Выберите **Область переходов > Модули > Управление складом > Настройка > Склад > Зоны**.
2. Нажмите кнопку **Создать**.
3. В поле **Код зоны** введите значение.
4. В поле **Имя зоны** введите значение.
5. В поле **Код группы зон** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. Поиск и выбор требуемой записи в списке.
7. В списке перейдите по ссылке в выбранной строке.
8. Закройте страницу.

## <a name="create-locations-using-the-location-setup-wizard"></a>Создание местонахождений с помощью мастера настройки местонахождения
1. Выберите **Область переходов > Модули > Управление складом > Настройка > Склад > Мастер настройки местонахождения**.
2. В поле **Склад** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
3. Поиск и выбор требуемой записи в списке.
4. В списке перейдите по ссылке в выбранной строке.
5. В поле **Код зоны** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. Поиск и выбор требуемой записи в списке.
7. В списке перейдите по ссылке в выбранной строке.
8. В поле **Код профиля местонахождения** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
9. Поиск и выбор требуемой записи в списке.
10. В списке перейдите по ссылке в выбранной строке.
11. В списке пометьте выбранную строку.
12. В поле **С номера** введите число. Поля "С номера" и "По номер" определяют, столько мест хранения будет создано. Например, если в поле "С номера" указать значение 1, а в поле "По номер" указать значение 3 для всех четырех строк в формате места хранения, будет создано 81 место хранения (3x3x3x3).   
13. В поле **По номер** введите число.
14. В списке найдите и выберите требуемую запись.
15. В поле **С номера** введите число.
16. В поле **По номер** введите число.
17. Поиск и выбор требуемой записи в списке.
18. В поле **С номера** введите число.
19. В поле **По номер** введите число.
20. Поиск и выбор требуемой записи в списке.
21. В поле **С номера** введите число.
22. В поле **По номер** введите число.
23. Щелкните "Создать".

## <a name="create-locations-manually"></a>Создание местонахождений вручную
1. Перейдите в раздел **Управление складом > Настройка > Склад > Местонахождения**. Места хранения в пределах склада легко создавать вручную. Имя местонахождения и код профиля местонахождения — обязательные значения. 
2. Нажмите кнопку **Создать**.
3. В поле **Склад** введите значение.
4. В поле **Местонахождение** введите значение. Обратите внимание, что здесь вы создаете новое место хранения, поэтому необходимо ввести новое уникальное имя, а не выбирать существующее.
5. В поле **Код профиля местонахождения** введите значение.
6. Закройте страницу.

## <a name="define-pack-size-categories"></a>Определение категорий размеров упаковки
1. Перейдите в раздел **Управление складом > Настройка > Склад > Категории размеров упаковки**. Категории размера упаковки можно использовать для группировки номенклатур с похожими физическими размерами упаковки. В этом примере категория размера упаковки используется для управления емкостью ячеек комплектации в конкретной зоне склада. Обратите внимание, что объекту запущенного в производство продукта должен быть назначен код категории размера упаковки, чтобы его можно было использовать в рамках обработки лимитов хранения.  
2. Нажмите кнопку **Создать**.
3. В поле **Код категории размеров упаковки** введите значение.
4. В поле **Имя категории размеров упаковки** введите значение.
5. Закройте страницу.

## <a name="define-location-stocking-limits"></a>Определение лимитов хранения местонахождения
1. Перейдите в раздел **Управление складом > Настройка > Склад > Лимиты хранения местонахождения**. Лимиты хранения мест хранения помогают избежать создания работ, в которых запрашивается помещение запасов в место хранения, в котором нет физической емкости под эти запасы.  
2. Нажмите кнопку **Создать**.
3. В поле **Склад** введите значение.
4. В поле **Код профиля местонахождения** введите значение.
5. В поле **Код категории размеров упаковки** введите значение.
6. В поле **Количество** введите число.
7. Нажмите кнопку **Сохранить**.
8. Закройте страницу.

## <a name="define-fixed-picking-locations"></a>Определение фиксированных ячеек комплектации
1. Перейдите в раздел **Управление складом > Настройка > Склад > Фиксированные местонахождения**. Можно определить места хранения, которые будут использоваться для продукта или для варианта продукта. Можно создать несколько фиксированных мест хранения для одного и того же продукта в пределах одного и того же склада.     
2. Нажмите кнопку **Создать**.
3. В поле **Код номенклатуры** введите значение.
4. В поле **Склад** введите значение.
5. В поле **Местонахождение** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. В списке перейдите по ссылке в выбранной строке.
7. Закройте страницу.
