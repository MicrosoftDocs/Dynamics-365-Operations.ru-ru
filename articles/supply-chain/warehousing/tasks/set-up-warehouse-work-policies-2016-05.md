# <a name="set-up-warehouse-work-policies"></a>Настройка политик работы склада 

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Процессы склада не всегда включают работу склада. Определяя политику работы, можно предотвратить создание работы для комплектации сырья и размещения готовой продукции для набора продуктов в определенных местонахождениях. В качестве компании с демонстрационными данными для создания этой записи используется USMF. Для выполнения этого руководства по задаче требуется приложение Dynamics AX 7.0.1 или более поздней версии.

1. Перейдите в раздел "Управление складом" > "Настройка" > "Работа" > "Политики работы".
2. Щелкните "Создать".
3. В поле "Имя политики работы" введите "Не работа размещения".
4. Нажмите кнопку "Сохранить".
5. Нажмите кнопку Добавить.
6. В списке пометьте выбранную строку.
7. В поле "Тип заказа на выполнение работ" выберите "Размещение готовой продукции".
8. Нажмите кнопку Добавить.
9. В списке пометьте выбранную строку.
10. В поле "Тип заказа на выполнение работ" выберите "Размещение побочного и сопутствующего продуктов".
11. Разверните раздел "Места хранения".
12. Нажмите кнопку Добавить.
13. В списке пометьте выбранную строку.
14. В списке "Склад" введите "51".
15. В поле "Местонахождение" введите или выберите "001".
16. Разверните раздел "Продукты".
17. В поле "Выбор продукта" выберите "Выбрано".
18. Нажмите кнопку Добавить.
19. В списке пометьте выбранную строку.
20. В поле "Код номенклатуры" введите или выберите "L0101".
21. Нажмите кнопку "Сохранить".
