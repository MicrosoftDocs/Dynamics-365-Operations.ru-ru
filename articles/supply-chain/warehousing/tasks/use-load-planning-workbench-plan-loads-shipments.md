# <a name="plan-loads-and-shipments-using-the-load-planning-workbench"></a>Планирование загрузок и отгрузок с помощью рабочего места планирования загрузки

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как использовать рабочее место планирования загрузки для создания загрузки для заказа на продажу. Для начала необходимо будет создать заказ на продажу. Эта процедура является элементом повседневной работы координатора транспортировки. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="create-a-sales-order"></a>Создать заказ на продажу
1. Перейдите в раздел "Расчеты с клиентами" > "Заказы" > "Все заказы на продажу".
2. Щелкните "Создать".
3. В поле "Счет клиента" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. Выберите счет US-004.
5. Нажмите кнопку OK.
6. В поле "Код номенклатуры" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
7. Выберите номенклатуру A0001.
    * A0001 поддерживает управление транспортировкой.  
8. В списке перейдите по ссылке в выбранной строке.
9. В поле "Количество" введите число.
10. В поле "Склад" введите "24".
    * В этом примере выберите склад 24. Этот склад поддерживает управление транспортировкой и комплексные функции управления складом.  
11. Нажмите кнопку "Сохранить".
12. Закройте страницу.

## <a name="create-a-new-load"></a>Создание новой загрузки
1. Перейдите в раздел "Управление транспортировкой" > "Планирование" > "Рабочее место планирования загрузки".
2. Перейдите на вкладку "Строки продаж".
    * Теперь вам предстоит построить загрузку для только что созданного заказа на продажу. Загрузки можно строить на основе предложения и спроса по заказам на покупку, заказам на перемещение и заказам на продажу.  
3. В области действий щелкните "Поставки и спрос".
4. Щелкните "Для новой загрузки".
5. В поле "Код шаблона загрузки" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
    * Шаблон загрузки определяет максимальные измерения для веса и объема всей загрузки. Например, шаблон загрузки может представлять размер контейнера или грузовика.  
6. В списке перейдите по ссылке в выбранной строке.
7. Нажмите кнопку "OК".

## <a name="rate-and-route-the-load"></a>Оценка и маршрутизация загрузки
1. Щелкните "Оценка и маршрутизация".
2. Щелкните "Рабочее место маршрута ставки".
3. Щелкните "Магазин ставок".
4. В списке найдите и выберите требуемую запись.
5. Щелкните "Назначить".
6. Закройте страницу.
7. Закройте страницу.
