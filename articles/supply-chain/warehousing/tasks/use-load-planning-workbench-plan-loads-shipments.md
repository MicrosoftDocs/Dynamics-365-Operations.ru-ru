# <a name="plan-loads-and-shipments-using-the-load-planning-workbench"></a>Планирование загрузок и отгрузок с помощью рабочего места планирования загрузки

[!include [banner](../../includes/banner.md)]

В этом разделе показано, как использовать рабочее место планирования загрузки для создания загрузки для заказа на продажу. Для начала необходимо будет создать заказ на продажу. Эта процедура является элементом повседневной работы координатора транспортировки. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="create-a-sales-order"></a>Создать заказ на продажу
1. Перейдите к **Область перехода > Модули > Расчеты с клиентами > Заказы > Все заказы на продажу**.
2. Выберите **Создать**.
3. В поле **Счет клиента** выберите кнопку раскрывающегося списка, чтобы открыть подстановку.
4. Выберите счет **US-004**.
5. Нажмите **ОК**.
6. В поле **Код номенклатуры** выберите кнопку раскрывающегося списка, чтобы открыть подстановку.
7. Выбор номенклатуры **A0001**. **A0001** поддерживает управление транспортировкой.  
8. В поле **Сайт** выберите кнопку раскрывающегося списка, чтобы открыть поиск, затем выберите номенклатуру.
9. В поле **Количество** введите число.
10. В поле **Склад** введите «24» для этого примера. Этот склад поддерживает управление транспортировкой и комплексные функции управления складом.  
11. Нажмите **Сохранить**.
12. Закройте страницу.

## <a name="create-a-new-load"></a>Создание новой загрузки
1. Перейдите в раздел **Область перехода > Модули > Управление транспортировкой > Планирование > Рабочее место планирования загрузки**.
2. Выберите вкладку **Строки продаж**. Теперь вам предстоит построить загрузку для только что созданного заказа на продажу. Загрузки можно строить на основе предложения и спроса по заказам на покупку, заказам на перемещение и заказам на продажу.  
3. На панели операций выберите **Поставки и спрос**.
4. Выберите **В новую загрузку**.
5. В поле **Код шаблона загрузки** нажмите кнопку раскрывающегося списка, чтобы открыть поиск. Шаблон загрузки определяет максимальные измерения для веса и объема всей загрузки. Например, шаблон загрузки может представлять размер контейнера или грузовика. Выберите элемент номенклатуры.
6. Нажмите **ОК**.

## <a name="rate-and-route-the-load"></a>Оценка и маршрутизация загрузки
1. Выберите **Рейтинг и маршрутизация**.
2. Выберите **Рабочее место маршрута ставки**.
3. Выберите **Магазин ставок**.
4. В списке найдите и выберите требуемую запись.
5. Выберите **Назначить**.
6. Закройте страницу.
