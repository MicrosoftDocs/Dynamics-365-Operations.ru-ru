# <a name="set-up-a-min-max-replenishment-process"></a>Настройка процесса пополнения "Мин./Макс."

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как настроить новый процесс пополнения, использующий стратегию пополнения "Мин./Макс.". Когда уровень запасов опускается ниже минимального, создается работа для пополнения местонахождения. В этой процедуре также показано, как использовать фиксированные ячейки комплектации, чтобы разрешить пополнение запасов, даже если уровень запасов ниже минимального, и как выполнять процесс пополнения регулярно, используя пакетное задание. Эти задачи обычно выполняются менеджером склада. Эту процедуру можно выполнить, используя компанию с демонстрационными данными USMF и примеры значений ниже, либо собственные данные. При использовании собственных данных убедитесь, что для процессов управления складом включен склад.


## <a name="create-a-fixed-picking-location"></a>Создание фиксированной ячейки комплектации
1. Выберите **Область переходов > Модули > Управление складом > Настройка > Склад > Фиксированные местонахождения**. Это необязательная задача для пополнения "Мин./Макс.", но если используется фиксированная ячейка комплектации, запасы можно пополнять, даже если значение опускается ниже минимального уровня, поскольку система может определить, какие номенклатуры должны быть пополнены, даже если их больше не осталось.
2. Нажмите кнопку **Создать**.
3. В поле **Код номенклатуры** введите или выберите значение. При использовании USMF можно выбрать номенклатуру A0001.  
4. В поле **Сайт** введите или выберите значение. При использовании USMF можно выбрать сайт 2.  
5. В поле **Склад** введите или выберите значение. При использовании USMF можно выбрать склад 24.  
6. В поле **Местонахождение** введите или выберите значение. При использовании USMF можно выбрать CP-003.  
7. Закройте страницу.

## <a name="create-a-replenishment-location-directive"></a>Создание директивы местонахождения пополнения
1. Перейдите в раздел **Управление складом > Настройка > Директивы местонахождений**. Директивы местонахождений используются для определения того, должны ли номенклатуры комплектоваться в процессе пополнения.
2. В поле **Тип заказа на выполнение работ** выберите "Пополнение".
3. В области **Панель операций** щелкните **Создать**.
4. В поле **Имя** введите значение.
5. В поле **Тип работы** выберите "Комплектация".
6. В поле **Сайт** введите или выберите значение. При использовании USMF можно выбрать сайт 2.  
7. В поле **Склад** введите или выберите значение. При использовании USMF можно выбрать склад 24.  
8. Нажмите кнопку **Сохранить**.
9. В разделе **Строки** щелкните **Создать**.
10. В списке пометьте выбранную строку.
11. В поле **Количество "До"** введите число. Например, можно задать значение 9999.  
12. Установите флажок **Разрешить разделение**. При выборе этого параметра в процессе создания работы можно будет разделить количества строк работы между несколькими местонахождениями.  
13. Нажмите кнопку **Сохранить**.
14. В разделе **Действия директивы для места хранения** щелкните **Создать**.
15. В списке пометьте выбранную строку.
16. В поле **Имя** введите значение.
17. Нажмите кнопку **Сохранить**.
18. В разделе **Область действий** щелкните **Изменить запрос**. Можно изменить этот запрос для добавления ограничений того, откуда можно выбирать запасы в процессе пополнения. Например, можно указать, что следует использовать запасы только из массовой области склада.
19. Щелкните **OK**.
20. Закройте страницу.

## <a name="create-a-replenishment-work-template"></a>Создание шаблона работы пополнения
1. Перейдите в раздел **Управление складом > Настройка > Работа > Шаблоны работ**. Шаблон работы используется для указания системе того, как следует создавать работу пополнения "Мин./Макс.". Как минимум, необходимо наличие строки шаблона работы для комплектации и размещения. Шаблон работы будет недействительным до тех пор, пока не будет заполнена вся необходимая информация. 
2. В поле **Тип заказа на выполнение работ** выберите "Пополнение".
3. В области **Панель операций** щелкните **Создать**.
4. В поле **Шаблон работы** введите значение.
5. Нажмите кнопку **Сохранить**.
6. В разделе **Сведения шаблона работы** щелкните **Создать**.
7. В поле **Тип работы** выберите "Комплектация".
8. В поле **Код класса работы** введите или выберите значение. Это должен быть класс работы, связанный с пополнением. При использовании компании USMF выберите "Пополнение".  
9. В разделе **Сведения шаблона работы** щелкните **Создать**.
10. В списке пометьте выбранную строку.
11. В поле **Тип работы** выберите "Поместить".
12. В поле **Код класса работы** введите или выберите значение.
13. Нажмите кнопку **Сохранить**.
14. Закройте страницу.

## <a name="create-a-new-replenishment-template"></a>Создание нового шаблона пополнения
1. Перейдите в раздел **Управление складом > Настройка > Пополнение > Шаблоны пополнения**. Шаблон пополнения используется для определения номенклатур, количеств и местонахождения для пополнения.
2. В области **Панель операций** щелкните **Создать**.
3. В поле **Шаблон пополнения** введите значение. Присвойте шаблону имя для указания того, что он предназначен для пополнения "Мин./Макс.".  
4. В поле **Описание** введите значение.
5. Установите флажок **Разрешить волне спроса использовать незарезервированные количества**. Если этот флажок установлен, при пополнении спроса волны можно использовать количества, связанные с пополнением "Мин./Макс.". Например, это может быть полезно, если работа пополнения "Мин./Макс." не обрабатывается немедленно, чтобы избежать создания ненужно работы пополнения спроса.
6. В разделе **Сведения о шаблоне пополнения** щелкните **Создать**.
7. В поле **Порядковый номер** введите число.
8. В поле **Описание** введите значение.
9. В списке пометьте выбранную строку.
10. В поле **Единица пополнения** введите или выберите значение. Например, выберите "шт.". Этот параметр является обязательным. Он позволяет определить единицу измерения для работы пополнения, отличную от единицы измерения, указанной для минимального и максимального уровня запасов в этом шаблоне.
11. В поле **Шаблон работы** введите или выберите значение. Выберите созданный ранее шаблон работы.  
12. В поле **Минимальное количество** введите число. Выберите минимальное количество, которое должно запускать пополнение. Например, выберите значение "50". Можно оставить значение нуль, если пополняется фиксированное местонахождение и для параметра **Пополнять пустые фиксированные местонахождения** задано значение "Да". Также рекомендуется выбрать параметр **Пополнять только фиксированные местонахождения** по соображениям эффективности.
13. В поле **Максимальное количество** введите число. Например, выберите значение "100".  
14. В поле **Единица измерения** введите или выберите значение. Назначьте единицу измерения для минимального и максимального количества. Например, выберите значение "шт.".  
15. Установите флажок **Пополнять пустые фиксированные местонахождения**. Установите этот флажок для пополнения фиксированных местонахождений, когда они пусты. В противном случае будут пополняться только местонахождения, в которых имеется количество в наличии.
16. Установите флажок **Пополнять только фиксированные местонахождения**.
17. Щелкните **Выбрать продукты**. В этой области можно определить, какие продукты должны быть пополнены. Если выбран параметр "Фиксированные ячейки комплектации", также необходимо указать местонахождения в этом запросе. Доступны запросы определенных вариантов и определенных продуктов.
18. Выберите строку **Номенклатуры**.
19. В поле **Критерии** введите значение. Выберите номенклатуры, которые должны быть пополнены для фиксированных местонахождений. Например, введите "A*", чтобы выбрать все коды номенклатур, начинающиеся с "A".
20. Нажмите кнопку **Добавить**. Добавьте объект местонахождения (если он еще не существует), чтобы иметь возможность ограничить работу пополнения фиксированными ячейкам комплектации в определенной области склада.
21. В списке пометьте выбранную строку.
22. Установите в поле **Таблица** значение "Местоположения".
23. В поле **Поле** выберите "Код профиля местонахождения".
24. В поле **Критерии** введите или выберите значение.
25. Щелкните **OK**.
26. Закройте страницу.

## <a name="set-the-replenishment-process-to-run-as-a-batch-job"></a>Настройка процесса пополнения для выполнения пакетного задания
1. Перейдите в раздел **Управление складом > Пополнение > Пополнения**. Страница "Пополнения" позволяет настроить пополнение для выполнения как пакетного задания или для того, чтобы указать, что его следует запускать вручную.
2. Щелкните **Фильтр**.
3. В списке пометьте выбранную строку.
4. В поле **Критерии** введите или выберите значение.
5. Щелкните **OK**.
6. Разверните раздел **Выполнять в фоновом режиме**.
7. Установите для параметра **Пакетная обработка** значение "Да".
8. Щелкните **Повторение**.
9. Выберите параметр **Дата завершения не указана**.
10. Задайте **Шаблон повторения**. В этом примере выберите "Дни".  
11. Щелкните **OK**.
12. Щелкните **OK**.
