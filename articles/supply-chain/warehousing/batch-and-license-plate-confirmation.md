# <a name="batch-and-license-plate-confirmation"></a>Подтверждение партии и грузоместа

[!include [banner](../includes/banner.md)]

Подтверждение партии позволяет подтвердить, что правильная партия комплектуется, на мобильном устройстве. Только при первоначальной комплектации работы для указанных выше номенклатур, где партия выше означает, что партия выше чем расположение в иерархии поиска, необходимо убедиться, что скомплектованная партия соответствует партии в строке работы.

Подтверждение грузоместа позволяет подтвердить, что правильное грузоместо комплектуется, на мобильном устройстве. При работе комплектации из местоположения стадии необходимо проверить, что комплектуемое грузоместо соответствует грузоместу, связанному с работой. Если работа начинается путем сканирования грузоместа, этот шаг подтверждения будет пропущен.

## <a name="where-it-applies"></a>Где он применяется.

Подтверждение применяется в следующих случаях:

- Подтверждение партии применяется к первоначальной комплектации работы для номенклатур "партия выше".
- Подтверждение грузоместа применяется к комплектации из расположения стадии.

> [!IMPORTANT]
> Пополнение не поддерживается для подтверждения грузомест. При выполнении работы по пополнению не создается шаг для подтверждения грузоместа.

## <a name="set-up-batch-and-license-plate-confirmation"></a>Настройка подтверждения партии и грузоместа

Можно настроить подтверждение партии и грузоместа в меню на мобильном устройстве.

1. В пунктах меню мобильного устройства введите настройки подтверждения работы.  
1. Выберите параметр для подтверждения партии или грузоместа. Оба параметра доступны для комплектаций типа работы, у которых не включено автоматическое подтверждение.
