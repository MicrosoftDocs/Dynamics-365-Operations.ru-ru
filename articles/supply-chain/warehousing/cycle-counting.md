<span data-ttu-id="4b35c-107">Подсчет циклов — это процесс склада, который можно использовать для аудита номенклатур запасов в наличии.</span><span class="sxs-lookup"><span data-stu-id="4b35c-107">Cycle counting is a warehouse process that you can use to audit on-hand inventory items.</span></span> <span data-ttu-id="4b35c-108">Процесс подсчета циклов можно описать в три шага:</span><span class="sxs-lookup"><span data-stu-id="4b35c-108">The cycle counting process can be described in three steps:</span></span>
