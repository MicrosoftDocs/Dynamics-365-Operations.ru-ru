<span data-ttu-id="e6d08-162">Директивы местоположения определяют, *где* номенклатуры должны комплектоваться и *где* они должны быть размещены.</span><span class="sxs-lookup"><span data-stu-id="e6d08-162">Location directives determine *where* items should be picked and *where* they should be put.</span></span> <span data-ttu-id="e6d08-163">Система оценивает директиву местоположения по каждой строке работы, а затем выбирает местоположение на основе сведений о строке работы.</span><span class="sxs-lookup"><span data-stu-id="e6d08-163">The system evaluates a location directive against each work line and then selects a location, based on the work line details.</span></span> <span data-ttu-id="e6d08-164">Система сначала находит все директивы местоположения, соответствующие определенной строке работы (например, они относятся к правильному складу и соответствуют запросу).</span><span class="sxs-lookup"><span data-stu-id="e6d08-164">The system first finds all location directives that match a particular work line (for example, they are for the correct warehouse and match the query).</span></span> <span data-ttu-id="e6d08-165">Затем последовательно выполняются найденные директивы.</span><span class="sxs-lookup"><span data-stu-id="e6d08-165">It then sequentially evaluates the directives that it has found.</span></span>
