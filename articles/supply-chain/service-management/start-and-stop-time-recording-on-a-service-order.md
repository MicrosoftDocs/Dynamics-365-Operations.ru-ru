# <a name="start-and-stop-time-recording-on-a-service-order"></a><span data-ttu-id="fd162-103">Начало и остановка записи времени в заказе на обслуживание</span><span class="sxs-lookup"><span data-stu-id="fd162-103">Start and stop time recording on a service order</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="fd162-104">Данная процедура используется для запуска и остановки записи времени для заказа на сервисное обслуживание, для которого определено соглашение об условиях обслуживания.</span><span class="sxs-lookup"><span data-stu-id="fd162-104">Use this procedure to start and stop time recording for a service order for which a service level agreement is defined.</span></span>

## <a name="start-time-recording"></a><span data-ttu-id="fd162-105">Начало записи времени</span><span class="sxs-lookup"><span data-stu-id="fd162-105">Start time recording</span></span>

1.  <span data-ttu-id="fd162-106">Щелкните **Управление сервисным обслуживанием** \> **Общий** \> **Заказы на сервисное обслуживание** \> **Заказы на сервисное обслуживание**.</span><span class="sxs-lookup"><span data-stu-id="fd162-106">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="fd162-107">Перейдите на вкладку **Заказ на сервисное обслуживание**. В разделе **Панель операций**, в группе **Соглашение об уровне обслуживания** щелкните **Начать**.</span><span class="sxs-lookup"><span data-stu-id="fd162-107">Click the **Service order** tab. On the **Action Pane**, in the **Service level agreement** group, click **Start**.</span></span>

3.  <span data-ttu-id="fd162-108">Введите дату и время, когда запись времени должна быть начата.</span><span class="sxs-lookup"><span data-stu-id="fd162-108">Enter the date and time that the time recording should be started.</span></span>

## <a name="stop-time-recording"></a><span data-ttu-id="fd162-109">Прекращение учета времени</span><span class="sxs-lookup"><span data-stu-id="fd162-109">Stop time recording</span></span>

1.  <span data-ttu-id="fd162-110">Щелкните **Управление сервисным обслуживанием** \> **Общий** \> **Заказы на сервисное обслуживание** \> **Заказы на сервисное обслуживание**.</span><span class="sxs-lookup"><span data-stu-id="fd162-110">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="fd162-111">Перейдите на вкладку **Заказ на сервисное обслуживание**. В разделе **Панель операций**, в группе **Соглашение об уровне обслуживания** щелкните **Остановить**.</span><span class="sxs-lookup"><span data-stu-id="fd162-111">Click the **Service order** tab. On the **Action Pane**, in the **Service level agreement** group, click **Stop**.</span></span>

3.  <span data-ttu-id="fd162-112">Введите дату и время, когда запись времени должна быть остановлена.</span><span class="sxs-lookup"><span data-stu-id="fd162-112">Enter the date and time that the time recording should be stopped.</span></span>

4.  <span data-ttu-id="fd162-113">Выберите **Добавить причину отзыва**, а затем выберите код причины в списке **Код причины этапа**, чтобы указать причину остановки записи времени.</span><span class="sxs-lookup"><span data-stu-id="fd162-113">Select **Add a revocation reason**, and select a reason code in the **Stage reason code** list to provide a reason for stopping the time recording.</span></span>


> [!NOTE]
> <P><span data-ttu-id="fd162-114">Если в форме <STRONG>Параметры управления сервисным обслуживанием</STRONG> выбрано <STRONG>Код основания при превышении по времени</STRONG>, необходимо указать код причины, прежде чем можно будет остановить запись времени.</span><span class="sxs-lookup"><span data-stu-id="fd162-114">If <STRONG>Reason code on exceeding time</STRONG> is selected in the <STRONG>Service management parameters</STRONG> form, you must provide a reason code before you can stop the time recording.</span></span></P>
