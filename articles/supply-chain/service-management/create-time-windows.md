# <a name="create-time-windows"></a><span data-ttu-id="518de-103">Создание окон времени</span><span class="sxs-lookup"><span data-stu-id="518de-103">Create time windows</span></span>   

[!include [banner](../includes/banner.md)]

<span data-ttu-id="518de-104">Окна времени можно использовать для оптимизации планирования строк заказа на обслуживание.</span><span class="sxs-lookup"><span data-stu-id="518de-104">You can use time windows to optimize the scheduling of service order lines.</span></span>

1. <span data-ttu-id="518de-105">Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Соглашения на обслуживание** \> **Окно времени**.</span><span class="sxs-lookup"><span data-stu-id="518de-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Time window**.</span></span>

3. <span data-ttu-id="518de-107">В поле **Окно времени** введите имя для окна времени.</span><span class="sxs-lookup"><span data-stu-id="518de-107">In the **Time window** field, enter a name for the time window.</span></span>
