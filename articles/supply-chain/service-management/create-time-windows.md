[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="488e9-104">Окна времени можно использовать для оптимизации планирования строк заказа на обслуживание.</span><span class="sxs-lookup"><span data-stu-id="488e9-104">You can use time windows to optimize the scheduling of service order lines.</span></span>

1. <span data-ttu-id="488e9-105">Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Соглашения на обслуживание** \> **Окно времени**.</span><span class="sxs-lookup"><span data-stu-id="488e9-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Time window**.</span></span>

3. <span data-ttu-id="488e9-107">В поле **Окно времени** введите имя для окна времени.</span><span class="sxs-lookup"><span data-stu-id="488e9-107">In the **Time window** field, enter a name for the time window.</span></span>
