# <a name="subscription-groups"></a>Группы подписок    

[!include [banner](../includes/banner.md)]


Подписки используют следующую общую информацию, указанную для группы подписок:

  - Код номерной серии подписки (если имеется).

  - Период выставления счетов, для которого созданы проводки подписок. Например, если для кода периода **Выставление накладных** в форме **Группы подписки** указано **Ежеквартально**, проводки по всем подпискам, входящим в группу подписки, создаются ежеквартально.

  - Периодичность начислений по подписке.

Для всех подписок, входящих в группу подписки, можно также выполнять следующие задачи:

  - Создание проводок по подписке.

  - Обновление базовой цены или указателя для подписок.

  - Разноска начисленного дохода по проводкам подписок.
