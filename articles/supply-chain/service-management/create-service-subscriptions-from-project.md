# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="def67-103">Создание подписок на обслуживание из проекта</span><span class="sxs-lookup"><span data-stu-id="def67-103">Create service subscriptions from a project</span></span>    

[!include [banner](../includes/banner.md)]


<span data-ttu-id="def67-104">Можно создать подписку на обслуживание из проекта в модуле **Управление и учет по проектам**.</span><span class="sxs-lookup"><span data-stu-id="def67-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="def67-105">Щелкните **Управление и учет по проектам** \> **Общее** \> **Проекты** \> **Все проекты**.</span><span class="sxs-lookup"><span data-stu-id="def67-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="def67-106">Откройте проект, для которого необходимо создать подписку.</span><span class="sxs-lookup"><span data-stu-id="def67-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="def67-107">В форме **Проекты**, в разделе **Панель операций**, на вкладке **Управление** щелкните **Обслуживание** \> **Подписка на обслуживание**.</span><span class="sxs-lookup"><span data-stu-id="def67-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="def67-108">В форме **Подписка**, в поле **Подписка на обслуживание** введите уникальный идентификационный номер.</span><span class="sxs-lookup"><span data-stu-id="def67-108">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="def67-109">Номер проекта будет автоматически введен в подписку для выбранного проекта.</span><span class="sxs-lookup"><span data-stu-id="def67-109">The project number is entered automatically into the subscription for the selected project.</span></span>

4.  <span data-ttu-id="def67-110">Введите остальные сведения в форму.</span><span class="sxs-lookup"><span data-stu-id="def67-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="def67-111">Теперь для этой подписки можно создавать транзакции по подписке.</span><span class="sxs-lookup"><span data-stu-id="def67-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="def67-112">Если для группы подписок настроена номерная серия, код подписки будет создан автоматически при указании группы подписок.</span><span class="sxs-lookup"><span data-stu-id="def67-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
