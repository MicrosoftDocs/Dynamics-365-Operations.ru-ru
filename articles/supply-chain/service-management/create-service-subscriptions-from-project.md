[!include [banner](../includes/banner.md)]


<span data-ttu-id="275d9-104">Можно создать подписку на обслуживание из проекта в модуле **Управление и учет по проектам**.</span><span class="sxs-lookup"><span data-stu-id="275d9-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

2.  <span data-ttu-id="275d9-107">В форме **Проекты**, в разделе **Панель операций**, на вкладке **Управление** щелкните **Обслуживание** \> **Подписка на обслуживание**.</span><span class="sxs-lookup"><span data-stu-id="275d9-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="275d9-110">Введите остальные сведения в форму.</span><span class="sxs-lookup"><span data-stu-id="275d9-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="275d9-111">Теперь для этой подписки можно создавать транзакции по подписке.</span><span class="sxs-lookup"><span data-stu-id="275d9-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="275d9-112">Если для группы подписок настроена номерная серия, код подписки будет создан автоматически при указании группы подписок.</span><span class="sxs-lookup"><span data-stu-id="275d9-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
