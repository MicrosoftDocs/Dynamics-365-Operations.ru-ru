<span data-ttu-id="054fe-105">Можно создать подписку на обслуживание из проекта в модуле **Управление и учет по проектам**.</span><span class="sxs-lookup"><span data-stu-id="054fe-105">You can create a service subscription from a project in **Project management and accounting**.</span></span>

2.  <span data-ttu-id="054fe-108">В форме **Проекты**, в разделе **Панель операций**, на вкладке **Управление** щелкните **Обслуживание** \> **Подписка на обслуживание**.</span><span class="sxs-lookup"><span data-stu-id="054fe-108">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="054fe-111">Введите остальные сведения в форму.</span><span class="sxs-lookup"><span data-stu-id="054fe-111">Enter the remaining information in the form.</span></span>

<span data-ttu-id="054fe-112">Теперь для этой подписки можно создавать проводки по подписке.</span><span class="sxs-lookup"><span data-stu-id="054fe-112">You can now create subscription transactions for the subscription.</span></span>
