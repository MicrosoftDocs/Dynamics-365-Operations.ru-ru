# <a name="indexed-sales-prices"></a>Индексированные цены продажи  

[!include [banner](../includes/banner.md)]


Настройка индекса для цены продажи подписки производится при создании платежа по подписке.

В форме **Создание проводок по сборам по подписке** задайте для поля **Получить цены из** значение **Индексированная базисная цена**, а затем умножьте базисную цену на процент из поля **Изменение цены в процентах**, чтобы получить цену продажи проводки по подписке.

Например, если базисная цена составляет 1 000 евро, а индекс равен 110, то продажная цена будет 1 100.
