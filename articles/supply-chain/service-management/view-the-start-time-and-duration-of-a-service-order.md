# <a name="view-the-start-time-and-duration-of-a-service-order"></a>Просмотр времени начала и длительности заказа на обслуживание 

[!include [banner](../includes/banner.md)]


Можно выполнять просмотр после начала и при завершении работы по заказу на обслуживание.

Можно также выполнять просмотр после начала и при остановке записи времени. При остановке заказа на обслуживание время, в которое должен быть выполнен заказ на обслуживание, переносится на более поздний срок.

## <a name="view-the-start-time-for-a-service-order"></a>Просмотр времени начала для заказа на обслуживание

1.  Щелкните **Управление сервисным обслуживанием** \> **Общий** \> **Заказы на сервисное обслуживание** \> **Заказы на сервисное обслуживание**. Выберите и дважды щелкните заказ, чтобы открыть форму со сведениями.

2.  На вкладке **Общее** просмотрите время начала работы для заказа на сервисное обслуживание в поле **Время начала**.

## <a name="view-the-time-remaining-to-complete-a-service-order"></a>Просмотр времени, оставшегося для выполнения заказа на обслуживание

1.  Щелкните **Управление сервисным обслуживанием** \> **Общий** \> **Заказы на сервисное обслуживание** \> **Заказы на сервисное обслуживание**. Выберите и дважды щелкните заказ, чтобы открыть форму со сведениями.

2.  На вкладке **Общее** просмотрите время, оставшееся для выполнения заказа на сервисное обслуживание, в поле **Последнее время выполнения**.

## <a name="view-the-start-time-and-stop-time-recording-entries-for-a-service-order"></a>Просмотрите строки записи времени начала и завершения для заказа об обслуживании

1.  Щелкните **Управление сервисным обслуживанием** \> **Общий** \> **Заказы на сервисное обслуживание** \> **Заказы на сервисное обслуживание**. Выберите и дважды щелкните заказ, чтобы открыть форму со сведениями.

2.  В разделе **Панель операций** перейдите на вкладку **Подготовить к отправке** \> **Учет времени**, чтобы открыть форму **Учет времени по соглашению об уровне обслуживания** и просмотреть строки записи времени для заказа на сервисное обслуживание.
