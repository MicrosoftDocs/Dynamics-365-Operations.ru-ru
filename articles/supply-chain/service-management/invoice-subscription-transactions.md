# <a name="invoice-subscription-transactions"></a>Обработка накладных для проводок по подписке 

[!include [banner](../includes/banner.md)]



1.  Щелкните **Управление сервисным обслуживанием** \> **Подписки на сервисное обслуживание** \> **Все подписки на сервисное обслуживание**.

2.  Выберите подписку на обслуживание, а затем щелкните **Изменить** либо щелкните **Подписка** для создания подписки на обслуживание.

3.  Щелкните **Анализировать** \> **Проводки по сборам**.

4.  Выберите одну или несколько проводок по сборам для выставления накладной, а затем щелкните **Функции** \> **Предложение по накладной**, чтобы открыть форму **Предложения по накладной**.

5.  Щелкните **Создать накладную** и введите соответствующие параметры в форме **Создать накладную**.

6.  Щелкните **OK**, чтобы создать предложение по накладной в форме **Предложения по накладной**.

7.  Проверьте содержимое накладной, щелкните **Разнести накладную**, чтобы открыть форму **Разноска накладной** для выполнения разноски накладной. Можно также распечатать накладную из формы **Разноска накладной**.
