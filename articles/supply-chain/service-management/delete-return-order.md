# <a name="delete-a-return-order"></a>Удаление заказа на возврат 

[!include [banner](../includes/banner.md)]


Форма **Удаление заказа на возврат** используется для удаления нескольких заказов на возврат.

1.  Щелкните **Продажи и маркетинг** \> **Периодические операции** \> **Очистка** \> **Удалить заказы на возврат**.

2.  Щелкните **Выбрать**, чтобы открыть форму **Обработка продаж**.

3.  Задайте критерии, а затем щелкните **OK** для возврата в форму **Удаление заказа на возврат**. Заказы на возврат, созданных для юридического лица, отображаются на основании заданных критериев.

4.  Выберите удаляемую строку заказа на возврат и щелкните **ОК**.

5.  Щелкните **Да** для удаления заказов на возврат.
