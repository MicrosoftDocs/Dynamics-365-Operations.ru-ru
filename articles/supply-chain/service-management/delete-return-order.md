[!include [banner](../includes/banner.md)]


<span data-ttu-id="b4b4c-104">Форма **Удаление заказа на возврат** используется для удаления нескольких заказов на возврат.</span><span class="sxs-lookup"><span data-stu-id="b4b4c-104">Use the **Delete return order** form to delete multiple return orders.</span></span>

1.  <span data-ttu-id="b4b4c-105">Щелкните **Продажи и маркетинг** \> **Периодические операции** \> **Очистка** \> **Удалить заказы на возврат**.</span><span class="sxs-lookup"><span data-stu-id="b4b4c-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>

2.  <span data-ttu-id="b4b4c-106">Щелкните **Выбрать**, чтобы открыть форму **Обработка продаж**.</span><span class="sxs-lookup"><span data-stu-id="b4b4c-106">Click **Select** to open the **Sales update** form.</span></span>

3.  <span data-ttu-id="b4b4c-107">Задайте критерии, а затем щелкните **OK** для возврата в форму **Удаление заказа на возврат**.</span><span class="sxs-lookup"><span data-stu-id="b4b4c-107">Specify the criteria, and then click **OK** to return to the **Delete return order** form.</span></span> <span data-ttu-id="b4b4c-108">Заказы на возврат, созданных для юридического лица, отображаются на основании заданных критериев.</span><span class="sxs-lookup"><span data-stu-id="b4b4c-108">Return orders that have been created in the legal entity are displayed, based on the specified criteria.</span></span>

5.  <span data-ttu-id="b4b4c-110">Щелкните **Да** для удаления заказов на возврат.</span><span class="sxs-lookup"><span data-stu-id="b4b4c-110">Click **Yes** to delete the return orders.</span></span>
