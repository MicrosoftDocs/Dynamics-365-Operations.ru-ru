# <a name="create-service-agreement-groups"></a>Создание групп соглашений на обслуживание 

[!include [banner](../includes/banner.md)]

Группы соглашений о сервисном обслуживании целесообразно использовать для сортировки и фильтрации соглашений о сервисном обслуживании.

1. Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Соглашения на обслуживание** \> **Группы соглашений о сервисном обслуживании**.
2. Нажмите **Alt+N**, чтобы создать новую группу соглашений о сервисном обслуживании.
3. В поле **Группа соглашений о сервисном обслуживании** введите код для группы соглашений о сервисном обслуживании.
4. В поле **Описание** введите описание группы соглашений о сервисном обслуживании.
5. Закройте страницу, чтобы сохранить изменения.

> [!NOTE]
> Если группа соглашений о сервисном обслуживании указана в поле **Группа соглашений о сервисном обслуживании** страницы **Параметры управления сервисным обслуживанием**, то программа предлагает эту группу для всех создаваемых новых соглашений о сервисном обслуживании.
