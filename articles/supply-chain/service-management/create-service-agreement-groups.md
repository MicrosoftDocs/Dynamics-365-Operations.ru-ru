# <a name="create-service-agreement-groups"></a><span data-ttu-id="8cf0c-103">Создание групп соглашений на обслуживание</span><span class="sxs-lookup"><span data-stu-id="8cf0c-103">Create service agreement groups</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="8cf0c-104">Группы соглашений о сервисном обслуживании целесообразно использовать для сортировки и фильтрации соглашений о сервисном обслуживании.</span><span class="sxs-lookup"><span data-stu-id="8cf0c-104">Service agreement groups are useful for sorting and filtering service agreements.</span></span>

1. <span data-ttu-id="8cf0c-105">Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Соглашения на обслуживание** \> **Группы соглашений о сервисном обслуживании**.</span><span class="sxs-lookup"><span data-stu-id="8cf0c-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.</span></span>
2. <span data-ttu-id="8cf0c-106">Нажмите **Alt+N**, чтобы создать новую группу соглашений о сервисном обслуживании.</span><span class="sxs-lookup"><span data-stu-id="8cf0c-106">Press **Alt+N** to create a new service agreement group.</span></span>
3. <span data-ttu-id="8cf0c-107">В поле **Группа соглашений о сервисном обслуживании** введите код для группы соглашений о сервисном обслуживании.</span><span class="sxs-lookup"><span data-stu-id="8cf0c-107">In the **Service agreement group** field, enter an ID for the service agreement group.</span></span>
4. <span data-ttu-id="8cf0c-108">В поле **Описание** введите описание группы соглашений о сервисном обслуживании.</span><span class="sxs-lookup"><span data-stu-id="8cf0c-108">In the **Description** field, enter a description for the service agreement group.</span></span>
5. <span data-ttu-id="8cf0c-109">Закройте страницу, чтобы сохранить изменения.</span><span class="sxs-lookup"><span data-stu-id="8cf0c-109">Close the page to save your changes.</span></span>

> [!NOTE]
> <span data-ttu-id="8cf0c-110">Если группа соглашений о сервисном обслуживании указана в поле **Группа соглашений о сервисном обслуживании** страницы **Параметры управления сервисным обслуживанием**, то программа предлагает эту группу для всех создаваемых новых соглашений о сервисном обслуживании.</span><span class="sxs-lookup"><span data-stu-id="8cf0c-110">If a service agreement group is defined in the **Service agreement group** field of the **Service management parameters** page, the program suggests this group for all new service agreements that you create.</span></span>
