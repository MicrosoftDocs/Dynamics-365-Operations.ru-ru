# <a name="create-service-agreements"></a>Создание соглашения на обслуживание

[!INCLUDE [banner](../includes/banner.md)]

В этом разделе описаны способы использования функций в модулях "Управление сервисным обслуживанием" и "Управление и учет по проектам" для создания соглашений о сервисном обслуживании.

## <a name="create-a-service-agreement-from-service-management"></a>Создание соглашения о сервисном обслуживании из модуля управления сервисным обслуживанием

1. Перейдите к модулю **Управление сервисным обслуживанием**.
2. Нажмите кнопку **Соглашения на обслуживание** для создания новой строки соглашения о сервисном обслуживании в заголовке страницы. 
3. Нажмите кнопку **Создать**. Введите описание, выберите ссылку на проект в поле **Код проекта** и заполните остальные поля и строки соглашения о сервисном обслуживании. Нажмите кнопку **Сохранить**.
4. Перейдите на вкладку **Отношения** и выберите **Объекты обслуживания** или **Задачи обслуживания** для создания связей объектов сервисного обслуживания или связей задач сервисного обслуживания для соглашения о сервисном обслуживании. Объекты и задачи обслуживания, для которых вы создали связи, можно приложить в строках соглашения на обслуживание.
5. В нижней половине страницы создайте строки соглашения на обслуживание посредством копирования строк из шаблона обслуживания или другого соглашения на обслуживание или вручную.

> [!NOTE]
> При копировании строк в соглашение на обслуживание из другого соглашения на обслуживание можно указать, хотите ли вы копировать связи объектов или задач обслуживания. При копировании этих связей они добавляются к существующим связям в соглашении на обслуживание. При копировании строк соглашения на обслуживание из шаблона обслуживания связи объектов и задач обслуживания автоматически копируются в качестве связей объектов и задач обслуживания в новые строки соглашения на обслуживание.

## <a name="create-service-agreement-lines-manually"></a>Создание строк соглашения на обслуживание вручную

1. На странице **Соглашения на обслуживание** добавьте строку соглашения на обслуживание в таблицу строк. 
2. Введите соответствующую информацию в строку соглашения на обслуживание. 
3. Нажмите сочетание клавиш **CTRL+S**, чтобы сохранить строку, а затем закройте страницу.

## <a name="create-a-service-agreement-from-project"></a>Создание соглашения на обслуживание из модуля Проект

1. Щелкните **Управление и учет по проектам**.
2. Щелкните **Все проекты**.
3. Выберите проект в списке.
4. В области **Панель операций** щелкните **Управлять**. В группе действий **Создать** щелкните **Сервисное обслуживание** и выберите **Соглашение о сервисном обслуживании**.
5. Следуйте шагам в разделе **Создание соглашения на обслуживание**, как описано ранее в этом разделе, чтобы ввести ссылку на проект.
