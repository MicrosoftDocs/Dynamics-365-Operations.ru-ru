# <a name="set-up-a-preferred-technician"></a>Настройка предпочтительного технического специалиста 

[!include [banner](../includes/banner.md)]


Можно выбрать любого работника в качестве предпочтительного специалиста для соглашения о сервисном обслуживании или заказа на сервисное обслуживание. Однако имеет смысл добавить работника к соответствующей группе исполнения, чтобы работник был включен в разделе **Панель исполнения**.

## <a name="assign-employee-to-a-dispatch-team"></a>Назначение сотрудника команде подготовки к отправке

1.  Щелкните **Управление персоналом** \> **Общий** \> **Работники** \> **Работники**. Дважды щелкните по имени работника, чтобы открыть страницу подробных сведений о работнике. В разделе **Панель операций** щелкните **Настройка** \>**Группа исполнения**, чтобы открыть форму **Подготовка работников к отправке**.

2.  В поле **Группа исполнения** выберите группу, для которой необходимо назначить работника.

## <a name="assign-a-preferred-technician-to-a-service-agreement"></a>Назначение предпочтительного специалиста к соглашению на обслуживание

1.  Щелкните **Управление сервисным обслуживанием** \> **Общее** \> **Соглашения на обслуживание** \> **Соглашения на обслуживание**. Дважды щелкните соглашение о сервисном обслуживании, чтобы открыть форму подробных сведений.

2.  На вкладке **Общие** выберите поле **Предпочитаемый техник** и выберите члена подходящей группы исполнения предпочтительного специалиста для соглашения о сервисном обслуживании.

## <a name="assign-a-preferred-technician-to-a-service-order"></a>Назначение предпочтительного специалиста к соглашению на обслуживание

1.  Щелкните **Управление сервисным обслуживанием** \> **Периодические операции** \> **Панель исполнения**.
    

    > [!NOTE]
    > <P>В форме <STRONG>Панель исполнения</STRONG> укажите диапазон дат для мероприятий по исполнению, которые необходимо просмотреть. Кроме того, укажите, необходимо ли отображать закрытые мероприятия и необходимо ли ограничить мероприятие по подготовке к отправке командами, к которым вы относитесь или которые вы уполномочены отслеживать. Нажмите <STRONG>OK</STRONG>, чтобы открыть диалоговое окно <STRONG>Панель исполнения</STRONG>.</P>



2.  Выберите строку действия по сервисному обслуживанию, которое необходимо изменить.

3.  На вкладке **Связанные** используйте список **Работник** для назначения члена подходящей группы исполнения в качестве предпочитаемого специалиста для сервисного вызова.
