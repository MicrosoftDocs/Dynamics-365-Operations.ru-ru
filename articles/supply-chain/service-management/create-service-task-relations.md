# <a name="create-service-task-relations"></a>Создание связей задач обслуживания    

[!include [banner](../includes/banner.md)]

Можно связывать задачи обслуживания с соглашениями о сервисном обслуживании или заказами на сервисное обслуживание для описания задачи сервисного обслуживания, которая должна быть выполнена для соглашения или заказа. Эта информация доступна техническим специалистам и клиентам.

## <a name="create-a-relation-with-a-service-agreement"></a>Создание связи с соглашением на обслуживание

1.  Перейдите **Управление сервисным обслуживанием** \> **Общее** \> **Соглашения на обслуживание** \> **Соглашения на обслуживание**.

2.  Выберите существующее соглашение или создайте новое.

3.  На панели операций выберите **Задачи сервисного обслуживания**.

4.  В форме **Задачи сервисного обслуживания** выберите **Создать**, чтобы создать новую строку, затем выберите сервисного обслуживания из списка из списка **Задача сервисного обслуживания**, чтобы присоединить задачу сервисного обслуживания к соглашению о сервисном обслуживании.

5.  На вкладке **Описание** введите в поле свободного текстового формата внутренние или внешние описания примечаний.

6.  Закройте форму, чтобы сохранить запись.

Повторяйте эту процедуру, пока не будут созданы все необходимые связи задач обслуживания для данного соглашения на обслуживание. Теперь эти задачи обслуживания можно определять в прикрепленных строках соглашения.

Связь с задачами обслуживания, создаваемая в соглашении на обслуживание, доступна из всех заказов на обслуживание, присоединенных к соглашению на обслуживание.

## <a name="create-a-relation-with-a-service-order"></a>Создание связи с заказом на обслуживание

1.  Перейдите **Управление сервисным обслуживанием** \> **Общий** \> **Заказы на сервисное обслуживание** \> **Заказы на сервисное обслуживание**.

2.  Выберите существующий заказ на обслуживание или создайте новый.

3.  На панели операций выберите **Задачи сервисного обслуживания**.

4.  В форме **Задачи сервисного обслуживания** выберите **Создать**, чтобы создать новую строку, затем выберите сервисного обслуживания из списка из списка **Задача сервисного обслуживания**, чтобы присоединить задачи сервисного обслуживания к заказу на сервисное обслуживание.

5.  На вкладке **Описание** введите в поле свободного текстового формата внутренние или внешние описания примечаний.

6.  Закройте форму, чтобы сохранить запись.

Повторяйте эту процедуру до тех пор, пока не будут созданы все необходимые связи задач обслуживания для данного заказа на обслуживание. Теперь при создании строк заказа на обслуживание можно выбирать задачу обслуживания, для которой создана связь.

Связи задач обслуживания, созданные для заказа на обслуживание, доступны для определенного заказа на обслуживание.
