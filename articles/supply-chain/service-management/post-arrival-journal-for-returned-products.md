# <a name="post-arrival-journal-for-returned-products"></a>Разноска журнала прихода возвращенных продуктов 

[!include [banner](../includes/banner.md)]


Чтобы обработать возврат, сначала проверьте количество возврата, обновите поле количества в журнале прибытия номенклатуры. Затем выберите код расстановки или укажите, что возвращенные номенклатуры должны быть проверены. После завершения этих шагов можно разнести журнал прибытия номенклатуры для этого заказа на возврат.

1.  Щелкните **Управление запасами** \> **Периодические операции** \> **Обзор прибытия**.

2.  В фильтре **Имя настройки** выберите **Заказ на возврат**.

3.  Если список приходов длинный, используйте поля в области **Диапазон**, чтобы сузить список.

4.  Найдите строку заказа на возврат, которую необходимо разнести, установите для нее флажок **Выбрать для прибытия**, а затем щелкните **Начать прибытие**.

5.  Щелкните **Журналы** \> **Показать прибытия по приходам**, чтобы открыть форму **Журнал местоположений**.
    

    > [!TIP]
    > <P>Для просмотра подобной информации выберите журнал, а затем щелкните <STRONG>Строки</STRONG>.</P>


6.  Выполните все необходимые обновления, а затем нажмите кнопку **Разнести**.

После разноски журнала возвращенные номенклатуры регистрируются в запасах, а форма **Заказы на возврат** показывает, что номенклатуры прибыли на склад.
