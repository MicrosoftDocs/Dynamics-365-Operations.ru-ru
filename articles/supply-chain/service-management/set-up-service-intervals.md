# <a name="set-up-service-intervals"></a>Настройка интервалов обслуживания  

[!include [banner](../includes/banner.md)]

Интервал обслуживания определяет частоту, с которой строки заказа на обслуживание создаются для строк соглашения на обслуживание при автоматическом создании заказов на обслуживание.

1. Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Соглашения на обслуживание** \> **Диапазоны сервисного обслуживания**.
2. Создайте новый интервал сервисного обслуживания.
3. Введите код и описание диапазона сервисного обслуживания.
4. В поле **Диапазон** выберите диапазон.
5. В поле **Частота** введите частоту. Частота — это множитель, на который необходимо умножить диапазон, чтобы получить интервал соглашения о сервисном обслуживании.
6. Нажмите комбинацию клавиш **Alt+S**, чтобы сохранить интервал сервисного обслуживания.

## <a name="example"></a>Пример

Предположим, необходимо создать 10-дневный интервал сервисного обслуживания.

**Создание 10-дневного интервала сервисного обслуживания**

1. Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Соглашения на обслуживание** \> **Диапазоны сервисного обслуживания**.
2. Создайте новый интервал сервисного обслуживания.
3. Введите код и описание диапазона сервисного обслуживания.
4. В поле **Диапазон** выберите **Ежедневно**.
5. В поле **Частота** введите значение 10.
6. Нажмите комбинацию клавиш **Alt+S**, чтобы сохранить интервал сервисного обслуживания.
