# <a name="set-up-service-intervals"></a><span data-ttu-id="e4652-103">Настройка интервалов обслуживания</span><span class="sxs-lookup"><span data-stu-id="e4652-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="e4652-104">Интервал обслуживания определяет частоту, с которой строки заказа на обслуживание создаются для строк соглашения на обслуживание при автоматическом создании заказов на обслуживание.</span><span class="sxs-lookup"><span data-stu-id="e4652-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="e4652-105">Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Соглашения на обслуживание** \> **Диапазоны сервисного обслуживания**.</span><span class="sxs-lookup"><span data-stu-id="e4652-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="e4652-106">Создайте новый интервал сервисного обслуживания.</span><span class="sxs-lookup"><span data-stu-id="e4652-106">Create a new service interval.</span></span>
3. <span data-ttu-id="e4652-107">Введите код и описание диапазона сервисного обслуживания.</span><span class="sxs-lookup"><span data-stu-id="e4652-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="e4652-108">В поле **Диапазон** выберите диапазон.</span><span class="sxs-lookup"><span data-stu-id="e4652-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="e4652-109">В поле **Частота** введите частоту.</span><span class="sxs-lookup"><span data-stu-id="e4652-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="e4652-110">Частота — это множитель, на который необходимо умножить диапазон, чтобы получить интервал соглашения о сервисном обслуживании.</span><span class="sxs-lookup"><span data-stu-id="e4652-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="e4652-111">Нажмите комбинацию клавиш **Alt+S**, чтобы сохранить интервал сервисного обслуживания.</span><span class="sxs-lookup"><span data-stu-id="e4652-111">Press **Alt+S** to save the service interval.</span></span>

<span data-ttu-id="e4652-113">Предположим, необходимо создать 10-дневный интервал сервисного обслуживания.</span><span class="sxs-lookup"><span data-stu-id="e4652-113">You want to create a service interval of 10 days.</span></span>

<span data-ttu-id="e4652-114">**Создание 10-дневного интервала сервисного обслуживания**</span><span class="sxs-lookup"><span data-stu-id="e4652-114">**Create a 10-day service interval**</span></span>

2. <span data-ttu-id="e4652-116">Создайте новый интервал сервисного обслуживания.</span><span class="sxs-lookup"><span data-stu-id="e4652-116">Create a new service interval.</span></span>
