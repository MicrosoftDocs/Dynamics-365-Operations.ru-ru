# <a name="set-up-dispatch-teams"></a>Настройка групп подготовки к оправке 

[!include [banner](../includes/banner.md)]


В этом разделе описывается создание группы исполнения. После создания группы исполнения можно назначить работников группе. После создания заказа на сервисное обслуживание можно связать члена группы исполнения с мероприятием в заказе на сервисное обслуживание.

1.  Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Группы исполнения**.

2.  В форме **Группы исполнения** щелкните **Создать**, чтобы создать новую группу.

3.  В поле **Группа исполнения** введите уникальное имя для группы исполнения.

4.  В поле **Владелец** выберите сотрудника, ответственного за эту группу.
    

    > [!NOTE]
    > <P>В этом поле можно выбрать только одного сотрудника. Выбрать поставщиков невозможно.</P>



5.  (Необязательно) В поле **Описание** введите краткое описание этой группы исполнения.


> [!NOTE]
> <P>Можно назначить работника этой группе исполнения в форме <STRONG>Работник</STRONG>. Дополнительные сведения о назначении работника группе исполнения см. в разделе <A href="https://technet.microsoft.com/library/dn776288(v=ax.60)">Подготовка работников к отправке (форма)</A>.</P>
