# <a name="service-subscriptions"></a>Подписки на сервисное обслуживание

[!include [banner](../includes/banner.md)]

Подписки основываются на проекте  «Время и расходы». Можно создавать подписки из проекта с помощью страницы **Подписки на сервисное обслуживание** или с помощью группы подписки на обслуживание.

Для каждой подписки можно создавать (по желанию) несколько платежей по подписке. Платежи по подписке представляют собой проводки, накладные по которым выставляются клиенту.

Код периода указывает продолжительность периода платежа по подписке или интервалы выставления счетов по подписке.

Код периода определяется в группе подписок на обслуживание. Он определяется для подписок с использованием группы подписок на обслуживание. Любой новый сбор по подписке, который вы создаете, содержит предлагаемую дату начала. Это дата начала периода, если это первый создаваемый период, или в конца предыдущего периода, если это второй или последующий период.
