# <a name="set-up-subscription-groups"></a>Настройка групп подписки 

[!include [banner](../includes/banner.md)]


1.  Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Подписки на сервисное обслуживание** \> **Группы подписки**.

2.  Нажмите сочетание клавиш Ctrl+N или щелкните **Создать**, чтобы создать новую группу подписки.

3.  В поле **Группа подписки** введите имя новой группы подписки.

4.  В поле **Выставление накладных** выберите частоту выставления накладных для подписки. Частота выставления накладных указана кодом периода, который настраивается в форме **Типы периодов**.

5.  Чтобы начислить выручку по проводкам подписки, установите флажок **Начислить выручку**, а затем в поле **Начисление** выберите частоту. Частота начисления основывается на коде периода, который настраивается в форме **Периоды**.
