# <a name="set-up-subscription-groups"></a><span data-ttu-id="66745-103">Настройка групп подписки</span><span class="sxs-lookup"><span data-stu-id="66745-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="66745-104">Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Подписки на сервисное обслуживание** \> **Группы подписки**.</span><span class="sxs-lookup"><span data-stu-id="66745-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="66745-105">Нажмите сочетание клавиш Ctrl+N или щелкните **Создать**, чтобы создать новую группу подписки.</span><span class="sxs-lookup"><span data-stu-id="66745-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="66745-106">В поле **Группа подписки** введите имя новой группы подписки.</span><span class="sxs-lookup"><span data-stu-id="66745-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="66745-107">В поле **Выставление накладных** выберите частоту выставления накладных для подписки.</span><span class="sxs-lookup"><span data-stu-id="66745-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="66745-108">Частота выставления накладных указана кодом периода, который настраивается в форме **Типы периодов**.</span><span class="sxs-lookup"><span data-stu-id="66745-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="66745-109">Чтобы начислить выручку по проводкам подписки, установите флажок **Начислить выручку**, а затем в поле **Начисление** выберите частоту.</span><span class="sxs-lookup"><span data-stu-id="66745-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="66745-110">Частота начисления основывается на коде периода, который настраивается в форме **Периоды**.</span><span class="sxs-lookup"><span data-stu-id="66745-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
