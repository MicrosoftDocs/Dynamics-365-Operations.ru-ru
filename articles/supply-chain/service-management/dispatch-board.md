#  <a name="dispatch-board"></a>Панель подготовки к отправке 

[!include [banner](../includes/banner.md)]

Панель исполнения — это центральное место, в котором можно просмотреть статус мероприятий в заказе на сервисное обслуживание. На панели исполнения можно выполнить следующие задачи:

  - Применить фильтр и просмотреть действия сервиса, попадающие в определенный диапазон дат.

  - Определить приоритет действия сервиса и отличить уровни приоритетов обслуживания на основе настраиваемой цветовой схемы.

  - Просмотреть работников, назначенных группе исполнения.

  - Изменить время обслуживания и назначенных техников для действия сервиса.

  - Просмотреть список действий сервиса, которые не были исполнены.

Дополнительные сведения о выполнении определенных задач на панели исполнения см. в подразделах раздела **См. также**.
