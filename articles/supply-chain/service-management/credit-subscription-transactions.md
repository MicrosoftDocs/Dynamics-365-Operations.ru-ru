[!include [banner](../includes/banner.md)]


## <a name="credit-subscription-transactions"></a><span data-ttu-id="a6ddc-104">Кредитование проводок по подписке</span><span class="sxs-lookup"><span data-stu-id="a6ddc-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="a6ddc-105">Щелкните **Управление сервисным обслуживанием** \> **Общий** \> **Подписки на сервисное обслуживание** \> **Все подписки на сервисное обслуживание**.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="a6ddc-106">Выберите подписку, присоединенную к проводке подписки, для которой необходимо создать кредит-ноту.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="a6ddc-107">Перейдите на вкладку **Анализировать** и нажмите кнопку **Проводки по сборам** на панели операций.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="a6ddc-108">В форме **Проводки по сборам** выберите проводку, для которой необходимо создать кредит-ноту.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="a6ddc-109">Щелкните **Функции** \> **Выбор для кредит-ноты**.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="a6ddc-110">В форме **Выбор для кредит-ноты** выберите проводку, которую необходимо кредитовать, и щелкните **OK**.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="a6ddc-111">При создании кредит-ноты обязательно выберите <STRONG>Кредит-ноты</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="a6ddc-112">Это доступно в списке <STRONG>Метод выставления накладной</STRONG> в диалоговом окне <STRONG>Создать накладную</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="a6ddc-113">Если значение поля **Реверсировать начисления на кредитование** в форме **Параметры управления сервисным обслуживанием** равно **Вручную**, необходимо отдельно реверсировать каждую проводку начисленного дохода, и только после этого создавать предложение кредит-ноты для проводки.</span><span class="sxs-lookup"><span data-stu-id="a6ddc-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
