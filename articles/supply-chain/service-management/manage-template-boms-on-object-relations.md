# <a name="manage-template-boms-on-object-relations"></a>Управление шаблонами спецификаций для связей объектов 

[!include [banner](../includes/banner.md)]


## <a name="attach-a-template-bom-to-a-service-object"></a>Присоединение шаблона спецификации к объекту обслуживания

1.  Щелкните **Управление сервисным обслуживанием** \> **Общее** \> **Соглашения на обслуживание** \> **Соглашения на обслуживание**.

2.  Дважды щелкните соглашение о сервисном обслуживание, в котором необходимо вложить спецификацию шаблона в связь объекта.

3.  Щелкните **Настройка** \> **Объекты сервисного обслуживания**.

4.  В форме **Объекты сервисного обслуживания** выберите объект, в который требуется вложить спецификацию, и щелкните **Функции** \> **Присоединение шаблона спецификации**.

5.  В диалоговом окне **Выбор шаблона спецификации** выберите шаблон спецификации, а затем нажмите кнопку **OK**.

6.  Закройте формы, чтобы сохранить изменения.

## <a name="delete-a-service-bom-from-a-service-object"></a>Удаление шаблона спецификации из объекта обслуживания

1.  Щелкните **Управление сервисным обслуживанием** \> **Общее** \> **Соглашения на обслуживание** \> **Соглашения на обслуживание**.

2.  Дважды щелкните соглашение о сервисном обслуживании, в котором необходимо удалить спецификацию обслуживания из связи объекта.

3.  Щелкните **Настройка** \> **Объекты сервисного обслуживания**.

4.  В форме **Объекты сервисного обслуживания** выберите объект, содержащий спецификацию обслуживания, которую необходимо удалить, и щелкните **Функции** \> **Удалить спецификацию обслуживания**.

5.  Закройте формы, чтобы сохранить изменения.

## <a name="move-the-service-bom-history-from-one-service-agreement-to-another"></a>Перемещение истории спецификации обслуживания из одного соглашения на обслуживание в другое

1.  Щелкните **Управление сервисным обслуживанием** \> **Общее** \> **Соглашения на обслуживание** \> **Соглашения на обслуживание**.

2.  Дважды щелкните соглашение о сервисном обслуживание, из которого необходимо переместить спецификацию обслуживания.

3.  Щелкните **Настройка** \> **Объекты сервисного обслуживания**.

4.  В форме **Объекты сервисного обслуживания** выберите объект, содержащий спецификацию обслуживания, которую необходимо переместить, и щелкните **Функции** \> **Переместить спецификацию обслуживания**.

5.  В форме **Выбор связи объекта обслуживания** выберите связь объекта сервисного обслуживания, в которую необходимо переместить спецификацию обслуживания, и щелкните **OK**.

6.  Закройте формы, чтобы сохранить изменения.

## <a name="modify-the-information-displayed-for-a-bom-line"></a>Изменение сведений, которые отображаются в строке спецификации

1.  Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Объекты обслуживания** \> **Шаблоны спецификаций**.

2.  Выберите шаблон спецификации и щелкните **Конструктор**.

3.  Перейдите на вкладку **Настройка**. В области **Спецификация** установите флажки для сведений, которые должны отображаться в строках спецификации для спецификаций обслуживания и спецификаций шаблонов.

4.  Закройте формы, чтобы сохранить изменения.
