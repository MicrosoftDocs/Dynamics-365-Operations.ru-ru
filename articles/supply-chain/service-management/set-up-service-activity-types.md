# <a name="set-up-service-activity-types"></a>Настройка типов действий сервиса 

[!include [banner](../includes/banner.md)]


1.  Щелкните **Продажи и маркетинг** \> **Настройка** \> **Мероприятия** \> **Типы мероприятий**.
    
    – или –
    
    Щелкните **Управление организацией** \> **Настройка** \> **Мероприятия** \> **Типы мероприятий**.

2.  В форме **Типы мероприятий** нажмите сочетание клавиш CTRL + N, чтобы добавить новую строку типа мероприятия.

3.  В поле **Тип** введите имя для типа мероприятия.

4.  В поле **Описание** введите текст описания этого типа мероприятия.

5.  Закройте форму, чтобы сохранить изменения.
