# <a name="create-service-objects"></a>Создание объектов обслуживания  

[!include [banner](../includes/banner.md)]
   
Объекты сервисного обслуживания — это основные средства и продукты клиента, сервисное обслуживание которых вы можете выполнить. 

1. Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Объекты обслуживания** \> **Объекты обслуживания**.
2. Нажмите кнопку **Создать** на панели инструментов.
3. В поле **Описание** введите описание объекта.
4. В поле **Группа объектов сервисного обслуживания** выберите группу, к которой относится объект. 
5. (Необязательно) Выберите номенклатуру для строки объекта Услуга или шаблон спецификации.
