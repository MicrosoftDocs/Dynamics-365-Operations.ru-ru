# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="13384-103">Настройка шаблона для представления объектов у клиента</span><span class="sxs-lookup"><span data-stu-id="13384-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="13384-104">Выполните эту процедуру для настройки шаблона, представляющего объекты на сайте клиента.</span><span class="sxs-lookup"><span data-stu-id="13384-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="13384-105">Щелкните **Управление сервисным обслуживанием** \> **Общее** \> **Соглашения на обслуживание** \> **Соглашения на обслуживание**.</span><span class="sxs-lookup"><span data-stu-id="13384-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="13384-106">Дважды щелкните соглашение о сервисном обслуживание или нажмите **Соглашение о сервисном обслуживании**, чтобы создать новое соглашение о сервисном обслуживании.</span><span class="sxs-lookup"><span data-stu-id="13384-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="13384-107">Перейдите на вкладку **Настройка**.</span><span class="sxs-lookup"><span data-stu-id="13384-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="13384-108">Щелкните **Объекты сервисного обслуживания**, чтобы прикрепить шаблон спецификации к соглашению о сервисном обслуживании.</span><span class="sxs-lookup"><span data-stu-id="13384-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="13384-109">В форме **Объекты сервисного обслуживания** выберите объект сервисного обслуживания и щелкните **Функции** \> **Присоединение шаблона спецификации**.</span><span class="sxs-lookup"><span data-stu-id="13384-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="13384-110">В форме **Выбор шаблона спецификации** выберите шаблон спецификации в поле **Шаблон спецификации** и нажмите кнопку **OK**.</span><span class="sxs-lookup"><span data-stu-id="13384-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="13384-111">В форме **Объекты сервисного обслуживания** щелкните **Конструктор**, чтобы изменить шаблон спецификации так, чтобы в спецификации были представлены нужные объекты.</span><span class="sxs-lookup"><span data-stu-id="13384-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="13384-112">Закройте формы, чтобы сохранить изменения.</span><span class="sxs-lookup"><span data-stu-id="13384-112">Close the forms to save your changes.</span></span>
