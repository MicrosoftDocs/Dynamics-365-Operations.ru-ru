# <a name="perform-invoice-updates-for-returns"></a>Выполнение обновлений накладной для возвратов 

[!include [banner](../includes/banner.md)]


Заказ на возврат является особым типом заказа на продажу, имеющим маркировку "Возврат". Поэтому, чтобы создать накладные для заказа на возврат, страница списка **Все заказы на продажу** используется вместо формы **Заказы на возврат**. Эта функция также поддерживает бизнес-процессы компаний, выбравших процедуру одновременного выставления накладных по заказам на возврат и заказам на продажу одним и тем же сотрудником.

Так как накладная по возвращенным номенклатурам используется для отрицательных сумм, она называется кредит-нотой.

Если была настроена пакетная обработка обновления накладной, заказ на продажу, относящийся к типу **Возвращенный заказ**, должен иметь статус строки возврата **Получено**, указывающий на то, что отборочная накладная этого заказа была обновлена.

## <a name="post-an-invoice-for-a-return-order"></a>Разноска накладной для заказа на возврат

1.  Щелкните **Расчеты с клиентами** \> **Общий** \> **Заказы на продажу** \> **Все заказы на продажу**.

2.  Выберите заказ на продажу, для которого **Возвращенный заказ** отображается в поле **Тип заказа**.

3.  В элементе управления "Панель операций" на вкладке **Накладная** в группе **Создать** щелкните **Накладная**.

4.  На вкладке **Параметры** установите флажок **Разноска**.

5.  Проверьте сведения в форме и внесите все необходимые изменения.

6.  Нажмите кнопку **ОК**. Кредит-нота разнесена.
