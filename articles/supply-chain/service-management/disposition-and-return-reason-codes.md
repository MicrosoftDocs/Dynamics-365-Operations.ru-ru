# <a name="customer-returns-overview"></a>Обзор возвратов клиентами

[!include [banner](../includes/banner.md)]


Создайте и используйте коды причин возврата и коды метода обработки в системе для поддержки процесса возврата продукции.

Используйте код причины возврата для описания причины, по которой клиент хочет вернуть номенклатуру. Можно назначить код причины в форме **Создание заказов на возврат**.

Назначьте код расстановки, когда номенклатура получается или при физическом осмотре возвращенной номенклатуры. Можно использовать коды расстановки для описания состояния номенклатуры. Можно также использовать коды расстановки, чтобы указать, требуется ли дополнительных действий для проводки. Например, создайте коды расстановки для следующих действий:

  - Утилизация в отходы возвращенной номенклатуры и предоставление сменной номенклатуры клиенту.

  - Возврат номенклатуры на склад и кредитование клиенту за стоимость номенклатуры.

  - Ремонт номенклатуры и возврат ее клиенту.
