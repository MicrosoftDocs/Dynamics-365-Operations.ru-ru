# <a name="accrue-subscription-revenue"></a>Начислить выручку по подписке 

[!include [banner](../includes/banner.md)]


1.  Щелкните **Управление сервисным обслуживанием** \> **Периодические операции** \> **Подписки на сервисное обслуживание** \> **Начисление выручки по подписке**.

2.  В поле **Код периода** выберите код периода. Доход разносится по всем проводкам по сборам по подписке с выбранным кодом периода.

3.  В поле **Дата разноски** введите дату разноски и нажмите кнопку **OK**.


> [!NOTE]
> <P>Накладные для проводок по сборам должны быть выставлены до разноски начисленного дохода.<P>
