<span data-ttu-id="36ad9-110">Эта процедура позволяет настраивать коды причин, которые можно использовать для указания причин возврата продукта клиентом.</span><span class="sxs-lookup"><span data-stu-id="36ad9-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>
