# <a name="set-up-return-reason-codes"></a>Настройка кодов причин возврата   

[!include [banner](../includes/banner.md)]


## <a name="set-up-return-reason-code-groups"></a>Настройка групп кодов причин возврата 

1.  Перейдите **Продажи и маркетинг** \> **Настройка** \> **Заказы на продажу** \> **Возвраты** \> **Группы кодов причин возврата**.

2.  Выберите **Создать**, чтобы создать новую группу кодов причин возврата.

3.  В поле **Группа кодов причин возврата** введите идентификатор этой группы.

4.  В поле **Описание** введите краткую фразу, объясняющую, как эта группа будет использоваться.

## <a name="set-up-return-reason-codes"></a>Настройка кодов причин возврата 

Эта процедура позволяет настраивать коды причин, которые можно использовать для указания причин возврата продукта клиентом.

1.  Перейдите **Продажи и маркетинг** \> **Настройка** \> **Заказы на продажу** \> **Возвраты** \> **Коды причин возврата**.

2.  Нажмите CTRL+N, чтобы создать новую строку кода причины возврата.

3.  В поле **Код причины возврата** введите текст для определения данного кода.

4.  В поле **Описание** введите текст описания данного кода причины возврата.

5.  В раскрывающемся списке **Группа кодов причин возврата** добавьте код в группу кодов причины возврата.
