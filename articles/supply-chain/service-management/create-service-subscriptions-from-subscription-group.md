# <a name="create-service-subscriptions-from-a-subscription-group"></a>Создание подписок на обслуживание из группы подписки 

[!include [banner](../includes/banner.md)]


1.  Щелкните **Управление сервисным обслуживанием** \> **Настройка** \> **Подписки на сервисное обслуживание** \> **Группы подписки**.

2.  Выбор группы подписок.

3.  Нажмите кнопку **Подписки на сервисное обслуживание**.

4.  Нажмите комбинацию клавиш Ctrl+N, чтобы создать новую подписку. В поле группы подписки отображаются данные из группы подписок в форме **Подписка**.

5.  Введите сведения в оставшиеся поля.

6.  Закройте форму, чтобы сохранить изменения.
