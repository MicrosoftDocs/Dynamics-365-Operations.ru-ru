# <a name="reverse-changes-in-a-dispatch-board"></a><span data-ttu-id="41fe1-103">Реверсирование изменений на панели исполнения</span><span class="sxs-lookup"><span data-stu-id="41fe1-103">Reverse changes in a dispatch board</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="41fe1-104">В этом разделе описывается, как реверсировать несохраненные изменения, вносимые на панели исполнения.</span><span class="sxs-lookup"><span data-stu-id="41fe1-104">This topic describes how to reverse unsaved modifications that you make in a dispatch board.</span></span> <span data-ttu-id="41fe1-105">Например, можно назначить работника действию сервиса, сохранить информацию, а позже решить назначить другого работника этому действию сервиса.</span><span class="sxs-lookup"><span data-stu-id="41fe1-105">For example, you assign a worker to a service activity, save the information, and then later decide to assign a different worker to the service activity.</span></span> <span data-ttu-id="41fe1-106">Вы изменили работника на панели исполнения, но перед сохранением изменения узнали, что этот работник недоступен.</span><span class="sxs-lookup"><span data-stu-id="41fe1-106">You modify the worker in the dispatch board, and then, before saving the change, learn that the worker just assigned is not available.</span></span> <span data-ttu-id="41fe1-107">Можно реверсировать несохраненные изменения, чтобы переназначить исходного работника заказу на сервисное обслуживание.</span><span class="sxs-lookup"><span data-stu-id="41fe1-107">You can reverse the unsaved modification so that the original worker is reassigned to the service order.</span></span>

<span data-ttu-id="41fe1-108">Выполните следующие действия, чтобы реверсировать несохраненные изменения на панели исполнения.</span><span class="sxs-lookup"><span data-stu-id="41fe1-108">Use the following steps to reverse unsaved changes in a dispatch board:</span></span>

1.  <span data-ttu-id="41fe1-109">Щелкните **Управление сервисным обслуживанием** \> **Периодические операции** \> **Панель исполнения**.</span><span class="sxs-lookup"><span data-stu-id="41fe1-109">Click **Service management** \> **Periodic** \> **Dispatch board**.</span></span>

2.  <span data-ttu-id="41fe1-110">В форме **Панель исполнения** введите соответствующую информацию в поля и нажмите кнопку **OK**.</span><span class="sxs-lookup"><span data-stu-id="41fe1-110">In the **Dispatch board** form, enter relevant information in the fields, and then click **OK**.</span></span> 

3.  <span data-ttu-id="41fe1-111">Чтобы реверсировать последнее несохраненное изменение, щелкните **Отменить**.</span><span class="sxs-lookup"><span data-stu-id="41fe1-111">To reverse the most recent change that is not saved, click **Undo**.</span></span>

4.  <span data-ttu-id="41fe1-112">Чтобы реверсировать серию изменений, которые еще не были сохранены, продолжайте нажимать кнопку **Отменить** до тех пор, пока не будут реверсированы по одному все изменения, которые необходимо реверсировать.</span><span class="sxs-lookup"><span data-stu-id="41fe1-112">To reverse a series of changes that are not saved, continue clicking **Undo** until each change that you want to discard is reversed.</span></span>
