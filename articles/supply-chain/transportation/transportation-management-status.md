# <a name="transportation-management-statuses"></a>Статусы управления транспортировкой

[!include [banner](../includes/banner.md)]

Настройте коды шаблонов для статусов транспортировки, чтобы интерпретировать коды, предоставленные вашими перевозчиками. Это позволяет интегрироваться с перевозчиками для предоставления статуса. С помощью статуса транспортировки, введенного для кода статуса шаблона транспортировки, можно отслеживать статус загрузки, отгрузки или контейнера. Конкретный статус транспортировки для загрузки, отгрузки или контейнера может быть обновлен только с помощью интеграции данных, а не вручную через интерфейс пользователя.

## <a name="create-a-transportation-status"></a>Создание статуса транспортировки

Чтобы создать статус транспортировки, выполните следующие шаги:

1. Перейдите в раздел **Управление транспортировкой \> Настройка \> Шаблоны статуса транспортировки**.
1. Выберите **Создать**, чтобы создать шаблон статуса транспортировки.
1. В поле **Шаблон статуса транспортировки** введите уникальный код для статуса транспортировки.
1. В поле **Тип транспортировки** выберите *Перевозчик* или *Центр* в качестве типа транспортировки.
1. Введите имя и статус транспортировки.
1. Закройте страницу.

## <a name="map-a-transportation-status-to-a-carrier-status"></a>Сопоставление статуса транспортировки статусу перевозчика

Для сопоставления статуса транспортировки статусу перевозчика выполните следующие шаги:

1. Перейдите в раздел **Управление транспортировкой \> Настройка \> Перевозчики \> Статус транспортировки перевозчика**.
1. Выберите **Создать**, чтобы сопоставить код перевозчика с кодом шаблона статуса транспортировки.
1. Выберите уникальный код для перевозчика и услуги перевозчика.
1. Выберите код статуса транспортировки, который требуется сопоставить с кодом выбранного перевозчика.
1. Введите внешний код, используемый перевозчиком.
1. Закройте страницу.
