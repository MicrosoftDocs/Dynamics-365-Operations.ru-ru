# <a name="rating-profiles"></a>Профили оценки

Профиль оценки напоминает договор логистики (но не юридический договор). Он используется для определения транспортных тарифов для загрузок. 

Каждый профиль оценки уникален для каждого перевозчика. В профиле вы связываете перевозчика с шаблоном ставки. Шаблон ставки определяет назначение базы ставки и базу ставки. База ставки определяет ставку перевозчика.

Профиль оценки можно настроить с помощью общей страницы, на которой представлен обзор всех существующих профилей оценки. Можно также настроить профиль оценки непосредственно из перевозчика. В обоих случаях сведения, настроенные для профиля оценки, одинаковы.

## <a name="create-or-edit-a-rating-profile-on-the-rating-profiles-page"></a>Создание или редактирование профиля оценки на странице профили оценки

На странице **Профили оценки** можно просмотреть все доступные профили оценки. Можно также редактировать существующие профили и создавать новые профили.

1. Перейдите в раздел **Управление транспортировкой \> Настройка \> Расчет ставок \> Профиль оценки**.
1. В области действий выберите **Создать**, чтобы добавить новый профиль оценки в сетку, или выберите **Правка**, чтобы изменить существующий профиль.
1. В строке для нового или существующего профиля оценки задайте следующие поля:

    - **Профиль оценки** — введите уникальный идентификатор для профиля оценки.
    - **Имя** — введите описательное имя для профиля оценки.
    - **Перевозчик** — выберите перевозчика. Профиль оценки, который вы настраиваете, также будет отображаться на странице **Перевозчики** для выбранного перевозчика.
    - **Сайт** и **Склад** — выберите сайт и склад.
    - **Механизм ставок** — выберите механизм ставок для профиля оценки.
    - **Шаблон ставки** — выберите шаблон ставок для профиля оценки. Шаблона ставки можно использовать для определения типа базы ставки и базы ставки. Дополнительные сведения см. в разделе [Настройка шаблонов ставок](set-up-rate-masters.md).
    - **Механизм расчета транзитного времени** — выберите механизм расчета транзитного времени для профиля оценки.
    - **Индекс топлива перевозчика** — выберите индекс топлива перевозчика для профиля оценки.
    - **Фактические дата и время начала** и **Фактические дата и время окончания** — определите период, в течение которого профиль оценки должен быть активным.

1. На панели операций выберите **Сохранить**.

## <a name="create-a-rating-profile-directly-on-the-shipping-carriers-page"></a>Создание профиля оценки непосредственно на странице перевозчиков

1. Перейдите в раздел **Управление транспортировкой \> Настройка \> Перевозчики \> Перевозчики, осуществляющий доставку**.
1. Выберите перевозчика в списке.
1. На экспресс-вкладке **Профили оценки** выберите **Создать**, чтобы создать профиль оценки.
1. Задайте поля для нового профиля оценки. Эти поля соответствуют полям на странице **Профили оценки**, как это описано в предыдущем разделе данной темы.

> [!NOTE]
> Профили, созданные на странице **Перевозчики**, также отображаются на странице **Профили оценки**.
