# <a name="set-up-accessorial-assignments"></a>Настройка дополнительных назначений

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как настроить назначение дополнения. Обычно это делает координатор транспортировки. Перед использованием этого руководства необходимо выполнить руководство "Настройка дополнительных расходов и справочников дополнений для узла".


## <a name="set-up-accessorial-assignment"></a>Настройка дополнительного назначения
1. Перейдите в раздел "Управление транспортировкой" > "Настройка" > "Расчет ставок" > "Дополнительные назначения".
2. Щелкните "Создать".
3. В поле "Имя" введите значение.
4. Переключите развертывание раздела "Подробности".
5. В поле "Узел" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. Выберите в списке узел, который вы создали в качестве при выполнении руководства "Настройка дополнительных расходов и справочников дополнений для узла". 
7. В поле "Код дополнительного узла" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
8. В списке перейдите по ссылке в выбранной строке.
9. Переключите развертывание раздела "Критерии".
    * В разделе "Критерии" можно выбрать точные критерии того, когда должен применяться сбор, в зависимости от различных предложенных здесь значений.  
10. Установите параметр "Применять всегда" в значение "Да".
11. В поле "Уровень дополнительной встречи" выберите один из вариантов.
12. Переключите развертывание раздела "Расчет".
13. В поле "Тип дополнительного сбора" выберите "Фиксированная".
    * Тип дополнительного сбора определяет, как рассчитывается фактическая сумма сбора. В данном примере это фиксированный сбор.  
14. В поле "Дополнительный сбор" введите число.
15. Нажмите кнопку "Сохранить".
