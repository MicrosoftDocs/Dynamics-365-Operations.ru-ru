# <a name="set-up-transportation-constraints-for-an-item"></a>Настройка ограничений по транспортировке для номенклатуры

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как настроить ограничение по транспортировке для предотвращения транспортировки выбранной номенклатуры через выбранный узел. Обычно эту задачу выполняет координатор транспортировки. Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USMF или собственные данные.


## <a name="create-an-item-constaint"></a>Создание ограничения номенклатуры
1. Перейдите в раздел "Ограничения".
2. Щелкните "Создать".
3. В поле "Ограничение номенклатуры" введите значение.
4. В поле "Имя" введите значение.
5. В поле "Сайт" введите или выберите значение.
6. В поле "Склад" введите или выберите значение.
7. В поле "Код номенклатуры" введите или выберите значение.
8. В поле "Узел" введите или выберите значение.
9. В поле "Действие ограничения" выберите параметр.
10. Нажмите кнопку "Сохранить".
11. Закройте страницу.
