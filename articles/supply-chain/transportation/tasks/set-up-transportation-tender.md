# <a name="set-up-a-transportation-tender"></a>Настройка способа доставки транспортировки

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как настроить способ доставки для транспортировки. Обычно это делает координатор транспортировки. Эту процедуру можно выполнить, используя компанию с демонстрационными данными USMF.


## <a name="select-a-route"></a>Выберите маршрут
1. Перейдите в раздел "Управление транспортировкой" > "Планирование" > "Рабочее место планирования загрузки".
2. Снимите флажок "Скрыть отгруженные и полученные".
3. Выберите строку с кодом груза 00006.
4. Щелкните "Оценка и маршрутизация".
5. Щелкните "Маршруты".

## <a name="create-the-transportation-tender"></a>Создание способа доставки транспортировки
1. Щелкните "Способ доставки транспортировки".
2. Щелкните "Создать".
3. Разверните раздел "Общие".
4. В поле "Запрошенные ставки" введите число.
5. Нажмите кнопку "Сохранить".
6. Щелкните "Изменить статус".
7. Щелкните Отправить.
8. Выберите маршрут.
