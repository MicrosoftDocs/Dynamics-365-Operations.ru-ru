# <a name="carrier-groups"></a>Группы перевозчиков

[!include [banner](../../includes/banner.md)]

Группа перевозчиков — это совокупность перевозчиков и услуг перевозчиков. Каждая группа перевозчиков определяет предпочтительную последовательность для перевозчиков и услуг перевозчиков, которые относятся к этой группе.

Когда для одного сегмента маршрута существует несколько перевозчиков и услуг перевозчиков, можно указать группу перевозчиков вместо конкретного перевозчика и услуги перевозчика в плане маршрута или в руководстве по маршруту.

## <a name="create-a-carrier-group"></a>Создание группы перевозчиков

1. Перейдите в раздел **Управление транспортировкой &gt; Настройка &gt; Перевозчики &gt; Группа перевозчиков**.
1. Выберите **Создать**.
1. В поле **Группа перевозчиков** введите уникальный код (ИД) для группы.
1. В поле **Имя** введите описательное имя для группы.
1. На экспресс-вкладке **Сведения** добавьте строку и выберите перевозчика и услугу перевозчика для него. Повторяйте этот шаг до тех пор, пока не будет добавлено столько перевозчиков, сколько требуется для группы.
1. Закройте страницу.
