# <a name="inventory-aging-report-discrepancies"></a>Несоответствия отчета о распределении запасов по срокам

## <a name="symptoms"></a>Симптомы

В **отчете о распределении запасов по срокам** отображаются различные значения при просмотре в разной аналитике хранения (например, на сайте или складе).

## <a name="resolution"></a>Решение

Дополнительные сведения о логике отчетности см. в разделе [Примеры и логика отчета о распределении запасов по срокам](/dynamics365/supply-chain/cost-management/inventory-aging-report.md).
