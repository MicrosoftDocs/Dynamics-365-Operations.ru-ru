# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Невозможно отследить серийные номера готовых товаров относительно потребленных товаров

## <a name="symptoms"></a>Симптомы

Вы не можете отслеживать серийные номера готовых изделий по серийным номерам материалов, которые потребляются производственным заказом, чтобы сделать эти готовые товары.

## <a name="workaround"></a>Обходное решение

Для решения проблемы создайте производственные заказы как количество, равное 1.
