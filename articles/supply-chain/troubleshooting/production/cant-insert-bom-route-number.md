# <a name="cant-insert-bill-of-material-and-route-when-creating-a-new-production-order"></a>Невозможно вставить спецификацию и маршрут при создании нового производственного заказа

## <a name="symptoms"></a>Симптомы

При создании нового производственного заказа после ввода номера номенклатуры поля **Сайт** и **Склад** автоматически устанавливаются в сайт и склад по умолчанию, которые определены на странице **Параметры заказов по умолчанию** для готовой номенклатуры товаров. Кроме того, автоматически вводится активный номер спецификации и номер маршрута, поэтому не выводится следующее сообщение, предлагающее ввести эти значения:

> Вставить активную версию спецификации и маршрута?

## <a name="resolution"></a>Решение

Вам не предлагается вставить номера спецификаций и маршрутов, если выбран продукт, для которого объект и склад уже на странице **Параметры заказа по умолчанию**. Запрос выводится только в том случае, если эти значения не определены для выбранного продукта.
