# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Не удается завершить производственный заказ из-за ошибки значения спецификации consumptionCost

## <a name="symptoms"></a>Симптомы

При попытке завершить производственный заказ выводится следующее сообщение об ошибке:

> Расчет значения спецификации consumptionCost при выпуске со склада должно быть отрицательным.

## <a name="resolution"></a>Решение

Эта проблема была исправлена в версии 10.0.15.
