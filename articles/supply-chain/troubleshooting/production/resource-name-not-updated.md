# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>После изменения имени работника в адресной книге имя ресурса не обновляется

## <a name="symptoms"></a>Симптомы

Если изменить имя работника в глобальной адресной книге, имя соответствующего ресурса не обновляется с шаблоне группы ресурсов.

## <a name="resolution"></a>Решение

В настоящее время соответствующее имя ресурса не обновляется автоматически. Чтобы устранить эту проблему, необходимо вручную обновить имя ресурса.
