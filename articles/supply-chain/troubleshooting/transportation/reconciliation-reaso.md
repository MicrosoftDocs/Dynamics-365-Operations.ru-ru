# <a name="you-can-add-only-the-main-account-as-the-credit-account-for-reconciliation-reasons"></a>В целях выверки можно добавить только счет ГК в качестве кредитного счета

Номер статьи базы знаний: 4603538

## <a name="symptoms"></a>Симптомы

При настройке причины выверки в модуле управления транспортировкой в качестве кредитного счета можно добавить только счет ГК. Нельзя добавить место возникновения затрат или любую другую аналитику в качестве кредитового счета. Однако дебетовый счет позволяет выбрать другие аналитики.

## <a name="resolution"></a>Приказ

Если выверка не выполнена для оплаты поставщику, но для кредитования определенного счета ГК, система не позволяет выбрать финансовую аналитику для кредитового счета при настройке причины выверки.

Если для структуры счета требуется конкретное значение финансовой аналитики для кредитного счета ГК, журнал итогового поставщика нельзя разнести автоматически, поскольку значение финансовой аналитики отсутствует. Таким образом, необходимо сначала указать счет кредита вручную, используя страницу **Журнал накладных**.

Поскольку значение аналитики требуется для счета кредита, разноска журнала накладных поставщика автоматически невозможна. Необходимо вручную разнести его после добавления значения аналитики к счету ГК для строки кредита.
