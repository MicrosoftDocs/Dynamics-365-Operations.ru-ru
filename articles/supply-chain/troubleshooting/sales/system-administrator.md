# <a name="system-administrators-cant-clear-order-holds-because-they-arent-authorized"></a>Системные администраторы не могут отменить удержания заказов, поскольку они не авторизованы

Номер статьи базы знаний: 4614642

## <a name="symptoms"></a>Симптомы

Системные администраторы не могут очищать удержания заказов на продажу, если только у них нет особой роли, назначенной в коде удержания заказа.

## <a name="resolution"></a>Приказ

Доступ к некоторым операциям, таким как удержания клирингового заказа на продажу, управляется настройкой бизнес-политики. Обычно системным администраторам не разрешается выполнять операции такого типа. 

Чаще доступ к выполнению определенной задачи осуществляется с помощью бизнес-политик, и только отдельные лица в организации утверждаются для выполнения этой задачи. Примерами могут служить утверждения, являющиеся результатом утверждения рабочего процесса, и отдельные задачи, которые являются результатом настройки рабочего процесса.

Хотя ни один из рабочих процессов не связан с удержанием заказов, принцип аналогичен. Соответствующая роль определяет конкретную группу лиц в организации, имеющих право очищать удержания заказов. Это право не обязательно предоставлять всем администраторам, поскольку этот подход нарушает определенную бизнес-политику.
