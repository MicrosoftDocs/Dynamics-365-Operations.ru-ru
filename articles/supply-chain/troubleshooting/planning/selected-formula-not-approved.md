# <a name="selected-formula-number-isnt-approved-for-a-batch-order"></a>Выбранный номер формулы не утверждается для заказа партии

Код ошибки: PRO2614

## <a name="symptoms"></a>Симптомы

При попытке утвердить спланированный заказ выводится следующее сообщение об ошибке:

> Выбранный номер формулы не утвержден для партионного заказа.

## <a name="cause"></a>Причина

Система проверяет операцию утверждения, чтобы убедиться, что утвержденная формула доступна для активной номенклатуры. Вероятно, необходимо утвердить формулу.

## <a name="resolution"></a>Решение

Чтобы утвердить формулу, выполните следующие действия.

1. Перейдите в раздел **Управление сведениями о продукте \> Спецификации и формулы \> Формулы**.
1. Выберите соответствующую формулу.
1. В области действий на вкладке **Формула** в группе **Поддержка** выберите **Утвердить формулу**.
1. В диалоговом окне **Утверждение спецификации или формулы** выберите утверждающего и затем нажмите **ОК**.
