# <a name="vendor-isnt-specified-when-planned-orders-are-firmed"></a>При утверждении спланированных заказов не указан поставщик

Код ошибки: SYS23532

## <a name="symptoms"></a>Симптомы

При попытке утвердить спланированные заказы выводится следующее сообщение об ошибке:

> Поставщик не указан

## <a name="resolution"></a>Решение

Для указания поставщика выполните следующие действия.

1. Откройте спланированный заказ, в котором отсутствует поставщик.
1. На экспресс-вкладке **Запланированные поставки** в поле **Поставщик** выберите поставщика.
