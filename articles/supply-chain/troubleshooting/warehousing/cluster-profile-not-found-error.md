# <a name="cluster-profile-cant-be-found"></a>Не удалось найти профиль кластера

## <a name="symptoms"></a>Симптомы

При работе с входящими складскими операциями можно получить следующее сообщение об ошибке:

> "Заказ контроля качества %1 создан. Не удалось найти профиль кластера. Проверьте конфигурацию."

Это сообщение об ошибке относится к процессу приемки, когда включено управление качеством (QMS). В зависимости от конфигураций в вашей среде, для устранения проблемы могут помочь дополнительные сведения о проводке, которая создает это сообщение об ошибке.

## <a name="resolution"></a>Решение

Сначала проверьте настройку комплектации кластера и убедитесь, что профили кластеров настроены правильно. Нельзя использовать пункт меню мобильного устройства для комплектации кластера, если не настроены профили кластера. В зависимости от среды, возможно, придется также просмотреть и другие связанные конфигурации.
