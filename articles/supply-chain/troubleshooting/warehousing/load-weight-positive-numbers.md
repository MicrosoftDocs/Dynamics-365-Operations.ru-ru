# <a name="load-weight-error-and-update-canceled-when-processing-work-between-locations"></a>Ошибка веса загрузки и обновление отменено при обработке работы между местоположениями

## <a name="symptoms"></a>Симптомы

Это сообщение об ошибке выводится в том случае, если имеется открытая работа при обработке работы из местоположения упаковки в промежуточные местоположения или из промежуточных местоположений в местоположения докинга:

> Поле 'Вес загрузки'(=-%1) может содержать только положительные числа. Обновление отменено.

## <a name="resolution"></a>Решение

Чтобы устранить эту проблему, перейдите на страницу **Администрирование системы \> Периодические задачи \> База данных \> Проверка целостности** и запустите процесс **Проверка целостности веса загрузки склада**.
