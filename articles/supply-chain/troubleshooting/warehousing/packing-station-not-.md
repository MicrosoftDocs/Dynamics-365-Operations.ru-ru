# <a name="packing-station-doesnt-show-product-notes"></a>Станция упаковки не показывает примечания к продуктам

Номер статьи базы знаний: 4614615

## <a name="symptoms"></a>Симптомы

Примечания упаковки не отображаются в форме упаковки при добавлении инструкций упаковки в качестве вложения к шаблону продукта или варианту продукта.

## <a name="resolution"></a>Приказ

Система работает в соответствии с разработанным поведением.

Текущая логика для отображения примечаний упаковки в форме упаковки требует, чтобы примечания были связаны с отгрузками.
