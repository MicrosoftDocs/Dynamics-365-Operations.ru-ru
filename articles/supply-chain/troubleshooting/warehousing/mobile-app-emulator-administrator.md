# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Может использовать только роль администратора для доступа к эмулятору приложения для мобильного устройства

## <a name="symptoms"></a>Симптомы

При настройке складов в Supply Chain Management нельзя использовать любую роль, кроме роли администратора, для доступа к эмулятору приложения мобильного устройства.

## <a name="resolution"></a>Решение

Эмулятор приложения для мобильного устройства настроен для работы только с учетной записью администратора. Для всех целей тестирования и динамических процессов рекомендуется использовать само мобильное приложение управления складом.
