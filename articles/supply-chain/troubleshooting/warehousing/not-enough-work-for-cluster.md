# <a name="not-enough-work-found-for-cluster-when-using-system-directed-cluster-picking"></a>Недостаточная работа кластера, если используется комплектация кластера под управлением системы

## <a name="symptoms"></a>Симптомы

При использовании процесса *Комплектация кластера, управляемая системой* при настройке профиля кластера, в котором, например, можно скомплектовать 10 позиций, процесс работает как запланировано, если для комплектации на 10 позиций достаточно работы. Однако если в комплектации только 8 позиций, появится следующее сообщение об ошибке:

> Для кластера не найдено достаточно работы.

## <a name="resolution"></a>Решение

Чтобы устранить эту проблему, отредактируйте профиль кластера и установите для параметра **Активировать позиции** значение *Нет*.
