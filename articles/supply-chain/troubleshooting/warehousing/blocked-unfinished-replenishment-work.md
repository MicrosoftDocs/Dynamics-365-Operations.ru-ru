# <a name="picking-work-cant-be-unblocked-because-of-unfinished-replenishment-work"></a>Работа по комплектации не может быть разблокирована из-за незаконченной работы пополнения

## <a name="symptoms"></a>Симптомы

При использовании пополнения для волнового спроса работа комплектации может быть заблокирована из-за зависимой работы пополнения. Если местонахождение комплектации должно быть пополнено для удовлетворения спроса по исходному заказу, система создает и работу пополнения, и работу комплектации. Однако она блокирует работы комплектации, пока не будут завершены работы пополнения и вы не получите следующее сообщение об ошибке:

> Невозможно разблокировать работу %1, поскольку она имеет незавершенную работу пополнения.

## <a name="resolution"></a>Решение

Такое поведение является умышленным, поскольку местонахождение комплектации не имеет достаточного количества запасов, если только работа по пополнению не завершена. Выполните работу по пополнению, а затем обработайте работу комплектации.
