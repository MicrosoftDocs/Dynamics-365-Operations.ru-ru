# <a name="the-weight-fields-on-load-lines-dont-match-the-weight-fields-on-the-load"></a>Поля веса в строках загрузки не соответствуют полям веса при загрузке

Коды ошибок: WHSLoadWeightOnLinesDoNotMatchLoadWarning

## <a name="symptoms"></a>Симптомы

Система отобразит следующее сообщение об ошибке:

> Поля веса в строках загрузки не соответствуют полям веса при загрузке %1. Выполните проверку целостности веса загрузки склада, чтобы пересчитать поля веса.

## <a name="cause"></a>Причина

В полях **Вес нетто** и **Вес тары** задано значение *0* (ноль) в строке загрузки. Однако поля веса не заданы как *0* для измерений веса продукта. Если в строке загрузки не заданы поля веса, любая корректировка количества в строке загрузки может привести к неправильному расчету веса загрузки. Эта проблема может возникнуть, если вес продукта был изменен с момента создания строки загрузки.

## <a name="resolution"></a>Приказ

По умолчанию при создании строки загрузки в нее вводятся поля веса продукта. Если вес равен нулю, его можно пересчитать, используя функцию *Проверка целостности веса загрузки склада*.

1. Перейдите к пункту **Администрирование системы \> Периодические задачи \> База данных \> Проверка целостности**.
1. В диалоговом окне **Проверка целостности** в поле **Модуль** задайте *Управление складом*.
1. Задайте в поле **Режим проверки** значение *Исправление ошибок*.
1. В списке флажков установите флажок **Проверка целостности веса загрузки склада** и убедитесь, что выделена только строка для этого флажка.
1. В верхней части списка флажков выберите кнопку с многоточием (**…**), а затем выберите пункт **Диалоговое окно** в меню.
1. В диалоговом окне **Проверка целостности веса загрузки склада** установите следующие поля, чтобы указать критерии, для которых должен применяться проверка целостности:

    - **Код загрузки:** укажите код загрузки. Оставьте поле пустым для проверки всех кодов загрузки.
    - **Код номенклатуры:** укажите код номенклатуры. Оставьте поле пустым для проверки всех номенклатур.
    - **Всегда пересчитывать вес при загрузке:** установите для этого параметра значение *Да*.
    - **Разрешить переопределение веса в строках загрузки:** установите для этого параметра значение *Да*.

1. Нажмите **ОК**, чтобы закрыть диалоговое окно **Проверка целостности веса загрузки склада**.
1. Нажмите кнопку с многоточием, а затем выберите **Выполнить** в меню.

Вес пересчитывается на основе заданных вами критериев. Выберите ссылку **Сведения о сообщении**, чтобы просмотреть результат проверки целостности.
