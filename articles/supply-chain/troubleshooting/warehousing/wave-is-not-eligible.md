# <a name="wave-isnt-eligible-for-cleanup"></a>Не разрешается очистка волны

Код ошибки: WaveNotEligibleForCleanup

## <a name="symptoms"></a>Симптомы

Система отобразит следующее сообщение об ошибке:

> Волна %1 не подлежит очистке

Невозможно очистить данные волны.  

## <a name="cause"></a>Причина

В настоящее время запись волны обрабатывается по одной из следующих причин:

- Для поля **Статус волны** задано значение *Выполнение*.
- При выборе **Пакетное задание** в группе **Волна** на вкладке **Волна** в области действий в поле **Статус** не устанавливается значение *Завершено*, *Ошибка* или *Отменено*. Поэтому в данный момент выполняется пакетное задание.

## <a name="resolution"></a>Приказ

В области действий на вкладке **Волна** в группе **Волна** выберите **Пакетное задание**, а затем выполните одно из следующих действий:

- Если в поле **Статус** указано значение *Выполняется*: в области действий на вкладке **Пакетное задание** в группе **Пакетное задание** выберите **Просмотр задач**. Можно отслеживать ход выполнения, обновляя страницу **Пакетные задания**.
- Если в поле **Статус** не указано значение *Выполняется*: в области действий на вкладке **Пакетное задание** в группе **Функции** выберите **Изменить статус**. В поле **Выбрать новый статус** выберите *Ожидание*. Затем выберите **OK**.
