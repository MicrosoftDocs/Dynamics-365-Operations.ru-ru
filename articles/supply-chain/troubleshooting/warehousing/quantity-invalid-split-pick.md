# <a name="quantity-is-not-valid-when-performing-a-split-pick-across-multiple-batches"></a>При выполнении разделения комплектации для нескольких партий количество является недопустимым

## <a name="symptoms"></a>Симптомы

При попытке выполнить *разделение комплектации* по нескольким партиям вы получаете следующее сообщение об ошибке:

> Недопустимое количество единицы %1.

## <a name="resolution"></a>Решение

Работник склада должен использовать процесс *недоукомплектованности* в мобильном приложении Warehouse Management. При попытке комплектации нескольких партий из одного местоположения можно также использовать параметр **Полный** в мобильном приложении управления складом.
