# <a name="direct-delivery-not-able-to-process-for-wms-enabled-warehouse"></a>Прямая поставка невозможно обработать для склада, на котором включена система управления складом

## <a name="symptoms"></a>Симптомы

Если номенклатура добавляется в строку продаж для прямой поставки со склада, для которого включена система управления складом, при обновлении строки продаж выводится следующее сообщение об ошибке:

> "Невозможна обработка прямой поставки для склада %1, так как для него включена система управления складом. Задайте другой склад, для которого управление складом не включено.

## <a name="resolution"></a>Решение

Корпорация Майкрософт проверила эту проблему и определила, что она является ограничением функции. В настоящее время WMS не поддерживает прямую поставку. Таким образом, для использования прямой поставки необходимо выбрать номенклатуру и склад, для которых не используется WMS.
