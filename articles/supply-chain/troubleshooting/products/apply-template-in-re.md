# <a name="you-cant-apply-a-template-to-create-a-released-product"></a>Невозможно применить шаблон, чтобы создать выпущенный продукт.

Номер статьи базы знаний: 4612097

## <a name="symptoms"></a>Симптомы

При создании нового запущенного продукта с помощью диалогового окна **Новый выпущенный продукт** можно выбрать шаблон. Шаблон должен предоставлять настройки по умолчанию для многих полей нового выпущенного продукта. Однако некоторые или все поля не настраиваются после выбора шаблона.

## <a name="cause"></a>Причина

Многие страницы в Microsoft Dynamics 365 Supply Chain Management позволяют создать шаблон, который устанавливает исходные настройки для записей, отображаемых на этих страницах. Можно создать один из этих шаблонов, выбрав **Сведения о записи** на вкладке **Параметры** панели операций. Однако выпущенные продукты значительно сложнее большинства других типов записей. Хотя можно выбрать **Сведения о записи** на странице **Выпущенные продукты** для создания шаблона и хотя этот шаблон можно выбрать при создании выпущенного продукта, шаблон не будет предоставлять ожидаемые значения полей для нового выпущенного продукта. Таким образом, нельзя использовать шаблоны "сведения о записи" для выпущенных продуктов. Вместо этого необходимо создать специальные шаблоны продуктов.

## <a name="resolution"></a>Приказ

Чтобы создать шаблон продукта, воспользуйтесь меню **Шаблон** на вкладке **Продукт** панели операций, а не кнопкой **Сведения о записи** на вкладке **Параметры**.

1. Выберите **Управление сведениями о продукте \> Продукты \> Выпущенные продукты**.
1. На панели операций на вкладке **Продукт** в группе **Создать** выберите **Шаблон**, а затем выберите вариант **Создать личный шаблон** или **Создать общий шаблон**, как необходимо.
