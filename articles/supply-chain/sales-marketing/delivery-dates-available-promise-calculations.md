## <a name="atp-calculations"></a><span data-ttu-id="6ee27-121">Расчеты ATP</span><span class="sxs-lookup"><span data-stu-id="6ee27-121">ATP calculations</span></span>
<span data-ttu-id="6ee27-122">Количество ATP вычисляется с помощью метода "суммарное ATP с прогнозированием".</span><span class="sxs-lookup"><span data-stu-id="6ee27-122">The ATP quantity is calculated by using the “cumulative ATP with look-ahead” method.</span></span> <span data-ttu-id="6ee27-123">Основное преимущество этого метода подсчета ATP заключается в том, что с его помощью можно обрабатывать случаи, когда сумма расходов между приходами больше, чем последний приход (т. е. когда необходимо использовать количество из предыдущего прихода для соответствия требованию).</span><span class="sxs-lookup"><span data-stu-id="6ee27-123">The main advantage of this ATP calculation method is that it can handle cases where the sum of issues among receipts is more than the latest receipt (for example, when a quantity from an earlier receipt must be used to meet a requirement).</span></span> <span data-ttu-id="6ee27-124">Метод расчета "суммарное ATP с прогнозированием” включает все расходы, пока суммарное получаемое количество не превысит суммарное количество расходов.</span><span class="sxs-lookup"><span data-stu-id="6ee27-124">The “cumulative ATP with look-ahead” calculation method includes all issues until the cumulative quantity to receive exceeds the cumulative quantity to issue.</span></span> <span data-ttu-id="6ee27-125">Следовательно этот метод расчета ATP оценивает, можно ли использовать некоторое количество из более раннего периода в более последнем периоде.</span><span class="sxs-lookup"><span data-stu-id="6ee27-125">Therefore, this ATP calculation method evaluates whether some of the quantity from an earlier period can be used in a later period.</span></span>  

<span data-ttu-id="6ee27-126">Доступное для резервирования (ATP) количество — это незадействованное сальдо склада за первый период,</span><span class="sxs-lookup"><span data-stu-id="6ee27-126">The ATP quantity is the uncommitted inventory balance in the first period.</span></span> <span data-ttu-id="6ee27-127">Оно обычно подсчитывается для каждого периода, на который запланировано поступление.</span><span class="sxs-lookup"><span data-stu-id="6ee27-127">Typically, it's calculated for each period in which a receipt is scheduled.</span></span> <span data-ttu-id="6ee27-128">Программа оценивает период доступного для резервирования (ATP) количества в днях и считает текущую дату первой датой доступного для резервирования (ATP) количества.</span><span class="sxs-lookup"><span data-stu-id="6ee27-128">The program calculates the ATP period in days and calculates the current date as the first date for the ATP quantity.</span></span> <span data-ttu-id="6ee27-129">В первом периоде в доступное для резервирования (ATP) количество входят запасы в наличии минус должные и просроченные заказы клиента.</span><span class="sxs-lookup"><span data-stu-id="6ee27-129">In the first period, ATP includes on-hand inventory minus customer orders that are due and overdue.</span></span>  

<span data-ttu-id="6ee27-130">ATP рассчитывается по следующей формуле:</span><span class="sxs-lookup"><span data-stu-id="6ee27-130">ATP is calculated by using the following formula:</span></span>  

<span data-ttu-id="6ee27-131">ATP = ATP за предыдущий период + приходы за текущий период - расходы за текущий период - количество чистых расходов за каждый период вплоть до периода, когда сумма приходов за все будущие периоды вплоть до будущего периода включительно, превышает сумму расходов вплоть до будущего периода включительно.</span><span class="sxs-lookup"><span data-stu-id="6ee27-131">ATP = ATP for the previous period + Receipts for the current period – Issues for the current period – Net issue quantity for each future period until the period when the sum of receipts for all future periods, up to and including the future period, exceeds the sum of issues up to and including the future period.</span></span>  

<span data-ttu-id="6ee27-132">Обратите внимание, что расчет доступного для резервирования количества не включает информацию, указанную в поле даты окончания срока годности, а также временную границу ATP, ожидаемую системой, когда любое количество может быть обещано.</span><span class="sxs-lookup"><span data-stu-id="6ee27-132">Notice that the ATP calculation does not include information around expiry date and beyond the ATP time fence that the system expects when any quantity can be promised.</span></span>

<span data-ttu-id="6ee27-133">Когда не останется расходов и приходов для рассмотрения, доступное для резервирования (ATP) количество для следующих дат совпадает с последним подсчитанным доступным для резервирования (ATP) количеством.</span><span class="sxs-lookup"><span data-stu-id="6ee27-133">When there are no more issues or receipts to consider, the ATP quantity for the following dates is the same as the latest calculated ATP quantity.</span></span>  

<span data-ttu-id="6ee27-134">Если не все аналитики номенклатуры заполнены во время проверки доступного для резервирования (ATP) количества, их все равно можно указать в расходах и поступлениях.</span><span class="sxs-lookup"><span data-stu-id="6ee27-134">If not all the dimensions that are used for an item are given when the ATP check is completed, they can still be specified on the issue and receipts.</span></span> <span data-ttu-id="6ee27-135">В этом случае во время расчета доступного для резервирования (ATP) количества поступления и расходы должны быть агрегированы по существующим аналитикам, чтобы уменьшить количество строк поступлений и расходов при расчете доступного для резервирования (ATP) количества.</span><span class="sxs-lookup"><span data-stu-id="6ee27-135">In this case, in the ATP calculation, the receipts and issues must be aggregated to the existing dimensions to reduce the number of receipt and issue lines that are used in the ATP calculation.</span></span>  

<span data-ttu-id="6ee27-136">Отображаемое количество ATP всегда больше или равно 0 (нулю).</span><span class="sxs-lookup"><span data-stu-id="6ee27-136">The ATP quantity that is shown is always greater than or equal to 0 (zero).</span></span> <span data-ttu-id="6ee27-137">Если при подсчете получается отрицательное доступное для резервирования (ATP) количество (например, если ранее было зарезервировано количество большее, чем доступное) количество автоматически устанавливает равным 0.</span><span class="sxs-lookup"><span data-stu-id="6ee27-137">If the calculation returns a negative ATP quantity (for example, if the quantity that was previously promised exceeds the available quantity), the quantity is automatically set to 0.</span></span>

<span data-ttu-id="6ee27-139">В поле **Временная граница требования ATP назад во времени** можно указать, какое время назад следует искать задержанные заказы требования или расходы запасов.</span><span class="sxs-lookup"><span data-stu-id="6ee27-139">The **ATP backward demand time fence** field controls how far back in time to look for delayed demand orders or inventory issues.</span></span> <span data-ttu-id="6ee27-140">В поле **Временная граница поставки ATP назад во времени** можно указать, какое время назад следует искать задержанные заказы поставки или поступления на склад.</span><span class="sxs-lookup"><span data-stu-id="6ee27-140">The **ATP backward supply time fence** field controls how far back in time to look for delayed supply orders or inventory receipts.</span></span> <span data-ttu-id="6ee27-141">Например, если считать заказы, которые задерживаются только на семь дней, следует учитывать в расчете ATP, в обоих полях будет задано значение **7**.</span><span class="sxs-lookup"><span data-stu-id="6ee27-141">For example, if orders that are delayed by only seven days should be considered in the ATP calculation, both fields should be set to **7**.</span></span>
