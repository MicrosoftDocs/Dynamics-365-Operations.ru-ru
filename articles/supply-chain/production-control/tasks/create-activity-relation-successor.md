# <a name="create-activity-relation---successor"></a>Создание связи мероприятия: последующий элемент

[!include [banner](../../includes/banner.md)]

Поток мероприятий в бережливом производственном потоке документируется через связи мероприятия. Эта запись показывает, как создать связь мероприятия.

Обязательные компоненты:

- Производственный поток и версия в черновом режиме. 

- Два мероприятия, следующих друг за другом в производственном потоке, создаются, но не связываются.


## <a name="find-the-production-flow-version"></a>Поиск версии производственного потока 
1. Перейдите в раздел "Управление производством" > "Настройка" > "Поток бережливого производства" > "Производственные потоки".
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.
4. В списке пометьте выбранную строку.
5. В списке выберите черновую версию.
    * Связи мероприятий можно добавить как в черновые, так и в активные версии производственного потока.  

## <a name="open-the-activity-overview"></a>Открыть обзор мероприятия
1. Щелкните "Мероприятия".
    * Обратите внимание, что в форме отображаются все мероприятия производственного потока, которые назначены для версии производственного потока, в которой вы работаете.  

## <a name="add-a-successor"></a>Добавить последующий элемент
1. Щелкните "Добавить последующий элемент".
2. В поле "Мероприятие" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
3. Поиск и выбор требуемой записи в списке.
4. В списке перейдите по ссылке в выбранной строке.
5. Установите флажок "Ограничение".
6. В поле "Значение ограничения" введите число.
    * Время ограничения — это время для планирования между плановым завершением предшествующего элемента (срок выполнения и время) и плановым началом последующего действия.  
7. В поле "Единица" введите значение.
8. В поле "Коэффициент времени цикла" введите число.
    * Если оба мероприятия выполняются в одном и том же такте, то коэффициент времени цикла должен быть равен 1. Если предшествующий элемент выполняется с удвоенной скоростью относительного последующего элемента, коэффициент должен быть равен 2.   Коэффициенты времени цикла используются для расчета отдельных времен циклов мероприятий производственного потока.  
9. Нажмите кнопку "OК".
10. Закройте страницу.
11. Перейдите на вкладку "Панель сетки".
12. Закройте страницу.
13. Обновите страницу.
