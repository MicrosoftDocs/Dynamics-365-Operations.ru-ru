# <a name="activate-a-production-flow-version"></a>Активация версии производственного потока

[!include [banner](../../includes/banner.md)]

Прежде чем можно будет создать задания канбана и обработать мероприятия для версии производственного потока, необходимо активировать версию. Рекомендуется активировать версию до того, как истекут ее дата и время вступления в силу. Для выполнения этой задачи необходимо иметь версию производственного потока со статусом "Черновик", чтобы активировать версию. 


## <a name="find-and-activate-a-production-flow-version"></a>Поиск и активация версии производственного потока
1. Перейдите в раздел "Производственные потоки".
2. В списке найдите и выберите требуемую запись.
    * Выберите версию производственного потока со статусом "Черновик" в поле "Статус плана".  
3. В списке перейдите по ссылке в выбранной строке.
4. В списке найдите и выберите требуемую запись.
    * Выберите версию производственного потока со статусом "Черновик" в поле "Статус плана".  
5. Нажмите кнопку Активировать.
6. Нажмите кнопку "OК".
