# <a name="release-a-production-order"></a>Выпуск производственного заказа

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как запустить в производство производственный заказ. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Это четвертая из семи процедур, которая объясняет жизненный цикл производственного заказа.

1. Перейдите в раздел "Управление производством" > "Производственные заказы" > "Все производственные заказы".
    * В сетке выберите производственный заказ со статусом "Запланировано".  
2. В области действий щелкните "Производственный заказ".
3. Щелкните "Запуск в производство".
    * На этой странице можно подтвердить выпуск выбранного диапазона производственных заказов. Щелкните "Выбрать" для добавления заказов.  
    * Шаг "Запуск в производство" указывает, когда производственный заказ запускается в производство в цехе производства, чтобы операторы цеха могли сообщать о ходе выполнения производственных заданий. Можно выпустить производственные документы, содержащие соответствующую информацию об обработке заданий. Работа склада для комплектации сырья создается для номенклатур, настроенных для процесса склада.  
4. Перейдите на вкладку "Общие".
    * Выберите, какие отчеты по производству следует напечатать. В отчете по карте задания печатается одна страница для каждого запланированного задания, и для него требуется запланировать производственный заказ на уровне работы. Отчет содержит сведения о запланированном времени начала и окончания, количестве для производства и ресурсах, обрабатывающих задание. В отчете о маршрутных заданиях указывается та же информация на той же странице, но не печатается страница для каждого задания. В отчете по карте маршрута отображаются только операции, но не задания. Поэтому для данного отчета не требуется планирование заданий, но его можно использовать, когда производственные заказы планируются на уровне операций.  
5. Установите флажок "Печать карты маршрута".
6. Нажмите кнопку "OК".
7. Закройте страницу.
