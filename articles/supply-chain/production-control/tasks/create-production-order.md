# <a name="create-a-production-order"></a>Создание производственного заказа

[!include [task guide banner](../../includes/task-guide-banner.md)]

Следующая процедура используется для создания производственного заказа. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Это первая из семи процедур, которая объясняет жизненный цикл производственного заказа.


## <a name="create-a-production-order"></a>Создание производственного заказа
1. Перейдите в раздел "Управление производством" > "Производственные заказы" > "Все производственные заказы".
2. Щелкните "Новый производственный заказ".
3. В поле "Код номенклатуры" введите "D0001".
4. В поле "Поставка" введите дату.
    * Дата поставки указывает, когда должен завершиться производственный заказ для своевременной поставки. Эту дату можно использовать в процессе планирования. Например, можно запланировать заказ назад от даты поставки.  
5. В поле "Количество" укажите "20".
    * Примечание. В поле "Код спецификации" автоматически отображается номер любой активной спецификации для текущей номенклатуры, но можно изменить спецификацию для производственного заказа, выбрав активную спецификацию из списка утвержденных версий спецификации    В поле "Код маршрута" автоматически отображается номер любого активного маршрута для текущей номенклатуры, но можно изменить маршрут для производственного заказа, выбрав активный маршрут из списка утвержденных версий маршрута.  
6. Щелкните "Создать".

## <a name="validate-the-production-order"></a>Проверка производственного заказа
1. В списке перейдите по ссылке в выбранной строке.
    * Перейдите по ссылке только что созданного номера производственного заказа. Откроется страница сведений для заказа.  
2. Щелкните "Изменить".
3. В поле "Поставка" введите дату.
    * Например, можно изменить дату поставки для производственного заказа.  
4. Нажмите кнопку "Сохранить".
5. Закройте страницу.

## <a name="update-the-bom"></a>Обновление номенклатуры
1. В области действий щелкните "Производственный заказ".
2. Щелкните "Спецификация".
    * Откройте страницу спецификации, чтобы проверить данные спецификации, которые были скопированы из данных по умолчанию при создании производственного заказа. В этой процедуре необходимо обновить количество для спецификации.  
3. Щелкните "Изменить".
4. В поле "Количество" введите число.
    * Изменение количества в строке спецификации влияет на оценку затрат потребления материалов для производственного заказа.  
5. Нажмите кнопку "Сохранить".
6. Закройте страницу.

## <a name="update-the-production-route"></a>Обновление маршрута производства
1. В области действий щелкните "Производственный заказ".
2. Щелкните "Маршрут".
    * Откройте страницу "Маршрут", чтобы проверить данные маршрута производства, которые были скопированы из данных по умолчанию при создании заказа. В этой процедуре необходимо обновить количество для одной из операции в маршруте производства.  
3. В списке найдите и выберите требуемую запись.
4. Щелкните "Изменить".
5. В поле "К-во процесса" введите число.
    * Изменение времени процесса влияет на расчетное потребление на маршруте и затраты производственного заказа.  
6. Нажмите кнопку "Сохранить".
7. Закройте страницу.
