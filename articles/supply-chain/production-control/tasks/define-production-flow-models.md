    * <span data-ttu-id="af5bb-125">После того как для ячейки создано резервирование мощности, тип периода планирования нельзя изменить.</span><span class="sxs-lookup"><span data-stu-id="af5bb-125">After a work cell has capacity reservations, the planning period type cannot by changed.</span></span> <span data-ttu-id="af5bb-126">Можно выбрать только модели производственного потока с тем же типом периода для этой конкретной производственной ячейки.</span><span class="sxs-lookup"><span data-stu-id="af5bb-126">You can only select production flow models with the same period type for this given work cell.</span></span>

    * <span data-ttu-id="af5bb-134">Возможны следующие варианты:   Отсрочка — откладывание полного спроса события планирования на следующий доступный день производства с доступной пропускной способностью.</span><span class="sxs-lookup"><span data-stu-id="af5bb-134">The options include:   Postpone -
