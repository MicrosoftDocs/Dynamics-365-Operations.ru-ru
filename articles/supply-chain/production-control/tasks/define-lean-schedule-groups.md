# <a name="define-lean-schedule-groups"></a>Определение групп графиков бережливого производства

[!include [task guide banner](../../includes/task-guide-banner.md)]

Группы графиков бережливого производства определены для группировки и различения продуктов в планировании канбана. Группирование может выполнить как универсальную связь на компанию или конкретно для производственной ячейки. Каждая группа имеет цветовой код, назначенный для визуальной индикации на странице планирования канбана. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="define-lean-scheduling-group"></a>Определение группы планирования бережливого производства
1. Перейдите в раздел "Управление сведениями о продуктах" > "Бережливое производство" > "Группы графиков бережливого производства".
2. Щелкните "Создать".
3. В поле "Группа планирования" введите значение.
    * Группу планирования можно определить как глобальную группу или конкретную группу для производственной ячейки. В этом примере определим глобальную группу, и производственная ячейка останется пустой. Настройки этой группы применяются ко всем производственным ячейкам, которые не имеют определенных групп планирования.  
4. Выберите цвет в списке цветов.
    * Цвета используются, чтобы выделить задания на странице списка планирования канбана или на панели процессов канбана.  
5. В списке пометьте выбранную строку.
6. В списке перейдите по ссылке в выбранной строке.

## <a name="associate-product"></a>Связь продукта
1. Связывание конкретного продукта
    * Существует два способа связи продуктов с группами графиков бережливого производства: как конкретный продукт (тип связи номенклатуры = номенклатура) или как часть ключа распределения номенклатуры (тип связи номенклатуры = группа).    
2. В поле "Тип связи номенклатуры" выберите "Номенклатура".
3. В поле "Код номенклатуры" введите значение.
4. В поле "Коэффициент пропускной способности" введите число.
    * Коэффициент пропускной способности по умолчанию равняется 1, это означает, что связанные продукты потребляют точно такую мощность, какая указана в мероприятиях производственных потоков. Коэффициент пропускной способности > 1 определяет более высокое потребление ресурсов, коэффициент пропускной способности < 1 определяет более низкое потребление ресурсов. Коэффициент используется при вычислении затрат и в расчете потребления для задания канбана.  

## <a name="associate-item-allocation-key"></a>Связь ключа распределения номенклатуры
1. Связь ключа распределения номенклатуры
    * Добавьте связь с ключом распределения номенклатуры с помощью типа связи номенклатуры "группа".   Обратите внимание, что для этого процесса необходимо в данных определить ключ распределения номенклатуры прогноза.  
2. В поле "Тип связи номенклатуры" выберите "Группа".
3. В поле "Ключ распределения номенклатуры" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. В списке перейдите по ссылке в выбранной строке.
