# <a name="define-discrete-manufacturing-resource-group"></a>Определение групп ресурсов для дискретного производства

[!include [banner](../../includes/banner.md)]

Группа ресурсов — это набор операционных ресурсов, которые обычно соответствуют физической организации производственных ячеек, определенных желтыми строками в цехе производства. Следующая процедура используется для определения группы ресурсов для использования в дискретном производстве. Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USMF или собственные данные.

1. Перейдите в раздел "Группы ресурсов".
2. Щелкните "Создать".
3. В поле "Группа ресурсов" введите значение.
4. В поле "Описание" введите значение.
5. В поле "Сайт" введите или выберите значение.
6. В поле "Производственное подразделение" введите или выберите значение.

## <a name="define-default-operational-parameters"></a>Определение рабочих параметров по умолчанию
1. Разверните раздел "Операция".
2. В поле "Процент отходов" введите число.
3. В поле "Категория настройки" введите или выберите значение.
4. В поле "Категория времени выполнения" введите или выберите значение.
5. В поле "Процент планирования операций" введите число.

## <a name="define-operating-hours"></a>Определение часов работы
1. Разверните раздел "Календари".
2. Нажмите кнопку Добавить.
3. В поле "Календарь" введите или выберите значение.

## <a name="add-operations-resources"></a>Добавление операционных ресурсов
1. Разверните раздел "Ресурсы".
2. Нажмите кнопку Добавить.
3. В поле "Ресурс" введите или выберите значение.
4. Нажмите кнопку Добавить.
5. В поле "Ресурс" введите или выберите значение.
6. В списке найдите и выберите требуемую запись.
7. В списке перейдите по ссылке в выбранной строке.
