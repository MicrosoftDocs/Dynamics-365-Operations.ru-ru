# <a name="add-an-existing-activity-to-a-production-flow-version"></a>Добавление имеющегося мероприятия в версию производственного потока

[!include [banner](../../includes/banner.md)]

При создании новых версий производственных потоков можно выбрать добавление мероприятий, созданных для более старых версий, в новую версию. Эта процедура демонстрирует порядок создания новой версии для существующего производственного потока без копирования мероприятий. В следующем шаге существующее мероприятие добавляется в новую версию. 

Эта задача требует производственного потока с уже созданными версией и мероприятиями.


## <a name="create-a-new-production-flow-version"></a>Создание новой версии производственного потока
1. Перейдите в раздел "Управление производством" > "Настройка" > "Поток бережливого производства" > "Производственные потоки".
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.
4. Выберите Изменить.
5. В списке пометьте выбранную строку.
6. В поле "Дата окончания срока действия" введите дату и время.
    * Обратите внимание, что перед созданием новой версии производственного потока необходимо проверить дату окончания срока действия и время активной версии. Новая версия будет создана с эффективной датой начала, которая связывается с датой окончания срока действия выбранной версии.  
7. Нажмите кнопку Добавить.
8. Выберите значение "Нет" в поле "Копировать из версии".
    * Выберите "Нет", чтобы начать с пустой версии, если большинство мероприятий скопированной версии будет заменено новыми мероприятиями. Добавьте неизменные мероприятия к функции "Добавить существующее" в форме мероприятия вручную.  
9. Выберите "Нет" в поле "Дублировать правила канбана".
    * Когда мероприятия не копируются в новую версию, невозможно скопировать правила канбана во время создания новой версии.   Вместо этого вы будете использовать функцию создания заменяющего канбана в форме правила канбана для замены правил канбана старой версии производственного потока на правила канбана с использованием мероприятий новой версии.  
10. Нажмите кнопку "OК".
11. В списке найдите и выберите требуемую запись.

## <a name="add-an-existing-activity"></a>Добавление существующего мероприятия
1. Щелкните "Мероприятия".
2. Щелкните "Добавить существующее", чтобы открыть раскрывающееся диалоговое окно.
    * Найдите и выберите существующее мероприятие для добавления в новую версию производственного потока.  Обратите внимание, что в списке отображаются все мероприятия, которые были созданы для этого производственного потока для всех предыдущих версий потока.  
3. В поле "Мероприятие" введите или выберите значение.
4. Нажмите кнопку "OК".
