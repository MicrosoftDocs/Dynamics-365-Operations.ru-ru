# <a name="prepare-a-process-kanban-job-when-materials-are-available-for-the-work-cell"></a>Подготовка обработки заданий канбана при наличии материалов для производственной ячейки

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой задаче особое внимание обращается на подготовку задания канбана обработки, когда для производственной ячейки имеются все материалы. В качестве компании с демонстрационными данными для создания этой задачи используется USMF. Эта задача предназначена для оператора.

1. Перейдите на доску канбана для заданий обработки.
2. В поле "Производственная ячейка" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
3. В списке перейдите по ссылке в выбранной строке.
    * Выберите производственную ячейку 1250 и нажмите "ОК".  
4. В списке выберите строку 4.
    * В чистой демонстрационной компании канбан 000329 в строке 4 — это первое задание, которое еще не завершено.  
5. Переключите развертывание раздела "Отгрузочная накладная".
    * Убедитесь, что статус поставки для всех номенклатур в листе комплектации — "доступно".  
    * Если выбрано несколько заданий, в листе комплектации отображается сумма всех номенклатур, необходимых для выбранных заданий.  
6. Щелкните "Подготовить".
    * Процесс подготовки завершен. Установленный флажок для всех строк в листе комплектации указывает, что статус поставки — "скомплектовано".
