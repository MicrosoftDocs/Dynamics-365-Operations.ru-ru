# <a name="prepare-a-process-kanban-job-when-materials-are-not-available-for-the-work-cell"></a>Подготовка обработки заданий канбана при отсутствии материалов для производственной ячейки

[!include [task guide banner](../../includes/task-guide-banner.md)]

Эта процедура заключается в подготовке задания канбана обработки, когда для производственной ячейки отсутствуют некоторые материалы, поэтому необходимо выбрать материалы со склада. Процедура "Подготовка задания канбана обработки при наличии материалов" является предварительным условием для создания этой процедуры. Эта процедура предназначена для оператора. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Перейдите в раздел "Управление производством" > "Канбан" > "Доска канбана для заданий процесса".
2. В поле "Производственная ячейка" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
3. В списке перейдите по ссылке в выбранной строке.
    * Выберите производственную ячейку 1250.  
4. В списке найдите и выберите требуемую запись.
    * Выберите канбан 000356.  
5. В списке найдите и выберите требуемую запись.
    * В списке отмените выбор строки 4. (Или выберите строку 4, если вы не выполнили задачу "Подготовка задания канбана обработки при наличии материалов").  
6. Переключите развертывание раздела "Отгрузочная накладная".
    * Значок "Отсутствует запись" в статусе поставки указывает, что для производственной ячейки отсутствует 48 шт. номенклатуры P0002.  

## <a name="transfer-materials-to-work-cell"></a>Перемещение материалов в производственную ячейку
1. Переключите развертывание раздела "Задачи перемещения".
2. В экспресс-фильтре по полю "Код номенклатуры" выберите значение "P0002".
3. В списке найдите и выберите требуемую запись.
4. Щелкните "Начать".
    * Идет перемещение.  
5. Щелкните "Завершить".
    * Номенклатура P0002 теперь присутствует в листе комплектации для задания канбана. Это означает, что мы можем подготовить канбан со всеми необходимыми материалами.  
6. Щелкните "Подготовить".
    * Обратите внимание, что значок "Статус задания" показывает, что задание теперь готово.
