# <a name="copy-co-products-from-an-existing-formula-version"></a>Копирование побочных продуктов из существующей версии формулы

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как скопировать сопутствующие продукты из существующей версии формулы в другую версию формулы для выпущенного продукта. Обязательным условием является наличие по крайней мере одной версии формулы, связанной с сопутствующими продуктами. В качестве компании с демонстрационными данными для создания этой процедуры используется USP2.


## <a name="find-a-released-product"></a>Поиск выпущенного продукта
1. Перейдите в раздел "Выпущенные продукты".
2. Щелкните "Показать фильтры".
    * Вы собираетесь добавить поле "Тип производства" в диалоговое окно фильтра.  
3. Нажмите кнопку "Добавить фильтр" для добавления поля "Тип производства".
    * На следующем шаге необходимо вручную ввести формулу в поле "Тип производства" перед нажатием кнопки "Применить". В результате фильтр будет задан для списка выпущенных продуктов.  
4. Вручную введите формулу в поле "Тип производства".
5. Нажмите кнопку Применить.

## <a name="select-a-released-product"></a>Выбор выпущенного продукта
1. В списке найдите и выберите требуемую запись.
2. Щелкните "Версии формулы".
    * На панели операций технического отдела щелкните "Версии формулы".  

## <a name="copy-co-products"></a>Копирование сопутствующих продуктов
1. В области действий щелкните "Версия формулы".
2. Щелкните Побочные продукты.
3. Щелкните Копировать.
4. В поле "Код номенклатуры" введите или выберите значение.
5. В поле "Версия формулы" введите или выберите значение.
6. Нажмите кнопку "OК".
7. Закройте страницу.
