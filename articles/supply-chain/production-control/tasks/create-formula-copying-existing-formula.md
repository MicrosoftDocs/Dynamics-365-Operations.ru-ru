# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Создание формулы путем копирования из существующей формулы

[!include [banner](../../includes/banner.md)]

Ниже описан порядок быстрого создания новой формулы путем копирования из существующей формулы. Можно использовать компании USMF, USP2 или USPI с демонстрационными данными для создания этой процедуры.


## <a name="create-a-new-formula"></a>Создание новой формулы
1. Перейдите в раздел "Формулы".
2. Щелкните "Создать".
3. В поле "Формула" введите значение.
4. В поле "Имя" введите значение.
5. В поле "Сайт" введите или выберите значение.
6. В поле "Номенклатурная группа" введите или выберите значение.
7. Нажмите кнопку "Сохранить".

## <a name="copy-from-an-existing-formula"></a>Копирование из существующей формулы
1. Щелкните Копировать.
2. В поле "Код номенклатуры" введите или выберите значение.
3. В поле "Версия формулы" введите или выберите значение.
4. Нажмите кнопку "OК".

## <a name="revise-the-copied-formula"></a>Пересмотр скопированной формулы
1. В списке найдите и выберите требуемую запись.
2. Нажмите кнопку Удалить.
3. Щелкните Да.

## <a name="approve-formula"></a>Утвердить формулу
1. Щелкните Утвердить формулу.
2. В поле "Кем утверждено" введите или выберите значение.
3. Нажмите кнопку "OК".
4. Закройте страницу.
