# <a name="create-a-kanban-rule-for-multiple-activities"></a>Создание правила канбана для нескольких мероприятий

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как создать правило канбана, которое включает несколько мероприятий из производственного потока. В качестве компании с демонстрационными данными для создания этой задачи используется USMF. Эта задача предназначена для инженер-технолога или менеджера потока создания ценности, так как он подготавливает производство нового или измененного продукта в среде бережливого производства.


## <a name="create-a-new-kanban-rule"></a>Создать новое правило канбана
1. Перейдите в раздел "Управление сведениями о продуктах" > "Бережливое производство" > "Правила канбана".
2. Щелкните "Создать".
3. В поле "Стратегия пополнения" выберите "Запланировано".
4. В поле "Первое мероприятие плана" введите или выберите значение.
    * Выберите SpeakerAssemblyAndPolish.  
5. Установите флажок "Несколько мероприятий".
    * Цель — включить несколько мероприятий в правило канбана. Вы выбираете путь в производственном потоке при выборе последнего мероприятия плана.  
6. В поле "Последнее мероприятие плана" введите или выберите значение.
    * Выберите SpeakerTestAndPackaging. После выбора значения страница откроется автоматически. Выберите поток канбана SpeakerAssemblyAndPolish > SpeakerTestAndPackaging. Нажмите кнопку "OК".  
7. Разверните раздел "Сведения".
8. В поле "Продукт" введите или выберите значение.
    * Выберите номенклатуру L0006.  

## <a name="create-kanban-and-view-jobs"></a>Создание канбана и просмотр заданий
1. Разверните раздел "Канбаны".
2. Нажмите кнопку Добавить.
3. В поле "Число новых канбанов" введите "1".
    * В результате будет создан один канбан.  
4. Установите количество продукта равным 3.
    * Канбан будет обрабатывать 3 продукта.  
5. В поле "Дата/время срока выполнения" введите дату и время.
    * Можно ввести значение "Сегодня".  
6. Щелкните "Создать".
7. Нажмите кнопку Состав.
    * Обратите внимание, что канбан имеет два задания процесса из производственного потока. Первое задание — это SpeakerAssemblyAndPolish, второе — SpeakerTestAndPackaging.  
    * Это последний шаг.
