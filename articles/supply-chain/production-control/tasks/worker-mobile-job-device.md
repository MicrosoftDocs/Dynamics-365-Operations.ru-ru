# <a name="configure-a-worker-using-the-mobile-job-device"></a>Настройка работника с использованием мобильного устройства задания

[!include [banner](../../includes/banner.md)]

В этом разделе показано, как назначить правильные роли учетной записи пользователя работника, а затем позволить работнику регистрироваться в цеху.

## <a name="verify-that-a-worker-is-assigned-a-certain-role"></a>Проверить, что работнику назначена определенная роль

В этом примере удостоверьтесь, что пользователю "SHANNON" присваивается роль оператора перед настройкой учетной записи работника.

1. Перейдите в раздел **Область перехода > Модули > Администрирование системы > Пользователи > Пользователи**.
2. Поиск пользователя в быстром фильтре. В этом примере введите `shannon`.
3. Выберите ссылку в столбце **Идентификатор пользователя** учетной записи пользователя, который появляется.
4. В дереве **Роли пользователя** выберите **Роли > Оператор станка**.
5. Закройте страницы **Сведения о пользователе** и **пользователи**, чтобы вернуться на домашнюю страницу.

## <a name="configure-worker-account"></a>Настройте учетную запись работника
1. Перейдите к **Область перехода > Модули > Управление персоналом > Работники > Работники**.
2. Поиск пользователя в быстром фильтре. В этом примере введите `shannon`.
3. Выберите ссылку в столбце **Имя** учетной записи пользователя, который появляется.
4. Выберите вкладку **Регистрация времени**.
5. Выберите **Активировать в терминалах регистрации**.
6. Введите или выберите значения в следующих полях:  

    - **Группа расчета**  
    - **Группа расчета по умолчанию**  
    - **Группа утверждения**  
    - **Стандартный профиль**  
    - **Группа графиков**  

7. Нажмите **ОК**.
8. Выберите **Изменить**, чтобы ввести номер пропуска для нового работника с регистрацией времени. Введите значение в поле **Код пропуска**.
9. Нажмите **Сохранить**.
10. Закройте страницы **Сведения о работнике** и **Работники**.

## <a name="assign-worker-to-device-group"></a>Назначьте работника группе устройств
1. Перейдите к **Управление производством > Настройка > Управление производством > Настроить карту заданий для устройств**.
2. Выберите **Добавить**.
3. В списке выберите требуемого работника. В этом примере выберите **SHANNON**.
4. Нажмите **ОК**.
5. Выберите **Правка**.
6. В поле **Производственное подразделение** можно установить фильтр по умолчанию для работника. В результате, когда работник входит в систему для устройства, будут отображаться только производственные задания для выбранного производственного подразделения. Введите требуемое значение.
7. Закройте страницу.
