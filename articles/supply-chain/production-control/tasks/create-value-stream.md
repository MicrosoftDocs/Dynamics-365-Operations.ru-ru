# <a name="create-a-value-stream"></a>Создание потока создания ценности

[!include [banner](../../includes/banner.md)]

Потоки создания ценности — это операционные единицы, группирующие один или несколько производственных потоков. Для определения производственного потока сначала необходимо определить поток создания ценности.


## <a name="open-the-form"></a>Открытие формы
1. Перейдите в раздел "Управление производством" > "Настройка" > "Поток бережливого производства" > "Потоки создания ценности".

## <a name="create-a-new-value-stream"></a>Создание нового потока создания ценности
1. Щелкните "Создать", чтобы открыть раскрывающееся диалоговое окно.
2. В поле "Имя" введите значение.

## <a name="select-a-value-stream-manager"></a>Выбор менеджера потока создания ценности
1. В поле "Менеджер" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
    * Менеджер потока создания ценности определяется для документирования организации.  
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.

## <a name="save-and-close"></a>Сохранить и закрыть
1. Нажмите кнопку "Сохранить".
2. Закройте страницу.
3. Обновите страницу.
