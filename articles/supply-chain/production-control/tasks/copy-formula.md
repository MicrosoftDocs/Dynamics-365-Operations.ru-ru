# <a name="copy-a-formula"></a>Копирование формулы

[!include[task guide banner](../../includes/task-guide-banner.md)]

Эта процедура заключается в создании формулы, содержащей те же компоненты, что и существующая формула, но с небольшими отличиями. Чтобы создать строки формулы, можно воспользоваться функцией Копировать для копирования существующей формулы, содержащей большинство необходимых компонентов. Затем можно внести любые необходимые изменения в отдельные строки в новой версии. При использовании функции Копировать нет необходимости создавать несколько формул, которые почти не отличаются друг от друга. В качестве компании с демонстрационными данными для создания этой задачи используется USP2.


## <a name="create-a-formula"></a>Создание формулы
1. Перейдите в раздел "Управление сведениями о продукте" > "Спецификации и формулы" > "Формулы".
2. Щелкните "Создать".
3. В поле "Формула" введите значение.
4. В поле "Имя" введите значение.
    * Введите информативное имя для формулы.  
5. В поле "Сайт" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. В списке перейдите по ссылке в выбранной строке.
7. В поле "Номенклатурная группа" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
8. В списке найдите и выберите требуемую запись.
9. В списке перейдите по ссылке в выбранной строке.
10. Нажмите кнопку "Сохранить".

## <a name="copy-formula-lines"></a>Копирование строк формулы
1. В области действий щелкните "Формула".
2. Щелкните Копировать.
3. В поле "Код номенклатуры" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. В списке перейдите по ссылке в выбранной строке.
5. В поле "Версия формулы" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. В списке перейдите по ссылке в выбранной строке.
7. Нажмите кнопку "OК".

## <a name="adjust-copied-formula-lines"></a>Корректировка скопированных строк формулы
1. В списке пометьте выбранную строку.
2. Нажмите кнопку Удалить.
3. Щелкните Да.

## <a name="approve-formula"></a>Утвердить формулу
1. В области действий щелкните "Формула".
2. Щелкните Утвердить формулу.
3. В поле "Кем утверждено" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. В списке перейдите по ссылке в выбранной строке.
5. Щелкните Выбрать.
6. Нажмите кнопку "OК".
