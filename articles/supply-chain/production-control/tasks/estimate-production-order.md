# <a name="estimate-a-production-order"></a>Оценка производственного заказа

[!include [banner](../../includes/banner.md)]

Эту процедуру можно выполнять с помощью компании с демонстрационными данными USMF или применительно к собственным данным. В обоих случаях необходимо располагать открытым производственным заказом со статусом "Создано". Это вторая из семи процедур, которая объясняет жизненный цикл производственного заказа.


## <a name="estimate-a-production-order"></a>Оценка производственного заказа
1. Перейдите в раздел "Управление производством" > "Производственные заказы" > "Все производственные заказы".
2. В сетке выберите заказ со статусом "Создано".
3. В области действий щелкните "Производственный заказ".
4. Щелкните "Оценка".
    * На этом этапе рассчитывается оценочная стоимость одного производственного заказа.   
5. Нажмите кнопку "OК".

## <a name="view-the-calculation-details"></a>Просмотр сведений расчета
1. В области действий щелкните "Управление затратами".
2. Щелкните "Просмотреть сведения расчета".
    * На этой странице отображается детализация затрат. Например, можно просматривать общую себестоимость на единицу измерения для готовой продукции в первой строке. Последующие строки содержат стоимость согласно спецификации, маршруту производства и косвенным затратам.
