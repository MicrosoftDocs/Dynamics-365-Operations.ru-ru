# <a name="add-a-predecessor-to-a-production-flow-activity"></a>Добавление предшествующего элемента к мероприятию производственного потока

[!include [task guide banner](../../includes/task-guide-banner.md)]

В версии производственного потока все мероприятия должны быть последовательными. Мероприятие может иметь один или несколько предыдущих или последующих мероприятий. 

Эта процедура демонстрирует порядок связывания предшествующего элемента с мероприятием. 

Для выполнения этой задачи требуется производственный поток, который имеет черновую версию, содержащую хотя бы два мероприятия, которые можно подключить. 

Дополнительные сведения см. в официальном документе "Производственные потоки и мероприятия в бережливом производстве".


## <a name="find-the-production-flow-and-version"></a>Поиск производственного потока и версии
1. Перейдите в раздел "Управление производством" > "Настройка" > "Поток бережливого производства" > "Производственные потоки".
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.
4. В списке найдите и выберите требуемую запись.
5. Щелкните "Мероприятия".

## <a name="select-an-activity-and-add-a-predecessor"></a>Выберите мероприятие и добавьте предшествующий элемент
1. В списке найдите и выберите требуемую запись.
2. Щелкните "Добавить предшествующий элемент".
3. В поле "Мероприятие" введите или выберите значение.
4. В поле "Коэффициент времени цикла" введите число.
    * Коэффициент времени цикла по умолчанию для связи мероприятий равен 1. Это предполагает, что оба мероприятия выполняются в одном темпе или с одинаковым временем такта. Если предшествующий элемент работает в более высоком темпе (с меньшим временем такта), коэффициент должен быть меньше 1, если предшествующий элемент работает в более медленном темпе (с большим временем такта) коэффициент времени цикла будет больше 1.  
5. Нажмите кнопку "OК".
