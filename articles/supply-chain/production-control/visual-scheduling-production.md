# <a name="gantt-chart-for-job-scheduling"></a>Диаграмма Ганта для планирования заданий

[!include [banner](../includes/banner.md)]

Диаграмма Ганта дает планировщикам производства возможность контролировать и оптимизировать производственный план. Диаграмма Ганта обеспечивает прозрачность потока операций и позволяет легко корректировать план производства и с учетом нехватки материалов или ресурсов. Это помогает планировщикам оптимально использовать доступные ресурсы, минимизировать незавершенное производство и оптимизировать обработки производственных заказов.

Диаграмма Ганта — это визуальное представление запланированных действий в течение определенного временного интервала. Действия планируются на ресурсах, имеющих мощность, определяемую календарем мощности. Следующие типы действий могут отображаться на диаграмме Ганта.

-   Задания из производственных заказов, запланированных как задания.
-   Задания из спланированных производственных заказов.
-   Действия по проекту типа "Часовой прогноз", запланированных как задания.

Можно открыть диаграмму Ганта в двух различных представлениях, **Представление заказов** и **Представление ресурсов**[](https://authoring.help.dynamics.com/en/?post_type=incsub_wiki&p=1665154&preview=true). В **Представлении заказов** действия группируются в производственных заказах. Это удобно, например, если вам хотелось бы видеть всех задания, принадлежащие к одним и тем же заказам. В **представлении ресурсов** все задания группируются внутри отдельных ресурсов. Этим представлением удобно пользоваться при оптимизации плана на уровне ресурсов, например станка или группы станков. На диаграммах Ганта на рисунках ниже показаны **представление заказов** и **представление ресурсов** с этими ключевыми элементами:

1.  Действие на диаграмме Ганта
2.  Значок нехватки материала
3.  Значок доступность материала
4.  Значок даты поставки заказа
5.  Панель мощности

## <a name="activities"></a>Действия
Действия отображаются в виде полос на сетке шкалы времени с запланированным временем начала и временем окончания, т.е. длина полос пропорциональна времени, необходимому для выполнения действия. Действия отображаются в соответствии со шкалой времени. Шкалу времени можно откорректировать в меню путем выбора даты начала и даты окончания и единицы времени, например часов или дней. Путем корректировки шкалы времени можно сосредоточиться на временном интервале, действиями в котором вы хотите управлять. 

Для получения более наглядного обзора существуют различные параметры для контроля цвета действий. Можно настроить отдельные цвета для действий, использовать цвет темы (общую цветовую тему, используемую для приложения) или сделать так, чтобы цвет определялся кодом цвета для производственных заказов. 

Интервал времени для действий имеет фоновое затенение. Периоды с белым затенением обозначают временной интервал с заданной мощностью ресурса для действия, тогда как периоды с серым затенением обозначают временные интервалы без заданной мощности. 

Слева от диаграммы имеется дополнительная информация о действии, например, ресурс, на котором запланировано действие, и номер производственного заказа. Связь между заданиями, относящимися к одному и тому же заказу, изображается в виде стрелки. 

Дополнительную информацию о действии можно получить в диалоговом окне действия. Чтобы открыть это диалоговое окно, дважды щелкните действие или выберите меню **информация**. В диалоговом окне действия можно просмотреть запланированное время начала и окончания, а также информацию о том, какие материалы планируется использовать при выполнении действия, с указанием времени. 

Действия могут быть сгруппированы по уровням группировки. Уровни группировки имеют иерархическую структуру и могут использоваться для получения логического группирования действий. Например, если у вас есть макет, где производственные действия сгруппированы по объекту, производственным подразделениям, группам ресурсов и ресурсов, можно использовать уровни группировки для группирования действий в соответствии с этим макетом. Уровни группировки можно разворачивать и сворачивать либо на уровне отдельных группировок, либо на всех уровнях в диаграмме с помощью кнопок **Развернуть все** и **Свернуть все** в меню. Можно также настроить уровни группировки, чтобы они разворачивались и сворачивались при открытии диаграммы.

### <a name="material-availability"></a>Доступность материалов

Диаграмму Ганта можно настроить так, чтобы у планировщика была детальная информация о статусе материалов для отдельных действий. Это бывает полезно, например, когда материал задерживается и это влияет на производственный план. В этом случае проблемы с материалом будут выделены на диаграмме Ганта, чтобы помочь планировщику оценить последствия и внести необходимые корректировки. 

Задание будет отображаться со значком нехватки материала, если дата начала задания по плану позднее, чем дата доступности материалов, потребляемых заданием. Дата доступности материалов рассчитывается на основании информации об источниках потребности в динамическом сводном плане. Значок нехватки материала, например, будет отображаться на задании, которое потребляет материал, привязанный к заказу на покупку, дата прихода по которому позднее запланированной даты начала задания.

### <a name="indicator-of-material-availability-date"></a>Индикатор даты доступности материала

При настройке диаграммы дна отображение заданий с нехваткой материалов может также отображаться значок с датой доступности материала для задания. Значок будет отображаться, только если дата доступности материала находится в пределах временного интервала, заданного для диаграммы. Если дата доступности материала находится вне заданного временного интервала, более подробную информацию о доступности материала можно получить из списка материалов в диалоговом окне задания. В списке также имеется значок, обозначающий запаздывающие материалы для задания. Вы можете перепланировать задание, используя дату доступности материала в качестве начальной даты.

### <a name="indicator-of-order-delivery-date"></a>Индикатор даты поставки заказа

Этот значок показывает дату поставки для производственного заказа. Этот значок отображается только в представлении заказов.

### <a name="capacity-bar"></a>Панель мощности

Диаграмму можно настроить так, чтобы отображалась панель мощности ресурсов. Этот панель позволяет получить представление о мощности ресурсов для действия в пределах временного интервала, заданного для диаграммы. Панель мощности не отображается для периодов, когда ресурс не зарезервирован. В периодах, когда ресурс зарезервирован на всю мощность, панель мощности отображается в виде сплошной полосы. В периодах, когда ресурс зарезервировано с превышением мощности, панель становится толще и отображается красным цветом. Например, если два задания перекрываются, на панели мощности будет показано превышение мощности в том временно интервале, в котором имеется перекрытие. Панель мощности обновляется динамически при планировании задания. Включить панель мощности можно в меню **Показать панель мощности**. Ее отображение возможно только в **представлении ресурсов**. Если вы хотите получить более детальное представление о загрузке мощности ресурса, используйте диаграмму **Максимальная мощность**, которую можно открыть из меню или контекстного меню выбранного действия.

## <a name="job-scheduling-in-the-gantt-chart"></a>Планирование заданий на диаграмме Ганта
На диаграмме Ганта предусмотрены различные варианты для внесения корректировок в план производства. На диаграмме Ганта можно перепланировать действия путем перетаскивания или из меню планирования. В процессе планирования вы может учитывать мощность ресурсов, возможности ресурсов и ограничения по материалам.

### <a name="schedule-a-job-as-a-drag-and-drop-interaction"></a>Планирование задания путем перетаскивания

Вы можете перепланировать задание в пределах заданного интервала времени путем перетаскивания. Перепланировать задание можно только на том же ресурсе, причем одновременно можно перепланировать только одно задание.

### <a name="schedule-a-job-from-the-menu"></a>Планирование задания из меню

В меню **Планирование заданий** можно запланировать одно или несколько выбранных заданий на диаграмме на основании направления планирования и даты и времени планирования. Существует три возможных направления планирования.

-   Вперед от даты планирования
-   Назад от даты планирования
-   Начиная с даты доступности материалов

Запланировать задание на время за пределами временного интервала, заданного для диаграммы Ганта, невозможно. Если это сделать, задание останется незапланированным, и появится сообщение об ошибке: "Не удалось запланировать задание в течение периода времени загрузки".

### <a name="schedule-previous-jobs"></a>Планировать предшествующие задания

При наличии сети действий, например заданий, принадлежащих к одному и тому же производственному заказу, можно использовать функцию **Планировать предшествующие задания**, чтобы запланировать предшествующие задания относительно выбранного задания в сети. В следующем примере выделенное действие — это выбранное задание. На диаграмме показана ситуация до того, как предшествующее задание запланировано, и после того, как предшествующее задание запланировано. 

[![Планирование предшествующего задания](./media/schprevjob3.png)](./media/schprevjob3.png)

### <a name="schedule-next-jobs"></a>Планировать следующие задания

Функцию **Планировать следующие задания** можно использовать, чтобы запланировать следующие задания относительно выбранного задания в сети действий. В следующем примере выделенное действие — это выбранное задание. На диаграмме показана ситуация до того, как следующее задание запланировано, и после того, как следующее задание запланировано. 

[![Планирование следующего задания](./media/schnxtjob.png)](./media/schnxtjob.png)

### <a name="schedule-around-job"></a>Планировать вокруг задания

Функцию **Планировать вокруг задания** можно использовать, чтобы запланировать следующее задание и предшествующее задание относительно выбранного задания в сети действий. В следующем примере выделенное действие — это выбранное задание. На диаграмме показана ситуация до того, как задание запланировано, и после того, как задание запланировано. 

[![Планировать вокруг задания](./media/scharoundjob1.png)](./media/scharoundjob1.png)

### <a name="arrange-jobs"></a>Упорядочить задания

Функцию **Упорядочить** можно использовать для упорядочения выбранных действий на одном и том же ресурсе. Эти действия могут быть в одной и той же сети действий, но также могут принадлежать к разным сетям. При использовании функции упорядочения временные промежутки между выбранными действиями устраняются. Эту функцию можно использовать для оптимизации использования мощности ресурсов. На диаграмме показана ситуация до того, как задание запланировано, и после того, как задание запланировано. 

[![Упорядочение задания](./media/arrangejobs1.png)](./media/arrangejobs1.png)

### <a name="reassign-activities-from-one-resource-to-another"></a>Переназначение действий с одного ресурса на другой

Вы можете переназначить задание с одного ресурса на другой. Это удобно делать в случаях, когда станок вышел из строя или зарезервирован с превышением, и вам нужно найти другой доступный ресурс, на котором можно выполнить задание.

### <a name="reassigning-an-activity-as-a-drag-and-drop-interaction"></a>Переназначение действия путем перетаскивания

В представлении **Ресурс** можно переназначить действие другому ресурсу на диаграмме Ганта путем перетаскивания. Это делается путем выбора строки, в которой запланировано действие. После выбора строки ее можно перетащить ее на ресурс на диаграмме, находящийся внутри другого уровня группировки ресурсов.

### <a name="reassigning-an-activity-from-the-schedule-jobs-menu"></a>Переназначение действия из меню "Планирование заданий"

Переназначить задание можно из диалогового окна **Планировать задание**, которое открывается из меню **Планировать задание**. Из этого меню переназначить задание можно только на ресурс, который уже загружен в диаграмму Ганта. Если выбрано только одно задание, раскрывающийся список для выбора ресурса будет отсортирован по применимым ресурсам. Если выбрано несколько заданий, информации о применимых ресурсах из списка ресурсов не будет.

## <a name="load-additional-resources-to-the-gantt-chart"></a>Загрузка дополнительных ресурсов на диаграмму Ганта
В **представлении ресурсов** у вас есть возможность загрузить дополнительные ресурсы на диаграмму Ганта. Это удобно делать, если вы хотите найти альтернативный ресурс для задания, запланированного на станке, который зарезервирован с превышением или вышел из строя. На странице **Загрузить дополнительные ресурсы** вы найдете список ресурсов на дату открытия списка. Сначала в списке идут ресурсы, применимые по отношению к выбранному заданию на диаграмме Ганта. Если вы выбрали несколько заданий, прежде чем открыть список, ресурсы будут отображаться без указания применимых ресурсов. На странице **Загрузить дополнительные ресурсы** можно выбрать один или несколько ресурсов, которые после подтверждения выбора будут загружены на диаграмму Ганта. Если на выбранном ресурсе не запланировано никаких заданий в пределах временного интервала диаграммы Ганта, ресурс будет помещен внутри уровня группировки ресурсов внизу списка действий на диаграмме Ганта.
