# <a name="default-descriptions-for-the-general-ledger"></a>Описания по умолчанию для главной книги

[!include [banner](../../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

Описания по умолчанию могут использоваться для обновления поля **Описание** в разносках ваучера в главной книге. Эта функция была улучшена для работы со стоимостью на складе.

Чтобы настроить описания по умолчанию для разносок ГК, откройте **Управление организацией \> Настройка \> Описания по умолчанию**.

В следующей таблице перечислены новые значения, которые были добавлены в поле **Описание** на странице **Описания по умолчанию** для поддержки стоимости на складе.

| Тип описания | Основание |
|---|---|
| Импорт затрат — начисление затрат | При разноске накладной по заказу на покупку начисление затрат обрабатывается для оценки затрат на рейс. Описания по умолчанию могут быть указаны для добавления номера рейса в описание. Используйте *%4* для номера отгрузки. |
| Учет затрат на импорт — заказ товаров в пути | При использовании обработки в пути накладная заказа на покупку разносится, и товары разносятся на счет товаров в пути. Описания по умолчанию могут быть указаны для добавления номера заказа в пути в описание. Используйте *%4* для номера заказа в пути. |
| Запасы — закрытие — коррекция | Когда накладная для расчетов с поставщиками (AP) обрабатывается для затрат на рейс, корректировка запасов обрабатывается для оценки затрат на рейс. Описания по умолчанию могут быть указаны для добавления номера рейса в описание. Используйте *%4* для номера отгрузки. |

Дополнительные сведения о работе со страницей **Описания по умолчанию** см. в разделе [Настройка описаний по умолчанию для автоматической разноски](../../finance/general-ledger/set-up-default-descriptions-for-automatic-posting.md).
