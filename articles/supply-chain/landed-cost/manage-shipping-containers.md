# <a name="manage-shipping-containers"></a>Управление контейнерами отгрузки

[!include [banner](../../includes/banner.md)]

Контейнеры отгрузки используются для группировки товаров, которые физически сгруппированы вместе. Они также используются в случаях, когда затраты должны быть общими только для этих товаров, обычно потому что они физически вместе.

Для просмотра и обработки товаров на странице контейнеров отгрузки перейдите к **Стоимость на складе \> Контейнеры отгрузки \> Все контейнеры отгрузки**. На странице **Все контейнеры отгрузки** отображается список всех доступных контейнеров отгрузки. Можно использовать кнопки на панели операций для создания, удаления и работы с контейнерами отгрузки. Выберите любой контейнер отгрузки в списке для просмотра его подробных сведений на странице **Контейнеры отгрузки**.

В верхней части страницы подробных сведений о контейнере отгрузки отображается информация о контейнере отгрузки и информация о затратах. В разделе **Строки** показаны листы, номенклатуры и заказы на покупку или заказы на перемещение, связанные с контейнером.

## <a name="action-pane"></a>Панель операций

Панель операций на страницах **Все контейнеры отгрузки** и **Контейнеры отгрузки** содержат кнопки, которые позволяют работать с выбранным контейнером отгрузки. Каждая кнопка выполняет одно действие. Панель операций также включает вкладки, каждая из которых, в свою очередь, содержит набор соответствующих кнопок. За исключением оговоренных случаев, все кнопки и вкладки, описанные в следующих подразделах, доступны как в представлении списка (то есть на странице **Все контейнеры отгрузки**), так и в подробном представлении (то есть на странице **Контейнеры отгрузки**).

### <a name="buttons-on-the-manage-tab"></a>Кнопки на вкладке "Управление"

В следующей таблице описаны кнопки на панели операций вкладки **Управление**.

| Кнопка | Описания |
|---|---|
| Разноска списка прихода | Разноска списка поступлений или просмотр списков поступлений продуктов для всех строк заказа на покупку в контейнере отгрузки. При использовании отгрузок с несколькими компаниями для каждой компании открывается новое диалоговое окно разноски списка поступлений. |
| Разноска поступления продуктов | Разнесите поступление продукта для всех строк заказа на покупку в контейнере отгрузки. |
| Разноска накладной | Разнесите накладную для всех строк заказа на покупку в контейнере отгрузки. При использовании отгрузок с несколькими компаниями для каждой компании открывается новое диалоговое окно разноски накладной. |
| Отгрузить заказ на перемещение | Разнесите отгрузку заказа на перемещение для всех строк заказа на перемещение в контейнере отгрузки. В диалоговом окне отображаются только те строки в контейнере отгрузки, которые являются типом заказа на перемещение. |
| Получить заказ на перемещение | Разнесите поступление заказа на перемещение для всех строк заказа на перемещение в контейнере отгрузки. Диалоговое окно поступления является самым простым способом получения товаров в контейнере отгрузки или рейсе, и является одним из трех доступных вариантов. Можно также получать с помощью журналов прибытия или обработки мобильных устройств. |
| Создать журнал прибытия | Можно создать журнал прибытия для организаций, используя расширенные функции склада. Эти параметры определяют _Инициализировать количество_ (рекомендуется) и либо _Создание из товаров в пути_, либо _Создание из заказов на покупку_. Последние два параметра зависят от того, используется ли обработка товаров в пути. |
| Переименовать | Откройте диалоговое окно, в котором можно переименовать выбранный контейнер отгрузки. |
| Изменить шаблон пути | Измените шаблон пути. После изменения шаблона пути, возможно, придется выбрать **Поиск автозатрат** или вручную добавить затраты снова, так как затраты на отгрузку будут удалены. |
| Преобразовать в аренду | Преобразование выбранного контейнера отгрузки в арендный контейнер отгрузки. |

Экспресс-вкладка **Строки** в представлении **Строки** отображает подробные сведения о каждой полной или частичной строке заказа на покупку, которая включена в текущий контейнер отгрузки.

В следующей таблице описаны кнопки на панели операций экспресс-вкладки **Строки** в представлении **Строки**.

| Кнопка | описание |
|---|---|

| Запасы \> Отображение аналитик | Откройте диалоговое окно, в котором можно выбрать складские аналитики, которые отображаются для проводок, которые вы просматриваете. |

### <a name="information-on-the-lines-details-fasttab"></a>Сведения на экспресс-вкладке "Сведения строк"

Экспресс-вкладка **Сведения строк** в представлении **Строки** содержит более подробную информацию о строке заказа на покупку, которая сейчас выбрана на экспресс-вкладке **Строки**.
