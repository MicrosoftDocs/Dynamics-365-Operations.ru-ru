# <a name="landed-cost-entities-overview"></a>Обзор объектов стоимости на складе

[!include [banner](../includes/banner.md)]
[!INCLUDE [preview-banner](../includes/preview-banner.md)]
<!-- KFM: Preview until GA with 10.0.28 -->

Модуль **Стоимость на складе** помогает предприятиям оптимизировать входящие операции по отгрузке, предоставляя пользователям полный финансовый и логистический контроль над фрахтом импорта: от производителя до склада. Объекты данных транзакций для затрат на выгрузку позволяет внешним источникам данных (например службам экспедитора грузов) создавать поездку и затраты, а также обновлять записи отслеживания контейнеров.

Вот некоторые ограничения:

- Создание и обновление записей о поездке, грузовом контейнере и листах для существующих строк входящего заказа.
- Создание и обновление оценочной стоимости для каждой области стоимости.
- Создание и обновление записей мероприятий по грузовом контейнере для сбора даты начала, предполагаемой даты окончания и фактической даты окончания для каждого участка пути контейнера.
- Создание распределений по накладной затрат поставщика.
