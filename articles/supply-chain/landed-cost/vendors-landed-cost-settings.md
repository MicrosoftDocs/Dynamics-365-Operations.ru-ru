# <a name="vendor-settings-added-for-landed-cost"></a>Настройки поставщика, добавленные в стоимость на складе

[!include [banner](../../includes/banner.md)]

Когда включен модуль **Стоимость на складе** некоторые новые поля добавляются на существующую страницу **Поставщики**. Эти поля используются для настройки поставщиков, которые будут использоваться вместе с функциями стоимости на складе.

Чтобы настроить соответствующие поля, перейдите **Закупки и источники \> Поставщики \> Все поставщики**. Откройте существующего поставщика или создайте нового поставщика, а затем выберите экспресс-вкладку **Прочие подробные сведения**. Все новые поля, которые добавляет модуль **Стоимость на складе**, будут отображаться под заголовком **Рейсы**. В следующей таблице приводится описание этих полей.

| Поле | описание |
|---|---|
| Тип отгрузки | <p>Выбор роли поставщика в отношении стоимости на складе:</p><ul><li>**Нет** — у поставщика нет особой роли, которая относится к стоимости на складе. Это значение является значением по умолчанию, так как у большинства поставщиков, возможно, нет будет какой-либо особой роли.</li><li>**Транспортная компания** — поставщик является транспортной компанией. Поставщики с этим типом отгрузки доступны для выбора в поле **Транспортная компания** на странице **Рейсы**.</li><li>**Таможенный брокер** — поставщик является таможенным брокером. Поставщики с этим типом отгрузки доступны для выбора в поле **Таможенный брокер** на странице **Листы**.</li><li>**Агент** — поставщик является агентом. Поставщики с этим типом отгрузки доступны для выбора в поле **Агент** на страницах **Поставщики** и **Заказы на покупку**.</li></ul> |
| Группа типов затрат | Назначьте поставщика для группы типов затрат, чтобы выбрать [автоматические затраты](auto-cost-setup.md). |
| Порт отправления | Выбор порта происхождения для рейса. |
| Агент | Агент по умолчанию, когда покупки выполняются от поставщика. |
| Поставщик учета затрат на импорт | <p>Указывается, является ли поставщик поставщиком стоимости на складе.</p><p>**Совет.** Можно использовать это поле вместе с защитой на уровне записей для ограничения заказов на покупку, которые отображаются при создании рейса.</p> |
| Компания по отгрузке | Выбор транспортной компании по умолчанию, используемой при создании заказов на покупку для поставщика. |
| Поставщик услуг | Указывает, является ли поставщик поставщиком услуг. |
| Группа допустимых отклонений перепоставки/недопоставки | Выбор группы допустимых отклонений перепоставки/недопоставки для поставщика. |
