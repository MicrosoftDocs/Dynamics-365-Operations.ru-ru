# <a name="manage-folios"></a>Управление листами

[!include [banner](../../includes/banner.md)]

Лист часто определяется таможенными правилами. Он может состоять из товаров одного поставщика для одной организации или компании на отгрузку. Товары в листе могут быть в одном контейнере или распределены между несколькими контейнерами.

Чтобы открыть страницу **Все листы**, перейдите **Стоимость на складе \> Листы \> Все листы**. На этой странице отображается список всех текущих листов. Можно использовать кнопки на панели операций для создания, удаления и работы с листами. Выберите лист в списке, чтобы просмотреть его сведения на странице **Листы**.

## <a name="action-pane"></a>Панель операций

Панель операций на страницах **Все листы** и **Листы** содержит кнопки, позволяющие работать с выбранным листом. Каждая кнопка выполняет одно действие. Панель операций также включает вкладки, каждая из которых, в свою очередь, содержит набор соответствующих кнопок. За исключением оговоренных случаев, все кнопки и вкладки, описанные в следующих подразделах, доступны как в представлении списка (то есть на странице **Все листы**), так и в подробном представлении (то есть на странице **Листы**).

### <a name="buttons-that-appear-directly-on-the-action-pane"></a>Кнопки, появляющиеся непосредственно на панели операций

В следующей таблице описаны кнопки непосредственно на панели операций.

| Кнопка | описание |
|---|---|
| Новое | Создать лист. |
| Удаление | Удалить открытый или выбранный лист. |
| Затраты на поездку | Откройте страницу **Затраты на рейс**, на которой можно просмотреть и добавить затраты на уровне листа для всех товаров в рейсе. Когда затраты листа добавляются в рейс вручную, они автоматически добавляются на страницу запроса затрат и распределяются по всем товарам в соответствии с методом, указанным на странице **Затраты на рейс**. |

### <a name="buttons-on-the-manage-tab"></a>Кнопки на вкладке "Управление"

В следующей таблице описаны кнопки на панели операций вкладки **Управление**.

| Кнопка | описание |
|---|---|
| Разноска списка прихода | Разноска списка поступлений для всех строк заказа на покупку на листе. При использовании отгрузок с несколькими компаниями для каждой компании открывается новое диалоговое окно разноски списка поступлений. |
| Разноска поступления продуктов | Разнесите поступление продукта для всех строк заказа на покупку на листе. При использовании рейсов с несколькими компаниями для каждой компании открывается новое диалоговое окно получения продукта. |
| Разноска накладной | Разнесите накладную для всех строк заказа на покупку на листе. При использовании рейсов с несколькими компаниями для каждой компании открывается новое диалоговое окно разноски накладной. |
| Отгрузить заказ на перемещение | Разнесите заказ на перемещение для всех строк заказа на перемещение, которые относятся к текущему листу в соответствующей отгрузке. |
| Получить заказ на перемещение | Разнесите получение заказа на перемещение для всех строк заказа на перемещение, которые относятся к текущему листу в соответствующей отгрузке. |
| Получить товары в пути | Получите все строки заказа, которые находятся в пути на листе. |
| Документы получены | Обновите параметр **Полученные документы** на *Да*. Эту кнопку можно использовать для блокировки номенклатуры и/или строки покупки, чтобы дальнейшее обновление было невозможно. |
| Поиск автоматических затрат | Найдите соответствующие затраты на рейс. Если эти затраты уже были найдены или обновлены, появится следующее сообщение: "Обнаружены строки затрат без накладных. Перезаписать их?" Обратите внимание, что затраты на рейс, приложенные к листу и по которым были выставлены накладные, не будут перезаписаны. |
| Создать журнал прибытия | Создайте журнал прибытия для организаций, используя расширенные функции склада. Можно выбрать **Инициализировать количество** (рекомендуется), **Создание из товаров в пути** и/или **Создание из заказов на покупку**. Последний параметр зависит от того, используется ли обработка товаров в пути. |
| Начислить затраты | Выполните начисление затрат, где у типа затрат есть счет ГК, указанный для дебета. Эта кнопка обычно используется, когда запасы находятся в пути или когда товары были получены и по ним выставлены накладные. |

### <a name="buttons-on-the-general-tab"></a>Кнопки на вкладке "Общее"

В следующей таблице описаны кнопки на панели операций вкладки **Общее**.

| Кнопка | описание |
|---|---|
| Список прихода | Разноска списка поступлений для всех строк заказа на покупку на листе. При использовании рейсов с несколькими компаниями для каждой компании открывается новое диалоговое окно разноски списка поступлений. |
| Поступление продуктов | Просмотр записи о поступлении продукта, если она используется. |
| Прибытие номенклатуры | Просмотрите журнал прибытия номенклатуры, если он используется. |
| Запрос затрат | Откройте страницу запроса затрат для просмотра всех затрат рейса, включая контейнер отгрузки, лист и заказ на покупку. Можно настроить точное представление страницы, используя действие просмотра. На странице запроса затрат можно просмотреть любые области, а также номенклатуру и код типа затрат. Удаление этих номенклатур позволяет скорректировать страницу путем объединения затрат вместе. Эта возможность может быть полезной при использовании размеров и цветов. Можно изменять аналитики, отображаемые на странице. На странице **Затраты** отображаются только коды типов затрат, для которых в записи **Дб** на вкладке **Разноска** указано значение *Номенклатура*. |

## <a name="header-view"></a>вид заголовка

Чтобы открыть представление **Заголовок**, откройте лист, а затем выберите вкладку **Заголовок** в верхнем правом углу заголовка листа.

### <a name="settings-on-the-general-fasttab"></a>Настройки на экспресс-вкладке "Общее"

В следующей таблице описаны поля, доступные на экспресс-вкладке **Общее** в представлении **Заголовок** листа.

| Поле | описание |
|---|---|
| Лист | Наименование листа. Это имя создается автоматически при создании листа.|
| Поездка | Рейс, связанный с листом. |
| Таможенный брокер | Выберите таможенного брокера для листа. Таможенные брокеры определяются у поставщика. Они позволяют автоматически определять созданные затраты. |
| Авиационная грузовая накладная/транспортная накладная компании | Укажите путевой лист домашней авианакладной или транспортную накладную, которая применяется к листу. |
| Организация | Юридическое лицо (компания), связанное с листом. |
| Контрольный номер груза | Это поле используется в таможенных органах в некоторых странах или регионах. |
| Измерение | Это поле позволяет указать измерение в модуле **Стоимость на складе**. Измерения часто используются организациями, которые не знают отдельный объем или вес товаров, но требуют более точного распределения по сравнению с суммой или количеством. Экспедитор груза предоставит сведения о весе или мере объема, их можно использовать на уровне номенклатуры или заказа на покупку. Он может быть автоматически обновлен, если выбран параметр, или его можно ввести вручную. |
| Единица измерения | Единица измерения, используемая для указанного измерения. |
| Количество коробок | Количестве коробок в листе. Это поле может быть автоматически обновлено в зависимости от выбора параметра. |
| Счет поставщика | Выберите поставщика, связанного с листом. Это поле предназначено только для информационных целей. Это не влияет на функциональность. |
| ФИО | Имя выбранного счета поставщика. |
| Примечания | Ввод дополнительной информации, связанной с листом. |
| Описание товаров | Выберите описание товаров, помогающего идентифицировать лист. Дополнительные сведения см. в разделе [Описание товаров](shipping-information-setup.md#description-of-goods). |
| Дата оценки | Это поле связано со страницей записи пошлины. В модуле **Стоимость на складе** будет использоваться таможенный валютный курс для даты, которая указана здесь. Значением по умолчанию является дата на странице записи пошлины. |
| ИД таможни | Ведите таможенный код. Этот код предоставляют таможенные органы в странах/регионах. |
| Код тарифа | Введите кода тарифа для связи с листом. Этот код обычно требуется (и определяется) страной/регионом, куда производится поставка. |

## <a name="lines-view"></a>Представление строк

Чтобы открыть представление **Строки**, откройте лист, а затем выберите вкладку **Строки** в верхнем правом углу заголовка листа.

### <a name="information-on-the-folio-fasttab"></a>Информация на экспресс-вкладка "Лист"

На экспресс-вкладке **Лист** в представлении **Строки** отображаются сведения о листе. Большая часть этих сведений также отображается в представлении **Заголовок**, как описано ранее в этом разделе.

### <a name="information-and-buttons-on-the-lines-fasttab"></a>Сведения и кнопки на экспресс-вкладке "Строки"

Экспресс-вкладка **Строки** в представлении **Строки** отображает подробные сведения о каждой полной или частичной строке заказа на покупку, которая включена в текущий лист.

В следующей таблице описаны кнопки на панели операций экспресс-вкладки **Строки** в представлении **Строки**.

| Кнопка | описание |
|---|---|
| Удалить | Удаление выбранной строки заказа на покупку из рейса. |
| Запасы \> Проводки | Просмотр проводок по запасам для выбранной строки заказа на покупку. Обратите внимание, что при использовании товаров в пути также отображаются исходный заказ и заказы товаров в пути. |
| Запасы \> Отображение аналитик | Откройте диалоговое окно, в котором можно выбрать складские аналитики, которые отображаются для проводок, которые вы просматриваете. |
| Обновить   | Обновление информации, которая относится к сумме строки, весу или объему выбранной строки заказа на покупку. |

### <a name="information-on-the-lines-details-fasttab"></a>Сведения на экспресс-вкладке "Сведения строк"

Экспресс-вкладка **Сведения строк** в представлении **Строки** содержит более подробную информацию о строке заказа на покупку, которая сейчас выбрана на экспресс-вкладке **Строки**.
