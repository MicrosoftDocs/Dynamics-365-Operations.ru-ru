# <a name="landed-cost-inquiries"></a>Запросы стоимости на складе

[!include [banner](../../includes/banner.md)]

## <a name="voyage-line-inquiries"></a>Запросы строки рейса

В запросе **Строки рейса** отображаются все строки рейса по мере того, как они относятся к запасам. Этот запрос можно использовать в качестве фильтра, помогающего найти номенклатуру, заказ на покупку или другие полезные данные. Он также может использоваться для определения ожидаемой даты поставки номенклатуры, которая может быть в одном или нескольких рейсах. Таким образом, это может помочь в управлении получением ожидаемых запасов.

Чтобы открыть этот запрос, перейдите **Стоимость на складе \> Запросы \> Строки рейса**.

### <a name="overview-tab"></a>Вкладка «Обзор»

На вкладке **Обзор** страницы запроса **Строки выполнения** имеется сетка, в которой отображаются основные сведения о каждой соответствующей строке рейса. В следующей таблице столбцы в сетке.

| Столбец | описание |
|---|---|
| **Код номенклатуры** | Номенклатура для строки рейса. |
| **Справка** | Тип заказа (заказ на покупку или заказ на перемещение). |
| **Число** | Номер заказа на покупку или номер заказа на перемещение. |
| **Лист** | Лист, связанный со строкой рейса. |
| **Грузовой контейнер** | Контейнер отгрузки, связанный со строкой рейса. |
| **Поездка** | Рейс, связанный со строкой рейса. |
| **Количество** | Количество номенклатуры для строки рейса. |
| (Другие аналитики) | Можно отобразить столбцы для дополнительных аналитик. Такие аналитики включают номер партии, статус запасов и склад. В области действий выберите **Запасы \> Отображение аналитик**, чтобы открыть диалоговое окно, в котором можно выбрать аналитики для включения в список. |

### <a name="general-tab"></a>Вкладка "Общие"

Перейдите на вкладку **Общие**, чтобы просмотреть дополнительные сведения о строке, выбранной на вкладке **Обзор**.

### <a name="dimensions-tab"></a>Вкладка «Аналитики»

Выберите вкладку **Аналитики** для просмотра значений для всех доступных складских аналитик для строки, выбранной на вкладке **Обзор** независимо от того, какие аналитики выбраны для отображения на этой вкладке.

## <a name="cost-estimate-inquiries"></a>Запросы оценки затрат

Каждый раз при создании оценки затрат, оценка добавляется на страницу запроса **Оценки затрат**. Полные сведения о создании, просмотре и работе с оценками затрат (включая оценки на странице запроса) см. в разделе [Оценка и управление стоимостью на складе](estimate-manage-landed-costs.md).

## <a name="item-tracking"></a>Отслеживание номенклатуры

Страница **Отслеживание номенклатуры** используется для просмотра строк открытых заказов на покупку и их текущего статуса. Строки, которые не должны быть связаны с рейсом, указываются здесь. Однако, если номенклатура связана с рейсом, в строке отслеживания номенклатуры отображается код рейса.

Чтобы открыть эту страницу, перейдите **Стоимость на складе \> Запросы \> Отслеживание \> Отслеживание номенклатуры**.

Так как ваша система, вероятно, содержит большое количество строк заказа на покупку, страница **Отслеживание номенклатуры** изначально не содержит записей. Начните с использования полей фильтра в верхней части страницы, чтобы определить набор строк заказа на покупку, который вы ищете. Затем в области действий выберите **Обновить** для создания списка. Звездочку (\*) можно использовать в качестве подстановочного знака в любом из полей фильтра. Например, чтобы найти все строки заказа на покупку для номенклатур, в имени которых содержится слово "DVD", задайте для поля **Тип** значение **Название продукта** и введите *\*DVD\** в поле **Значение поля**.

> [!NOTE]
> В настоящее время недопоставки включают только заказы на продажу. Предложения по продажам не отображаются или считаются недопоставками.

В следующей таблице приводится описание столбцов сетки на странице **Отслеживание номенклатуры**.

| Столбец | описание |
|---|---|
| **Порт назначения** | Конечное место назначения. |
| **Подтверждено** | Подтвержденная дата строки заказа на покупку. |
| **Дата поставки** | Дата поставки строки заказа на покупку. |
| **Поездка** | Если строка связана с рейсом, код рейса. |
| **Контейнер отгрузки** | Если строка связана с контейнером отгрузки, код контейнера. |
| **Порт отгрузки** | Текущий порт на основе первого отрезка в форме отслеживания, где нет фактической даты, установленной для контейнера отгрузки, связанного со строкой заказа на покупку. |
| **Деятельность** | Текущее действие на основе первого отрезка в форме отслеживания, где нет фактической даты, установленной для контейнера отгрузки, связанного со строкой заказа на покупку. |
| **Предполагаемая конечная дата** | Дата, в которую ожидается прибытие рейса на приемку на складе назначения. |
| **ФИО** | Название поставщика. |
| **Статус рейса** | Статус отгрузки, который связан со строкой заказа на покупку. |
| **Код номенклатуры** | Номер номенклатуры для строки заказа на покупку. |
| **Внешний** | Название номенклатуры поставщика. |
| **Наименование продукта** | Название номенклатуры для строки заказа на покупку. |
| **Количество** | Количество по строке заказа на покупку для строки заказа на покупку. |
| **Единица измерения** | Единица измерения для строки заказа на покупку. |
| **Справка** | Тип заказа (заказ на покупку или заказ на перемещение) |
| **Номер ссылки** | Номер заказа на покупку или номер заказа на перемещение. |
| **Недопоставка** | Символ указывает, что имеются недопоставки продаж для номенклатуры. |
| (Другие аналитики) | Можно отобразить столбцы для дополнительных аналитик. Такие аналитики включают номер партии, статус запасов и склад. В области действий выберите **Отображение аналитик**, чтобы открыть диалоговое окно, в котором можно выбрать аналитики для включения в список. |

### <a name="related-information-about-backorders"></a>Связанные сведения о недопоставках

Чтобы просмотреть дополнительные сведения о недопоставках, выберите вкладку **Связанные сведения** в правой части страницы, а затем выберите **Недопоставки**. Чтобы просмотреть дополнительные сведения о конкретной недопоставке, выберите строку для нее, а затем щелкните ссылку **Дополнительно**.

## <a name="individual-shipping-container-tracking"></a>Отслеживание отдельного грузового контейнера

В запросе **Отслеживание отдельного контейнера отгрузки** есть фильтр, который позволяет найти контейнер отгрузки, а затем определить все строки рейса в этом контейнере.

Чтобы открыть этот запрос, перейдите **Стоимость на складе \> Запросы \> Отслеживание \> Отслеживание отдельного контейнера отгрузки**.

## <a name="multiple-shipping-container-tracking"></a>Отслеживание нескольких грузовых контейнеров

В запросе **Отслеживание нескольких контейнеров отгрузки** есть фильтр, который позволяет найти несколько контейнеров отгрузки, а затем определить все строки рейса в этих контейнерах.

Чтобы открыть этот запрос, перейдите **Стоимость на складе \> Запросы \> Отслеживание \> Отслеживание нескольких контейнеров отгрузки**.
