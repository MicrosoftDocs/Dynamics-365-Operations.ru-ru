# <a name="turn-on-the-landed-cost-module-for-your-system"></a>Включение модуля "Стоимость на складе" для вашей системы

[!include [banner](../includes/banner.md)]

Прежде чем использовать модуль "Стоимость на складе", он должен быть включен в системе. Администраторы могут использовать параметры [управления функциями](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) для проверки статуса функции и ее включения при необходимости. В рабочей области **Управление функциями** эта функция перечисляется следующими способами:

- **Модуль:** *Управление транспортировкой*
- **Название компонента:** *Стоимость на складе*
