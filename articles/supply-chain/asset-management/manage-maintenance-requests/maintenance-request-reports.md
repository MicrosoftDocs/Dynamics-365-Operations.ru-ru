# <a name="maintenance-request-reports"></a>Отчеты по запросам на обслуживание

[!include [banner](../../includes/banner.md)]

 

В «Управлении активами» можно создать два отчета, связанных с запросами на обслуживание. В одном отчете представлены подробные сведения, а в другом — список, который может быть использован для планирования и обработки результатов.

## <a name="create-a-maintenance-request-details-report"></a>Создание отчета по сведениям запроса на обслуживание

Отчет **по сведениям запроса на обслуживание** показывает различную информацию, связанную с запросами на обслуживание.

1. Выберите **Управление активами** \> **Отчеты** \> **Запросы на обслуживание** \> **Сведения о запросах на обслуживание**.
2. На экспресс-вкладке **Включаемые записи** FastTab можно выбрать конкретные запросы на обслуживание, чтобы включить их в отчет.
3. На экспресс-вкладке **Выполнять в фоновом режиме** можно настроить генерацию отчетов в виде пакетного задания, по мере необходимости.
4. Выберите **ОК**, чтобы создать отчет.

На следующем рисунке показан пример отчета **Сведения о запросах на обслуживание**.

![Отчет по сведениям запроса на обслуживание.](media/09-manage-maintenance-requests.png)

## <a name="create-a-maintenance-request-list-report"></a>Создание отчета по списку запроса на обслуживание

Отчет **Список запросов на обслуживание** показывает список всех запросов на обслуживание одного и того же типа.

1. Выберите **Управление активами** \> **Отчеты** \> **Запросы на обслуживание** \> **Список запросов на обслуживание**.
2. На экспресс-вкладке **Включаемые записи** можно сделать выбор для определения, какие запросы на обслуживание включаются в отчет.
3. На экспресс-вкладке **Выполнять в фоновом режиме** можно настроить генерацию отчетов в виде пакетного задания, по мере необходимости.
4. Выберите **ОК**, чтобы создать отчет.

На следующем рисунке показан пример отчета **Список запросов на обслуживание** для всех активных запросов на обслуживание.

![Отчеты по списку запросов на обслуживание.](media/10-manage-maintenance-requests.png)
