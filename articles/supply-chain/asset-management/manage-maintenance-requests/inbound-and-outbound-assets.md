# <a name="inbound-and-outbound-assets"></a>Входящие и исходящие активы

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

Если ваша компания выполняет ремонтные работы или задания обслуживания по активам, полученные из других местоположений или клиентов, «Управление активами» может отслеживать как входящие активы, которые находятся на пути к вашей компании, так и исходящие активы, которые возвращаются.

> [!NOTE]
> Если требуется использовать состояния входящего и исходящего жизненного цикла для управления активами, которые поступают и возвращаются, необходимо настроить состояния жизненного цикла и модели жизненного цикла запроса на обслуживание для поддержки этих действий. Дополнительные сведения о запросах новых поставщиков см. в разделе [Настройка запросов на обслуживание](../setup-for-maintenance-requests/requests.md).

Настройка «Управления активами» определяет, можно ли работать с входящими и исходящими активами.

## <a name="register-assets-as-inbound"></a>Регистрация активов в качестве входящих

1. Выберите **Управление активами** \> **Общее** \> **Запросы на обслуживание** \> **Активные запросы на обслуживание**.
2. Выберите запрос на обслуживание.
3. Выберите **Обновить состояние запроса на обслуживание**.
4. Выберите **Входящие** (или другое состояние жизненного цикла, созданное для входящих активов), а затем выберите **OK.**

![Рисунок 1](media/07-manage-maintenance-requests.png)

## <a name="register-inbound-assets-as-received"></a>Регистрация входящих активов в качестве полученных

1. Выберите **Управление активами** \> **Общее** \> **Входящие/Исходящие** \> **Входящие активы**.
2. Выберите актив или запрос на обслуживание.
3. ВЫберите **Получить активы**.
4. В поле **Получено** введите дату и время. Затем выберите **OK**. Запись удаляется со страницы списка **Входящие активы**.

![Рисунок 2](media/08-manage-maintenance-requests.png)

## <a name="register-assets-as-outbound"></a>Регистрация активов в качестве исходящих

После завершения задания обслуживания или ремонта актив можно зарегистрировать в качестве возвращенного.

1. Выберите **Управление активами** \> **Общее** \> **Запросы на обслуживание** \> **Активные запросы на обслуживание**.
2. Выберите запрос на обслуживание.
3. Выберите **Обновить состояние запроса на обслуживание**.
4. Выберите **Исходящие** (или другое состояние жизненного цикла, созданное для исходящих активов), а затем выберите **OK.**

## <a name="register-outbound-assets-as-delivered"></a>Регистрация исходящих активов в качестве отобранных

1. Выберите **Управление активами** \> **Общее** \> **Входящие/Исходящие** \> **Исходящие активы**.
2. Выберите актив или запрос на обслуживание.
3. Выберите **Доставить активы**.
4. В поле **Отобрано** введите дату и время. Затем выберите **OK**. Запись удаляется со страницы списка **Исходящие активы**.
