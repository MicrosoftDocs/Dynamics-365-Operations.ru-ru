# <a name="warranties-on-assets-and-asset-types"></a>Гарантии на основные средства и типы основных средств

[!include [banner](../../includes/banner.md)]

 


В этой статье объясняется, как настроить гарантии на активы и типы активов в модуле "Управление активами".

## <a name="set-up-a-warranty-on-an-asset-type"></a>Настройка гарантии на тип основных средств

1. Выберите **Управление активами** \> **Настройка** \> **Типы активов** \> **Типы активов**.
2. В левой области выберите тип актива, к которому прикрепляются гарантийное соглашение поставщика, затем выберите **Значения по умолчанию для типа активов**.
3. На экспресс-вкладке **Общее** в поле **Гарантия поставщика** выберите соглашение.

## <a name="set-up-a-warranty-on-an-asset"></a>Настройка гарантии на основное средство

1. Выберите **Управление активами** \> **Общие** \> **Активы** \> **Все активы**.
2. Выберите актив, затем выберите **Изменить**.
3. На экспресс-вкладке **Поставщик** в разделе **Гарантия поставщика** в поле **Гарантия** выберите гарантийное соглашение.
4. В полях **Начало гарантии** и **Окончание гарантии** выберите начальную и конечную даты.

    > [!IMPORTANT]
    > Если дата выбрана в поле **Начало гарантии** в заказе на работу, гарантия становится действительной для заказа на работу на эту дату. При создании заказа на работу в поле **Начало гарантии** автоматически задается дата создания. Однако дату можно изменить таким образом, чтобы она соответствовала, например, дате начала гарантийного соглашения.
    >
    > ![Страница заказов на работу.](media/02-warranty.png)

> [!NOTE]
> При создании заказа на работу для актива, охватываемого гарантией поставщика, если заказ на работу имеет ожидаемую дату начала в течение гарантийного периода, вы получите уведомление о гарантийном соглашении. Затем можно отменить заказ на работу, если требуется.
