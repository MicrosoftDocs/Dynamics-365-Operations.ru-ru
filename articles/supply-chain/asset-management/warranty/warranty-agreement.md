# <a name="warranty-agreements"></a>Гарантийные соглашения

[!include [banner](../../includes/banner.md)]

 


В управлении активами можно настроить условия гарантии, которые могут быть связаны с активом или с типом актива. Условия гарантии создаются за конкретный период. Гарантию можно настроить для обеспечения полного покрытия или частичного покрытия, а также можно настроить условия, которые относятся к часам, расходам и номенклатурам.

Первым шагом является создание любых гарантийных соглашений по поставщику для вашего оборудования. Затем соглашения о гарантии присоединяются к активам или типам активов. Гарантийные соглашения поставщика используются только в информационных целях. Если гарантия поставщика настроена для актива, можно просмотреть период покрытия гарантии для актива.

## <a name="create-a-warranty-agreement"></a>Создание гарантийного соглашения

Соглашение о гарантии может включать несколько строк соглашения для покрытия гарантии на рабочие часы, расходы и номенклатуры.

1. Выберите **Управление активами** \> **Настройка** \> **Активы** \> **Гарантия**.
2. Выберите **Создать**, чтобы создать продукт.
3. В поле **Гарантия** введите код гарантии. 
4. В поле **Имя** введите описание.

    На экспресс-вкладке **Сведения** в поле **Активы** отображается число активных активов, использующих соглашение о гарантии.

5. На экспресс-вкладке **Строки гарантии** выполните следующие шаги, чтобы добавить строки, которые должны быть включены в гарантийное соглашение:

    1. Выберите **Добавить строку**, чтобы добавить новое условие в гарантию. Порядковый номер строки автоматически вводится в поле **Строка**.
    2. В поле **Период** выберите тип периода гарантии.
    3. В поле **Интервал** введите число. Это поле определяет число периодов действия гарантии.
    4. В поле **Процент** введите процент покрытия для строки гарантии. Процентное значение указывает, сколько покрывает ваша компания.
