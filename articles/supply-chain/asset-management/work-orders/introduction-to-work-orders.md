# <a name="introduction-to-work-orders"></a>Введение в заказы на работу

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

Заказы на работу используются для управления, предоставления необходимой информации и регистрации потребления по заданиям обслуживания. Заказ на работу может содержать одно или несколько заданий заказов на работу, а также один или несколько активов, которые могут быть связаны с заказом на работу. Каждая строка заказа на работу определяет задание по обслуживанию, запланированное для актива.

Заказы на работу можно создавать автоматически или вручную:

- Автоматически с помощью формы **Составление графика планов обслуживания** для планов обслуживания на основе календаря, для которых задано значение "Автосоздание".  

- Автоматически с помощью формы **Составление графика циклов обслуживания** для циклов обслуживания, для которых задано значение "Автосоздание".  

- Создание из графика обслуживания, который может быть заданиями профилактического обслуживания или запросами на обслуживание.  

- Создание заказа на работу вручную.  

- Создание заказа на работу из раздела **Все запросы на обслуживание**, **Активные запросы на обслуживание** или **Запросы на обслуживание моего функционального местоположения**.

>[!NOTE]
>Задания заказа на работу, связанные с одним и тем же активом, связаны с одним и тем же кодом проекта.

## <a name="all-work-orders"></a>Все заказы на работу

Щелкните **Управление активами** > **Общее** > **Заказы на работу** > **Все заказы на работу**, чтобы открыть список. Пункт **Все заказы на работу** содержит список всех заказов на работу и отображает некоторые сведения, имеющие отношение к заказу на работу.

![Рисунок 1](media/01-work-orders.png)

- Щелкните **Управление активами** > **Общее** > **Заказы на работу** > **Активные заказы на работу** для просмотра списка всех активных заказов на работу.

- Щелкните **Управление активами** > **Общее** > **Заказы на работу** > **Задания по обслуживанию заказов на работу для моих функциональных местоположений** для просмотра списка заданий заказов на работу, содержащих активы, установленные в функциональных местоположениях, с которые вы связаны как сотрудник (настройка в разделе [Специалисты и группы специалистов по обслуживанию](../setup-for-objects/workers-and-worker-groups.md)).

- В списке **Все заказы на работу** (представление сетки) щелкните ссылку в столбце **Заказ на работу** для просмотра представления сведений о выбранной записи. Щелкните кнопку **Правка**, чтобы открыть для редактирования.  

- В представлении сведений отображается детальная информация, связанная с заказом на работу.  

- В представлении сведений выберите ссылку **Строки**, чтобы просмотреть сведения о заданиях заказа на работу, или выберите ссылку **Заголовок** для просмотра сведений о заказе на работу.  

- Вертикальная область **Связанные сведения** с правой стороны экрана содержит дополнительные сведения, имеющие отношение к заказу на работу. Разверните область, чтобы посмотреть связанную информацию для выбранного заказа на работу.
