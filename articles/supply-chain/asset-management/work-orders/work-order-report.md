# <a name="work-order-report"></a>Отчет по заказу на работу

[!include [banner](../../includes/banner.md)]


Можно создать отчет о заказе на работу, в котором отображаются подробные сведения об одном или более заказе на работу, которые выбраны.

1. Выберите **Управление активами** > **отчеты** > **Заказы на работу** > **Отчет по заказу на работу**.

2. Для определения сведений, включаемых в отчет, выберите значение "Да"/"Нет" на переключателях при необходимости.  
    - В разделе **Параметры печати** можно выбрать, следует ли включать вложения из настройки связанного типа задания на обслуживание.

3. На экспресс-вкладке **Записи для добавления** можно выполнить фильтрацию содержимого отчета по полю **Заказ на работу**.

4. На экспресс-вкладке **Выполнять в фоновом режиме** можно настроить генерацию отчета по заказу на работу в виде пакетного задания, по мере необходимости.

5. Выберите **ОК**, чтобы создать отчет.
