[!include [banner](../../includes/banner.md)]


<span data-ttu-id="40554-104">Можно создать отчет о заказе на работу, в котором отображаются подробные сведения об одном или более заказе на работу, которые выбраны.</span><span class="sxs-lookup"><span data-stu-id="40554-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="40554-105">Выберите **Управление активами** > **отчеты** > **Заказы на работу** > **Отчет по заказу на работу**.</span><span class="sxs-lookup"><span data-stu-id="40554-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="40554-106">Для определения сведений, включаемых в отчет, выберите значение "Да"/"Нет" на переключателях при необходимости.</span><span class="sxs-lookup"><span data-stu-id="40554-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>  
    - <span data-ttu-id="40554-107">В разделе **Параметры печати** можно выбрать, следует ли включать вложения из настройки связанного типа задания на обслуживание.</span><span class="sxs-lookup"><span data-stu-id="40554-107">In the **Print settings** section, you can select whether to include attachments from the setup of the related maintenance job type.</span></span>

3. <span data-ttu-id="40554-108">На экспресс-вкладке **Записи для добавления** можно выполнить фильтрацию содержимого отчета по полю **Заказ на работу**.</span><span class="sxs-lookup"><span data-stu-id="40554-108">On the **Records to include** FastTab, you can filter the contents of the report by **Work order**.</span></span>

4. <span data-ttu-id="40554-109">На экспресс-вкладке **Выполнять в фоновом режиме** можно настроить генерацию отчета по заказу на работу в виде пакетного задания, по мере необходимости.</span><span class="sxs-lookup"><span data-stu-id="40554-109">On the **Run in the background** FastTab, you can set up generation of the work order report as a batch job, as you require.</span></span>
