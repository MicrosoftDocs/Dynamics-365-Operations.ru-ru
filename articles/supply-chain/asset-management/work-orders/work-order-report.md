# <a name="work-order-report"></a>Отчет по заказу на работу


[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]


Можно создать отчет о заказе на работу, в котором отображаются подробные сведения о заказе на работу. Можно выбрать один или несколько заказов на работу, которые будут отображаться в отчете.

1. Щелкните **Управление активами** > **Отчеты** > **Заказы на работу** > **Отчет по заказу на работу**.

2. Для определения сведений, включаемых в отчет, выберите значение "Да"/"Нет" на переключателях и введите нужные значения полей.  
  a. В разделе **Параметры печати** можно выбрать, следует ли включать в печать вложения из соответствующей настройки типа задания на обслуживание.

3. На экспресс-вкладке **Записи для добавления** можно выполнить фильтрацию содержимого отчета по полю **Заказ на работу**.

4. При необходимости можно настроить создание отчета о заказе на работу как пакетное задание, заполнив поля на экспресс-вкладке **Выполнять в фоновом режиме**.

5. Щелкните **ОК**, чтобы создать отчет.

Далее вы видите пример того, как можно настроить параметры, и связанный отчет по заказу на работу.
