<span data-ttu-id="8038b-104">В разделе **Активные задания обслуживания для заказа на работу** можно просмотреть обзор заказов на работу в отношении того, сколько заказов на обслуживание было создано для определенных активов, типов активов, изготовителей, моделей, типов заданий обслуживания и так далее.</span><span class="sxs-lookup"><span data-stu-id="8038b-104">In **Active work order maintenance jobs**, you can get an overview of work orders in relation to how many work orders have been created on specific assets, asset types, manufacturers, models, maintenance job types, and so on.</span></span> <span data-ttu-id="8038b-105">Если выбрать задание заказа на работу и щелкнуть **Правка**, открывается связанный заказ на работу.</span><span class="sxs-lookup"><span data-stu-id="8038b-105">If you select a work order job and click **Edit**, you open the related work order.</span></span> <span data-ttu-id="8038b-106">Если выбрать задание и щелкнуть одну из кнопок на вкладке **Задания обслуживания по заказу на работу**, будут отображены данные о заказе на работу, с которым связано задание заказа на работу.</span><span class="sxs-lookup"><span data-stu-id="8038b-106">If you select a job and click one of the buttons on the **Work order maintenance jobs** tab, you see data for the work order to which the work order job is related.</span></span>

<span data-ttu-id="8038b-107">Щелкните **Управление активами** > **Общее** > **Заказы на работу** > **Активные задания обслуживания для заказа на работу**, чтобы открыть список.</span><span class="sxs-lookup"><span data-stu-id="8038b-107">Click **Asset management** > **Common** > **Work orders** > **Active work order maintenance jobs** to open the list.</span></span> <span data-ttu-id="8038b-108">Список содержит все активные задания заказа на работу и отображает некоторые сведения, имеющие отношение к заказу на работу или заданию заказа на работу.</span><span class="sxs-lookup"><span data-stu-id="8038b-108">The list contains all active work order jobs and displays some of the information related to the work order or work order job.</span></span>

<span data-ttu-id="8038b-109">В столбце **%** число означает степень завершения заказа на работу в процентах.</span><span class="sxs-lookup"><span data-stu-id="8038b-109">In the **%** column, a number indicates completion of the work order in percent.</span></span> <span data-ttu-id="8038b-110">Степень завершения основана на двух расчетах: разнесенные часы в сравнении с прогнозируемыми часам, и количество завершенных контрольных списков обслуживания.</span><span class="sxs-lookup"><span data-stu-id="8038b-110">Completion is based on two calculations: Posted hours compared to forecasted hours, and number of maintenance checklists completed.</span></span>
