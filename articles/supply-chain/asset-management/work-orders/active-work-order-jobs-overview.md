<span data-ttu-id="46149-104">На странице списка **Активные задания обслуживания для заказа на работу** можно просмотреть обзор заказов на работу, с точки зрения количества заказов на работу, созданных для определенных активов, типов активов, изготовителей, моделей, типов заданий обслуживания и так далее.</span><span class="sxs-lookup"><span data-stu-id="46149-104">On the **Active work order maintenance jobs** list page, you can get an overview of work orders, in terms of the number of work orders that have been created on specific assets, asset types, manufacturers, models, maintenance job types, and so on.</span></span> <span data-ttu-id="46149-105">Выбирая задание заказа на работу и затем выбрав **Правка**, открывается связанный заказ на работу.</span><span class="sxs-lookup"><span data-stu-id="46149-105">By selecting a work order job and then selecting **Edit**, you can open the related work order.</span></span> <span data-ttu-id="46149-106">Если выбрать задание заказа на работу, а затем выбрать одну из кнопок на вкладке **Задания обслуживания для заказа на работу** в области действий, можно просмотреть данные для заказа на работу, с которым связано задание в заказе на работу.</span><span class="sxs-lookup"><span data-stu-id="46149-106">By selecting a work order job and then selecting one of the buttons on the **Work order maintenance jobs** tab of the Action Pane, you can view data for the work order that the work order job is related to.</span></span>

<span data-ttu-id="46149-107">Выберите **Управление активами** > **Общее** > **Заказы на работу** > **Активные задания обслуживания для заказа на работу**, чтобы открыть страницу списка.</span><span class="sxs-lookup"><span data-stu-id="46149-107">To open the list page, select **Asset management** > **Common** > **Work orders** > **Active work order maintenance jobs**.</span></span> <span data-ttu-id="46149-108">Страница содержит все активные задания заказа на работу, а также отображает некоторые сведения, имеющие отношение к заказу на работу или заданию заказа на работу.</span><span class="sxs-lookup"><span data-stu-id="46149-108">The page lists all active work order jobs, and it also shows some of the information that is related to the work order or work order job.</span></span>

<span data-ttu-id="46149-109">В столбце **%** число показывает степень завершения заказа на работу в процентах.</span><span class="sxs-lookup"><span data-stu-id="46149-109">In the **%** column, a number shows completion of the work order as a percentage.</span></span> <span data-ttu-id="46149-110">Степень завершения основана на двух расчетах: разнесенные часы в сравнении с прогнозируемыми часам, и количество завершенных контрольных списков обслуживания.</span><span class="sxs-lookup"><span data-stu-id="46149-110">Completion is based on two calculations: posted hours compared to forecasted hours, and the number of maintenance checklists that have been completed.</span></span>
