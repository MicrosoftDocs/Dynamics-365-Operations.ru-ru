# <a name="active-work-order-maintenance-jobs-overview"></a>Обзор активных заданий обслуживания для заказа на работу

[!include [banner](../../includes/banner.md)]



На странице списка **Активные задания обслуживания для заказа на работу** можно просмотреть обзор заказов на работу, с точки зрения количества заказов на работу, созданных для определенных активов, типов активов, изготовителей, моделей, типов заданий обслуживания и так далее. Выбирая задание заказа на работу и затем выбрав **Правка**, открывается связанный заказ на работу. Если выбрать задание заказа на работу, а затем выбрать одну из кнопок на вкладке **Задания обслуживания для заказа на работу** в области действий, можно просмотреть данные для заказа на работу, с которым связано задание в заказе на работу.

Выберите **Управление активами** > **Общее** > **Заказы на работу** > **Активные задания обслуживания для заказа на работу**, чтобы открыть страницу списка. Страница содержит все активные задания заказа на работу, а также отображает некоторые сведения, имеющие отношение к заказу на работу или заданию заказа на работу.

В столбце **%** число показывает степень завершения заказа на работу в процентах. Степень завершения основана на двух расчетах: разнесенные часы в сравнении с прогнозируемыми часам, и количество завершенных контрольных списков обслуживания.
