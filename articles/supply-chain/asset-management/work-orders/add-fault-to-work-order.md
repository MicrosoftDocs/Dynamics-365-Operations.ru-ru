# <a name="add-fault-to-work-order"></a>Добавление ошибки в заказ на работу

[!include [banner](../../includes/banner.md)]



Можно добавить неисправности, настроенные в конструкторе неисправностей, в заказ на работу. Одна или несколько записей о сбоях должны быть связаны с типами активов, которые используются для актива, выбранного в заказе на работу. Дополнительные сведения о настройке см. в разделе [Управление ошибками](../setup-for-work-orders/fault-management.md).

1. Выберите **Управление активами** > **Общее** > **Заказы на работу** > **Все заказы на работу** или **Активные заказы на работу**.

2. Выберите заказ на работу, для которого необходимо выполнить регистрацию неисправности, а затем в области действий на вкладке **заказ на работу** в группе **Актив** выберите **сбой актива**.

3. На экспресс-вкладке **Признаки** выберите **Добавить строку**. Последовательной номер ошибки автоматически вводится в поле **Ошибка**.

4. Выберите соответствующий симптом в поле **Симптом неисправности**.

5. В полях **Область неисправности** и **тип неисправности** выберите соответствующие значения.

6. В поле **Дата неисправности** автоматически вставляется текущая дата. Можно выбрать другую дату, при необходимости.

7. На экспресс-вкладке **Причины для выбранного симптома** добавьте строку, описывающую причину проблемы.

8. На экспресс-вкладке **Меры устранения для выбранного симптома** добавьте строку, описывающую возможный способ устранения проблемы.

9. Нажмите **Сохранить**.

## <a name="view-asset-faults"></a>Просмотр неисправностей активов

В списке **Ошибки по активам** можно получить обзор всех ошибок, зарегистрированных для активов.

На странице списка **Ошибки по активам** можно получить обзор всех неисправностей, зарегистрированных для активов. Выберите **Управление активами** > **Запросы** > **Ошибка актива** > **Ошибки активов**, чтобы открыть страницу.


## <a name="print-asset-fault-report"></a>Печать отчет о неисправностях активов

На странице списка **Все активы** можно напечатать отчет об ошибках активов, отображающий все регистрации ошибок, а также графический обзор статистики ошибок.

1. Выберите **Управление активами** > **Общие** > **Активы** > **Все активы**.

2. Выбрать актив для печати отчета о неисправностях.

3. В области действий на вкладке **Общие** в группе **Отчеты** выберите **Ошибка актива**.

4. Введите указанный период или выберите типа ошибки.

5. Выберите **OK**, чтобы напечатать отчет.

>[!NOTE]
>Чтобы напечатать отчет об ошибке для нескольких активов или типов активов, выберите **Управление активами** > **Отчеты** > **Активы** > **Ошибка актива**.
