# <a name="add-fault-to-work-order"></a>Добавление ошибки в заказ на работу

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]


Можно добавить неисправности, настроенные в конструкторе неисправностей, в заказ на работу. Актив, выбранный в заказе на работу, должен содержать типы активов, к которым привязаны одна или несколько записей о сбоях. Дополнительные сведения о настройке см. в разделе [Управление ошибками](../setup-for-work-orders/fault-management.md).

1. Щелкните **Управление активами** > **Общее** > **Заказы на работу** > **Все заказы на работу** или **Активные заказы на работу**.

2. В списке выберите заказ на работу, для которого необходимо выполнить регистрацию неисправности, и выберите пункт **Ошибка актива**.

3. На экспресс-вкладке **Признаки** щелкните **Добавить строку**. Последовательной номер ошибки автоматически вставляется в поле **Ошибка**.

4. Выберите соответствующий признак в поле **Признак неисправности**.

5. Выберите **Область неисправности** и **Тип неисправности**.

6. В поле **Дата неисправности** автоматически вставляется текущая дата. Если необходимо, можно выбрать другую дату.

7. На экспресс-вкладке **Причины для выбранного признака** добавьте строку, описывающую причину проблемы.

8. На экспресс-вкладке **Меры устранения для выбранного признака** добавьте строку, описывающую возможный способ устранения проблемы.

9. Нажмите кнопку **Сохранить**.

## <a name="view-asset-faults"></a>Просмотр неисправностей активов

В списке **Ошибки по активам** можно получить обзор всех ошибок, зарегистрированных для активов.

Щелкните **Управление активами** > **Запросы** > **Ошибка актива** > **Ошибки активов**, чтобы открыть список.


## <a name="print-asset-fault-report"></a>Печать отчет о неисправностях активов

На странице списка **Все активы** можно напечатать отчет об ошибках активов, отображающий все регистрации ошибок, а также графический обзор статистики ошибок.

1. Щелкните **Управление активами** > **Общие** > **Активы** > **Все активы**.

2. В списке **Активы** выберите актив, для которого требуется распечатать отчет о неисправностях.

3. На вкладке **Общие** > раздел **Отчеты** щелкните **Неисправность актива**.

4. Вставьте указанный период или выберите типа ошибки.

5. Щелкните **OK**, чтобы напечатать отчет.

>[!NOTE]
>Можно также напечатать отчет об ошибке для нескольких активов или типов активов, щелкнув **Управление активами** > **Отчеты** > **Активы** > **Неисправность актива**.
