# <a name="maintenance-request-types"></a>Типы запросов на обслуживание

[!include [banner](../../includes/banner.md)]

 

Типы запросов на обслуживание используются для классификации запросов на обслуживание. Например, могут быть типы запросов на обслуживание, связанные с профилактическим обслуживанием и корректирующим обслуживанием. Или у вас может быть специальный тип запроса на обслуживание, который используется для управления ремонтом активов (ремонт в хранилище).

Тип запроса на обслуживание определяет связь с группой состояния жизненного цикла запроса на обслуживание (модель жизненного цикла обслуживания). Модели жизненного цикла запроса на обслуживание определяют состояния жизненного цикла, которые могут быть установлены для запроса на обслуживание. (Примеры состояния жизненного цикла запроса на обслуживание включают **Создано**, **Активно** и **Завершено**).

1. Выберите **Управление активами** \> **Настройка** \> **Запросы на обслуживание** \> **Типы запросов на обслуживание**.
2. Выберите **Создать** для создания типа запроса на обслуживание.
3. В поле **Тип запроса обслуживания** введите идентификатор для типа запроса на обслуживание.
4. В поле **Имя** введите имя.
5. На экспресс-вкладке **Общее** в поле **Модель жизненного цикла запроса на обслуживание** выберите модель жизненного цикла запроса на обслуживание.
6. В поле **Тип заказа на выполнение работ** выберите тип заказа на работу. При преобразовании запроса на обслуживание в заказ на работу заказ автоматически получает тип заказа на работу, связанный с типом запроса на обслуживание.
