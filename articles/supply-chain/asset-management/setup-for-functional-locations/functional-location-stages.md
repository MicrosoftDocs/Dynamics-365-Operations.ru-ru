## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="a6d64-108">Настройка состояний жизненного цикла функционального местоположения</span><span class="sxs-lookup"><span data-stu-id="a6d64-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="a6d64-109">Выберите **Управление активами** > **Настройка** > **Функциональные местоположения** > **Состояния жизненного цикла**.</span><span class="sxs-lookup"><span data-stu-id="a6d64-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="a6d64-110">Выберите **Создать**, чтобы создать новое состояние функционального местоположения.</span><span class="sxs-lookup"><span data-stu-id="a6d64-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="a6d64-111">Вставьте идентификатор в поле **Состояния жизненного цикла** и имя для состояния функционального местоположения в поле **Имя**.</span><span class="sxs-lookup"><span data-stu-id="a6d64-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="a6d64-112">В поле **Модели жизненного цикла** можно увидеть количество моделей жизненного цикла функционального местоположения, использующих состояние функционального местоположения.</span><span class="sxs-lookup"><span data-stu-id="a6d64-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>
4. <span data-ttu-id="a6d64-113">На экспресс-вкладке **Общее** выберите «Да» на кнопке-переключателе **Активый**, если функциональное местоположение должно быть активным в этом состоянии.</span><span class="sxs-lookup"><span data-stu-id="a6d64-113">On the **General** FastTab, select "Yes" on the **Active** toggle button if the functional location should be active at this state.</span></span>
5. <span data-ttu-id="a6d64-114">Выберите «Да» на кнопке-переключателе **Создать активы**, если требуется возможность автоматически создать актив с тем же именем, что и функциональное местоположение, и установить его в функциональном местоположении в этом состоянии.</span><span class="sxs-lookup"><span data-stu-id="a6d64-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="a6d64-115">Эта кнопка-переключатель относится к полю **Тип актива** на экспресс-вкладке **Общее** в форме **Типы функциональных местоположений** (**Управление активами** > **Настройка** > **Функциональные местоположения** > **Типы функциональных местоположений**).</span><span class="sxs-lookup"><span data-stu-id="a6d64-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="a6d64-116">Выберите «Да» на кнопке-переключателе **Переименовать местоположение**, если требуется возможность изменить имя функционального местоположения в этом состоянии.</span><span class="sxs-lookup"><span data-stu-id="a6d64-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="a6d64-117">Выберите «Да» на кнопке-переключателе **Создать вложенные местоположения**, если требуется возможность добавить новые вложенные функциональные местоположения в функциональное местоположение в этом состоянии.</span><span class="sxs-lookup"><span data-stu-id="a6d64-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="a6d64-118">Выберите «Да» на кнопке-переключателе **Установить активы**, если требуется возможность установить активы в функциональное местоположение в этом состоянии.</span><span class="sxs-lookup"><span data-stu-id="a6d64-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="a6d64-119">Выберите «Да» на кнопке-переключателе **Удалить функциональное местоположение**, если требуется возможность удалить функциональное местоположение в этом состоянии.</span><span class="sxs-lookup"><span data-stu-id="a6d64-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="a6d64-120">Выберите состояние актива в поле **Состояние жизненного цикла**, если требуется, чтобы состояние жизненного цикла актива для всех активов, установленных в функциональном местоположении, было автоматически обновлено в этом состоянии.</span><span class="sxs-lookup"><span data-stu-id="a6d64-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="a6d64-121">Пример: Если вы закроете функциональное местоположение и установите состояние жизненного цикла функционального местоположения на «Завершено», может потребоваться автоматически изменить состояние жизненного цикла активов, установленных в этом функциональном местоположении, на «Не используется».</span><span class="sxs-lookup"><span data-stu-id="a6d64-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="a6d64-122">Состояния жизненного цикла функционального местоположения, модели жизненного цикла и типы связаны и используются так же, как состояния жизненного цикла заказа на работу, модели жизненного цикла заказа на работу и типы заказа на работу.</span><span class="sxs-lookup"><span data-stu-id="a6d64-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="a6d64-123">Настройка моделей жизненного цикла функционального местоположения</span><span class="sxs-lookup"><span data-stu-id="a6d64-123">Set up functional location lifecycle models</span></span>

2. <span data-ttu-id="a6d64-128">Выберите **Создать**, чтобы создать новую модель жизненного цикла.</span><span class="sxs-lookup"><span data-stu-id="a6d64-128">Select **New** to create a new lifecycle model.</span></span>
3. <span data-ttu-id="a6d64-129">Вставьте идентификатор модели жизненного цикла в поле **Модель жизненного цикла** и имя для модели жизненного цикла в поле **Имя**.</span><span class="sxs-lookup"><span data-stu-id="a6d64-129">Insert the lifecycle model ID in the **Lifecycle model** field and a name for the lifecycle model in the **Name** field.</span></span> <span data-ttu-id="a6d64-130">В поле **Типы функционального местоположения** и **Состояния жизненного цикла** можно увидеть количество типов функционального местоположения, использующих модель жизненного цикла, и количество состояний, выбранных в модели жизненного цикла.</span><span class="sxs-lookup"><span data-stu-id="a6d64-130">In the **Functional location types** and **Lifecycle states** fields, you can see the number of functional location types that uses the lifecycle model and the number of states selected in the lifecycle model.</span></span>
4. <span data-ttu-id="a6d64-131">На экспресс-вкладке **Состояния жизненного цикла** выберите состояния, которые должны быть включены в модель жизненного цикла.</span><span class="sxs-lookup"><span data-stu-id="a6d64-131">On the **Lifecycle states** FastTab, select the states that should be included in the model.</span></span> <span data-ttu-id="a6d64-132">Для этого щелкните состояние в разделе **Осталось состояний жизненного цикла** и щелкните ![стрелка вперед](media/02-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="a6d64-132">This is done by clicking on a state in the **Lifecycle states remaining** section and clicking the ![forward arrow](media/02-setup-for-functional-locations.png) button.</span></span>
5. <span data-ttu-id="a6d64-133">Если вы хотите выбрать все доступные состояния для модели, нажмите кнопку ![Выбрать все доступные этапы](media/03-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="a6d64-133">If you want to select all the available states for a model, click the ![select all available stages](media/03-setup-for-functional-locations.png) button.</span></span> <span data-ttu-id="a6d64-134">Все состояния перемещаются в раздел **Выбранные состояния жизненного цикла**.</span><span class="sxs-lookup"><span data-stu-id="a6d64-134">All states are transferred to the **Lifecycle states selected** section.</span></span>

7. <span data-ttu-id="a6d64-136">Выберите **Обновления состояния жизненного цикла** для определения состояний жизненного цикла, которые могут следовать выбранноому состоянию.</span><span class="sxs-lookup"><span data-stu-id="a6d64-136">Select **Lifecycle state updates** to define which lifecycle states can follow a selected state.</span></span>
