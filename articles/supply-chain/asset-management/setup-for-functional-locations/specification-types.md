# <a name="maintenance-attribute-types"></a>Типы атрибутов обслуживания

[!include [banner](../../includes/banner.md)]

 

В этом разделе объясняется, как создавать типы атрибуты в «Управлении активами». Атрибуты используются для описания свойств различных элементов. Можно настроить атрибуты по следующим элементам:

- [Типы функциональных местоположений](../setup-for-functional-locations/functional-location-types.md)
- [Создание функциональных местоположений](../functional-locations/create-functional-locations.md)
- [Типы активов](../setup-for-objects/object-types.md)
- Активы

Атрибуты, которые можно настроить, варьируются в зависимости от элемента. Например, для функционального местоположения можно настроить атрибуты для конфигурации и физического размера местоположения. Для типа актива или актива можно настроить атрибуты для объема двигателя, энергопотребления и максимальной емкости нагрузки при различных условиях.

## <a name="create-attribute-types"></a>Создание типов атрибутов

Можно создавать собственные типа атрибутов. Кроме того, можно перенести аналитики продуктов на страницу **Типы атрибутов**.

1. Выберите **Управление активами** \> **Настройка** \> **Типы атрибутов**.
2. При первой настройке типов атрибута выберите **Создать аналитики продукта** для автоматической передачи стандартных аналитик продукта.
3. Выберите **Создать** для создания нового типа атрибута.
4. В поле **Тип атрибута** введите название для типа атрибута.
5. В поле **Описание** введите описание.
6. В поле **Единица измерения**, по мере необходимости, выберите соответствующую единицу измерения атрибута.
7. В поле **Тип данных** выберите тип данных для единицы измерения.
8. Если вы выбрали **Строка** в качестве типа данных, выполните следующие действия, чтобы создать значения для типа атрибута:

    1. Выберите тип атрибута, а затем выберите **Значения**.
    2. В поле **Значения атрибута** выберите **Создать**.
    3. В поле **Тип атрибута** выберите тип атрибута (аналитика).
    4. В поле **Значение** введите связанное значение.
    5. В поле **Описание** введите описание.
    6. Сохранение записи.
    7. Вернитесь на страницу **Типы атрибута**.

9. Сохранение записи.

    Поле **Типы функционального местоположения** показывает количество функциональных местоположений, которые используют тип атрибута. Поле **Типы актива** показывает количество типов актива, которые используют это.
