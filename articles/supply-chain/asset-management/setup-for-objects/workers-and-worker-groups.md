# <a name="maintenance-workers-and-worker-groups"></a><span data-ttu-id="62bc5-103">Специалисты и группы специалистов по обслуживанию</span><span class="sxs-lookup"><span data-stu-id="62bc5-103">Maintenance workers and worker groups</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="62bc5-104">В этом разделе описываются специалисты и группы специалистов по обслуживанию в «Управлении активами».</span><span class="sxs-lookup"><span data-stu-id="62bc5-104">This topic explains maintenance workers and worker groups in Asset Management.</span></span> <span data-ttu-id="62bc5-105">В «Управлении активами» можно подключить специалистов по обслуживанию к функциональным местоположениям.</span><span class="sxs-lookup"><span data-stu-id="62bc5-105">In Asset Management, you can connect maintenance workers to functional locations.</span></span> <span data-ttu-id="62bc5-106">(Для получения дополнительной информации о функциональных местоположениях см. раздел [Создание функциональных местоположений](../functional-locations/create-functional-locations.md).) ## <a name="create-workers"></a><span data-ttu-id="62bc5-110">Создание работников</span><span class="sxs-lookup"><span data-stu-id="62bc5-110">Create workers</span></span>

1. <span data-ttu-id="62bc5-111">Выберите **Управление активами** \> **Настройка** \> **Работники** \> **Работники**.</span><span class="sxs-lookup"><span data-stu-id="62bc5-111">Select **Asset management** \> **Setup** \> **Workers** \> **Workers**.</span></span>
2. <span data-ttu-id="62bc5-112">Выберите **Создать** чтобы добавить работника в список.</span><span class="sxs-lookup"><span data-stu-id="62bc5-112">Select **New** to add a worker to the list.</span></span>
3. <span data-ttu-id="62bc5-113">В поле **Работник** выберите работника.</span><span class="sxs-lookup"><span data-stu-id="62bc5-113">In the **Worker** field, select the worker.</span></span>
4. <span data-ttu-id="62bc5-114">Установите параметр **Активный** на **Да**, чтобы спланировать работника по заказу на работу.</span><span class="sxs-lookup"><span data-stu-id="62bc5-114">Set the **Active** option to **Yes** to schedule the worker on work orders.</span></span>

    <span data-ttu-id="62bc5-115">На экспресс-вкладке **Общее** поля **Ресурс** и **Описание** автоматически заполняются, если ресурс был выбран для работника.</span><span class="sxs-lookup"><span data-stu-id="62bc5-115">On the **General** FastTab, the **Resource** and **Description** fields are automatically filled in if a resource has been selected for the worker.</span></span> <span data-ttu-id="62bc5-116">Поле **Календарь** также автоматически заполняется при условии, что вы настроили работника в качестве ресурса и распределили календарь этому ресурсу на странице **Ресурсы** (**Администрирование организации** \> **Ресурсы** \> **Ресурсы**).</span><span class="sxs-lookup"><span data-stu-id="62bc5-116">The **Calendar** field is also automatically filled in, provided that you've set up the worker as a resource and allocated a calendar to that resource on the **Resources** page (**Organization administration** \> **Resources** \> **Resources**).</span></span>

5. <span data-ttu-id="62bc5-117">На экспресс-вкладке **Группы** выберите **Добавить**, а затем выберите группу специалистов по обслуживанию для работника.</span><span class="sxs-lookup"><span data-stu-id="62bc5-117">On the **Groups** FastTab, select **Add**, and then select a maintenance worker group for the worker.</span></span> <span data-ttu-id="62bc5-118">Работник может связан с несколькими группами.</span><span class="sxs-lookup"><span data-stu-id="62bc5-118">A worker can be affiliated with more than one group.</span></span>
6. <span data-ttu-id="62bc5-119">В стандартной настройке принадлежность работника к группе вступает в силу с даты добавления группы, и срок ее действия не ограничен.</span><span class="sxs-lookup"><span data-stu-id="62bc5-119">In the standard setup, a worker's affiliation with a group is effective from the date when you add the group, and it never expires.</span></span> <span data-ttu-id="62bc5-120">Эта дата отображается в поле **Действует с**.</span><span class="sxs-lookup"><span data-stu-id="62bc5-120">This date is shown in the **Effective** field.</span></span> <span data-ttu-id="62bc5-121">Чтобы увидеть поле **Действует с**, выберите **Просмотр** \> **Все**.</span><span class="sxs-lookup"><span data-stu-id="62bc5-121">To see the **Effective** field, select **View** \> **All**.</span></span> <span data-ttu-id="62bc5-122">Если принадлежность работника к группе должна быть ограничена определенным периодом, используйте поля **Действует с** и **Действует по** для определения периода.</span><span class="sxs-lookup"><span data-stu-id="62bc5-122">If the worker's affiliation with a group should be limited to a specific period, use the **Effective** and **Expiration** fields to define the period.</span></span>
7. <span data-ttu-id="62bc5-123">На экспресс-вкладке **Функциональные местоположения** выберите **Добавить**, а затем выберите функциональное местоположение для специалиста по обслуживанию.</span><span class="sxs-lookup"><span data-stu-id="62bc5-123">On the **Functional locations** FastTab, select **Add**, and then select a functional location for the maintenance worker.</span></span> <span data-ttu-id="62bc5-124">Также укажите, какое местоположение является основным функциональным местоположением для работника.</span><span class="sxs-lookup"><span data-stu-id="62bc5-124">Also specify which location is the primary functional location for the worker.</span></span>

    > [!NOTE]
    > <span data-ttu-id="62bc5-125">При добавлении функциональных местоположений работнику все активные активы, связанные с этими функциональными местоположениями, отображаются в различных пунктах меню, таких как **Мои активные активы** и **Мои активные функциональные местоположения**.</span><span class="sxs-lookup"><span data-stu-id="62bc5-125">When you add functional locations to a worker, all active assets that are related to those functional locations appear on various menu items, such as **My active assets** and **My active functional locations**.</span></span> <span data-ttu-id="62bc5-126">Они также отображаются в поисках активов, которые отображаются при создании нового актива, запроса на обслуживание или заказа на работу.</span><span class="sxs-lookup"><span data-stu-id="62bc5-126">They also appear in the asset lookups that are shown when you create a new asset, maintenance request, or work order.</span></span>

    <span data-ttu-id="62bc5-127">Поля на экспресс-вкладке **Подробно** показывают количество групп специалистов по обслуживанию и функциональных местоположений, с которыми связан выбранный специалист по обслуживанию.</span><span class="sxs-lookup"><span data-stu-id="62bc5-127">The fields on the **Details** FastTab show the number of maintenance worker groups and functional locations that the selected maintenance worker is related to.</span></span>
