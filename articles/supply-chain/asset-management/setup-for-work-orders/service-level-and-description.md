# <a name="service-level-and-description"></a>Уровень обслуживания и описание

[!include [banner](../../includes/banner.md)]

 

При создании заказа на работу может быть необходимо определить уровни обслуживания для него и добавить к нему общее описание. Можно создать уровни обслуживания для заказов на работу на странице **Уровни обслуживания для заказа на работу** и описания на странице **Описание заказа на работу**.

## <a name="create-a-service-level"></a>Создание уровня обслуживания

1. Выберите **Управление активами** \> **Настройка** \> **Заказы на работу** \> **Уровень обслуживания**.
2. Выберите **Создать**.
3. В поле **Уровень обслуживания** введите уровень обслуживания (например, номер).
4. В поле **Имя** введите имя.

    На экспресс-вкладке **Заказы на работу** можно определить ожидаемые даты и время начала и окончания. Поля на этой экспресс-вкладке определяют период, в течение которого заказы на работу должны быть начаты и завершены. Они используются для заказов на работу, созданных вручную, и заказов на работу, созданных на основе запросов обслуживания. 

5. В поле **День начала** введите количество дней, чтобы определить период, в течение которого должен начаться заказ на работу. Количество дней рассчитывается с даты создания заказа на работу. Например, если заказ на работу должен начаться сразу после его создания, введите **0**. Если заказ на работу должен начаться в течение недели после его создания, введите **7**.
6. Чтобы задать время начала для заказа на работу в дополнение к дате начала, установите для параметра **Задать время начала** значение **Да**. Затем введите время начала в поле **Время начала**. Если для параметра выбрано значение **Нет**, используется текущее время дня.
7. В поле **День завершения** введите количество дней, чтобы определить период, в течение которого заказ на работу должен быть завершен. Количество дней рассчитывается с даты начала заказа на работу. Например, если заказ на работу должен завершиться в течение одной недели с даты начала, введите **7**.
8. Чтобы задать время завершения для заказа на работу в дополнение к дате завершения, установите для параметра **Задать время завершения** значение **Да**. Затем введите время завершения в поле **Время завершения**. Если для параметра выбрано значение **Нет**, используется текущее время дня.
9. Нажмите **Сохранить**.

## <a name="create-a-description"></a>Создание описания

1. Выберите **Управление активами** \> **Настройка** \> **Заказы на работу** \> **Описания**.
2. Выберите **Создать**.
3. В поле **Описание** введите описание.
4. Нажмите **Сохранить**.
