# <a name="scheduled-work-order-maintenance-jobs"></a>Запланированные задания обслуживания заказа на работу

[!include [banner](../../includes/banner.md)]

 

Страница **Запланированные задания обслуживания для заказа на работу** отображает обзор заказов на работу, назначенных ресурсу. Заказы на работу с использованием типов ресурсов "Управление персоналом", "Инструмент" и "Станок" отображаются. Например, если специалист по обслуживанию сообщил, что заболел, вы можете использовать эту страницу, чтобы быстро найти заказы на работу, назначенные этому работнику, а затем выделить другого специалиста по обслуживанию для этого задания.

## <a name="view-scheduled-work-order-maintenance-jobs"></a>Просмотр запланированных заданий обслуживания для заказа на работу

1. Щелкните **Управление активами** > **Общее** > **Заказы на работу** > **Запланированные задания обслуживания для заказа на работу**. Отображается список всех заказов на работу, которые установлены в состояние жизненного цикла заказа на работу "Запланировано" или "Выполняется".

2. Можно отсортировать список, например, по специалистам по обслуживанию. Можно также использовать фильтр, чтобы ограничить список для показа заказов на работу, назначенных конкретному ресурсу или специалисту по обслуживанию.

3. Можно просмотреть примечания по заказу на работу и, если это необходимо, добавить новые примечания, выбрав задание заказа на работу и щелкнув **Примечания**.

4. Если требуется назначить одного специалиста по обслуживанию для заказа на работу, выберите заказ на работу и щелкните **Заказ на работу**.

5. Будет открыта страница **Заказ на работу**. Щелкните **Подготовить к отправке**, чтобы запланировать заказ на работу конкретному специалисту по обслуживанию.

>[!NOTE]
>Более подробную информацию о планировании нескольких заказов на работу или одного заказа на работу см. в разделах [Запланировать заказы на работу](../work-order-scheduling/schedule-work-orders.md) и [Отправка заказа на работу](../work-order-scheduling/dispatch-work-order.md).
