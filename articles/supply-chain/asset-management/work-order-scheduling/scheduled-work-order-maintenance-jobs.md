<span data-ttu-id="1c881-104">Страница **Запланированные задания обслуживания для заказа на работу** отображает обзор заказов на работу, назначенных ресурсу.</span><span class="sxs-lookup"><span data-stu-id="1c881-104">The **Scheduled work order maintenance jobs** page shows an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="1c881-105">Заказы на работу с использованием типов ресурсов "Управление персоналом", "Инструмент" и "Станок" отображаются.</span><span class="sxs-lookup"><span data-stu-id="1c881-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown.</span></span> <span data-ttu-id="1c881-106">Например, если специалист по обслуживанию сообщил, что заболел, вы можете использовать эту страницу, чтобы быстро найти заказы на работу, назначенные этому работнику, а затем выделить другого специалиста по обслуживанию для этого задания.</span><span class="sxs-lookup"><span data-stu-id="1c881-106">For example, if a maintenance worker calls in sick, you can use this page to quickly find work orders allocated to the worker, and then allocate another maintenance worker to the job.</span></span>

1. <span data-ttu-id="1c881-108">Щелкните **Управление активами** > **Общее** > **Заказы на работу** > **Запланированные задания обслуживания для заказа на работу**.</span><span class="sxs-lookup"><span data-stu-id="1c881-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="1c881-109">Отображается список всех заказов на работу, которые установлены в состояние жизненного цикла заказа на работу "Запланировано" или "Выполняется".</span><span class="sxs-lookup"><span data-stu-id="1c881-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="1c881-110">Можно отсортировать список, например, по специалистам по обслуживанию.</span><span class="sxs-lookup"><span data-stu-id="1c881-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="1c881-111">Можно также использовать фильтр, чтобы ограничить список для показа заказов на работу, назначенных конкретному ресурсу или специалисту по обслуживанию.</span><span class="sxs-lookup"><span data-stu-id="1c881-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="1c881-112">Можно просмотреть примечания по заказу на работу и, если это необходимо, добавить новые примечания, выбрав задание заказа на работу и щелкнув **Примечания**.</span><span class="sxs-lookup"><span data-stu-id="1c881-112">You can see notes on the work order and, if required, add new notes by selecting the work order job, and then click **Notes**.</span></span>

4. <span data-ttu-id="1c881-113">Если требуется назначить одного специалиста по обслуживанию для заказа на работу, выберите заказ на работу и щелкните **Заказ на работу**.</span><span class="sxs-lookup"><span data-stu-id="1c881-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>
