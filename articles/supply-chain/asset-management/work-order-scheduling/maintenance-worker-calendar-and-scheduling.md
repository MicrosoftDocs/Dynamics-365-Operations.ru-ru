# <a name="maintenance-worker-calendar-and-scheduling"></a>Календарь специалистов по обслуживанию и планирование

[!include [banner](../../includes/banner.md)]

 

При планировании заказов на работу создается график для специалистов по обслуживанию, инструментов и активов. Чтобы планировать специалистов по обслуживанию календарь должен быть настроен для каждого специалиста по обслуживанию. Специалисты по обслуживанию связаны с ресурсом, а календари рабочего времени настраиваются для ресурсов. Вы настраиваете ресурс и календарь в разделе **Управление активами** > **Настройка** > **Работники** > **Работники**, что описано в разделе [Специалисты и группы специалистов по обслуживанию](../setup-for-objects/workers-and-worker-groups.md).

На следующем снимке экрана показан пример специалиста по обслуживанию, который связан с ресурсом, использующим календарь рабочего времени "Производство".

![Рисунок 1](media/01-work-order-scheduling.png)

Настройка календаря для инструментов и активов не требуется в связи с планированием заказов на работу. Предполагается, что средства и активы доступны 24 часа в день для обслуживания.
