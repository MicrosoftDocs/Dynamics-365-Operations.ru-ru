# <a name="asset-kpis"></a>Ключевые показатели эффективности актива

[!include [banner](../../includes/banner.md)]

 

В управлении активами можно рассчитать различные ключевые показатели эффективности (КПЭ) для активов и типов активов. Ключевые показатели эффективности используются, чтобы получить обзор эффективности по активам в отношении, например, времени работы, времени простоя, времени ремонта и среднего времени наработки на отказ.

1. Щелкните **Управление активами** > **Запросы** > **Активы** > **Ключевые показатели эффективности актива**.

2. В диалоговом окне **Расчет ключевых показателей эффективности актива** выберите **Шкала времени**, которая будет использоваться в расчете, и период в полях **Начальная дата** и **Конечная дата**. 

3. На экспресс-вкладке **Записи для добавления** можно выбрать отдельные активы и типы активов, которые будут включены в расчет, если это необходимо.

4. Нажмите **ОК**, чтобы начать расчет.

5. На экспресс-вкладке **Обзор** результаты расчета отображаются в виде сетки. Каждый актив отображается в отдельной строке.

6. На экспресс-вкладке **Ключевые показатели эффективности для выбранной строки** отображаются расчеты для актива, выбранного на экспресс-вкладке **Обзор**. Значения ключевых показателей эффективности классифицируются по параметрам **Время**, **Доступность**, **Заказы на работу**, **Обслуживание**, **Неисправности**, **Простой из-за обслуживания** и **Затраты**.

В приведенной ниже таблице вы найдете описание полей на странице **Ключевые индикаторы эффективности актива**.

| Поле                   | Описание                                                                                                                                                                                                                                                                                           |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ОС                   | Идентификатор актива.                                                                                                                                                                                                                                                                                             |
| Общее время              | Общее время, настроенное в календаре, который используется в расчете. Если актив связан с ресурсом, используется связанный календарь ресурса. Если актив не связан с ресурсом, используется календарь, выбранный в поле **Стандартный календарь** в пункте **Параметры управления активами**. |
| Время работы                  | Общее время минус время простоя.                                                                                                                                                                                                                                                                            |

| Время ремонта             | Общее количество рабочих часов, потраченных на ремонтным заказам на работу.                                                                                                                                                                                                                                            |
| Доступность %          | Время работы, деленное на общее время и умноженное на 100.                                                                                                                                                                                                                                                   |
| Количество ошибок        | Количество причин неисправностей, зарегистрированных по симптомам неисправности актива в выбранном периоде.                                                                                                                                                                                                             |
| MTBF                    | Среднее время наработки на отказ, которое является общим временем, деленным на количество неисправностей, зарегистрированных для актива в выбранном периоде. Если количество неисправностей равно нулю, для MTBF устанавливается значение общего времени.                                                                                                                   |
| Частота ошибок               | 1, деленная на MTBF.                                                                                                                                                                                                                                                                                    |
| MRT                     | Среднее время ремонта, которое является временем ремонта, деленным на количество неисправностей, зарегистрированных для актива в выбранном периоде. Если количество неисправностей равно нулю, для MRT устанавливается значение времени ремонта.                                                                                                                           |
| Количество остановок         | Количество регистраций простоя из-за обслуживания, созданных по активу.                                                                                                                                                                                                                                     |
| MTBS                    | Среднее время между остановками, которое является общим временем, деленным на количество регистраций простоя из-за обслуживания. Если количество регистраций простоя из-за обслуживания равно нулю в выбранном периоде, значение MTBS устанавливается равным общему времени.                                                                                      |
| Профилактическая стоимость         | Затраты, разнесенные по активу, относящимися к типу затрат "Профилактические" в выбранном периоде. Типы затрат настраиваются для типов заказов на работу.                                                                                                                                                                       |
| Корректирующая стоимость         | Затраты, разнесенные по активу, относящимися к типу затрат "Корректирующие" в выбранном периоде. Типы затрат настраиваются для типов заказов на работу.                                                                                                                                                                       |
| Значение замены       | Значение, определенное для актива в качестве восстановительной стоимости.                                                                                                                                                                                                                                                  |
| Вид актива             | Указание типа актива, выбранного в активе.                                                                                                                                                                                                                                             |
| Производитель           | Указание производителя, выбранного в активе.                                                                                                                                                                                                                                                 |
| Модель                   | Указание модели производителя, выбранной в активе.                                                                                                                                                                                                                                           |
| с даты               | Дата начала расчета ключевых показателей эффективности. Если актив был создан после даты начала, выбранной для расчета, в этом поле отображается дата начала актива.                                                                                                                                  |
| До даты                 | Конечная дата расчета ключевых показателей эффективности. Если актив был зарегистрирован как неактивный до даты окончания, выбранной для расчета, в этом поле отображается дата, начиная с которой актив более не активен.                                                                                               |
| Шкала времени              | При вычислении ключевых показателей эффективности выберите используемую шкалу времени: часы, дни или недели.                                                                                                                                                                                                            |
| Доступность            | Время работы, деленное на общее время.                                                                                                                                                                                                                                                                         |
| Заказы на работу             | Общее количество заказов на работу, включенных в расчет КПЭ.                                                                                                                                                                                                                                          |
| Время заказа на работу         | Общее количество рабочих часов, потраченных на заказам на работу.                                                                                                                                                                                                                                               |
| Первичные заказы на работу     | Количество основных заказов на работу, включенных в расчет КПЭ.                                                                                                                                                                                                                                        |
| Вторичные заказы на работу   | Количество дополнительных заказов на работу, включенных в расчет КПЭ.                                                                                                                                                                                                                                      |
| Заказы на работу по обслуживанию | Количество заказов на работу по обслуживанию, включенных в расчет КПЭ. Заказ на работу по обслуживанию — это заказ на работу, в котором нет связанных причин неисправностей.                                                                                                                                                             |
| Время обслуживания        | Общее количество рабочих часов, потраченных на заказам на работу по обслуживанию.                                                                                                                                                                                                                                       |
| Заказы на работу по ремонту      | Количество заказов на работу по ремонту, включенных в расчет КПЭ. Заказ на работу по ремонту — это заказ на работу, с которым связанна причина неисправности.                                                                                                                                                                        |
| Надежность %           | Расчет основан на ожидаемом экспоненциальном росте регистраций неисправностей для актива, что означает, что со временем актив становится менее надежным из-за износа. Расчет этого ключевого показателя эффективности базируется на MTBF и общем времени.                                                            |
| MTPS                    | Среднее время остановок производства, которое является временем простоя из-за обслуживания, деленным на количество регистраций простоя из-за обслуживания. Если количество регистраций простоя из-за обслуживания равно нулю в выбранном периоде, значение MTPS устанавливается равным нулю.                                                                               |
| Общие затраты              | Общие затраты, разнесенные по активу в выбранном периоде.                                                                                                                                                                                                                                              |
| Стоимость инвестиций         | Затраты, разнесенные по активу, относящимися к типу затрат "Инвестиции" в выбранном периоде. Типы затрат настраиваются для типов заказов на работу.                                                                                                                                                                       |

- Имеется возможность выбирать несколько активов в разделе **Все активы** и нажать кнопку **КПЭ активов** на вкладке **Общее**. Затем нажмите кнопку **ОК** в диалоговом окне **Расчет ключевых показателей эффективности активов**, чтобы рассчитать ключевые показатели эффективности для выбранных активов.  
- Результаты вычисления ключевых показателей эффективности могут включать или не включать [регистрации простоя из-за обслуживания](../work-orders/maintenance-downtime.md), в зависимости от настройки и использования кодов причин простоя из-за обслуживания.
