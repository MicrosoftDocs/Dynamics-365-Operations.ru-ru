# <a name="asset-attribute-overview"></a>Обзор атрибутов актива

[!include [banner](../../includes/banner.md)]

 

В этом разделе содержится обзор атрибутов актива в «Управлении активами». Атрибуты активов являются свойствами, связанными с типом актива или активом. Если вы настроили [типы атрибута](../setup-for-functional-locations/specification-types.md) и использовали их по активам, можно получить обзор значений атрибутов, настроенных на активы. Этот обзор отображается на двух страницах в «Управлении активами»: **Обзор атрибутов актива** и **Атрибут актива**.

Страница **Обзор атрибута актива** показывает отдельную строку для каждого актива. Линия отображает все типы атрибутов, связанные с этим активом. Выполните следующие действия, чтобы открыть и использовать страницу **Обзор атрибута актива**.

1. Выберите **Управление активами** \> **Запросы** \> **Активы** \> **Обзор атрибута актива**.
2. В поле **Тип актива** выберите тип актива.
3. Нажмите **ОК**. Список показывает все активы, которые используют выбранный тип актива. Для каждого актива все связанные типы атрибута актива отображаются в одной строке.

Страница **Атрибут актива** показывает отдельную строку для каждого типа атрибута, который используется на активе. Чтобы открыть эту страницу, выберите **Управление активами** \> **Запросы** \> **Активы** \> **Атрибут актива**.
