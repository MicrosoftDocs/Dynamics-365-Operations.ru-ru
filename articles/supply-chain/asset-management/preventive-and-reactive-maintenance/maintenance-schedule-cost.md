# <a name="maintenance-schedule-cost"></a>Стоимость для графика обслуживания

[!include [banner](../../includes/banner.md)]

 

В модуле "Управление активами" можно рассчитать бюджетные затраты по строкам графика обслуживания. Это полезно, если требуется получить обзор ожидаемых затрат, например, затраты, связанные с запланированными заданиями профилактического обслуживания для следующего года. Расчеты основаны на существующих строках графика обслуживания с типами "Планы обслуживания", "Циклы обслуживания" и "Запросы на обслуживание".

1. Щелкните **Управление активами** > **Запросы** > **Активы** > **Затраты для графика обслуживания**.

2. В диалоговом окне **Затраты для графика обслуживания** можно выбрать **Набор финансовых аналитик**, если требуется просмотреть затраты, сгруппированные в финансовые аналитики.

>[!NOTE]
>Наборы финансовых аналитик настраиваются в разделе **Главная книга** > **План счетов** > **Аналитики** > **Наборы финансовых аналитик**.

3. Можно использовать поле **Уровень**, чтобы указать, насколько подробными должны быть строки графика обслуживания относительно функциональных местоположений. Например, если вставить число "1" в это поле, и имеется структура функциональных местоположений с несколькими уровнями, все строки графика обслуживания для функционального местоположения будут показаны на верхнем уровне, и поэтому часы по строке могут быть добавлены из функциональных местоположений, расположенных на более низком уровне. Если вставить число "0" в поле **Уровень**, будет выведен подробный результат, отображающий все строки графика обслуживания на всех уровнях функциональных местоположений, к которым они относятся.

4. Если требуется выполнить расчет для отдельных активов, щелкните **Фильтр** на экспресс-вкладке **Записи для добавления** и выберите соответствующие активы. Если необходимо, можно также указать значение **Ожидаемая дата начала** для расчета затрат или выбрать другое **Состояние** для расчета затрат.

5. Нажмите **ОК**, чтобы начать расчет затрат.

6. На вкладке **Затраты для графика обслуживания** в группах панели операций **Группировать по...** щелкните соответствующие кнопки, чтобы отобразить необходимый уровень детализации для расчета. Выбранные кнопки группы панели операций выделяются. Нажимайте кнопку, чтобы активировать или деактивировать ее.

7. Нажмите кнопку **Рассчитать затраты**, если необходимо сделать новый расчет затрат.
