# <a name="create-an-intercompany-plan"></a>Создание внутрихолдингового плана

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Эта процедура демонстрирует порядок создания внутрихолдингового плана. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="set-up-an-intercompany-planning-group"></a>Настройка группы внутрихолдингового планирования 
1. Перейдите к группам внутрихолдингового планирования.
    * "Сводное планирование" > "Настройка" > "Группы внутрихолдингового планирования"  
2. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле "Имя" по значению "10".
3. В списке пометьте выбранную строку.
4. Нажмите кнопку Удалить.
    * Это действие необходимо, чтобы ускорить выполнение внутрихолдингового планирования.   Внутрихолдинговое планирование будет выполнять сводное планирование во всех компаниях в группе планирования, начиная с наименьшей запланированной последовательности.  
5. Щелкните Да.
6. Закройте страницу.

## <a name="create-an-intercompany-plan"></a>Создание внутрихолдингового плана
1. Щелкните "Внутрихолдинговое сводное планирование".
    * Это в рабочей области сводного планирования.  
2. В поле "Группа внутрихолдингового планирования" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
3. В списке перейдите по ссылке в выбранной строке.
    * Выберите группу внутрихолдингового планирования 10.  
4. В поле "Количество внутрихолдинговых итераций планирования" введите "2".
    * Группа внутрихолдингового планирования 10 содержит двух участников. Чтобы распространить задержки из компании-источника (USMF) на компанию клиента (DEMF), потребуется выполнить внутрихолдинговое планирование в обеих компаниях два раза. Первая итерация распространяет спрос и определяет задержки в компании-источнике (USMF). Вторая итерация распространяет задержки из USMF на DEMF.  
5. В поле "Первая итерация" выберите вариант.
6. В поле "Первая итерация" выберите "Пересоздание".
7. В поле "Последующие итерации" выберите "Пересоздание".
8. В поле "Количество потоков" введите число.
    * Представляет собой количество параллельных потоков, которые будут использоваться для планирования.  
9. Нажмите кнопку "OК".

## <a name="view-the-result-of-the-plan"></a>Просмотр результата плана
1. В поле "План" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
2. В списке перейдите по ссылке в выбранной строке.
    * Нажмите на ссылку для StaticPlan. Необходимо находиться в компании USMF.  
3. Щелкните "Спланированные заказы".
