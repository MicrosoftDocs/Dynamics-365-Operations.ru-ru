# <a name="create-an-intercompany-plan"></a>Создание внутрихолдингового плана

[!include [banner](../../includes/banner.md)]

Эта процедура демонстрирует порядок создания внутрихолдингового плана. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

## <a name="set-up-an-intercompany-planning-group"></a>Настройка группы внутрихолдингового планирования

1. В **области переходов** выберите **Модули > Сводное планирование > Настройка > Группы внутрихолдингового планирования**.
2. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле "Имя" по значению "10".
3. В списке пометьте выбранную строку.
4. Выберите **Удалить**. Это действие необходимо, чтобы ускорить выполнение внутрихолдингового планирования.   Внутрихолдинговое планирование будет выполнять сводное планирование во всех компаниях в группе планирования, начиная с наименьшей запланированной последовательности.  
5. Выберите **Да**.
6. Закройте страницу.

## <a name="create-an-intercompany-master-plan"></a>Создание внутрихолдингового сводного плана

1. В **области переходов** выберите **Модули > Сводное планирование > Рабочие области > Сводное планирование**.
2. Выберите **Внутрихолдинговое сводное планирование**.  
3. В поле **Группа внутрихолдингового планирования** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
4. В списке выберите ссылку в выбранной строке. Выберите группу внутрихолдингового планирования 10.  
5. В поле **Количество внутрихолдинговых итераций планирования** введите "2". Группа внутрихолдингового планирования 10 содержит двух участников. Чтобы распространить задержки из компании-источника (USMF) на компанию клиента (DEMF), потребуется выполнить внутрихолдинговое планирование в обеих компаниях два раза. Первая итерация распространяет спрос и определяет задержки в компании-источнике (USMF). Вторая итерация распространяет задержки из USMF на DEMF.  
6. В поле **Первая итерация** выберите "Пересоздание".
7. В поле **Последующие итерации** выберите "Пересоздание".
8. В поле **Количество потоков** введите число. Представляет собой количество параллельных потоков, которые будут использоваться для планирования.  
9. Нажмите **ОК**.

## <a name="view-the-result-of-the-plan"></a>Просмотр результата плана

1. В поле **План** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
2. В списке выберите ссылку в выбранной строке. Выберите ссылку для StaticPlan. Необходимо находиться в компании USMF.  
3. Выберите **Спланированные заказы**.
