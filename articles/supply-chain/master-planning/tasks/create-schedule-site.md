# <a name="create-a-schedule-for-a-site"></a>Создание графика для сайта

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как запланировать производственные заказы, которые еще не начаты для сайта.  Для выполнения этой процедуры используется компания с демонстрационными данными USMF.


## <a name="identify-production-orders-that-are-not-started"></a>Определение неначатых производственных заказов
1. Перейдите в раздел "Управление производством" > "Производственные заказы" > "Все производственные заказы".
2. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле "Сайт" по значению "1".
    * 1 означает сайт в USMF. Если вы не используете USMF, выберите сайт из собственной компании.  
3. Откройте фильтра столбца "Статус".
4. Примените фильтр в поле "Статус" со значением "Запланировано", используя оператор фильтра "совпадает с".

## <a name="create-a-schedule"></a>Создание графика
1. В списке отметьте все строки или отмените отметку всех строк.
2. В области действий щелкните "План".
3. Щелкните "Планирование заданий".
4. В поле "Направление планирования" выберите "Назад от даты поставки".
5. Выберите значение "Нет" в поле "Ограничение по мощности".
6. Выберите значение "Нет" в поле "Ограничение по материалам".
7. Нажмите кнопку "OК".
    * Это может занять некоторое время.  

## <a name="view-the-result-of-scheduled-production-orders"></a>Просмотр результата спланированных производственных заказов
1. В списке пометьте выбранную строку.
    * Можно пометить любую строку.  
2. В области действий щелкните "Производственный заказ".
3. Щелкните "Все задания".
    * На этой странице можно просмотреть список заданий. На вкладке "Планирование" можно просмотреть дату начала и окончания задания.  
4. Щелкните "Материалы".
    * На этой странице можно просмотреть расчетное потребление материала для операций по производственному заказу и доступные в данный момент запасы.
