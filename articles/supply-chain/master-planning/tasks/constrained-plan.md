# <a name="generate-a-constrained-plan"></a>Создание плана с ограничениями

[!include [task guide banner](../../includes/task-guide-banner.md)]

Следующая процедура используется для создания плана, в котором учитываются ограничения по материалам и мощности. План гарантирует, что производство не начнется, пока не будут доступны материалы, и что ресурсы не будут зарезервированы с превышением. 

В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Эта процедура предназначена для планировщика производства.


## <a name="set-up-a-constrained-plan"></a>Настройка плана с ограничениями
1. Щелкните "Сводное планирование".
2. Щелкните "Сводные планы".
3. В списке найдите и выберите требуемую запись.
    * Пример: StaticPlan  
4. Выберите значение "Да" в поле "Ограничение по мощности".
5. В поле "Временная граница ограничения по мощности" введите значение "30".
6. Разверните раздел "Временные границы в днях".
7. Выберите значение "Да" в поле "Мощность".
8. В поле "Временная граница планирования мощности (в днях)" введите число.
    * Пример: 60  
9. Выберите значение "Да" в поле "Рассчитанные задержки".
10. В поле "Временная граница расчета задержек (в днях)" введите число.
    * Пример: 60  
11. Разверните раздел "Рассчитанные задержки".
12. Выберите значение"Да" в поле "Добавить рассчитанную задержку к дате потребности".
13. Выберите значение"Да" в поле "Добавить рассчитанную задержку к дате потребности".
14. Выберите значение"Да" в поле "Добавить рассчитанную задержку к дате потребности".
15. Закройте страницу.

## <a name="create-a-constrained-plan"></a>Создание плана с ограничениями
1. Щелкните "Выполнить".
2. В поле "Сводный план" введите или выберите значение.
    * Выберите план, для которого настроены ограничения.  
3. Нажмите кнопку "OК".
    * Это может занять некоторое время.  
4. Щелкните "Спланированные заказы".
