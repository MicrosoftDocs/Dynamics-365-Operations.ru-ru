# <a name="create-a-material-plan-for-co-products"></a>Создание плана материалов для сопутствующих продуктов

[!include [banner](../../includes/banner.md)]

Планировщик производства планирует материальные потребности в номенклатурах, представляющих собой сопутствующие продукты формулы. В качестве компании с демонстрационными данными для создания этой процедуры используется USP2.

## <a name="create-requirement-for-a-co-product"></a>Создание потребности для сопутствующего продукта

1. Перейдите **Продажи и маркетинг \> Рабочие области \> Обработка и запрос заказов на продажу**.
1. Выберите **Создать**.
1. Выберите **Заказ на продажу**.
1. В поле **Счет клиента** введите значение.
    * Пример: US-001  
1. Нажмите **ОК**.
1. В поле **Код номенклатуры** введите значение.
    * Пример: P6003  
1. В поле **Количество** введите число.
    * Пример: 50000  
1. Нажмите **Сохранить**.

## <a name="create-a-material-plan-for-co-products"></a>Создание плана материалов для сопутствующих продуктов

1. Закройте страницу.
1. Закройте страницу.
1. Выберите **Сводное планирование**.
1. В поле **План** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
1. В списке выберите ссылку в выбранной строке.
    * Пример: MasterPlan  
1. Выберите **Выполнить**.
1. Разверните или сверните раздел **Записи для добавления**.
1. Выберите **Фильтр**.
1. Выберите в списке строку, где **Поле** = *Номер номенклатуры*.
1. В поле **Критерии** введите значение.
    * Пример: P6003  
1. Нажмите **ОК**.
1. Нажмите **ОК**.
1. Выберите **Спланированные заказы**.
1. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле **Номер номенклатуры** по значению "P6000".
    * Установите фильтр по номенклатуре-формуле, в которой присутствует сопутствующий продукт номенклатуры, для которой вы создали заказ на продажу.  
1. В списке пометьте выбранную строку.
    * Выберите любую из строк, возращенных фильтром.  
1. В списке выберите ссылку в выбранной строке.
1. Разверните раздел **Информация об источниках потребности**.
1. В списке выберите ссылку в выбранной строке.
    * Спланированный заказ прикрепляется к заказу на продажу сопутствующего продукта.  
1. Закройте страницу.

## <a name="create-a-second-material-plan-for-co-products"></a>Создание второго плана материалов для сопутствующих продуктов

1. В поле **План** выберите кнопку раскрывающегося списка, чтобы открыть поиск.
2. В списке выберите ссылку в выбранной строке.
    * Пример: MasterPlan  
3. Выберите **Выполнить**.
4. Разверните или сверните раздел **Записи для добавления**.
5. Выберите **Фильтр**.
6. Выберите в списке строку, где **Поле** = *Номер номенклатуры*.
7. В поле *Критерии* введите значение.
    * Пример: P6003  
8. Нажмите **ОК**.
9. Нажмите **ОК**.
10. Выберите **Спланированные заказы**.
11. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле **Номер номенклатуры** по значению "P6000".
    * Установите фильтр по номенклатуре-формуле, в которой присутствует сопутствующий продукт номенклатуры, для которой вы создали заказ на продажу.  
12. В списке пометьте выбранную строку.
    * Выберите любую из строк, возращенных фильтром.  
13. В списке выберите ссылку в выбранной строке.
14. Разверните или сверните раздел **Информация об источниках потребности**.
15. В списке выберите ссылку в выбранной строке.
    * Спланированный заказ прикрепляется к заказу на продажу сопутствующего продукта.  
16. Закройте страницу.
17. Выберите **Сводное планирование**.
18. Перейдите в раздел **Сводное планирование \> Настройка \> Параметры сводного планирования**.
19. Выберите *Нет* в поле **Отключить все процессы планирования**.
20. Закройте страницу.
