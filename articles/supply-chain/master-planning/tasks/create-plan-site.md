# <a name="create-a-plan-for-a-site"></a>Создание плана для сайта

[!include [banner](../../includes/banner.md)]

Планировщик производства рассчитывает потребности в материалах и потребности в мощностях для производства конкретной номенклатуры. После создания исходных предложений они обнаруживают заказы на сайте, для которого они планируют и утверждают заказы, начиная со срочных. Самые срочные заказы — это заказы, которые необходимо утвердить на текущую дату. Для выполнения этих задач используйте компанию с демонстрационными данными USMF.


## <a name="create-a-materials-and-capacity-plan-for-an-item"></a>Создание плана материально-технического обеспечения и мощностей для номенклатуры
1. Щелкните "Сводное планирование".
    * Необходимо перейти к панели мониторинга по умолчанию.  
2. Щелкните "Выполнить".
3. Разверните раздел "Записи для добавления".
4. Щелкните "Фильтр".
5. В списке пометьте выбранную строку.
6. В поле "Критерии" введите значение.
    * Пример: D0001  
7. Нажмите кнопку "OК".
8. Нажмите кнопку "OК".
    * Это может занять несколько минут.  
9. Обновите страницу.

## <a name="identify-the-urgent-planned-orders-for-the-item"></a>Определение срочных спланированных заказов для номенклатуры
1. Откройте фильтр столбца "Код номенклатуры".
2. Примените фильтр к полю "Код номенклатуры" со значением "D0001", используя оператор фильтра "начинается с".
3. Откройте фильтр столбца "Дата заказа".
4. Примените фильтр в поле "Дата заказа" со значением текущей даты, используя оператор фильтра "совпадает с".

## <a name="firm-all-the-urgent-orders-for-the-item"></a>Утверждение срочных заказов для номенклатуры
1. В списке отметьте все строки или отмените отметку всех строк.
2. Щелкните "Утверждение".
3. Нажмите кнопку "OК".
