# <a name="cancel-a-master-planning-job"></a>Отмена задания сводного планирования

[!include [banner](../includes/banner.md)]

В Microsoft Dynamics 365 Supply Chain Management есть несколько вариантов отмены задания сводного планирования. Например, можно отменить задание сводного планирования, если оно было запущено по ошибке или работает дольше, чем ожидалось, и вы хотите закончить его. Лучший способ отменить задание планирования — это страница **Незавершенные процессы планирования**. Альтернативные варианты на страницах **Пакетные задания** и **Расширенные пакетные задания** должны использоваться только в том случае, если отмена задания сводного планирования на странице **Незавершенные процессы планирования** не была завершена в течение нескольких минут.

## <a name="preferred-cancel-option"></a>Предпочтительный вариант отмены
### <a name="cancel-master-planning-job-from-unfinished-planning-processes-page"></a>Отмена задания сводного планирования на странице **Незавершенные процессы планирования**
1. Перейдите в раздел **Сводное планирование > Запросы и отчеты > Сводное планирование > Незавершенные процессы планирования**.
2. Выберите строку процесса планирования, который следует отменить.
3. Щелкните **Отмена**.

## <a name="additional-cancel-options"></a>дополнительные параметры отмены
Они должны использоваться, только если отмена задания сводного планирования на странице **Незавершенные процессы планирования** не завершилась в течение нескольких минут.

### <a name="delete-master-planning-job-from-the-batch-jobs-page"></a>Удаление задания сводного планирования на странице **Пакетных заданий**
1. Перейдите в раздел **Администрирование системы > Запросы > Пакетные задания**.
2. Выберите строку задания планирования, которое следует удалить.
3. Нажмите кнопку **Удалить**.

### <a name="abort-master-planning-job-task-from-the-batch-jobs-enhanced-page"></a>Отмена задачи задания сводного планирования на странице **Расширенные пакетные задания**
1. Перейдите в раздел **Администрирование системы > Запросы > Пакетные задания**.
2. Если идентификатор задания не отображается в списке, щелкните **Переключить форму в расширенное представление**, в противном случае приступите к следующему шагу.
3. Откройте пакетное задание. Нажмите **Идентификатор задания** для пакетного задания с задачами, которые вы хотите закончить.
4. В **Пакетные задания** выберите задачи для завершения.
5. На экспресс-вкладке **Пакетные задачи** нажмите **Отмена**.
