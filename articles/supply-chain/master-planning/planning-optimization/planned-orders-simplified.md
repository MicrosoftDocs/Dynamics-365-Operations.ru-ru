# <a name="planned-orders-simplified"></a>Упрощенные спланированные заказы

[!include [banner](../../includes/banner.md)]
[!INCLUDE [preview-banner](../../includes/preview-banner.md)]
<!--KFM: preview until 10.0.25 GA -->

В этом разделе описывается, как можно просматривать, управлять и утверждать спланированные заказы, используя страницу **Упрощенные спланированные заказы**.

Страница **Упрощенные спланированные заказы** — это упрощенная версия страницы **Спланированные заказы**. Она содержит меньше параметров, но она быстрее, когда достаточно лишь проверить, утвердить и утвердить подтвердить спланированные заказы.

## <a name="turn-on-this-feature-for-your-system"></a>Включите эту функцию для своей системы

Если ваша система еще не содержит функций, описанных в этом разделе, перейдите в [Управление функциями](../../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) и включите функцию *Упрощенные спланированные заказы*.

## <a name="view-manage-and-firm-planned-orders"></a>Просмотр, управление и подтверждение спланированных заказов

Страница **Упрощенные спланированные заказы** предназначена в основном для подтверждения. На ней отображаются все спланированные заказы, созданные для выбранного плана так же, как и на стандартной странице **Спланированные заказы**. Таким образом, ее можно использовать в качестве оптимизированной альтернативы для просмотра результатов выполнения сводного планирования.

Чтобы открыть страницу **Упрощенные спланированные заказы**, перейдите в раздел **Сводное планирование \> Сводное планирование \> Упрощенные спланированные заказы**. Эту страницу можно использовать для завершения следующих задач:

- Просмотр всех спланированных заказов для определенного плана.
- Создайте новые планируемые заказы.
- Удаление одного или нескольких спланированных заказов.
- Подтверждение одного или нескольких спланированных заказов.
- Изменение статуса спланированного заказа для одного или нескольких спланированных заказов.
- Простое открытие сведений о спланированном заказе для отдельного спланированного заказа для просмотра полной информации о заказе.
- Создайте настраиваемые представления, используя сохраненные представления.

Дополнительные сценарии, такие как разбиение, поддерживаются стандартной страницей **Спланированные заказы**.
