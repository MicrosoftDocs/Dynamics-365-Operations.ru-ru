# <a name="use-tracing-for-explosion"></a>Использование трассировки для развертывания

[!include [banner](../includes/banner.md)]

Эта статья описывает, как можно использовать трассировку, чтобы исследовать причины результата развертывания заказа.

Включив трассировку, вы можете просматривать информацию о факторах, которые участвуют в результатах развертывания определенного заказа. Ниже приведены примеры, как можно использовать сведения трассировки:

-   Просмотр связей между действиями по запланированным заказам для оптимизации цепочки поставок и резервирования запасов.
-   Просмотр связей с заказами, которые уже утверждены. Вы можете сосредоточиться на автоматическом подтверждении производных потребностей и более точной поставной приоритетов заказов.
-   Моделирование результатов планирования, чтобы определить, оптимальны ли параметры планирования.
-   Выявление того, как определяется информация, такая как даты производство, количества и приоритеты для заказов.

Вы можете просматривать информацию о фьючерсах и действия для выбранного заказа. На странице **Развертывание** отслеживание информации доступно на вкладке **Объяснение** в верхней области. Трассировка происходит при развертывании заказа. Чтобы начать трассировку для заказа щелкните **Обновить**, а затем установите флажок **Включить трассировку**. Можно использовать поле **Найти текст** для поиска конкретной информации в журнале. Результаты поиска будут выделены в дереве.
