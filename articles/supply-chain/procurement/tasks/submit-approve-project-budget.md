# <a name="submit-and-approve-project-budget"></a>Отправка и утверждение бюджета проекта

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как создать и отправить бюджет для проекта. 

При создании бюджета проекта можно ввести расчетные значения доходов и затрат по проекту, а затем использовать их для контроля фактических проводок по проекту. В бюджетировании проекта все исходные бюджеты и изменения должны отправляться в workflow-процесс проекта на утверждение. Workflow-процессы обеспечивают более жесткий контроль над процессами и позволяют вести историю изменений.

Эта задача была создана с использованием набора данных USSI.

1. В **области переходов** выберите **Модули > Управление и учет по проектам > Проекты > Все проекты**.
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.
4. В области **Панель операций** щелкните **План**.
5. Щелкните **Бюджет проекта**.
6. В поле **Описание** введите значение.
7. Разверните экспресс-вкладку **Затраты**.
8. Нажмите кнопку **Создать**.
9. В поле **Тип проводки** выберите параметр.
10. В поле **Категория** введите или выберите значение.
11. В поле **Исходный бюджет** введите число.
12. Разверните экспресс-вкладку **Выручка**.
13. Нажмите кнопку **Создать**.
14. В списке пометьте выбранную строку.
15. В поле **Тип проводки** выберите параметр.
16. В поле **Категория** введите или выберите значение.
17. В поле **Исходный бюджет** введите число.
18. Нажмите кнопку **Сохранить**.
19. Щелкните **Рабочий процесс**.
20. Щелкните **Отправить**.
21. В поле **Комментарий** введите значение.
22. Щелкните **Отправить**.
