# <a name="create-and-submit-a-project-budget-workflow"></a>Рабочий процесс создания и отправки бюджета проекта

[!include [banner](../../includes/banner.md)]

При создании бюджета проекта можно ввести расчетные значения доходов и затрат по проекту, а затем использовать эти значения для контроля фактических проводок по проекту. Для бюджетирования проекта требуется, чтобы все исходные бюджеты и изменения прошли через рабочий процесс проекта для утверждения. Рабочий процесс увеличивает ваш контроль над бюджетированием и создает запись истории изменений. После [создания проекта](/dynamicsax-2012/appuser-itpro/create-a-project) используйте эту процедуру, чтобы создать и отправить бюджет.

1. Перейдите в раздел **Модули** > **Управление и учет по проектам** > **Проекты** > **Все проекты**.
1. Из списка проектов выберите проект.
1. На странице сведений о проекте выберите вкладку **План**.
1. В группе **Бюджет** выберите **Бюджет проекта**.
1. На экспресс-вкладке **Общие** введите следующие сведения:
   - В поле **Описание** введите значение.
   - Выберите параметр для поля **Исходный бюджет**.
   - Выберите параметр для поля **Оставшийся бюджет**.
1. Разверните экспресс-вкладку **Затраты** и выберите **Создать**. Затем установите следующие параметры:
   - Выберите параметр для поля **Тип проводки**.
   - Выберите соответствующую **Категорию**.
   - Ввод значения в поле **Исходный бюджет**.
1. Разверните экспресс-вкладку **Выручка** и выберите **Создать**. Затем установите следующие параметры:
   - Выберите параметр для поля **Тип проводки**.
   - Выберите **Категорию**.
   - Ввод значения для поля **Исходный бюджет**.
1. Нажмите **Сохранить**.
1. Выберите **Workflow-процесс \> Отправить**.
1. На странице **Обзор рабочего процесса исходного бюджета — отправка** введите **Комментарий** и выберите **Отправить**.
