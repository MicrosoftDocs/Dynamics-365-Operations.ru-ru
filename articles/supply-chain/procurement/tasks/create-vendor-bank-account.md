# <a name="create-a-vendor-bank-account"></a>Создание банковского счета поставщика

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как создать банковский счет для поставщика. Эту процедуру можно выполнить, используя компанию с демонстрационными данными USMF.

1. Выберите **Область переходов > Модули > Закупки и источники > Поставщики > Все поставщики**.
2. Выберите поставщика, для которого необходимо создавать банковский счет, и перейдите по ссылке в поле **Код счета поставщика**.
3. В **области действий** щелкните **Поставщик**.
4. Щелкните **Банковские счета**.
5. В **Области действий** щелкните **Создать**.
6. В поле **Банковский счет** введите значение. Этот код будет использоваться для идентификации банковского счета в запись поставщика.  
7. В поле **Имя** введите значение.
8. В поле **Банковские группы** введите или выберите значение.
9. В поле **Тип кода банка** выберите параметр. Это тип внутреннего номера, используемого для международных платежей.  
10. В поле **Номер банковского счета** введите значение.
11. В поле **SWIFT-код** введите значение.
12. В поле **IBAN** введите значение.
    - Номер IBAN должен указываться в правильном формате. Например, можно использовать DE89370400440532013000.  
    - Банковский счет будет иметь статус "Активно", если достигнуто значение в поле "Действует с" и не превышено значение в поле "Дата окончания срока действия". Он также будет активным, если поля "Действует с" и "Дата окончания срока действия" пусты. Ели обе даты, указанные в полях "Действует с" и "Дата окончания срока действия", относятся к будущему периоду, электронные платежи недоступны. Доступны другие типы платежей, и банковский счет является активным.  
13. Разверните раздел **Настройка**.
14. В поле **Код текста** введите значение. В этом поле указывается код, который отображается в банковской выписке получателя.  
15. В поле **Сообщение в банк** введите значение.
16. В поле **Ссылка на курс** введите значение. Это номер ссылки любого валютного курса с изменяемыми или фиксированными условиями.
17. В поле **Валюта** введите или выберите значение. Если выпускаются контрольные операции, в этом разделе содержится обзор их статуса ("Ожидание" или "Утверждено").  
18. Разверните раздел **Адрес**.
19. Разверните раздел **Контрольные операции**.
20. Разверните раздел **Контактная информация**.
21. В поле **Телефон** введите значение.
22. Закройте страницу.
23. Выберите **Изменить**.
24. Разверните раздел **Платеж**.
25. В поле **Банковский счет** выберите созданный счет.
26. Нажмите кнопку **Сохранить**. Адрес может наследоваться от банковской группы, если оно указана, либо его можно добавить здесь.
