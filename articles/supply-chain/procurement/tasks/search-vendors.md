# <a name="search-for-vendors"></a>Поиск поставщиков

[!include [task guide banner](../../includes/task-guide-banner.md)]

Научитесь выполнять поиск поставщиков по определенным критериям. В этом примере показано, как искать поставщиков, утвержденных для определенной категории закупаемой продукции, с получением их основного адреса в конкретной стране. Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USMF или собственные данные. Эта задача обычно выполняется специалистом по закупкам.

1. Перейдите в раздел "Закупки и источники" > "Поставщики" > "Поиск поставщика".
2. Щелкните значок "плюс", чтобы открыть страницу "Выбор категории закупаемой продукции".  
3. В дереве выберите "CORP PROCUREMENT CATEGORIES\OFFICE MACHINES".
    * Если вы выполняете эту процедуру как руководство по задаче, вам может потребоваться нажать кнопку "Разблокировать", прежде чем вы сможете выбрать правильный узел. Если вы используете не USMF, выберите одну из имеющихся у вас категорий.  
4. Нажмите кнопку Добавить.
    * Здесь можно выбрать несколько категорий. При этом при поиске будут найдены все поставщики, утвержденные по крайней мере для одной из категорий.  
5. Нажмите кнопку "OК".
6. В поле "Страна/регион" введите значение.
7. Нажмите кнопку "OК".
