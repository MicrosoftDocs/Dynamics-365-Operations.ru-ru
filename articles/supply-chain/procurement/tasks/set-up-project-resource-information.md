# <a name="set-up-project-resource-information"></a>Настройка сведений о ресурсах для проекта

[!include [banner](../../includes/banner.md)]

В этой процедуре показано, как настроить ресурс проекта. В этой задаче используется набор данных компании USMF.

1. Перейдите в раздел "Управление и учет по проектам" > "Ресурсы проекта" > "Список ресурсов".
2. Щелкните "Новый ресурс".
3. В поле "Работник" введите или выберите значение.
4. В поле "Календарь" введите или выберите значение.
5. В поле "Код периода" введите или выберите значение.
6. Щелкните "Создать".
7. Щелкните "Себестоимость".
8. Щелкните "Себестоимость (час)".
9. Щелкните "Создать".
10. В поле "Действует с" введите дату.
11. В поле "Себестоимость" введите число.
12. Нажмите кнопку "Сохранить".
13. Закройте страницу.
14. Щелкните "Цены продажи".
15. Щелкните "Цена продажи (час)".
16. Щелкните "Создать".
17. В поле "Действует с" введите дату.
18. В поле "Ценообразование" введите число.
19. В поле "Допустимо для" выберите один из вариантов.
20. Нажмите кнопку "Сохранить".
21. Закройте страницу.
