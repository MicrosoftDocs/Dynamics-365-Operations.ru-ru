# <a name="create-solicitation-types-and-scoring-criteria-for-rfqs"></a>Создание типов обращений и критериев оценки для запросов предложений

[!include [banner](../../includes/banner.md)]

В этом руководстве показано, как создать тип обращения и связать его с методом оценки. В нем также показано, как использовать тип обращения в запросе предложения, который задает метод оценки по умолчанию. Эти задачи обычно выполняются менеджером по закупкам. Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USMF или собственные данные. Необходимо иметь доступный метод оценки перед началом работы.


## <a name="create-a-solicitation-type"></a>Создание типа обращения
1. Перейдите в раздел "Закупки и источники" > "Настройка" > "Запрос предложения" > "Тип обращения".
2. Щелкните "Создать".
3. В поле "Имя" введите значение.
4. В поле "Описание" введите значение.
5. В поле "Метод оценки" выберите метод оценки, который требуется использовать для этого типа обращения.
6. Нажмите кнопку "Сохранить".
7. Закройте страницу.

## <a name="use-the-solicitation-type"></a>Использование типа обращения
1. Перейдите в раздел "Закупки и источники" > "Запросы предложений" > "Все запросы предложений".
2. Щелкните "Создать".
3. В поле "Тип обращения" выберите созданный тип обращения. 
    *   
4. Нажмите кнопку "OК".
5. Щелкните "Критерии оценки".
    * Отображаемые критерии оценки — это критерии из метода оценки, связанного с типом обращения. Можно добавить или удалить критерии на этой странице. Также можно добавить новый критерий, скопировав его из других методов оценки.  
6. Щелкните "Копировать критерии".
7. В поле "Метод оценки" введите или выберите значение.
8. Нажмите кнопку "OК".
9. Закройте страницу.
