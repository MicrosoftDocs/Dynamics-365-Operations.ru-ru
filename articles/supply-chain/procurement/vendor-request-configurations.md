# <a name="vendor-request-configurations"></a>Конфигурации запросов поставщика
[!include [banner](../includes/banner.md)]

Для выполнения запроса поставщика контактное лицо поставщика должно заполнить мастер регистрации потенциального поставщика.

В форме **Конфигурации запросов поставщика** можно создать профили, которые определяют обязательные поля и видимые поля в мастере регистрации потенциального поставщика.

Мастер регистрации поставщика будет начинаться с запроса у пользователя потенциального поставщика страны или региона, в которых поставщик ведет свой бизнес. Эта информация определяет используемую конфигурацию. Если для страны/региона не определена конкретная конфигурация, будет использоваться конфигурация по умолчанию.

### <a name="set-up-a-vendor-request-configuration"></a>Настройка конфигурации запросов поставщиков

По умолчанию имеется доступная конфигурация поставщика в форме конфигураций запроса поставщика.

Невозможно выбрать страну или регион для конфигурации по умолчанию, поэтому раздел **Страна/регион** не может быть изменен.

1. Щелкните **Закупки и источники** > **Настройка** > **Поставщики**, затем щелкните **Конфигурации запросов поставщика**.
2. Щелкните вкладку **Поля**, чтобы задать статус перечисленных полей.
3. Скрыто (не отображается)
4. Отображается (видно, но не является обязательным)
5. Обязательно (видно и обязательно)
6. Щелкните вкладку **Содержимое**, чтобы указать, будет ли отображаться текст в мастере, и требуется ли подтверждение, что пользователь потенциального поставщика должен принять этот текст перед переходом к следующему шагу в мастере. Подтверждение будет запрашиваться для любых положений и условий, которые пользователь должен принять для продолжения.

Можно также ввести подтверждающее сообщение, которое будет отображаться, когда мастер завершен, и можно добавить одну или несколько анкет.

### <a name="create-a-vendor-configuration-for-a-specific-countryregion"></a>Создание конфигурации поставщика для определенной страны или региона
1.  Щелкните **Закупки и источники** > **Настройка** > **Поставщики**, затем щелкните **Конфигурации запросов поставщика**.
2.  Щелкните **Создать**, чтобы создать новую конфигурацию, и введите имя для конфигурации.
3.  Нажмите кнопку **Сохранить**.
4.  Откройте вкладку **Страна/регионы**, чтобы выбрать страну или регион, для которых должна использоваться конфигурация.
5.  Завершите настройку конфигурации в соответствии с инструкциями для конфигурации по умолчанию.
