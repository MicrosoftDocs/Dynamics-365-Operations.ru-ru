# <a name="register-goods-shipped-to-customers-russia"></a>Регистрация товаров, отгруженных клиентам (Россия)

[!include [banner](../includes/banner.md)]

В соответствии с законодательством Российской Федерации компании необходимо зарегистрировать отгрузку товаров при транспортировке товаров клиенту. Когда товары отгружаются, право собственности на товары передается от продавца покупателю. Такая передача права владения также называется передачей прав собственности.
Кроме того, можно отгружать товары, когда переход прав собственности откладывается на более позднюю дату. Клиент несет ответственность за товары только в том случае, когда происходит передача прав собственности.
Включена следующая функциональность:

- Регистрация отгрузки в случае наличия отложенной передачи права собственности.

  > [!ПРИМЕЧАНИЕ.] Хотя отгрузка не изменяет баланс клиента, поставщик все равно должен заплатить налог на добавленную стоимость (НДС). Товары перемещаются на специальный склад для отгруженных товаров.
  
- Регистрация передачи прав владения. Эта регистрация включает регистрацию задолженности клиента и списание товаров, которые были проданы.
- Отмена отгрузки в случае наличия отложенной передачи права собственности.
- Распределение расходов по накладной по продаже.

## <a name="settings"></a>Настройки
### <a name="set-up-a-warehouse"></a>Настройка склада
1.  Перейдите в раздел **Управление запасами** \> **Настройка** \> **Разделение запасов** \> **Склады**.
2.  Создайте склад типа **Товары отгруженные**.

Для склада любого другого типа в поле **Склад для товаров отгруженных** можно указать склад **Товары отгруженные**, который должен использоваться для отгрузки товаров, когда имеется отсроченная передача прав собственности.
Для склада типа **Товары отгруженные** можно задать местоположение для отгруженных номенклатур, если для номенклатур активирована аналитика **Местоположение**.

### <a name="set-up-a-posting-type-and-number-sequences-for-passing-of-property-rights"></a>Настройка типа разноски и номерных серий для передачи прав собственности
1.  Перейдите в раздел **Расчеты с клиентами** \> **Настройка** \> **Параметры модуля расчетов с клиентами**.
2.  На вкладке **Общие** на экспресс-вкладке **Значения по умолчанию для продаж** в поле **Тип разноски** выберите тип разноски:

- **Стандартная** — права собственности передаются покупателю, когда разносится накладная по продаже.
- **Отлож. переход прав собств-ти** — передача права собственности покупателю откладывается при разноске накладной заказа на продажу.

3.  На вкладке **Номерные серии** в поле **Код номерной серии** выберите номерную серию для пунктов **Журнал перехода прав собственности**, **Ваучер перехода прав собственности** и **Ссылки ваучера накладных расходов**.

### <a name="set-up-a-ledger-posting-group-to-post-shipped-goods-tax"></a>Настройка группы разноски ГК для разноски налога за отправленные товары
1.  Перейдите в раздел **Налог** \> **Настройка** \> **Налог** \> **Группы разноски ГК**.
2.  Выберите группу разноски главной книги и на экспресс-вкладке **Общее** в поле **Налог на товары отгруженные** выберите счет ГК для налога на отгруженные товары.

### <a name="set-up-a-charge-code-that-uses-a-transit-account"></a>Настройка кода накладных расходов, который использует транзитный счет
Страница **Код накладных расходов** служит для настройки кодов накладных расходов. Можно также распределить расходы, воспользовавшись транзитным счетом.

1.  Перейдите в раздел **Расчеты с клиентами** \> **Настройка** \> **Настройка накладных расходов** \> **Код накладных расходов**.
2.  На экспресс-вкладке **Разноска** в разделе **Дебет** в поле **Тип** выберите **Клиент/Поставщик**.
3.  В разделе **Кредит** в поле **Тип** выберите **Счет ГК**.
4.  В разделе **Кредит** в поле **Разноска** выберите тип разноски для кредитового счета.
5.  В разделе **Кредит** в поле **Счет** выберите номер счета, на который должны быть кредитованы накладные расходы.
6.  Установите для параметра **Использовать транзитный счет** значение **Да**, чтобы использовать транзитный счет ГК для накладных расходов при разноске накладной заказа на продажу, где существует отложенная передача прав собственности.

### <a name="set-up-revenue-consumption-and-transit-accounts-to-post-miscellaneous-sales-charges"></a>Настройка счетов выручки, потребления и транзитного счета для разноски накладных расходов продажи
1.  Перейдите в раздел **Расчеты с клиентами** \> **Настройка накладных расходов** \> **Разноска расходов**.
2.  В группе полей **Тип разноски накладных расходов** выберите тип разноски для накладных расходов:

  - **Выручка** — настройка счетов учета, на которые следует разносить выручку для накладных расходов продажи.
  - **Потребление** — настройка счетов учета, на которые следует разносить потребление для накладных расходов продажи.
  - **Транзитный счет** — настройка счетов ГК, на которые требуется разносить накладные расходы продажи для отложенной передачи прав собственности.

> [!NOTE]
> Необходимо настроить счета выручки, потребления и транзита для разноски заказов на продажу, в которых имеются накладные расходы и отложенный перенос прав собственности.

3.  В поле **Код счета** выберите связь счета:

  - **Таблица** — задать связь счетов для конкретного клиента.
  - **Группа** — задать связь счетов для списка групп клиентов.
  - **Все** — задать связь счетов для всех клиентов.
  
4.  В поле **Связь счета** выберите клиента или группу клиентов для настройки связи счетов.
  > [!NOTE]
  > Это поле доступно, только если выбран вариант **Таблица** или **Группа** в поле **Связь счета**.
  
5.  В поле **Связь накл. расходов** выберите код накладных расходов:
  - **Таблица** — разноска настраивается для определенного кода накладных расходов.
  - **Все** — разноска настраивается для всех кодов накладных расходов.
  
6.  В поле **Код накладных расходов** выберите код накладных расходов.
  > [!NOTE]
  > Это поле доступно, только если выбран вариант **Таблица** в поле **Связь накл. расходов**.
  
7.  В поле **Счет ГК** выберите счет учета, на который разносятся проводки.
8.  В поле **Тип разноски** выберите тип разноски для разноски накладных расходов.

### <a name="set-up-ledger-accounts-and-offset-ledger-accounts-for-shipped-goods"></a>Настройка счетов учета и корр. счетов учета для отгруженных товаров
1.  Выберите **Управление запасами** \> **Настройка** \> **Разноска** \> **Разноска**.
2.  На вкладке **Заказ на продажу** в левой области выберите тип счета учета для настройки отгруженных товаров:

  - **Товары отгруженные** — настройка счетов учета для разноски проводок главной книги для отгруженных товаров.
  - **Корр. счет товаров отгруженных** — настройка корр. счетов учета для разноски проводок главной книги для отгруженных товаров.
  
3.  В поле **Код номенклатуры** выберите связь номенклатуры:

  - **Таблица** — код номенклатуры относится к конкретной номенклатуре.
  - **Группа** — код номенклатуры относится к конкретной группе номенклатур.
  - **Все** — код номенклатуры для всей номенклатуры.
  
4.  В поле **Связь номенклатуры** выберите номенклатуру или номенклатурную группу.

  >[!NOTE]
  > Это поле доступно, только если выбран вариант **Таблица** или **Группа** в поле **Код номенклатуры**.
  
5.  В поле **Код счета** выберите связь счета:

  - **Таблица** — задать связь счетов для конкретного клиента.
  - **Группа** — задать связь счетов для списка групп клиентов.
  - **Все** — задать связь счетов для всех клиентов.
  
6.  В поле **Связь счета** выберите клиента или группу клиентов для настройки связи счетов.

  > [!NOTE]
  > Это поле доступно, только если выбран вариант **Таблица** или **Группа** в поле **Связь счета**.
  
7.  В поле **Настройка налоговой группы** выберите налоговую группу.
8.  В поле **Счет ГК** выберите номер счета ГК.

### <a name="set-up-a-posting-type-for-a-customer-and-an-agreement"></a>Настройка типа разноски для клиента и соглашения
1.  Перейдите в раздел **Расчеты с клиентами** \> **Клиенты** \> **Все клиенты**.
2.  Выберите счет клиента.
3.  На экспресс-вкладке **Значения заказов на продажу по умолчанию** в поле **Тип разноски** выберите тип разноски:

  - **Стандартная** — права собственности передаются клиенту, когда разносится накладная по продаже.
  - **Отлож. переход прав собств-ти** — передача права собственности клиенту откладывается при разноске накладной заказа на продажу.
  
4.  Перейдите в раздел **Продажи и маркетинг** \> **Договоры продажи** \> **Договоры продажи**.
5.  Выберите договор продажи.
6.  Перейдите к представлению **Заголовок**.
7.  На экспресс-вкладке **Финансы** в поле **Тип разноски** выберите тип разноски:

  - **Стандартная** — права собственности передаются клиенту, когда разносится накладная по продаже.
  - **Отлож. переход прав собств-ти** — передача права собственности клиенту откладывается при разноске накладной заказа на продажу.

## <a name="working-with-sales-invoices-where-there-is-postponed-passing-of-property-rights"></a>Работа с накладными по продаже, в которых имеется отсроченный переход прав собственности
Выполните процедуры, описанные в этом разделе, чтобы создать заказ на продажу, в котором имеется отсроченный перенос прав собственности, разнесите накладную заказа на продажу и зарегистрируйте передачу прав собственности.

### <a name="create-a-sales-order-and-post-a-sales-invoice-where-there-is-postponed-passing-of-property-rights"></a>Создание заказа на продажу и разноска накладной заказа на продажу, где имеется отложенный перенос прав собственности
1.  Перейдите в раздел **Расчеты с клиентами** \> **Заказы** \> **Все заказы на продажу**.
2.  Создание заказа на продажу.
3.  Перейдите к представлению **Заголовок**.
4.  На экспресс-вкладке **Настройка** в поле **Тип разноски** выберите **Отлож. переход прав собств-ти**.
5.  В поле **Налоговая группа** выберите налоговую группу для заказа на продажу.
6.  Перейдите обратно в представление **Строки**.
7.  Создайте строки заказа на продажу. В поле **Налоговая группа номенклатур** выберите налоговую группу.
8.  Разноска накладной.

  > [!NOTE]
  > Чтобы разнести накладную типа **Отлож. переход прав собств-ти**, убедитесь, что условия оплаты, назначенные для заказов на продажу, не используют оплату наличными, и что для поля **Вида деятельности** не установлено значение **Хранитель** или **Комиссионер**.

Для каждой складской проводки со статусом расхода **Продано** создаются две дополнительные складские проводки для **Склад отгруженных товаров**: одна проводка имеет статус прихода **Куплено**, а другая проводка имеет статус выпуска **Списано**.
На этом этапе не создаются проводки сальдо клиента и выручки.
Следующие проводки ГК создаются:

  - **Дебет**: заказ на продажу, корр. счет товаров отгруженных; **Кредит**: стоимость единиц измерения с выставленными накладными
  - **Дебет**: заказ на продажу, отгруженные товары; **Кредит**: заказ на продажу, корр. счет товаров отгруженных
  - **Дебет**: налог на товары отгруженные; **Кредит**: исходящий налог

### <a name="allocate-miscellaneous-charges-to-a-sales-invoice"></a>Распределение накладных расходов для накладной заказа на продажу
Диалоговое окно **Распределить накладные расходы** служит для отнесения расходов на разнесенную накладную по продаже.
1.  Перейдите в раздел **Расчеты с клиентами** \> **Запросы и отчеты** \> **Накладные** \> **Журнал накладных**.
2.  Выберите накладную заказа на продажу типа разноски **Отлож. переход прав собств-ти**, затем выберите **Корректировка**, чтобы открыть диалоговое окно **Распределить накладные расходы**.
3.  В поле **Дата разноски** введите дату, когда накладные расходы необходимо разнести.
4.  Установите для параметра **Корректировка по кредиту** значение **Да**, чтобы указать, что накладные расходы должны разноситься как корректировки по кредиту.
5.  В поле **Код накладных расходов** выберите код расходов. В этом поле перечислены только расходы, для которых поле **Тип** в разделах **Дебет** и **Кредит** имеет значение **Счет ГК**.
6.  В поле **Значение расходов** введите значение расхода.
7.  Выберите **ОК** для разноски расхода на накладную по продаже.

### <a name="cancel-a-shipment-for-the-passing-of-property-rights"></a>Отмена отгрузки для передачи прав собственности
Диалоговое окно **Отмена отгрузки** используется для отмены отгрузки. Можно отменить отгрузку для товаров, если передача прав собственности не была полностью или частично зарегистрирована. При отмене отгрузки создается кредит-нота.
1.  Перейдите в раздел **Расчеты с клиентами** \> **Запросы и отчеты** \> **Накладные** \> **Журнал накладных**.
2.  Выберите накладную заказа на продажу для типа разноски **Отлож. переход прав собств-ти**.
3.  Выберите **Переход прав собственности** \> **Отмена отгрузки**, чтобы открыть диалоговое окно **Отмена отгрузки**.
4.  В разделе **Обработка** в поле **Дата** выберите дату отмены.
5.  Установите для параметра **Корректировка по кредиту (физическая)** значение **Да**, чтобы указать, что физическое обновление запасов должно разноситься как корректировка по кредиту.

  > [!NOTE]
  > Этот параметр автоматически устанавливается в значение **Да**, если для параметра **Сторно для физической разноски** задано значение **Да** на вкладке **Учет запасов** на странице **Параметры модуля "Управление запасами и складами"**.
  
6.  Установите для параметра **Корректирующая кредит-нота** значение **Да**, чтобы разнести проводку по кредит-ноте как корректировку по кредиту.

  > [!NOTE]
  > Этот параметр автоматически устанавливается в значение **Да**, если для параметра **Корректирующая кредит-нота** выбрано значение **Да** на вкладке **Обновления** на странице **Параметры модуля расчетов с клиентами**.
  
7.  Установите для параметра **Кредитовать оставшееся количество** значение **Да**, чтобы восстановить открытое количество заказа на продажу.

  > [!NOTE]
  > Когда накладная по продаже отменяется, количество, определенное в заказе на продажу, открыто для другой отгрузки.
  
8.  Установите для параметра **Корректирующий документ** значение **Да**, чтобы предоставить ссылку на исходную накладную заказа на продажу в кредит-ноте.
9.  Для выбора одной или нескольких строк накладной для отмены используется поле **Пометка**.

  > [!NOTE]
  > Чтобы выбрать все строки накладной, можно установить для параметра **Выбрать все** значение **Да**.
  
10. В поле **Для обработки** введите количество, для которого должна быть отменена отгрузка.
11. Выберите **ОК**, чтобы отменить отгрузку.
12. Закройте страницу **Журнал накладных**.

### <a name="register-the-passing-of-property-rights-for-a-posted-sales-invoice"></a>Регистрация передачи прав собственности для разнесенной накладной по продаже
Для регистрации передачи прав собственности для разнесенной накладной заказа на продажу используется диалоговое окно **Регистрация перехода прав собственности**. Каждая строка накладной в диалоговом окне содержит количество, для которого должна быть зарегистрирована передача прав собственности. Для подтверждения передачи прав собственности можно указать дату и количество в строке накладной. Начисленный налог и накладные расходы пересчитываются на основе количества, даты отгрузки и даты передачи прав собственности.
1.  Перейдите в раздел **Расчеты с клиентами** \> **Запросы и отчеты** \> **Накладные** \> **Журнал накладных**.
2.  Выберите накладную заказа на продажу для типа разноски **Отлож. переход прав собств-ти**.
3.  Выберите **Переход прав собственности** \> **Переход прав собственности**, чтобы открыть диалоговое окно **Регистрация перехода прав собственности**.
4.  В разделе **Обработка** в поле **Дата** выберите дату перехода прав собственности.
5.  Для выбора одной или нескольких строк накладной для перехода прав собственности используется поле **Пометка**.

  > [!NOTE]
  > Чтобы выбрать все строки накладной, можно установить для параметра **Выбрать все** значение **Да**.
  
6.  В поле **Для обработки** выберите количество, регистрируемое для передачи прав собственности.
7.  Выберите **ОК**, чтобы зарегистрировать передачу прав собственности.
На этом этапе создаются проводки сальдо клиента и выручки.

  > [!NOTE]
  > Накладные расходы обрабатываются как общая сумма, независимо от того, была ли регистрация передачи прав собственности полностью обработана или частично обработана.
  
Следующие проводки ГК разносятся:

  - **Дебет**: себестоимость проданных товаров, по которым выставлена накладная; **Кредит**: заказ на продажу, корр. счет товаров отгруженных
  - **Дебет**: корр. счет уплаты (налог); **Кредит**: исходящий налог
