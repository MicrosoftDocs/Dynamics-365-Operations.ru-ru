# <a name="change-the-banner-or-logo"></a><span data-ttu-id="762af-103">Изменение баннера или эмблемы</span><span class="sxs-lookup"><span data-stu-id="762af-103">Change the banner or logo</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="762af-104">Следующая процедура перечисляет этапы, которые системные администраторы могут использовать для обновления изображения баннера или логотипа, отображаемого для юридического лица.</span><span class="sxs-lookup"><span data-stu-id="762af-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="762af-105">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="762af-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="762af-106">Перейдите в раздел "Управление организацией" > "Организации" > "Юридические лица".</span><span class="sxs-lookup"><span data-stu-id="762af-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="762af-107">В списке в верхнем левом углу выберите юридическое лицо, для которого необходимо обновить баннер или эмблему.</span><span class="sxs-lookup"><span data-stu-id="762af-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="762af-108">Если оно уже выбрано, перейдите к следующему шагу.</span><span class="sxs-lookup"><span data-stu-id="762af-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="762af-109">Разверните раздел "Изображения".</span><span class="sxs-lookup"><span data-stu-id="762af-109">Expand the Images section.</span></span> <span data-ttu-id="762af-110">Если он уже развернут, перейдите к следующему шагу.</span><span class="sxs-lookup"><span data-stu-id="762af-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="762af-111">Выберите Изменить.</span><span class="sxs-lookup"><span data-stu-id="762af-111">Click Edit.</span></span>
5. <span data-ttu-id="762af-112">В поле "Тип изображения компании на панели мониторинга" выберите "Баннер".</span><span class="sxs-lookup"><span data-stu-id="762af-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="762af-113">Щелкните "Изменить".</span><span class="sxs-lookup"><span data-stu-id="762af-113">Click Change.</span></span>
    * <span data-ttu-id="762af-114">Совет: идеальное разрешение изображения баннера составляет 1920 x 281 пикселей.</span><span class="sxs-lookup"><span data-stu-id="762af-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="762af-115">Идеальная ширина изображения баннера составляет 350 пикселей.</span><span class="sxs-lookup"><span data-stu-id="762af-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="762af-116">После завершения отправки вы должны видеть новый баннер и изображение эмблемы в разделе "Изображение на панели мониторинга".</span><span class="sxs-lookup"><span data-stu-id="762af-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="762af-117">Нажмите кнопку "Сохранить".</span><span class="sxs-lookup"><span data-stu-id="762af-117">Click Save.</span></span>
8. <span data-ttu-id="762af-118">Перейдите в раздел "Общее" > "Общее" > "Панель мониторинга по умолчанию".</span><span class="sxs-lookup"><span data-stu-id="762af-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="762af-119">Вы должны видеть новый баннер и изображение эмблемы на панели мониторинга.</span><span class="sxs-lookup"><span data-stu-id="762af-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="762af-120">Вам может потребоваться изменить свою компанию, используя элемент управления выбора компании, на ту, для которой вы загрузили баннер.</span><span class="sxs-lookup"><span data-stu-id="762af-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
