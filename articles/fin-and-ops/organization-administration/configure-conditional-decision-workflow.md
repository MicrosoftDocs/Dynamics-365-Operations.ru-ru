# <a name="configure-a-conditional-decision-in-a-workflow"></a>Настройте условное решение в workflow-процессе

[!include [banner](../includes/banner.md)]

Используйте следующую процедуру для настройки свойств условного решения.

Условное решение - Точка, в которой workflow-процесс делится на две ветви. Чтобы настроить условное решение, в редакторе workflow-процесс, щелкните правой кнопкой мыши условное решение, а затем щелкните **Свойства**, чтобы открыть форму **Свойства**.

## <a name="name-a-decision"></a>Задание имени решения
Чтобы ввести имя условного решения, необходимо выполнить следующие действия.
1.  В левой области нажмите **Основные настройки**.
2.  В поле **Имя** введите уникальное имя для условного решения.

## <a name="set-conditions"></a>Установка условий
Система определяет, какая ветвь будет использоваться, оценивая документ отправленный, чтобы определить, отвечает ли документ указанным условиям.
1.  В левой области нажмите **Основные настройки**.
2.  Щелкните **Добавить условие**.
3.  Введите условие.
4.  При необходимости введите дополнительные условия.
5.  Чтобы убедиться, что введенные условия настроены верно, выполните следующие действия.
    1.  Щелкните **Проверка** для открытия формы **Условия тестового workflow-процесса**.
    2.  Выберите запись в области **Проверка условия** формы.
    3.  Щелкните **Тест**. Система оценит запись и определит, соответствует ли она определенным вами условиям.
    4.  Щелкните **OK** или **Отмена** для возврата к форме **Свойства**.
