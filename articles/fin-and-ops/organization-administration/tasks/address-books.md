# <a name="configure-address-books"></a>Настройка адресных книг

[!include [task guide banner](../../includes/task-guide-banner.md)]

Используйте эту процедуру и решения, принятые в разделе "Планирование конфигурации глобальной адресной книги и дополнительных адресных книг", чтобы настроить дополнительные адресные книги для вашей организации.

В качестве компании с демонстрационными данными для создания этой задачи используется USMF. Эта запись предназначена для членов группы планирования и конфигурации.


## <a name="configure-address-books"></a>Настройка адресных книг
1. Перейдите в раздел "Управление организацией" > "Глобальная адресная книга" > "Адресные книги".
2. Щелкните "Создать".
3. В поле "Имя" введите значение.
4. В поле "Описание" введите значение.
5. Нажмите кнопку Сохранить.
6. В списке найдите и выберите требуемую запись.
7. Щелкните стрелку, чтобы добавить в адресную книгу выбранные доступные группы.
8. Нажмите кнопку "Сохранить".
