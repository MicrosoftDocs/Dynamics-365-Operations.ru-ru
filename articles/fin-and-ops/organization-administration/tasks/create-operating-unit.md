# <a name="create-operating-units"></a>Создание операционных единиц

[!include [task guide banner](../../includes/task-guide-banner.md)]

Операционная единица — это организация, используемая для разделения управления экономическими ресурсами и операционными процессами в бизнесе. Сотрудники операционной единицы обязаны использовать ограниченные ресурсы максимально эффективно, оптимизировать процессы и отчитываться за свою деятельность. Ттипы операционных единиц включают центры затрат, бизнес-единицы, подразделения и потоки создания ценности. Для создания операционной единицы используется следующая процедура. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Перейдите в раздел "Управление организацией" > "Организации" > "Операционные единицы".
2. Щелкните "Создать", чтобы открыть раскрывающееся диалоговое окно.
3. В списке найдите и выберите требуемую запись.
    * Выберите тип операционной единицы, которую необходимо создать.  
4. В списке перейдите по ссылке в выбранной строке.
5. В поле "Имя" введите значение.
    * Разверните раздел "Общее", если необходимо.  
    * Введите общие сведения о операционной единице, например идентификационный номер, номер DUNS и менеджер.    
    * Разверните раздел "Адреса", если необходимо.  
    * Введите информацию об адресе, например улицу и номер дома, почтовый индекс и город. Щелкните "Добавить", чтобы ввести новую запись адреса, или "Изменить", чтобы изменить существующую запись адреса.   
    * При необходимости разверните раздел "Контактная информация".  
    * Введите сведения о способах связи, например адреса электронной почты, URL-адреса и номера телефона. Для ввода новой записи щелкните "Создать". Для изменения существующей записи щелкните "Дополнительные параметры" > "Дополнительно".   
6. Нажмите кнопку "Сохранить".
