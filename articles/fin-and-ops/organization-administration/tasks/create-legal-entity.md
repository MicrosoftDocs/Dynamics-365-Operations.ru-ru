# <a name="create-a-legal-entity"></a><span data-ttu-id="ae1dc-103">Создание юридического лица</span><span class="sxs-lookup"><span data-stu-id="ae1dc-103">Create a legal entity</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="ae1dc-104">Юридическое лицо - это организация, которая определяется посредством регистрации в органе власти.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-104">A legal entity is an organization that is identified through registration with a legal authority.</span></span> <span data-ttu-id="ae1dc-105">Юридические лица могут заключать контракты и обязаны подготавливать отчетность по результатам своей деятельности.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-105">Legal entities can enter into contracts and are required to prepare statements that report on their performance.</span></span> <span data-ttu-id="ae1dc-106">В следующей процедуре показано, как создать юридическое лицо.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-106">The following procedure explains how to create a legal entity.</span></span> <span data-ttu-id="ae1dc-107">В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="ae1dc-108">Перейдите в раздел "Управление организацией" > "Организации" > "Юридические лица".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-108">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="ae1dc-109">Щелкните "Создать".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-109">Click New.</span></span>
3. <span data-ttu-id="ae1dc-110">В поле "Имя" введите значение.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-110">In the Name field, type a value.</span></span>
4. <span data-ttu-id="ae1dc-111">В поле "Компания" введите значение.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-111">In the Company field, type a value.</span></span>
5. <span data-ttu-id="ae1dc-112">В поле "Страна/регион" введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-112">In the Country/region field, enter or select a value.</span></span>
6. <span data-ttu-id="ae1dc-113">Нажмите кнопку "OК".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-113">Click OK.</span></span>
    * <span data-ttu-id="ae1dc-114">В разделе "Общее" укажите следующие общие сведения о юридическом лице: Введите краткое наименование, если имя поиска необходимо.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-114">In the General section, provide the following general information about the legal entity: Enter a search name, if a search name is required.</span></span> <span data-ttu-id="ae1dc-115">Краткое наименование — это альтернативное имя, которое можно использовать для поиска этого юридического лица.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-115">A search name is an alternate name that can be used to search for this legal entity.</span></span> <span data-ttu-id="ae1dc-116">Выберите используется ли это юридическое лицо как консолидированная компания.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-116">Select whether this legal entity is being used as a consolidation company.</span></span> <span data-ttu-id="ae1dc-117">Выберите используется ли это юридическое лицо как Компания элиминирования.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-117">Select whether this legal entity is being used as an elimination company.</span></span>  
7. <span data-ttu-id="ae1dc-118">Разверните раздел "Адреса".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-118">Expand the Addresses section.</span></span>
    * <span data-ttu-id="ae1dc-119">В разделе "Адреса" введите информацию об адресе, например название улицы, номер дома, почтовый индекс и название города.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-119">In the Addresses section, enter address information, such as the street name and number, postal code, and city.</span></span>  
8. <span data-ttu-id="ae1dc-120">Разверните раздел "Контактная информация".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-120">Expand the Contact information section.</span></span>
    * <span data-ttu-id="ae1dc-121">В разделе "Контактная информация" введите сведения о методах связи, например адреса электронной почты, URL-адреса и номера телефонов.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-121">In the Contact information section, enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span>  
9. <span data-ttu-id="ae1dc-122">Разверните раздел "Предусмотренная отчетность".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-122">Expand the Statutory reporting section.</span></span>
    * <span data-ttu-id="ae1dc-123">В разделе "Предусмотренная отчетность" введите регистрационные номера, которые используются для предусмотренной отчетности.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-123">In the Statutory reporting section, enter the registration numbers that are used for statutory reporting.</span></span>  
10. <span data-ttu-id="ae1dc-124">Разверните раздел "Регистрационные номера".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-124">Expand the Registration numbers section.</span></span>
    * <span data-ttu-id="ae1dc-125">В разделе "Регистрационные номера" введите любую информацию, требуемую юридическим лицом.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-125">In the Registration numbers section, enter any information required by the legal entity.</span></span>  
11. <span data-ttu-id="ae1dc-126">Разверните раздел "Сведения о банковском счете".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-126">Expand the Bank account information section.</span></span>
    * <span data-ttu-id="ae1dc-127">В разделе "Сведения о банковском счете" введите банковские счета и внутренние номера для юридического лица.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-127">In the Bank account information section, enter bank accounts and routing numbers for the legal entity.</span></span>  
12. <span data-ttu-id="ae1dc-128">Разверните раздел "Внешняя торговля и логистика".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-128">Expand the Foreign trade and logistics section.</span></span>
    * <span data-ttu-id="ae1dc-129">В разделе "Внешняя торговля и логистика" введите сведения об отгрузке для юридического лица.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-129">In the Foreign trade and logistics section, enter shipping information for the legal entity.</span></span>  
13. <span data-ttu-id="ae1dc-130">Разверните раздел "Номерные серии".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-130">Expand the Number sequences section.</span></span>
    * <span data-ttu-id="ae1dc-131">В разделе "Номерные серии" можно просмотреть номерные серии, которые связаны с юридическим лицом.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-131">In the Number sequences section, you can view the number sequences that are associated with the legal entity.</span></span>  
14. <span data-ttu-id="ae1dc-132">Разверните раздел "Изображения".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-132">Expand the Images section.</span></span>
    * <span data-ttu-id="ae1dc-133">В разделе "Изображения" просмотрите или измените изображение эмблемы или панели мониторинга, связанной с юридическим лицом.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-133">In the Images section, view or change the logo and/or dashboard image that are associated with the legal entity.</span></span>  
15. <span data-ttu-id="ae1dc-134">Разверните раздел "Налоговая регистрация".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-134">Expand the Tax registration section.</span></span>
    * <span data-ttu-id="ae1dc-135">В разделе "Налоговая регистрация" введите регистрационные номера, которые используются для отчета в налоговые органы.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-135">In the Tax registration section, enter the registration numbers that are used to report to tax authorities.</span></span>  
16. <span data-ttu-id="ae1dc-136">Разверните раздел "Налог по форме 1099".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-136">Expand the Tax 1099 section.</span></span>
    * <span data-ttu-id="ae1dc-137">В разделе "Налог по форме 1099" введите сведения 1099 для юридического лица.</span><span class="sxs-lookup"><span data-stu-id="ae1dc-137">In the Tax 1099 section, enter 1099 information for the legal entity.</span></span>  
17. <span data-ttu-id="ae1dc-138">Нажмите кнопку "Сохранить".</span><span class="sxs-lookup"><span data-stu-id="ae1dc-138">Click Save.</span></span>
