# <a name="set-a-users-preferred-time-zone"></a>Настройка основного часового пояса пользователя

[!include [task guide banner](../../includes/task-guide-banner.md)]

В следующей процедуре объясняется, как пользователь в роли "Системный администратор" может настроить часовой пояс для пользователя.

1. Перейдите в раздел "Администрирование системы" > "Пользователи" > "Пользователи".
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.
4. Щелкните "Параметры пользователя".
5. Перейдите на вкладку "Параметры".
6. В поле "Часовой пояс" выберите параметр.
7. Нажмите кнопку "Сохранить".
