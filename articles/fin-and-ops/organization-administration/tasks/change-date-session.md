# <a name="change-the-date-for-a-session"></a>Изменение даты для сеанса

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой теме объясняется, как изменить дату для сеанса в Dynamics 365 for Finance and Operations. По умолчанию текущая дата используется при вводе и разноске записей журналов или документов-источников. Можно изменить дату для текущего сеанса. Используйте эту функцию для пометки задним числом записей журналов или документов-источников, если необходимо.

1. В области переходов выберите **Модули > Общие > Общие > Дата и время сеанса**.
2. В поле **Дата** введите дату.
3. Нажмите **ОК**.
