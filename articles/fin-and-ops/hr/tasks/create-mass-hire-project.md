# <a name="create-a-mass-hire-project"></a>Создание проекта по массовому набору сотрудников

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как настроить проект по массовому набору сотрудников. Специалист по найму может использовать проекты по массовому набору сотрудников для простого создания нескольких должностей и найма нескольких сотрудников на эти должности. Чтобы начать эту процедуру, перейдите в раздел "Управление персоналом" > "Набор сотрудников" > "Проекты по массовому набору сотрудников". В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.

1. Щелкните "Создать".
2. В поле "Проект по массовому набору сотрудников" введите значение.
3. В поле "Описание" введите значение.
4. В поле "Начало проекта" введите дату.
5. В поле "Конец проекта" введите дату.
6. Щелкните "Открыть проект".
7. Щелкните Да.
8. Щелкните "Создание должностей".
9. В поле "Количество" введите количество должностей, которые требуется создать.
    * Дата начала становится датой приема на работу для новых работников.  
    * Дата окончания становится датой увольнения для новых работников.  
    * Укажите тип новых работников: сотрудники или подрядчики.  
10. В поле "Задание" нажмите кнопку раскрывающегося списка, чтобы выбрать задание, для которого требуется создать должности.
11. В списке найдите и выберите требуемую запись.
12. В списке перейдите по ссылке в выбранной строке.
    * Значение эквивалента полной занятости по умолчанию берется из выбранного задания. Это значение можно изменить при необходимости.  
    * При желании выберите подразделение для новых должностей.  
13. Нажмите кнопку "OК".
