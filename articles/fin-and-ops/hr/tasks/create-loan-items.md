# <a name="create-loan-items"></a>Создать одалживаемые номенклатуры

[!include[task guide banner](../../includes/task-guide-banner.md)]

Номенклатуры временного пользования (одалживаемые номенклатуры) — это записи, позволяющие отслеживать физические номенклатуры, например телефоны или компьютеры, выдаваемые организацией работникам во временное пользование. Каждая физическая номенклатура должна иметь соответствующую номенклатуру временного пользования. В каждой записи арендованной номенклатуры должно быть описано, что было предоставлено в аренду, кто несет ответственность за аренду и количество дней, на которые была предоставлена в аренду номенклатура. Одновременно можно создать несколько номенклатур временного пользования, таких как ключи, карточки доступа или комплекты униформы. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="create-loan-types"></a>Создание типов займов
1. Перейдите в раздел "Управление персоналом" > "Работники" > "Одалживаемые номенклатуры" > "Типы займа".
2. Щелкните "Создать".
3. В поле "Тип займа" введите значение.
4. В поле "Описание" введите значение.
5. Введите количество дней, на которое может быть просрочен возврат номенклатур, принадлежащих к этому типу. 
6. Нажмите кнопку "Сохранить".
7. Закройте страницу.
8. Обновите страницу.

## <a name="create-loan-items"></a>Создание одалживаемых номенклатур
1. Перейдите в раздел "Управление персоналом" > "Работники" > "Одалживаемые номенклатуры" > "Одалживаемые номенклатуры".
2. Щелкните "Создать одалживаемые номенклатуры".
3. В поле "Кол-во" введите число.
4. В поле "Описание" введите значение.
5. В поле "Тип займа" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. Поиск и выбор требуемой записи в списке.
7. В списке перейдите по ссылке в выбранной строке.
8. Введите количество дней, на которое номенклатура может быть выдана во временное пользование.
    * Значение по умолчанию в поле "Планируемый возврат" на странице "Одолженное оборудование" вычисляется как текущая дата плюс это число.  
9. В поле "Ответственное лицо" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
10. Щелкните Выбрать.
11. В поле "Начальное значение" введите число.
12. В поле "Интервал" введите число.
13. В поле "Формат" введите значение.
    * Например, если начальный номер для номенклатуры временного пользования — 10, в поле "Формат" нужно ввести два символа числа.  
14. Нажмите кнопку "OК".
15. Обновите страницу.
