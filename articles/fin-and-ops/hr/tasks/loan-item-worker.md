# <a name="loan-items-to-a-worker"></a>Одалживание номенклатур работнику

[!include[task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как выдать номенклатуру во временное пользование работнику и зарегистрировать возврат номенклатуры работником. Работники могут также запрашивать выдачу номенклатур во временное пользование на портале самообслуживания сотрудников. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="loan-item-to-a-worker"></a>Выдача номенклатуры во временное пользование работнику
1. Перейдите в раздел "Управление персоналом" > "Работники" > "Одалживаемые номенклатуры" > "Одолженное оборудование".
2. Щелкните "Создать".
3. В поле "Человек" введите или выберите значение.
4. В поле "Одалживаемая номенклатура" введите или выберите значение.
5. В поле "Планируемый возврат" введите дату, когда сотрудник должен вернуть переданную во временное пользование номенклатуру.
6. Нажмите кнопку "Сохранить".
7. Закройте страницу.

## <a name="return-a-loan-item"></a>Возврат выданной во временное пользование номенклатуры
1. Перейдите в раздел "Управление персоналом" > "Работники" > "Одалживаемые номенклатуры" > "Одолженное оборудование".
2. Щелкните "Изменить".
3. Введите дату в поле "Фактический возврат".
