# <a name="develop-and-open-job-requisitions"></a>Разработка и открытие заявок на должность

[!include [task guide banner](../../includes/task-guide-banner.md)]

Проекты по набору сотрудников помогают управлять процессом найма. Для каждого проекта по набору сотрудников можно настроить сведения, например должность, для которой производится набор, имя агента по найму, статус проекта и отдел, в котором будет находиться должность. После создания проекта по набору сотрудников можно написать объявление о найме, опубликовать его на страницах портала самообслуживания сотрудников, связать заявления о приеме на работу с проектом и отслеживать действия для проекта. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF. Чтобы начать процедуру, перейдите в раздел "Управление персоналом" > "Набор сотрудников" > "Проекты по набору сотрудников" > "Проекты по набору сотрудников".

1. Щелкните "Создать".
2. В поле "Проект по набору сотрудников" введите значение.
3. В поле "Описание" введите значение.
4. В поле "Агентство по найму" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
5. Поиск и выбор требуемой записи в списке.
6. В списке перейдите по ссылке в выбранной строке.
7. Щелкните Выбрать.
8. В поле "Подразделение" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
9. В списке перейдите по ссылке в выбранной строке.
10. В поле "Задание" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
11. Поиск и выбор требуемой записи в списке.
12. В списке перейдите по ссылке в выбранной строке.
13. В поле "Количество вакансий" введите число.
14. В поле "Специалист по комплектации штата" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
15. Поиск и выбор требуемой записи в списке.
16. В списке перейдите по ссылке в выбранной строке.
17. Щелкните Выбрать.
18. В поле "Крайний срок подачи заявления" введите дату.
19. Щелкните "СМИ".
    * Проекты по набору сотрудников включают параметр для определения медиаплощадок, которые требуется использовать для рекламы открытых позиций.  
20. Щелкните "Создать".
21. В поле "СМИ" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
22. В списке перейдите по ссылке в выбранной строке.
23. В поле "Дата начала" введите дату.
24. В поле "Дата окончания" введите дату.
25. Нажмите кнопку "Сохранить".
26. Закройте страницу.
27. Щелкните "Объявление о приеме на работу".
28. Нажмите кнопку "Сохранить".
29. Закройте страницу.
30. Установите или снимите флажок "Отобразить на портале самообслуживания сотрудников".
    * Установите флажок "Отобразить на портале самообслуживания сотрудников", чтобы проект по набору сотрудников отображался для сотрудников на страницах портала самообслуживания сотрудников.  
31. Щелкните "Статус проекта по набору сотрудников".
32. Щелкните "Начать".
    * Статус "Начато" означает, что проект готов получать заявления.  
33. Нажмите кнопку "OК".
