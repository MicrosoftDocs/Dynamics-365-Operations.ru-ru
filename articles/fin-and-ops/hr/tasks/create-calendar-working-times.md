# <a name="create-calendar-and-generate-working-times"></a>Создание календаря и создание рабочих времен

[!include [task guide banner](../../includes/task-guide-banner.md)]

Календари описывают производительность и рабочее время операционных ресурсов. Эта процедура поможет определить рабочий календарь на основе шаблона рабочего времени. Чтобы выполнить эту процедуру, используйте компанию с демонстрационными данными USMF или собственные данные.

1. Перейдите в раздел "Все рабочие области" > "Управление жизненным циклом ресурса".
2. Щелкните "Календари".
3. Щелкните "Создать".
4. В поле "Календарь" введите значение.
    * Это код календаря, используемый в качестве ссылки при назначении календарей, таких как операционные ресурсы или группа ресурсов.  
5. В поле "Имя" введите значение.
6. В поле "Стандартный рабочий день в часах" введите число.
7. В списке пометьте выбранную строку.
8. Щелкните "Рабочее время".
9. Щелкните "Формирование операционных времен".
    * Создание рабочих часов для каждого дня периода, где можно будет планировать работу. С течением времени можно создать рабочее время для дополнительных периодов.  
10. В поле "Дата начала" введите дату.
    * Это первый день, в который должен начинаться этот календарь.  
11. В поле "Дата окончания" введите дату.
    * Это последний день, в который календарь открыт.  
12. В поле "Шаблон расписания" введите или выберите значение.
    * Шаблон рабочего времени определяет рабочие часы для каждого дня недели.  
13. Нажмите кнопку "OК".
14. Закройте страницу.
