# <a name="enroll-and-remove-benefits-from-workers"></a>Регистрация и удаление льгот для работников

[!include[task guide banner](../../includes/task-guide-banner.md)]

Эта процедура демонстрирует, как для одного работника можно зарегистрировать одну или несколько льгот, а также как для нескольких работников можно зарегистрировать льготу. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="enroll-a-single-worker-in-benefits"></a>Регистрация одного работника на льготы
1. Перейдите в раздел "Управление персоналом" > "Работники" > "Сотрудники"
2. В списке найдите и выберите требуемую запись.
3. Щелкните "Льготы".
4. Нажмите Создать.
5. В поле "Льгота" введите или выберите значение.
6. В поле "Начальная дата покрытия" введите дату и время.
7. В поле "Конечная дата покрытия" введите дату и время.
8. Разверните раздел "Бенефициары", если бенефициары должны быть добавлены к льготе. Можно также добавить иждивенцев на этой странице, если это применимо к льготе.
9. Можно также изменить сведения о регистрации льготы или удалить регистрацию на этой странице. По завершении внесения изменений в регистрацию льготы закройте страницу.

## <a name="enroll-multiple-workers-in-a-benefit"></a>Регистрация нескольких работников для льготы
1. Закройте страницу.
2. Перейдите в раздел "Управление персоналом" > "Работники" > "Сотрудники"
3. В списке пометьте выбранную строку.
4. В списке найдите и выберите требуемую запись.
5. Поиск и выбор требуемой записи в списке.
6. Щелкните "Зарегистрировать для льгот".
7. В поле "Льгота" введите или выберите значение.
8. В поле "Начальная дата покрытия" введите дату и время.
9. В поле "Конечная дата покрытия" введите дату и время.
10. Щелкните "Зарегистрировать".
11. Закройте страницу.
12. Перейдите в раздел "Управление персоналом" > "Льготы" > "Регистрация" > "Результаты регистрации льготы"
13. Найдите запись результатов льготы, которую ищете.
14. В списке перейдите по ссылке в выбранной строке.
15. Эта страница позволяет просматривать сотрудников, которые были зарегистрированы для получения льготы, а также любых других сотрудников, которые не были зарегистрированы.
