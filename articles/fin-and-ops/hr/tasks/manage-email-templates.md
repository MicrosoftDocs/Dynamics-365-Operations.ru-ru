# <a name="manage-email-templates"></a>Управление шаблонами электронной почты

[!include [task guide banner](../../includes/task-guide-banner.md)]

Информацию из базы данных вашей организации можно перенести в закладки в новом документе и использовать ее в шаблонах, помогающих эффективно переписываться с отправителями заявлений и кандидатами. Для этого необходимо создать шаблон, содержащий стандартный текст и закладки в местах вставки системных данных. Например, можно вставить адрес и контактные данные кандидата в документ Microsoft Word и использовать этот документ при переписке с этим кандидатом. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="select-which-bookmarks-to-use-in-your-email-templates"></a>Выбор закладок для использования в шаблонах сообщений электронной почты
1. Перейдите в раздел "Закладки приложения".
2. Найдите в списке требуемое действие по обмену корреспонденцией и выберите его.
3. Щелкните "Изменить".
4. В списке найдите и выберите требуемую запись.
    * Выберите поля, которые вы хотите использовать в шаблоне электронного сообщения для выбранного действия по обмену корреспонденцией, и переместите их в поля закладок.  
5. Закройте страницу.

## <a name="create-an-email-template"></a>Создание шаблона эл. почты
1. Перейдите в раздел "Управление персоналом" > "Набор сотрудников" > "Связь" > "Шаблоны электронной почты приложения".
2. Щелкните "Создать".
3. В поле "Соответствующее действие" выберите "Собеседование".
    * Выберите действие по обмену корреспонденцией, содержащее закладки, которые требуется использовать для данного типа переписки по электронной почте.  
4. В поле "Шаблон сообщения" введите значение.
5. В поле "Тема" введите значение.
6. В поле "Текст" введите значение.
7. Найдите в списке требуемое поле закладки и выберите его.
8. Продолжайте вводить электронное сообщение, вставляя, где необходимо, поля закладок.
    * Продолжайте вводить свое электронное сообщение, вставляя, где необходимо, поля закладок.  
9. Нажмите кнопку "Сохранить".
