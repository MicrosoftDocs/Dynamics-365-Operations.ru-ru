# <a name="employment-verification-using-form-i-9"></a>Проверка сотрудников с помощью формы I-9

[!include[task guide banner](../../../includes/task-guide-banner.md)]

Закон об иммиграционной реформе и контроле над иммиграцией требует, чтобы работодатели США проверяли наличие права на трудоустройство у вновь нанимаемых сотрудников. Эта процедура демонстрирует последовательность действий при записи документов, необходимых для проверки информации в форме I-9. В процедуре используется демонстрационная компания USMF.

1. Перейдите в раздел "Управление персоналом" > "Работники" > "Сотрудники".
2. Используйте экспресс-фильтр для поиска записей. Например, отфильтруйте поле "Имя" по значению "Vince".
3. Выбор сотрудника. Пример: Vince Prado
4. Разверните экспресс-вкладку "Личные данные".
5. Щелкните "Идентификационные номера".
6. Нажмите Создать.
7. Выберите тип удостоверения личности, запись о котором вы делаете. Пример: "Паспорт"
8. В поле "Число" введите значение.
9. Выберите значение "Да" в поле "Основной".
10. В поле "Описание" введите краткое описание записи удостоверения личности.
11. В списке выпускающих агентств выберите организацию, которая выдала работнику это удостоверения личности. Пример: "Правительство"
12. Введите дату выдачи работнику этого удостоверения личности соответствующей организации. Пример: 15.02.2011
13. Введите дату истечения срока действия этого удостоверения личности. Пример: 15.02.2021
14. Нажмите кнопку "Сохранить".
15. Закройте страницу.
16. Перейдите на вкладку "Занятость".
17. Щелкните "I-9".
18. Щелкните "Создать".
19. В поле "Право на трудоустройство" выберите один из вариантов.
    * Если сотрудник не является гражданином США, необходимо ввести его номер разрешения на постоянное место жительства или разрешения на въезд.  
20. Выберите вариант "GroupListA".
    * Выбранный список зависит от того, какого типа удостоверения личности предоставил работник. Работник должен предоставить один документ из списка A или один документа из списка B и C. Например, если сотрудник предоставил паспорт, можно выбрать список A. Однако если работник предоставил только свои водительские права и карточку социального обеспечения, необходимо выбрать список B и C.  
21. В поле "Тип документа I-9" выберите тип документа, который предоставил работник.
22. В поле "Номер документа" введите или выберите значение.
23. Нажмите кнопку "Сохранить".
