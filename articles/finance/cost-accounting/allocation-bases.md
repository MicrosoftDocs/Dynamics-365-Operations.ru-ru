# <a name="allocation-bases"></a>Базы распределения 

[!include [banner](../includes/banner.md)]

База распределения — база, по которой учет затрат распределяет накладные расходы. База распределения может быть количественной, например используемое машинное время, потребленные киловатт-часы (кВт-ч) или занятая площадь. Базы распределения используются главным образом для назначения накладных расходов в запасы, которые производятся. Например, отдел ИТ распределяет свои расходы по числу компьютеров, используемых в каждом отделе.

Существует три типа баз распределения в учете затрат:

- Предопределенные базы распределения элементов аналитики
- Базы распределения иерархии
- Базы распределения формулы

## <a name="predefined-dimension-member-allocation-bases"></a>Предопределенные базы распределения элементов аналитики

Предопределенные базы распределения элементов аналитики создаются автоматически, когда создается элемент аналитики одного из следующих типов:

- Элементы статистической аналитики
- Элементы аналитики элементов затрат

> [!NOTE]
> Предопределенные базы распределения элемента аналитики, основанные на элементе аналитики элемента затрат учитывают только значения от поставщика источника данных, например главной книги или бюджета.

## <a name="hierarchy-allocation-bases"></a>Базы распределения иерархии

Бухгалтеры по учету затрат могут вручную создавать базы распределения иерархии путем применения узла иерархии аналитики объекта затрат к существующей базе распределения. Таким образом можно ограничить диапазон исходной предопределенной базы распределения элемента аналитики. Одну предопределенную базу распределения элемента аналитики можно использовать для создания нескольких баз распределения иерархии. Диапазонами можно управлять в иерархии аналитик объекта затрат, связанной с базами распределения иерархии.

## <a name="formula-allocation-bases"></a>Базы распределения формулы

Базы распределения формулы позволяют определить дополнительные формулы для достижения правильной базу распределения. Можно вручную создать базы распределения формул.

При создании базы распределения формул следует выбрать, какая статистическая аналитика и аналитика элемента затрат должны быть основой для формулы. Все базы распределения, полученные из ранее выбранных аналитик, можно использовать в базе распределения формулы.

> [!NOTE]
> Предварительно определенные базы распределения формул могут использоваться для определения новой базы распределения формул.

В коэффициентах баз распределения формул можно создать псевдоним и связать его с базой распределения или константой. Затем псевдонимы используются для определения формулы.

Чтобы определить формулу, можно использовать следующие операторы.

| Символы | Текст           |
|---------|----------------|
| ( )     | Круглые скобки    |
| \<      | Меньше   |
| \>      | Больше    |
| +       | Сложение       |
| –       | Вычитание    |
| \*      | Умножение |

Традиционные выписки **IF** не поддерживаются. Тем не менее можно создавать выписки и проверять,выполняются ли они.

| Оператор | Проверка | Результат |
|-----------|------------|--------|
| a \> b    | Истина       | 1      |
| a \> b    | Ложь      | 0      |

### <a name="example-2-an-advanced-formula"></a>Пример 2. Расширенная формула

| Псевдоним | Константа  | База распределения |
