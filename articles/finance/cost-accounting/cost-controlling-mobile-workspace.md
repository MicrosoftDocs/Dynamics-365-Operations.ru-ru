# <a name="cost-controlling-mobile-workspace"></a>Мобильная рабочая область управления затратами

[!include [banner](../includes/banner.md)]
[!include [mobile app deprecation](../../fin-ops-core/dev-itpro/includes/mobile-app-deprecation-banner.md)]

В этой статье содержится информация о мобильной рабочей области **Управление затратами**. Эта рабочая область позволяет менеджерам центра затрат просматривать сведения о производительности центра затрат в любое время и в любом месте.

Эту мобильную рабочую область можно использовать с мобильным приложением для управления финансами и операциями (Dynamics 365).

## <a name="overview"></a>Обзор
Мобильная рабочая область **Управление затратами** обеспечивает мгновенное отображение текущей производительности центров затрат путем сравнения фактических затрат с бюджетными затратами. Можно выполнить детализацию до статуса отдельных элементов затрат.

Например, сотрудник получает приглашение на международную конференцию, но организация должна оплатить все командировочные расходы. Сотрудник запрашивает у своего менеджера разрешение на посещение этой конференции. Менеджер открывает мобильную рабочую область **Управление затратами** на своем мобильном устройстве и смотрит, имеется ли бюджет для оплаты посещения конференции сотрудником.

### <a name="data-security"></a>Безопасность данных
Данные в мобильной рабочей области **Управление затратами** защищены учетными данными пользователя. Менеджеры центра затрат могут просматривать сведения только для своего собственного центра затрат. Управление безопасностью уровня доступа осуществляется в модуле **Учет затрат**.

Бухгалтеры по учету затрат определяют конфигурацию мобильной рабочей области **Управление затратами** в модуле **Учет затрат**. После публикации рабочей области в мобильном приложении она становится доступна в приложении. Таким образом, все менеджеры центров затрат в рамках организации могут просматривать данные в одинаковом формате.

### <a name="actions-views-and-links"></a>Действия, представления и ссылки
Мобильная рабочая область **Управление затратами** содержит действия, представления и ссылки:

-   **Действия:**

    -   Используйте **Выбрать конфигурацию** для выбора макета.
    -   Используйте **Выбрать объект затрат** для выбора центров затрат, по которым требуется фильтровать данные.
    
        > [!NOTE]
        > Центры затрат, которые отображаются в списке, зависят от уровня доступа, предоставленного в модуле **Учет затрат**.

-   **Представления:** на основании выбранных действий и конфигурации в модуле **Учет затрат** можно просмотреть следующую информацию на карточках.

    -   Факт/бюджет (текущий период)
    -   Факт/Пересмотренный бюджет (текущий период)
    -   Факт/бюджет (предыдущий период)
    -   Факт/пересмотренный бюджет (предыдущий период)
    -   Факт/бюджет (с начала года)
    -   Факс/пересмотренный бюджет (с начала года)

    В каждой карточке показаны следующие суммы: фактическая, бюджет, отклонение и % отклонения.

-   **Ссылки:**

    -   Сведения для текущего периода
    -   Сведения для предыдущего периода
    -   Сведения с начала года

    При выборе ссылки отображается карта для каждого элемента затрат. На каждой карточке отображаются следующее суммы: фактическая, бюджетная, отклонение от бюджета, отклонение от бюджета в процентах, пересмотренный бюджет, отклонение от пересмотренного бюджета и отклонение от пересмотренного бюджета в процентах.

## <a name="download-and-install-the-mobile-app"></a>Загрузите и установите мобильное приложение
Загрузите и установите мобильное приложение для управления финансами и операциями (Dynamics 365):

-   [Для телефонов Android](https://go.microsoft.com/fwlink/?linkid=850662)
-   [Для iPhone](https://go.microsoft.com/fwlink/?linkid=850663)

## <a name="sign-in-to-the-mobile-app"></a>Войдите в систему в мобильном приложении

1.  Запустите приложение на мобильном устройстве.
2.  Введите URL-адрес Dynamics 365.
3.  При первом входе появится запрос имени пользователя и пароля. Введите свои учетные данные.
4.  После входа будут показаны доступные рабочие области для вашей компании. Обратите внимание, что если позже системный администратор опубликует новую рабочую область, вам потребуется обновить список мобильных рабочих областей.

[![Потянуть для обновления.](./media/pull-to-refresh-list-of-workspaces-183x300.png)](./media/pull-to-refresh-list-of-workspaces.png)

## <a name="view-the-performance-of-your-cost-center-by-using-the-cost-controlling-mobile-workspace"></a>Просмотр производительности центра затрат с помощью мобильной рабочей области управления затратами

1.  На мобильном устройстве выберите рабочую область **Управление затратами**.
2.  Выберите **Управление объектами затрат**.
3.  Выберите **Действия**.
4.  Выберите **Выбрать конфигурацию**, чтобы выбрать макет управления затратами.
5.  Выберите **Готово**.
6.  Выберите **Действия**.
7.  Выберите **Выбрать объект затрат** для выбора центров затрат, к которым вам предоставлен доступ.
8.  Выберите **Готово**.
9.  Просмотрите общую производительность вашего центра затрат.
10. Выберите ссылку **Сведений для текущего периода**.
11. Просмотрите производительность конкретного центра затрат.
12. Можно также выполнять поиск конкретных элементов затрат.
