# <a name="define-cost-control-units"></a>Определение единиц управления затратами

[!include [banner](../../includes/banner.md)]

Эта процедура показывает, как определить единицы управления затратами в книге учета затрат. Вам нужно выбрать аналитику объектов затрат, для которой вы хотите управлять затратами в книге. Элементы аналитики объектов затрат преобразуются в объекты затрат единиц управления затратами. В качестве компании с демонстрационными данными для создания этой процедуры используется USP2.

1. Перейдите в раздел "Учет затрат" > "Настройка главной книги" > "Книги учета затрат".
2. В списке найдите и выберите требуемую запись.
3. Щелкните "Единицы управления".
4. Щелкните "Создать".
5. В поле "Имя" введите значение.
6. В поле "Аналитика" введите или выберите значение.
7. Нажмите кнопку "Сохранить".
8. Щелкните "Создать".
9. В поле "Имя" введите значение.
10. В поле "Аналитика" введите или выберите значение.
11. Нажмите кнопку "Сохранить".
