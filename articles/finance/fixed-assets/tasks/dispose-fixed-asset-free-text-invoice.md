# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="bd3c7-103">Выбытие основного средства с использованием накладной с произвольным текстом</span><span class="sxs-lookup"><span data-stu-id="bd3c7-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="bd3c7-104">В этой теме объясняется порядок списания основного средства с использованием накладной с произвольным текстом.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="bd3c7-105">В области переходов выберите **Модули > Расчеты с клиентами > Накладные > Все накладные с произвольным текстом**.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="bd3c7-106">Выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-106">Select **New**.</span></span>
3. <span data-ttu-id="bd3c7-107">В поле **Счет клиента** введите или выберите значение.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="bd3c7-108">Проверьте дату **Накладной** по умолчанию и отредактируйте ее, если необходимо.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="bd3c7-109">Проверьте остальные поля заголовка по умолчанию, такие как **Валюта**, и отредактируйте их, если необходимо.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="bd3c7-110">В разделе **Строки накладной** проверьте поля **Описание** и **Счет ГК**.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="bd3c7-111">Проверьте поля **Налоговая группа** и **Налоговая группа номенклатур**, подставленные по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="bd3c7-112">Введите значения **Цена за единицу** или **Сумма** продажи основного средства.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="bd3c7-113">Выберите раздел **Сведения по строке**.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="bd3c7-114">Введите или выберите значение в поле **Основное средство**.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="bd3c7-115">Выберите **Разнести**.</span><span class="sxs-lookup"><span data-stu-id="bd3c7-115">Select **Post**.</span></span>
