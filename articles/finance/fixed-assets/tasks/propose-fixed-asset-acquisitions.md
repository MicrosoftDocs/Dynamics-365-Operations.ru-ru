# <a name="propose-fixed-asset-acquisitions"></a>Предложение по приобретению ОС

[!include [banner](../../includes/banner.md)]

Этот раздел описывает, как приобрести основное средство с помощью предложения по приобретению в журнале основных средств. В нем используется роль бухгалтера и демонстрационные данные для юридического лица USMF. Чтобы получить основные средства с помощью журнала предложений по основным средствам, необходимо сначала создать запись ОС, а затем определить цену приобретения в журнале активов.

1. В области перехода, перейдите к **Модули > Основные средства > Записи журнала > Журнал основных средств**.
2. Выберите **Создать**.
3. В поле **Имя** введите или выберите значение.
4. На панели операций выберите **Строки**.
5. Выберите **Предложения**.
6. Выберите в **Предложение по вводу в эксплуатацию**.
7. Выберите **Фильтр**. Выберите **Сброс**, чтобы сбросить вне предыдущие значения.
8. Выберите строку **Инв. номер ОС**.
9. В поле **Критерии** введите или выберите значение. Настройте оставшиеся критерии для основных средств, которые необходимо приобрести с этим предложением.  
10. Выберите **OK** дважды, чтобы выйти из панели.
- Проверьте созданные строки проводки.  
- Только основные средства, для которых заданы параметры даты приобретения и цены приобретения в книге, будут включены в предложение по приобретению.  
11. На странице выберите вкладку **Книги**.
12. Выберите **Разнести**.
