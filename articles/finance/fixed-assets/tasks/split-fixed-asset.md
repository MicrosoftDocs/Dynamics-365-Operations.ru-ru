# <a name="split-a-fixed-asset"></a>Разбивка ОС

[!include [banner](../../includes/banner.md)]

В этой теме показано, как разделить процент одной модели стоимости основного средства в новую модель стоимости основного средства. В нем используется роль бухгалтера и демонстрационные данные USMF.

## <a name="create-a-new-fixed-asset"></a>Создание нового основного средства

1. В области перехода, перейдите к разделу **Модули \> Основные средства \> Основные средства \> Основные средства**.
2. Выберите **Создать**.
3. В поле **Группа ОС** введите или выберите значение. Обратите внимание на номер основного средства для использования в процессе разбиения позже.
4. В поле **Имя** введите значение.
5. Закройте форму.

## <a name="split-a-fixed-asset"></a>Разбивка ОС

Перед разбиением полностью амортизированного актива статус журнала ОС должен быть изменен вручную с **Закрыто** на **Открыто**. Этот шаг необходим, поскольку статус книги должен быть **Открыто**, если необходимо разнести проводки для актива (например, для продажи выбытия). Также необходимо включить параметр **Разрешить несколько проводок в одном ваучере** на вкладке **Общие** на странице **Параметры главной книги**. После изменения статуса книги активов и разрешения нескольких проводок в одном ваучере выполните следующие шаги, чтобы разделить актив.

1. Найдите и выберите в списке ссылку на основное средство для разбиения.
2. Выберите **Книги**. Выберите книгу для разбиения в новое основное средство.
3. Выберите **Функции**.
4. Выберите **Разбивка ОС**.
5. В поле **В основные средства** введите или выберите значение.
6. В поле **В книгу** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
7. В поле **Дата проводки** введите дату.
8. В поле **Процент** введите число.
9. В поле **Наименование журнала** введите или выберите значение.
10. Нажмите **ОК**.

## <a name="post-the-journal-transaction"></a>Разноска проводки по журналу

1. В области перехода, перейдите к разделу **Модули \> Основные средства \> Записи журнала \> Журнал основных средств**.
2. В списке выберите журнал, созданный в результате процесса разбиения.
3. Выберите **Строки**.

    - Проверьте созданные строки журнала.
    - Проводка о корректировке приобретения создается для исходного основного средства для уменьшения стоимости на процент, указанный в процессе разбиения.
    - Проводка приобретения создается для нового основного средства на ту же сумму.

4. Выберите **Разнести**.
