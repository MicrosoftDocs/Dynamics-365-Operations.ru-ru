# <a name="change-a-fixed-asset-group"></a>Изменение группы ОС

[!include [banner](../../includes/banner.md)]

Основные средства должны быть назначены правильной группе основных средств. Группа основных средств используется в следующих ситуациях:

 - Создание запросов и отчетов

 - Настройка новых ОС

 - Интеграция главных книг и проводок разноски ОС с соответствующими счетами ГК

В данном руководстве используется демонстрационная компания USMF.

1. Перейдите в раздел "Основные средства" > "Основные средства" > "Основные средства".
2. Выберите основное средство, для которого требуется изменить группу ОС.
3. Щелкните "Изменение группы основных средств".
4. В поле "Новая группа" введите или выберите значение.
5. Выберите этот параметр, чтобы назначить инвентарный номер ОС для выбранного ОС.
    * Поле "Инв. номер ОС" доступно, если выбран параметр "Новый инвентарный номер ОС".   Если для основных средств настроена автоматическая нумерация, в этом поле отображается следующий доступный номер ОС. Этот номер можно изменить.   Если настроена нумерация вручную, это поле остается пустым, и следует ввести новый номер ОС.     
6. Нажмите кнопку "OК".
7. Щелкните Да.
