# <a name="set-up-fixed-asset-groups"></a>Настройка групп ОС

[!include [banner](../../includes/banner.md)]

В этой теме показано, как создать новую группу основных средств. В нем используется роль бухгалтера и демонстрационные данные для юридического лица USMF.

1. В области перехода, перейдите к **Модули > Основные средства > Настройка > Группы ОС**.
2. Выберите **Создать**.
3. В поле **Группа ОС** введите значение.
4. В поле **Имя** введите значение. Параметры "Автонумерация ОС" и "Код номерной серии" в группе **Основное средство** переопределяют настройки в параметрах основных средств. Значения можно изменить здесь, если основные средства в этой группе основных средств имеют нумерацию, отличную от других групп.  
5. Выберите **Книги**.
6. В поле **Книга** введите или выберите значение. Если в поле **Расчет амортизации** задано значение **Да**, журнал основных средств будет включен в предложения по амортизации. Если в поле **Расчет амортизации** задано значение **Нет**, основное средство не будет амортизироваться автоматически.  
7. Задайте срок службы основного средства в годах. Обратите внимание, что значение поля **Периоды амортизации** рассчитывается после настройки срока службы.  
8. В поле **Соглашение по амортизации** выберите вариант.
9. Закройте страницу.
