# <a name="calculate-fixed-asset-depreciation-across-legal-entities"></a>Расчет амортизации ОС в разных юридических лицах

[!include [banner](../../includes/banner.md)]

Можно запустить амортизации основных средств для нескольких юридических лиц за один шаг. Эта процедура показывает, как настроить и запустить процесс для нескольких юридических лиц. В нем используется роль бухгалтера и демонстрационные данные для юридического лица USMF.


## <a name="set-up-cross-company-depreciation-run-journals"></a>Настройка журналов запуска амортизации по нескольким компаниям
1. Перейдите в раздел "Основные средства" > "Настройка" > "Параметры основных средств".
2. Разверните раздел "Предложения по ОС".
3. Нажмите кнопку Добавить.
4. В поле "Слой разноски" введите или выберите значение.
5. В поле "Наименование журнала" введите или выберите значение.
    * Повторите настройку журнала на странице "Параметры основных средств" в каждом юридическом лице.  

## <a name="depreciation-run"></a>Запуск амортизации
1. Перейдите в раздел "Основные средства" > "Записи в журнале" > "Создать предложение по амортизации".
2. В поле "Слой разноски" введите или выберите значение.
    * Имя журнала по умолчанию берется из параметров основных средств. Его можно изменить здесь для текущего юридического лица.  
3. В поле "Дата окончания" введите дату.
    * Выберите юридические лица для включения в прогон амортизации.  
    * В списке будут отображаться только юридические лица с журналами, настроенными для предложений ОС на странице "Параметры основных средств".  
4. В поле "Разнести журналы" выберите значение "Да".
    * Поля фильтрации включают все основные средства, группы и книги для юридических лиц, выбранных для этого прогона амортизации.  
    * Параметр пакетной обработки по умолчанию включен. Если этот параметр включен, создание и разноска журнала амортизации будут выполняться в фоновом режиме.  
5. Щелкните "Создать журнал".
6. Перейдите в раздел "Основные средства" > "Записи в журнале" > "Журнал основных средств".
