# <a name="budget-funds-available"></a>Доступные бюджетные средства

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

В этой статье представлена функция доступных бюджетных средств и представлены сведения, которые могут помочь выполнить настройку бюджетного контроля для оптимизации управления финансовыми ресурсами организации.

## <a name="enhanced-calculation-feature-for-budget-funds-available"></a>Расширенная функция расчета для доступных бюджетных средств

Функция **Отслеживать суммы только в расчетах доступных бюджетных средств** позволяют отслеживать базовые таблицы бюджетного контроля для типов документов и состояний на основе настроек на странице **Определение параметров бюджетного контроля**.

Некоторые параметры конфигурации бюджетного контроля должны иметь определенные настройки для надлежащей работы функции. Эти параметры устанавливаются или снимаются на вкладке **Доступные бюджетные средства** на странице **Определение параметров бюджетного контроля**. В следующей таблице показаны параметры, которые требуются для функции доступных бюджетных средств.

| Если этот параметр выбран | Этот параметр также должен быть выбран |
| ------------------------- | -------------------------------- |
| Резервирования бюджета по предварительным бюджетным обязательствам | Резервирования бюджета по бюджетным обязательствам *и* фактическим расходам |
| Резервирования бюджета по бюджетным обязательствам | Фактические расходы |
| Резервирования бюджета для бюджетных обязательств с документами типа заявок на закупку | Резервирования бюджета по предварительным бюджетным обязательствам |

Эта функция влияет только на новые документы. Суммы для существующих документов по-прежнему будут отслеживаться и отображаться в запросе статистики контроля бюджета, пока не будет изменена настройка доступных бюджетных средств и активирована новая конфигурация бюджетного контроля. В этот момент данные отслеживания бюджета будут удалены для документов, которые были удалены из расчета доступных бюджетных средств.

Рекомендуется оставить параметр **Неразнесенные фактические расходы** снятым. Если этот флажок установлен, требующий большого времени расчет бюджетного контроля будет выполняться для неразнесенных документов, таких как ожидаемые накладные поставщиков.
