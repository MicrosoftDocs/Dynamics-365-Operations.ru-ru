# <a name="consolidation-account-groups-and-additional-consolidation-accounts"></a>Группы счетов консолидации и дополнительные счета консолидации

[!include [banner](../includes/banner.md)]

В этой статье представлены сведения о группах счетов консолидации и о дополнительных счетах консолидации и объясняется, как они используются.

## <a name="consolidation-account-groups"></a>Группы счетов консолидации

Группы счетов консолидации позволяют создавать группы счетов, которые требуется использовать для консолидации данных. Обычно группа счетов консолидации представляет утвержденный органами власти план счетов. Группа счетов консолидации также может сопоставлять счета с группой, которая определена головным офисом компании. Группы счетов консолидации можно найти в области **Настройка** модуля **Консолидации**. При добавлении новой группы вводится уникальный идентификатор для группы счетов, а также имя.

## <a name="additional-consolidation-accounts"></a>Дополнительные счета консолидации
Дополнительные счета консолидации позволяют назначить счет из существующего плана счетов для группы счетов консолидации. Затем можно указать значение и имя счета консолидации. 

Дополнительные счета консолидации можно найти в области **Настройка** модуля **Консолидации**. При создании нового счета консолидации необходимо указать следующие сведения:

-   **Счет ГК** — это поле является полем поиска, которое показывает все счета ГК, основанные на плане счетов, выбранном на странице. При выборе счета его имя автоматически вводится в поле **Наименование счета ГК**.
-   **Группы счетов консолидации** — это поле используется для указания группы, которой будет назначен счет. При консолидации двумя разными способами необходимо добавить один и тот же счет во все четыре группы счетов консолидации.
-   **Счет консолидации** — введите значение счета консолидации. Это значение не обязательно должен быть счетом из плана счетов. Это может быть любое требуемое вам значение.
-   **Имя счета консолидации** — введите имя счета, которое должно отображаться в запросах и отчетах.
-   **Уровень SAT** — это поле используется для передачи отчета по выпискам по счетам налоговым органам Мексики. 

После завершения создания групп счетов консолидации и дополнительных счетов консолидации можно выбрать эту группу в процессе интерактивной консолидации.


Дополнительные сведения см. в разделе [Создание групп консолидации и дополнительных счетов консолидации](../general-ledger/tasks/create-consolidation-groups.md).
