# <a name="set-up-workflows-for-expense-management"></a>Настройка рабочих процессов для управления расходами

[!include [banner](../includes/banner.md)]

Можно настроить workflow-процесс, который используется для просмотра и утверждения документов командировок и расходов. Документы, для которых может быть определен workflow-процесс, включают в себя отчеты о расходах, заявки на командировки и запросы денежных авансов.

Бизнес-правило представляет бизнес-процесс. Он определяет путь движения документа по системе и указывает, кто должен выполнить ту или иную задачу или утвердить тот или иной документ. Имеется несколько преимуществ использования системы workflow-процессов в организации:

-   **Непротиворечивые процессы** — можно определить процесс утверждения для конкретных документов, таких как заявки на покупку и отчеты по расходам. Использование системы workflow-процессов позволяет обеспечить обработку и утверждение документов непротиворечивым и эффективным образом.

-   Видимость процессов — можно отслеживать статус, историю и метрики производительности конкретного экземпляра workflow-процесса. Это позволяет определить, должны ли быть внесены изменения в workflow-процесс для повышения эффективности.

-   Централизованный список работ — пользователи могут просматривать централизованный список работ для просмотра назначенных им задач и утверждений workflow-процесса. 

**Могут быть созданы следующие типы workflow-процессов**

В следующей таблице перечислены типы workflow-процессов, которые можно создавать в модуле **Расходы**.


|              <strong>Тип</strong>              |                   <strong>Тип, который будет использоваться</strong>                   |
|-------------------------------------------------|-----------------------------------------------------------------------|
|         <strong>Отчет о расходах</strong>         |            Создайте workflow-процессы утверждения для отчетов о расходах.             |
|  <strong>Автоматическая разноска отчета о расходах</strong>   |        Создайте workflow-процессы автоматической разноски для отчетов о расходах.        |
|       <strong>Элемент строки расхода</strong>        |     Создайте workflow-процессы утверждения для номенклатур строк в отчетах о расходах.      |
| <strong>Автоматическая разноска номенклатуры строки расхода</strong> | Создайте workflow-процессы автоматической разноски для номенклатур строк в отчетах о расходах. |
|       <strong>Заявка на командировку</strong>       |          Создайте workflow-процессы утверждения для заявок на командировки.           |
|      <strong>Запрос денежного аванса</strong>      |         Создайте workflow-процессы утверждения для запросов на денежный аванс.          |
|        <strong>Возмещение НДС</strong>        | Создайте workflow-процессы утверждения для возмещения НДС.  |
