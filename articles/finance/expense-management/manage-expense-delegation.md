# <a name="manage-expense-delegation"></a>Управление делегированием расходов

[!include [banner](../includes/banner.md)]

Пользователь с делегированием расходов может создавать отчеты о расходах и управлять ими от имени другого сотрудника в организации.

## <a name="configuring-expense-delegation"></a>Настройка делегирования расходов

Чтобы настроить пользователя как делегата по расходам , перейдите **Управление расходами > Настройка > Общие > Делегаты**, чтобы открыть страницу **Делегаты**. Выберите **Создать**, а затем выберите сотрудника, для которого будет назначен определенный делегат. Введите псевдоним делегированного пользователя и даты начала и окончания периода делегирования.

## <a name="managing-expense-delegation-on-behalf-of-another-employee"></a>Управление делегированием расходов от имени другого сотрудника

Если включен ключ функции **Включить страницу списка делегатов расходов**, на страницу **Делегированные мне расходы** можно перейти **Управление расходами > Мои расходы > Делегированные мне расходы**.

Делегированные пользователь может быстро выполнить фильтрацию и поиск существующих отчетов по расходам, которые были делегированы данному пользователю. Пользователь также может быстро создать новый отчет о расходах от имени других пользователей, щелкнув **Создать отчет по расходам**.

Делегированные пользователи могут также создавать отчеты о расходах и управлять ими от имени других сотрудников, перейдя **Управление расходами > Мои расходы > Отчеты по расходам** и нажав кнопку **Открыть расходы другого пользователя**.
