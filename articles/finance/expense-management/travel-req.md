# <a name="travel-requisitions"></a><span data-ttu-id="a072b-104">Заявки на командировки</span><span class="sxs-lookup"><span data-stu-id="a072b-104">Travel requisitions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a072b-105">*Заявка на командировку* является исходным документом, в котором перечислены расходы, которые требуются для целей командировки.</span><span class="sxs-lookup"><span data-stu-id="a072b-105">A *travel requisition* is a source document that lists the expenses that will be incurred for the purpose of travel.</span></span> <span data-ttu-id="a072b-106">Заявка на командировку представляется на рассмотрение, а затем может быть использована для авторизации расходов.</span><span class="sxs-lookup"><span data-stu-id="a072b-106">A travel requisition is submitted for review and can then be used to authorize expenses.</span></span>

<span data-ttu-id="a072b-107">В организации может требоваться, чтобы заявка на командировку подавалась перед тем, как сотрудник понес любые расходы, которые оплачивает организация.</span><span class="sxs-lookup"><span data-stu-id="a072b-107">Your organization might require that a travel requisition be submitted before employees incur any expense that is charged to the organization.</span></span> <span data-ttu-id="a072b-108">Это требование применимо независимо от того, оплачивает ли сотрудник расходы кредитной картой компании, тратит наличные деньги, полученные авансом, или оплачивает расходы собственными средствами, которые затем будут ему возвращены компанией.</span><span class="sxs-lookup"><span data-stu-id="a072b-108">This requirement applies, regardless of whether employees charge expenses to a corporate credit card, spend cash that they received from a cash advance, or incur out-of-pocket expenses that will be reimbursed by the organization.</span></span>

<span data-ttu-id="a072b-109">Заявки на командировки и политики командировок могут быть использованы для бюджетного контроля.</span><span class="sxs-lookup"><span data-stu-id="a072b-109">Travel requisitions and policies can be used to help with budget control.</span></span> <span data-ttu-id="a072b-110">Например, если организация занимается проектом Фиксированная цена, для которого необходимы командировки, то расходы на командировки должны проходить по бюджету проекта.</span><span class="sxs-lookup"><span data-stu-id="a072b-110">For example, if your organization is working on a fixed-price project that requires travel, the travel expenses of the project's team members must fit within the budget for the project.</span></span> <span data-ttu-id="a072b-111">Требуя утверждения расходов на командировку перед самой командировкой, организация может помочь сохранить проект в рамках бюджета.</span><span class="sxs-lookup"><span data-stu-id="a072b-111">By requiring that travel expenses be approved before they are incurred, the organization can help make sure that the project remains within budget.</span></span>

<span data-ttu-id="a072b-112">Политики заявок на командировку можно настроить на странице **Заявка на командировку**.</span><span class="sxs-lookup"><span data-stu-id="a072b-112">You can set up policies for travel requisitions from the **Travel requisition** page.</span></span>
