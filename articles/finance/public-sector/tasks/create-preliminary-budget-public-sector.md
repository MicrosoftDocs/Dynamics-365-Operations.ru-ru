# <a name="create-a-preliminary-budget-for-public-sector"></a>Создание предварительного бюджета для государственного сектора

[!include [banner](../../includes/banner.md)]

Вы можете создать предварительные записи бюджетного регистра для определенной бюджетной модели и значений аналитик. После того как фактический бюджет будет утвержден, можно будет создать исходные записи бюджетного регистра. Эта процедура была создана с использованием демонстрационных данных компании PSUS в разделе государственного сектора.

1. Перейдите в раздел **Бюджетирование > Записи бюджетного регистра**.
2. Нажмите кнопку **Создать**.
3. В поле **Бюджетная модель** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. Поиск и выбор требуемой записи в списке.
5. В поле **Код бюджета** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. Поиск и выбор требуемой записи в списке.
7. В поле **Код основания** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
8. В списке щелкните требуемую запись.
9. Нажмите кнопку **Сохранить**.
10. Щелкните **Добавить строку**.
    * Необязательно. Если вы хотите изменить дату в одном из заголовков, введите новую дату. Эта дата определяет финансовый период, для которого будет записан бюджет. При просмотре руководства по задаче для заполнения других полей щелкните **Разблокировать** в верхней части страницы.  
11. В поле **Структура счета** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
12. Поиск и выбор требуемой записи в списке.
13. В поле **Значения аналитик** укажите требуемые значения.
14. В поле **Сумма** введите число.
    * Можно также ввести тип суммы.  
15. В поле **Валюта** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
16. В списке найдите и выберите требуемую запись.
17. Нажмите кнопку **Сохранить**.
18. Щелкните **Обновить сальдо бюджета**.
19. Щелкните **Обновить**.
    * Чтобы просмотреть результаты обновления, щелкните **Сведения о сообщении** на синей панели.
