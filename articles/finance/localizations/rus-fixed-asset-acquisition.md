# <a name="acquiring-fixed-assets-and-putting-them-into-operation-russia"></a>Приобретение ОС и их ввод в эксплуатацию (Россия)

[!include [banner](../includes/banner.md)]

Следующие амортизируемые основные средства могут приобретаться и вводиться в эксплуатацию.

- Купленные основные средства
- Основные средства, получаемые в качестве вклада в уставный капитал
- Основные средства, получаемые по договору дарения (т. е. получаемые без компенсации)
- Основные средства, получаемые в обмен на другое имущество
- Неучтенные основные средства, обнаруженные в ходе инвентаризации
- Арендованные основные средства, возращаемые арендатором

В сценариях, в которых фигурируют эти основные средства, используется кредитовый счет. Кредитный счет предоставляет несколько вариантов для создания корректирующих проводок в книге учета. Для указания счета создайте вручную проводку в журнале основных средств. Также можно на странице **Профили разноски** (**Основные средства (Россия) > Настройка > Профили разноски)** настроить профили разноски для проводок по различным типам основных средств.

Стоимость приобретенного основного средства может отличаться от стоимости покупки. Например, стоимость может быть увеличена за счет накладных расходов на приобретение, например расходов на испытания и работы по вводу в эксплуатацию.

Прежде чем разносить ваучер приобретений основных средства, необходимо проверить параметры амортизации на странице **Модели стоимости ОС** (**Основные средства (Россия)** > **Основные средства** и в области действий выберите **Модели стоимости**). Эти параметры включают группу амортизации, срок службы группы ОС и метод амортизации.

При разноске проводки **Ввод в эксплуатацию** происходят следующие изменения:

- Создается проводка модели основных средств типа **Ввод в эксплуатацию**.
- Создаются проводки по ваучеру с типом проводки **Основные средства (Россия)** и типом разноски **Основные средства (Россия), дебет**.
- В случае сборки основных средств создаются проводки по запасам со статусом расхода **Продано**.
- Баланс по основным средствам изменяется для каждой модели стоимости в диалоговом окне **Баланс по ОС** (выберите **Основные средства (Россия)** > **Основные средства**, выберите **Модели стоимости** в области действий, а затем на странице **Модели учета основных средств** в области действий выберите **Баланс**).

Ввести в эксплуатацию можно одно основное средство, группу основных средств или все основные средства со статусом **Куплено** или **Запланировано**.

Существует два способа приобретения основных средств (т. е. их ввода в эксплуатацию):

- Можно вручную создать проводку для одного основного средства.
- Можно автоматически сформировать проводку для одного основного средства, группы основных средств или всех основных средств со статусом **Куплено** или **Запланировано**.

## <a name="acquire-a-fixed-asset"></a>Приобретение основного средства 

Выполните следующие действия, чтобы создать проводки для одного основного средства с помощью журнала основных средств.

1. Выберите **Основные средства (Россия) > Журналы > Журнал ОС**.
2. Выберите **Создать**, чтобы создать журнал ОС.
3. В поле **Имя** выберите имя журнала. В поле **Описание** введите описание журнала.
4. В области действий выберите **Строки**, чтобы открыть страницу **Ваучер журнала**.
5. Выберите **Создать**, чтобы создать ваучер журнала.
6. В диалоговом окне **Добавление в журнал** в поле **Дата проводки** задайте дату проводки.
7. В поле **Тип проводки** выберите **Ввод в действие**.
8. В поле **Инвентарный номер ОС** выберите инвентарный номер основного средства.
9. В полях **Амортизационная премия** и **Код основания** выберите код амортизационной премии и код основания, если эти значения необходимы.

    > [!NOTE]
    > Изменить значение поля **Модель стоимости** в этом диалоговом окне нельзя, потому что проводки типа **Ввод в эксплуатацию** создаются одновременно для всех моделей стоимости.

10. Нажмите **ОК**. Строки, которые имеют тип **Ввод в эксплуатацию** для моделей стоимости и которые зарегистрированы на счете основных средств, создаются на странице **Ваучер журнала**.
11. В области действий выберите **Проверить** > **Проверить** для проверки проводок.
12. В области действий выберите **Разнести** > **Разнести**, чтобы разнести проводки.

    Создаются транзации основного средства и главной книги. Статус основного средства меняется на **Эксплуатация**.

    > [!NOTE] 
    > Можно изменить текст в поле **Текст проводки** поле и сумму приобретенного основного средства в поле **Сумма**. По умолчанию в поле **Сумма** установлена предлагаемая цена покупки.

    Значения полей **Тип корр. счета** и **Корр. счет** основаны на выбранном профиле разноски. Эти значения можно изменить.

Выполните следующие действия, чтобы создать проводки сразу для нескольких основных средств.

1. На странице **Ваучер журнала** (т. е. в строках журнала основных средств) в панели действий выберите **Групповые операции** \> **Ввод в эксплуатацию**.
2. В диалоговом окне **Ввод в эксплуатацию** на экспресс-вкладке **Включаемые записи** выберите **Фильтр**, чтобы открыть диалоговое окно **Основные средства**, где можно указать значения для одного основного средства, списка основных средств или группы основных средств. Закончив, выберите **OK**.
3. Введите значения в поля **Дата проводки**, **Амортизационная премия** и **Код причины**, если эти значения необходимы.
4. Нажмите **ОК**. Строки, которые имеют тип **Ввод в эксплуатацию** для моделей стоимости и которые зарегистрированы на счете основных средств, создаются на странице **Ваучер журнала**.
5. В области действий выберите **Проверить** \> **Проверить** для проверки проводок.
6. В области действий выберите **Разнести** \> **Разнести**, чтобы разнести проводки.

Можно просмотреть разнесенные проводки в строках журнала, выбрав **Запросы \> Проводки** в области действий. Со страницы **Модели стоимости** можно просмотреть разнесенные проводки для каждой модели стоимости (выберите **Основные средства (Россия) \> Основные средства**, выберите **Модели стоимости** в области действий, а затем на странице **Модели учета основных средств** в области действий выберите **Проводки**).

## <a name="create-standard-printing-forms"></a>Создание стандартных форм печати
Выполните следующие действия, чтобы создать стандартные формы печати.

1. В записи основного средства в области действий выберите **Документы**, а затем выберите одну из следующих форм печати:

    - Инв.карточка (№ ОС-6)
    - Акт приемки оборудования (№ ОС-14)

2. На странице **Проводки по ОС** (выберите **Основные средства (Россия) > Основные средства**, выберите **Модели стоимости** в области действий, а затем на странице **Модели учета основных средств** в области действий выберите **Проводки**) выберите проводку типа **Ввод в эксплуатацию**. В области действий выберите **Документы**, а затем выберите одну из следующих пар форм печати:

    - **Акт приемки (№ ОС-1)** и **Акт о приеме-передаче (№ ОС-1)** — приемка всех основных средств, кроме зданий и правопритязаний.
    - **Акт приемки (№ ОС-1а)** и **Акт о приеме-передаче (№ ОС-1а)** — приемка зданий и сооружений.

3. Выберите **Создать**, создать запись документа.
4. Выберите **Печать** для создания документа в формате Microsoft Excel.

Все созданные записи документов можно просмотреть, выбрав **Основные средства (Россия)** > **Запросы > Документы**.

## <a name="fixed-asset-assembly"></a>Сборка основных средств
Можно собрать основное средство из складируемых номенклатуры и ввести это основное средство в эксплуатацию. Когда собранное основное средство вводится в эксплуатацию, комплектующие выдаются со склада по своей текущей себестоимости.

1. Выберите **Основные средства (Россия) > Основные средства** и в области действий выберите **Комплектующие**, чтобы открыть страницу **Комплектующие** (если основное средство не находится в статусе **Куплено** или **Запланировано**, ввести комплектующие нельзя).
2. В верхней области на вкладке **Комплектующие** выберите **Добавить**.
3. В поле **Номенклатура** выберите номенклатуру, из которой собирается основное средство.
4. В поле **Количество** введите количество номенклатур, которое должно использоваться для сборки основного средства.

    Поле **Дата проводки** заполняется автоматически и показывает дату ввода основного средства в эксплуатацию (разноски проводки).

5. На вкладке **Складская аналитика** укажите аналитики склада, конфигурацию (если нужно) и прочие параметры, используемые для списания номенклатуры из запасов.

    Если основное средство собирается их компонентов, купленных в рамках одной закупки, на вкладке **Комплектующие** выберите **Из закупки**. Затем в диалоговом окне **Заявка** выберите закупку, которая включает в себя номенклатуры для сборки, а затем в области действий выберите **Добавить**. Все включенные в закупку номенклатуры добавляются на страницу **Комплектующие**.
7. Введите основное средство в эксплуатацию с помощью журнала основных средств (**Основные средства (Россия)** > **Журналы** > **Журнал ОС**).

    > [!NOTE]
    > По умолчанию стоимость приобретения основного средства вычисляется как себестоимость всех компонентов. Тем не менее эту стоимость можно изменить.

## <a name="reverse-acquisition-transactions"></a>Реверсирование проводок приобретения
По умолчанию при реверсировании проводок, дата реверсирования равна дате исходной проводки. Тем не менее можно указать другую дату реверсирования. 

1. Выберите **Основные средства (Россия)  Основные средства**. В области действий выберите **Модели стоимости**. Затем на вкладке **Модели учета основных средств** в области действий выберите **Проводки**, чтобы открыть страницу **Проводки по ОС**. 
2. Выберите проводку, а затем в области действий выберите **Сторнировать проводку**. В диалоговом окне **Сторнировать проводку** можно изменить дату реверсирования проводки.
3. Нажмите **ОК**. На странице **Проводки по ОС** создается проводка для реверсирования исходной проводки.
4. Выберите **Ваучер**, чтобы открыть страницу **Проводки ваучера**, где можно просмотреть проводки в книге учета.

При разноске реверсирующих проводок для всех моделей стоимости статус основного средства меняется на **Запланировано** или **Куплено**, в зависимости от исходного статуса основного средства.

При реверсировании проводки типа **Ввод в эксплуатацию** для основного средства, которое включает в себя комплектующие, номенклатуры возвращаются в запасы по той же себестоимости, по которой они были отпущены со склада при разноске проводки **Ввод в эксплуатацию**. Складские проводки помечаются и после закрытия запасов имеют ту же себестоимость.

> [!NOTE]
> Проводки типа **Ввод в эксплуатацию** можно реверсировать через журнал основных средств (**Основные средства (Россия) \> Журналы \> Журнал ОС**).
