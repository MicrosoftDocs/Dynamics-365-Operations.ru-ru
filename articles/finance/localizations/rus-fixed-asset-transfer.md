# <a name="transfer-fixed-assets-russia"></a>Перенос основных средств (Россия)

[!include [banner](../includes/banner.md)]

Можно переместить основные средства от одного местоположения или ответственного в другое местоположение или другому ответственному. Можно также перемещать основные средства между компаниями.

## <a name="transfer-multiple-assets-by-using-the-fixed-asset-transfer-journal"></a>Перевод нескольких основных средств с использованием журнала переноса ОС

Выполните следующие шаги для переноса основных средств или групп основных средств из одного склада или от одного ответственного лица в другое место или другому ответственному лицу.

1. Выберите **Основные средства (Россия) \> Журналы \> Журналы перемещений \> Перемещение ОС**.
2. Выберите **Создать** и введите требуемые сведения.
3. Выберите **Строки**, чтобы открыть диалоговое окно **Создание строк журнала перемещения**.
4. Выберите **Фильтр**, чтобы открыть диалоговое окно **Активы**, в котором можно выбрать требуемые критерии.
5. Выберите **OK**, чтобы перенести сведения в диалоговом окне **Создание строк журнала перемещения**.
6. Выберите **OK**, чтобы открыть страницу **Строки журнала перемещения ОС**.

    > [!NOTE]
    > В журнале записи создаются для всех активов или складских запасов, у которых имеются значения в полях **Расположение** и **Предыдущее ответственное лицо**. В заголовке страницы приведена информация, заданная в журнальном пакете. Можно добавить или удалить строки на странице **Строки журнала перемещения ОС**, как требуется.

7. Выберите **Закрыть**. Сведения о перемещении активов сохраняются в истории.
8. Выберите **Печать**, чтобы задать параметры печати и распечатать накладную на перемещение активов.

## <a name="transfer-assets-between-companies"></a>Перевод средств между компаниями

Для перемещения активов в другую компанию можно создавать отдельные проводки перемещения. Если компании находятся в отдельных базах данных, проводки перемещения создаются в компании, из которой перемещают актив, а проводки поступлений создаются в компании, в которую перемещают актив.

### <a name="adjust-the-value-model-settings-to-transfer-an-asset"></a>Корректировка параметров модели стоимости для перемещения актива

Прежде чем создать перемещения основных средств между компаниями, необходимо скорректировать параметры моделей стоимости.

1. Выберите **Основные средства (Россия) \> Настройка \> Модели учета** и создайте модель стоимости.
2. На экспресс-вкладке **Карта** в поле **Код компании** выберите код для компании, в которую перемещается актив.
3. В поле **Модель стоимости** выберите номер модели стоимости, который соответствует номеру модели стоимости для счета компании, выбранной вами на предыдущем шаге.

### <a name="transfer-an-asset-to-another-company"></a>Перемещение актива в другую компанию

1. Выберите **Основные средства (Россия) \> Общее \> Основные средства** и выберите основное средство для перемещения.
2. В области действий на вкладке **Основные средства** в группе **История** выберите **Передача**.
3. В поле **Дата** введите дату перемещения.
4. Необязательно: в поле **Код компании** введите код компании, которая получает актив.
5. Необязательно: в поле **Основные средства** измените код для ОС.
6. После выбора компании, в которую передается актив, можно выбрать инвентарный номер основного средства, которое актив будет иметь в получающей компании после завершения перемещения. Статус этого основного средства должен быть **Запланировано** или **Списано**.

    После разноски транзакции передачи в журнале основных средств (**Основные средства (Россия) \> Журналы \> Журнал ОС**) следующая информация на странице **Передача в другую компанию** обновляется автоматически:

    - На вкладке **Обзор** флажок **Разнесено** выбирается, если операции передачи была разнесена для всех моделей стоимости.
    - На вкладке **Модели учета** балансовая стоимость, бухгалтерская амортизация и срок службы обновляются для моделей учета, для которых была разнесена операция передачи.

Запись может также быть автоматически создана на странице **Передача в другую компанию** после закрытия журнала **Передача в другую компанию** (**Основные средства (Россия) \> Журналы \> Журналы перемещений**). Информацию о том, как создать запись в журнале **Передача в другую компанию**, см. в разделе [Передача группы основных средств](#transfer-a-group-of-fixed-assets).

> [!NOTE]
> После разноски журнала перемещения статус основного средства будет **Списано**. Кроме того, на в получающей компании на странице **Прием из другой компании** поля на вкладке **Модели учета** обновляются для основного средства, которое было указано на странице **Передача в другую компанию**.

### <a name="transfer-a-group-of-fixed-assets"></a>Передача группы основных средств

1. Выберите **Основные средства (Россия) \> Журналы \> Журналы перемещений \> Передача в другую компанию**.
2. Выберите **Создать**, чтобы создать строку, а затем в поле **Дата** введите дату передачу основного средства.
3. В поле **Идентификатор счетов компании** введите идентификатор счета компании, принимающей активы, если компания находится в той же базе данных.

    > [!NOTE]
    > Поле **Номер журнала** автоматически заполняется на основе настроенной номерной серии. Однако можно вручную ввести номер журнала.

4. Выберите **Строки**.
5. На странице **Строки журнала перемещения ОС** создайте строку.
6. В поле **Источник** выберите инвентарный номер для передаваемого основного средства.

    > [!NOTE]
    > Доступные активы имеют статус **В эксплуатации**. Поле **Имя счета** показывает имя перемещаемого актива. Если указана получающая компания для журнала, в поле **Компания назначения** отображается ее название.

7. В поле **Место назначения** выберите в получающей компании номер актива, который соответствует перемещаемому активу.

    > [!NOTE]
    > Доступные активы имеют статус **Запланированный** или **Списанный**. В поле **Имя счета** отображается счет, на который перемещается актив, на основе значения, выбранного в поле **Место назначения**.

8. Повторите шаги с 5 по 7, чтобы создать все дополнительные строки, которые необходимы, затем закройте страницу.
9. На странице **Передача в другую компанию** выберите **Закрыть**. Флажок **Разнесено** установлен и отображаются сведения о перемещении в истории основных средств или складских активов, которые указаны в строках журнала (страница **Передача в другую компанию**).
10. Выберите **Основные средства (Россия) \> Журналы \> Журнал ОС**.
11. Выберите **Строки** и введите проводки передачи в строки журнала. Этот процесс похож на процесс для создания проводки приобретения.

    > [!NOTE]
    > Дата проводки на странице **Основное средство** не может предшествовать дате регистрации передачи в другую компанию в истории активов. Для создания строк необходимо использовать групповую проводку. Строки автоматически создаются для всех основных средств, которые имеют сведения о передаче в другую компанию в истории.

12. Выберите **Проверить \> Проверить**, чтобы проверить журнал.
13. Выберите **Разнести \> Разнести** для создания ваучера и проводок основных средств.
14. Обзор всех ваучеров и проводок с основными средствами можно просмотреть с помощью запросов. Выберите **Основные средства (Россия) \> Запросы \> Проводки** или **Основные средства (Россия) \> Запросы \> Проводки по ваучерам**.

### <a name="receive-a-group-of-fixed-assets-from-another-company"></a>Получение группы ОС из другой компании

Можно зарегистрировать одновременно получение нескольких основных средств или складских активов.

1. Выберите **Основные средства (Россия) \> Журналы \> Журналы перемещений \> Прием из другой компании** и создайте журнал.
2. В поле **Дата** введите дату перемещения активов из другой компании.
3. В поле **Идентификатор счетов компании** введите идентификатор счета для компании, из которой принимаются активы, если эта компания находится в той же базе данных.

    > [!NOTE]
    > Поле **Номер журнала** автоматически заполняется на основе настроенной номерной серии. Однако можно вручную ввести номер журнала.

4. Выберите **Строки**.
5. На странице **Строки журнала перемещения ОС** создайте строку.
6. В поле **Источник** выберите номер склада для актива, принимаемого из другой компании.

    > [!NOTE]
    > Если выбран номер актива в поле **Источник** и настроено полное соответствие моделей стоимости настроен между передающей и принимающей компаниями, отображаются сведения в полях на вкладке **Модели учета**. В противном случае необходимо вручную ввести или изменить эти сведения.

7. В поле **Место назначения** выберите в получающей компании номер актива, который соответствует перемещаемому активу.
8. Повторите шаги с 5 по 7, чтобы создать все дополнительные строки, которые необходимы.
9. Выберите **Создание основного средства**.

    > [!NOTE]
    > Данная кнопка доступна только в том случае, если в поле **Источник** отображается номер актива, но место назначения не выбрано. Такая ситуация возможна, если получатель не указан в проводке перемещения для перемещения актива в другую компанию.

10. В диалоговом окне **Создание основного средства** выберите группу основных средств, к которой принадлежит актив. Введите инвентарный номер, если это поле не заполняется автоматически.
11. Нажмите **ОК**. Создается запись об активе и отображается место назначения.

#### <a name="automatically-create-lines"></a>Автоматическое создание строк

Чтобы автоматически создавать строки на странице **Прием из другой компании**, выберите **Создать из журнала передачи ОС**. затем выполните следующие действия.

1. На странице **Добавление основных средств из журналов отпуска для другой компании** в верхней области выберите журнал. Затем выберите **Копировать журнал**, чтобы скопировать все сведения из указанного журнала перемещения в журнал поступления.
2. В нижней области выберите **Копировать строку журнала**, чтобы скопировать отдельные строки журнала перемещения в журнал поступления.
3. На странице **Прием из другой компании** выберите **Закрыть**. Сведения из поступления обновляются в записях об активах.
4. Выберите **Основные средства (Россия) \> Журналы \> Журнал ОС**.
5. Выберите **Строки** и введите проводку поступления в строки журнала. Этот процесс похож на процесс для создания проводки приобретения.

    > [!NOTE]
    > Дата проводки на странице **Основное средство** не может предшествовать дате регистрации поступления в истории активов. Для создания строк необходимо использовать групповую проводку. Строки автоматически создаются для всех основных средств, которые имеют сведения о приеме из другой компании в истории.

6. Выберите **Проверить \> Проверить**, чтобы проверить журнал.
7. Выберите **Разнести \> Разнести** для создания ваучера и проводок основных средств.
8. Обзор всех ваучеров и проводок с основными средствами можно просмотреть с помощью запросов. Выберите **Основные средства (Россия) \> Запросы \> Проводки** или **Основные средства (Россия) \> Запросы \> Проводки по ваучерам**.

    > [!NOTE]
    > Сведения о перемещении активов между компаниями можно просмотреть на странице **Прием из другой компании**.
