# <a name="foreign-currency-sales-purchases-and-transfers"></a>Продажи, покупки и переводы в иностранной валюте

[!include [banner](../includes/banner.md)]

Dynamics 365 Finance позволяет выполнять следующие задачи:

- Регистрировать проводки для продажи, покупки или перевода иностранной валюты, которые выполняются с помощью счета учета **Переводы в пути**.
- Автоматически рассчитывать суммы прибыли и убытков, вызванных конвертацией иностранной валюты, для продаж и покупок иностранной валюты. (В расчете учитывается валютный курс иностранной валюты центрального банка и валютный курс банка юридического лица на дату перевода иностранной валюты.) Также автоматически создаются проводки для прибылей/убытков от конвертации иностранной валюты.
- Автоматическое создание проводок для нереализованной курсовой разницы, возникающей, когда валютные курсы в иностранной валюте на даты проводок отличаются.
- Печать заказа на перемещение иностранной валюты.


## <a name="set-up-the-sale-purchase-and-transfer-of-foreign-currency"></a>Настройка продажи, покупки и перевода иностранной валюты

Ниже приведен обзор задач, которые необходимо выполнить для настройки системы для расчета курсовой разницы и прибыли/убытков на счете **Промежуточный счет предъявления к оплате**:

- [Настройка валютного курса](#set-up-an-exchange-rate)
- [Настройка банка для использования проводок конвертации иностранной валюты](#set-up-a-bank-to-use-for-foreign-currency-conversion-transactions)
- [Настройка типа банковских проводок](#set-up-a-bank-transaction-type)
- [Настройка способа оплаты](#set-up-a-method-of-payment)
- [Настройка номерной серии](#set-up-a-number-sequence)
- [Настройка банковского счета](#set-up-a-bank-account)

### <a name="set-up-an-exchange-rate"></a>Настройка валютного курса

Страница **Счета валютной переоценки** используется для расчета сумм прибыли или убытка для конвертации иностранной валюты.

1. Откройте **Главная книга \> Валюты \> Параметры валюты**.
2. В поле **Юридические лица** выберите компанию.
3. На экспресс-вкладке **Покупки/Поставщики** определяет следующие счета учета для конвертации валюты: **Конвертация - прибыль** и **Конвертация - убытки**.
4. Определите следующие коды выручки и дохода для конвертации иностранной валюты: **Код дохода (конвертация валюты)** и **Код расхода (конвертация валюты)**.

### <a name="set-up-a-bank-to-use-for-foreign-currency-conversion-transactions"></a>Настройка банка для использования проводок конвертации иностранной валюты

Используйте страницу **Банки**, чтобы связать счет поставщика с банком, который должен использоваться для проводок продажи или покупки в иностранной валюте.

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Настройка \> Банковские группы**.
2. Создайте банк или выберите существующий банк.
3. На экспресс-вкладке **Общее** в поле **Счет поставщика** выберите счет поставщика.

При связывании счета поставщика с банком становится доступна кнопка **Проводки** на панели действий. Параметры **Сопоставление открытых проводок** и **Редактирование сопоставленных проводок** на кнопке меню **Функции** также станут доступны.

### <a name="set-up-a-bank-transaction-type"></a>Настройка типа банковских проводок

Страница **Типы банковских проводок** служит для настройки типа банковской проводки для проводок продажи, покупки и перевода иностранной валюты.

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Настройка \> Типы банковских проводок**.
2. Создайте тип банковской проводки.
3. В поле **Расширенный тип проводки** выберите один из следующих типов перевода иностранной валюты: **Продажа валюты**, **Покупка валюты** или **Перевод валюты**.
4. В поле **Профиль разноски по поставщику** выберите профиль разноски по поставщику, который должен использоваться для разноски на счет учета **Переводы в пути**.

### <a name="set-up-a-method-of-payment"></a>Настройка способа оплаты

Используйте страницу **Способы оплаты**, чтобы настроить способ оплаты для продажи, покупки или перевода иностранной валюты.

1. Перейдите в раздел **Расчеты с поставщиками \> Настройка платежей \> Способы оплаты**.
2. Создайте способ оплаты.
3. На экспресс-вкладке **Общее** в поле **Тип счета** выберите **Банк**.
4. В поле **Счет оплаты** выберите банковский счет, с которого переводится иностранная валюта для данного метода оплаты.
5. В поле **Тип банковской проводки** выберите тип банковской проводки для продажи, покупки или перевода иностранной валюты.
6. На экспресс-вкладке **Форматы файлов** в поле **Формат экспорта** выберите **Поручение на конвертацию/перевод валюты**.

    > [!NOTE]
    > Если параметр **Поручение на конвертацию/перевод валюты** недоступен, выберите **Настройка**, чтобы открыть страницу **Форматы файлов для способов оплаты**. На вкладке **Экспорт** в списке **Доступно** выберите **Поручение на конвертацию/перевод валюты**, затем выберите кнопку со стрелкой вправо, чтобы добавить формат в список **Выбранные**.

### <a name="set-up-a-bank-account"></a>Настройка банковского счета

Используйте страницу **Банковские счета** для настройки конкретных сведений для продажи или покупки иностранной валюты.

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Банковские счета \> Банковские счета**.
2. Создайте банковский счет или выберите существующий.
3. На экспресс-вкладке **Управление платежами** в полях **Шаблон поручения (продажа валюты)** и **Шаблон поручения (покупка валюты)** выберите шаблон Microsoft Word.

Дополнительные сведения см. в разделе [Настройка банковских счетов (Россия)](./rus-local-settings-requisites-bank-module.md).

### <a name="set-up-a-number-sequence"></a>Настройка номерной серии

Используйте страницу **Параметры управления банком и кассовыми операциями**, чтобы назначить номерную серию для проводок в иностранной валюте.

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Настройка \> Параметры управления банком и кассовыми операциями**.
2. На вкладке **Номерные серии** в поле **Код номерной серии** выберите номерную серию для ссылки **Поручение на конвертацию/перевод валюты**.

## <a name="foreign-currency-sales"></a>Продажи иностранной валюты

### <a name="create-a-foreign-currency-sale-transaction"></a>Создание проводки продажи иностранной валюты

1. Перейдите в раздел **Расчеты с поставщиками \> Платежи \> Журнал платежей**.
2. Создайте журнал платежей.
3. Выберите **Строки** и создайте строку журнала.
4. На вкладке **Список** в поле **Счет** выберите счет поставщика, связанный с банковским счетом, который используется для проводок конвертации иностранной валюты.
5. В поле **Дебет** введите сумму проводки в иностранной валюте, которую необходимо продать.
6. В поле **Валюта** введите код иностранной валюты, на продажу которой дается поручение.
7. В поле **Тип корр. счета** выберите **Банк**.
8. В поле **Корр. счет** выберите банковский счет.
9. В поле **Способ оплаты** выберите способ оплаты, созданный ранее для продажи иностранной валюты.
10. На вкладке **Банк** в поле **Тип банковской проводки** проверьте значение, которое заполняется автоматически на основе метода оплаты, или выберите тип проводки для проводки продажи иностранной валюты.
11. На вкладке **Платеж** в поле **Профиль разноски** убедитесь, что значение равно профилю, который используется для разноски на счет учета **Переводы в пути**.
12. Выберите **Создать платежи**.
13. В диалоговом окне **Создание платежей** выберите параметр **Формат экспорта**, затем в поле **Формат экспорта** выберите **Поручение на конвертацию/перевод валюты**. Можно также выбрать параметр **Способ оплаты**, затем в поле **Способ оплаты** выбрать метод оплаты для продажи иностранной валюты.
14. В поле **Банковский счет** выберите банковский счет для платежа.
15. Нажмите **ОК**.
16. В диалоговом окне **Параметры платежа** введите требуемую информацию.

    | Поле              | Описание |
    |--------------------|-------------|
    | Рабочий             | Выберите работника, ответственного за продажу иностранной валюты. |
    | Транзитный счет    | Выберите банковский счет, на который должны поступить рубли за проданную иностранную валюту (результат конвертирования иностранной валюты). Шаблон, который будет использоваться для печати поручения на продажу иностранной валюты, берется из данного банковского счета. |
    | Валютное           | Выберите код валюты, которая должна быть получена на транзитный банковский счет после продажи иностранной валюты. |
    | Обменный курс банка | Определите валютный курс иностранной валюты для конвертации иностранной валюты. Эта информация печатается в поручении на продажу иностранной валюты. |
    | Счет поставщика   | Выберите сведения о контракте в иностранной валюте, который является основанием для проводки продажи иностранной валюты. |
    | Договор покупки | Выберите сведения о контракте в иностранной валюте, который является основанием для проводки продажи иностранной валюты. |
    | Печатать документ     | Задайте для этого параметра значение "Да", чтобы печатать поручение на продажу иностранной валюты при создании платежей. |



17. Нажмите **ОК**.

    В строке журнала платежей значение поля **Статус платежа** обновляется до **Отправлено**.

    На вкладке **Платеж** обновляются поля **Транзитный счет**, **Иностранный контрагент**, **Группа договоров** и **Контракт по сделке**.

18. Выберите **Печать \> Поручения на продажу валюты** для печати поручения на продажу иностранной валюты. В отчете содержится следующая информация:
19. Проверьте и разнесите журнал.

### <a name="create-transactions-for-the-bank-commission-and-the-local-currency-rubles-amount-that-is-received-as-a-result-of-the-foreign-currency-sale"></a>Создание проводок по комиссии банка и суммы в основной валюте (в рублях), полученной в результате продажи иностранной валюты

После учета платежного поручения для продажи иностранной валюты можно зарегистрировать комиссию банка и в местную валюту, полученную в результате конвертации иностранной валюты. (В данном случае получаются российские рубли.)

1. Перейдите в раздел **Расчеты с поставщиками \> Платежи \> Журнал платежей**.
2. Создайте журнал платежей.
3. Выберите **Строки**, создайте строку журнала для списания комиссии банка и введите следующие сведения.

    | Вкладка     | Поле                 | Описание |
    |---------|-----------------------|-------------|
    | Список    | Счет               | Выберите счет поставщика, который был использован при проводке по продаже иностранной валюты в предыдущей процедуре. |
    |         | по кредиту                | Введите сумму комиссии банка. |
    |         | Валютное              | Выберите валюту комиссии банка. |
    |         | Тип корр. счета   | Выберите **Главная книга**. |
    |         | Корр. счет        | Выберите счет ГК для убытков от банковской комиссии. |
    | Банк    | Тип банковской проводки | Выберите тип проводки, который имеет расширенный тип **Продажа валюты**. |
    | Платеж | Профиль разноски       | Убедитесь, что значение равно профилю, который используется для разноски на счет ГК **Переводы в пути**. |

4. Создайте строку журнала для полученной местной валюты и введите следующие сведения.

    | Вкладка     | Поле                                              | Описание |
    |---------|----------------------------------------------------|-------------|
    | Список    | Счет                                            | Выберите счет поставщика, который был использован при проводке по продаже иностранной валюты в предыдущей процедуре. |
    |         | по кредиту                                             | Введите сумму валюты учета (в рублях), которая была получена. |
    |         | Валютное                                           | Выберите валюту учета, которая является валютой, полученной в результате продажи иностранной валюты (рубли). |
    |         | Тип корр. счета                                | Выберите **Банк**. |
    |         | Корр. счет                                     | Выберите банковский счет, на который получена валюта учета. |
    | Банк    | Тип банковской проводки                              | Выберите тип проводки, который имеет расширенный тип **Продажа валюты**. |
    | Платеж | Валюта (в пункте **Конвертация валюты**)           | Выберите иностранную валюту, которая была продана. Этот код иностранной валюты должен отличаться от валюты учета. |
    |         | Обменный курс банка (в пункте **Конвертация валюты**) | Введите валютный курс банка, который применялся при продаже иностранной валюты. |
    |         | Профиль разноски                                    | Убедитесь, что значение равно профилю, который используется для разноски на счет ГК **Переводы в пути**. |

5. Разноска журнала.

### <a name="post-the-settlement-of-foreign-currency-sale-transactions"></a>Разноска сопоставления проводок продажи иностранной валюты

Сопоставление проводок по продаже иностранной валюты разносится для создания проводок прибылей/убытков от них и от проводок курсовой разницы иностранной валюты.

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Настройка \> Банковские группы**.
2. Выберите банк, затем выберите **Функции \> Сопоставление открытых проводок** для сопоставления проводок продажи иностранной валюты.
3. В поле **Дата разноски сопоставления** выберите тип даты разноски: **Последняя дата**, **Текущая дата** или **Выбранная дата**.
4. Если вы выбрали **Выбранная дата** в поле **Дата разноски сопоставления**, вручную введите дату разноски.
5. Просмотрите проводки, которые будут сопоставлены:

    - На вкладке **Обзор** поля **Транзитный счет** и **Банковский счет (поступление)** специфичны для продажи иностранной валюты:

        - Для проводок с положительным значением значение поля **Транзитный счет** равно транзитному банковскому счету, который был выбран для проводки по продаже иностранной валюты в поле **Транзитный счет** в диалоговом окне **Параметры платежа**.
        - Для проводок с отрицательным значением поле **Банковский счет (поступление)** равно банковскому счету, на который получена сумма в валюте учета в результате проводки продажи иностранной валюты.

        Значения в этих полях должны быть равны для проводок, помеченных для сопоставления.

    - На вкладке **Общие** поля в разделе **Конвертация валюты** специфичны для продажи иностранной валюты.

6. Пометить проводки для сопоставления.

    Во время сопоставления проверяется следующая информация:

    - Тип проводки обоих проводок должен иметь расширенный тип **Продажа валюты**.
    - Значение поля **Валюта** в разделе **Конвертация валюты** на вкладке **Общие** для положительной проводки (продажа иностранной валюты) должно быть равно значение поля **Валюта** на вкладке **Обзор** для отрицательной проводки (поступление в валюте учета в результате продажи иностранной валюты).
    - Банковский счет, указанный в поле **Транзитный счет** для положительной проводки, должен быть равен банковскому счету, указанному в поле **Банковский счет (поступление)** для отрицательной проводки.

7. Выберите **Разнести**. Следующие проводки создаются:

    - Нереализованная курсовая разница иностранной валюты (если, для иностранной валюты, которая продана, валютный курс иностранной валюты на дату продажи иностранной валюты отличается от курс обмена иностранной валюты на дату получения валюты учета)
    - Прибыль или убыток от продажи валюты (если, для проданной иностранной валюты, валютный курс банка отличается от валютного курса Центрального банка на дату проводки поступления валюты учета)

8. Можно просмотреть проводки, которые будут созданы. На страницы **Банковские группы** выберите **Проводки**.
9. Имеется возможность разноски отмены сопоставления проводок. На странице **Банковские группы** выберите **Функции \> Редактирование сопоставленных проводок**, отметьте проводки, затем выберите **Сторнировать**.

## <a name="foreign-currency-purchases"></a>Покупка иностранной валюты

### <a name="create-a-transaction-of-rubles-movement-for-a-foreign-currency-purchase"></a>Создание проводки перевода в рублях для покупки иностранной валюты

1. Перейдите в раздел **Расчеты с поставщиками \> Платежи \> Журнал платежей**.
2. Создайте журнал платежей.
3. Выберите **Строки**, создайте строку журнала и введите следующие сведения.

    | Вкладка     | Поле                 | Описание |
    |---------|-----------------------|-------------|
    | Список    | Счет               | Выберите счет поставщика, связанный с банковским счетом, который используется для проводок конвертации иностранной валюты. Если ввести сумму в поле **Дебет** и если в поле **Тип банковской проводки** задан тип проводки расширенного типа **Значение**, это поле заполняется автоматически при выборе банковского счета в поле **Корр. счет**. |
    |         | по дебету                 | Введите сумму проводки в валюте учета (рубли). |
    |         | Валютное              | Введите код валюты учета (рубли). |
    |         | Тип корр. счета   | Выберите **Банк**. |
    |         | Корр. счет        | Выберите банковский счет, на который переводятся рубли для покупки иностранной валюты. |
    |         | Способ оплаты     | Выберите способ оплаты, созданный ранее для покупки иностранной валюты. |
    | Банк    | Тип банковской проводки | Проверьте значение, которое заполняется автоматически на основе метода оплаты, или выберите тип проводки для проводки покупки иностранной валюты. |
    | Платеж | Профиль разноски       | Убедитесь, что значение равно профилю, который используется для разноски на счет ГК **Переводы в пути**. |

4. Выберите **Создать платежи**.
5. В диалоговом окне **Создать платежи** выберите параметр **Способ оплаты**, затем в поле **Способ оплаты** выберите метод оплаты для покупки иностранной валюты.
6. В поле **Банковский счет** выберите банковский счет для платежа.
7. Нажмите **ОК**.
8. В диалоговом окне **Параметры платежа** введите требуемую информацию.

    | Поле              | Описание |
    |--------------------|-------------|
    | Рабочий             | Выберите работника, ответственного за покупку иностранной валюты. |
    | Транзитный счет    | Выберите банковский счет, на который должна поступить приобретенная иностранная валюта (результат конвертирования иностранной валюты). |
    | Валютное           | Выберите код иностранной валюты, которая должна быть получена на транзитный банковский счет после покупки иностранной валюты. |
    | Обменный курс банка | Определите валютный курс иностранной валюты для конвертации иностранной валюты. Эта информация печатается в поручении на покупку иностранной валюты. |
    | Счет поставщика   | Выберите сведения о контракте в иностранной валюте, который является основанием для проводки покупки иностранной валюты. |
    | Договор покупки | Выберите сведения о контракте в иностранной валюте, который является основанием для проводки покупки иностранной валюты. |
    | Печатать документ     | Задайте для этого параметра значение "Да", чтобы печатать поручение на покупку иностранной валюты при создании платежей. |

9. Нажмите **ОК**.

    В строке журнала платежей поле **Статус платежа** обновляется до **Отправлено**.

    На вкладке **Платеж** обновляются поля **Транзитный счет**, **Иностранный контрагент**, **Группа договоров** и **Контракт по сделке**.

    Во время создания поручения на покупку иностранной валюты проверяется следующая информация:

    - Код валюты транзитного банковского счета должен отличаться от валюты учета.
    - Код валюты транзитного банковского счета должен быть равен код валюты, которая выбрана в диалоговом окне **Параметры платежа**.
    - Код валюты из строки журнала платежа должен быть в валюте учета.

10. Выберите **Печать \> Поручения на покупку валюты** для печати поручения на покупку иностранной валюты. (Шаблон берется из транзитного банковского счета, который выбран в диалоговом окне **Параметры платежа**.)
11. Проверьте и разнесите журнал.

### <a name="create-transactions-for-the-bank-commission-and-the-foreign-currency-amount-that-is-received-as-a-result-of-the-foreign-currency-purchase"></a>Создание проводок по комиссии банка и суммы в иностранной валюте, полученной в результате покупки иностранной валюты

После учета платежного поручения для покупки иностранной валюты можно зарегистрировать комиссию банка и сумму в иностранной валюте, полученную в результате конвертации иностранной валюты.

1. Перейдите в раздел **Расчеты с поставщиками \> Платежи \> Журнал платежей**.
2. Создайте журнал платежей.
3. Выберите **Строки**, создайте строку журнала для списания комиссии банка и введите следующие сведения.

    | Вкладка     | Поле                 | Описание |
    |---------|-----------------------|-------------|
    | Список    | Счет               | Выберите счет поставщика, который был использован при проводке по покупке иностранной валюты в предыдущей процедуре. |
    |         | по кредиту                | Введите сумму комиссии банка. |
    |         | Валютное              | Выберите код валюты комиссии банка. |
    |         | Тип корр. счета   | Выберите **Главная книга**. |
    |         | Корр. счет        | Выберите счет ГК для убытков от банковской комиссии. |
    | Банк    | Тип банковской проводки | Выберите тип проводки, который имеет расширенный тип **Покупка валюты**. |
    | Платеж | Профиль разноски       | Убедитесь, что значение равно профилю, который используется для разноски на счет ГК **Переводы в пути**. |

4. Создайте строку журнала для полученной иностранной валюты и введите следующие сведения.

    | Вкладка     | Поле                                              | Описание |
    |---------|----------------------------------------------------|-------------|
    | Список    | Счет                                            | Выберите счет поставщика, который был использован при проводке по покупке иностранной валюты в предыдущей процедуре. |
    |         | по кредиту                                             | Введите сумму в иностранной валюте, которая была получена. |
    |         | Валютное                                           | Выберите код купленной иностранной валюты. |
    |         | Тип корр. счета                                | Выберите **Банк**. |
    |         | Корр. счет                                     | Выберите банковский счет, на который получена иностранная валюта. Банк этого банковского счета должен иметь связь со счетом поставщика, который выбран в поле **Счет**. |
    | Банк    | Тип банковской проводки                              | Выберите тип проводки, который имеет расширенный тип **Покупка валюты**. |
    | Платеж | Валюта (в пункте **Конвертация валюты**)           | Выберите валюту учета, используемую для покупки иностранной валюты. |
    |         | Обменный курс банка (в пункте **Конвертация валюты**) | Введите валютный курс банка, который применялся при покупке иностранной валюты. |
    |         | Профиль разноски                                    | Убедитесь, что значение равно профилю, который используется для разноски на счет ГК **Переводы в пути**. |

5. Разноска журнала.

    Во время разности проверяется следующая информация:

    - Код валюты проводки должен отличаться от валюты учета компании.
    - Код валюты, который выбран в поле **Валюта** на вкладке **Платеж**, должен быть равен валюте учета компании.
    - Значения должны быть введены в поля **Валюта** и **Обменный курс банка** на вкладке **Платеж**.

### <a name="post-the-settlement-of-foreign-currency-purchase-transactions"></a>Разноска сопоставления проводок покупки иностранной валюты

Разноска сопоставления проводки покупки иностранной валюты выполняется для создания проводок прибылей и убытков от них.

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Настройка \> Банковские группы**.
2. Выберите банк, затем выберите **Функции \> Сопоставление открытых проводок** для сопоставления проводок покупки иностранной валюты.
3. Пометить проводки для сопоставления.

    Во время сопоставления проверяется следующая информация:

    - Тип проводки обоих проводок должен иметь расширенный тип **Покупка валюты**.
    - Значение поля **Валюта** в разделе **Конвертация валюты** на вкладке **Общие** для проводки покупки иностранной валюты должно быть равно значение поля **Валюта** на вкладке **Обзор** для проводки поступления иностранной валюты.
    - Банковский счет, указанный в поле **Транзитный счет** для проводки покупки иностранной валюты, должен быть равен банковскому счету, на который получена иностранная валюта.

4. Выберите **Разнести**. Проводка для прибыли или убытка от покупки иностранной валюты создается (если, для купленной иностранной валюты, курс обмена банка отличается от валютного курса Центрального банка на дату поступления иностранной валюты).
5. Можно просмотреть проводки, которые будут созданы. На страницы **Банковские группы** выберите **Проводки**.
6. Имеется возможность разноски отмены сопоставления проводок. На странице **Банковские группы** выберите **Функции \> Редактирование сопоставленных проводок**, отметьте проводки, затем выберите **Сторнировать**.

### <a name="post-the-settlement-of-foreign-currency-transfer-transactions"></a>Разноска сопоставления проводок перевода иностранной валюты

Разноска сопоставления проводки покупки иностранной валюты выполняется для создания проводок курсовой разницы для иностранной валюты.

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Настройка \> Банковские группы**.
2. Выберите банк, затем выберите **Функции \> Сопоставление открытых проводок** для сопоставления проводок покупки иностранной валюты.
3. Пометить проводки для сопоставления.

    Во время сопоставления проверяется следующая информация:

    - Тип проводки обоих проводок должен иметь расширенный тип **Перевод валюты**.
    - Коды валют по проводкам должны быть равны.

4. Выберите **Разнести**. Создается проводка разницы валютных курсов. Счета ГК для курсовой разницы иностранной валюты берутся из счетов **Реализованная прибыль** и **Реализованный убыток**, которые указаны в настройке параметров иностранной валюты.
5. Можно просмотреть проводки, которые будут созданы. На страницы **Банковские группы** выберите **Проводки**.
6. Имеется возможность разноски отмены сопоставления проводок. На странице **Банковские группы** выберите **Функции \> Редактирование сопоставленных проводок**, отметьте проводки, затем выберите **Сторнировать**.
