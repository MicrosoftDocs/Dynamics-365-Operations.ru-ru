# <a name="date-of-vendor-vat-register"></a>Дата регистрации НДС поставщика

В Microsoft Dynamics 365 Finance версии 10.0.24 доступно новое поле **Дата регистрации НДС поставщика** для накладных поставщика. В этом поле указывается дата налогооблагаемой поставки для покупки.

Чтобы включить новое поле, перейдите в рабочую область **Управление функциями**, найдите и выберите функцию **Включить дату регистрации НДС поставщика в накладных поставщика**, затем выберите **Включить**.

Входящие накладные от поставщиков могут иметь две даты: дату, когда поставщик выпустил накладную, и дату налогооблагаемой поставки (т. е. дата, когда поставщику начисляется налог на добавленную стоимость [НДС]). В некоторых сценариях эти две даты могут различаться.

Можно вычесть сумму входящего НДС для накладной по закупке и включить эту накладную в отчеты по НДС позже, на дату, отличную от обеих ранее упомянутых дат. Эта дата контролируется правилами локального законодательства по отложенному вычету входящего НДС для некоторых сценариев. Она зависит от страны или региона.

Некоторые отчеты по НДС, например [Отчет о инструкциях по контролю НДС](emea-cze-vat-declaration-tax-declaration-model.md#vat-control-statement) в Чешской Республике, требуют, чтобы была указана дата налогооблагаемой поставки для документа покупки. Эта дата должна быть включена в отчет, чтобы налоговые органы могли сверить отчеты по НДС между сторонами.

В Finance даты можно определить в следующих полях:

- **Дата накладной** — дата, когда накладная была выпущена поставщиком.
- **Дата регистрации НДС** — дата вычета суммы НДС для накладной по закупке.
- **Дата регистрации НДС поставщика** — дата налогооблагаемой поставки поставщика.
- **Дата получения документа** — дата получения накладной.

Эти поля включены на следующих страницах:

- Накладная поставщика
- Регистр накладных поставщика
- Утверждение накладной поставщика
- Журнал накладных поставщиков

После разноски накладной поставщика можно проверить даты на страницах **Журнал накладных** и **Проводки по поставщику**.
