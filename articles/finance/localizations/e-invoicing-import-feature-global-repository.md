# <a name="import-features-from-the-global-repository"></a>Импорт функций из глобального репозитория

[!include [banner](../includes/banner.md)]

Глобальный репозиторий содержит функции электронного выставления накладных, которые используются совместно с поставщиком конфигурации. Все функции выставления электронных накладных, предоставляемые корпорацией Майкрософт, являются общими для всех компаний. У вас автоматически есть доступ ко всем функциям электронного выставления накладных, которые корпорация Майкрософт выпускает и публикуют в глобальном репозитории.

Чтобы начать работать с функциями электронного выставления накладных, которые совместно используются с поставщиком конфигурации, импортируйте их в ваш экземпляр службы Regulatory Configuration Service (RCS) из глобального репозитория. Затем просмотрите сведения о функциях, такие как конфигурации электронной отчетности (ER) и конвейеры обработки.

## <a name="import-a-feature-from-the-global-repository"></a>Импорт функции из глобального репозитория

1. Войдите в учетную запись RCS.
2. В рабочей области **Функция глобализации** в разделе **Функции** выберите плитку **Электронное выставление накладных**.
3. Выберите **Импорт**, чтобы открыть поле подстановки **Импорт функции из глобального репозитория**.
4. Чтобы просмотреть функции электронного выставления накладных в глобальном репозитории, доступные для конкретного поставщика конфигурации, выполните синхронизацию экземпляра RCS организации, выбрав **Синхронизировать**. После завершения синхронизации будет обновлен список доступных функций электронного выставления накладных из глобального репозитория. Это обновление может занять несколько минут.
5. Выберите функцию для импорта, затем выберите **Импорт**.

Для просмотра импортированной функции электронного выставления накладных убедитесь, что выбран правильный поставщик конфигурации. По умолчанию функции, созданные поставщиком активной конфигурации, автоматически отфильтровываются. Фильтр можно отрегулировать для просмотра функций, которые были созданы другими поставщиками, например поставщиком конфигурации Майкрософт.

Теперь можно работать с импортированной функцией. Можно просмотреть ее сведения и создать новую функцию, используя импортированную функцию в качестве шаблона.

> [!NOTE]
> Изменить функцию можно только в том случае, если она была создана поставщиком конфигурации, который активен в данный момент. Можно создать новую функцию, используя исходную функцию в качестве базы. Затем можно выполнить требуемые изменения или настроить параметры.

Если корпорация Майкрософт или другая компания, которая использует функции глобализации совместно с вашей компанией, обновляет импортированные вами функции, можно проверить наличие обновленных версий, выбрав пункт **Проверить наличие обновлений** в разделе **Связанные параметры** в рабочей области **Функции глобализации**.

Если имеются обновления для функции, используемой в качестве шаблона, можно импортировать новую версию после синхронизации списка опубликованных функций в глобальном репозитории.
