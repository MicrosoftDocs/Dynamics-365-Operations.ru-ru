# <a name="sync-the-tax-setup-from-the-tax-calculation-service-to-dynamics-365-finance"></a>Перенос параметров налогов из службы расчета налогов в Dynamics 365 Finance

[!include [banner](../includes/banner.md)]

В этой статье объясняется, как передать справочник настройки налогов из службы расчета налогов в Microsoft Dynamics 365 Finance в целях синхронизации.

После выполнения необходимых шагов настройки в разделе [Начала работы с расчетом налогов](global-get-started-with-tax-calculation-service.md) следующие данные настройки налога автоматически синхронизируются из службы расчета налогов в Finance.

## <a name="sales-tax-code"></a>Код налога

| Служба расчета налогов           | Finance                             |
| --------------------------------- | ----------------------------------- |
| Налоговый код                          | Код налога                      |
| Описание                       | Имя                                |
| Пользовательский ввод                        | Период сопоставления                   |
| Пользовательский ввод                        | Группа разноски ГК                |
| Пользовательский ввод                        | Валюта налога                  |
| Настроена отрицательная ставка налога | Возможна отрицательная налоговая ставка |

## <a name="tax-value"></a>Налоговая стоимость

| Служба расчета налогов | Finance                   |
| ----------------------- | ------------------------- |
| С даты проводки   | С даты                 |
| До даты проводки     | По дату                   |
| Минимальная сумма          | Нижний предел             |
| Максимальная сумма          | Максимальный лимит             |
| Ставка налога                | Значение                     |
| Не подлежащая вычету ставка     | Не подлежащий вычету процент |

## <a name="tax-limits"></a>Лимиты налогов

| Служба расчета налогов | Finance           |
| ----------------------- | ----------------- |
| С даты проводки   | С даты         |
| До даты проводки     | По дату           |
| Минимальная сумма налога      | Минимум налога |
| Максимальная сумма налога      | Максимум налога |

## <a name="sales-tax-group"></a>Налоговая группа

| Служба расчета налогов                         | Finance                                    |
| ----------------------------------------------- | ------------------------------------------ |
| Налоговая группа                                       | Налоговая группа                            |
| Налоговые коды для этой налоговой группы                  | Налоговые коды для этой налоговой группы |
| Налоговый код помечен как **Освобожденный**         | Освобождается                                     |
| Налоговый код помечен как **Освобожденный**         | Код налогового освобождения                                |
| Налоговый код помечен как **Является возмещением расходов** | Удержание с покупателя                             |
| Налоговый код помечен как **Является налогом за пользование**        | Налог за пользование                                    |

## <a name="item-sales-tax-group"></a>Налоговая группа номенклатур

| Служба расчета налогов             | Finance                                         |
| ----------------------------------- | ----------------------------------------------- |
| Налоговая группа номенклатуры                      | Налоговая группа номенклатур                            |
| Налоговые коды для этой налоговой группы номенклатур | Налоговые коды для этой налоговой группы номенклатур |

После завершения синхронизации продолжайте управлять остальными параметрами в Finance для целей разноски и отчетности.

> [!NOTE]
> Синхронизация настройки налога из Finance в службу расчета налогов не поддерживается. Для существующей среды Finance сначала необходимо создать настройку налога в службе расчета налогов. Затем синхронизируйте данные настройки обратно в среду Finance.
