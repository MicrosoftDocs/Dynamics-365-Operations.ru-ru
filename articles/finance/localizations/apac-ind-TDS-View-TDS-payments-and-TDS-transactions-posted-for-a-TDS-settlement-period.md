# <a name="view-posted-tds-payments-and-transactions-for-a-tds-settlement-period"></a>Просмотр разнесенных платежей TDS и проводок для периода сопоставления TDS

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как просмотреть платежи и проводки по налогу, удержанный в источнике (TDS), которые были разнесены для периода сопоставления.

1. Перейдите в раздел **Налог \> Косвенные налоги \> Подоходный налог \> Периоды сопоставления подоходного налога**.

    [![Страница периодов сопоставления подоходного налога.](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)

2. На странице **Периоды сопоставления подоходного налога** выберите **Платежи подоходного налога**, чтобы открыть страницу **Платежи подоходного налога**, на которой можно просмотреть сопоставления TDS, сделанные для конкретного периода сопоставления TDS.

    На вкладке **Обзор** показаны следующие данные:

    - **Дата** — дата сопоставления TDS.
    - **Ваучер** — код ваучера проводки сопоставления TDS.
    - **Тип налога** — тип налога, для которого запускается процесс сопоставления.
    - **Номер налогового счета (TAN)** — номер налогового счета (TAN) сопоставленной проводки TDS.
    - **Период сопоставления** — период сопоставления TDS, для которого необходимо выполнить процесс сопоставления.
    - **Начальная дата** — начальная дата периода сопоставления.
    - **Конечная дата** — конечная дата периода сопоставления.
    - **Версия платежа подоходного налога** — версия платежа по подоходному налогу: **Исходная**, **Последние корректировки** или **Итоговый список**.

5. Выберите **Ваучер** для просмотра записей ваучера для проводки TDS.
6. Выберите **Проводки подоходного налога** для просмотра подробных сведений о проводках TDS, которые были сопоставлены для определенного периода в течение периода сопоставления. Выберите **Ваучер** для просмотра записей ваучера для проводки TDS. Выберите **Компоненты подоходного налога** для просмотра TDS, рассчитанного по компонентам налогов для конкретного налогового кода TDS.
7. Закройте страницу **Платежи подоходного налога**, чтобы вернуться на страницу **Периоды сопоставления подоходного налога**.
8. Выберите **Проводки подоходного налога** для просмотра подробных сведений о проводках TDS, которые были сопоставлены для периода сопоставления.
