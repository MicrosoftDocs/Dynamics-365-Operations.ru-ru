# <a name="tds-calculation-on-invoices"></a>Расчет TDS по накладным

[!include [banner](../includes/banner.md)]

В этой статье содержится справка для проводок, в которых налог, удерживаемый у источника (TDS), рассчитывается на уровне накладной.

| Серийный номер | Тип проводки                                 | Сумма проводки | Имя страницы и путь выбора                                 | Тип счета и тип корр. счета                         |
| ------------- | ------------------------------------------------ | ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 1.            | Покупка сделана от поставщика / расходы на запись   | Дебит или кредит  | Страница Общие журналы (Главная книга > Записи журнала > Общие журналы) страница Журнал утверждения накладных (Расчеты с поставщиками > Накладные > Утверждение накладных), страница Журнал накладных (Расчеты с поставщиками > Накладные > Журнал накладных) | Главная книга (Дб.) Поставщик (Кр.).  Подоходный налог рассчитывается для комбинации "Поставщик-Книга" только в том случае, когда счет ГК имеет тип разноски **Покупка** **наличности**. |
| 2.            | Покупка сделана от клиента / расходы на запись | Дебит или кредит  | Страница Общие журналы (Главная книга > Записи журнала > Общие журналы), страница Журнал накладных (Расчеты с поставщиками > Накладные > Журнал накладных) | Главная книга (Дб.) Клиент (Кр.)                                 |
| 3.            | Покупка ОС у поставщика              | Дебит или кредит  | Страница Общие журналы (Главная книга > Записи журнала > Общие журналы) страница Журнал регистрации накладных (Расчеты с поставщиками > Накладные > Регистрация накладных), страница Журнал накладных (Расчеты с поставщиками > Накладные > Журнал накладных) | Основные средства (Дб.) Поставщик (Кр.)                             |
| 4.            | Покупка ОС у клиента            | Дебит или кредит  | Страница Общие журналы (Главная книга > Записи журнала > Общие журналы), страница Журнал накладных (Расчеты с поставщиками > Накладные > Журнал накладных) | Основные средства (Дб.) Клиент (Кр.)                           |
| 5.            | Накладная по покупке (оплата TDS)                  |                    | Страница Заказ на покупку (Расчеты с поставщиками > Заказы на покупку > Все заказы на покупку) |                                                              |
| 6.            | Накладная по продаже (TDS получается)                  |                    | Страница Заказ на продажу (Расчеты с клиентами > Заказы > Все заказы на продажу), страница Накладная с произвольным текстом (Расчеты с клиентами > Накладные > Все накладные с произвольным текстом) |                                                              |
