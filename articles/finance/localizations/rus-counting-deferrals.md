# <a name="deferrals-counting-russia"></a>Инвентаризация РБП (Россия)

[!include [banner](../includes/banner.md)]

Функциональность РБП поддерживает процесс инвентаризации РБП и позволяет печатать отчет **Акт инвентаризации (ИНВ-11)**.

Перед созданием отчета **Акт инвентаризации (ИНВ-11)** необходимо создать РБП и разнести проводки списания в журнале РБП.

1. Перейдите к **Главная книга** \> **Периодические задачи** \> **Расходы будущих периодов** \> **Журнал инвентаризации РБП**.

    Можно использовать страницу **Журнал инвентаризации РБП** для создания журнала инвентаризации РБП и печати отчета **Акт инвентаризации (ИНВ-11)**.

2. Выполните следующие действия, чтобы создать новый журнал инвентаризации РБП:

    1. В области действий выберите **Создать**.

        > [!NOTE]
        > Вы не можете создать более одного журнала инвентаризации РБП с одинаковой датой инвентаризации.

    2. В поле **Имя** выберите наименование журнала, которое имеет тип журнала **Расходы будущих периодов**.
    3. В поле **Номер разрешения** вводится номер разрешения для журнала инвентаризации.
    4. В поле **Дата разрешения** выберите дату разрешения для журнала инвентаризации.
    5. В поле **Дата начала инвентаризации** выберите дату начала периода инвентаризации РБП. По умолчанию для этого поля задана текущая дата.

        Поле **Дата окончания инвентаризации** установлена на дату окончания, которая была установлена для периода инвентаризации РБП в диалоговом окне **Завершить инвентаризацию РБП**.

    6. Выберите вкладку **Должностные лица** для настройки имен, должностей и званий должностных лиц.
    7. В поле **Должность** выберите название должности должностного лица.
    8. В поле **Имя сотрудника** выберите имя должностного лица в списке сотрудников компании.
    9. В поле **Название должности** выберите название должности сотрудника.

3. На панели операций выберите **Строки**, чтобы открыть страницу **Строка журнала инвентаризации РБП**.

    Можно использовать страницу **Строка журнала инвентаризации РБП** для просмотра или изменения существующих строк журнала или для создания вручную новых строк.

    В следующей таблице описаны поля на странице **Строка журнала инвентаризации РБП**.

    | Поле               | Описание                                                                                                         |
    |---------------------|---------------------------------------------------------------------------------------------------------------------|
    | Код модели        | Выберите номер модели РБП.                                                                                   |
    | Дата начала инвентаризации | Дата начала, указанная для периода инвентаризации РБП на странице **Журнал инвентаризации РБП**.       |
    | Дата окончания инвентаризации   | Дата окончания, указанная для периода инвентаризации РБП на странице **Журнал инвентаризации РБП**.         |
    | Идентификатор расхода         | Выберите идентификационный код, связанный с РБП.                                                    |
    | Название                | Введите имя расхода будущего периода.                                                                                            |
    | Дата вложения       | Выберите дату создания расхода будущего периода.                                                                                  |
    | Срок списания    | Выберите период списания для расхода будущего периода.                                                                       |
    | Сумма списания  | Введите сумму, списанную между созданием РБП и началом периода инвентаризации. |
    | Оставшаяся сумма    | Введите сумму, которая должна быть списана в будущем.                                                            |
    | Сумма РБП    | Выберите сумму РБП в валюте по умолчанию.                                                                 |

4. Выполните следующие действия, чтобы изменить существующие строки журнала инвентаризации РБП:

    1. В поле **Номер модели** выберите номер модели РБП из строк журнала.

        > [!NOTE]
        > Поля **Дата начала инвентаризации** и **Дата окончания инвентаризации** настроены на даты начала и окончания, которые были установлены для периода инвентаризации РБП на странице **Журнал инвентаризации РБП**.

    2. В поле **Идентификатор РБП** измените код РБП.
    3. В поле **Имя** измените имя РБП.
    4. В поле **Дата присоединения** выберите дату, когда был создан РБП.
    5. В поле **Время списания** выберите период списания для РБП.
    6. В поле **Сумма списания** введите сумму, списанную между созданием РБП и началом периода инвентаризации.
    7. В поле **Сумма остатка** введите сумму, которая должна быть списана в будущем.
    8. В поле **Сумма РБП** выберите сумму РБП в валюте по умолчанию.
    9. Закройте страницу **Строка журнала инвентаризации РБП**.

    ![Страница строки журнала инвентаризации РБП](media/rus-counting-deferrals-01.jpg)

5. Чтобы распечатать отчет **Акт инвентаризации (ИНВ-11)**, выполните следующие действия:

    1. На панели операций выберите **Закрыть**, чтобы открыть диалоговое окно **Завершить инвентаризацию РБП**.
    2. В поле **Дата окончания инвентаризации** введите дату, когда требуется закрыть журнал.

        > [!NOTE]
        > Указанная вами дата отображается в поле **Дата окончания инвентаризации** страницы **Журнал инвентаризации РБП**.

    3. Нажмите кнопку **ОК**, чтобы закрыть журнал.
    4. На панели операций выберите **Печать**, чтобы открыть диалоговое окно **Инвентаризация РБП**.
    5. В поле **Код модели** выберите код модели Расхода будущего периода.
    6. Выберите **OK** для печати **Акта инвентаризации (ИНВ-11)** с помощью шаблона Excel для номера выбранной модели.
