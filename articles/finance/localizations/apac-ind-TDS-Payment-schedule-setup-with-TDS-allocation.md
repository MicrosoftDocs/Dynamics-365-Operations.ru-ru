# <a name="set-up-payment-schedules-with-tds-allocation"></a>Настройка графиков оплаты с распределением TDS

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как настроить графики оплаты с распределением налогов, которые удерживаются в источнике (TDS).

1. Перейдите в раздел **Расчеты с поставщиками \> Настройка платежей \> Графики платежей**.

    [![Страница графиков оплаты.](./media/apac-ind-TDS-27.png)](./media/apac-ind-TDS-27.png)

2. В области действий выберите **Создать**, чтобы создать график оплаты, и введите требуемые сведения.
3. В поле **Распределение** выберите способ, который будет использоваться для распределения оплаты для графика оплаты:

    - Итоговый
    - Фиксированная сумма
    - Фиксированное количество
    - Задано

    В поле **Распределение подоходного налога** отображается метод распределения TDS для графика оплаты. Если выбрано **Итого** в поле **Распределение**, в поле **Распределение подоходного налога** автоматически устанавливается значение **Итого**. При выборе **Фиксированная сумма**, **Фиксированное количество** или **Указано** в поле **Распределение** в поле **Распределение подоходного налога** автоматически устанавливается значение **Пропорционально**.

    > [!NOTE]
    > Если для поля **Распределение подоходного налога** установлено значение **Итого**, платежные взносы рассчитываются на основе валовой суммы, которая включает сумму TDS. Если для поля **Распределение подоходного налога** установлено значение **Пропорционально**, платежные взносы рассчитываются на основе валовой суммы накладной после удержания суммы TDS.

4. Введите остальные необходимые данные и закройте страницу.
