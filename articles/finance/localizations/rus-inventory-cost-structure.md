# <a name="inventory-cost-structure"></a>Структура стоимости запасов
[!include [banner](../includes/banner.md)]

Структуру затрат на номенклатуру можно сохранить в складских проводках в качестве накладных расходов. После этого можно просмотреть структуру затрат на странице **Анализатор себестоимости**. Можно изменить стоимость складских проводок, используя следующие функции:

- Распределите накладные расходы из заказов на покупку и заказов на продажу.
- Настройте затраты на поступление на склад, используя периодическую операцию **Закрытие и коррекция**.

При распределении накладных расходов по заказу на покупку создаются сопоставления запасов. Затем можно просмотреть складские проводки прихода в виде кодов накладных расходов. Для работы со стоимостью складских проводок необходимо создать код накладных расходов для складских проводок расходов. Для этой цели доступны следующие функции системы:

- Настройка параметра **Структура накладных расходов**, чтобы разрешить или запретить создание сопоставлений запасов в виде кодов накладных расходов.
- Периодическая операция **Закрытие и коррекция** используется для регистрации корректировки затрат по складским проводкам или по запасам в наличии, которые имеют коды накладных расходов.
- Страница **Анализатор себестоимости** предназначена для просмотра и изучения структуры затрат в виде кодов накладных расходов.

## <a name="preliminary-setup"></a>Предварительная настройка

### <a name="set-up-the-miscellaneous-charges-structure"></a>Настройка структуры накладных расходов

1. Перейдите в раздел **Управление запасами** \> **Настройка** \> **Параметры управления запасами и складами**.
2. На вкладке **Общие** в разделе **Корректировка** задайте параметр **Структура накладных расходов**:

    - Если для этого параметра установить значение **Нет**, одна проводка сопоставления для общей суммы создается для каждой проводки прихода. Код накладных расходов не введен в сопоставление запасов.
    - При выборе значения **Да** сопоставления запасов создаются как коды накладных расходов при разноске накладной по покупке.

### <a name="set-up-a-charges-code-in-inventory-management"></a>Настройка кода расходов в управлении запасами

1. Перейдите **Управление запасами** \> **Настройка** \> **Накладные расходы** \> **Коды накладных расходов**.
2. Создание кода расходов. Коды накладных расходов, которые создаются в модуле **Управление запасами**, имеют следующие ограничения по конфигурации:

    - В поле **Тип** в разделе **Дебет** можно выбрать только **Номенклатура**.
    - В поле **Тип** в разделе **Кредит** можно выбрать только **Счет ГК**.

![Страница накладных расходов](media/1_Charges_codes.png)

## <a name="adjustment-of-item-cost-in-terms-of-miscellaneous-charges-codes"></a>Корректировка затрат по номенклатуре в виде кодов накладных расходов

В модуле **Управление запасами** накладные расходы могут распределяться несколькими способами:

- Коррекция запасов в наличии вручную.
- Корректировка проводок вручную.
- Корректировка запасов в наличии или проводок с помощью мастера.

Существующие коды накладных расходов можно указать при корректировке себестоимости номенклатур на страницах **Наличие**, **Корректировать проводки** и **Мастер коррекции себестоимости складских проводок или всех запасов в наличии**.

### <a name="close-inventory"></a><a name="close-inventory"></a>Закрыть запасы

Перед корректировкой запасов в наличии необходимо закрыть запасы. Дополнительные сведения о закрытии склада см. в разделе [Закрытие запасов](../../supply-chain/cost-management/inventory-close.md).

1. Перейдите **Управление запасами** \> **Периодические задачи** \> **Закрытие и коррекция**.
2. На панели операций выберите **Процедура закрытия** \> **Закрыть запасы**.
3. В диалоговом окне **Закрыть запасы** в поле **Закрыть запасы до** укажите дату, с которой должны быть закрыты запасы.
4. В поле **Детализация** выберите **Все**.

    ![Диалоговое окно закрытия запасов](media/2_Close_inventory.png)

5. Нажмите **ОК**.

### <a name="cancel-inventory-closing"></a><a name="cancel-inventory-closing"></a>Отмена закрытия запасов

Прежде чем можно будет корректировать проводки, необходимо убедиться, что запасы не закрыты.

1.  Перейдите **Управление запасами** \> **Периодические задачи** \> **Закрытие и коррекция**.
2.  Выберите операцию закрытия запасов, а затем в области действий выберите **Отмена**.
3.  В диалоговом окне **Отмена - Начать** выберите **ОК**.

### <a name="adjust-on-hand-inventory"></a><a name="adjust-on-hand-inventory"></a>Корректировка запасов в наличии

Дополнительные сведения о корректировке запасов в наличии см. в разделе [Корректировка себестоимости запасов в наличии](../../supply-chain/cost-management/adjust-hand-inventory-cost-values.md).

1.  Закройте складские запасы, как описано в разделе [Закрытие запасов](#close-inventory) ранее в этой теме.
2.  Перейдите **Управление запасами** \> **Периодические задачи** \> **Закрытие и коррекция**.
3.  В области действий выберите **Корректировка \> Наличие**.
4.  На панели операций выберите **Выбрать**.
5.  В диалоговом окне **Выбрать запасы в наличии** укажите складские аналитики, которые должны отображаться после завершения корректировки.

    ![Диалоговое окно выбора параметров запасов в наличии](media/3_Select_on-hand_inventory.png)

6.  Нажмите **ОК**. На странице **Наличие** показаны строки запасов, выбранные для корректировки, вместе с указанными складскими аналитиками.
7.  Выберите строку для номенклатуры, а затем в поле **Код накладных расходов** выберите код накладных расходов для номенклатуры. Выбранный код накладных расходов вводится в сопоставление запасов после разноски корректировки.

    ![Страница наличия](media/4_On-hand.png)

8.  Чтобы ввести код накладных расходов во всех строках на странице **Наличие**, в области действий выберите **Корректировка** \> **Код накладных расходов**.
9.  В диалоговом окне **Изменить код накладного расхода.** задайте следующие значения:

    - В поле **Код накладных расходов** выберите код накладных расходов.
    - Установите параметр **Применить ко всем** как **Да**. Если этот параметр имеет значение **Нет**, значение в поле **Код накладных расходов** будет скорректировано только в строке, выбранной на странице **Наличие**.

    ![Диалоговое окно изменения кода накладных расходов](media/5_Change_misc._charges_code.png)

10. Нажмите **ОК**.
11. На панели операций выберите **Разнести**.
12. В диалоговом окне **Коррекция запасов в наличии** в поле **Детализация** выберите способ разноски корректировок проводок:

    - **Все** — корректировки разносятся для всех номенклатур с одинаковыми параметрами в профиле разноски.
    - **Номенклатурная группа** — разнесение корректировок выполняется для всех номенклатур из такой же номенклатурной группы.
    - **Код номенклатуры** — корректировки разносятся для каждой номенклатуры.

13. В поле **Примечание** введите примечание для корректировки.

    ![Диалоговое окно коррекции запасов в наличии](media/6_Adjustment_of_on-hand_inventory.png)

14. Выберите **ОК**, чтобы разнести корректировку.

При разноске корректировки корр. счет главной книги определяется следующим образом:

   - Если код накладных расходов введен в строке номенклатуры, корректировка разносится на корр. счет, настроенный для кода накладных расходов.
   - Если код накладных расходов не введен в строке номенклатуры, корректировка назначается для корр. счета прибылей и убытков, который настраивается на вкладке **Запасы** на странице **Разноска** (**Управление запасами** \> **Настройка** \> **Разноска** \> **Разноска**).

### <a name="adjust-transactions"></a>Корректировать проводки

1. Убедитесь, что запасы не закрыты. Если они были закрыты, отмените закрытие запасов, как описано в разделе [Отмена закрытия запасов](#cancel-inventory-closing) ранее в этой теме.
2. Перейдите **Управление запасами** \> **Периодические задачи** \> **Закрытие и коррекция**.
3. В области действий выберите **Корректировка** \> **Проводки**.
4. На панели операций выберите **Выбрать**.
5. В диалоговом окне **Корректировка проводок** укажите критерии выбора для проводок прихода на склад, которые необходимо скорректировать ,а затем нажмите **ОК**.

    ![Диалоговое окно корректировки проводок](media/7_Adjust_transactions.png)

6. Выберите коды накладных расходов для номенклатур и разнесите корректировку, как описано в разделе [Корректировка запасов в наличии](#adjust-on-hand-inventory) ранее в этой теме. Корр. счет главной книги будет определяться в соответствии с тем же принципом.

### <a name="use-a-wizard-to-adjust-item-cost"></a>Используйте мастера для корректировки затрат на номенклатуру 

Для получения дополнительной информации о см. раздел [Мастер корректировки запасов](rus-inventory-adjustment-wizard.md).

1. Перейдите **Управление запасами** \> **Периодические задачи** \> **Закрытие и коррекция**.
2. В области действий выберите **Корректировка** \> **Мастер**.
3. На странице **Добро пожаловать** выберите **Далее**.
4. На странице **Метод коррекции себестоимости** выберите метод корректировки:

    - **В наличии** — коррекция запасов в наличии.
    - **Проводки** — корректировка проводок.

5. Выберите **Далее**. Отображается либо страница **Выбрать запасы в наличии**, либо страница **Корректировать проводки** в зависимости от выбранного метода корректировки.
6. Укажите критерии выбора, а затем нажмите **ОК**.
7. На странице **Результат выбора** можно просмотреть номенклатуры в наличии или проводки по номенклатуре, которые выбраны для корректировки, затем нажмите **Далее**.

    ![Страница результатов выбора](media/8_Selection_result.png)

8. На странице **Функции для вычисления корректирующих сумм** выберите метод расчета корректирующих сумм, затем выберите **Далее**.

    ![Страница функций для вычисления корректирующих сумм](media/9_Functions_for_calculating_adjustments_amounts.png)

9. На странице **Результаты распределения** можно просмотреть результаты распределения. В поле **Код накладных расходов** выберите код накладных расходов для строки.

    ![Страница результатов распределения](media/10_Results_of_allocation.png)

10. Выберите **Далее**.
11. На странице **Разноска** укажите сведения для разноски корректировки:

   - В поле **Дата корректировки** укажите дату корректировки.
   - В поле **Детализация** выберите способ разноски корректировок проводок:

     - **Все** — корректировки разносятся для всех номенклатур с одинаковыми параметрами в профиле разноски.
     - **Номенклатурная группа** — разнесение корректировок выполняется для всех номенклатур из такой же номенклатурной группы.
     - **Код номенклатуры** — корректировки разносятся для каждой номенклатуры.
   - Задайте параметр **Корр. счет Прибыли/Убытки** и поле **Корр. счет** в зависимости от счета, которому должна быть назначена корректировка во время разноски:

     - Если код накладных расходов введен в строке номенклатуры, корректировка назначается корр. счету, настроенному для кода накладных расходов.
     - Если код накладных расходов не введен в строке номенклатуры, а параметр **Корр. счет Прибыли/Убытки** на странице **Разноска** имеет значение **Да**, корректировка назначается обычным способом для корр. счета прибылей и убытков, который настраивается на вкладке **Запасы** на странице **Разноска** (**Управление запасами \> Настройка \> Разноска \> Разноска**).
     - Если код накладных расходов не введен в строке номенклатуры, а параметр **Корр. счет Прибыли/Убытки** на странице **Разноска** имеет значение **Нет**, корректировка назначается счету, указанному в поле **Корр. счет**.

   ![Страница разноски](media/11_Posting.png)

12. Выберите **Далее**.
13. На странице **Готово** установите флажок **Показать список ваучеров ГК**, чтобы отобразить список ваучеров ГК.
14. Выберите **Готово**, чтобы разнести корректировку.

## <a name="view-item-cost-structure-in-the-inventory-settlements-and-cost-explorer"></a>Просмотр структуры затрат номенклатуры в сопоставлениях запасов и анализаторе себестоимости

На странице **Сопоставления** можно просмотреть проводки корректировки себестоимости запасов. В результате закрытия запасов проводки прихода сопоставляются с проводками расхода. После завершения закрытия запасов можно открыть страницу **Анализатор себестоимости** для просмотра и изучения структуры затрат номенклатуры с точки зрения накладных расходов.

1. Закройте складские запасы, как описано в разделе [Закрытие запасов](#close-inventory) ранее в этой теме.
2. Щелкните **Управление сведениями о продукте** \> **Продукты** \> **Выпущенные продукты**.
3. На странице **Сведения об используемых продуктах** выберите номенклатуру, затем, в области действий, на вкладке **Управление затратами**, в группе **Проводки затрат**, выберите **Проводки**.
4. На странице **Проводки по запасам** выберите проводку для полученной номенклатуры, для которой необходимо просмотреть структуру затрат. Затем в области действий на вкладке **Склад** в группе **Учет затрат** выберите **Сопоставления**.
5. На странице **Сопоставления** изучите следующую информацию:

    - В поле **Код расходов** показан код накладных расходов, который был распределен для проводки прихода номенклатуры.
    - В поле **Счет поставщика** показан счет поставщика, у которого накладные расходы были приобретены. Дополнительные сведения о приобретении накладных расходов от стороннего поставщика см. в разделе [Накладные расходы третьей стороны](rus-third-party-misc-charges.md).

    - В поле **Накладная** показан номер накладной по закупке накладных расходов.

    > [!NOTE]
    > Если поля **Счет поставщика** и **Накладная** не заполнены, операция корректировки затрат была выполнена стандартным методом распределения накладных расходов.

    ![Страница сопоставлений](media/12_Settlements.png)

6. На странице **Проводки по запасам** в области действий на вкладке **Запасы** в группе **Учет затрат** выберите **Анализатор себестоимости**.

    ![Страница анализатора себестоимости](media/13_Cost_explorer.png)

На страницах **Сопоставление** и **Анализатор себестоимости** коды накладных расходов с типом дебета **Номенклатура** и типом кредита **Счет ГК** отображаются в виде отдельной строки. Коды накладных расходов с другими типами дебета и кредита включаются в себестоимость номенклатуры как прямые расходы (стандартная функция). Они не распределяются на странице **Сопоставления**. Следовательно, они не отображаются на странице **Анализатор себестоимости**.

Если поле **Код накладных расходов** на странице **Сопоставления** не задано, код накладных расходов не будет показан на странице **Анализатор себестоимости**. Если поле **Код накладных расходов** на странице **Сопоставления** задано, код накладных расходов будет показан как отдельная строка на странице **Анализатор себестоимости**.
