# <a name="settle-tds-payments-to-tds-authority-vendors-and-generate-tds-challan"></a>Сопоставление проводок TDS с поставщиками органа TDS и создание Challan TDS

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как сопоставлять платежи налога, удерживаемого в источнике (TDS), с поставщиками органа TDS.

1. Перейдите в раздел **Расчеты с поставщиками \> Платежи \> Журнал платежей поставщикам**.

    [![Страница журнала платежей поставщика.](./media/apac-ind-TDS-51.png)](./media/apac-ind-TDS-51.png)

2. На странице **Журнал платежей поставщикам** выберите **Создать**, создать строку журнала.
3. В поле **Счет** выберите поставщика органа TDS, для которого требуется сопоставить платежи TDS.
4. Выберите **Проводки сопоставления**, чтобы открыть страницу **Проводки сопоставления**, на которой можно просмотреть сопоставленные проводки TDS для поставщика органа TDS.

    Сопоставленные проводки TDS для периода сопоставления отображаются следующим образом:

    - Проводки TDS, в которых суть категории налогоплательщика является **Компания**, отображаются как одна строка проводки.
    - Проводки TDS, в которых суть категории налогоплательщика — **HUF**, **Фирма**, **Физическое лицо**, **AOP**, **BOI**, **Локальный орган** или **Прочее**, отображаются как одна строка проводки.
    - В поле **Сумма** отображается общая сумма TDS, выплачиваемая поставщику органа TDS.

5. Выберите **Проводки по подоходному налогу** для просмотра других проводок TDS, включенных для записи сопоставления. На этой странице можно просмотреть разбиение каждой проводки TDS, которая была включена в процесс сопоставления для периода сопоставления.
6. На вкладке **Обзор** установите флажок **Отметить** для проводок TDS, которые будут сопоставляться с поставщиком органа TDS.

    Вкладка **Обзор** содержит следующие сведения для каждой открытой проводки TDS:

    - **Дата** — дата проводки TDS.
    - **Ваучер** — код ваучера.
    - **Источник** — модуль, в котором была разнесена проводка TDS.
    - **Поставщик/Клиент** — номер счета поставщика или клиента, из которого вычитается TDS.
    - **Имя лица/стороны, в пользу которой производится налоговый вычет** — имя поставщика или клиента, из которого вычитается TDS.
    - **Суть налогоплательщика** — суть категории налогоплательщика, к которой принадлежит лицо, в пользу которого производится налоговый вычет.
    - **Сумма** — сумма накладной, для которой рассчитан TDS.
    - **Сумма налога** — сумма TDS, рассчитанная для проводки.

    > [!NOTE]
    > Для всех проводок TDS, которые не должны сопоставлены с поставщиком TDS, следует снять флажок **Отметить**.

    На вкладке **Общие** в поле **PAN** отображается постоянный номер счета (PAN) лица, в пользу которого производится налоговый вычет. В поле **Дата** отображается дата расчета TDS, а в поле **Значение** отображается общее процентное значение, которое использовалось для расчета TDS.

7. Выберите **Ваучер** для просмотра записей ваучера для проводки TDS.
8. Закройте страницу.
10. Выберите **Компоненты подоходного налога**, чтобы открыть страницу **Компоненты подоходного налога**, на которой можно просмотреть TDS, рассчитанный на основе каждого налогового кода TDS, для конкретного налога TDS.

    В поле **Компонент налога** на вкладке **Обзор** отображается налоговый компонент TDS, который использовался для проводки. В поле **Сумма** отображается сумма TDS, рассчитанная для компонента налога TDS в базовой валюте. В поле **Накопленная сумма** отображается общая сумма TDS, рассчитанная для компонента налога TDS для всех сопоставленных проводок.

    На вкладке **Сумма** в разделе **Валюта по умолчанию** отображается сумма TDS, рассчитанная для компонента налога TDS в валюте по умолчанию. В разделе **Вторичная валюта** отображается сумма во вторичной валюте.

11. Закройте страницу **Компоненты подоходного налога**.
12. На странице **Редактирование открытых проводок** в поле **Сумма** обратите внимание на то, что общая сумма, которая должна быть сопоставлена с поставщиком органа TDS для периода сопоставления, обновляется.
13. Чтобы выполнить сопоставление проводок TDS для других периодов сопоставления TDS поставщику органа TDS, установите флажок **Отметить** для проводок.
14. Закройте страницу **Редактирование открытых проводок**.

    > [!NOTE]
    > Если для сопоставления на странице **Проводки по подоходному налогу** выбрано несколько проводок, общая сумма TDS по выбранным проводкам обновляется в поле **Корректировка** на странице **Редактирование открытых проводок**. Сумма корректировки обновляется в строке журнала на странице **Ваучер журнала** и закрывается страница **Редактирование открытых проводок**.

    На странице **Ваучер журнала** в поле **Дебет** отображается общая сумма, подлежащей оплате поставщику органа TDS.

15. Введите сведения о корр. счете.

    > [!NOTE]
    > Если проводки TDS имеют разный номер налогового счета (TAN), строки журнала создаются для TAN на странице **Ваучер журнала**.

16. На вкладке **Сборы по платежам** в поле **Код сбора** выберите код сбора, который имеет тип сбора **Процент** или **Прочее**, чтобы начислить сборы по платежам для отложенных платежей, сделанных для поставщика органа TDS.

    На вкладке **Сведения о налоге** в разделе **Данные о компании** в поле **Имя** указано название компании. В разделе **Подоходный налог** в поле **Номер налогового счета (TAN)** отображается TAN, присоединенный к строке проводки.

17. Проверьте и разнесите журнал.
18. Выберите **Подоходный налог \> Сведения о Challan**, чтобы ввести сведения о Challan для проводки.

    В поле **Ваучер** показан код ваучера проводки.
    
19. Установите флажок **TDS, депонированный по записи книги**, если сумма TDS депонируется с помощью записи книги.
20. В поле **Номер Challan** введите номер Challan, который используется для создания платежа поставщику органа TDS.
21. В поле **Дата** введите дату Challan.
22. В поле **Наименование банка** выберите имя банка, в который следует депонировать сумму TDS, оплачиваемую поставщику органа TDS. В этом поле перечислены все банковские счета, которые были настроены для поставщика органа TDS в **Расчеты с поставщиками \> Все поставщики \> Настройка \> Банковские счета**.
23. В поле **Код БСР** введите код основного статистического возврата (BSR) банка.
24. Закройте страницу.

### <a name="example"></a>Пример

Период 01.04.2009 сопоставляется для группы компонентов TDS **Аренда** с помощью периодического процесса сопоставления TDS. Общая сумма TDS 141 625,00 разносится на счет поставщика органа TDS для периода сопоставления TDS. Эту сумму можно просмотреть в поле **Сумма** на странице **Редактирование открытых проводок** для поставщика органа TD.

Если выбрать **Проводки по подоходному налогу** для просмотра других проводок TDS, которые были сопоставлены для этого периода, отображаются следующие сведения.

| Сумма TDS |
|------------|
| 16,995.00  |
| 22,660.00  |
| 28,325.00  |
| 16,995.00  |
| 28,325.00  |
| 16,995.00  |
| 11,330.00  |

Для конкретной суммы TDS можно выбрать **Компоненты подоходного налога**, просмотреть TDS, рассчитанный на основе каждого налогового компонента TDS, для конкретного кода налога TDS. В данном примере выбираются **Компоненты подоходного налога** для суммы TDS 16 995,00. Отображается сумма налога, рассчитанная на компонент для проводки.

| Налоговый компонент | Сумма, руб.    | Нарастающее сальдо |
|---------------|-----------|--------------------|
| TDS           | 1,5000.00 | 125,000.00         |
| Доплата     | 1,500.00  | 12,500.00          |
| PE-Cess       | 330.00    | 2,750.00           |
| SHE Cess      | 165.00    | 1,375.00           |

Если выбраны только суммы TDS 16 995,00, 22 660,00, и 28 325,00 для сопоставления на странице **Проводки по подоходному налогу**, общая сумма для сопоставления отображается как **67 980,00** в поле **Корректировка** на странице **Редактирование открытых проводок**. Если эта проводка помечена для сопоставления, а страница **Редактирование открытых проводок** закрыта, сумма **67 980,00** отображается в поле **Дебет** на странице **Ваучер журнала**.

Теперь можно разнести журнал и создать TDS Challan.

### <a name="adjustment-of-advance-payments-that-are-made-to-tds-authority-vendors"></a>Корректировка авансовых платежей, выполненных поставщикам органа TDS

Чтобы скорректировать авансовый платеж, сделанный поставщику TDS в отношении фактического платежа, перейдите **Расчеты с поставщиками \> Поставщики \> Все поставщики \> Редактирование проводок**. Если фактический платеж превышает авансовый платеж, для проводки создаются два номера Challan. Однако в запросе TDS показан только первый номер Challan.
