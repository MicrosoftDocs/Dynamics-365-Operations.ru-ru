# <a name="work-with-configurations"></a>Работа с конфигурациями

[!include [banner](../includes/banner.md)]

Конфигурации электронной отчетности (ER) являются одним из основных наборов компонентов функций электронного выставления накладных. Конфигурация электронной отчетности содержит настройку структуры файлов и набор правил преобразования для преобразования данных двумя способами:

- **Экспорт конфигурации формата электронной отчетности** — эта конфигурация преобразует данные из единой внутренней структуры (модель электронной отчетности) в формат электронного файла.
- **Импорт конфигурации формата электронной отчетности** — эта конфигурация преобразует данные из формата электронного файла в единую внутреннюю структуры (модель электронной отчетности).

В дополнение к созданию исходящих электронных файлов в заранее определенном формате, конфигурации электронной отчетности широко используются для анализа входящих сообщений из внешних веб-служб как ответов. Эта функция позволяет создать настраиваемую логику для получения результатов связи с внешними каналами, преобразовать эти результаты (коды, сообщения и журналы) в структуру, доступную для чтения пользователем, а затем передать эту информацию клиентскому приложению.

Чтобы начать использовать конфигурации электронной отчетности в функции электронного выставления накладных, необходимо связать их с функцией и сделать их доступными на вкладке **Конфигурации** для текущей версии функции. Для работы со связанной конфигурацией электронной отчетности следует выберите **Добавить**, **Удалить**, **Изменить** или **Просмотреть**.

Прежде чем можно будет добавить ссылку на конфигурацию электронной отчетности, эта конфигурация должна существовать в репозитории службы Regulatory Configuration Service (RCS). Чтобы просмотреть конфигурации электронной отчетности, доступные в репозитории RCS, выполните следующие действия.

1. Войдите в учетную запись RCS.
2. В рабочей области **Электронная отчетность** в разделе **Конфигурации** выберите плитку **Конфигурации отчетности**.

Сведения о том, как создать новую конфигурацию электронной отчетности или импортировать существующую конфигурацию электронной отчетности из глобального репозитория, см. раздел [Электронная отчетность](../../fin-ops-core/dev-itpro/analytics/general-electronic-reporting.md).

Чтобы откорректировать связанную конфигурацию электронной отчетности, выберите **Изменить** на вкладке **Конфигурации** для текущей функции электронного выставления накладных. Система автоматически создает версию конфигурации электронной отчетности. Если текущая версия не была создана активным поставщиком конфигураций, система создает производную версию, использующую ваш поставщик конфигурации. Если текущая версия была создана активным поставщиком конфигурации, система создает новую версию. В обоих случаях можно изменить созданную версию, а затем автоматически выполнять все необходимые обновления.
