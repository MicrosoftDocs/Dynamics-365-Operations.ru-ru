# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="4fe5f-103">Настройка групп компонентов подоходного налога для налогов типа TDS</span><span class="sxs-lookup"><span data-stu-id="4fe5f-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="4fe5f-104">В этой теме объясняется, как настроить группы компонентов подоходного налога, такие как **Аренда** и **Подрядчик** для типа налога, удерживаемого в источнике (TDS).</span><span class="sxs-lookup"><span data-stu-id="4fe5f-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="4fe5f-105">Перейдите **Налог \> Настройка \> Подоходный налог \> Группы компонентов подоходного налога**.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="4fe5f-106">[![Страница "Группы компонентов подоходного налога"](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="4fe5f-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="4fe5f-107">В поле **Тип налога** выберите **TDS**, чтобы настроить группы компонентов подоходного налога для типа налога TDS.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="4fe5f-108">На панели операций выберите **Создать** для создания строки.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="4fe5f-109">В поле **Группа компонентов подоходного налога** введите имя группы компонентов TDS.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="4fe5f-110">В поле **Описание** введите описание.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="4fe5f-111">На экспресс-вкладке **Общие** в поле **Статус** выберите статус резидента группы компонентов TDS.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="4fe5f-112">Параметры: **Резидент** и **Нерезидент**.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="4fe5f-113">В поле **Код раздела** введите код раздела, который применяется к группе компонентов TDS.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="4fe5f-114">Закройте страницу.</span><span class="sxs-lookup"><span data-stu-id="4fe5f-114">Close the page.</span></span>
