# <a name="run-the-periodic-tds-settlement-process"></a><span data-ttu-id="869a4-103">Выполнение периодического процесса сопоставления TDS</span><span class="sxs-lookup"><span data-stu-id="869a4-103">Run the periodic TDS settlement process</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="869a4-104">В этой теме объясняется, как сопоставить периодический налог, который удерживается в источнике (TDS).</span><span class="sxs-lookup"><span data-stu-id="869a4-104">This topic explains how to settle periodic Tax Deducted at Source (TDS).</span></span>

1. <span data-ttu-id="869a4-105">Перейдите **Налог \> Декларации \> Подоходный налог \> Платеж подоходного налога**.</span><span class="sxs-lookup"><span data-stu-id="869a4-105">Go to **Tax \> Declarations \> Withholding tax \> Withholding tax payment**.</span></span>

    <span data-ttu-id="869a4-106">[![Диалоговое окно платежа подоходного налога](./media/apac-ind-TDS-47.png)](./media/apac-ind-TDS-47.png)</span><span class="sxs-lookup"><span data-stu-id="869a4-106">[![Withholding tax payment dialog box](./media/apac-ind-TDS-47.png)](./media/apac-ind-TDS-47.png)</span></span>

2. <span data-ttu-id="869a4-107">В диалоговом окне **Платеж подоходного налога** в поле **Тип налога** выберите **TDS**.</span><span class="sxs-lookup"><span data-stu-id="869a4-107">In the **Withholding tax payment** dialog box, in the **Tax type** field, select **TDS**.</span></span>
3. <span data-ttu-id="869a4-108">В поле **Номер налогового счета (TAN)** выберите номер налогового счета (TAN), для которого необходимо выполнить процесс сопоставления.</span><span class="sxs-lookup"><span data-stu-id="869a4-108">In the **Tax Account Number (TAN)** field, select the Tax Account Number (TAN) to run the settlement process for.</span></span>
4. <span data-ttu-id="869a4-109">В поле **Группа компонентов подоходного налога** выберите группу компонентов TDS, для которой необходимо выполнить процесс сопоставления.</span><span class="sxs-lookup"><span data-stu-id="869a4-109">In the **Withholding tax component group** field, select the TDS component group to run the settlement process for.</span></span>
5. <span data-ttu-id="869a4-110">В поле **Период сопоставления** выберите период сопоставления TDS, для которого необходимо выполнить процесс сопоставления.</span><span class="sxs-lookup"><span data-stu-id="869a4-110">In the **Settlement period** field, select the TDS settlement period to run the settlement process for.</span></span>

    > [!NOTE]
    > <span data-ttu-id="869a4-111">Процесс сопоставления TDS выполняется для всех периодов, настроенных для периода сопоставления TDS на вкладке **Периоды** на странице **Периоды сопоставления подоходного налога** (**Налог \> Косвенные налоги \> Подоходный налог \> Периоды сопоставления подоходного налога**).</span><span class="sxs-lookup"><span data-stu-id="869a4-111">The TDS settlement process is run for all periods that are set up for the TDS settlement period on the **Periods** tab of the **Withholding tax settlement periods** page (**Tax \> Indirect taxes \> Withholding tax \> Withholding tax settlement periods**).</span></span>

6. <span data-ttu-id="869a4-112">В поле **Начальная дата** выберите дату начала выполнения процесса сопоставления TDS.</span><span class="sxs-lookup"><span data-stu-id="869a4-112">In the **From date** field, select the start date to run the TDS settlement process from.</span></span>

    <span data-ttu-id="869a4-113">Для определенного периода в периоде сопоставления дата начала, определенная для периода, берется в качестве начальной даты.</span><span class="sxs-lookup"><span data-stu-id="869a4-113">For a specific period within the settlement period, the start date that is defined for the period is taken as the "from" date.</span></span> <span data-ttu-id="869a4-114">Например, период сопоставления TDS имеет два периода: с 1 апреля по 30 апреля 2009 г. и с 1 мая по 31 мая 2009 г.</span><span class="sxs-lookup"><span data-stu-id="869a4-114">For example, the TDS settlement period has two periods: April 1 through April 30, 2009, and May 1 through May 31, 2009.</span></span> <span data-ttu-id="869a4-115">Если выбрать **06.04.2009** (6 апреля, 2009 г.) в качестве даты начала в поле **Начальная дата**, процесс сопоставления по-прежнему будет выполняться с 1 апреля 2009 г.</span><span class="sxs-lookup"><span data-stu-id="869a4-115">If you select **04/06/2009** (April 6, 2009) as the start date in the **From date** field, the settlement process will still run from April 1, 2009.</span></span>

    <span data-ttu-id="869a4-116">При вводе последующего периода в поле **Дата начала**, но без сопоставления предыдущего периода в пределах периода сопоставления, сопоставление для каких-либо предыдущих периодов выполняться не будет.</span><span class="sxs-lookup"><span data-stu-id="869a4-116">If you enter a later period in the **From date** field, but without settling a previous period within the settlement period, the settlement won't occur for any previous periods.</span></span> <span data-ttu-id="869a4-117">Например, период сопоставления TDS имеет три периода: с 1 апреля по 30 апреля 2009 г., с 1 мая по 31 мая 2009 г. и с 1 июня по 30 июня 2009 г.</span><span class="sxs-lookup"><span data-stu-id="869a4-117">For example, the TDS settlement period has three periods: April 1 through April 30, 2009, May 1 through May 31, 2009, and June 1 through June 30, 2009.</span></span> <span data-ttu-id="869a4-118">Если выбрано **01.05.2009** (1 мая 2009 г.) в качестве даты начала в поле **Начальная дата**, процесс сопоставления будет запускаться только с 1 мая по 31 мая 2009 г.</span><span class="sxs-lookup"><span data-stu-id="869a4-118">If you select **05/01/2009** (May 1, 2009) as the start date in the **From date** field, the settlement process will run only from May 1 through May 31, 2009.</span></span> <span data-ttu-id="869a4-119">Сопоставление не выполняется для "с 1 апреля по 30 апреля, 2009 г.".</span><span class="sxs-lookup"><span data-stu-id="869a4-119">The settlement won't occur for April 1 through April 30, 2009.</span></span>

7. <span data-ttu-id="869a4-120">В поле **Дата проводки** выберите дату для разноски проводки сопоставления TDS.</span><span class="sxs-lookup"><span data-stu-id="869a4-120">In the **Transaction date** field, select the date to post the TDS settlement transaction.</span></span>
8. <span data-ttu-id="869a4-121">В поле **Версия платежа подоходного налога** выберите версию сопоставления TDS:</span><span class="sxs-lookup"><span data-stu-id="869a4-121">In the **Withholding tax payment version** field, select the TDS settlement version:</span></span>

     - <span data-ttu-id="869a4-122">**Исходный** — этот параметр используется для первого выполнения процесса сопоставления TDS.</span><span class="sxs-lookup"><span data-stu-id="869a4-122">**Original** – Use this option to run the TDS settlement process for the first time.</span></span> <span data-ttu-id="869a4-123">Исходная версия платежа используется только один раз для выполнения процесса сопоставления TDS для комбинации TAN, группы компонентов подоходного налога и периода сопоставления подоходного налога.</span><span class="sxs-lookup"><span data-stu-id="869a4-123">The original payment version is used only one time to run the TDS settlement process for a combination of a TAN, a withholding tax component group, and a withholding tax settlement period.</span></span>
    - <span data-ttu-id="869a4-124">**Последние версии** — этот параметр используется, если процесс сопоставления TDS уже выполнялся для указанного периода.</span><span class="sxs-lookup"><span data-stu-id="869a4-124">**Latest versions** – Use this option if the TDS settlement process has already been run for the specified period.</span></span> <span data-ttu-id="869a4-125">Включите проводки с прошлым числом, которые были разнесены после запуска процесса сопоставления ранее для этого периода.</span><span class="sxs-lookup"><span data-stu-id="869a4-125">Include back-dated transactions that were posted after the settlement process was previously run for the period.</span></span> <span data-ttu-id="869a4-126">Этот параметр можно использовать, чтобы выполнить любое количество процессов сопоставления.</span><span class="sxs-lookup"><span data-stu-id="869a4-126">You can use this option to run the settlement process any number of times.</span></span>

9. <span data-ttu-id="869a4-127">Установите флажок **Обновить**, чтобы выполнить процесс сопоставления TDS, и выполните разноску сумм на счета ГК.</span><span class="sxs-lookup"><span data-stu-id="869a4-127">Select the **Update** check box to run the TDS settlement process and post the amounts to the ledger accounts.</span></span> <span data-ttu-id="869a4-128">Если этот флажок снят, процесс сопоставления не будет выполняться, и финансовые записи создаваться не будут.</span><span class="sxs-lookup"><span data-stu-id="869a4-128">If this check box is cleared, the settlement process won't be run, and the financial entries won't be generated.</span></span>
10. <span data-ttu-id="869a4-129">Нажмите **ОК**, чтобы выполнить процесс сопоставления TDS и создать отчет по платежам подоходного налога.</span><span class="sxs-lookup"><span data-stu-id="869a4-129">Select **OK** to run the TDS settlement process and generate the withholding tax payment report.</span></span> <span data-ttu-id="869a4-130">Статус проводок TDS, включенных в процесс сопоставления, отображается как **Сопоставлено** на странице **Сопоставление** (перейдите **Расчеты с поставщиками \> Платежи \> Журнал платежей поставщикам**, выберите **Строки**, выберите **Функции**, а затем выберите **Сопоставление**).</span><span class="sxs-lookup"><span data-stu-id="869a4-130">The status of TDS transactions that are included in the settlement process is shown as **Settled** on the **Settlement** page (go to **Accounts payable \> Payments \> Vendor payment journal**, select **Lines**, select **Functions**, and then select **Settlement**).</span></span>

### <a name="important-points"></a><span data-ttu-id="869a4-131">Важные моменты</span><span class="sxs-lookup"><span data-stu-id="869a4-131">Important points</span></span>

- <span data-ttu-id="869a4-132">Если в процессе сопоставления не выбрана группа компонентов подоходного налога, сопоставление выполняется для всех групп компонентов подоходного налога для выбранных TAN и периода сопоставления.</span><span class="sxs-lookup"><span data-stu-id="869a4-132">If a withholding tax component group isn't selected during the settlement process, the settlement occurs for all the withholding tax component groups for the selected TAN and settlement period.</span></span> <span data-ttu-id="869a4-133">Отдельная строка создается для каждой группы компонентов подоходного налога на странице **Редактирование открытых проводок**.</span><span class="sxs-lookup"><span data-stu-id="869a4-133">A separate line is created for each withholding tax component group on the **Open transaction editing** page.</span></span>
- <span data-ttu-id="869a4-134">Процесс сопоставления базируется на сути категории налогоплательщика периода сопоставления.</span><span class="sxs-lookup"><span data-stu-id="869a4-134">The settlement process is based on the nature of assessee category for a settlement period.</span></span> <span data-ttu-id="869a4-135">Проводки, в которых суть категории налогоплательщика является **Компания**, сопоставляются и отображаются как одна строка на странице **Редактирование открытых проводок**.</span><span class="sxs-lookup"><span data-stu-id="869a4-135">Transactions where the nature of assessee category is **Company** are settled and are shown as one line on the **Open transaction editing** page.</span></span> <span data-ttu-id="869a4-136">Проводки, в которых суть налогоплательщика отличается от **Компания**, сопоставляются и отображаются как одна строка на странице **Редактирование открытых проводок**.</span><span class="sxs-lookup"><span data-stu-id="869a4-136">Transactions where the nature of assessee is something other than **Company** are settled and are shown as one line on the **Open transaction editing** page.</span></span>
- <span data-ttu-id="869a4-137">Дата выполнения для строк сопоставленных проводок TDS на открытой странице **Редактирование открытых проводок** основана на условиях оплаты, которые определены для поставщика проводок TDS на странице **Поставщики**.</span><span class="sxs-lookup"><span data-stu-id="869a4-137">The due date for the settled TDS transaction lines on the **Open transaction editing** page is based on the terms of payment that are defined for the TDS authority vendor on the **Vendors** page.</span></span> <span data-ttu-id="869a4-138">Если условия оплаты не определены для поставщика проводок TDS, в качестве даты выполнения отображается последний день периода сопоставления.</span><span class="sxs-lookup"><span data-stu-id="869a4-138">If the terms of payments aren't defined for the TDS authority vendor, the last day of the settlement period is shown as the due date.</span></span>
