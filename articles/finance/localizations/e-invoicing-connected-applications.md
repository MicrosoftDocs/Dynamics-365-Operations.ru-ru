# <a name="connected-applications"></a>Подключенные приложения

[!include [banner](../includes/banner.md)]

Подключенные приложения — это экземпляры Microsoft Dynamics 365 Finance и Dynamics 365 Supply Chain Management, которые могут быть доступны через службу Regulatory Configuration Service (RCS). С помощью подключенных приложений можно настроить часть функции глобализации в Finance или Supply Chain Management, чтобы работал сценарий электронного выставления счетов.

При развертывании функции глобализации сведения о настройке, применимые к приложению Finance или Supply Chain Management, могут быть опубликованы непосредственно из RCS в соответствующее подключенное приложение.

Доступность параметров Finance или Supply Chain Management в RCS полезна, если имеется несколько сред приложений, таких как приемочное тестирование пользователем (UAT) и производственные среды, и необходимо обеспечить согласованность настройки путем развертывания одних и тех же параметров в различных средах.

## <a name="create-a-connected-application"></a>Создание подключенного приложения

1. Войдите в учетную запись RCS.
2. В рабочей области **Функция глобализации** в разделе **Среда** выберите плитку **Электронное выставление накладных**.
3. На странице **Настройка среды** на панели действий выберите **Подключенные приложения**.
4. Выберите **Создать** для создания подключенного приложения.
5. В поле **Имя** введите имя подключаемого приложения.
6. В поле **Тип** выберите **Dynamics 365 Finance**.
7. В поле **Приложение** введите URL-адрес подключаемой среды.
8. В поле **Клиент** введите клиент среды.
9. Нажмите **Сохранить**.
10. В области действий выберите **Проверить подключение** для проверки подключения к среде. Затем закройте страницу.

## <a name="link-connected-applications-to-environments"></a>Связывание подключенных приложений со средами

1. На странице **Настройка среды** выберите **Создать**, чтобы назначить подключенное приложение среде.
2. В поле **Подключенное приложение** выберите подключенное приложение.
3. В поле **Среда службы** выберите среду службы.
4. Выберите **Сохранить** и закройте страницу.
