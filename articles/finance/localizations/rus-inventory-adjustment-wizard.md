# <a name="inventory-adjustment-wizard"></a>Мастер корректировки запасов

[!include [banner](../includes/banner.md)]


Мастер **Корректировка запасов** используется для корректировки запасов в наличии или складских проводок. Перед корректировкой запасов в наличии его необходимо закрыть. Дополнительные сведения о закрытии запасов см. в разделе [Закрытие запасов](../../supply-chain/cost-management/inventory-close.md) и [Структура стоимости запасов](rus-inventory-cost-structure.md).

Можно скорректировать сумму номенклатуры на складе в сопоставлении с произвольным счетом ГК, и можно разнести отрицательную корректировку в качестве сторнирования.

В дополнение к корректировке затрат в основной валюте, можно скорректировать затраты во вторичной валюте.

В русской локализации эти функции доступны с помощью мастера **Корректировка запасов**. Мастер позволяет использовать следующие методы корректировки: **Себестоимость номенклатуры**, **Фиксированная себестоимость**, **Сумма**, **Значение**, **Процент** и **Со счета ГК**. Можно скорректировать себестоимость номенклатур, используя накладные расходы. Можно также использовать мастера, чтобы определить сумму, которая должна быть распределена в себестоимость номенклатуры на основе сальдо счета ГК. Можно выбрать набор аналитик для определения сальдо счета.

## <a name="preliminary-setup"></a>Предварительная настройка

Чтобы можно было скорректировать стоимость номенклатуры, используя накладные расходы и мастер **Корректировка запасов**, убедитесь, что выполнены предварительные условия в структуре стоимости запасов (здесь будет ссылка на статью, относящуюся к поставке содержимого 148943).

Кроме того, чтобы можно было выполнить корректировку стоимости номенклатуры во вторичной валюте, убедитесь, что выполнены все необходимые условия в статье «Оценка запасов во вторичной валюте» (здесь будет приведена ссылка на статью, относящуюся к поставке содержимого 148936).

## <a name="adjust-item-cost-by-using-the-inventory-adjustment-wizard"></a><a name="adjust-item-cost-using-inventory-adjustment-wizard"></a>Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов

1. Перейдите **Управление запасами \> Периодические задачи \> Закрытие и коррекция** или **Управление запасами \> Периодические задачи \> Закрытие и коррекция в валюте**.
2. В области действий выберите **Корректировка \> Мастер**.
3. На странице **Добро пожаловать** выберите **Далее**.
4. На странице **Метод коррекции себестоимости** выберите метод корректировки:

    - **В наличии** — коррекция запасов в наличии.
    - **Проводки** — корректировка проводок.

5. Выберите **Далее**. Отображается либо страница **Выбрать запасы в наличии**, либо страница **Корректировать проводки** в зависимости от выбранного метода корректировки.
6.  Укажите критерий выбора, а затем нажмите **ОК**.

    > [!NOTE]
    > Если известно, что при корректировке затрат будут выполнены как положительные, так и отрицательные корректировки, эту процедуру следует выполнить двумя отдельными этапами: положительная корректировка и отрицательная корректировка.

7. На странице **Результат выбора** можно просмотреть номенклатуры в наличии или проводки по номенклатуре, которые выбраны для корректировки. Нажмите кнопку с многоточием (**...**), а затем выберите **Печать** для печати списка строк корректировки.
    
    ![Страница результатов выбора.](media/1%20Selection%20result.png)

8. Выберите **Далее**.
9. На странице **Функции для вычисления корректирующих сумм** выберите метод расчета корректирующих сумм:

    - **Себестоимость номенклатуры**
    - **Фиксированная себестоимость** (Дополнительные сведения см. в разделе [Метод расчета корректировки для фиксированной себестоимости](#method-for-calculating-an-adjustment-to-a-fixed-cost-price).)
    - **Сумма** (Дополнительные сведения см. в разделе [Метод расчета корректировки с использованием суммы](#method-for-calculating-an-adjustment-by-using-an-amount).)
    - **Значение** (Дополнительные сведения см. в разделе [Метод расчета корректировки с использованием значения](#method-for-calculating-an-adjustment-to-a-value).)
    - **Процент** (Дополнительные сведения см. в разделе [Метод расчета корректировки с использованием процента](#method-for-calculating-a-percentage-adjustment).)
    - **Со счета ГК** (Дополнительные сведения см. в разделе [Метод расчета корректирующих сумм со счета ГК](#method-for-calculating-adjustment-amounts-from-a-ledger-account).)
        
     ![Страница функций для расчета суммы корректировки, на которой можно просмотреть номенклатуры в наличии или проводки по номенклатуре](media/2%20Functions%20for%20calculating%20adjustment%20amounts.png)

10. Выберите **Далее**.
11. Настройка параметров для выбранного метода корректировки. Параметры могут различаться в зависимости от выбранного метода.
12. На странице **Результаты распределения** можно просмотреть результаты распределения.

    - В поле **Код накладных расходов** выберите код накладных расходов для строки.

        > [!NOTE]
        > Можно также нажать кнопку с многоточием (**...**), а затем выбрать **Функции \> Код накладных расходов**, чтобы открыть страницу **Изменить код накладного расхода**. Затем в поле **Код накладных расходов** выберите код накладных расходов для строки. Чтобы применить код накладных расходов, которые были введены во все строки, установите для параметра **Применить ко всем** значение **Да**.
        
        ![Страница результатов распределения.](media/3%20Change%20misc.%20charge%20code.png)

    - В поле **Сумма распределения** можно вручную редактировать суммы распределения, за исключением сумм, рассчитанных с использованием метода **Со счета ГК**. При изменении этого поля вручную сумма распределения должна быть отрицательной для всех строк или положительной для всех строк.
    
    ![Страница результатов распределения, поле суммы распределения.](media/4%20Results%20of%20allocation.png)

13. Выберите **Далее**.
14. На странице **Разноска** укажите сведения для разноски корректировки:

    - В поле **Дата корректировки** укажите дату корректировки.
    - Установите для параметра **Сторно** значение **Да**, чтобы разнести ваучеры в ГК как сторно. Этот параметр можно изменить только в том случае, если значение **Сумма распределения** имеет отрицательное значение. Если значение **Сумма распределения** является отрицательным, а параметр **Сторно** имеет значение **Нет**, система разносит ваучер корректировки в качестве сторнирования.
    - В поле **Спецификация** выберите способ разноски корректировок проводок:

      - **Все** — корректировки разносятся для всех номенклатур с одинаковыми параметрами в профиле разноски.
      - **Номенклатурная группа** — разнесение корректировок выполняется для всех номенклатур из такой же номенклатурной группы.

         - **Код номенклатуры** — корректировки разносятся для каждой номенклатуры.

     -   В поле **Примечание** введите примечание к корректировке.
     -   Задайте параметр **Корр. счет Прибыли/Убытки** и поле **Корр. счет** в зависимости от счета, которому должна быть назначена корректировка во время разноски:

         - Если код накладных расходов введен в строке номенклатуры, корректировка назначается корр. счету, настроенному для кода накладных расходов.
         - Если код накладных расходов не введен в строке номенклатуры, а параметр **Корр. счет Прибыли/Убытки** на странице **Разноска** имеет значение **Да**, корректировка назначается стандартным способом для корр. счета прибылей и убытков, который настраивается на вкладке **Запасы** на странице **Разноска** (**Управление запасами \> Настройка \> Разноска \> Разноска**).
         - Если код накладных расходов не введен в строке номенклатуры, а параметр **Корр. счет Прибыли/Убытки** на странице **Разноска** имеет значение **Нет**, корректировка назначается счету, указанному в поле **Корр. счет**.
                
     ![Страница разноски.](media/5%20Posting.png)

15. Выберите **Далее**.
16. На странице **Готово** установите флажок **Показать список ваучеров ГК**, чтобы отобразить список ваучеров ГК, которые были разнесены, а затем выберите **Готово** для разноски корректировки.

Чтобы отменить изменения, выполненные мастером, на странице **Закрытие и коррекция** на панели операций выберите **Отмена**.

### <a name="method-for-calculating-an-adjustment-to-a-fixed-cost-price"></a>Метод расчета корректировки по фиксированной себестоимости

1. Откройте мастер **Корректировка запасов**, выберите метод корректировки и выберите номенклатуры в наличии или проводки по номенклатуре, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).
2. На странице **Функции для вычисления корректирующих сумм** выберите **Фиксированная себестоимость**, затем нажмите **Далее**.
3. На странице **Корректировка по цене постоянных затрат** в поле **Цена** введите значение фиксированной себестоимости, а затем нажмите **Далее**.

    ![Страница корректировки по фиксированной себестоимости.](media/6%20Adjustment%20to%20fixed%20cost%20price.png)

4. Выполните остальные шаги настройки, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).

### <a name="method-for-calculating-an-adjustment-by-using-an-amount"></a>Метод для вычисления корректировки с помощью суммы

1. Откройте мастер **Корректировка запасов**, выберите метод корректировки и выберите номенклатуры в наличии или проводки по номенклатуре, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).
2. На странице **Функции для вычисления корректирующих сумм** выберите **Сумма**, затем нажмите **Далее**.
3. На странице **Коррекция с суммой** задайте следующие поля:

    - В поле **Сумма** введите сумму.
    - В поле **Принцип распределения** выберите **Значение** или **Количество**.

    ![Страница функций для вычисления корректирующих сумм, где вы выбираете "Сумма", затем нажмите "Далее"](media/7%20Adjustment%20with%20amount.png)

4. Выберите **Далее**.
5. Выполните остальные шаги настройки, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).

### <a name="method-for-calculating-an-adjustment-to-a-value"></a>Метод вычисления корректировки с помощью значения

1. Откройте мастер **Корректировка запасов**, выберите метод корректировки и выберите номенклатуры в наличии или проводки по номенклатуре, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).
2. На странице **Функции для вычисления корректирующих сумм** выберите **Значение**, затем нажмите **Далее**.
3. На странице **Корректировка суммы** задайте следующие поля:

    - В поле **Значение** введите значение.
    - В поле **Принцип распределения** выберите **Значение** или **Количество**.

    ![Страница функций для вычисления корректирующих сумм, где вы выбираете "Значение", затем нажмите "Далее".](media/8%20Adjustment%20to%20value.png)

4. Выберите **Далее**.
5.  Выполните оставшиеся шаги корректировки, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-by-using-the-inventory-adjustment-wizard).

### <a name="method-for-calculating-a-percentage-adjustment"></a>Метод для вычисления корректировки с помощью процента

1. Откройте мастер **Корректировка запасов**, выберите метод корректировки и выберите номенклатуры в наличии или проводки по номенклатуре, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).
2. На странице **Функции для вычисления корректирующих сумм** выберите **Процент**, затем нажмите **Далее**.
3. На странице **Коррекция процента** задайте следующие поля:

    - В поле **Процент** введите процент НДС.
    - В поле **Корректировка** выберите **Положительные запасы в наличии** или **Всего запасов в наличии**.

    ![Страница функций для вычисления корректирующих сумм, где вы выбираете "Процент", затем нажмите "Далее".](media/9%20Percentage%20adjustment.png)

4. Выберите **Далее**.
5. Выполните остальные шаги настройки, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).

### <a name="method-for-calculating-adjustment-amounts-from-a-ledger-account"></a>Метод расчета корректирующих сумм со счета ГК

Этот метод позволяет определить сумму корректировки на основе сальдо счета ГК на конкретную дату. Суммы корректировки могут быть всеми сальдо и значениями выбранных комбинаций финансовых аналитик. Этот метод может использоваться, например, для закрытия каких-либо счетов, таких как затраты на закупку или расходы на обслуживание склада, по себестоимости номенклатуры для складских затрат.

1. Откройте мастер **Корректировка запасов**, выберите метод корректировки и выберите номенклатуры в наличии или проводки по номенклатуре, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).
2. На странице **Функции для вычисления корректирующих сумм** выберите **Со счета ГК**, затем нажмите **Далее**.
3. На странице **Сальдо со счета ГК или аналитик** задайте следующие поля:

    - В поле **Счет ГК** выберите счет ГК, на котором определяется сальдо.
    - В поле **Сальдо на дату** укажите дату, в которую определяется сальдо.
    - Выберите параметр **Сальдо по счету**, чтобы определить сальдо по счету, или выберите параметр **Сальдо со счета ГК и аналитик**, чтобы определить сальдо по счету с помощью определенных значений аналитик.
    - В поле **Набор аналитик** выберите набор аналитик, по которому будет определяться сальдо счета. Это поле доступно, только когда выбран параметр **Сальдо со счета ГК и аналитик**.

    ![Страница сальдо со счета ГК или аналитик.](media/10%20Balance%20from%20ledger%20account%20or%20dimensions.png)

4. Выберите **Далее**.
5. Выбор сумм для включения в корректировку. Можно использовать ссылки **Выбрать все** и **Отменить выбор всех**, чтобы выбрать и отменить выбор всех строк.
6. В поле **Принцип распределения** выберите **Значение** или **Количество**.

    ![Страница сальдо со счета ГК или аналитик, поле принципа распределения.](media/11%20Balance%20from%20ledger%20account%20or%20dimensions.png)

7. Выберите **Далее**.
8. Выполните остальные шаги настройки, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard).

Этот метод определяет сальдо счета в контексте финансовых аналитик только для определения суммы корректировки. Оно не закрывает автоматически сальдо по аналитикам в себестоимости номенклатуры на складе.

#### <a name="example"></a>Пример

На счете 26.100 аналитика **Подразделение** накапливает суммы, которые необходимо распределить по затратам на склад, которые учтены в конкретных складах. Чтобы выполнить этот процесс в системе, выполните следующие действия.

1. Закройте складские запасы до окончания периода закрытия.
2. Откройте мастер **Корректировка запасов**, выберите метод корректировки **Наличие**, а затем нажмите **Далее**.
3. На странице **Выбрать запасы в наличии** настройте следующие поля:

    - На экспресс- вкладке **Параметры** задайте для параметра **Склад** значение **Да**.
    - На экспресс-вкладке **Записи для добавления** выберите **Фильтр** и укажите фильтр для конкретного склада или складов.

4. Выберите **Далее**.
5. На странице **Функции для вычисления корректирующих сумм** выберите **Со счета ГК**, затем нажмите **Далее**.
6. На странице **Сальдо со счета ГК или аналитик** задайте следующие поля:

    - В поле **Счет ГК** выберите счет ГК **26.100**.
    - В поле **Сальдо на дату** укажите дату, в которую определяется сальдо.
    - Выберите параметр **Сальдо со счета ГК и аналитик**, а затем в поле **Набор аналитик** выберите набор аналитик.

7. Выберите **Далее**.
8. На следующей странице выберите строки для значений аналитики **Подразделение**, из которых сальдо должны быть назначены запасам выбранного склада или складов, а затем нажмите **Далее**.
9. Установите для параметра **Корр. счет Прибыли/Убытки** значение **Да** и задайте поле **Корр. счет**.
10. Выполните остальные шаги настройки, как описано в разделе [Скорректируйте стоимость номенклатуры с помощью мастера корректировки запасов](#adjust-item-cost-using-inventory-adjustment-wizard). Повторите операцию распределения для всех складов и аналитик, для которых необходимо выполнить распределение.

Как вы видите, система позволяет определить сумму корректировки на основе сальдо счета в контексте аналитик. Ее можно также распределить по себестоимости запасов или складских приемок.
