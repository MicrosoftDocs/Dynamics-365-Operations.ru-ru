# <a name="set-up-and-create-agreements"></a>Настройка и создание договоров
[!include [banner](../includes/banner.md)]

Все договоры покупки можно просмотреть на странице списка **Договоры покупки** и все договоры продажи на странице списка **Договоры продажи**. Эти страницы списков хранят данные, такие как даты и суммы договоров, и условия оплаты. Все договоры группируются в классификации договоров.

## <a name="create-sales-agreement-classifications"></a>Создание классификаций договоров продажи

1. Перейдите **Продажи и маркетинг** \> **Настройка** \> **Договоры продажи** \> **Классификации договоров продажи** или **Расчеты с клиентами** \> **Настройка** \> **Классификации договоров продажи**.
2. Создайте классификацию договоров продажи.
3. На экспресс-вкладке **Общие** в поле **Связанные классификации** выберите группу классификации договоров, связанную с классификацией договоров.
4. В поле **Код номерной серии** выберите код номерной серии, который должен использоваться для нумерации договоров, относящихся к классификации договоров.
5.  Нажмите **Сохранить**.

![Страница классификации договоров продажи.](media/1_Sales_agreement_classifications.png)

> [!NOTE]
> Нельзя удалить классификацию договоров, используемую для существующих договоров.

## <a name="create-purchase-agreement-classifications"></a>Создание классификаций договоров покупки

1. Перейдите в раздел **Закупки и источники** \> **Настройка** \> **Классификация договоров покупки**.
2. Укажите сведения, как описано в предыдущем разделе [Создание классификаций договоров продажи](#create-sales-agreement-classifications) данной статьи.

![Страница классификаций договоров покупки.](media/2_Purchase_agreement_classifications.png)

## <a name="create-a-sales-agreement"></a><a name="create-sales-agreement"></a>Создание договора продажи

Дополнительные сведения о создании договоров продажи см. в разделе [Ввод договоров продажи](../../supply-chain/sales-marketing/tasks/enter-sales-agreements.md).

1. Перейдите **Расчеты с клиентами** \> **Заказы** \> **Договоры продажи** или **Продажи и маркетинг** \> **Договоры продажи** \> **Договоры продажи**.
2. На панели операций выберите **Создать** для создания договора продажи.
3. В диалоговом окне **Создание договора продажи** на экспресс-вкладке **Клиент** укажите следующие сведения:

    -  Выберите счет клиента в поле **Счет клиента**.
    -  В разделе **Документ** в поле **Классификация договоров продажи** выберите классификацию договоров продажи, к которой относится данный договор.

    На экспресс-вкладке **Общие** в разделе **Документ** поле **Код договора продажи** заполнено на основе номерной серии, указанной для классификации соглашения.

4. Нажмите **ОК**.

    ![Диалоговое окно создания классификаций договоров продажи.](media/3_Create_sales_agreement.png)

5. На странице **Договоры продажи** на экспресс-вкладке **Заголовок договора продажи** укажите следующие сведения:

    -  В поле **Валюта** валюту договора.
    -  В поле **Действует с** укажите дату вступления в силу договора.
    -  В поле **Дата окончания срока действия** укажите дату окончания срока действия договора.
    -  В поле **Статус** выберите статус соглашения: **на удержании**, **действует** или **закрыто**.

    > [!NOTE]
    > Проводки могут быть разнесены только для договоров со статусом **действует**.

    ![Экспресс-вкладка заголовка договора продажи.](media/4_Sales_agreements.png)

6. Переключитесь в представление **Заголовок**, а затем на экспресс-вкладке **Общие** выполните следующие шаги:

  - В разделе **Документ** укажите следующие сведения:

     - В поле **Заголовок документа** укажите заголовок договора.
     - В поле **Дата** укажите дату договора.

  -   В разделе **Предмет договора** в поле **Предмет договора** введите комментарий или примечания о договоре.

   ![Экспресс-вкладка "Общее" договора продажи.](media/5_Sales_agreements.png)

7. На экспресс-вкладке **Финансы** выполните следующие действия.

    - В разделе **Сумма договора** в поле **Сумма договора** укажите сумму договора.
    - В разделе **Кредит** в поле **Кредитный лимит** укажите кредитный лимит для договора. Кредитный лимит комплексно проверяется для всех договоров контрагентов и для общего лимита контрагентов.


    > [!NOTE]
    > При сопоставлении накладных с платежами, относящимися к разным договорам, контроль лимита выполняется в соответствии с договором, которому назначен платеж.
    >
    > При отмене сопоставления накладных и платежей, относящихся к разным договорам, контроль лимита выполняется в соответствии с договором, которому назначена накладная.
    >
    > При сопоставлении накладных с платежами, которые не показывают, что они принадлежат договору, контроль лимита выполняется контрагентом.
    >
    > При сопоставлении накладных без указания, что они относятся к договору с платежами, относящимися к договору, контроль лимита выполняется в соответствии с договором, которому назначен платеж.
    >
    > При отмене сопоставления накладных без указания, что они относятся к договору с платежами, и платежей, относящихся к договору, контроль лимита выполняется контрагентом. Накладные с открытым сальдо и без указания на договор включаются.

   - В разделе **Профиль разноски** укажите следующие сведения:

    - Если в **Профиль разноски** указан договор для проводок, можно выбрать профиль разноски контрагента, который должен использоваться при разноске проводок.
     - В поле **Профиль разноски с ваучером журнала предоплат** можно выбрать отдельный профиль разноски, который должен использоваться при регистрации предоплаты для договора. Если это поле не задано, значение по умолчанию берется со страницы **Параметры модуля расчетов с клиентами**.

   ![Страница договоров на продажу.](media/6_Sales_agreements.png)

8. На экспресс- вкладке **Условия** в разделе **Оплата** задайте поля для условий оплаты. В поле **Платежный день** выберите календарь платежных дней.
9. На экспресс-вкладке **Контактная информация** укажите контактную информацию для договора.

    ![Страница договоров продажи, экспресс-вкладка контактной информации.](media/7_Sales_agreements.png)

10. На экспресс-вкладке **Финансовые аналитики** укажите финансовые аналитики. При указании договора в документе финансовые аналитики договора вводятся в аналитику документа.
11. Нажмите **Сохранить**.

### <a name="set-up-a-commission"></a>Настройка комиссии

1. На странице **Договоры продажи** выберите соглашение, для которого необходимо настроить комиссию.
2.  На вкладке **Договор продажи** в разделе **Настройка** выберите **Расчет комиссии**.
3.  На странице **Расчет комиссии** настройте комиссию для выбранного договора. Дополнительные сведения о настройке комиссий см. в [Регистрация комиссий продаж](../../supply-chain/sales-marketing/tasks/register-sales-commissions.md).

![Страница расчета комиссии.](media/8_Commission_calculation.png)

## <a name="create-a-purchase-agreement"></a>Создание договора покупки

Для получения дополнительной информации о том, как создать договоры покупки, см. [Создание договора покупки](../../supply-chain/procurement/tasks/create-purchase-agreement.md).

1. Перейдите **Расчеты с поставщиками** \> **Заказы на покупку** \> **Договоры покупки** или **Закупки и источники** \> **Договоры покупки** \> **Договоры покупки**.
2. На панели операций выберите **Создать** для создания договора.
3. В диалоговом окне **Создание договора покупки** на экспресс-вкладке **Поставщик** укажите следующие сведения:

    - В поле **Счет поставщика** выберите счет поставщика.
    - В разделе **Документ** в поле **Классификация договоров покупки** выберите классификацию договоров покупки, к которой относится данный договор.

    На экспресс-вкладке **Общие** в разделе **Документ** поле **Код договора покупки** заполнено на основе номерной серии, указанной для классификации соглашения.

4. Нажмите **ОК**.

    ![Диалоговое окно создания классификаций договоров покупки.](media/9_Create_purchase_agreement.png)

5. На странице **Договоры покупки** укажите сведения, как описано в разделе [Создание договора продажи](#create-sales-agreement) ранее в этой статье.

## <a name="set-up-agreement-dimension-control-for-settlements"></a>Настройка контроля аналитик договоров для сопоставлений

Можно настроить контроль аналитик для сопоставлений. Дополнительные сведения см. в разделе [Настройка контроля аналитик для сопоставлений](rus-transactions-settlement-date.md) Для сопоставления в контексте договоров выполните следующие действия.

1.  Перейдите в раздел **Главная книга** \> **Настройка главной книги** \> **Параметры главной книги**.
2.  На вкладке **Договоры** установите для параметра **Отключить контроль аналитики договоров** значение **Нет**.
