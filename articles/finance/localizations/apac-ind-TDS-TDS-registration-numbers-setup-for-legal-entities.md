# <a name="set-up-tds-registration-numbers-for-legal-entities"></a>Настройка регистрационных номеров TDS для юридических лиц

[!include [banner](../includes/banner.md)]

В этой статье объясняется, как настроить регистрационные номера для юридических лиц для налогов, которые удерживаются у источника (TDS).

1. Перейдите в раздел **Управление организацией \> Организации \> Юридические лица**.

    [![Страница "Юридические лица".](./media/apac-ind-TDS-4.png)](./media/apac-ind-TDS-4.png)

2. На экспресс-вкладке **Сведения о налоге** в поле **Постоянный номер счета** введите постоянный номер счета (PAN) юридического лица.
3. В поле **Номер круга** введите номер круга органа TDS.
4. В поле **Номер района** введите номер района органа TDS.
5. В поле **Номер налогового оценщика** введите сведения о налоговом оценщике для органа TDS.
6. В поле **Тип удержания** выберите категорию типа удержания для юридического лица.
7. В области действий выберите **Регистрационные номера**, чтобы открыть страницу **Управление адресами**.
8. На экспресс-вкладке **Сведения о налоге** выберите **Добавить** или **Изменить**, чтобы открыть страницу **Управление сведениями о налогах**, где можно вести запись регистрации налога.

    [![Страница "Управление адресами".](./media/apac-ind-TDS-5.png)](./media/apac-ind-TDS-5.png)

9. На экспресс-вкладке **Подоходный налог** в поле **Номер налогового счета (TAN)** введите регистрационный номер. Этот номер должен состоять из четырех букв, затем пяти цифр, а затем одной буквы. Пример: **AXDF87645F**.
10. В поле **Имя или описание** введите описание регистрационного номера подоходного налога.

    [![Страница "Управление сведениями о налогах".](./media/apac-ind-TDS-5-1.png)](./media/apac-ind-TDS-5-1.png)

11. Закройте страницу.
