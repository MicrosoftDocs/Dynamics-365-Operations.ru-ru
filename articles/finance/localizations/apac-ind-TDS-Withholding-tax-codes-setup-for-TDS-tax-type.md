# <a name="set-up-withholding-tax-codes-for-the-tds-tax-type"></a>Настройка налоговых кодов по подоходному налогу для налогов типа TDS

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как настроить налоговые коды для налога, который удерживается в источнике (TDS).

1. Перейдите в раздел **Налог \> Косвенные налоги \> Подоходный налог \> Коды подоходного налога**.

    [![Страница "Коды подоходного налога".](./media/apac-ind-TDS-17.png)](./media/apac-ind-TDS-17.png)

2. В области действий выберите **Создать**, чтобы создать код подоходного налога для TDS, и введите требуемые сведения.
3. На экспресс-вкладке **Общие** в поле **Тип налога** выберите **TDS** для классификации налогового кода как налогового кода TDS.
4. В поле **Период сопоставления** выберите период сопоставления TDS для налогового кода TDS.
5. В поле **Счет ГК** выберите счет учета, на который разносится сумма TDS.
6. В поле **Счет дебиторской задолженности** выберите счет, на который должны быть разнесены суммы TDS, вычитаемые в проводках по продаже.

    Поле **Исходный** автоматически устанавливается как **Процент от валовой суммы**, и значение не может быть изменено.

    > [!NOTE]
    > Невозможно установить источник как **Процент от чистой суммы** для налоговых кодов типа TDS.

7. В поле **Компонент подоходного налога** выберите компонент налога TDS для налогового кода TDS.
8. Выберите **Значения** на панели операций, чтобы открыть страницу **Значения подоходного налога**.
9. В поле **Начальная дата** введите дату начала для значения TDS. В поле **Конечная дата** введите дату окончания.

    > [!NOTE]
    > Поля **Минимальный лимит**, **Верхний лимит** и **Процент исключения** недоступны для налоговых кодов типа налога TDS.

10. В поле **Значение** введите значение процента TDS для кода налога TDS.
11. Закройте страницу **Значения подоходного налога**, чтобы вернуться на страницу **Коды подоходного налога**.

    > [!NOTE]
    > Кнопка **Лимиты** на странице **Коды подоходного налога** недоступна для налоговых кодов типа налога TDS.

12. Закройте страницу.
