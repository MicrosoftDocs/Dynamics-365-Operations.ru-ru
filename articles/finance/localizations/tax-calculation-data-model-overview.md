# <a name="tax-calculation-data-model"></a>Модель данных для расчета налога

В этой теме приводятся сведения о том, как значения полей в модели данных налога определяются в контексте расчета налога.

*Модель налоговых данных* состоит из полей, которые необходимы для расчета налогов. Компания Microsoft предоставляет *модель данных для расчета налога*, которая включает поля заголовка и поля строки документов проводок в приложениях Finance and Operations. Поля, определенные в модели данных расчета налога, являются доступными столбцами в таблицах правила применимости в конфигурации функции расчета налогов.

> [!NOTE] 
> Некоторые узлы, определенные в модели данных, такие как **Код записи** и **Код таблицы**, предназначены для технических целей. Они не являются доступными столбцами в конфигурации функции расчета налога.

Выполните следующие действия, чтобы просмотреть модель данных для расчета налога.

1. Войдите в Regulatory Configuration Service (RCS) и перейдите в **Электронная отчетность** \> **Конфигурации налога**.
2. Выберите **Модель налоговых данных** \> **Модель данных для расчета налога**.
3. На экспресс-вкладке **Версии** выберите версию.
4. Выберите **Конструктор**.

    [![Кнопка Конструктор.](./media/tax-calculation-model-mapping-1.png)](./media/tax-calculation-model-mapping-1.png)

5. Разверните узел **Налоговая служба**.
6. Разверните узел **Заголовок** для просмотра моделей данных заголовка.
7. Разверните узел **Строки** для просмотра моделей данных строк.

    [![Модели данных.](./media/tax-calculation-model-mapping-2.png)](./media/tax-calculation-model-mapping-2.png)

Можно также просмотреть модель данных для расчета налога и доступные столбцы в таблице правил применимости.

1. В RCS перейдите в **Функции глобализации** \> **Расчета налога**.
2. Найдите функцию налога, имеющую статус **Черновик**, а затем выберите **Правка**.
3. На вкладке **Общие** выберите версию конфигурации на основе версии модели данных для расчета налога. Например, выберите **40.46.212**.
4. На вкладке **Применимость налоговой группы** выберите **Управление столбцами**.

    [![Управление столбцами.](./media/tax-calculation-model-mapping-3.png)](./media/tax-calculation-model-mapping-3.png)

Модель данных для расчета налога интегрируется с приложениями Finance and Operations. Во время каждой проводки, которая относится к расчету налога, значения полей, которые определены в таблицах правил применимости, собираются и отправляются в службу расчета налогов для вычисления. В версии 40.46 доступно 57 предопределенных моделей данных для следующих типов проводок: заказы на покупку, заказы на продажу, заказы на перемещение, заявки на покупку, запросы предложения с расценками и предложения по продажам.
