# <a name="goods-in-transit-from-vendor-russia"></a>Товары в пути от поставщика (Россия)

[!include [banner](../includes/banner.md)]

Процесс инвентаризации номенклатуры (ИНВ-6) используется для определения количества и стоимости складских номенклатур, которые находятся в транзите при инвентаризации запасов.

Процесс инвентаризации номенклатур выполняется в контексте номенклатур, отгрузочных документов и поставщиков.

## <a name="set-up-an-inventory-profile-for-a-transferable-item"></a>Настройка профиля учета для переносимой номенклатуры
Чтобы отразить товары и материалы в транзите, можно настроить вид запасов для вида деятельности **Основной**. Поле **Вид запасов** может иметь следующие значения:

- Общий
- Купленные товары в пути

1. Перейдите в раздел **Управление запасами** \> **Настройка** \> **Аналитики** \> **Профили учета**.
2. Выберите **Создать**, чтобы создать профиль учета для номенклатуры.
3. В поле **Профиль учета** введите имя профиля учета.
4. В поле **Имя** введите описание профиля учета.
5. На экспресс-вкладке **Настройка** в поле **Вид деятельности** выберите **Основной**.
6. Установите для параметра **Не подбирать** значение **Да**, чтобы запретить автоматическое сопоставление запасов в наличии из данного профиля учета с совместимым профилем учета.
7. В поле **Вид запасов** выберите **Купленные товары в пути**. Для параметров **Инициализировать аналитику** и **Контролировать аналитику в заказе на покупку** автоматически устанавливается значение **Да**.

> [!NOTE]
> Поле **Вид запасов** доступно только в том случае, если выбрано значение **Основной** в поле **Вид деятельности**.

8. На экспресс-вкладке **Приоритет подбора** выберите **Вверх** или **Вниз**, чтобы изменить порядок профиля запасов.
9. Выберите **Сохранить** и закройте страницу.

  ![Страница профилей учета.](media/goods-in-transit-vendor-01.png)
 
## <a name="set-up-a-number-sequence-for-the-counting-act-inv-6-report"></a>Настройка номерной серии для отчета по акту инвентаризации (ИНВ-6)

1. Перейдите в раздел **Управление запасами** \> **Настройка** \> **Параметры управления запасами и складами**.
2. На вкладке **Номерные серии** в поле **Код номерной серии** выберите номерную серию для ссылки **Акт инвентаризации (ИНВ-6)**.
3. Выберите **Сохранить** и закройте страницу.

## <a name="set-up-an-inventory-journal-name-for-the-counting-act-inv-6-report"></a>Настройка имени журнала инвентаризации запасов для отчета об акте инвентаризации (INV-6)

1. Перейдите в раздел **Управление запасами** \> **Настройка** \> **Имена журналов** \> **Запасы**.
2. Создайте имя журнала.
3. В поле **Имя** введите имя журнала запасов.
4. В поле **Описание** введите описание.
5. В поле **Тип журнала** выберите **Инвентаризация**.
6. На экспресс-вкладке **Отчеты** в поле **Доступно** выберите **Акт инвентаризации (ИНВ-6)**, затем выберите кнопку со стрелкой влево, чтобы переместить отчет в поле **Выбрано**.

> [!NOTE]
> Если поле **Доступно** пусто, выберите **Обновить** на панели операций, чтобы обновить список доступных отчетов.

7. Выберите **Сохранить** и закройте страницу.

  ![Страница имен журналов запасов.](media/goods-in-transit-vendor-2.png)
 
## <a name="add-officials-to-the-counting-act-inv-6-report"></a>Добавление должностных лиц для отчета "Акт инвентаризации ИНВ-6"
Можно указать должностных лиц компании, участвующих в процессе инвентаризации номенклатур, для отчета **Акт инвентаризации (ИНВ-6)**.

1. Откройте **Управление организацией** \> **Настройка** \> **Контакты** \> **Должностные лица**.
2. На вкладке **Запасы** выберите параметр **Акт инвентаризации (ИНВ-6)**.
3. Выберите **Создать**, чтобы создать должностное лицо для отчета **Акт инвентаризации (ИНВ-6)**.
4. В поле **Должность** выберите назначение должностного лица.
5. В поле **Имя** введите имя должностного лица.
6. В поле **Название должности** выберите название должности должностного лица.

> [!NOTE]
> По умолчанию название должности копируется из записи выбранного сотрудника.
 
  ![Страница должностных лиц.](media/goods-in-transit-vendor-03.png)
 
7. Выберите **Сохранить** и закройте страницу.

### <a name="create-an-inventory-tracking-dimension-group-for-the-counting-act-inv-6-report"></a>Создание группы аналитик отслеживания запасов для отчета по акту инвентаризации (ИНВ-6)

1. Откройте **Управление сведениями о продукте** \> **Настройка** \> **Группы аналитик и вариантов** \> **Группы аналитик отслеживания**.
2. Выберите **Создать**, чтобы создать группу аналитик.
3. В поле **Имя** введите полное имя группы аналитик.
4. В поле **Описание** введите описание.
5. На экспресс-вкладке **Аналитики отслеживания** в строке **Профиль учета** установите флажки **Активный**, **Первичная аналитика хранения**, **Физические запасы**, **Финансовые запасы**, **План покрытия по аналитикам** и **Перемещение**.
6. В строке **Ответственный** установите флажки **Активный**, **Физические запасы**, **Финансовые запасы** и **Перемещение**.

> [!NOTE]
> Чтобы в отчете отображались детали отгрузки и платежных документов, в строке **Номер партии** установите флажки **Активный** и **Физические запасы**.

  ![Страница групп аналитик отслеживания.](media/goods-in-transit-vendor-04.png)
 
7. Нажмите **Сохранить**.

## <a name="create-a-purchase-agreement"></a>Создание договора покупки

1. Перейдите в раздел **Расчеты с поставщиками** \> **Заказы на покупку** \> **Договоры покупки**.
2. Выберите **Создать**, чтобы открыть диалоговое окно **Создание договора покупки**.
3. На экспресс-вкладке **Поставщик** в поле **Счет поставщика** выберите счет поставщика.
4. В поле **Классификация договоров покупки** выберите **Общий договор покупки**.
5. На экспресс-вкладке **Общие сведения** в разделе **Документ** в поле **Договор покупки** введите код договора покупки.
6. Укажите другие сведения, затем выберите **ОК**.
7. На странице **Договоры покупки** переключитесь в представление **Заголовок**, затем на экспресс-вкладке **Финансы** в разделе **Профиль учета** настройте следующие поля:

- В поле **Вид деятельности** выберите **Основной**.
- В поле **Профиль учета** выберите профиль учета, созданный ранее.
 
  ![Страница договоров покупки.](media/goods-in-transit-vendor-05.png)
 
8. На панели операций на вкладке **Договор покупки** в группе **Создать** выберите **Подтверждение** для обновления статуса договора покупки до **Действует**.

## <a name="add-an-inventory-owner-to-the-counting-act-inv-6-report"></a>Добавление владельца запасов в отчет по акту инвентаризации (ИНВ-6)

1. Перейдите в раздел **Управление запасами** \> **Настройка** \> **Аналитики** \> **Владельцы запасов**.
2. Выберите **Создать**, чтобы добавить владельца запасов.
3. В поле **Владелец** введите код владельца.
4. В поле **Тип счета** выберите **Поставщик**.
5. В поле **Счет** выберите код принципала. Поле **Имя** заполняется автоматически.
6. В поле **Код договора** выберите договор покупки, созданный ранее. Таким образом, новый владелец связывается с договором.

  ![Страница владельцев запасов.](media/goods-in-transit-vendor-06.png)

7. Нажмите **Сохранить**.

## <a name="register-goods-and-materials-that-are-in-transit"></a>Регистрация товаров и материалов в пути

1. Создание нового заказа на покупку. В поле **Договор покупки** выберите ранее созданный договор покупки, затем в строке заказа на покупку в поле **Код номенклатуры** выберите код номенклатуры.

> [!NOTE]
> В поле **Аналитика отслеживания** для данной номенклатуры должен быть указан профиль учета, созданный ранее.

2. На экспресс-вкладке **Сведения по строке** на вкладке **Продукт** в разделе **Аналитика отслеживания** проверьте, что в поле **Профиль учета** выбран профиль учета, созданный ранее.
3. В поле **Владелец** выберите запись, созданную ранее.

  ![Страница заказа на покупку.](media/goods-in-transit-vendor-07.png)

4. Выполните разноску накладной обычным способом.

## <a name="generate-a-counting-act-inv-6-report"></a>Создание отчета по акту инвентаризации (ИНВ-6)
Можно использовать диалоговое окно **Печать акта инвентаризации (ИНВ-6)** для создания отчета **Акт инвентаризации (ИНВ-6)** в виде файла Microsoft Excel. Этот отчет необходимо создать для отслеживания номенклатур, перемещаемых между складами, и для создания инвентарного списка приобретенных номенклатур, находящихся в состоянии перемещения. Инвентаризационная опись может содержать запасы в наличии.

1. Перейдите в раздел **Управление запасами** \> **Записи журнала** \> **Инвентаризация номенклатуры** \> **Инвентаризация**.
2. Выберите **Создать**, чтобы открыть диалоговое окно **Отображение аналитик**.
3. На экспресс-вкладке **Обзор** убедитесь, что в поле **Имя** указано имя журнала запасов, который был создан ранее.
4. В разделе **Запасы магазина** задайте поля **Место** и **Склад**.
5. На экспресс-вкладке **Подсчет по** убедитесь, что для параметров **Профиль учета** и **Владелец** задано значение **Да**.

> [!NOTE]
> Можно также установить для параметров **Номер партии** и **Склад** значение **Да**, если эти аналитики применимы.
 
  ![Диалоговое окно отображения аналитик.](media/goods-in-transit-vendor-08.png)
 
6. Выберите **ОК**, чтобы создать журнал инвентаризации запасов для перемещаемых номенклатур.
7. На странице **Инвентаризация** на экспресс-вкладке **Строки журнала** создайте строку и выберите код номенклатуры, созданный ранее.
8. На экспресс-вкладке **Сведения по строке** на вкладке **Складские аналитики** в полях **Профиль учета** и **Владелец** выберите записи, которые были созданы ранее. Поле **В наличии** в строке журнала обновляется автоматически.
9. На экспресс-вкладке **Строки журнала** выберите **Функции** \> **Создать Инвентаризационную опись**, чтобы создать инвентаризационную опись. Для каждой строки поле **Фактический остаток** обновляется значением, которое указано в поле **В наличии**.

  ![Страница инвентаризации.](media/goods-in-transit-vendor-09.png)

10. На панели операций выберите **Печать** \> **Акт инвентаризации (ИНВ-6)**, чтобы открыть диалоговое окно **Печать акта инвентаризации (ИНВ-6)**.
11. В поле **Дата составления акта** выберите дату, когда процесс инвентаризации запланирован для выполнения.
12. В полях **Дата начала** и **Дата окончания**, выберите начальную и конечную даты периода инвентаризации запасов.
13. В поле **Номер заказа** введите номер заказа, для которого создается отчет **Акт инвентаризации ИНВ-6**.
14. В поле **Дата приказа** выберите дату проводки заказа, для которого создается отчет **Акт инвентаризации ИНВ-6**.

> [!NOTE]
> Значения в полях **Журнал**, **Номер аналитики** и **Тип складских запасов** основаны на проводках инвентаризации запасов. Чтобы изменить значения этих полей, выберите **Фильтр.** Например, можно изменить эти значения для создания отчета для другого журнала или другой аналитики.

  ![Печать акта инвентаризации (ИНВ-6).](media/goods-in-transit-vendor-10.png)
 
15. Выберите **OK**, чтобы создать отчет **Акт инвентаризации (ИНВ-6)**.

  ![Распечатанный отчет акта инвентаризации (ИНВ-6).](media/goods-in-transit-vendor-11.png)
