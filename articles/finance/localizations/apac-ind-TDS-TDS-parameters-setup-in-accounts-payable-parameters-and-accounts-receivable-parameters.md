# <a name="set-tds-parameters-in-accounts-payable-and-accounts-receivable"></a>Настройте параметры TDS в модуле "Расчеты с поставщиками" и "Расчеты с клиентами"

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как настроить параметры модуля расчетов с поставщиками и расчетов с клиентами для поддержки вычета налога, удерживаемого в источнике (TDS).

1. Перейдите в раздел **Налог \> Настройка \> Параметры \> Параметры модуля расчетов с клиентами**.
2. На вкладке **Обновления** выберите **Обновление строк заказа**, чтобы открыть диалоговое окно **Обновление строк заказа**.
3. В разделе **Группа TDS** в поле **Обновление группы TDS** укажите метод, который будет использоваться для обновления группы TDS на уровне строк. Этот параметр используется, если группа TDS обновлена в заголовке заказа. Имеются следующие варианты:

    - **Никогда** — группа TDS не обновляется в строках заказа при обновлении заголовка заказа.
    - **Всегда** — группа TDS обновляется в строках заказа при обновлении заголовка заказа.
    - **Запрос** — пользователи получат сообщение, предлагающее обновить группу TDS в строках заказа.
4. Нажмите **ОК**.

    [![Диалоговое окно "Обновление строк заказа"](./media/apac-ind-TDS-26.PNG)](./media/apac-ind-TDS-26.PNG)

5. Перейдите в раздел **Налог \> Настройка \> Параметры \> Параметры модуля расчетов с поставщиками**.
6. На вкладке **Общие** на экспресс-вкладке **Разбивка по сведениям о доставке** установите для параметра **Поступление продуктов** значение **Да**, чтобы разнести и разделить поступление продуктов, у которых другие адреса поставки и номера налоговых счетов (TAN). Если для этого параметра установлено значение **Нет**, разноска отборочной накладной по покупке с другими адресами доставки и номерами TAN.
7. Установите для параметра **Накладная** значение **Да**, чтобы разнести и разбить накладную по покупке, имеющую другие адреса доставки и номера TAN.

    [![Экспресс-вкладка разбиения на основе информации о доставке](./media/apac-ind-TDS-25.png)](./media/apac-ind-TDS-25.png)

8. Закройте страницу.
