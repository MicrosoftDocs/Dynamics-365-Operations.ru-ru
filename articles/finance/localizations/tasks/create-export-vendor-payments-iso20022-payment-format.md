# <a name="create-and-export-vendor-payments-using-iso20022-payment-format"></a>Создание и экспорт платежей поставщикам с помощью формата платежей ISO20022

[!include [task guide banner](../../includes/task-guide-banner.md)]

В этом разделе объясняется, как создать строки платежей в журнале платежей поставщику и как создать файл платежей поставщикам с использованием примера перемещения кредита ISO2022.

Это пятая процедура из пяти, которые иллюстрируют процесс платежа поставщикам с помощью конфигурации электронной отчетности. Для выполнения этого примера используйте демонстрационные данные DEMF.

## <a name="example"></a>Пример

1.  Перейдите в раздел **Расчеты с поставщиками > Платежи > Журнал платежей**.
2.  Нажмите кнопку **Создать**.
3.  В поле **Имя** введите или выберите значение.
4.  Щелкните **Предложение по оплате -> Создать предложение по оплате**.
5.  Разверните раздел **Записи для добавления**.
6.  Щелкните **Фильтр**.
7.  В списке выберите строку для **таблицы "Поставщики"** и **поля "Счет поставщика"**.
8.  В поле **Критерии** введите или выберите значение. Можно применить любые критерии для выбора проводок поставщиков для оплаты, для данного примера в качестве счета поставщика используется DE-001.
12. Нажмите кнопку **ОК**.
13. Нажмите кнопку **ОК**.
14. Щелкните **Создать платежи**.
15. Создайте файл платежа ISO20022.
    1.  Щелкните **Создать платежи**.
    2.  В поле **Способ оплаты** введите или выберите значение.
    3.  В поле **Имя файла** введите значение. Так как в этом примере платеж производится в ЕВРО, созданный файл будет SEPA-совместимым. Перенос кредита ISO20022, а также другие форматы платежей поставщикам могут также использоваться для создания платежей в других валютах.
    4.  В поле **Банковский счет** введите или выберите значение.
