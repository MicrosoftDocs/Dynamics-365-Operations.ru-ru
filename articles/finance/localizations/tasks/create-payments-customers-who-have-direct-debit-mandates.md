# <a name="create-payments-for-a-customer-who-have-direct-debit-mandates"></a>Создание платежей для клиента с поручениями на безакцептное списание

[!include [banner](../../includes/banner.md)]

В этой процедуре демонстрируется создание файла платежа прямого дебетования ISO20022 для клиента, для которого настроено прямое дебетование и накладная, подлежащая оплате. Создание и разноска накладной не являются обязательными. Вместо задания накладной, подлежащей оплате, можно выбрать поручение в журнале перед созданием файла платежа для поддержки сценарий предоплаты клиента.



В качестве компании с демонстрационными данными для создания этой процедуры используется DEMF.



Это пятая процедура из пяти, которые демонстрируют процесс платежа клиентов с помощью конфигурации электронной отчетности. Перед выполнением этой задачи необходимо выполнить предыдущие задачи. Сначала следует импортировать конфигурации электронной отчетности по платежам клиентов, настроить метод платежей и настроить компанию и сведения о клиенте. 


## <a name="post-a-free-text-invoice-with-direct-debit-information"></a>Разноска накладной с произвольным текстом со сведениями прямого дебетования
1. Перейдите в раздел "Расчеты с клиентами" > "Накладные" > "Все накладные с произвольным текстом".
2. Щелкните "Создать".
3. В поле "Счет клиента" введите или выберите значение.
    * Например, выберите DE-010.  
4. В поле "Способ оплаты" введите или выберите значение.
    * Например, выберите "Электронно".  
5. В поле код предписания прямого дебетования введите или выберите значение.
6. Щелкните "Добавить строку".
7. В поле "Описание" введите значение.
8. В поле "Счет ГК" укажите требуемые значения.
9. В поле "Цена за единицу" введите число.
10. Щелкните "Разнести".
11. Нажмите кнопку "OК".

## <a name="create-a-payment"></a>Создание платежа
1. Перейдите в раздел "Расчеты с клиентами" > "Настройка платежей" > "Журнал платежей".
2. Щелкните "Создать".
3. В поле "Имя" введите или выберите значение.
4. Щелкните "Строки".
5. Щелкните "Предложение по оплате".
6. Щелкните "Создать предложение по оплате".
7. Разверните раздел "Записи для добавления".
8. Щелкните "Фильтр".
9. В списке выберите строку для таблицы "Проводки клиента" и поля "Способ оплаты".
    * Можно применить любые критерии выбора проводки клиента для оплаты. В этом примере для фильтрации проводок используется способ оплаты "Электронно".  
10. В поле "Критерии" введите или выберите значение.
11. Нажмите кнопку "OК".
12. Нажмите кнопку "OК".
13. Щелкните "Создать платежи".
