[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="35a54-104">В этой теме приводятся сведения, которые помогут выбрать один из месячных, квартальных, полугодовых и годовых интервалов начисления сложных процентов.</span><span class="sxs-lookup"><span data-stu-id="35a54-104">This topic provides information that will help you choose among monthly, quarterly, semiannual, and annual compounding intervals.</span></span> <span data-ttu-id="35a54-105">Функция интервалов начисления сложных процентов используется для определения количества периодов начисления сложных процентов за год в графике оплаты аренды.</span><span class="sxs-lookup"><span data-stu-id="35a54-105">The compounding interval functionality is used to determine the number of compounding periods per year in a lease's payment schedule.</span></span> <span data-ttu-id="35a54-106">В каждом из четырех примеров данного раздела показано, как будет выглядеть график платежей аренды для различных интервалов.</span><span class="sxs-lookup"><span data-stu-id="35a54-106">Each of the four examples in this topic shows what a lease's payment schedule will look like for a different interval.</span></span>

<span data-ttu-id="35a54-107">Интервал начисления сложных процентов не может быть выбран, если его частота меньше частоты платежа, указанного в аренде.</span><span class="sxs-lookup"><span data-stu-id="35a54-107">You can't select a compounding interval that is less frequent than the lease's payment frequency.</span></span> <span data-ttu-id="35a54-108">Например, ежеквартальный интервал начисления сложных процентов не может использоваться с частотой ежемесячной оплаты, а ежегодный интервал начисления сложных процентов не может использоваться с полугодовой частотой оплаты.</span><span class="sxs-lookup"><span data-stu-id="35a54-108">For example, a quarterly compounding interval can't be used with a monthly payment frequency, and an annual compounding interval can't be used with a semiannual payment frequency.</span></span> <span data-ttu-id="35a54-109">Если попробовать выбрать интервал начисления сложных процентов, частота которого меньше частоты платежа, указанного в аренде, вы получите сообщение об ошибке.</span><span class="sxs-lookup"><span data-stu-id="35a54-109">If you try to select a compounding interval that is less frequent than the lease's payment frequency, you receive an error message.</span></span>

> <span data-ttu-id="35a54-110">Во всех четырех примерах в этой теме интервал начисления сложных процентов совпадает с частотой платежа.</span><span class="sxs-lookup"><span data-stu-id="35a54-110">In all four examples in this topic, the compounding interval matches the payment frequency.</span></span>
