# <a name="compounding-interval-functionality"></a>Функция интервалов начисления сложных процентов

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

В этой теме приводятся сведения, которые помогут выбрать один из месячных, квартальных, полугодовых и годовых интервалов начисления сложных процентов. Функция интервалов начисления сложных процентов используется для определения количества периодов начисления сложных процентов за год в графике оплаты аренды. В каждом из четырех примеров данного раздела показано, как будет выглядеть график платежей аренды для различных интервалов.

Интервал начисления сложных процентов не может быть выбран, если его частота меньше частоты платежа, указанного в аренде. Например, ежеквартальный интервал начисления сложных процентов не может использоваться с частотой ежемесячной оплаты, а ежегодный интервал начисления сложных процентов не может использоваться с полугодовой частотой оплаты. Если попробовать выбрать интервал начисления сложных процентов, частота которого меньше частоты платежа, указанного в аренде, вы получите сообщение об ошибке.

> [!NOTE]
> Во всех четырех примерах в этой теме интервал начисления сложных процентов совпадает с частотой платежа.
