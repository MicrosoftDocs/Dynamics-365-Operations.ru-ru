# <a name="view-liability-asset-and-expense-transactions"></a>Просмотр проводок по задолженности, активам и расходам

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как просмотреть проводки для арендованного актива. Эти проводки включают проводки по обязательству по аренде и проводки по затратам по осуществлению аренды, которые были разнесены. Учетные стоимости обязательств и актива в форме права пользования (ФПП) используются в нескольких отчетах. Они также используются для расчета значений коррекции.

## <a name="liability-transactions"></a>Транзакции по обязательствам

Чтобы просмотреть проводки по обязательствам по аренде, выберите аренду на странице **Сводка по аренде**, а затем выберите **Книги**, чтобы открыть книги, связанные с записью аренды. После разноски первоначального признания, накладной или журнала процентов выберите **Проводки по обязательствам**.

На странице **Проводки по обязательствам** отображаются проводки, которые либо увеличивают, либо уменьшают арендное обязательство. Отрицательные суммы на этой странице представляют собой кредитные проводки в стандартном приложении. Все начисления процентов отображаются как отрицательные значения и увеличивают общую сумму арендного обязательства. Любые корректировки аренды отображаются как положительные или отрицательные значения, в зависимости от природы и воздействия книги по аренде. Текущее чистое общее сальдо арендного обязательства для выбранной аренды отображается в левом нижнем углу страницы.

## <a name="asset-transactions"></a>Транзакции по активу

Чтобы просмотреть проводки по активам по аренде, выберите аренду на странице **Сводка по аренде**, а затем выберите **Книги**, чтобы открыть книги аренды, связанные с записью аренды. Затем выберите **Проводки по активам**.

На странице **Проводка по активу** отображаются проводки, которые либо увеличивают, либо уменьшают счета аренды актива и накопленной амортизации. Дебеты отображаются как положительные значения, а кредиты отображаются как отрицательные значения, как на странице **Проводки по обязательствам**. Учтенное первоначальное признание и следующая накопленная амортизация показаны в нижней левой части страницы в качестве сальдо актива. 

## <a name="expenses-transactions"></a>Проводки по расходам

Чтобы просмотреть проводки по расходам по аренде, выберите аренду на странице **Сводка по аренде**, а затем выберите **Книги**, чтобы открыть книги аренды, связанные с записью аренды. Затем выберите **Проводки по расходам**.

На странице **Проводки по расходам** отображаются все расходы, которые были разнесены по аренде, такие как расходы на уплату процентов, расходы на амортизацию и любые затраты по осуществлению аренды.
