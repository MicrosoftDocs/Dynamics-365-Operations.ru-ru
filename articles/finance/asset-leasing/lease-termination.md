# <a name="propose-a-lease-for-termination"></a>Предложение аренды для завершения

[!include [banner](../includes/banner.md)]

Если аренда была прекращена раньше, аренда актива может записать запись журнала завершения, чтобы списать арендное обязательство, актив в форме права пользования (ФПП) и накопленной амортизации, а также журнала прибылей или убытков. Процесс раннего завершения прерывает аренду и ее связанные журналы аренды. Он не прерывает отдельные книги аренды. В этой теме описывается функция, позволяющая предложить аренду для завершения и обработать запись в журнале прекращения аренды.

Если аренда не классифицируется как отсроченный арендный лизинг и не связана с основными средствами, аренда актива создает следующую запись журнала о прекращении.

| Проводка                           | Дебет (Дб.) | Кредит (Кр.) |
|---------------------------------------|-------------|--------------|
| Дб. Арендное обязательство                   | Х           |              |
| Дб. Накопленная амортизация          | Х           |              |
| Дб. Прибыли (убытки) от изменения аренды | Х           |              |
| Кр. Арендуемый актив                       |             | Х            |
| Кр. Прибыли (убытки) от изменения аренды |             | Х            |

Если журнал аренды классифицируется как журнал отложенной ренты, запись списывает сальдо отложенной ренты до завершения на счет прибылей и убытков, как показано здесь.

| Проводка                           | Дебет (Дб.) | Кредит (Кр.) | 
|---------------------------------------|-------------|--------------|
| Дб. Отложенная рента                     | Х           |              |
| Кр. Прибыли (убытки) от изменения аренды |             | Х            |
| Кр. Отложенная рента                     |             | Х            |
| Дб. Прибыли (убытки) от изменения аренды | Х           |              |

Если журнал аренды связан с основными средствами, актив ФПП учитывается в модуле основных средств. Этот счет включает учет для раннего завершения. Аренда актива создает следующую запись в журнале для списания обязательств по аренде.

| Проводка                           | Дебет (Дб.) | Кредит (Кр.) |
|---------------------------------------|-------------|--------------|
| Дб. Арендное обязательство                   | Х           |              |
| Кр. Прибыли (убытки) от изменения аренды |             | Х            |

Сведения о правильном способе ликвидации актива ФПП см. в разделе [Списание основного средства в утиль](../fixed-assets/dispose-of-a-fixed-asset-as-scrap.md).

## <a name="propose-a-lease-for-termination"></a>Предложение аренды для завершения

1. Перейдите к аренде, которая должна быть завершена, а затем в области действий выберите **Предложение о завершении**.

    > [!NOTE]
    > Кнопка **Предложение завершения** недоступна, если имеются неразнесенные записи журнала в какой-либо книге. Перед прекращением аренды необходимо выполнить разноску или удаление всех записей журнала, которые были созданы для аренды.

2. В появившемся диалоговом окне установите поля **Дата вступления в силу** и **Дата разноски** для записи журнала прекращения.
3. Выбор **Предложение завершения** для предложения аренды для завершения.
4. Выберите **Разнести завершение аренды** для автоматической разноски записи журнала завершения аренды.
5. На странице **Прекращение аренды** выберите код аренды для аренды, предлагаемой для завершения, для просмотра строк завершения. В строках завершения показаны учетные значения для актива ФПП, обязательства по аренде, накопленная амортизация, отложенная рента (если применимо), а также прибыли и убытки, которые должны быть распознаны при прекращении аренды.

Аренда готова к завершению. Значение поля **Статус завершения** для книги аренды изменяется на **Готово для завершения**. На этом этапе нельзя будет выполнять разноску записей журнала для аренды, а также корректировать или обесценивать ее. 

## <a name="process-leases-that-are-ready-for-termination"></a>Обработка аренд, готовых для завершения

Для обработки аренд, готовых к завершению, и разноски записи журнала завершения, выполните следующие действия.

1. На странице **Завершения аренд** выберите аренду для обработки, затем выберите **Завершить**.
2. В появившемся диалоговом окне выберите **ОК**.

Система выполняет разноску записи журнала завершения. Поле **Статус аренды** для данного журнала аренды установлено на **Завершено**, а в поле **Статус предложения по завершению** установлено значение **Завершено**.

## <a name="reverse-a-lease-termination"></a>Отмена прекращения аренды

Чтобы отменить запись журнала завершения аренды и открыть прекращенную аренду, выполните этот шаг.

- На странице **Завершения аренд** выберите завершенную аренду для отмены, затем выберите **Отменить завершение**.

Система отменяет запись журнала завершения. В поле **Статус аренды** для данного журнала аренды устанавливается значение **Открыта**. Аренда больше не отображается на странице **Прекращения аренды** и может быть предложена снова для завершения.

## <a name="example-of-a-lease-termination"></a>Пример завершения аренды

В данном примере аренда связана с неспециализированным активом, и она не передает владение активом или не предоставляет арендополучателю возможность приобретения актива.

### <a name="setup"></a>Настройка

В следующих таблицах показаны значения, заданные на вкладках **Общие** и **Строки графика оплаты** для аренды, используемой в этом примере.

**Вкладка "Общие"**

| Поле                      | значение            |
|----------------------------|------------------|
| Реальная стоимость актива    | 600,000          |
| Валютное                   | USD              |
| Первоначальные прямые затраты       | 1000            |
| Приростная ставка процента на заемный капитал | 7%               |
| Интервал объединения       | Ежегодно         |
| Срок службы актива (месяцы) | 600              |
| Тип аннуитета               | Аннуитет постнумерандо |
| Дата начала          | 1/1/2019         |

**Вкладка "Строки графика платежей"**

| Поле             | значение      |
|-------------------|------------|
| Начальная дата        | 1/1/2019   |
| Интервал периода   | ежемесячно    |
| Периоды           | 120        |
| Конечная дата          | 31.12.2028 |
| Частота платежей | Ежегодно   |
| Сумма оплаты    | 10,000     |

### <a name="steps-for-terminating-the-lease"></a>Шаги для прекращения аренды

1. После создания аренды, как описано ранее в этом разделе, перейдите в книгу аренды и подтвердите график оплаты. Затем разнесите запись журнала первоначального признания. Первоначально актив ФПП стоит 71 235,81 доллара США, и арендное обязательство должны быть 70 235,81 доллара США. В данном примере аренда классифицируется как операционная аренда по теме 842 кодификации бухгалтерских стандартов (ASC 842).
2. Три раза выполните процесс журнала пакетных заданий, чтобы смоделировать три года для арендных платежей, расходов на уплату процентов и затрат на амортизацию.
3. После завершения выполнения всех трех пакетных заданий вернитесь в книгу аренды и откройте таблицы проводок по обязательствам и активам, чтобы просмотреть текущую учетную стоимость актива ФПП и арендного обязательства. Через три года стоимость обязательства должна составлять примерно -53 893,00 долларов США, а стоимость актива должна быть примерно равна 54 593,00 долларов США.

    По истечении трех лет компания и арендодатель взаимно договариваются о прекращении аренды. Таким образом, теперь необходимо завершить аренду.

4. Перейдите к аренде, которая должна быть завершена, а затем в области действий выберите **Предложение о завершении**. 
5. В появившемся диалоговом окне в полях **Дата вступления в силу** и **Дата разноски** введите **1/1/2021**.
6. Выбор **Предложение завершения** для предложения аренды для завершения.

    Отображается страница **Завершение аренды**, в которой отображается аренда, которая будет прекращена.

7. Для просмотра строк прекращения выберите код аренды для аренды, предлагаемой для завершения. В строках завершения отображаются учетные значения аренды. В следующей таблице показано, какие эти значения должны быть для этого примера. 

    | Поле                                                 | значение      |
    |-------------------------------------------------------|------------|
    | Учетное сальдо обязательств в валюте проводки | 53 892,89 долларов США |
    | Актив в форме права пользования в валюте проводки            | 71 235,81 доллара США |
    | Накопленная амортизация в валюте проводки      | 16 642,92 доллара США |
    | Прибыль (убыток) в валюте проводки                   | -700,00 доллара США   |

8. Чтобы разнести запись журнала завершения, выберите аренду на странице **Завершения аренд**, затем выберите **Завершить**.
9. В появившемся диалоговом окне выберите **ОК**.
10. Чтобы просмотреть созданную и разнесенную запись журнала завершения, перейдите к журналу аренды в книге аренды. В следующей таблице показано, как эта запись должна выглядеть для этого примера.

    | Проводка                           | Дебет (Дб.) | Кредит (Кр.) |
    |---------------------------------------|-------------|--------------|
    | Дб. Арендное обязательство                   | 53,892.89   |              |
    | Дб. Накопленная амортизация          | 16,642.92   |              |
    | Дб. Прибыли (убытки) от изменения аренды | 700.00      |              |
    | Кр. Арендуемый актив                       |             | 71,235.81    |

11. Для просмотра чистого эффекта завершения, где актив ФПП и обязательство по аренде будут иметь значение 0 (ноль), откройте таблицы проводок обязательств и активов.

Статус аренды должен быть **Завершена**. Никакие дополнительные записи журнала не будут разноситься по этой аренде, пока прекращение не будет отменено.
