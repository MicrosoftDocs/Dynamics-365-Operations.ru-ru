# <a name="create-a-lease-group"></a>Создание группы аренды

[!include [banner](../includes/banner.md)]

В этом разделе описывается, как настроить группы аренды. Группы аренды необходимы для создания новых аренд. Книги по аренде сопоставляются с каждой группой аренды. Книги по аренде определяют книги по умолчанию, которые должны быть созданы для каждой аренды. Можно назначить отдельные счета для группы аренды на странице **Параметры разноски аренды**.

## <a name="create-a-lease-book-and-add-a-lease-group"></a>Создание книги аренды и добавление группы арены

1. Перейдите в раздел **Аренда активов \> Настройка \> Группы аренды**.
2. На панели операций выберите **Создать** для добавления группы аренды. В сетку добавляется строка.
3. В новой строке в поле **Группа аренды** введите значение.
4. В поле **Описание** введите значение.

Введенные сведения добавляются в поле **Группа аренды** на странице **Добавить аренду**.

## <a name="associate-a-book-with-a-lease-group"></a>Связывание книги с группой аренды

После создания групп аренды можно назначить книги для каждой группы. При создании аренды и назначении ее группе аренды система создает набор графиков для каждой книги, связанной с данной группой аренды.

> [!NOTE]
> Прежде чем книги могут быть назначены группе аренды, необходимо настроить книги.

1. Перейдите в раздел **Аренда активов \> Настройка \> Группа аренды**.
2. Выберите группу аренды, а затем выберите **Книги**.
3. Выберите **Создать**, а затем в поле **Тип книги** выберите книгу, которую необходимо назначить группе аренды. Имеется возможность назначить несколько книг группе аренды, если аренда должна быть учтена другими способами.
