[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="4b58a-105">В этой теме объясняется, как создать номерную серию для кодов аренды.</span><span class="sxs-lookup"><span data-stu-id="4b58a-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="4b58a-106">Здесь также объясняется, как создавать уникальные коды, используемые в процессе переоценки индекса.</span><span class="sxs-lookup"><span data-stu-id="4b58a-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>

1. <span data-ttu-id="4b58a-107">Перейдите в раздел **Аренда активов \> Настройка \> Параметры аренды активов**.</span><span class="sxs-lookup"><span data-stu-id="4b58a-107">Go to **Asset leasing \> Setup \> Asset leasing parameters**.</span></span>

3. <span data-ttu-id="4b58a-109">Выберите **Номерные серии** на боковой панели.</span><span class="sxs-lookup"><span data-stu-id="4b58a-109">Select **Number sequences** in the side bar.</span></span>
