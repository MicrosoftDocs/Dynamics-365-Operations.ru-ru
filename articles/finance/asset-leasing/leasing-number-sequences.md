# <a name="assign-number-sequences"></a>Назначение номерных серий

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как создать номерную серию для кодов аренды. Здесь также объясняется, как создавать уникальные коды, используемые в процессе переоценки индекса.

1. Перейдите в раздел **Аренда активов \> Настройка \> Параметры аренды активов**.
2. Перейдите на боковую вкладку **Номерные серии**.
3. Выберите **Номерные серии** на боковой панели.
4. Выберите номерную серию для ссылки **Код аренды**. Эта номерная серия будет использоваться для создания уникального идентификатора для каждой аренды.
5. Выберите номерную серию для ссылки **Код процесса**. Эта номерная серия будет использоваться для создания уникального идентификатора для каждого процесса переоценки индекса.
6. Выберите номерную серию для ссылки **Код предложения завершения**. Эта номерная серия будет использоваться для создания уникального идентификатора для каждого предложения завершения.
