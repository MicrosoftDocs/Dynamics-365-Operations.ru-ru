[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="50565-105">В этой теме объясняется, как создать номерную серию для кодов аренды.</span><span class="sxs-lookup"><span data-stu-id="50565-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="50565-106">Здесь также объясняется, как создавать уникальные коды, используемые в процессе переоценки индекса.</span><span class="sxs-lookup"><span data-stu-id="50565-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>

1. <span data-ttu-id="50565-107">Перейдите в раздел **Аренда активов \> Настройка \> Параметры аренды активов**.</span><span class="sxs-lookup"><span data-stu-id="50565-107">Go to **Asset leasing \> Setup \> Asset leasing parameters**.</span></span>
2. <span data-ttu-id="50565-108">Перейдите на боковую вкладку **Номерные серии**.</span><span class="sxs-lookup"><span data-stu-id="50565-108">Select the **Number sequences** side tab.</span></span>
3. <span data-ttu-id="50565-109">Выберите **Номерные серии** на боковой панели.</span><span class="sxs-lookup"><span data-stu-id="50565-109">Select **Number sequences** in the side bar.</span></span>

5. <span data-ttu-id="50565-112">Выберите номерную серию для ссылки **Код процесса**.</span><span class="sxs-lookup"><span data-stu-id="50565-112">Select the number sequence for the **Process ID** reference.</span></span> <span data-ttu-id="50565-113">Эта номерная серия будет использоваться для создания уникального идентификатора для каждого процесса переоценки индекса.</span><span class="sxs-lookup"><span data-stu-id="50565-113">This number sequence will be used to generate the unique identifier for each index revaluation process.</span></span>
6. <span data-ttu-id="50565-114">Выберите номерную серию для ссылки **Код предложения завершения**.</span><span class="sxs-lookup"><span data-stu-id="50565-114">Select the number sequence for the **Termination Proposal ID** reference.</span></span> <span data-ttu-id="50565-115">Эта номерная серия будет использоваться для создания уникального идентификатора для каждого предложения завершения.</span><span class="sxs-lookup"><span data-stu-id="50565-115">This number sequence will be used to generate the unique identifier for each termination proposal.</span></span>
