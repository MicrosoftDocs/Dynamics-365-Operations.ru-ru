# <a name="asset-leasing-conventions"></a>Соглашения по аренде активов

[!include [banner](../includes/banner.md)]

В этом разделе описываются соглашения для аренды активов. Соглашения по аренде используются для определения даты начала книги аренды. Если соглашение по аренде имеет значение **Нет**, дата начала совпадает с начальной датой аренды (то есть, со значением поля **Дата начала аренды**). Если соглашение по аренде настроено на **Полный месяц**, дата начала — это первый день месяца, на который приходится начальная дата аренды.

Дата начала определяет начальную дату периода для амортизации обязательств и графиков амортизации актива. Расходы на уплату процентов и расходы на амортизацию разносятся на дату окончания периода соответствующих расписаний. Начальное признание и запись журнала корректировки разносятся на дату начала.

> [!NOTE]
> Функцию для соглашений по аренде необходимо включить через механизм управления функциями. В рабочей области **Управление функциями** найдите и выберите функцию, которая называется **Соглашение по аренде для аренды актива**, затем выберите **Включить**.

Соглашения по аренде назначаются настройке для книги аренды активов.

Чтобы просмотреть или назначить соглашение по аренде, выполните следующие действия.

1. Перейдите в раздел **Аренда активов \> Настройка \> Книги аренды**.
2. В поле **Соглашение по аренде** выберите одно из следующих значений.

    | Соглашение об аренде | описание |
    |--------------------|-------------|
    | Не допускается               | Амортизация обязательств и графики амортизации актива начинаются с даты начала аренды, так как дата начала совпадает с начальной датой аренды. Дата окончания на один месяц позже. Это соглашение по аренде не гарантирует, что процент будет разнесен в последний день каждого месяца. |
    | Полный месяц         | Для аренд с датой начала, которая указана в любое время в течение месяца, амортизация обязательств и график амортизации актива начинают начислять расходы на первый день этого месяца. Это соглашение по аренде гарантирует, что процент начисляется в последний день каждого месяца для всего месяца. |

3. Нажмите **Сохранить**.

При создании аренды дата начала каждой книги автоматически вводится на основе начальной даты, введенной для аренды, и соглашения по аренде, которое указано для книги.
