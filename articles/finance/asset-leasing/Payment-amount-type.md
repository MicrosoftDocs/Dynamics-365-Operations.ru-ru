# <a name="add-payment-amount-types"></a>Добавление типов сумм платежей

[!include [banner](../includes/banner.md)]

В этой статье описывается, как настраивать типы сумм платежей в модуле "Аренда активов". Таким образом можно детализировать сумму платежа по аренде вместо сложения суммы общей суммы в строках графика платежей.

Чтобы добавить типы сумм платежей, выполните следующие действия.

1. Перейдите в раздел **Аренда активов \> Настройка \> Типы сумм платежей**.
2. Выберите **Создать**.
3. Введите новый тип платежа и описание.

> [!NOTE]
> Для переоценки индекса МСФО 16 необходимо создать один тип суммы платежей и пометить его как **Используется для переоценки индекса IFRS16**. Этот тип суммы платежей будет использоваться, когда процесс переоценки индекса выполняется в отношении МСФО 16, и будут учитывать изменения, произошедшие из-за процесса переоценки индекса.
>
> Если параметр **Детализаций платежей** в аренде имеет значение **Да**, то если при выполнении переоценки индекса для МСФО 16 отсутствует тип платежа для МСФО 16, процесс не будет завершен.

Только одна запись может быть помечена как **Используется для переоценки индекса IFRS16**.
