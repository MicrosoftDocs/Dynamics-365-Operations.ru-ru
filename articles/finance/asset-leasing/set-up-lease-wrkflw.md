# <a name="set-up-lease-approval-workflows"></a>Настройка рабочих процессов утверждения аренды

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как настроить рабочий процесс утверждения, который будет выполняться при создании новой аренды. Дополнительные сведения об использовании рабочего процесса см. в разделе [Использование рабочих процессов утверждения аренды](use-create-lease-wrkflw.md). 

1. Перейдите в раздел **Аренда активов \> Настройка \> Рабочий процесс аренды**.
2. На странице **Рабочий процесс аренды** нажмите **Создать**.
3. В диалоговом окне, которое отображается, в разделе **Тип рабочего процесса** выберите ссылку **Рабочий процесс аренды**.

    Приложение открывается. После его выполнения выполните вход в Azure Active Directory (Azure AD) для перенаправления в приложение рабочего процесса.

4. Перетащите элемент **Утверждение рабочего процесса аренды** в рабочий процесс.
5. Соедините один узел от **Начало** до **Утверждение рабочего процесса аренды**. Затем соедините **Утверждение рабочего процесса для аренды** с **Готово**.
6. Дважды щелкните **Утверждение рабочего процесса аренды**.
7. Выберите **Свойства**, а затем в разделе **Основные параметры** введите имя рабочего процесса.

    На этой странице можно также настроить дополнительные параметры для рабочего процесса. Если вы включили **Автоматические действия**, система автоматически выполнит конкретное действие. Уведомления могут быть отправлены, если они указаны на вкладке **Уведомления**. На вкладке **Дополнительные параметры** можно указать окончательного утверждающего, установить предельное время и назначить определенные действия, которые необходимо выполнить.

8. После завершения настройки параметров рабочего процесса выберите **Закрыть**.
9. Выберите **Шаг 1**, а затем выберите **Свойства**.
10. В разделе **Основные параметры** введите имя шага, создайте строку темы для утверждения и укажите инструкции для утверждения.
11. На странице **Назначение** выберите тип назначения.
12. Чтобы назначить определенных пользователей для утверждения, выберите **Пользователь**, выберите пользователей, которые утверждают аренды, а затем выберите **Закрыть**.
13. Выберите **Сохранить и закрыть**, чтобы создать рабочий процесс. Затем, когда появится запрос, выберите **ОК**.
14. На странице **Создать рабочий процесс** выберите **Закрыть**.
14. Выберите новый рабочий процесс, затем выберите **Версии**. Затем выберите **Активировать**, чтобы убедиться, что рабочий процесс активен.
15. Выберите **Закрыть**. Появится новая активная версия.
