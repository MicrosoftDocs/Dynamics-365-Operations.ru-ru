# <a name="confirm-asset-leasing-payment-schedules-in-a-batch"></a>Подтверждение графиков оплаты аренды активов в пакетном режиме

[!include [banner](../includes/banner.md)]

В этой теме объясняется, как подтвердить несколько графиков оплаты в пакетном режиме. Графики оплаты подтверждаются либо на поарендной основе, либо с помощью процесса пакетной обработки подтверждения. Запись журнала может быть разнесена только по аренде, которая имеет подтвержденный график оплаты. Подтверждение графика оплаты служит окончательным утверждением финансовой информации для аренды. Все будущие изменения финансовой информации для аренды, такие как платежи и срок аренды, составляют корректировку аренды и должны обрабатываться таким образом.

Чтобы подтвердить несколько графиков оплаты, выполните следующие действия.

1. Перейдите в раздел **Аренда активов \> Периодические задачи \> Пакетное подтверждение**.
2. На странице **Пакетное подтверждение** выберите **Пакетное подтверждение**.
3. В появившемся диалоговом окне отфильтруйте книги, которые требуется подтвердить.

    - Чтобы подтвердить все книги в конкретной группе аренды, выберите группу в поле **Группа аренды**.
    - Чтобы подтвердить отдельные книги, выберите книги в поле **Код книги**.
    - Чтобы подтвердить все книги, включите параметр **Для всех книг**.

Сведения для вновь подтвержденных книг отображаются на странице **Подтвержденные книги**. После подтверждения графиков оплаты записи журнала начального признания могут быть разнесены по арендам.
