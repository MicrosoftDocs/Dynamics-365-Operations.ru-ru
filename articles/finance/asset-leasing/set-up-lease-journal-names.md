# <a name="set-up-lease-journal-names"></a>Настройка наименований журналов аренды

[!include [banner](../includes/banner.md)]

Названия журналов аренды определяют журналы, в которые разносятся проводки аренды активов. Только имена журналов, назначенных для типа журнала **Аренда активов**, отображаются в полях **Первоначальное признание** и **Имя ежемесячного журнала** на странице **Параметры аренды активов**. Только тип журнала **запись накладных поставщика** может быть назначен в поле **Наименование журнала накладных**.

Для настройки наименований журналов аренды выполните следующие действия.

1. Перейдите в раздел **Аренда активов \> Настройка \> Параметры аренды активов**.
2. На вкладке **Общие** в поле **Имя журнала первоначального признания** выберите журнал. Все записи журнала первоначального признания будут разнесены по этому имени журнала.
3. В поле **Имя журнала накладных** выберите журнал. Если для параметра **Оплата поставщику** установлено значение **Да** для книги аренды, накладные по арендной плате и по расходам будут разнесены по этому имени журнала.
4. В поле **Имя журнала аренды** выберите журнал. Все амортизации, проценты и записи изменения классификации краткосрочной аренды будут разнесены по этому имени журнала. Если для параметра **Оплата поставщику** установлено значение **Нет** для книги аренды, записи арендных платежей и платежей по расходам также будут разнесены по этому имени журнала.
