# <a name="asset-leasing-reports"></a>Отчеты об аренде активов

[!include [banner](../includes/banner.md)]

В этой теме перечислены и кратко описаны отчеты, доступные в аренде активов. Большинство отчетов отображаются путем выполнения этих шагов или очень похожих шагов, как отмечено). 

- Для просмотра большинства отчетов по аренде активов перейдите в раздел **Аренда активов > Запросы и отчеты > Отчеты об аренде**, а затем выберите отчет для просмотра. Для отчетов, требующих другой путь выбора, шаги, необходимые для открытия отчета, включаются вместе с описанием этого отчета. 
- При выборе отчета, который будет напечатан, откроется страница параметров, которая позволяет отфильтровать сведения, включенные в отчет. Введите критерий фильтра, а затем выберите **ОК**, чтобы создать отчет. Созданный отчет будет содержать сведения, которые попадают в указанные вами фильтры.

## <a name="asset-movement"></a>Проводка ОС
Отчет о движении активов выступает в качестве отчета о движении для балансов активов в форме права пользования для каждой аренды. Этот отчет позволяет просматривать проводки по активу для аренды за указанный период. Этот отчет включает следующие поля. 

|     Поля отчета                  |     описание                                                                |
|------------------------------------|--------------------------------------------------------------------------------|
|     Дата начала              |     Дата начала срока действия самой ранней версии аренды.                     |   
|     Срок аренды                     |     Срок аренды для ранней версии аренды.                            |
|     Краткосрочная аренда               |     Если аренда классифицируется как краткосрочная аренда, будет отображаться как **Да**.         |
|     Аренда малоценного актива                |     Если аренда классифицируется как аренда малоценного актива, будет отображаться как **Да**.          |
|     Первоначальный актив в форме права пользования     |     Исходное значение актива в форме права пользования из первоначальной записи журнала признания.      |
|     Начальное сальдо              |     Чистая остаточная стоимость актива в форме права пользования по состоянию на дату **С даты**.                         |
|     Расходы на амортизацию за период    |     Сумма расходов на амортизацию, попадающих в диапазон дат, определенный для отчета.        |
|     Накопленная амортизация       |     Сумма накопленной амортизации с даты **Дата начала**.                               |
|     Обесценение                     |     Сумма снижения стоимости актива в диапазоне дат, определенном для отчета.               |
|     Изменения                  |     Сумма корректировок актива в форме права пользования между параметрами даты.                            |
|     Остаточная стоимость                 |     Конечная остаточная стоимость актива в форме права пользования, которая является чистой суммой накопленной амортизации по состоянию на дату **Конечная дата**.    |

## <a name="differences-report"></a>Отчет о разницах
В отчете о различиях отображаются различия между информацией, загруженной в инфраструктуру импорта аренды, и информацией, которая находится в системе. Это позволяет сравнить, что было скорректировано, обновлено или добавлено через структуру импорта аренды.  

Значения в отчете будут различаться в зависимости от выбранной аренды. В отчете будут показаны только те поля, которые отличаются от текущих в системе и от того, что находится в промежуточных таблицах. Старым значением является то, что в настоящее время находится в системе, или что ранее было в системе, в зависимости от того, был ли выполнен процесс импорта. Новое значение отображает значение в промежуточной таблице, которое заменит старое значение.

## <a name="five-years-undiscounted-payment-forecast"></a>Прогноз платежей на 5 лет без дисконтирования
В отчете по прогнозу платежей на пять лет без дисконтирования отображаются предполагаемые выплаты по аренде без скидки в течение следующих пяти лет с даты, указанной в параметрах отчета. Этот отчет включает следующие поля. 

|     Поля отчета         |     описание                                                                                       |
|-------------------------- |---------------------------------------------------------------------------------------------------    |
|     Описание аренды     |     Описание аренды из заголовка аренды.                                                      |
|     ИД аренды              |     Уникальный идентификатор аренды.                                                                              |
|     Резервирование                  |     Книга аренды, связанная с кодом книги.                                                         |
|     Классификация        |     Классификация аренды.                                                                  |
|     Слой разноски         |     Слой, в который выполняется разноска данной аренды.                                                         |
|     Валютное              |     Валюта аренды.                                                                        |
|     Текущие               |     Сумма всех выплат арендной платы в течение следующих 12 месяцев с даты отчетности.          |
|     13–24 месяца          |     Сумма всех выплат арендной платы с 13 по 24 месяц с даты отчетности.           |
|     25–36 месяцев          |     Сумма всех выплат арендной платы с 25 по 36 месяц с даты отчетности.           |
|     37–48 месяцев          |     Сумма всех выплат арендной платы с 37 по 48 месяц с даты отчетности.           |
|     49–60 месяцев          |     Сумма всех выплат арендной платы с 49 по 60 месяц с даты отчетности.           |
|     Позже            |     Сумма всех выплат арендной платы в или после 61-го месяца с даты отчетности.              |

## <a name="gaap-cash-flows-report"></a>Отчет GAAP по движению денежных средств

|     Поля отчета                                 |     описание                                                                                                                                               |
|------------------------------------------------   |-----------------------------------------------------------------------------  |
|     с даты <br> Конечная дата                        |     Определяет диапазон дат, используемых для ограничения информации, включаемой в отчет.      |
|     Информация о компании                                  |     Юридическое лицо, связанное с арендами.                                      |
|     Тип аренды                                    |     Классификация аренды как финансовая или эксплуатационная.           |
|     ИД аренды                                      |     Уникальный идентификатор аренды.                                                      |
|     Описание аренды                             |     Описание аренды из заголовка аренды.                              |
|     Книга аренды                                    |     Книга аренды, на которую ссылается строка.                        |
|     Слой разноски                                 |     Каждая книга, которая присоединяется к основным средствам, настраивается для определенного слоя разноски, имеющего общее задание на амортизацию.                          |
|     Группа аренды                                   |     Группа, с которой связана аренда.                                     |
|     Валютное                                      |     Сокращение для использованной валюты проводки. Все отчеты преобразуют валюту проводки в валюту отчетности.                      |
|     Финансовые аренды — операционные денежные потоки         |     Сумма итоговых разнесенных переменных платежей и общих процентных выплат, разнесенных из графика амортизации между параметрами даты.        |
|     Финансовые аренды — финансовые денежные потоки           |     Сумма итоговых выплат по основным платежам из плана амортизации между параметрами даты.       |
|     Операционные аренды — операционные денежные потоки       |     Сумма всех разнесенных платежей по аренде и разнесенных переменных платежей между параметрами даты.            |

## <a name="lease-balances-forecast"></a>Прогноз сальдо по аренде
В прогнозе сальдо аренды отображаются сведения непосредственно из графика амортизации задолженностей и графика амортизации актива. В отчете отображаются прогнозируемые суммы по прогнозируемым обязательствам по аренде и активы в форме права пользования за определенный период времени, включая все прогнозируемые расходы для этих аренд. Этот отчет включает следующие поля.

|     Поля отчета                 |     описание                                                                                                                                                                               |
|---------------------------------  |--------------------------------------------------------------------------------------------------------------------   |
|     Начальное сальдо             |     Начальное сальдо в расписании амортизации аренды для периода, содержащего дату начала отчета.            |
|     Первоначальное признание           |     Если дата начала срока аренды попадает в диапазон дат, определенный для отчета, в этом столбце будет отображено значение счета задолженности первоначальной записи журнала признания.      |
|     Оплаты                      |     Сумма платежей аренды, попадающих в диапазон дат, определенный для отчета.                               |
|     Интересы                      |     Сумма расходов на уплату процентов по аренде, попадающих в диапазон дат, определенный для отчета.                    |
|     Конечное сальдо                |     Сальдо задолженности для аренды на дату **Дата окончания**.                                                             |
|     Краткосрочное обязательство          |     Сумма краткосрочных обязательств в графике амортизации аренды на дату **Дата окончания**.                           |
|     Долгосрочные обязательства           |     Сумма долгосрочных обязательств в графике амортизации аренды на дату **Дата окончания**.                            |
|     Начальная остаточная стоимость      |     "Начальная остаточная стоимость" в графике амортизации аренды для периода, содержащего дату начала отчета      |
|     Первоначальное признание           |     Если дата начала срока аренды попадает в между параметрами дат отчета, в этом столбце будет отображено значение счета актива записи журнала первоначального признания.            |
|     Расходы на амортизацию          |     Сумма расходов на амортизацию по аренде, попадающих в диапазон дат, определенный для отчета.                  |
|     Конечное сальдо                |     Сальдо аренды для аренды на дату **Дата окончания**.                                                              |
|     Корректировка собственного капитала             |     Начальное сальдо за вычетом начальной остаточной стоимости.                                                             |

## <a name="lease-commencements-report"></a>Отчет о начавшихся арендах
В отчете о начавшихся арендах отображаются все аренды, которые были начаты в указанном диапазоне дат, включая начальное обязательство и сальдо актива в форме права пользования. Этот отчет включает следующие поля. 

|     Поля отчета                 |     описание                                                                                       |
|---------------------------------  |---------------------------------------------------------------------------------------------------    |
|     Дата начала             |     Дата записи в журнале первоначального признания, которая была разнесена для данной версии аренды.         |
|     Сумма арендного обязательства      |     Сумма обязательств из записи журнала первоначального признания.                                  |
|     Первоначальная сумма актива          |     Сумма актива из записи журнала первоначального признания.                                      |

## <a name="lease-modification-report"></a>Отчет об изменении аренды
Отчет об изменении аренды показывает все аренды, которые были изменены в пределах указанного диапазона дат. В отчете также отображается пользователь, выполнивший корректировку аренды, и общая сумма скорректированных обязательств. Этот отчет включает следующие поля. 

|     Поля отчета                 |     описание           |
|---------------------------------  |-------------------------  |
|     Кем скорректировано:                   |     Имя пользователя, изменившего аренду.                                |
|     Дата корректировки               |     Дата разноски записи журнала корректировки.                        |
|     Корректировки                   |     Сумма всех записей журнала корректировок, разнесенных на счет обязательств аренды между параметрами даты. Кредиты будут отображаться как положительные, в то время как дебеты будут отрицательными.       |
|     Сумма прибылей (убытков)            |     Сумма всех записей журнала корректировок, разнесенных на счет прибылей/убытков аренды между параметрами даты. Кредиты будут отображаться как положительные, в то время как дебеты будут отрицательными.       |
|     Нераспределенная прибыль             |     Сумма всех записей журнала корректировок, разнесенных на счет нераспределенной чистой прибыли от аренды между параметрами даты.      |
|     Конечное сальдо по обязательству      |     Конечное сальдо обязательств на дату корректировки аренды.           |

## <a name="lease-movement-report"></a>Отчет по перемещению аренды
Отчет о движении аренды выступает в качестве отчета о движении для сальдо арендных обязательств для каждой аренды. Этот отчет позволяет пользователю просматривать проводки обязательств по аренде за указанный период.

|     Поля отчета             |     описание                                               |
|----------------------------   |-------------------------------------------------------------- |
|     Дата начала         |     Дата начала срока действия самой ранней версии аренды.    |
|     Срок аренды                |     Срок аренды для ранней версии аренды.           |
|     Осталось платежей        |     Число платежей, которые еще не были разнесены для аренды.                       |
|     Начальное сальдо         |     Сумма всех проводок, влияющих на обязательства по аренде, произошедших до даты начала отчета.             |
|     Первоначальное признание       |     Сумма всех начальных проводок признания для аренды, которая была разнесена в пределах диапазона дат, определенного для отчета.     |
|     Оплаты                  |     Сумма проводок платежей по аренде, разнесенных в диапазон дат, определенный для отчета. Значения в этом столбце будут отображаться как отрицательные суммы, поскольку платежи уменьшают сальдо арендных обязательств.  |
|     Интересы                  |     Сумма начислений процентов, разнесенных по аренде в диапазон дат, определенный для отчета.            |
|     Корректировки               |     Сумма разнесенных проводок корректировки по аренде, попадающих в диапазон дат, определенный для отчета.                               |
|     Конечное сальдо            |     Сальдо всех проводок обязательств по аренде, разнесенных на дату **Дата окончания** отчета.                  |

## <a name="lease-transactions-report"></a>Отчет по проводкам аренды
В запросе проводок по аренде отображаются все записи журнала, созданные по аренде активов. Каждая запись в журнале связана с кодом книги, из которой она была получена. Это позволяет легко связать запись в журнале с соответствующей арендой. Запрос о проводках по аренде функционирует аналогично странице **Проводки ваучера** в главной книге.

## <a name="weighted-average-discount-rate-report"></a>Отчет о средне-взвешенном коэффициенте скидки

Чтобы просмотреть этот отчет, перейдите в раздел **Аренда активов > Запросы и отчеты > Раскрытия > Средне-взвешенный коэффициент скидки**. Этот отчет включает следующие поля.

|     Информация о компании                      |     Юридическое лицо, связанное с арендой.                           |
|     ИД аренды                          |     Уникальный идентификатор аренды.                                                  |
|     Описание аренды                 |     Описание аренды из заголовка аренды.                          |

|     Слой разноски                     |     Каждая книга, которая присоединяется к основным средствам, настраивается для определенного слоя разноски, имеющего общее задание на амортизацию.      |
