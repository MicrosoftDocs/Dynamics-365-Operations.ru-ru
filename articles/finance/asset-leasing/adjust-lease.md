# <a name="adjust-leases"></a>Корректировка аренд

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

В этой теме объясняется, как корректировать аренду. Корректировка может потребоваться, если условия аренды изменяются, аренда продляется или другие обстоятельства изменяются. Аренда активов удовлетворяет рекомендациям по изменениям в аренде, приведенным в теме 842 кодификации стандартов учета (ASC 842) и международном стандарте финансовой отчетности 16 (МСФО 16). ASC 842-20-15-1 определяет изменение аренды как любое изменение условий контракта, которые вызывают изменение в области или в отношении аренды. Параграф 39 МСФО 16 указывает, что арендополучатель должен выполнить переоценку арендного обязательства, чтобы оно отражало изменения платежей по аренде.

Для организаций, в которых используется ASC 842 или МСФО 16, аренда заново измеряется для учета изменения в текущей стоимости будущих минимальных арендных платежей (PVFMLP). При увеличении PVFMLP создаваемая запись журнала будет дебетовать счет актива в форме права пользования (ФПП) и кредитовать счет арендного обязательства на разницу между новым PVFMLP и предыдущим PVFMLP. Если PVFMLP уменьшается, запись журнала будет дебетовать счет арендного обязательства и кредитовать счет актива ФПП на разницу.

Если корректировка приводит к уменьшению актива ФПП ниже 0 (нуля), остаток будет кредитоваться на прибыль счета разноски изменения аренды, который указан на странице **Счета разноски по аренде**. Системные счета для этих проводок и другие записи корректировки, такие как изменения классификации, первоначальные прямые затраты, арендные льготы, предоплаты и расходы на демонтаж, возникающие в результате изменений аренды.

Специальные указания относительно проводок корректировки аренды рекомендуется смотреть в МСФО 16 и ASC 842.

## <a name="lease-adjustment-wizard"></a>Мастер корректировки аренды

Выполните следующие шаги, чтобы скорректировать аренду. Измененные данные будут обновлять графики аренды после выполнения разноски с помощью мастера **Корректировка аренды**. Можно сохранить работу и закрыть мастер в любое время, а затем вернуться позже, чтобы возобновить работу.

Чтобы открыть мастер **Корректировка аренды** из сводки аренды, выполните следующие действия.

1. Перейдите в раздел **Аренда активов \> Аренды \> Сводка по аренде**. 
2. Выберите аренду для корректировки, затем выберите **Мастер корректировки**.
3. Следуйте инструкциям мастера для ввода необходимых изменений.

Чтобы открыть мастер **Корректировка аренды** на странице **Корректировка аренды**, для выполнения корректировки, которая уже выполняется, выполните следующие действия.

1.  Перейдите в раздел **Аренда \> Аренды \>Корректировки аренды**.
2.  Выберите аренду со статусом корректировки **Выполняется**, затем выберите **Мастер корректировки**.
3.  В полях **Дата начала изменения** и **Дата разноски** введите соответствующие даты.
4.  Выберите **Далее**.

    Аренда теперь добавляется на страницу **Корректировки аренды**, где можно ввести новые сведения о ней.

    После корректировки аренды к ней применяются поля параметров права пользования. Если первоначальные прямые затраты, арендные льготы, предоплаты или расходы на демонтаж связаны с измененной арендой, следует оставить соответствующие поля пустыми. Исходные суммы не будут применяться к обновленным арендам. 

    Например, арендодатель предоставляет льготу 1 000 долларов США за согласования продления аренды. В этом случае при корректировке аренды на счет для продления введите **1 000** в поле **Арендные льготы, возникающие в результате корректировки**.

    В строках графика платежей в поле **Дата начала изменения** отображаются все платежи с этого месяца и за последующие месяцы. Поскольку изменения являются потенциальными, строки графика оплаты не могут начаться до начала изменения. Чтобы просмотреть строки графика оплаты, начиная с даты начала изменения, перейдите на страницу **История версий аренды**.

8.  Выберите **Далее**.

    На следующей странице показаны ключевые сведения об ожидаемой корректировке аренды, такие как переносимые значения сумм обязательств по аренде до изменения и новые ожидаемые обязательства по аренде после изменения. На странице также отображается предварительный просмотр записи в журнале для ожидаемой корректировки аренды.

9.  Выберите **Отправить в рабочий процесс**, чтобы отправить корректировку аренды в систему рабочего процесса, если рабочий процесс корректировки аренды активен и корректировка еще не была утверждена. Дополнительные сведения об использовании рабочего процесса корректировки аренды см. в разделе [Использование рабочих процессов корректировок аренды](use-create-lease-wrkflw.md).

    > [!NOTE]
    > В этот момент система пересчитывает переменные корректировки для проверки того, что проводки не были разнесены по аренде, так как предварительно были рассчитаны общие сведения о корректировке и запись в журнале корректировки. Если какие-либо значения изменяются, сетка обзора корректировок обновляется, и можно просмотреть сведения до повторной отправки корректировок аренды в систему рабочих процессов.

10. Если рабочий процесс не активен или если корректировка аренды утверждена, выберите **Готово**, чтобы обработать изменения и разнести запись журнала корректировки.

    > [!NOTE] 
    > В этот момент система пересчитывает переменные корректировки для проверки того, что проводки не были разнесены по аренде, так как предварительно были рассчитаны общие сведения о корректировке и запись в журнале корректировки. Если какие-либо значения изменяются, сетка обзора корректировок обновляется, и вы сможете просмотреть изменения до того, как будет выбрано значение **Готово**. Если рабочий процесс активен и корректировка аренды была утверждена, любые изменения, внесенные в обзор корректировок, приводят к тому, что статус утверждения снова изменится на **Не отправлено**. В этом случае необходимо повторно отправить корректировку аренды в систему рабочих процессов.

    На странице **Корректировки аренды** статус корректировки теперь установлен как **Завершено**.

    На странице **Корректировки аренды** все равно можно просмотреть экспресс-вкладки **Обзор корректировки** и **Предварительный просмотр записей корректировки**. Сведения об аренде и дата отображаются в журнале версий этой аренды.

    Новая версия аренды и новый набор графиков будут созданы с использованием измененных сведений. 

13. Чтобы просмотреть новые расписания, перейдите к аренде, затем выберите **Книги**.
14. Для просмотра вновь созданного графика оплаты выберите **График платежей**.
15. Для просмотра нового графика амортизации арендных обязательств, созданного на основе новой информации, закройте страницу **График оплаты** и откройте страницу **График амортизации обязательств по аренде**.
16. Чтобы просмотреть вновь созданный график амортизации актива, откройте страницу **График амортизации актива** на странице **Сведения о книге**.
17. Для просмотра записи журнала корректировки выберите **Журналы \> Журнал аренды активов**.

## <a name="cancel-a-lease-adjustment"></a>Отмена корректировки аренды

Чтобы удалить выполняемую корректировку аренды, выполните следующие действия.

1.  Перейдите в раздел **Аренда \> Аренды \>Корректировки аренды**.
2.  Выберите, чтобы отменить выполняемую корректировку аренды.
3.  Выберите **Отменить корректировку**. 
4.  Нажмите **ОК**.

    > [!NOTE] 
    > При выборе **Отмена** в мастере **Корректировка аренды** выполняется откат последнего изменения, которое было завершено в мастере, но не удаляется выполняемая корректировка.

## <a name="use-the-lease-adjustment-workflow"></a>Использование рабочего потока корректировки аренды

В этом разделе объясняется, как использовать рабочий процесс коррекции аренды. Рабочий процесс корректировки аренды помогает в согласованном порядке управлять корректировками аренды путем предоставления набора шагов утверждения и назначения их тем пользователям, которые утверждают корректировку аренды до того, как она станет окончательной. После утверждения корректировки аренды в рабочем процессе можно воспользоваться мастером **Корректировка аренды**, чтобы выполнить корректировку аренды.

1.  Чтобы отправить корректировку аренды для утверждения, перейдите в раздел **Аренда \> Аренды \> Корректировки аренды**.
2.  Выберите код аренды для корректировки аренды, затем выберите **Мастер корректировки**.
3.  На последней странице мастера выберите **Отправить для утверждения**.
4.  Введите комментарий о корректировке, затем выберите **ОК**.

    Статус рабочего процесса изменится на **Ожидание утверждения**, и все поля в мастере будут заблокированы для редактирования.

5.  Чтобы утвердить корректировку аренды, перейдите в раздел **Аренда \> Аренды \> Корректировки аренды**.
6.  Выберите код аренды для корректировки аренды и проверьте экспресс-вкладки **Обзор корректировки** и **Предварительный просмотр записей корректировки**.
7.  Выберите **Рабочий процесс \> Утверждено**.

    На странице **Корректировки аренды** статус рабочего процесса теперь установлен как **Утверждено**. На этом этапе корректировка аренды готова к разноске через запись журнала корректировки.

8.  Чтобы продолжить обработку корректировки аренды и разнести запись корректировки, перейдите в раздел **Аренда \> Аренды \> Корректировки аренды**.
9.  Выберите код аренды для корректировки аренды, затем выберите **Мастер корректировки**.
10. Выбирайте **Далее**, пока не появится последняя страница мастера, затем выберите **Готово**.

Система пересчитывает текущие значения, используемые при аренде, чтобы гарантировать, что переменные корректировки, которые были утверждены, являются текущими. Если имеются какие-либо изменения, статус утверждения снова устанавливается в **Черновик**, и необходимо повторно отправить корректировку аренды в систему рабочих процессов.

## <a name="view-lease-versions"></a>Просмотр версий аренды

Если аренда была скорректирована, можно просмотреть другие ее версии. Также можно просмотреть исторические графики и сведения о прошлых арендах.

1. На странице **Сводка по аренде** выберите аренду, а затем на панели операций выберите **Журнал версий аренды**.

    Если выбранная аренда скорректирована, на странице **Журнал версий аренды** отображаются другие версии. Исходная аренда помечается как **1**, а последующие версии имеют нумерацию по возрастанию.

    Для выбранной версии аренды можно просмотреть финансовые аналитики, сведения о контракте, местоположение и строки графика оплаты.

2. Для просмотра исторических графиков откройте измененную аренду со страницы **Сводка по аренде**, выберите нужную книгу, а затем на панели операций выберите **Журнал версий книги**.
3. На странице **Версия книги** выберите версию и расписание для просмотра.

## <a name="adjust-a-lease-book"></a>Корректировка книги аренды

Чтобы откорректировать только книгу аренду, выполните следующие действия.

1. Перейдите в раздел **Аренда активов** \> **Аренды** \> **Сводка по аренде**.
2. Выберите и откройте аренду.
3. На странице **Сведения об аренде** выберите **Книги**.
4. На странице **Сведения о книгах** на панели действий в группе **Поддержка** выберите **Скорректировать книгу**. 
5. Удалите строки графика платежей.
6. В поле **Дата изменения аренды** введите дату изменения. Затем рассмотрите возможность удаления всех дополнительных замечаний по активам и задолженности (первоначальные прямые затраты, арендная льгота, предоплата аренды, стоимость разборки и гарантия остаточной стоимости), если таковые имеются. 
7. Чтобы помочь предотвратить неточные расчеты для корректировки аренды, добавьте новые строки графика платежей для новых дат платежей, которые соответствуют дате изменения. 

> [!NOTE] 
> Рекомендуется использовать Мастер **Корректировка аренды** для корректировки аренды. Мастер сокращает число шагов, выполняемых вручную, обеспечивает предварительный просмотр балансов после корректировки и позволяет изменять суммы перед разноской.
