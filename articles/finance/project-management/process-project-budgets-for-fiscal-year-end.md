# <a name="transfer-project-budgets-at-fiscal-year-end"></a>Перенос бюджетов проектов в конце финансового года

[!include [banner](../includes/banner.md)]

При работе с многолетним проектом в конце финансового года может остаться бюджет. Вы можете переместить оставшиеся бюджетные суммы на будущий год и создать сведения о регистре бюджета для этих сумм на связанных счетах главной книги. Перед переносом оставшихся сумм проверьте и проанализируйте оставшиеся бюджетные суммы.

## <a name="review-and-analyze-remaining-budget-amounts"></a>Просмотр и анализ оставшихся бюджетных сумм

Выполните следующие действия, чтобы проверить бюджетные суммы на конце года для проектов, но пока не переносить эти суммы.

1. Выберите **Управление и учет по проектам** > **Периодические** > **Бюджеты** > **Перенос бюджетов**. 
2. На странице **Процесс переноса бюджета проекта** на вкладке **Параметры конца года** убедитесь, что пункт **Перенести суммы остатков бюджета по проекту** не активирован.
3. На вкладке **Параметры** в поле **Год бюджета проекта** выберите финансовый год, для которого требуется просмотреть оставшуюся бюджетную сумму. 
4. В поле **Начало финансового года** выберите начало финансового года, для которого требуется просмотреть оставшуюся бюджетную сумму. 
5. В поле **Из прогнозной модели** выберите **Оставшийся бюджет**. 
6. Чтобы включить проекты, которые удовлетворяют выбранным критериям и не имеют остатков бюджета, установите флажок **Показать нулевые остатки**.  
7. На вкладке **Выберите бюджеты** выберите **Извлечь все бюджеты**, чтобы загрузить все бюджеты, соответствующие выбранным критериям, затем выберите **Обработать**. 
8. Чтобы разработать запрос к базе данных, который будет загружать в эту область определенный набор бюджетов, выберите **Извлечь выбранные бюджеты**.

Для получения дополнительных сведений о конкретной строке в области выберите эту строку, затем выберите **Просмотр сведений о бюджете** или **Просмотр счетов**.

## <a name="carry-forward-remaining-budget-amounts"></a>Перенести суммы остатков бюджета 

После обработки оставшихся бюджетных сумм можно создать проводки в главной книге для сумм, переносимых на следующий год. Чтобы создать проводки главной книги, выполните шаги из раздела [Перенос сумм бюджета и создание проводок главной книги](#carry-forward). 

> [!NOTE]
> Бюджетные суммы, переносимые на следующий год, перемещаются в прогнозную модель, которая выбрана на странице **Прогнозные модели** в качестве прогнозной модели для переноса.  

## <a name="carry-forward-budget-amounts-and-create-general-ledger-transactions"></a><a name="carry-forward"></a>Перенос бюджетных сумм на следующий год с созданием проводок ГК

1.  Выберите **Управление и учет по проектам** > **Периодические** > **Бюджеты** > **Перенос бюджетов**. 
2. На странице **Процесс переноса бюджета проекта** выберите **Конец года**, затем включите параметры **Перенести суммы остатков бюджета по проекту** и **Создать записи регистра бюджета в главной книге**. 
3. На вкладке **Параметры** в группе полей **Параметры проекта** выберите следующее:

   - **Год бюджета проекта**: выберите начало финансового года, для которого требуется просмотреть оставшиеся бюджетные суммы. 
   - **Прибыли и убытки** создайте проводки прибылей и убытков в главной книге. 
   -  **НЗП**: создайте проводки незавершенного производства (НЗП) в главной книге.
   -  **Зарплата**: создайте проводки распределения зарплаты в главной книге. 

5. В группе полей **Главная книга** укажите следующие сведения: 

   - В поле **Начало финансового года** выберите финансовый год, в который требуется переместить оставшиеся бюджетные суммы для проектов. Значение по умолчанию — на год позже значения в поле **Год бюджета проекта**.
   -  В поле **Период переноса** выберите период, для которого требуется создать сведения о регистре бюджета в главной книге. Обычно это первый период в открывающем финансовом году.

6. В группе полей **Копировать из/в** укажите следующие сведения:

   - В поле **Из прогнозной модели** выберите прогнозную модель бюджета по проекту, связанную с оставшимися бюджетными суммами, которые требуется переместить для проектов. 
   - В поле **В бюджетную модель книги учета** выберите модель бюджета ГК, связанную с бюджетными суммами, которые требуется переместить в ГК. 
   -  Чтобы использовать для создаваемых при перемещении бюджетных сумм для проектов проводок ГК валюту реализации проекта, выберите **Валюта продаж для переноса**. Если этот параметр не выбран, в проводках используется валюта учета. 
   -  Выберите **Показать нулевые остатки**, чтобы включить в проекты, по которым нет оставшихся бюджетных сумм, однако которые соответствуют другим критериям, выбранным в нижней панели.

7. На вкладке **Выберите бюджеты** выберите **Извлечь все бюджеты**, чтобы загрузить все бюджеты, соответствующие выбранным критериям. Если вы предпочитаете разработать запрос к базе данных, который будет загружать в эту область определенный набор бюджетов проектов, выберите **Извлечь выбранные бюджеты**.
8. Для каждого проекта, который требуется обработать, выберите параметр в начале строки для проекта.

    > [!TIP]
    > Чтобы выбрать все или большинство проектов, установите флажок в левом верхнем углу. Чтобы исключить обработку любого проекта, снимите флажок для этого проекта.

9. Чтобы переместить оставшиеся бюджетные суммы для выбранных проектов на выбранный финансовый год и создать сведения о регистре бюджета в главной книге, выберите **Обработать**.

## <a name="carry-forward-budget-amounts-without-creating-general-ledger-transactions"></a>Перенос бюджетных сумм без создания проводок ГК

1. Выберите **Управление и учет по проектам** > **Периодические** > **Бюджеты** > **Перенос бюджетов**.
2. На странице **Процесс переноса бюджета проекта** в поле **Параметры конца года** выберите **Перенести суммы остатков бюджета по проекту**.
3. В группе **Параметры** в поле **Год бюджета проекта** выберите финансовый год, для которого требуется просмотреть оставшиеся бюджетные суммы.
4. В группе **Копировать из/в** укажите следующие сведения:

   - В поле **Из прогнозной модели** выберите прогнозную модель бюджета по проекту, связанную с оставшимися бюджетными суммами, которые требуется переместить для проектов. 
   - Выберите **Показать нулевые остатки**, чтобы включить проекты, которые не имеют остатков бюджета, но удовлетворяют другим выбранным критериям.
   - В группе **Выберите бюджеты** выберите **Извлечь все бюджеты**, чтобы загрузить все бюджеты, соответствующие выбранным критериям. Чтобы разработать запрос к базе данных, который будет загружать в эту область определенный набор бюджетов проектов, выберите **Извлечь выбранные бюджеты**.

5. Для каждого проекта, который требуется обработать, выберите параметр в начале строки для проекта. 
6. Выберите **Обработать**, чтобы переместить оставшиеся бюджетные суммы для выбранных проектов в выбранный финансовый год.
