# <a name="eliminate-a-project-estimate"></a>Удаление оценки проекта

[!include [banner](../includes/banner.md)]

Оценки проекта обеспечивают финансовое представление для работы, которая оценивается и планируется для проекта. Для обработки оценок для проекта, следует присоединить проект к проекту НЗП. Проекты НЗП всегда основаны на существующем проекте, однако несколько проектов могут ссылаться на один проект НЗП. К проектам НЗП могут быть привязаны только проекты с фиксированной ценой и инвестиционные проекты, которые должны принадлежать той же группе проектов, что и проект НЗП.

Чтобы удалить оценку проекта, он должен быть завершен. Следующие шаги описывают, как удалить оценку.

1. Перейдите в раздел **Управление и учет по проектам** > **Все проекты** и откройте проект. 
2. На вкладке **Управление** выберите **Оценки**, а на странице **Оценка** выберите **Удалить**.
3. На странице **Удалить оценку** на вкладке **Общие** задайте следующие параметры:

   - **Код периода**: выберите код периода для выбора соответствующих проектов НЗП. 
   - **Дата оценки**: выберите соответствующую дату оценки для удаления.
   - **Удалить с предупреждениями о НЗП**: включите этот параметр, чтобы выдать уведомление, если оценка, связанная с незавершенным производством (НЗП), будет удалена. Если этот параметр не задан, удаление невозможно продолжить, если существуют не оцененные проводки. 
   > [!NOTE]
   > Этот параметр доступен только в том случае, когда закрытие должно применяться к проекту НЗП. Он недоступен при использовании периодических разносок. Этот параметр используется с настройками на вкладке **Оценка** на странице **Параметры проекта** в группе полей **Разрешить удаление, когда существуют неоцененные проводки**.
   - **Задать этап как "Завершено"**: включите этот параметр, чтобы после выполнения удаления этап проекта НЗП был **Завершено**.
   - **Печать списка оценок**: выберите информацию, которую надо будет включить, когда будет напечатан список оценок.
   - **Показать Infolog**: включите этот параметр, чтобы отобразить Infolog.
   - **Дата разноски**: выберите дату разноски ГК для оценки.

4.  Нажмите **ОК**.
5. После завершения процесса удаления отображается отрицательное значение для удаленного проекта НЗП. 

Если не требуется удалять оценку, можно выбрать удаленную оценку и выбрать **Отменить удаление**.
