<span data-ttu-id="721ab-104">После создания и отправки предложения по проекту клиенту необходимо получить подтверждение от клиента перед обновлением статуса предложения до значения **Отправлено**.</span><span class="sxs-lookup"><span data-stu-id="721ab-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="721ab-105">Любые изменения, запрошенные клиентом, могут быть обновлены в предложении.</span><span class="sxs-lookup"><span data-stu-id="721ab-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="721ab-106">После обновления статуса предложения до значения **Отправлено** его невозможно изменить.</span><span class="sxs-lookup"><span data-stu-id="721ab-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="721ab-107">Следующая процедура описывает параметры для отправки предложения на подтверждение, обновления на основе обратной связи и отправки предложения.</span><span class="sxs-lookup"><span data-stu-id="721ab-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

## <a name="send-a-project-quotation-confirmation"></a><span data-ttu-id="721ab-108">Отправка подтверждения предложения по проекту</span><span class="sxs-lookup"><span data-stu-id="721ab-108">Send a project quotation confirmation</span></span>  

<span data-ttu-id="721ab-109">Можно отправить существующее предложение по проекту для подтверждения по электронной почте или в виде печатной копии.</span><span class="sxs-lookup"><span data-stu-id="721ab-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span> 

1. <span data-ttu-id="721ab-110">Перейдите в раздел **Управление и учет по проектам** > **Предложения** > **Предложение по проекту.**</span><span class="sxs-lookup"><span data-stu-id="721ab-110">Go to **Project management and accounting** > **Quotations** > **Project quotation.**</span></span> 
2. <span data-ttu-id="721ab-111">На странице **Предложение по проекту** выберите предложение, которое необходимо отправить на подтверждение.</span><span class="sxs-lookup"><span data-stu-id="721ab-111">On the **Project quotation** page, select the quotation you want to send for confirmation.</span></span> 
3. <span data-ttu-id="721ab-112">На вкладке **Обработка результатов** в группе **Создать** выберите **Подтвердить**.</span><span class="sxs-lookup"><span data-stu-id="721ab-112">On the **Follow up** tab, in the **Generate** group, select **Confirm**.</span></span> 
4. <span data-ttu-id="721ab-113">На странице **Подтвердить предложение** задайте требуемые параметры.</span><span class="sxs-lookup"><span data-stu-id="721ab-113">On the **Confirm quotation** page, set the required parameters.</span></span> <span data-ttu-id="721ab-114">Например, чтобы напечатать подтверждение, в разделе параметров **Печать** включите **Печать подтверждения**, а затем нажмите **ОК**.</span><span class="sxs-lookup"><span data-stu-id="721ab-114">For example, to print the confirmation, under **Print** options, turn on **Print confirmation**, and then select **OK**.</span></span>
5. <span data-ttu-id="721ab-115">На странице **Отправить предложение** выберите **Параметры**, а затем на странице **Настройки назначения печати** выберите, нужно ли отправлять предложение на **экран**, **электронную почту**, в **файл**, **Архив печати** или на **принтер**, а затем выполните необходимые настройки в области **Детализация** для маршрутизации предложения.</span><span class="sxs-lookup"><span data-stu-id="721ab-115">On the **Send quotation** page, select **Options**, and on the **Print destination settings** page, select whether you want the quotation sent to **Screen**, **E-mail**, **File**, **Print archive**, or **Printer**, and then make any adjustments that you want in the **Specification** area to route the quotation.</span></span>
6. <span data-ttu-id="721ab-116">На странице **Настройки назначения печати** выберите **ОК**.</span><span class="sxs-lookup"><span data-stu-id="721ab-116">On the **Print destination settings** page, select **OK**.</span></span>
