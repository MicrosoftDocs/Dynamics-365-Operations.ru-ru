# <a name="confirm-update-and-send-a-project-quotation"></a>Подтверждение, обновление и отправка предложения по проекту

[!include [banner](../includes/banner.md)]

После создания и отправки предложения по проекту клиенту необходимо получить подтверждение от клиента перед обновлением статуса предложения до значения **Отправлено**. Любые изменения, запрошенные клиентом, могут быть обновлены в предложении. После обновления статуса предложения до значения **Отправлено** его невозможно изменить. Следующая процедура описывает параметры для отправки предложения на подтверждение, обновления на основе обратной связи и отправки предложения.

## <a name="send-a-project-quotation-confirmation"></a>Отправка подтверждения предложения по проекту  

Можно отправить существующее предложение по проекту для подтверждения по электронной почте или в виде печатной копии. 

1. Перейдите в раздел **Управление и учет по проектам** > **Предложения** > **Предложение по проекту.** 
2. На странице **Предложение по проекту** выберите предложение, которое необходимо отправить на подтверждение. 
3. На вкладке **Обработка результатов** в группе **Создать** выберите **Подтвердить**. 
4. На странице **Подтвердить предложение** задайте требуемые параметры. Например, чтобы напечатать подтверждение, в разделе параметров **Печать** включите **Печать подтверждения**, а затем нажмите **ОК**.
5. На странице **Отправить предложение** выберите **Параметры**, а затем на странице **Настройки назначения печати** выберите, нужно ли отправлять предложение на **экран**, **электронную почту**, в **файл**, **Архив печати** или на **принтер**, а затем выполните необходимые настройки в области **Детализация** для маршрутизации предложения.
6. На странице **Настройки назначения печати** выберите **ОК**.  

## <a name="update-a-project-quotation"></a>Обновление предложения по проекту

Чтобы изменить существующее предложение по проекту, статус предложения должен быть **Создано**. Выполните следующие действия, чтобы обновить существующее предложение. 

1. Перейдите в раздел **Управление и учет по проектам** > **Предложения** > **Предложения по проекту**.
2. На странице **Предложения по проекту** выберите предложение, которое требуется обновить, и в области действий выберите **Изменить**.
3. Обновите необходимые сведения в поле и на панели операций выберите **Сохранить**.  

## <a name="send-a-project-quotation-to-a-customer"></a>Отправка предложение по проекту клиенту 

1. Перейдите **Управление и учет по проектам** > **Предложения** > **Предложения по проекту** и выберите предложение по проекту, которое необходимо отправить.
2. На странице **Предложения по проекту** на вкладке **Предложение** в группе **Процесс** выберите **Отправить предложение**. Статус предложения обновляется на **Отправлено**.

> [!NOTE]
> Предложение по проекту невозможно изменить после того, как статус изменится на **Отправлено**.
