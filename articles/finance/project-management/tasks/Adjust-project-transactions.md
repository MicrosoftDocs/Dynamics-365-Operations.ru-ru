# <a name="adjust-project-transactions"></a>Корректировка проводок по проекту

[!include [task guide banner](../../includes/task-guide-banner.md)]

В данном руководстве по задаче показано, как корректировать проводку по проекту. Однако можно корректировать проводки по проекту по одной проводке за раз или выбирать из списка всех проводок по проекту. Обычно эта задача выполняется супервизором проекта. В этом руководстве по задаче используется набор данных компании USSI.

1. Перейдите в раздел "Управление и учет по проектам" > "Проекты" > "Все проекты". 
2. В списке пометьте выбранную строку. 
3. Щелкните "Корректировать проводки". 
4. Щелкните Выбрать. 
5. Выберите значение "Да" в поле "Часы". 
6. В поле "Дата окончания" введите дату. 
7. Нажмите кнопку "OК". 
8. В списке пометьте выбранную строку. 
9. Щелкните "Корректировка". 
10. В списке пометьте выбранную строку. 
11. В поле "Свойство строки" введите или выберите значение. 
12. Нажмите кнопку "OК". 
13. В списке пометьте выбранную строку. 
14. В поле "Себестоимость" введите число. 
15. Щелкните "Разнести".
