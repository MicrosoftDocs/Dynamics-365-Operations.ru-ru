# <a name="rename-or-modify-a-project-stage"></a>Переименование или изменение этапа проекта

[!include [banner](../../includes/banner.md)]

Этапы проекта могут служить проводником по жизненному циклу проекта. Определять этапы проекта, которые требуется включать в проекты, можно в области "Этап проекта" на странице "Параметры управления и учета по проектам". Dynamics 365 Finance предоставляет пять заранее определенных этапов проекта и три определяемых пользователем этапа. Можно обновить имя этапа проекта, которое используется для отслеживания хода выполнения проектов. Также можно указать имена для названий определяемый пользователем статусов. Эта задача демонстрирует переименование или изменение этапа проекта, а также использование набора данных USSI.

1. Перейдите в раздел "Управление и учет по проектам" > "Настройка" > «Параметры модуля "Управление и учет по проектам"».
2. Перейдите на вкладку "Этап проекта".
3. В поле "Язык" выберите язык, который требуется использовать для имени нового этапа проекта.
4. В списке выберите предопределенный или определяемый пользователем этап, который нужно изменить. 
5. В поле "Этап проекта" введите новое имя.
6. Нажмите кнопку "Сохранить".
