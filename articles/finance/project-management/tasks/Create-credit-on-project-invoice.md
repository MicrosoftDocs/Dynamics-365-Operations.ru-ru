# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="2bdcb-103">Создание кредит-нот для накладных по проекту</span><span class="sxs-lookup"><span data-stu-id="2bdcb-103">Create a credit note on project invoices</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="2bdcb-104">Перейдите в раздел "Управление и учет по проектам" > "Проекты" > "Все проекты".</span><span class="sxs-lookup"><span data-stu-id="2bdcb-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="2bdcb-105">В списке найдите и выберите требуемую запись.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="2bdcb-106">Выберите код проекта, который используется для кредит-ноты.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="2bdcb-107">В списке перейдите по ссылке в выбранной строке.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="2bdcb-108">В области действий щелкните "Управлять".</span><span class="sxs-lookup"><span data-stu-id="2bdcb-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="2bdcb-109">Щелкните "Журналы накладных".</span><span class="sxs-lookup"><span data-stu-id="2bdcb-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="2bdcb-110">Выберите проводку для кредит-ноты.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-110">Select the transaction for credit note.</span></span> <span data-ttu-id="2bdcb-111">В списке пометьте выбранную строку.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="2bdcb-112">Щелкните Функции.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-112">Click Functions.</span></span> 
8. <span data-ttu-id="2bdcb-113">Щелкните "Выбрать" для кредит-ноты.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="2bdcb-114">Установите флажок, чтобы выбрать проводку для кредит-ноты.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="2bdcb-115">Нажмите кнопку "OК".</span><span class="sxs-lookup"><span data-stu-id="2bdcb-115">Click OK.</span></span> 

<span data-ttu-id="2bdcb-116">После этого этапа кредит-нота будет доступна для предложения по накладной.</span><span class="sxs-lookup"><span data-stu-id="2bdcb-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
