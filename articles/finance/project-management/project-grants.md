# <a name="project-grants"></a>Гранты по проекту

[!include [banner](../includes/banner.md)]

Грант — это передаваемая безвозмездно сумма денег на конкретную цель или проект. Как правило, есть ограничения на то, как можно потратить грант. В управлении и учете по проектам можно ввести и отслеживать гранты и определять их связи с проектами и контрактами по проекту. Также можно выполнить следующие задачи:

- Свяжите гранты с проектами и источниками финансирования, используя ссылки на страницы **Проект** и **Сведения о контрактах по проекту**.
- Вводите и отслеживайте изменения гранта при его переходе из одного статуса в другой.
- Вводите и отслеживайте стоимости, запрошенные суммы и предоставленные суммы.
- Создание основных грантов и связь с ними дополнительных грантов.

Можно создать грант путем ввода всех сведений в новую запись или можно копировать сведения из существующего гранта и затем обновлять их.

## <a name="create-a-new-grant"></a>Создать новый грант

1. Выберите **Управление и учет по проектам** \> **Гранты** \> **Гранты**.
2. Выберите **Создать** \> **Грант**.
3. На странице сведений о гранте на экспресс- вкладке **Общие** в поле **Код гранта** введите алфавитно-цифровой код для гранта.
4. В поле **Имя гранта** введите имя гранта.
5. В поле **Описание** добавьте сведения о новом гранте.

    Большинство других полей на этой странице не являются обязательными, и можно ввести столько информации, сколько нужно.

    В следующем списке описываются сведения, указанные на каждой экспресс-вкладке страницы сведений о гранте:

    - **Общее** — введите имя, статус, описании, цель и сумму гранта.
    - **Контактная информация** — введите подробные сведения о членах персонала, подразделении, которое управляет грантом, и о клиенте или организации гранта, являющемся источником гранта. Можно также указать, является ли ваша организация транзитным субъектом, принимающим грант и передающим его другому получателю. Выберите **Добавить**, чтобы добавить сведения о контакте, например, номера телефона и адреса электронной почты для организации, которая предоставляет грант.
    - **Даты и крайние сроки** — введите дату, связанную с грантом и заявкой на грант. Примерами являются срок подачи заявления, дата отправки и дата утверждения или отклонения гранта.
    - **Связанные проекты и контракты по проектам** — можно ввести информацию на этой экспресс-вкладке только в том случае, если в поле **Статус гранта** на вкладке **Общее** выбрано значение **Активно** или **Выдано**. Выберите один из следующих вариантов, чтобы выполнить соответствующую задачу:

        - **Добавить источник финансирования** — добавьте новый источник финансирования для гранта. Можно ввести все сведения теперь или можно использовать сведения по умолчанию и затем обновить их позже.
        - **Добавить контракт по проекту** — добавьте или обновите сведения о контракте по проекту.
        - **Добавить проект** — добавление или обновление сведений о проекте.

    - **Настройка** — введите сведения о соответствующих фондах, если эти сведения необходимы. Многие организации, дающие гранты, требуют, чтобы получатели потратили определенную сумму собственных средств или ресурсов, дополняющую сумму гранта. В поле **Локальный проект или код отслеживания** можно ввести код, отличный от идентификатора проекта.

        > [!NOTE]
        > Если часть гранта будет выделена другой организации, установите для параметра **Дополнительный грантодатель** значение **Да**. Для грантов, выданных в США, можно указать, будет ли грант предоставлен в рамках предписания штата или федерального предписания.

    - **Сведения об использовании** — добавление или обновление сведений о том, как часто можно снимать, выставлять счета или тратить средства гранта.

## <a name="create-a-new-grant-from-a-copy"></a>Создание нового гранта из копии

1. Выберите **Управление и учет по проектам** \> **Гранты** \> **Гранты**.
2. Выберите **Создать** \> **Копировать из гранта**.
3. Введите буквенно-цифровой код и имя для нового гранта, а затем выберите **ОК**.
4. На странице сведений о грантах просмотрите сведения о скопированных грантах и внесите все необходимые изменения. Большинство полей на странице являются необязательными.

## <a name="view-or-modify-grant-details"></a>Просмотр или изменение сведений о гранте

1. Выберите **Управление и учет по проектам** \> **Гранты** \> **Гранты**.
2. Выберите изменяемый грант.
3. В области действий на вкладке **Грант** в группе **Ведение** выберите **Изменить**.
4. Просмотрите сведения о гранте и внесите необходимые изменения.
