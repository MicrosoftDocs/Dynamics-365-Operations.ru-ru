# <a name="project-mobile-applications-overview"></a>Обзор мобильных приложений Project

[!include [banner](../includes/banner.md)]

Доступны три мобильных приложения, относящихся ко времени проекта, для Microsoft Dynamics 365 Project Timesheet 365, Регистрация времени проекта и Мои табели учета рабочего времени/Табели учета рабочего времени для просмотра мною (оптимизированные для мобильных устройств).

## <a name="project-timesheet-mobile-app"></a>Мобильное приложение табеля учета рабочего времени проекта Project Timesheet

Мобильное приложение Dynamics 365 Project Timesheet позволяет пользователям отправлять и утверждать табели учета рабочего времени для проектов на своем мобильном устройстве. Это мобильное приложение использует функциональную возможность табеля учета рабочего времени в области управления и учета проекта в Dynamics 365 Finance. Это помогает повысить производительность и эффективность труда пользователей, и также помогает своевременно вводить и утверждать табели учета рабочего времени для проекта.

## <a name="project-time-entry-workspace"></a>Рабочая область регистрации времени по проектам

Рабочая область регистрации времени по проектам позволяет пользователям вводить и сохранять время по проекту. Однако она не позволяет пользователям отправлять или утверждать табели учета рабочего времени для проекта. Она может использоваться только для ввода времени. Эту мобильную рабочую область можно использовать вместе с мобильным приложением Finance and Operations.

## <a name="my-timesheetstimesheets-for-my-review"></a>Мои табели учета рабочего времени/Табели учета рабочего времени для просмотра мною

Мои табели учета рабочего времени (оптимизированные для мобильных устройств) и табели учета рабочего времени для просмотра мною (оптимизированные для мобильных устройств) составляют решение, которое позволяет сотрудникам создавать, отправлять и утверждать табели учета рабочего времени проекта на мобильном устройстве. Работники могут получить доступ к функциям Finance через URL-адрес для приложения. Это решение не требует приложения Finance and Operations.
