# <a name="resource-fulfillment-for-planned-resources"></a>Обеспечение ресурсов для запланированных ресурсов

[!include [banner](../includes/banner.md)]

Менеджер проекта может запланировать необходимые роли ресурсов для проекта. Менеджер по ресурсам видит эти запланированные ресурсы в виде запросов на странице **Обеспечение ресурса** и может назначить фактические ресурсы.

1. На странице **Все проекты** выберите проект **Этап 2 обновления XYZ**.
2. Выберите **Проект**, а затем выберите **Изменить**.
3. На вкладке **Проектная группа и планирование** выберите **Добавить**.
4. В диалоговом окне **Добавить роли** выберите роль **Разработчик программного обеспечения**.
5. Выберите **Создать** и закройте страницу проекта.
6. На странице **Обеспечение ресурса** выберите **Разработчик программного обеспечения 1** для проекта **Этап 2 проекта обновления XYZ**.
7. Выберите работника, а затем выберите **Назначить**.
8. Убедитесь, что строка для **Разработчик программного обеспечения 1** удалена из проекта **Этап 2 проекта обновления XYZ**.
9. На вкладке **Проектная группа и планирование** для проекта **Этап 2 обновления XYZ** убедитесь, что работник, выбранный на предыдущем шаге, добавлен как **Разработчик программного обеспечения**.
