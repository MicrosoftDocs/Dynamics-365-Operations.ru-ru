# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="6506e-103">Обеспечение ресурсов для запланированных ресурсов</span><span class="sxs-lookup"><span data-stu-id="6506e-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="6506e-104">Менеджер проекта может запланировать необходимые роли ресурсов для проекта.</span><span class="sxs-lookup"><span data-stu-id="6506e-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="6506e-105">Менеджер по ресурсам видит эти запланированные ресурсы в виде запросов на странице **Обеспечение ресурса** и может назначить фактические ресурсы.</span><span class="sxs-lookup"><span data-stu-id="6506e-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="6506e-106">На странице **Все проекты** выберите проект **Этап 2 обновления XYZ**.</span><span class="sxs-lookup"><span data-stu-id="6506e-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="6506e-107">Выберите **Проект**, а затем выберите **Изменить**.</span><span class="sxs-lookup"><span data-stu-id="6506e-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="6506e-108">На вкладке **Проектная группа и планирование** выберите **Добавить**.</span><span class="sxs-lookup"><span data-stu-id="6506e-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="6506e-109">В диалоговом окне **Добавить роли** выберите роль **Разработчик программного обеспечения**.</span><span class="sxs-lookup"><span data-stu-id="6506e-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="6506e-110">Выберите **Создать** и закройте страницу проекта.</span><span class="sxs-lookup"><span data-stu-id="6506e-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="6506e-111">На странице **Обеспечение ресурса** выберите **Разработчик программного обеспечения 1** для проекта **Этап 2 проекта обновления XYZ**.</span><span class="sxs-lookup"><span data-stu-id="6506e-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="6506e-112">Выберите работника, а затем выберите **Назначить**.</span><span class="sxs-lookup"><span data-stu-id="6506e-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="6506e-113">Убедитесь, что строка для **Разработчик программного обеспечения 1** удалена из проекта **Этап 2 проекта обновления XYZ**.</span><span class="sxs-lookup"><span data-stu-id="6506e-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="6506e-114">На вкладке **Проектная группа и планирование** для проекта **Этап 2 обновления XYZ** убедитесь, что работник, выбранный на предыдущем шаге, добавлен как **Разработчик программного обеспечения**.</span><span class="sxs-lookup"><span data-stu-id="6506e-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
