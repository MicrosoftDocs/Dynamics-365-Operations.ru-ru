## <a name="set-up-competencies-certificates"></a><span data-ttu-id="925e7-112">Настройка компетенций: сертификаты</span><span class="sxs-lookup"><span data-stu-id="925e7-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="925e7-113">На странице списка **Работники** выберите строку работника, для которого нужно добавить сведения о сертификатах.</span><span class="sxs-lookup"><span data-stu-id="925e7-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="925e7-114">В области действий на вкладке **Работник** в группе **Компетенции** выберите **Сертификаты**.</span><span class="sxs-lookup"><span data-stu-id="925e7-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>
3. <span data-ttu-id="925e7-115">Выберите **Создать**, а затем в поле **Тип сертификата** выберите **PMP**.</span><span class="sxs-lookup"><span data-stu-id="925e7-115">Select **New**, and then, in the **Certificate type** field, select **PMP**.</span></span>
4. <span data-ttu-id="925e7-116">В поле **Дата начала** выберите **01/10/2015** и выберите **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="925e7-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="925e7-117">Настройка компетенций: навыки</span><span class="sxs-lookup"><span data-stu-id="925e7-117">Set up competencies: Skills</span></span>

1. <span data-ttu-id="925e7-118">На странице списка **Работники** убедитесь, что все еще выбран работник, который использовался в предыдущей процедуре.</span><span class="sxs-lookup"><span data-stu-id="925e7-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="925e7-119">Затем в области действий на вкладке **Работник** в группе **Компетенции** выберите **Навыки**.</span><span class="sxs-lookup"><span data-stu-id="925e7-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>
2. <span data-ttu-id="925e7-120">Выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="925e7-120">Select **New**.</span></span>
3. <span data-ttu-id="925e7-121">В поле **Навык** выберите **Управление проектами**.</span><span class="sxs-lookup"><span data-stu-id="925e7-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="925e7-122">В поле **Уровень** выберите **5 Эксперт**.</span><span class="sxs-lookup"><span data-stu-id="925e7-122">In the **Level** field, select **5 Expert**.</span></span>
5. <span data-ttu-id="925e7-123">В поле **Дата уровня** выберите **14.1-.2014**.</span><span class="sxs-lookup"><span data-stu-id="925e7-123">In the **Level date** field, select **1-/14/2014**.</span></span>
6. <span data-ttu-id="925e7-124">В поле **Опыт в годах** введите **10**.</span><span class="sxs-lookup"><span data-stu-id="925e7-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="925e7-125">Выберите **Сохранить** и закройте страницу.</span><span class="sxs-lookup"><span data-stu-id="925e7-125">Select **Save**, and then close the page.</span></span>
