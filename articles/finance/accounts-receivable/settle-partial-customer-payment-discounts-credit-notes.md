# <a name="settle-a-partial-customer-payment-that-has-discounts-on-credit-notes"></a>Сопоставление частичного платежа клиента, имеющего скидки на кредит-ноты

[!include [banner](../includes/banner.md)]

В этой статье рассматривается сценарий, в котором скидка по оплате используется для кредит-ноты, когда исходная накладная также имела скидку по оплате. 

Fabrikam позволяет клиентам получать скидки по оплате при частичных платежах, а также с кредит-нотами (кредитными авизо). Скидку по оплате можно получить по кредит-ноте, если кредит-нота выдана для накладной, по которой клиент получил скидку по оплате. Вместо предоставления кредита на полную сумму вы можете кредитовать сальдо клиента на сумму, которая не включает процент скидки по оплате, полученную клиентом. Параметры сопоставления расположены на странице **Параметры модуля расчетов с клиентами**.

## <a name="invoice-and-credit-note"></a>Накладная и кредит-нота
Клиент 4035 имеет счет на 1 000,00 и кредит-ноту на 100,00. По каждому документу предоставляется скидка 1 % при оплате в течение 14 дней. Арни может просмотреть эту информацию на странице **Проводки по клиенту**.

| Операция    | Тип транзакции | Дата      | Счет  | Дебетовая сумма в валюте проводки | Сумма кредита в валюте проводки | Сальдо  | Валюта |
|------------|------------------|-----------|----------|--------------------------------------|---------------------------------------|----------|----------|
| FTI-10050  | Накладная          | 28.06.2020 | 10050    | 1,000.00                             |                                       | 1,000.00 | USD      |
| CCRN-10050 | Кредит-нота      | 28.06.2020 | CR-10050 |                                      | 100.00                                | -100,00  | USD      |

## <a name="settle-a-credit-note-with-an-invoice"></a>Сопоставление кредит-ноты с накладной
Со страницы **Проводки по клиенту** Арни открывает страницу **Сопоставление проводок**. Арни может использовать страницу **Сопоставление проводок**, чтобы сопоставить накладную и кредит-ноту. В рамках процесса сопоставления Арни просматривает даты и суммы скидки по оплате. Арни помечает два документа, а затем щелкает **Разнести**, чтобы сопоставить проводки. Используется скидка -1,00 на кредит-ноту, потому что Fabrikam разрешает использовать скидки на кредит-ноты.

| Пометка     | Использовать скидку по оплате | Операция    | Учетная запись | Дата      | Срок выполнения  | Счет  | Сумма в валюте проводки | Валюта | Сумма сопоставления |
|----------|-------------------|------------|---------|-----------|-----------|----------|--------------------------------|----------|------------------|
| Выбрано | Обычное округление            | FTI-10050  | 4035    | 28.06.2020 | 28.07.2020 | 10050    | 1,000.00                       | USD      | 990.00           |
| Выбрано | Обычное округление            | CCRN-10050 | 4035    | 28.06.2020 | 28.07.2020 | CR-10050 | -100,00                        | USD      | -99,00           |

Данные по скидкам появляются внизу страницы **Сопоставление проводок**.

- **Дата скидки по оплате**: 12.07.2020 
- **Сумма скидки по оплате**: -1,00     
- **Использовать скидку по оплате**: Обычный    
- **Скидка по оплате**: 0,00      
- **Сумма скидки по оплате**: -1,00     

Сопоставление будет равно 100,00 и будет включать платеж 99,00 и скидку 1,00.
