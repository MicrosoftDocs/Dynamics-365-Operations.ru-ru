# <a name="customer-posting-profiles"></a>Профили разноски по клиенту

[!include [banner](../includes/banner.md)]

В этой теме описываются профили разноски по клиенту, которые управляют разноской проводок клиента в ГК.

## <a name="customer-posting-profiles"></a>Профили разноски по клиенту

Профили разноски клиентов позволяют назначить счета ГК и настройки документов всем клиентам, группе клиентов или отдельным клиентам. Эти настройки будут использоваться для создания накладных по заказам на продажу, накладных с произвольным текстом, накладных по проекту, журналов платежей, писем-напоминаний и процент-нот. 

Профиль разноски по умолчанию определен на вкладке **Главная книга и налог** на странице **Параметры расчетов с клиентами**. Затем он автоматически включается в заголовок новых документов. Его можно изменить, если требуется другой профиль разноски. 

Организации, которые принимают предоплату от клиентов, часто настраивают второй профиль разноски для предоплаты и связывают его в параметрах как профиль разноски по умолчанию для предоплаты. Дополнительные сведения см. в разделе [Предоплата клиентов](customer-prepayments.md).

Также можно связать определения разноски с типами разноски проводок на странице **Определения разноски проводок**. Определения разноски используются вместо профилей разноски для управления разноской проводок клиента в ГК. Дополнительные сведения см. в разделе [Определения разноски](../general-ledger/posting-definitions.md).

## <a name="creating-a-posting-profile"></a>Создание профиля разноски
Укажите счета ГК, используемые для разноски проводок с выбранным профилем разноски. Выбор кода счета и там, где это возможно, счета или группы номеров для выбранного профиля разноски. В процессе разноски будет выполняться поиск наиболее подходящего профиля разноски для каждой проводки посредством поиска самого подходящего кода счета, номера счета или комбинации номеров и группы в соответствии со следующим приоритетом.

| Значение поля Код счета | Значение поля Номер счета/группы                | Приоритет поиска |
|--------------------------|-------------------------------------------------|-----------------|
| Таблица                    | Определенный счет клиента                       | 1               |
| Групповое                    | Группа клиентов, назначенная клиенту | 2               |
| Все                      | Пустой                                           | 3               |

Если необходимо всем проводкам клиентов присвоить одинаковый профиль разноски, настройте только один профиль разности с указанием значения **Все** в поле **Код счета**. Определите следующие значения, чтобы настроить профиль разноски.

### <a name="table-restrictions"></a>Табличные ограничения

Для проводок с выбранным профилем разноски укажите, будут ли проводки сопоставляться автоматически, будет ли рассчитываться процент и будут ли отправляться письма-напоминания. Также можно выбрать счет, используемый при закрытии проводок с выбранным профилем разноски.

Определите следующие значения, чтобы настроить профиль разноски:

| Поле                 | Описание                                           |
|-----------------------|-------------------------------------------------------|
| Населенный пункт        | Используйте этот переключатель, чтобы включить автоматическое сопоставление проводок с этим профилем разноски. Если этот переключатель не установлен, необходимо вручную сопоставить проводки на странице **Сопоставление открытых проводок** или на странице **Ввод платежей клиента**. |
| Интересы          | Установите этот переключатель, если следует рассчитывать процент по остаточным сальдо для счетов клиентов с этим профилем. Если этот переключатель не установлен, процент не будет рассчитываться для этих клиентов.                                           |
| Письмо-напоминание | Установите этот переключатель, если следует создавать письма-напоминания для счетов клиентов с этим профилем. Если этот переключатель не установлен, письма-напоминания не будут создаваться для этих клиентов.                                                 |
| Закрыть             | Выберите профиль разноски, на который нужно переключиться после закрытия проводок с этим профилем разноски. Проводка считается закрытой, если она полностью сопоставлена.             |



Дополнительные сведения см. в разделе [Обзор платежей клиентов](../cash-bank-management/tasks/customer-payment-overview.md).
