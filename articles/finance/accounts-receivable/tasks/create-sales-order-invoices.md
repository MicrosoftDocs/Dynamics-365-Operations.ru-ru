# <a name="create-sales-order-invoices"></a>Создание накладных по заказам на продажу

[!include [banner](../../includes/banner.md)]

В этой теме по задаче описывается процесс выставления накладных по заказу на продажу, включая объединения накладных и пакетную обработку. В данной процедуре используется демонстрационная компания USMF.


## <a name="create-an-invoice-from-a-sales-order"></a>Создание накладной из заказа на продажу
1. Перейдите к **Область перехода > Модули > Расчеты с клиентами > Заказы > Отправленные заказы на продажу, для которых не получены накладные**.
2. Выберите заказ на продажу в списке. 
3. В области **Панель операций** щелкните **Накладная > Создать > Накладная**. Обратите внимание, что этот заказ на продажу имеет несколько отборочных накладных, связанных с ним. Будет отображаться только слово *несколько* вместо числа отборочных накладных.  
4. Разверните раздел **Параметры**.
    - Для разноски необходимо установить значение "Да", чтобы разнести накладную. Также можно отключить разноску и просто напечатать накладную. Однако этот же результат можно получить, создав проформу вместо накладной.  
    - Этот параметр используется для пакетных заданий. Запрос выполняется при выполнении пакетного задания.
5. В поле **Печать** выберите "После".
6. Выберите значение **Да** для параметра **Печать накладной**. С помощью функции управления печатью можно напечатать несколько копий накладной, а также отправить накладную по электронной почте в формате PDF.  
7. В поле **Печать расходов** выберите значение "Суммировать".
8. В поле **Проверка кредитного лимита** выберите значение "Сальдо".
9. Щелкните **Отмена**.

## <a name="combine-orders-into-a-single-invoice"></a>Объединение заказов в одну накладную
1. Перейдите к **Область перехода > Модули > Расчеты с клиентами > Заказы > Все заказы на продажу**.
2. Найдите клиента, для которого открыто несколько накладных.
3. Выберите несколько открытых заказов на продажу для одного и того же клиента.
4. В области **Панель операций** щелкните **Накладная > Создать > Накладная**.
5. Разверните раздел **Параметры**.
6. В поле **Количество** выберите значение "Все". Обратите внимание, что в разделе обзора представлено две накладные. Объединим их в одну накладную.  
7. В поле **Суммарная обработка для** выберите значение "Счет в накладной".
8. Щелкните **Упорядочить**, чтобы объединить заказы на продажу в одну накладную. Два заказа на продажу объединены в одну накладную.   
9. Щелкните **Отмена**.
10. Нажмите кнопку **Да**.

## <a name="post-invoices-in-a-batch"></a>Разноска накладных в партии
1. Выберите **Область переходов > Модули > Расчеты с клиентами > Накладные > Пакетное выставление накладных > Накладная**.
2. Щелкните **Выбрать**.
3. Щелкните **OK**.
4. Щелкните **Партия**.
5. В поле **Пакетная обработка** выберите **Да**.
6. Щелкните **Повторение**.
7. Выберите **Дни**.
8. Щелкните **OK**.
9. Щелкните **OK**.
10. Щелкните **Отмена**.
11. Нажмите кнопку **Да**.
