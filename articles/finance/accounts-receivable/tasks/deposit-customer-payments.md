# <a name="deposit-customer-payments"></a>Внесение платежей клиентов

[!include [banner](../../includes/banner.md)]

Депонируйте платежи клиента. В этой задаче используется демонстрационная компания USMF.

1. Выберите **Область перехода > Модули > Расчеты с клиентами > Платежи > Журнал платежей**.
2. Выберите **Создать**.
3. В поле **Имя** выберите **CustPay** в раскрывающемся меню.
4. Выберите **Строки**.
5. В поле **Счет** выберите клиента, для которого записывается платеж.
6. В поле **Кредит** введите сумму платежа. Можно оставить сумму незаполненной, чтобы система рассчитывала ее путем выбора оплаченных накладных.  
7. В поле **Ссылка на платеж** введите значение. Ссылкой на платеж может быть номер чека для вводимого платежа. Ссылка на платеж требуется для включения платежа в бланк депозита.  
8. Установите флажок "Использовать бланк депозита". Если платеж должен быть включен в депозит, измените значение этого параметра на "Да".  
9. Выберите **Создать**.
10. В поле **Счет** выберите клиента для следующего платежа.
11. В поле **Кредит** введите сумму платежа.
12. В поле **Ссылка на платеж** введите значение.
13. Установите флажок **Использовать бланк депозита**.
14. Выберите **Разнести**. Платежи необходимо разнести до создания бланка депозита. Это гарантирует, что платежи не изменятся после создания бланка депозита.  
15. Выберите **Функции**.
16. Выберите **Бланк депозита**.
17. Нажмите **ОК**. Первая страница используется для создания бланка депозита.  
18. Нажмите **ОК**. Второй этап — печать бланка депозита, но этот шаг не является обязательным.
