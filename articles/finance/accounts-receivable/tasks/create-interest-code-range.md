# <a name="create-an-interest-code-with-a-range"></a>Создание кода процента с диапазоном

[!include [banner](../../includes/banner.md)]
Коды процента можно настроить для расчета различных сумм процента на основе диапазона значений. В этой процедуре показано, как добавить код процента и добавить в него диапазон.

1. Перейдите в раздел **Кредит и коллекции > Процент > Настройка кодов процентов**.
2. Нажмите кнопку **Создать**.
3. В поле **Код процента** введите полное имя кода процента.
4. В поле **Описание** введите описание кода процента.
5. Выберите **Месяц**.
6. Разверните раздел **Доход**.
7. Разверните раздел **Прибыль по валюте**.
8. В поле **Счет для разноски ГК** укажите требуемые значения.
9. В поле **Проценты по диапазону** выберите "Месяцы".
10. Нажмите кнопку **Добавить**.
11. В поле **Описание** введите описание текущей валюты и диапазона.
12. Нажмите кнопку **Сохранить**.
13. Щелкните **Диапазоны**.
14. Нажмите кнопку **Создать**.
15. Введите 0 в поле **Со значения**, а затем введите процент в месяц, который будет использоваться для расчета процента. В нашем примере он равен 1,5.
16. Нажмите кнопку **Создать**.
17. Введите следующее значение "С" как 4, что означает первый месяц, когда будет рассчитываться новый уровень процентов.
18. Введите процент в месяц, который будет использоваться для расчета процента начиная с месяца 4.  В этом примере он равен 2,0.
19. Нажмите кнопку **Создать**.
20. Введите следующее значение "С" как 7, что означает следующий месяц, с которого будет рассчитываться новый уровень процентов.
21. Введите процент в месяц, который будет использоваться для расчета процента начиная с месяца 7. В этом примере он равен 2,5.
22. Щелкните **Закрыть**, чтобы завершить настройку.
