# <a name="generate-and-post-recurring-free-text-invoices"></a>Формирование и разноска повторяющихся накладных с произвольным текстом

[!include [banner](../../includes/banner.md)]

Периодические накладные используются для выставления счетов клиентам регулярно на одни и те же суммы. В данной записи используется демонстрационная компания USMF. Запись предназначена для пользователя, ответственного за обработку накладных расчетов с клиентами и управление ими.


## <a name="generate-recurring-invoices"></a>Создавать периодические счета

## <a name="post-recurring-invoices"></a>Разноска периодических накладных
1. Перейдите в раздел **Расчеты с клиентами > Накладные > Периодические накладные > Разноска периодических накладных**.
    * Используйте эту страницу для просмотра и печати повторяющихся накладных, которые уже были созданы.  
2. В списке перейдите по ссылке в выбранной строке.
    * Выберите группу периодических накладных.  
3. Щелкните **Итоги**.
    * Проверьте итоги для группы периодических накладных.  
4. Нажмите кнопку **Закрыть**.
    * Каждая строка ниже является периодической накладной с произвольным текстом. Можно выбрать строку и нажать кнопку **Сведения**, чтобы просмотреть сведения о накладной с произвольным текстом.  
5. Щелкните **Проверить**.
    * Убедитесь, что выбранные накладные не содержат ошибок, но не разносите накладные.  
6. Щелкните **Разнести**.
    * Разнесите выбранные накладные.
