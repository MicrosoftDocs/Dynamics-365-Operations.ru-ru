# <a name="copy-a-billing-schedule"></a>Копирование графика выставления счетов

[!include [banner](../includes/banner.md)]

Можно использовать страницу **Копировать график** для создания нового графика выставления счетов из существующего графика выставления счетов. Новый график выставления счетов будет иметь те же параметры, что и исходный график выставления счетов. Можно также использовать страницу для копирования номенклатур из одного графика выставления счетов в другой.

## <a name="copy-a-billing-schedule"></a>Копирование графика выставления счетов

Для копирования графика выставления счетов выполните следующие действия.

1. На странице списка **Все графики выставления счетов** или **Активные графики выставления счетов** выберите существующий график выставления счетов.
2. На вкладке **График выставления счетов** выберите **Копировать график**.
3. Выберите счет клиента в поле **Счет клиента**.
4. При необходимости обновите даты начала и окончания выставления счетов
5. При необходимости обновите значения следующих дополнительных полей: **Счет конечного пользователя**, **Счет партнера**, **Ссылка на клиента**, **Заявка клиента** и **Дата начала РБП**.
6. Выберите **Предварительный просмотр**, чтобы просмотреть и выбрать строки графика выставления счетов, которые требуется скопировать.
7. Выберите **Копировать**.

После завершения процесса копирования новый график выставления счетов появится в нижней части страницы списка **Все графики выставления счетов** или **Активные графики выставления счетов**, а также на странице **Графики выставления счетов**.

## <a name="copy-items-from-one-billing-schedule-to-another"></a>Копирование элементов из одного графика выставления счетов в другой

1. На странице списка **Все графики выставления счетов** или **Активные графики выставления счетов** выберите существующий график выставления счетов.
2. На вкладке **График выставления счетов** выберите **Копировать график**.
3. Выберите счет клиента в поле **Счет клиента**.
4. В поле **Код графика выставления счетов** выберите код графика выставления счетов.
4. Обновите даты начала и окончания выставления счетов.
5. При необходимости обновите значения следующих полей: **Счет конечного пользователя**, **Счет партнера**, **Ссылка на клиента**, **Заявка клиента** и **Дата начала РБП**.
6. Выберите **Предварительный просмотр**, чтобы просмотреть и выбрать строки графика выставления счетов, которые требуется скопировать в существующий график выставления счетов.
7. Выберите **Копировать**.

После завершения процесса копирования проверьте график выставления счетов, в который были скопированы элементы, и убедитесь, что элементы были добавлены так, как ожидалось.
