# <a name="subscription-billing-overview"></a>Обзор выставления счетов по подпискам

[!include [banner](../includes/banner.md)]

Выставление счетов по подпискам позволяет организациям управлять возможностями выручки по подпискам и периодическим выставлением счетов через графики выставления счетов. Сложными моделями ценообразования и выставления счетов и распределением выручки легко управлять, и они выставляются и распознаются на уровне строк. Распределение выручки с несколькими элементами позволяет распределить выручку в соответствии с международными стандартами бухгалтерского учета (Стандарт международной финансовой отчетности 15 \[МСФО 15\]) и стандартами общепринятых принципов учета в США (US GAAP) (тема 606 кодификации бухгалтерских стандартов \[ASC 606\]).

В решении имеются три модуля, которые могут использоваться независимо. Кроме того, все три модуля могут использоваться вместе.

- **Периодическое выставление счетов по контракту** — этот модуль обеспечивает управление периодическим выставлением счетов и ценами для обеспечения управления ценами и параметрами выставления счетов, обновления контракта и консолидированного выставления накладных.
- **Выручка и расходы будущих периодов** — этот модуль исключает ручные процессы и зависимости внешних систем путем управления выручкой и включения аналитики в реальном времени в месячную повторяющуюся выручку.
- **Распределение выручки из нескольких элементов** — этот модуль помогает в обеспечении соответствия выручки путем обработки ценообразования и распределения выручки по нескольким номенклатурам.

Дополнительные сведения о выставлении счетов по подписке см. в разделе [Содержимое Power BI выставления счетов по подписке](sub-bill-power-bi.md).

Выставление счетов по подпискам включается через **Управление функциями**. Однако его нельзя использовать с функцией **Признание выручки**. Поэтому следует отключить эту функцию перед тем, как включить выставление счетов по подпискам.

1. В рабочей области **Управление функциями** на вкладке **Все** введите **Признание выручки** в фильтре, затем выберите имя функции в качестве фильтра.
2. Выберите функцию **Признание выручки**, затем выберите **Отключить**.
3. В фильтре **Имя функции** введите **Выставление счетов по подпискам**, затем выберите фильтр модулей.
4. Выберите функцию **Выставление счетов по подпискам**, затем выберите **Включить**.
5. Выберите один из трех модулей из предыдущего списка и затем выберите **Включить**. Повторите этот шаг для каждого из двух остальных модулей.

    > [!IMPORTANT]
    > Прежде чем можно будет включить любой из трех модулей, должна быть включена функция **Выставление счетов по подпискам**.
