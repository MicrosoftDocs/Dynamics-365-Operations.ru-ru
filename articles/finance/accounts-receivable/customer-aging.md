# <a name="customer-aging-report"></a>Отчет по срокам оплаты по клиентам 

Отчет **Распределение клиентов по срокам оплаты** содержит дебетовые сальдо клиентов, отсортированные по интервалу дат или периоду распределения по срокам.

При создании этого отчета отображаются следующие параметры по умолчанию. Эти параметры можно использовать для фильтрации данных, которые будут отображаться в отчете. Дополнительные сведения см. в разделе [Настройка коллекций](set-up-collections.md).

<div class="alert">

**Примечание.** Этот элемент управления доступен, только если выбран конфигурационный ключ <STRONG>Государственный сектор</STRONG>.</P>

<div class="alert">

<td><p>Выбор типа даты, на котором будет базироваться отчет.</p>
<ul>
<li><p><strong>Дата проводки</strong> — дата разноски проводок. Например, это может быть дата накладной, которая является базой для расчета срока выполнения.</p></li>
<li><p><strong>Срок выполнения</strong> — срок выполнения проводок с учетом условий оплаты.</p></li>
<li><p><strong>Дата документа</strong> — определяемая пользователем дата по документу, которая является базой для расчета срока оплаты.</p></li>
</ul></td>
</tr>

<td><p>Выбор определения периода распределения по срокам. Если выбрано определение периода распределения по срокам, поле <strong>Интервал</strong> не используется.</p>
<p>Определения периода распределения по строкам, которые имеют более 6 периодов распределения по срокам, нельзя использовать в распечатанном отчете.</p>
<div class="alert">

**Примечание.** Периоды распределения по срокам можно настроить на странице <STRONG>Определения периодов распределения по срокам</STRONG>.</P>


</div></td>
</tr>

</tr>

<div class="alert">

**Примечание.** Сведения, указанные в данном поле, используются только в том случае, если определение периода распределения по срокам не выбрано. В противном случае направление печати определяется в определении периодов распределения по срокам оплаты.</P>


</div></td>
</tr>
<tr class="odd">
<td><p><strong>День/мес.</strong></p></td>
<td><p>Выбор единицы времени: <strong>День</strong> или <strong>Месяц</strong>, которая будет определять период в поле <strong>Интервал</strong>.</p></td>
</tr>
<tr class="even">
<td><p><strong>Направление печати</strong></p></td>

Даты оцениваются по отношению к дате, выбранной в поле <strong>Сальдо на</strong>. Выберите <strong>Назад</strong>, чтобы показать сведения для прошлых периодов. Выберите <strong>Вперед</strong>, чтобы показать сведения для будущих периодов.</p>
<div class="alert">
  
<STRONG>Примечание.</STRONG> Сведения, указанные в данном поле, используются только в том случае, если определение периода распределения по срокам не выбрано.</P>


</div></td>
</tr>
