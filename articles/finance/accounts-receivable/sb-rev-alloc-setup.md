# <a name="set-up-multiple-element-revenue-allocation"></a>Настройка распределения выручки по нескольким элементам

Распределение выручки по нескольким элементам позволяет настроить различные шаблоны, используемые для расчета и распределения выручки по нескольким номенклатурам. Эти функции могут помочь обеспечить соответствие теме 606 кодификации бухгалтерских стандартов (ASC 606) и/или стандарту международной финансовой отчетности 15 (МСФО 15).

## <a name="multiple-element-revenue-allocation-parameters"></a>Параметры распределения выручки по нескольким элементам

Страница **Параметры распределения выручки из нескольких элементов** используется для настройки параметров для распределения выручки нескольких элементов.

### <a name="standalone-selling-price-origin"></a>Источник отдельной цены продажи

Поле **Источник отдельной цены продажи** определяет, откуда берется отдельная цена продажи. Можно обновить это значение на странице **Отдельная цена продажи номенклатуры** и в проводке. Имеются следующие варианты.

| Параметр | Описание |
|--------|-------------|
| Сумма | Отдельная цена продажи — это сумма, которую вы указываете и которая больше 0 (нуля). При необходимости сумма конвертируется между функциональной и исходной валютами. |
| Базовая цена продажи | Отдельная цена продажи соответствует базовой цене продажи номенклатуры. |
| Цена накладной | Отдельная цена продажи соответствует цене в накладной для номенклатуры. |
| Процент номенклатуры | Отдельная цена продажи указывается как процентное значение и рассчитывается на основе цены номенклатуры. Если выбран этот параметр, укажите процентное значение по умолчанию. |
| Распределить сумму остатка | <p>Происхождение отдельной цены продажи рассчитывается как *Общая сумма контракта по родительской номенклатуре* – *Общая цена продажи дочерних номенклатур*:</p><ul><li>*Общая сумма контракта по родительской номенклатуре* — чистая сумма или сумма, на которую выставлены счета.</li><li>*Общая отдельная продажа по дочерним номенклатурам* — это сумма расширенной или контрактной отдельной цены продажи для всех дочерних номенклатур, за исключением дочерней номенклатуры, в которой используется этот источник отдельной цены продажи.</li></ul><p>Если рассчитанная сумма является отрицательным значением, в качестве суммы устанавливается значение 0 (ноль).</p><p>**Примечание.** Этот параметр может быть выбран только для одной дочерней номенклатуры в разделении выручки.</p> |
| Нет | Источник отдельной цены продажи базируется на дочерних номенклатурах. Этот параметр применяется к номенклатурам, которые определены как родительские номенклатуры в шаблоне разделения выручки. Если установлен флажок **Разделение выручки**, этот параметр выбирается автоматически, и настройка не может быть изменена. |
| Процент родительской цены накладной | Источником отдельной цены продажи является процент от цены по накладной родительской номенклатуры. Вы можете изменить значение по умолчанию. Этот параметр доступен только для дочерних номенклатур в шаблоне разбиения выручки. |
| Процент родительской стандартной цены | Источником отдельной цены продажи является процент от стандартной цены родительской номенклатуры. Вы можете изменить значение по умолчанию. Этот параметр доступен только для дочерних номенклатур в шаблоне разбиения выручки. Это параметр по умолчанию для дочерних номенклатур. Когда параметр для дочерней номенклатуры изменяется с **Процент от родительской стандартной цены** на **Процент от родительской цены по накладной** или с **Процента от родительской цены по накладной** на **Процент от родительской стандартной цены**, вычисляемые значения также обновляются. |

### <a name="account-for-revenue-allocation-rounding-differences"></a>Счет для ошибок округления распределения выручки

Поле **Счет для разницы округления распределения выручки** указывает счет, используемый для записи каких-либо корректировок округления в сумме выручки по контракту. Оно доступно, когда используется повторяющееся выставление счетов по контракту.

## <a name="item-standalone-selling-price"></a>Отдельная цена продажи номенклатуры

Страница **Отдельная цена продажи номенклатуры** используется для указания исходной и отдельной отпускной цены номенклатуры. Значения, указанные на этой странице, используются как значения по умолчанию на странице **Распределение выручки** в распределении выручки по нескольким элементам и в периодическом выставлении счетов по контракту.

### <a name="specify-item-standalone-selling-price"></a>Указание отдельной цены продажи номенклатуры

Чтобы указать для номенклатуры отдельную цену, выполните следующие действия.

1. На странице **Отдельная цена продажи для номенклатуры** в поле **Источник отдельной цены продажи** выберите происхождение отдельной цены продажи.
2. Выполните одно из следующих действий в зависимости от значения, выбранного в поле **Источник отдельной цены продажи**.

    * Если выбран **Процент номенклатуры**, укажите процент в поле **Процент**.
    * Если выбрано значение **Сумма**, укажите сумму в поле **Отдельная цена продажи**.

2. Для каждой номенклатуры, которую следует добавить в список, выберите **Добавить**.
3. В поле **Код номенклатуры** выберите код номенклатуры. В поле **Связь номенклатуры** выберите связь номенклатуры. Для номенклатур, для которых снят флажок **Разбиение выручки**, выбранное сочетание кода номенклатуры и связи номенклатуры должно быть уникальным.
4. Измените значение по умолчанию для полей **Источник отдельной цены продажи**, **Отдельная цена продажи** или **Процент**.
5. Для каждой родительской номенклатуры, которую следует добавить в список, выберите **Добавить**.
6. В поле **Код номенклатуры** выберите код номенклатуры. В поле **Связь номенклатуры** выберите связь номенклатуры.
7. Установите флажок **Разделение выручки**.
8. В поле **Связь номенклатуры** выберите связь номенклатуры. Список обновляется родительской номенклатурой и всеми дочерними номенклатурами.
9. Для дочерней номенклатуры внесите требуемые изменения в значение по умолчанию поля **Источник отдельной цены продажи**, **Процент родительской стандартной цены**, **Процента родительской цены по накладной** или **Распределение остаточной суммы**.
10. Чтобы добавить несколько номенклатур за один раз, выберите **Добавить номенклатуры**.
11. Обновите запрос, чтобы выбрать диапазон номенклатур, который требуется добавить.
12. Выберите **ОК**, просмотрите список добавленных номенклатур и выберите **ОК**.

## <a name="mea-templates"></a>Шаблоны MEA

Страница **Шаблоны МЕА** используется для создания и изменения кодов шаблонов структуры с несколькими элементами (МЕА). Эти коды используются на странице **Распределение выручки** для объединения номенклатур в группы.

### <a name="create-a-template"></a>Создать шаблон

Чтобы настроить шаблон MEA, выполните следующие действия.

1. На странице **Шаблоны MEA** нажмите **Создать**.
1. В поле **Код шаблона MEA** введите уникальный код.
1. В поле **Описание** введите описание.
1. В поле **Счет выручки будущих периодов по контракту** выберите счет, который будет использоваться для записей журнала при создании накладной контракта MEA. Это поле доступно, если включено и используется повторяющееся выставление счетов по контракту.
1. Нажмите **Сохранить**.
