# <a name="main-account-types"></a>Типы счетов ГК

[!include [banner](../includes/banner.md)]

Типы счетов ГК определяют, каким образом счет ГК используется в плане счетов.

Можно также воспользоваться автоматическим выбором типа счета ГК при создании счета ГК на странице **Счета ГК — план счетов**.
-   **Прибыли и убытки**, **Выручка** и **Расход** — эти счета ГК используются как счета разноски. Счета **прибыли и убытков** используются для записи выручки и расходов. Сумма всех счетов прибылей и убытков, выручки и расходов определяет результаты в конце года. Счета ГК **Выручка** и **Расходы** выполняют те же функции, что и счета ГК **Прибыль и убытки**.
-   **Балансовый отчет**, **Основное средство**, **Задолженность** и **Собственный капитал** — эти счета ГК используются в качестве счетов проводок для записи сумм, которые компания должна или дает в долг.
-   **Итого** — эти счета ГК используются для добавления диапазонов счетов. Настройте тип счета ГК **Итого** с помощью страницы **Диапазон счетов**, которая отображает сумму групповых сальдо для всех диапазонов счетов. Откройте страницу **Диапазон счетов**, нажав **Итоги** на вкладке **Общие** после выбора типа счета **Итого**.
-   **Отчетность** — эти счета ГК используются для составления финансовых отчетов только для Бразилии.

Дополнительные сведения см. в разделе [Настройка категорий счетов ГК](tasks/set-up-main-account-categories.md).
