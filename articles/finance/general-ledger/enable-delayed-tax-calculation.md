# <a name="enable-delayed-tax-calculation-on-journals"></a>Включение отсроченного расчета налога для журналов
[!include [banner](../includes/banner.md)]


В этом разделе объясняется, как можно отложить расчет налога по журналам. Эта возможность позволяет улучшить производительность налоговых расчетов, если имеется много строк журнала.

По умолчанию суммы налога в строках журнала рассчитываются каждый раз, когда обновляются поля, соответствующие налогам. В эти поля включаются поля для налоговых групп и налоговых групп номенклатур. Любое обновление строки журнала приводит к пересчету налоговых сумм для всех строк журнала. Хотя это позволяет пользователю видеть суммы налога, рассчитанные в реальном времени, это также может повлиять на производительность, если количество строк журнала очень велико.

Функция отсроченного расчета налога позволяет отложить расчет налогов по журналам и, таким образом, помочь в устранении проблем с производительностью. Когда эта функция включена, суммы налога будут вычисляться только тогда, когда пользователь выбирает команду **Налог** или разносит журнал.

Можно отложить расчет налогов на трех уровнях:

- Информация о компании
- Наименование журнала
- Заголовок журнала

Система назначает приоритет параметру для заголовка журнала. По умолчанию этот параметр берется из наименования журнала. По умолчанию значение для наименования журнала берется из настройки на странице **Параметры главной книги** при создании наименования журнала. В следующих разделах объясняется, как включить Отсроченный расчет налога для юридических лиц, наименований журналов и заголовков журналов.

## <a name="turn-on-delayed-tax-calculation-at-the-legal-entity-level"></a>Включите Отсроченный расчет налога на уровне юридических лиц

1. Перейдите в раздел **Главная книга \> Настройка главной книги \> Параметры главной книги**.
2. На вкладке **Налог** на экспресс-вкладке **Общие** задайте для параметра **Отсроченный расчет налога** значение **Да**.

![Изображение параметров главной книги](media/delayed-tax-calculation-gl.png)

## <a name="turn-on-delayed-tax-calculation-at-the-journal-name-level"></a>Включите Отсроченный расчет налога на уровне наименование журнала

1. Перейдите в раздел **Главная книга \> Настройка журнала \> Наименования журналов**.
2. На экспресс-вкладке **Общее** в разделе **Налог** задайте для параметра **Отсроченный расчет налога** значение **Да**.

![Изображение наименований журналов](media/delayed-tax-calculation-journal-name.png)

## <a name="turn-on-delayed-tax-calculation-at-the-journal-header-level"></a>Включите Отсроченный расчет налога на уровне заголовка журнала

1. Перейдите к **Главная книга \> Записи в журнале \> Общие журналы**.
2. Выберите **Создать**.
3. Выберите название журнала.
4. На вкладке **Настройка** установите для параметра **Отсроченный расчет налога** значение **Да**.

![Изображение страницы общего журнала](media/delayed-tax-calculation-journal-header.png)
