# <a name="set-up-global-withholding-tax"></a>Настройка глобального подоходного налога

В этом разделе перечислены этапы настройки глобального подоходного налога для продаж и покупок. 

1. Настройте налоговые органы для подоходного налога на странице **Налоговые органы по подоходному налогу**.

2. Настройте периоды сопоставления налогов на странице **Периоды сопоставления подоходного налога**.

3. Настройте группы разноски ГК для удержания налога на странице **Подоходный налог > Группа разноски ГК**.

   > [!Note] 
   >
   > Счету **Подоходный налог** будет назначен счет ГК с параметером **Тип разноски** со значением подоходного налога. Счету **Корреспондирующий подоходного налога** также будет назначен счет ГК с параметером **Тип разноски** со значением "Корреспондирующий подоходного налога". Перейдите в раздел **Главная книга > План счетов > Счета > Счета ГК**, настройте **Тип разноски** во вложенной форме **Проверка разноски** для счетов ГК.

4. Настройте коды подоходного налога на странице **Коды подоходного налога**.

5. Настройте группы подоходного налога на странице **Группы подоходного налога**.

6. Настройте типы выручки подоходного налога на странице **Типы** **выручки по подоходному налогу**.

7. Настройте группы подоходного налога на странице **Группы подоходного налога для номенклатуры** для номенклатуры или услуги.

8. Настройте параметр **Минимальная сумма накладной** на странице **Параметры главной книги > Подоходный налог**.
