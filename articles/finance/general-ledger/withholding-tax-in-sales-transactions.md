# <a name="withholding-tax-in-sales-transactions"></a>Подоходный налог в проводках по продажам

В этой статье перечислены шаги, которые необходимо выполнить, чтобы исключить расчет подоходного налога для выбранных клиентов. Для клиентов, которые указывают в своих платежах вам подоходный налог, можно назначить параметр **Группа подоходного налога** по умолчанию на странице **Клиенты**. 

1. Перейдите в раздел **Область перехода > Модули > Расчеты с клиентами > Клиенты > Все клиенты**.

2. Щелкните соответствующий счет клиента, щелкните **Правка**.

3. На вкладке **Накладная и поставка** установите в поле **Рассчитать подоходный налог** значение **Да**.

   > [!NOTE] 
   > Подоходный налог не будет рассчитываться, если поле **Рассчитать подоходный налог** не включено для этого клиента в справочнике.

4. Выберите группу подоходного налога в поле **Группа подоходного налога**.

5. Нажмите кнопку **Сохранить**.

Для номенклатур/услуг, облагаемых подоходным налогом, можно назначить значение по умолчанию в поле **Группа подоходного налога для номенклатуры** в разделе **Выпущенные продукты**.

1. Перейдите в раздел **Область переходов > Модули > Управление сведениями о продукте > Продукты > Выпущенные продукты**.

2. Щелкните соответствующий номер номенклатуры, щелкните **Правка**.

3. На вкладке **Продажа** щелкните **Рассчитать подоходный налог**.

   > [!NOTE] 
   > Подоходный налог не будет рассчитываться, если в поле **Рассчитать подоходный налог** не установлено значение **Да** для этой номенклатуры на вкладке **Продажа** на странице **Выпущенный продукт**.

4. Выберите группу подоходного налога для номенклатуры в списке **Группа подоходного налога для номенклатуры**.

5. Нажмите кнопку **Сохранить**.

Группы подоходного налога и группы подоходного налога для номенклатур могут быть назначены с помощью страницы **Заказ на продажу**. 

Группа подоходного налога по умолчанию и группа подоходного налога для номенклатуры будут использоваться в качестве записей по умолчанию в строках заказа на продажу при создании нового заказа на продажу.

Подоходный налог рассчитывается и разносится с помощью **Журнала платежей клиентов**. Можно вручную скорректировать соответствующий код подоходного налога, а также фактическую сумму подоходного налога на вкладке **Подоходный налог** на странице **Сопоставление проводок**.

Расчетная сумма подоходного налога будет вычтена из платежа клиента и разнесена на **Корреспондирующий счет подоходного налога** в соответствующей операции.
