# <a name="set-up-a-ledger-account-alias"></a>Настройка псевдонима счета ГК

[!include [banner](../../includes/banner.md)]

Ниже описан порядок создания псевдонима счета, который обеспечивает ярлык для ввода номера счета. В этой процедуре используется компания с демонстрационными данными USMF.

1. Перейдите в раздел "Главная книга" > "План счетов" > "Счета" > "Псевдоним счета учета".
2. Щелкните "Создать".
3. В поле "Псевдоним счета учета" введите значение.
4. В поле "Структура счета" выберите структуру, которой принадлежат счет и аналитики.
5. В поле "Компания" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. В списке найдите и выберите компанию, к которой применяется псевдоним.
7. В списке перейдите по ссылке в выбранной строке.
8. В поле "Определение псевдонима счета учета" укажите счет и аналитики.
    * Счет и аналитики будут заполнены при использовании ярлыка.  
9. В поле "Первоначальное фокусирование" выберите аналитику, которая будет фокусироваться, где будет использоваться псевдоним.
    * После ввода ярлыка и заполнения счета и аналитик поле "Первоначальное фокусирование" находится там, где курсор, или куда перемещается фокус.
