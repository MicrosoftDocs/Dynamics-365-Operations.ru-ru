# <a name="journalize-posted-journal-entries"></a>Учет разнесенных записей в журнале в субкниге

[!include [banner](../../includes/banner.md)]

Процесс журнализации в главной книге предоставляет способ группировки и отчетности по разнесенным ваучерам для главной книги. На основании предоставленного критерия обработка создает список ваучеров, использующих уникальную номерную серию, которая имеет значение **Номер журнала** главной книги в качестве ссылки.

Выполните следующие действия для журнализации разнесенных записей в журнале. В этой процедуре используется компания с демонстрационными данными **USMF**.

1. Перейдите в раздел **Главная книга** \> **Настройка главной книги** \> **Параметры главной книги**.
2. В поле **Расширенный журнал ГК** выберите значение. Если выбрать **Да**, выходные данные отчета будут другими.
3. Выберите, можно ли закрыть период, если процесс журнализации не был запущен. Если выбрано значение **Да**, период нельзя закрыть до тех пор, пока не будет завершен процесс журнализации за этот период.
4. Закройте страницу.
5. Перейдите к пункту **Главная книга** \> **Периодические задачи** \> **Журнализация** и выберите **Фильтр**.
6. Выберите строку для критериев фильтра, которые вы хотите указать.
7. В поле **Критерии** введите или выберите критерий фильтра.
8. Нажмите кнопку **OK**, чтобы закрыть страницу.
9. Выберите **ОК**, чтобы начать процесс журнализации. Отчет будет создан после завершения процесса.
