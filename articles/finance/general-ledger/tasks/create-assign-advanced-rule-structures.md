# <a name="create-and-assign-advanced-rule-structures"></a>Создание и назначение структур дополнительных правил

[!include [banner](../../includes/banner.md)]

В этой статье объясняется, как создать структуру дополнительного правила и назначить ее структуре счета. В данном руководстве используется демонстрационная компания USMF.

## <a name="create-an-advanced-rule-structure"></a>Создание новой структуры дополнительного правила
1. Выберите **Область переходов > Модули > Главная книга > План счетов > Структуры > Структуры дополнительных правил**.
2. Выберите **Создать**, чтобы открыть раскрывающееся диалоговое окно.
3. В поле **Структура дополнительных правил** введите имя для описания структуры правил.
4. Нажмите **ОК**.
5. Выберите **Добавить сегмент**.
6. В списке сегментов выберите финансовую аналитику. Например, **Магазин**.  
7. Выберите **Добавить сегмент**.
8. Выберите **Активировать**.

## <a name="apply-an-advanced-rule-structure-to-an-account-structure"></a>Применение структуры дополнительного правила к структуре счета
1. Выберите **область переходов > Модули > Главная книга > План счетов > Структуры > Настройка структур счетов**.
2. В списке найдите и выберите структуру счета, к которой необходимо применить дополнительное правило.
3. Выберите **Правка**. Можно также выбрать **Дополнительные правила**, и будет предложено перевести структуру счета в **Режим черновика**.  
4. Выберите **Дополнительные правила**.
5. Выберите **Создать**, чтобы открыть раскрывающееся диалоговое окно.
6. В поле **Дополнительное правило** введите значение.
7. В поле **Имя** введите значение.
8. Выберите **Создать**.
9. Выберите **Добавить новые критерии**.
10. В поле **Где** выберите счет ГК или финансовую аналитику.
11. В поле **Оператор** выберите параметр, например **находится в диапазоне** и **включает**.
12. В поле **Значение** введите значение.
13. В поле **через** введите значение.
14. Выберите **Добавить**, чтобы открыть раскрывающееся диалоговое окно.
15. В списке найдите структуру дополнительного правила, которая должна использоваться при соответствии указанных критериев.
16. Выберите **Добавить**.
17. Закройте страницу.
18. Выберите **Активировать**.
