# <a name="set-up-sales-tax-authorities"></a>Настройка налоговых органов

[!include [task guide banner](../../includes/task-guide-banner.md)]

Налоговые органы — это юридические лица, которым отправляются налоговые отчеты и выплачиваются налоги. Вы можете платить налоги непосредственно налоговому органу или через счет поставщика, который создается для налогового органа. Если это сделать, компания может использовать свои обычные платежные процедуры для своевременных платежей налоговому органу. Если не настроить налоговый орган в качестве поставщика, кто-то должен будет готовить платежи налоговому органу в соответствующий срок. В этой задаче используется демонстрационная компания USMF.

1. Перейдите в раздел "Налог" > "Косвенные налоги" > "Налог" > "Налоговые органы".
2. Щелкните "Создать".
3. В поле "Налоговый орган" введите значение.
4. В поле "Имя" введите значение.
5. В поле "Счет поставщика" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. Поиск и выбор требуемой записи в списке.
7. В списке перейдите по ссылке в выбранной строке.
8. Выберите макет отчета для своей страны или региона, если он имеется. Если макеты отчетов не соответствуют требованиям налогового органа, используйте макет по умолчанию.
9. Введите значения в форму "Округление" и поля "Округление", чтобы указать, как должна округляться выплачиваемая общая сумма налога. Все расхождения при округлении будут разнесены в счета для автоматических проводок, настроенные в главной книге.
10. В поле "Округление" введите число.
11. Нажмите кнопку "Сохранить".
