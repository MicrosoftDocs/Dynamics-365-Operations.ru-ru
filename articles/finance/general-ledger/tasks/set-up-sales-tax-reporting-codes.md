# <a name="set-up-sales-tax-reporting-codes"></a>Настройка кодов налоговой отчетности

[!include [banner](../../includes/banner.md)]

Эти **коды налоговой отчетности** ссылаются на номер поля, указанный в налоговом отчете. Они используются в форматах отчетов для определенной страны или региона. Они также используются в отчете о налоговых платежах по коду. Этот отчет показывает суммы налогов для периода сопоставления, суммированные по каждому коду отчетности. После создания **кодов налоговой отчетности** на эти коды можно ссылаться на экспресс-вкладке **Настройка отчетов**, на которую можно перейти на странице **Налоговый код**. 

В данной записи используется демонстрационная компания DEMF.

1. В области **Область переходов** выберите **Налог > Настройка > Налог > Коды налоговой отчетности**.
2. Нажмите кнопку **Создать**.
3. Выберите макет отчета, к которому принадлежит код отчетности. Этот макет используется для фильтрации доступных кодов отчетности для налогового кода. Каждый налоговый код принадлежит к периоду сопоставления, который принадлежит к налоговому органу, использующему макета отчета.  
4. В поле **Код отчетности** введите номер.
5. В поле **Текст отчета** введите описание для отображения в отчетах.
6. В поле **Краткое описание** введите описание для внутренних целей.
7. Нажмите кнопку **Сохранить**.
