# <a name="create-a-journal-entry-using-template"></a>Создание записи в журнале с помощью шаблона

[!include [banner](../../includes/banner.md)]

Разнесенные ваучеры журнала можно сохранить как шаблоны ваучеров и применить в новом ваучере журнала. В данной процедуре используется демонстрационная компания USMF.

1. Выберите **Область переходов > Модули > Главная книга > Записи журнала > Общие журналы**.
2. В области **Панель операций** щелкните **Создать**. Эта процедура начинается с создания и разноски ваучера журнала, но любой ранее разнесенный ваучер журнала можно сохранить как шаблон.  
3. В поле **Имя** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. Поиск и выбор требуемой записи в списке.
5. В списке перейдите по ссылке в выбранной строке.
6. Выберите **Строки**.
7. В поле **Тип счета** введите значение.
8. В поле **Описание** введите значение.
9. В поле **Дебет** введите значение.
10. Нажмите кнопку **Создать**.
11. В поле **Тип счета** введите значение.
12. В поле **Описание** введите значение.
13. В поле **Дебет** введите значение.
14. Нажмите кнопку **Создать**.
14. В поле **Счет** укажите требуемые значения.
15. В поле **Описание** введите значение.
16. В поле **Кредит** введите значение для балансировки ваучера.
17. В области **Панель операций** щелкните **Разнести**.
18. Щелкните **Функции**.
19. Щелкните **Сохранить шаблон ваучера**.
20. В этой процедуре предполагается использование типа **Шаблон процента**. Нажмите кнопку "OК".
    - Процент: суммы в ваучере преобразовываются в процентные коэффициенты, что позволяет применить любую сумму при выборе шаблона ваучера.
    - Сумма: фактические суммы будут сохранены и применены.  
21. Щелкните **Общие журналы**.
22. Нажмите кнопку **Создать**.
23. В поле **Имя** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
24. В списке перейдите по ссылке в выбранной строке.
25. Выберите **Строки**.
26. Щелкните **Функции**.
27. Щелкните **Выбрать шаблон ваучера**.
28. Найдите созданный ранее шаблон. Щелкните **OK**. Возможно, потребуется щелкнуть **Предыдущий шаг** и выбрать правильный шаблон, если существуют другие шаблоны.  
29. В поле **Сумма** введите сумму, которая должна применяться к ваучеру. Поле **Сумма** отображается, только если шаблон ваучера принадлежит к типу "Процент".  
30. Щелкните **OK**.
