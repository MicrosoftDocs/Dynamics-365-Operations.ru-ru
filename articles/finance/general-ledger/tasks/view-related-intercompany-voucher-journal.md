# <a name="view-related-intercompany-voucher-from-journal"></a>Просмотр связанного внутрихолдингового ваучера из журнала

[!include [banner](../../includes/banner.md)]

В окне связанного ваучера отображается ваучер в корреспондирующей компании при разноске внутрихолдинговой проводки из общего журнала.


## <a name="post-an-intercompany-journal"></a>Разноска внутрихолдингового журнала
1. Перейдите в **Общие журналы**, нажмите кнопку **Создать**.
2. В списке пометьте выбранную строку.
3. В поле **Имя** введите или выберите имя внутрихолдингового журнала.
4. Выберите пункт **Строки** в списке, пометьте выбранную строку.
5. В поле **Счет** укажите требуемые значения.
6. В поле **Описание** введите или выберите значение.
7. Закройте страницу.
8. В поле **Дебет** введите число.
9. В поле **Корреспондирующая компания** введите или выберите корреспондирующую компанию.
10. В поле **Корр.счет** укажите требуемые значения.
11. Щелкните **Разнести**.

## <a name="view-related-intercompany-voucher"></a>Просмотр связанного внутрихолдингового ваучера
1. Щелкните **Операция**.
2. Выберите **Связанные ваучеры** в списке, пометьте выбранную строку.
3. Щелкните **Операция**.
