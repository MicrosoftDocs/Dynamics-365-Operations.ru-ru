# <a name="create-a-main-account"></a><span data-ttu-id="c149b-103">Создание счета ГК</span><span class="sxs-lookup"><span data-stu-id="c149b-103">Create a main account</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="c149b-104">В этом руководстве описано добавление счета ГК в существующий план счетов.</span><span class="sxs-lookup"><span data-stu-id="c149b-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="c149b-105">В данной записи используется демонстрационная компания USMF.</span><span class="sxs-lookup"><span data-stu-id="c149b-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="c149b-106">Выберите **Область переходов > Модули > Главная книга > План счетов > Счета > Счета ГК**.</span><span class="sxs-lookup"><span data-stu-id="c149b-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="c149b-107">Нажмите кнопку **Создать**.</span><span class="sxs-lookup"><span data-stu-id="c149b-107">Click **New**.</span></span>
3. <span data-ttu-id="c149b-108">В поле **Счет ГК** введите значение.</span><span class="sxs-lookup"><span data-stu-id="c149b-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="c149b-109">В поле **Имя** введите значение.</span><span class="sxs-lookup"><span data-stu-id="c149b-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="c149b-110">В поле **Тип счета ГК** выберите тип, который наилучшим образом соответствует сальдо счетов и местоположению финансовых отчетов.</span><span class="sxs-lookup"><span data-stu-id="c149b-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="c149b-111">В списке выберите категорию счетов, которой принадлежит счет ГК.</span><span class="sxs-lookup"><span data-stu-id="c149b-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="c149b-112">Категория счета используется для финансовых отчетов по умолчанию и содержимого панели мониторинга Power BI.</span><span class="sxs-lookup"><span data-stu-id="c149b-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="c149b-113">В списке перейдите по ссылке в выбранной строке.</span><span class="sxs-lookup"><span data-stu-id="c149b-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="c149b-114">Изменение дебета по умолчанию или кредитового сальдо.</span><span class="sxs-lookup"><span data-stu-id="c149b-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="c149b-115">В поле **Валюта по умолчанию** выберите значение из списка валют.</span><span class="sxs-lookup"><span data-stu-id="c149b-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="c149b-116">Поиск и выбор требуемой записи в списке.</span><span class="sxs-lookup"><span data-stu-id="c149b-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="c149b-117">В списке перейдите по ссылке в выбранной строке.</span><span class="sxs-lookup"><span data-stu-id="c149b-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="c149b-118">Переключите расширение раздела **Переопределения юридического лица**.</span><span class="sxs-lookup"><span data-stu-id="c149b-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="c149b-119">Щелкните **Добавить** для выбора юридического лица.</span><span class="sxs-lookup"><span data-stu-id="c149b-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="c149b-120">В списке выберите юридическое лицо.</span><span class="sxs-lookup"><span data-stu-id="c149b-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="c149b-121">Нажмите кнопку **Добавить**.</span><span class="sxs-lookup"><span data-stu-id="c149b-121">Click **Add**.</span></span>
15. <span data-ttu-id="c149b-122">В списке пометьте выбранную строку.</span><span class="sxs-lookup"><span data-stu-id="c149b-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="c149b-123">Поставьте или снимите флажок **Приостановлено**.</span><span class="sxs-lookup"><span data-stu-id="c149b-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="c149b-124">Разверните раздел **Финансовая отчетность**.</span><span class="sxs-lookup"><span data-stu-id="c149b-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="c149b-125">В поле **Тип валютного курса** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.</span><span class="sxs-lookup"><span data-stu-id="c149b-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="c149b-126">В списке выберите **Тип валютного курса для счета**.</span><span class="sxs-lookup"><span data-stu-id="c149b-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="c149b-127">В списке перейдите по ссылке в выбранной строке.</span><span class="sxs-lookup"><span data-stu-id="c149b-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="c149b-128">В поле **Тип преобразования валюты** выберите способ расчета валютных курсов для счета.</span><span class="sxs-lookup"><span data-stu-id="c149b-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="c149b-129">Закройте страницу.</span><span class="sxs-lookup"><span data-stu-id="c149b-129">Close the page.</span></span>
