# <a name="create-accrual-schemes"></a>Создание схем начисления

[!include [banner](../../includes/banner.md)]

В этой статье описывается, как создать схему начисления. В этой задаче используется демонстрационная компания USMF.

1. Выберите **Область переходов > Модули > Главная книга > Настройки журнала > Схемы начисления**.
2. Выберите **Создать**.
3. В поле **Код начисления** введите значение.
4. В поле **Описание схемы начислений** введите значение.
5. В поле **Дебет** укажите требуемые значения. Определенный счет ГК заменит счет ГК дебета в строке ваучера журнала и также будет использоваться для реверсирования расхода будущего периода на основе проводок начисления ГК.  
6. В поле **Кредит** укажите требуемые значения. Определенный счет ГК заменит счет ГК кредита в строке ваучера журнала и также будет использоваться для реверсирования расхода будущего периода на основе проводок начисления ГК.  
7. В поле **Ваучер** выберите способ определения ваучера, когда выполняется разноска проводок.
8. В поле **Описание** введите значение для описания проводок, которые будут разнесены.
9. В поле **Частота периода** выберите, как часто должны происходить проводки.
10. В поле **Количество случаев по периоду** введите число.
11. В поле **Разнести проводки** выберите, когда проводки должны быть разнесены, например, **Ежемесячно**.
