# <a name="define-audit-policies-for-source-documents"></a>Определение политик аудита для документов-источников

[!include [banner](../../includes/banner.md)]

В этой теме объясняется, как настроить и запустить правила политики аудита. В этом примере используются отчеты по расходам с типом расходов на проживание в гостинице. В данной процедуре используется демонстрационная компания USMF. Роль аудитора содержит правильные разрешения для выполнения этих задач.

1. В области переходов выберите **Модули > Рабочее место аудита > Настройка > Тип правила политики**.
2. Выберите **Создать**.
3. В поле **Имя правила** введите значение.
4. В поле **Описание** введите значение.
5. В поле **Имя запроса** выберите строку **Отчет по расходам**.
6. В поле **Тип запроса** выберите **Агрегировано**.
7. В поле **Юридическое лицо** выберите **Юридическое лицо**.
8. В поле **Ссылка на дату документа** выберите **Дата и время изменения**.
9. Нажмите **Сохранить**.
10. В области переходов выберите **Модули > Рабочее место аудита > Настройка > Политики аудита**.
11. Выберите **Создать**.
12. В поле **Имя** введите значение.
13. Разверните раздел **Организации политики**.
14. В дереве выберите **Contoso Entertainment System USA**, затем выберите **Добавить**.
15. В дереве выберите **Contoso Consulting USA**, затем выберите **Добавить**.
16. В дереве выберите **Contoso Retail USA**, затем выберите **Добавить**.
17. Сверните раздел **Организации политики**.
18. Разверните раздел **Правила политики**.
19. Найдите в списке ранее созданное правило политики и выберите его.
20. Выберите **Создать правило политики**.
21. В поле **Действует с** введите дату и время.
22. Выберите **Фильтр**.
23. В списке выберите строку для **Категория расходов** и задайте для сведений значение **Гостиница**.
24. В поле **Критерии** введите или выберите значение.
25. Выберите вкладку **Агрегировано**.
26. Выберите **Добавить**.
27. В списке выберите значение поля **Сумма проводки**.
28. В поле **Поле** введите или выберите значение.
29. В поле **Функция расчета** выберите **Сумма**.
30. Выберите вкладку **Группировать по**.
31. Выберите **Добавить**.
32. В списке выберите значение **Сотрудник**.
33. Выберите **Добавить**.
34. В списке выберите значение **Категория расходов**.
35. В поле **Поле** введите или выберите значение.
36. Выберите вкладку **Владение**.
37. Выберите **Добавить**.
38. Выберите **Сумма проводки**.
39. В поле **Поле** введите или выберите значение.
40. В поле **Функция расчета** выберите **Сумма**.
41. В поле **Критерии** введите `>2000`.
42. Нажмите **ОК**.
43. Выберите **Проверка**.
44. В поле **Начальная дата выбора документов** введите дату и время.
45. В поле **Конечная дата выбора документов** введите дату и время.
46. Выберите **Запустить тест**.
47. В области действий выберите **Политика аудита**.
48. Выберите **Дополнительные параметры**.
49. В поле **Дата начала** введите дату и время.
50. В поле **Дата окончания** введите дату и время.
51. Выберите **Партия**.
52. Разверните раздел **Выполнять в фоновом режиме**.
53. Выберите значение **Да** в поле **Пакетная обработка**.
54. Нажмите **ОК**.
55. В области переходов выберите **Модули > Рабочее место аудита > Настройка > Обращения аудита**.
56. В списке найдите и выберите требуемую запись.
57. Разверните раздел **Ассоциации**.
58. В списке найдите и выберите требуемую запись.
