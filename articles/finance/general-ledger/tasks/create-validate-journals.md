# <a name="create-and-validate-journals"></a>Создание и проверка журналов

[!include [banner](../../includes/banner.md)]

Эта процедура служит для создания и проверки журналов и строк журналов. Эту процедуру можно опробовать, используя компанию с демонстрационными данными USMF.  

1. Перейдите в раздел **Главная книга > Записи в журнале > Общие журналы**.
2. Нажмите кнопку **Создать**.
3. В поле **Имя** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. Поиск и выбор требуемой записи в списке.
5. Выберите **Строки**.
6. В поле **Счет** введите соответствующий счет на основе типа счета.
7. В поле **Описание** введите значение.
8. Введите сумму для счета как **Кредит** или **Дебет**. 
9. В поле **Корр.счет** введите соответствующий счет на основе типа корр. счета.
10. Щелкните **Проверить**.
11. Щелкните **Проверить**.
12. Щелкните **Разнести**.
13. Щелкните **Операция**.
