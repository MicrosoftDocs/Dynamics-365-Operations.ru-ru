# <a name="create-ledger-accrual-transactions"></a>Создание проводок начисления ГК

[!include [banner](../../includes/banner.md)]

Это руководство описывает создание проводок начисления ГК, которые основаны на схемах начисления

1. Перейдите в раздел "Главная книга" > "Записи в журнале" > "Общие журналы".
2. В списке найдите и выберите требуемый журнал или создайте новый.
3. Щелкните для перехода по ссылке в поле "Номер партии журнала".
4. В списке пометьте выбранную строку.
5. В поле "Счет" укажите требуемые значения.
    * В этом примере определяются расходы для страховки. Они станут периодической суммой расходов.  
6. В поле "Описание" введите значение.
7. В поле "Дебет" введите число.
8. В поле "Корр.счет" укажите требуемые значения.
9. Щелкните Функции.
10. Щелкните "Начисления ГК".
11. В поле "Код начисления" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
12. В списке найдите и выберите схему начисления, которую необходимо применить.
13. В списке перейдите по ссылке в выбранной строке.
14. В поле "Дата начала" введите дату.
15. Щелкните "Проводки".
16. Закройте страницу.
17. Нажмите кнопку "OК".
18. Щелкните "Разнести".
