# <a name="create-account-structures"></a>Создание структур счетов

[!include [banner](../../includes/banner.md)]

В этом руководства рассказывается о создании структуры счета. Для выполнения шагов используется компания с демонстрационными данными USMF.

1. Выберите **Область переходов > Модули > Главная книга > План счетов > Структуры > Настройка структур счетов**.
2. На **Панели операций** щелкните **Создать**, чтобы открыть раскрывающееся диалоговое окно.
3. В поле **Структура счета** введите имя для описания назначения структуры счета.
4. В поле **Описание** введите описание для указания назначения структуры счета.
5. Щелкните **Создать**.
6. В области **Сегменты и допустимые значения** щелкните **Добавить сегмент**.
7. В списке аналитик выберите аналитику для добавления в структуру счета.
8. В конце списка щелкните **Добавить сегмент**.
9. При необходимости повторите шаги с 6 по 9.
10. В разделе **Подробные сведения о допустимых значениях** выберите сегмент для редактирования допустимых значений.
    Например, щелкните поле **Счет ГК**.  
11. В поле **Оператор** выберите параметр, например "находится в диапазоне и включает".
12. В поле **Значение** введите значение. Например, 600000.  
13. В поле **через** введите значение. Например, 699999.  
14. В разделе **Подробные сведения о допустимых значениях** щелкните **Применить**.
15. При необходимости повторите шаги с 10 по 15.  
16. В разделе **Подробные сведения о допустимых значениях** щелкните **Добавить новые критерии**.
17. В поле "Оператор" выберите параметр, например "находится в диапазоне и включает".
18. В поле **Значение** введите значение. Например, 033.  
19. В поле **через** введите значение. Например, 034.  
20. Нажмите кнопку **Применить**.
21. В сетке выберите сегмент для редактирования допустимых значений. Например, "Место возникновения затрат".  
22. В поле **CostCenter** введите значение. Например, 007..021.  
23. В области **Сегменты и допустимые значения** щелкните **Добавить**.
24. В поле **MainAccount** введите значение. Например, 600000..699999  
25. В сетке выберите сегмент для редактирования допустимых значений. Например, "Отдел".  
26. В поле "Подразделение" введите значение. Например, 032.  
27. В поле CostCenter введите значение. Например, 086.  
28. В области **Панель операций** щелкните **Проверить**.
29. В области **Панель операций** щелкните **Активировать**.
30. Нажмите кнопку **Активировать**.
