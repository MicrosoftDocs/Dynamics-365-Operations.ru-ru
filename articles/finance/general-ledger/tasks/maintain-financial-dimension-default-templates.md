# <a name="maintain-financial-dimension-default-templates"></a>Ведение шаблонов финансовой аналитики по умолчанию

[!include [banner](../../includes/banner.md)]

В этой статье показано, как настраивать шаблоны по умолчанию для финансовых аналитик. Шаблоны можно использовать, чтобы быстро распределить суммы по исходным документам.

1. В области переходов выберите **Модули > Главная книга > План счетов > Аналитики > Шаблоны финансовой аналитики по умолчанию**.
2. Выберите **Создать**.
3. В поле **Код шаблона** введите код шаблона.
4. Ввод процент для первых значений аналитики.
5. Введите значения финансовой аналитики, которые соответствуют введенному проценту.
6. Выберите **Добавить**.
7. Введите другой процент для следующих значений аналитики. В этом примере введите процент, чтобы все еще было назначено менее 100%. 
8. Введите значения финансовой аналитики, которые соответствуют введенному проценту.
9. Выберите **Добавить**.
10. Введите другой процент, который доведет сумму процентных значений до 100%.
11. Введите значения финансовой аналитики, которые соответствуют введенному проценту.
12. Нажмите **Сохранить**.
