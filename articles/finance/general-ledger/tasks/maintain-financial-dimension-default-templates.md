# <a name="maintain-financial-dimension-default-templates"></a><span data-ttu-id="e9b03-103">Ведение шаблонов финансовой аналитики по умолчанию</span><span class="sxs-lookup"><span data-stu-id="e9b03-103">Maintain financial dimension default templates</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="e9b03-104">Эта тема показывает порядок настройки шаблонов финансовых аналитик по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="e9b03-104">This topic shows how to set up financial dimension default templates.</span></span> <span data-ttu-id="e9b03-105">Шаблоны можно использовать, чтобы быстро распределить суммы по исходным документам.</span><span class="sxs-lookup"><span data-stu-id="e9b03-105">The templates can be used to quickly distribute amounts on source documents.</span></span>

1. <span data-ttu-id="e9b03-106">В области переходов выберите **Модули > Главная книга > План счетов > Аналитики > Шаблоны финансовой аналитики по умолчанию**.</span><span class="sxs-lookup"><span data-stu-id="e9b03-106">In the navigation pane, go to **Modules > General ledger > Chart of accounts > Dimensions > Financial dimension default templates**.</span></span>
2. <span data-ttu-id="e9b03-107">Выберите **Создать**.</span><span class="sxs-lookup"><span data-stu-id="e9b03-107">Select **New**.</span></span>
3. <span data-ttu-id="e9b03-108">В поле **Код шаблона** введите код шаблона.</span><span class="sxs-lookup"><span data-stu-id="e9b03-108">In the **Template ID** field, enter a Template ID.</span></span>
4. <span data-ttu-id="e9b03-109">Ввод процент для первых значений аналитики.</span><span class="sxs-lookup"><span data-stu-id="e9b03-109">Enter the percentage for the first dimension values.</span></span>
5. <span data-ttu-id="e9b03-110">Введите значения финансовой аналитики, которые соответствуют введенному проценту.</span><span class="sxs-lookup"><span data-stu-id="e9b03-110">Enter the financial dimension values that correspond to the percent entered.</span></span>
6. <span data-ttu-id="e9b03-111">Выберите **Добавить**.</span><span class="sxs-lookup"><span data-stu-id="e9b03-111">Select **Add**.</span></span>
7. <span data-ttu-id="e9b03-112">Введите другой процент для следующих значений аналитики.</span><span class="sxs-lookup"><span data-stu-id="e9b03-112">Enter another percent for the next dimension values.</span></span> <span data-ttu-id="e9b03-113">В этом примере введите процент, чтобы все еще было назначено менее 100%.</span><span class="sxs-lookup"><span data-stu-id="e9b03-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span> 
8. <span data-ttu-id="e9b03-114">Введите значения финансовой аналитики, которые соответствуют введенному проценту.</span><span class="sxs-lookup"><span data-stu-id="e9b03-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="e9b03-115">Выберите **Добавить**.</span><span class="sxs-lookup"><span data-stu-id="e9b03-115">Select **Add**.</span></span>
10. <span data-ttu-id="e9b03-116">Введите другой процент, который доведет сумму процентных значений до 100%.</span><span class="sxs-lookup"><span data-stu-id="e9b03-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
11. <span data-ttu-id="e9b03-117">Введите значения финансовой аналитики, которые соответствуют введенному проценту.</span><span class="sxs-lookup"><span data-stu-id="e9b03-117">Enter the financial dimension values that correspond to the percent entered.</span></span>
12. <span data-ttu-id="e9b03-118">Нажмите **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="e9b03-118">Select **Save**.</span></span>
