# <a name="view-journal-entries-and-transactions"></a>Просмотр записей журнала и проводок

[!include [banner](../includes/banner.md)]

Эта статья описывает различные методы, которыми можно посмотреть записи и проводки в журнале 

Пользователи, которые хотят просмотреть журналы и проводки, имеют несколько способов получить доступ к данным. Они могут использовать страницы запросов, которые предлагают возможность детализации, или различные параметры отчетов в главной книге.

## <a name="voucher-transactions"></a>Проводки по ваучеру
**Проводки по ваучеру** — это страница запроса, на которой можно выбрать элементы в различных таблицах и полях для того, чтобы определить критерии для искомых сальдо или проводок. Например, можно просмотреть все проводки за определенную дату или для определенного счета либо все проводки типа **Работа**, которые находятся в определенном слое разноски. По умолчанию на странице отображается номер журнала, ваучер, дата и счет ГК, но можно добавить дополнительные таблицы, поля и критерии для того, чтобы сузить поиск.

## <a name="audit-trail"></a>Аудиторский след
**Аудиторский след** — это страница запроса, на которой отображаются типы проводок, описания, авторы проводок и дата создания проводок. На странице запроса **Аудиторский след** можно просмотреть проводки по ваучеру.

## <a name="financial-reports"></a>Финансовые отчеты
Проводки ГК также можно изучить и проанализировать, выполнив финансовые отчеты. Поскольку дизайн финансовых отчетов может основываться на счетах, аналитиках, категориях счетов или сочетании этих трех параметров, проводки можно просмотреть, выполнив различную детализацию. Если вам требуется больше информации о проводках ГК, вы также можете включить несколько свойств проводки как часть дизайна отчета. Кроме того, если требуется просмотреть проводки, которые составляют сальдо ГК, можно выполнить детализацию до проводок по счету, так же как на странице списка **Пробный баланс**.

## <a name="ledger-reports"></a>Отчеты по главной книге
В дополнение к финансовым отчетам можно использовать следующие отчеты по главной книге для просмотра проводок ГК:

-   **Отчет по аналитике** — в этом отчете отображаются проводки по дню и счету. Также можно просмотреть проводки по аналитике и периоду.
-   **Список проводок ГК** — в этом отчете отображаются проводки в валютах проводки, учета и отчетности для счета.
-   **Журнал печати** — в этом отчете отображается результат разнесенного журнала. Вы можете выполнить отчет по номеру партии журнала или типу журнала либо добавить дополнительные поля.
-   **Разнесенные проводки по журналу** — в этом отчете отображаются проводки, которые были разнесены в журнал, сгруппированные по ваучеру.
-   **Список проводок по датам** — в этом отчете отображаются все проводки по дате вместе с номером журнала, ваучером и счетом ГК. В нем также отображаются проводки в валютах проводки, учета и отчетности.
-   **Основание проводки** — в этом отчете по проводкам отображается счет по журналу, а также по валюте проводки, учета и отчетности. В нем также отображается каждая строка журнала, которая использовалась как смещение.
