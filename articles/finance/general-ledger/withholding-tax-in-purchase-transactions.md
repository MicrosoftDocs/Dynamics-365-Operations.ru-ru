# <a name="withholding-tax-in-purchase-transactions"></a>Подоходный налог в проводках по покупкам

Для поставщиков, которые облагаются подоходным налогом, можно назначить **Группу подоходного налога** по умолчанию на странице **Все поставщики**.

1. Перейдите в раздел **Область перехода > Модули > Расчеты с поставщиками > Поставщики > Все поставщики**.

2. Щелкните соответствующий счет поставщика, щелкните **Правка**.

3. На вкладке **Накладная и поставка** установите в поле **Рассчитать подоходный налог** значение **Да**.

   > [!NOTE] 
   > Подоходный налог не будет рассчитываться, если поле **Рассчитать подоходный налог** не включено для этого поставщика в данных.

4. Выберите группу подоходного налога в поле **Группа подоходного налога**.

5. Нажмите кнопку **Сохранить**.

Для номенклатур/услуг, облагаемых подоходным налогом, можно назначить значение по умолчанию в поле **Группа подоходного налога для номенклатуры** в разделе **Выпущенные продукты**.

1. Перейдите в раздел **Область переходов > Модули > Управление сведениями о продукте > Продукты > Выпущенные продукты**.

2. Щелкните соответствующий номер номенклатуры, щелкните **Правка**.

3. На вкладке **Покупка** щелкните **Рассчитать подоходный налог**.

   > [!NOTE] 
   > Подоходный налог не будет рассчитываться, если в поле **Рассчитать подоходный налог** не установлено значение **Да** для этой номенклатуры на вкладке **Покупка** на странице **Выпущенный продукт**.

4. Выберите группу подоходного налога для номенклатуры в списке **Группа подоходного налога для номенклатуры**.

5. Нажмите кнопку **Сохранить**.

Группы подоходного налога и группы подоходного налога для номенклатур могут быть назначены на страницах: 

- **Заказ на покупку**
- **Накладная поставщика**
- **Журнал накладных**

Группа подоходного налога и группа подоходного налога для номенклатуры по умолчанию будут перенесены в строки при создании **Заказов на покупку** и/или **Ожидающих накладных поставщиков**. Для **журнала накладных поставщиков** можно переключаться на **Рассчитать подоходный налог** и выбирать **Группа подоходного налога для номенклатуры** на вкладке **Общие** в журнале.

Временная сумма подоходного налога доступна в поле **Скорректированный подоходный налог** на вкладке **Итоги** на странице **Заказ на покупку**.

![Подоходный налог включается в заказ на покупку](media/withholding-tax-adjusted.png)

Подоходный налог рассчитывается в пункте **Журнал платежей поставщикам**. Можно вручную скорректировать соответствующие коды подоходного налога, а также фактические суммы подоходного налога на вкладке **Подоходный налог** на странице **Сопоставление проводок**.

![Удержание может быть скорректировано вручную на странице сопоставления проводок](media/withholding-tax-vendor-payment-tab.png)

Расчетная сумма подоходного налога будет вычтена из платежа поставщику и разнесена на **Счет подоходного налога** в соответствующей операции.

![Счет подоходного налога, показывающий соответствующую операцию](media/withholding-tax-adjusted.png)
