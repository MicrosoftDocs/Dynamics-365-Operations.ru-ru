# <a name="vendor-payments-workspace"></a>Рабочая область платежей поставщику

[!include [banner](../includes/banner.md)]

В рабочей области **Платежи поставщику** отображается информация, относящаяся к обработке платежей поставщикам. Эта рабочая область содержит представление **Моя работа** и страницу **Аналитика**. В представлении **Моя работа** отображаются сводные плитки, таблицы проводок по поставщикам, а также связанная информация о поставщиках. Страница **Аналитика** использует возможности Microsoft Power BI для отображения визуальных объектов, которые относятся к платежам поставщику.

## <a name="setup-needed-to-view-power-bi-content"></a>Для просмотра содержимого Power BI требуется настройка

Следующая настройка должна быть выполнена для отображения данных в визуальных элементах **Платежи поставщикам** Power BI.
1. Перейдите в раздел **Администрирование системы > Настройка > Системные параметры**, чтобы задать параметры **Системная валюта** и **Валютный курс системы**.
2. Перейдите к пункту **Главная книга > Календари > Финансовые календари**, чтобы проверить даты финансового календаря, назначенные активному периоду времени.
3. Перейдите в раздел **Главная книга > Настройка > Главная книга**, чтобы задать параметры **Валюта учета** и **Тип валютного курса**. 
4. Определите валютные курсы между валютами проводок и валютой учета, а также между валютой учета и системной валютой. Для этого перейдите в раздел **Главная книга > Валюты > Валютные курсы**.
5. Перейдите к раздел **Администрирование системы > Настройка > Хранилище объектов**, чтобы обновить агрегированное измерение **VendPaymentBIMeasureV2**.

## <a name="my-work-view"></a>Представление "Моя работа"

### <a name="summary-tiles"></a>Плитки сводки

Плитки в разделе **Сводка** позволяют получить представление о состоянии вашей информации по платежам. Вы можете видеть журналы платежей, которые еще не разнесены, просроченные накладные, всех поставщиков и заблокированных поставщиков. Из раздела **Сводка** можно создать новую выплату.

Информация в разделе **Сводка** — это информация по компании, в которую вы выполнили вход.

### <a name="vendor-transactions-grids"></a>Таблицы проводок по поставщикам

Раздел **Проводки по поставщику** содержит таблицы, в которых отображаются просроченные накладные и несопоставленные платежи. Из таблицы **Просроченные накладные** можно просмотреть историю сопоставлений для выбранной накладной. Из таблицы **Несопоставленные платежи** можно просмотреть историю сопоставлений для выбранной накладной и сопоставить накладную.

Сотрудники, обрабатывающие централизованные платежи, могут пользоваться фильтром в верхней части каждой таблицы для выбора компании. В этом случае таблица фильтруется и отображает только те компании, определенные в организационной иерархии централизованных платежей, право на просмотр которых есть у сотрудника.

Вкладка **Найти проводки** в разделе **Проводки по поставщику** позволяет выполнить поиск проводки по поставщику.

### <a name="related-information"></a>Связанные сведения

Можно просмотреть отчет **Классификация поставщиков по срокам оплаты** и отчет **Платежи по датам** с помощью ссылок в разделе **Связанные сведения** рабочей области.

## <a name="analytics-page"></a>Страница "Аналитика"

На одной странице приведены обзорные сведения, а на остальных восьми — подробные сведения о показателях платежей по расчетам с поставщиками.
