# <a name="apply-a-payment-schedule-to-the-invoice-journal"></a>Применить график платежей к журналу накладных

[!include [banner](../includes/preview-banner.md)]

В Microsoft Dynamics 365 Finance выпуска 10.0.25 график платежей теперь поддерживается в **Журнале накладных поставщиков**.

Для использования этой функции необходимо включить функцию **Применить график платежей к журналу накладных** в модуле управления функциями.

После включения функции новое поле **Графика оплаты** добавляется на странице **Журнал накладных**. При создании строки журнала накладных, если условия оплаты поддерживаются поставщиком и условия платежа выбраны в графике оплаты, поле **График оплаты** обновляется на странице **Журнал накладных**.

Можно изменить используемый график оплаты в соответствии с требованиями компании. Во время разноски журнала накладных поставщиков открытые проводки поставщика будут созданы в соответствии с графиком оплаты.

 - Чтобы просмотреть несколько открытых проводок поставщика, созданных из графика платежей, перейдите к **Расчеты с поставщиками \> Накладные \> Открыть накладные поставщика** и введите номер накладной или счет поставщика.
 - Чтобы просмотреть или настроить график оплаты, перейдите в **Расчеты с поставщиками \> Настройка оплаты \> График оплаты**.
 - Чтобы настроить условия оплаты и назначить график оплаты, перейдите на страницу **Расчеты с поставщиками \> Настройка оплаты \> Условия оплаты**.
 - Чтобы настроить условия оплаты для поставщика, перейдите **Расчеты с поставщиками \> Все поставщики**, выберите счет поставщика, а затем на вкладке **Оплата** задайте поле **Условия оплаты**.

Функция графика оплаты доступна также в процессе **Регистрация накладных поставщиков**. Если в журнале регистрации накладных выбран график оплаты, несколько строк платежа поставщика **не будут** создаваться при разноске регистрации накладных. Строки платежа поставщику будут созданы при утверждении накладной.

## <a name="limitation"></a>Ограничение

Если для ожидающей накладной поставщика график оплаты находится в заголовке накладной, имеется дополнительная страница, которая позволяет пользователям редактировать строки платежа. (Например, пользователи могут изменять срок и значение для каждой строки платежа.) Строки платежа, созданные из журнала накладных, будут иметь значение из графика оплаты.

Эта функция будет доступна для **Журнала накладных поставщиков** и **Ожидающих накладных** в будущих выпусках.
