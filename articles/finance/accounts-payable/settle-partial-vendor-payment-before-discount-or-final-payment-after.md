# <a name="settle-a-partial-vendor-payment-before-the-discount-date-with-a-final-payment-after-the-discount-date"></a>Сопоставление частичного платежа поставщика до даты скидки с окончательным платежом после даты скидки

[!include [banner](../includes/banner.md)]

Эта статья — пошаговое руководство по сценарию, где многие частичные платежи выполняются, некоторые в период скидки по оплате и другие вне периода скидки по оплате.

Компания Fabrikam покупает товары у поставщика 3057. Компания Fabrikam получает скидку на оплату в размере 1 %, если счет будет оплачен в течение 14 дней. Накладные должны быть оплачены в течение 30 дней. Поставщик также позволяет Fabrikam использовать скидки на оплату по частичным платежам. Параметры сопоставления расположены на странице **Параметры модуля расчетов с поставщиками**.

## <a name="invoice-on-june-25"></a>Накладная от 25 июня
25 июня Эйприл вводит и разносит накладную на 1 000,00 для поставщика 3057. На странице **Проводки по поставщику** Эйприл может просмотреть эту проводку.

| Операция   | Тип транзакции | Дата      | Счет | Дебетовая сумма в валюте проводки | Сумма кредита в валюте проводки | Сальдо   | Валютное |
|-----------|------------------|-----------|---------|--------------------------------------|---------------------------------------|-----------|----------|
| Inv-10020 | Счет          | 25.06.2015 | 10020   |                                      | 1 000,00                              | -1 000,00 | американский доллар      |

## <a name="partial-payment-on-july-2"></a>Частичный платеж от 2 июля
2-го июля, Эйприл хочет сопоставить 300,00 из этой накладной. На платеж распространяется скидка, потому что Fabrikam принимает скидки по частичным платежам. Следовательно, Эйприл платит 297,00 и использует скидку 3,00. Она создает журнал платежей и вводит строку для поставщика 3057. Она после этого открывает страницу **Сопоставление проводок**, чтобы отметить накладную для сопоставления.

| Пометка     | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
| Выбрано | Обычная            | Inv-10020 | 3057    | 25.06.2015 | 25.07.2015 | 10020   | -1 000,00                      | американский доллар      | -297,00          |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 09.07.2015 |
| Сумма скидки по оплате         | -10,00    |
| Использовать скидку по оплате            | Обычная    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | –3,00     |

Затем Эйприл разносит платеж. Баланс по накладной теперь — 700,00. На странице **Проводки по поставщику** Эйприл может просмотреть эту проводку.

| Операция    | Тип транзакции | Дата      | Счет | Дебетовая сумма в валюте проводки | Сумма кредита в валюте проводки | Сальдо | Валютное |
|------------|------------------|-----------|---------|--------------------------------------|---------------------------------------|---------|----------|
| Inv-10020  | Счет          | 25.06.2015 | 10020   |                                      | 1 000,00                              | -700,00 | американский доллар      |
| APP-10020  | Платеж          | 01.07.2015  |         | 297.00                               |                                       | 0,00    | американский доллар      |
| DISC-10020 | Скидка по оплате    | 01.07.2015  |         | 3,00                                 |                                       | 0,00    | американский доллар      |

## <a name="remaining-payment-on-july-15-use-cash-discount--normal"></a>Оставшийся платеж на 15 июля, использование скидки на оплату = нормальный
Эйприл платит остаток по накладной 15 июля, то есть после периода скидки. На странице **Сопоставление открытых проводок** сумма скидки не появляется в поле **Оцененная скидка по оплате**, и значение в поле **Сумма скидки по оплате** равно **0,00**. Когда Эйприл оплачивает оставшиеся 700,00, дополнительная скидка не используется.

| Пометка     | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
| Выбрано | Обычная            | Inv-10020 | 3057    | 25.06.2015 | 25.07.2015 | 10020   | -700,00                        | американский доллар      | -700,00          |

Данные по скидкам появляются внизу страницы **Сопоставление проводок**. Эйприл видит, что она уже использовала скидку 3,00.

|                              |           |
|------------------------------|-----------|

| Использовать скидку по оплате            | Обычная    |
| Скидка по оплате          | –3,00     |
| Сумма скидки по оплате | 0,00      |

Затем Эйприл разносит платеж. Когда она открывает страницу **Проводки по поставщику**, она видит, что накладная имеет сальдо 0,00. Также она видит, что платежей два. Один платеж — на 297,00 со скидкой 3,00, другой платеж — на 700,00.

## <a name="remaining-payment-on-july-15-use-cash-discount--always"></a>Оставшийся платеж на 15 июля, использование скидки на оплату = всегда
Если поставщик разрешает Эйприл использовать скидку, хотя она платит после окончания периода предоставления скидок, она может изменить значение в поле **Использовать скидку по оплате** на **Всегда**. Настройка **Вычислять скидки по оплате для частичных платежей** переопределена, скидка использована. Сумма платежа — 693,00, остальные 7,00 — скидка.

|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------------|---------------------------------------|----------|------------------|
| Выбрано | Всегда            | Inv-10020 | 3057    | 25.06.2015 | 25.07.2015 | 10020   | 700.00                               |                                       | американский доллар      | -693,00          |

Данные по скидкам появляются внизу страницы **Сопоставление проводок**.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 09.07.2015 |
| Сумма скидки по оплате         | 7,00      |

| Скидка по оплате          | –3,00     |
