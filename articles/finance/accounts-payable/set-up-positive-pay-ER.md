# <a name="set-up-positive-pay-files-by-using-electronic-reporting"></a>Настройка файлов положительных платежей с помощью электронной отчетности

Настройте положительные платежи для формирования электронного списка чеков, предоставляемых банку. Затем, когда чек передается в банк, банк сравнивает чек со списком чеков. Если чек соответствует чеку в списке, банк производит клиринг чека. Если чек не соответствует чеку в списке, банк отправляет чек на рассмотрение.

## <a name="set-up-the-electronic-reporting-configuration"></a>Настройка конфигурации электронной отчетности

1. Перейдите в раздел **Рабочие области \> Электронная отчетность**.
2. На плитке поставщика конфигурации **Microsoft** выберите **Репозитории**.
3. Выберите **Глобальный**, затем выберите **Открыть**.
4. Если необходимо установить соединение с репозиторием, выберите синюю ссылку в диалоговом окне.
5. В списке конфигураций найдите и выберите **Модель положительных платежей \> Формат положительного платежа**.
6. На экспресс-вкладке **Версии** выберите последнюю версию, затем выберите **Импорт**.

## <a name="set-up-a-positive-pay-format"></a>Настройка формата положительного платежа

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Настройка \> Форматы положительного платежа**.
2. Выберите **Создать**.
3. Задайте поля **Формат платежа** и **Описание**.
4. Установите флажок **Универсальный электронный формат экспорта**.
5. Задайте в поле **Конфигурация формата экспорта** значение **Формат положительного платежа**.

## <a name="assign-a-positive-pay-format-to-a-bank-account"></a>Назначение формата положительных платежей банковскому счету

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Банковские счета \> Банковские счета**.
2. Откройте банковский счет.
3. На экспресс-вкладке **Общие** задайте в поле **Формат положительного платежа** формат, который был создан ранее.
4. Установите в поле **Дата начала положительного платежа** текущую дату.

## <a name="generate-a-positive-pay-file"></a>Создать файл положительных платежей

1. Перейдите в раздел **Управление банком и кассовыми операциями \> Банковские счета \> Банковские счета**.
2. Откройте банковский счет, для которого настроен положительный платеж.
3. Выберите **Управление платежами \> Положительный платеж \> Файл положительного платежа**.
4. Установите значение в поле **Дата прекращения**. Чеки, которые были созданы до этой даты, будут включены.

Будет загружен результирующий XML-файл.
