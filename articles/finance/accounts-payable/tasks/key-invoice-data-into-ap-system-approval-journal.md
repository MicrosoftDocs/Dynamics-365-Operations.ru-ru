# <a name="key-invoice-data-into-accounts-payable-using-an-approval-journal"></a>Ввод данных счета в модуль расчетов с поставщиками с использованием журнала утверждения

[!include [banner](../../includes/banner.md)]

В этой статье показано, как использовать реестр накладных для создания накладных и последующего использования журнала утверждения для обновления счетов расходов.

## <a name="create-and-post-and-invoice"></a>Создание и разноска накладной
1. В области переходов выберите **Модули > Расчеты с поставщиками > Накладные > Регистрация накладных**.
2. Выберите **Создать**.
3. Выберите имя регистра накладных для использования.
4. Выберите **Строки**, чтобы открыть регистр и ввести строки расходов.
5. Выберите поставщика. Например, введите или выберите `US-104`.
6. В поле **Накладная** введите значение.
7. В поле **Описание** введите значение.
8. В поле **Кредит** введите число.
9. В поле **Кем утверждено** выберите утверждающего в раскрывающемся меню.
10. Выберите **Разнести**.

## <a name="approve-an-invoice"></a>Утверждение накладной
1. В области переходов выберите **Модули > Расчеты с поставщиками > Накладные > Утверждение накладной**.
2. Выберите **Создать**.
3. Выберите имя журнала утверждений накладных для использования.
4. Выберите **Строки** для отображения страницы, на которой можно будет выбрать накладные, которые требуется утвердить.
5. Выберите **Найти ваучеры** для отображения всех накладных, готовых к утверждению.
6. Отметьте созданную накладную, затем щелкните **Выбрать**. Выбранные выше ваучеры перемещаются в этот список после их выбора.  
7. Нажмите **ОК**.
8. Выберите поле **Номер счета** для добавления счета расходов в накладную.
9. Введите номер счета и выйдите из поля. Например, введите `600120`.
10. Выберите **Разнести**.
11. Выберите **Ваучер** для просмотра разнесенных записей. Счет утверждения ожидающих накладных реверсируется и заменяется на счет фактических расходов.
