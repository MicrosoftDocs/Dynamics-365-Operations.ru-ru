# <a name="define-vendor-payment-fees"></a>Определение сборов по платежам поставщиков

[!include [banner](../../includes/banner.md)]

Настройте сборы по платежам поставщику. В этой задаче используется демонстрационная компания USMF.

1. Перейдите в раздел "Расчеты с поставщиками" > "Настройка платежей" > "Сборы по платежам".
2. Щелкните "Создать".
3. В поле "Код сбора" введите значение.
    * Код сбора должен описывать ситуации, когда этот сбор будет использоваться, например "Сбор_EFT".  
4. В поле "Имя" введите значение.
5. В поле "Описание сборов" введите значение.
    * Добавьте описание, чтобы предоставить дополнительные сведения о том, когда оценивается сбор.  
6. В поле "Накладные расходы" выберите значение "Поставщик" или "Главная книга".
    * Главная книга используется, когда сбор будет списан в расход в организации.  Поставщик используется, когда сбор будет оценен для поставщика.  
7. Введите счет ГК, для которого сбор будет списан в расход.
    * Этот параметр доступен только в случае, если выбрано значение "Главная книга" для параметра "Накладные расходы".  
8. Выберите журнал, в котором может использоваться этот сбор. 
    * Для сборов по платежам поставщику можно выбрать журнал "Выплата поставщику".  
9. Нажмите кнопку "Сохранить".
10. Щелкните "Настройка сборов по платежам".
    * Перейдите к настройке сборов по платежам, чтобы указать, когда сбор должен использоваться по умолчанию в выбранном журнале.  
11. Выберите значение "Таблица", "Группа" или "Все".
    * Значение "Таблица" используется для выбора одного банковского счета, значение "Группа" используется для выбора банковской группы, значение "Все" используется для указания того, что данная настройка сборов применяется ко всем банковским счетам.  
12. Выбор банковскую группу или банковский счет.
    * В результатах поиска отобразится банковская группа, если выбрано значение "Группа", или банковские счета, если выбрано значение "Таблица".  
13. Выберите способ оплаты, для которого будет оцениваться этот сбор.
14. Выберите спецификацию оплаты для выбранного способа оплаты.
    * Спецификация оплаты используется при использовании электронного платежа как способа оплаты.  
15. Выберите значение сбора: процент, сумма или интервал.
16. Введите процент или сумму сбора.
    * Если для параметра "Сбор" задано значение "Процент", введите процент. Если для параметра "Сбор" задано значение "Сумма", введите сумму сбора. Если для параметра "Сбор" задано значение "Интервал", воспользуйтесь вкладкой "Интервал", чтобы определить многоуровневые сборы.  
17. В поле "Валюта сбора" выберите валюту, в которой будет оцениваться сбор.
    * Эта валюта используется для сбора. Валюта платежа используется для определения того, когда должно оцениваться правило сбора на основании валюты платежа. Например, ваш банк может взимать сбор при осуществлении платежа в евро, но все другие платежи не оцениваются как сбор.  
18. Нажмите кнопку "Сохранить".
