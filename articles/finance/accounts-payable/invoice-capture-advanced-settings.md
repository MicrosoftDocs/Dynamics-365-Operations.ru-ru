# <a name="invoice-capture-solution-advanced-settings"></a>Расширенные параметры решения Invoice Capture

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

В этой статье содержится информация о расширенных параметрах в решении Invoice Capture.

## <a name="create-additional-connections-for-channels"></a>Создание дополнительных подключений для каналов

Необходимо создать подключения к электронной почте или хранилищу файлов, чтобы включить мониторинг входящих накладных с других каналов. Необходимо зарегистрировать подключения в начале, чтобы предоставить доступ для автоматических потоков, используемых в решении.

Для импорта накладных используются следующие типы подключений:

- Office 365 Outlook
- Outlook.com
- OneDrive
- SharePoint

Канал импорта накладных будет использовать подключения в дальнейших шагах настройки. Прежде чем пользователи смогут создать канал определенного подключения, роль безопасности **Администратор** должна быть предоставлена им, и они должны создать подключения.

Чтобы создать подключение к Microsoft Dataverse, выполните следующие действия.

1. Перейдите к пункту **Администрирование системы \> Решение по умолчанию**.
2. Выберите **Создать**, затем выберите **Ссылка на подключение**.
3. В поле **Отображаемое имя** введите имя.
4. Выберите **Microsoft Dataverse** в качестве соединителя.
5. Если настройка подключения выполняется в первый раз, выберите **Создать подключение**.
6. В появившемся диалоговом окне создайте подключение Dataverse, затем выберите **Создать**.
7. Введите учетную запись и пароль Dataverse.
8. После прохождения проверки перейдите на страницу подключения, выберите **Обновить**, выберите учетную запись, затем выберите **Создать**.

Чтобы создать подключение по электронной почте или к хранилищу файлов, выполните следующие действия.

1. На странице **Создание подключения** в поле **Тип подключения** выберите **Office 365 Outlook**.
2. Для подключения к электронной почте можно выбрать **Outlook.com** или **Office 365 Outlook** в качестве соединителя. Для подключения к хранилищу файлов можно выбрать **OneDrive** или **SharePoint**.

Чтобы просмотреть существующие подключения, перейдите к пункту **Решение по умолчанию \> Объекты \> Ссылки на подключение**. Пользователь, создающий каналы, должен иметь по крайней мере одно подключение Dataverse в дополнение к конкретным подключениям к электронной почте или хранилищу файлов. Создателем нового канала должен быть владелец подключения.
