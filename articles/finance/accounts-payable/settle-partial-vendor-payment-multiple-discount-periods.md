# <a name="settle-a-partial-vendor-payment-that-has-multiple-discount-periods"></a>Сопоставление частичного платежа поставщика с несколькими периодами скидок

[!include [banner](../includes/banner.md)]

В этой статье рассматривается сценарий, в котором производится несколько частичных платежей поставщику, предлагающему несколько скидок по оплате. 

Поставщик 3054 предлагает Fabrikam скидку по оплате в размере 2 %, если накладная оплачивается за пять дней и скидку по оплате в размере 1 %, если накладная оплачивается в течение 14 дней.

## <a name="invoice"></a>Счет
28 июня Эйприл создает накладную на 1 000,00 для поставщика 3054. На странице **Проводки по поставщику** Эйприл может просмотреть эту проводку.

| Операция   | Дата      | Счет | Дебетовая сумма в валюте проводки | Сумма кредита в валюте проводки | Сальдо   | Валюта |
|-----------|-----------|---------|--------------------------------------|---------------------------------------|-----------|----------|
| Inv-10060 | 28.06.2020 | 10060   |                                      | 1,000.00                              | -1 000,00 | USD      |

Следующие даты и суммы скидки по оплате доступны для данной накладной.

| Дата скидки по оплате | Сумма скидки по оплате | Сумма в валюте проводки |
|--------------------|----------------------|--------------------------------|
| 03.07.2020           | 20.00                | 980.00                         |
| 12.07.2020          | 10.00                | 990.00                         |
| 25.07.2020          | 0,00                 | 1,000.00                       |

## <a name="payment-on-july-2"></a>Платеж от 2 июля
2-го июля Эйприл хочет заплатить 300,00 по этой накладной. Она создает одноразовую оплату путем использования страницы **Журнал платежей** в модуле "Расчеты с поставщиками". Она добавляет строку для поставщика 3054 и входит сумму платежа **300,00**. Эйприл после этого открывает страницу **Сопоставление проводок**, чтобы можно было отметить накладную, которая будет сопоставлена. Она обновляет значение в поле **Сумма сопоставления** на **300,00** и замечает, что значение в поле **Сумма скидки по оплате** изменяется на **6,12**. Поскольку этот платеж осуществляется в первый период скидки, используется скидка 2 процента.

| Пометка | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валюта | Сумма сопоставления |
|------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
|      | Обычное округление            | Inv-10060 | 3054    | 28.06.2020 | 28.07.2020 | 10060   | 1,000.00                       | USD      | 300,00           |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**.

| Поле                        | Значение     |
|------------------------------|-----------|
| Дата скидки по оплате           | 02.07.2020 |
| Сумма скидки по оплате         | -20,00    |
| Использовать скидку по оплате            | Обычное округление    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | -6,12     |

Так как доступна скидка по оплате, Эйприл хочет изменить сумму оплаты так, чтобы общая сопоставленная сумма составляла 300,00 как для оплаты, так и для скидки по оплате. Она изменяет значение в поле **Сумма сопоставления** на **294,00** и замечает, что значение в поле **Сумма скидки по оплате** изменяется на **6,00**.

| Пометка | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валюта | Сумма сопоставления |
|------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
|      | Обычное округление            | Inv-10060 | 3054    | 28.06.2020 | 28.07.2020 | 10060   | 1,000.00                       | USD      | 294,00           |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**.

| Поле                        | Значение     |
|------------------------------|-----------|
| Дата скидки по оплате           | 02.07.2020 |
| Сумма скидки по оплате         | -20,00    |
| Использовать скидку по оплате            | Обычное округление    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | -6,00     |

Эйприл разносит платеж. На странице **Проводки по поставщику** она может просмотреть эту проводку. Она видит, что 300,00 применены к накладной. Эта сумма включает скидку 6,00. Поэтому остаток — 700,00.

| Операция    | Дата      | Счет | Дебетовая сумма в валюте проводки | Сумма кредита в валюте проводки | Сальдо | Валюта |
|------------|-----------|---------|--------------------------------------|---------------------------------------|---------|----------|
| Inv-10060  | 28.06.2020 | 10060   |                                      | 1,000.00                              | -700,00 | USD      |
| APP-10060  | 02.07.2020  |         | 294,00                               |                                       | 0,00    | USD      |
| DISC-10060 | 02.07.2020  |         | 6,00                                 |                                       | 0,00    | USD      |

## <a name="payment-on-july-8"></a>Платеж от 8 июля
8-го июля Эйприл делает дополнительный платеж по накладной. Чтобы ввести количество, она открывает страницу **Сопоставление проводок** и после этого выбирает вкладку **Скидка по оплате**. Она видит даты и суммы двух доступных скидок по оплате. Поскольку этот платеж осуществляется во второй период скидки, доступна скидка 1 процент или 5,00. Эта сумма рассчитывается как половина скидки 1 % на 1 000,00 или половина от 10,00.

| Дата скидки по оплате | Сумма скидки по оплате | Сумма в валюте проводки |
|--------------------|----------------------|--------------------------------|
| 03.07.2020           | 20.00                | 680,00                         |
| 12.07.2020          | 10.00                | 690,00                         |
| 25.07.2020          | 0,00                 | 700.00                         |

Эйприл решает оплатить 495,00 и получить скидку по оплате 5,00. Поэтому полная сопоставленная сумма равна 500,00.
