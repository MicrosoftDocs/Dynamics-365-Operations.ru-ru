# <a name="create-a-bank-facility-agreement-for-a-letter-of-credit"></a>Создание договора о банковских услугах для аккредитива

[!include [banner](../../includes/banner.md)]

В этой задаче представлены шаги создания договора о банковских услугах для обработки аккредитива. До выполнения этой задачи необходимо настроить банковские услуги и профили разноски.  В данной задаче используется демонстрационная компания USMF.  


## <a name="create-bank-facility-agreement"></a>Создание договорах о банковских услугах
1. Перейдите в раздел **Управление банком и кассовыми операциями > Аккредитивы > Договоры о банковских услугах**.
2. Нажмите кнопку **Создать**.
3. В поле **Номер договора** введите номер договора согласно договору с банком.
4. В поле **Банковский счет** введите номер счета в выставляющем банке.
5. В списке перейдите по ссылке в выбранной строке.
6. В поле **Дата начала** введите дату и время.
7. В поле **Дата окончания** введите дату и время.
8. Разверните или сверните раздел "Общие".
9. Щелкните **Добавить строку**.
10. В поле **Тип ссуды** нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
11. Поиск и выбор требуемой записи в списке.
12. В списке перейдите по ссылке в выбранной строке.
13. В поле **Лимит** введите сумму ссуды, согласованную с банком.
14. Нажмите кнопку **Сохранить**.
15. Щелкните **Развернуть**, чтобы открыть раскрывающееся диалоговое окно.
16. В поле **Новый номер договора** введите значение.
17. В поле **Дата окончания** введите дату и время.
18. Щелкните **Развернуть**.
19. Закройте страницу.
