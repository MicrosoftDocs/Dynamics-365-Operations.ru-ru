# <a name="set-up-retail-products"></a><span data-ttu-id="aa254-103">Настройка розничных продуктов</span><span class="sxs-lookup"><span data-stu-id="aa254-103">Set up retail products</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="aa254-104">В этой статье описывается, как настроить розничные продукты в Microsoft Dynamics 365 for Retail.</span><span class="sxs-lookup"><span data-stu-id="aa254-104">This article describes how to set up retail products in Microsoft Dynamics 365 for Retail.</span></span>

<span data-ttu-id="aa254-105">Прежде чем можно будет предлагать продукты для перепродажи в каналах розничной торговли, необходимо создать и настроить продукты в Dynamics 365 for Retail.</span><span class="sxs-lookup"><span data-stu-id="aa254-105">Before you can offer products for resale in your retail channels, you must create and configure the products in Dynamics 365 for Retail.</span></span> <span data-ttu-id="aa254-106">Характеристики продукта в Dynamics 365 for Retail используются в модуле розничной торговли для создания продуктов в справочнике продуктов для всей организации.</span><span class="sxs-lookup"><span data-stu-id="aa254-106">Retail uses the product features in Dynamics 365 for Retail to create organization-wide products in the product master.</span></span> <span data-ttu-id="aa254-107">Можно создать продукты, определить свойства и атрибуты продукции и назначить продукты иерархиям розничных категорий.</span><span class="sxs-lookup"><span data-stu-id="aa254-107">You can create the products, define the product properties and attributes, and assign the products to retail category hierarchies.</span></span> <span data-ttu-id="aa254-108">Для того чтобы сделать продукты доступными розничным каналам и затем добавить их в активный ассортимент, необходимо предоставить их юридическим лицам, для которых они доступны.</span><span class="sxs-lookup"><span data-stu-id="aa254-108">To make the products available to your retail channels and add them to an active assortment, you must release the products to the legal entities where they are available.</span></span> <span data-ttu-id="aa254-109">Чтобы настроить продукты для продажи в розничной сети выполните следующие задачи.</span><span class="sxs-lookup"><span data-stu-id="aa254-109">To set up the products that you sell by using retail channels, complete the following tasks.</span></span>

1. <span data-ttu-id="aa254-110">Определите иерархию розничных продуктов.</span><span class="sxs-lookup"><span data-stu-id="aa254-110">Define a retail product hierarchy.</span></span> <span data-ttu-id="aa254-111">С помощью функций иерархии категорий в Dynamics 365 for Retail можно определить розничные иерархии категорий, позволяющие сгруппировать и классифицировать предоставляемые розничным каналам продукты.</span><span class="sxs-lookup"><span data-stu-id="aa254-111">By using the category hierarchy features in Dynamics 365 for Retail, you can define retail category hierarchies to group and categorize the products that you distribute to your retail channels.</span></span> <span data-ttu-id="aa254-112">Пользовательские и системные атрибуты определяются на уровне категорий.</span><span class="sxs-lookup"><span data-stu-id="aa254-112">User-defined and system attributes can be defined at the category level.</span></span> <span data-ttu-id="aa254-113">Затем все продукты присваиваются категории, которая наследует эти атрибуты.</span><span class="sxs-lookup"><span data-stu-id="aa254-113">Then, all products that are assigned to the category inherit those attributes.</span></span> <span data-ttu-id="aa254-114">Можно определить несколько иерархий категорий, и каждый продукт присвоить нескольким иерархиям.</span><span class="sxs-lookup"><span data-stu-id="aa254-114">Multiple category hierarchies can be defined, and each product can be assigned to multiple hierarchies.</span></span> <span data-ttu-id="aa254-115">Однако в одной иерархии каждый продукт может быть присвоен только одной категории.</span><span class="sxs-lookup"><span data-stu-id="aa254-115">However, in a single hierarchy, each product can be assigned to only one category.</span></span>
2. <span data-ttu-id="aa254-116">Добавьте продукты и варианты продуктов в шаблон продуктов.</span><span class="sxs-lookup"><span data-stu-id="aa254-116">Add products and product variants to the product master.</span></span> <span data-ttu-id="aa254-117">Продукты, добавляемые в шаблон продуктов, представляют глобальный список продуктов.</span><span class="sxs-lookup"><span data-stu-id="aa254-117">Products that are added to the product master represent a global list of products.</span></span> <span data-ttu-id="aa254-118">Можно добавить продукты вручную, по одному или импортировать данные о продуктах от поставщиков.</span><span class="sxs-lookup"><span data-stu-id="aa254-118">You can add products manually, one at a time, or you can import product data from your vendors.</span></span>
3. <span data-ttu-id="aa254-119">Отпуск продуктов юридическому лицу.</span><span class="sxs-lookup"><span data-stu-id="aa254-119">Release the products to legal entities.</span></span> <span data-ttu-id="aa254-120">Только продукты, предоставленные юридическим лицам, будут доступны каналам розничной торговли.</span><span class="sxs-lookup"><span data-stu-id="aa254-120">Only products that have been released to legal entities can be made available to your retail channels.</span></span> <span data-ttu-id="aa254-121">При первом определении продукта он определяется на общеорганизационном уровне.</span><span class="sxs-lookup"><span data-stu-id="aa254-121">When you first define a product, you define it on an organization-wide level.</span></span> <span data-ttu-id="aa254-122">Затем можно выбрать одно или несколько юридических лиц и предоставить им продукт.</span><span class="sxs-lookup"><span data-stu-id="aa254-122">You can then select one or more legal entities to release the product to.</span></span> <span data-ttu-id="aa254-123">После этого продукт становится доступным нескольким розничным каналам в организации.</span><span class="sxs-lookup"><span data-stu-id="aa254-123">The product then becomes available to multiple retail channels across your organization.</span></span> <span data-ttu-id="aa254-124">Вы можете использовать эту функцию для создания продукта один раз, затем добавить или обновить атрибуты и свойства продукта в одном месте и распространить продукт по организации, по розничным каналам, в которых он будет доступен.</span><span class="sxs-lookup"><span data-stu-id="aa254-124">You can use this functionality to create a product one time, add and update product attributes and properties in one place, and then distribute the product across your organization, to the retail channels where it's available.</span></span>
4. <span data-ttu-id="aa254-125">Добавьте продукты в ассортименты.</span><span class="sxs-lookup"><span data-stu-id="aa254-125">Add products to assortments.</span></span> <span data-ttu-id="aa254-126">Ассортимент представляет набор продуктов, которые вы предлагаете в ваших каналах розничной торговли.</span><span class="sxs-lookup"><span data-stu-id="aa254-126">An assortment represents a collection of products that you offer in your retail channels.</span></span> <span data-ttu-id="aa254-127">Можно определить один или несколько ассортиментов, каждый продукт может быть назначен одному или нескольким ассортиментам.</span><span class="sxs-lookup"><span data-stu-id="aa254-127">You can define one or more assortments, and each product can be assigned to one or more assortments.</span></span> <span data-ttu-id="aa254-128">Чтобы назначить продукты розничным каналам, нужно назначить им ассортименты.</span><span class="sxs-lookup"><span data-stu-id="aa254-128">To assign products to retail channels, you assign the assortments to those retail channels.</span></span> <span data-ttu-id="aa254-129">При создании ассортимента продукты, которые еще не были предоставлены юридическим лицам, можно добавить в ассортимент.</span><span class="sxs-lookup"><span data-stu-id="aa254-129">When you create an assortment, you can add products that haven't yet been released to a legal entity.</span></span> <span data-ttu-id="aa254-130">Однако чтобы продукты стали доступны в розничной сети, их нужно предоставить юридическим лицам.</span><span class="sxs-lookup"><span data-stu-id="aa254-130">However, you must release the products to a legal entity before those products can be made available to the retail channels.</span></span>

6. <span data-ttu-id="aa254-133">Добавьте продукты в каталоги.</span><span class="sxs-lookup"><span data-stu-id="aa254-133">Add products to catalogs.</span></span> <span data-ttu-id="aa254-134">Хотя этот шаг необязателен для пункта продаж (POS), интернет-магазины требуют, чтобы продукты были включены по крайней мере в один каталог.</span><span class="sxs-lookup"><span data-stu-id="aa254-134">Although this step is optional for POS, online stores require that products be included in at least one catalog.</span></span>
