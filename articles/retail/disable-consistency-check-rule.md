# <a name="disable-rules-in-the-retail-transaction-consistency-checker"></a>Отключение правил в программе проверки согласованности проводок 

[!include [banner](../includes/banner.md)]

[!include [banner](../includes/preview-banner.md)]

Многие розничные компании используют уникальные бизнес-сценарии и процессы. Поэтому не все правила, по умолчанию включенные в программу проверки согласованности проводок розничной торговли, применимы для всех продавцов. Для учета этих различий в Microsoft Dynamics 365 Retail предусмотрена возможность отключения неприменимых правил.

Чтобы посмотреть список доступных правил программы проверки согласованности проводок и статус каждого правила, перейдите в раздел **Розничная торговля \> Настройка центрального офиса \> Параметры \> Параметры розничной торговли** и откройте вкладку **Проверка проводок**.

По умолчанию каждое правило имеет статус **Включено**. Поэтому для проверки проводок перед их передачей в журналы операций розницы используются все правила. Чтобы отключить правило, измените его статус на **Отключено**. Отключенные правила не учитываются при проверке проводок в процессе формирования журнала операций.

Чтобы обойти весь процесс проверки независимо от состояния правил, выберите **Розничная торговля \> Настройка центрального офиса \> Параметры \> Параметры розничной торговли** и на вкладке **Проверка проводок** выберите для параметра **Отключение программы проверки согласованности для проводок розничной торговли** значение **Да**. Если для этого параметра установлено значение **Нет**, его невозможно изменить на **Да** в пользовательском интерфейсе.
