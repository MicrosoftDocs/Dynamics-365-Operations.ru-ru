# <a name="create-online-channel-and-define-channel-attributes"></a>Создание канала продаж через интернет и определение атрибутов канала

[!include[task guide banner](../includes/task-guide-banner.md)]

Эта процедура содержит указания по созданию нового канала и его добавлению в организационную иерархию. Необходимо создать организационную иерархию перед созданием нового интернет-канала. В этой процедуре используется компания с демонстрационными данными USRT.


## <a name="create-a-new-online-channel"></a>Создание нового интернет-канала
1. Перейдите в раздел "Розничная торговля и коммерция" > "Каналы" > "Интернет-магазины".
2. Щелкните "Создать".
3. В поле "Имя" введите значение.
4. В поле "Склад" введите или выберите значение.
5. В поле "Часовой пояс магазина" выберите параметр.
6. В поле "Клиент по умолчанию" введите или выберите значение.
7. В поле "Адресная книга клиентов" введите или выберите значение.
8. В поле "Условия оплаты" введите или выберите значение.
9. В поле "Способ оплаты" введите или выберите значение.
10. В поле "Профиль уведомлений по электронной почте" введите или выберите значение.
11. Разверните раздел "Финансовые аналитики".
12. В поле "BusinessUnit" введите или выберите значение.
    * Аналогично задайте значение всех остальных аналитик по умолчанию.  
13. Нажмите кнопку "Сохранить".

## <a name="add-the-online-channel-to-organization-hierarchy"></a>Добавление интернет-канала в организационную иерархию
1. Закройте страницу.
2. Перейдите в раздел "Управление организацией" > "Организации" > "Организационные иерархии".
3. В списке найдите и выберите требуемую запись.
4. Щелкните "Вид".
5. Щелкните "Изменить".
    * Вы можете выбрать любой узел иерархии, под который вы хотите поместить новый канал.  
6. Нажмите Вставить.
7. Щелкните "Канал розничной торговли".
8. Нажмите кнопку "OК".
9. Щелкните "Опубликовать", чтобы открыть раскрывающееся диалоговое окно.
10. В поле "Действует с" введите дату и время.
11. Щелкните "Опубликовать".
