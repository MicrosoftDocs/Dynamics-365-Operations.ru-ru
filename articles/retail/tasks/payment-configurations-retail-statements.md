# <a name="configure-payment-method-settings-that-affect-retail-statements"></a>Настройка параметров метода оплаты, влияющих на розничные отчеты

[!include [task guide banner](../includes/task-guide-banner.md)]

Эта процедура демонстрирует конфигурации способов оплаты в розничном магазине, которые влияют на то, как создаются и разносятся журналы операций розницы.

В данной записи используется демонстрационная компания USRT.

1. Перейдите в раздел Розничная торговля и коммерция > Каналы > Магазины розничной торговли > Все розничные магазины.
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.
4. В области действий щелкните "Настроить".
5. Щелкните "Способы оплаты".
6. Разверните или сверните раздел "Разноска".
7. Щелкните "Изменить".
    * Выберите, следует ли суммы, полученные для этого способа оплаты, разносить на счет ГК или на банковский счет.  
    * Выберите счет, на который должны быть разнесены суммы, полученные через этот способ оплаты.  
    * Выберите счет для разноски возможных расхождений между общей полученной суммой транзакции и суммой, учтенной для данного способа оплаты.  
    * В этом поле можно ввести сумму для управления тем, когда сумма расхождений должна быть разнесена на другой счет расхождения. Это поле можно использовать, чтобы отслеживать большие расхождения.  
    * Выберите счет для разноски возможных расхождений между общей полученной суммой транзакции и учтенной суммой, когда она превышает значение, указанное в поле "Максимальная сумма расхождения".  
    * Выберите "Да", чтобы разнести сумму инкассации на отдельный счет.  
    * В этом поле можно выбрать, следует ли суммы инкассации разносить на счет ГК или на банковский счет.  
    * Выберите счет для разноски сумм инкассации.  
    * Выберите тип банковской проводки, который будет использоваться при разноске суммы инкассации на банковский счет.  
    * Выберите "Да", чтобы разнести сумму сдачи наличных средств в сейф на отдельный счет.  
    * Выберите, следует ли суммы сдачи наличных средств в сейф разносить на счет ГК или на банковский счет.  
    * Выберите счет для разноски сумм сдачи наличных средств в сейф.  
8. Нажмите кнопку "Сохранить".
