# <a name="define-call-center-channel-and-channel-attributes"></a>Определение канала центра обработки вызовов и атрибутов этого канала

[!include [task guide banner](../includes/task-guide-banner.md)]

Эта процедура содержит указания по созданию нового канала розничной торговли и определению атрибутов канала. В качестве компании с демонстрационными данными для создания этой задачи используется USRT. Эта процедура предназначена для роли "ИТ розничной торговли".


## <a name="create-new-store"></a>Создание нового магазина
1. Пойдите к Все рабочие области > Развертывание канала.
2. Щелкните "Новый канал".
3. Щелкните "Магазин".
4. В поле "Имя" введите значение.
5. В поле "Номер магазина" введите значение.
6. В поле "Склад" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
7. В списке найдите и выберите требуемую запись.
8. В списке перейдите по ссылке в выбранной строке.
9. В поле "Часовой пояс магазина" выберите параметр.
10. В поле "Профиль канала" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
11. В списке перейдите по ссылке в выбранной строке.
12. В поле "Язык" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
13. В списке найдите и выберите требуемую запись.
14. В списке перейдите по ссылке в выбранной строке.
15. В поле "Налоговая группа" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
16. В списке найдите и выберите требуемую запись.
17. В списке перейдите по ссылке в выбранной строке.
18. В поле "Адресная книга клиентов" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
    * Выберите адресную книгу, используемую для связывания клиентов с этим магазином.  
19. В списке найдите и выберите требуемую запись.
20. В списке перейдите по ссылке в выбранной строке.
21. Щелкните Выбрать.
22. В поле "Адресная книга сотрудников" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
    * Выберите адресную книгу, используемую для связывания кассиров с этим каналом.  
23. В списке найдите и выберите требуемую запись.
24. В списке перейдите по ссылке в выбранной строке.
25. Щелкните Выбрать.
26. В поле "Клиент по умолчанию" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
27. В списке перейдите по ссылке в выбранной строке.
28. Разверните или сверните раздел "Макет экрана".
29. В поле "Код макета экрана" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
    * Выберите макета экрана POS по умолчанию для этого магазина.  
30. В списке найдите и выберите требуемую запись.
31. В списке перейдите по ссылке в выбранной строке.
32. В области действий щелкните "Настроить".
33. Щелкните "Атрибуты канала".
34. Щелкните "Создать".
35. В поле "Имя" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
36. В списке найдите и выберите требуемую запись.
37. В списке перейдите по ссылке в выбранной строке.
38. Нажмите кнопку "Сохранить".
39. Закройте страницу.
40. В области действий щелкните "Настроить".
41. Щелкните "Способы оплаты".
42. Щелкните "Создать".
43. В поле "Способ оплаты" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
44. В списке перейдите по ссылке в выбранной строке.
45. Разверните или сверните раздел "Разноска".
46. В поле "Номер счета" укажите требуемые значения.
47. Нажмите кнопку "Сохранить".
48. Закройте страницу.
49. В области действий щелкните "Настроить".
50. Щелкните "Декларирование наличности".
51. Щелкните "Создать".
52. В поле "Сумма в валюте проводки" введите число.
53. В поле "Валюта" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
54. В списке найдите и выберите требуемую запись.
55. В списке перейдите по ссылке в выбранной строке.
56. Нажмите кнопку "Сохранить".
57. Закройте страницу.
58. В области действий щелкните "Настроить".
59. Щелкните "Назначение групп указателей магазинов".
60. Щелкните "Создать".
61. В списке пометьте выбранную строку.
62. В поле "Группа указателей" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
63. В списке найдите и выберите требуемую запись.
64. В списке перейдите по ссылке в выбранной строке.
65. Нажмите кнопку "Сохранить".
66. Закройте страницу.
