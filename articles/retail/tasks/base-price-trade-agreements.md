# <a name="base-price-and-trade-agreements"></a>Базовая цена и коммерческие соглашения

[!INCLUDE [task guide banner](../includes/task-guide-banner.md)]

Эта процедура содержит инструкции по созданию коммерческих предложений по ценам продажи для конкретных каналов. В этой процедуре используется компания с демонстрационными данными USRT.

1. Перейдите в раздел "Розничная торговля и коммерция" > "Цены и скидки" > "Группы цен" > "Все группы цен".
    * Ценовые группы определяют, каким образом коммерческие соглашения назначаются конкретным каналам. Использование ценовых групп для назначения коммерческих соглашений каналам делает возможным ценообразование на уровне каналов.  
2. Щелкните "Создать".
3. В поле "Группы цен" введите значение.
4. В поле "Имя" введите значение.
5. Нажмите кнопку "Сохранить".
6. Закройте страницу.
7. Перейдите в раздел Розничная торговля и коммерция > Каналы > Магазины розничной торговли > Все розничные магазины.
8. В списке выберите "Нью-Йорк".
9. В области действий щелкните "Магазин".
10. Щелкните "Ценовые группы".
11. Щелкните "Создать".
12. В поле "Группа цен" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
13. В списке найдите и выберите требуемую запись.
14. Нажмите кнопку "Сохранить".
15. Закройте страницу.
16. Закройте страницу.
17. Перейдите в раздел "Розничная торговля и коммерция" > "Продукты и категории" > "Выпущенные продукты по категориям".
18. В списке перейдите по ссылке в выбранной строке.
19. Выберите Изменить.
20. Переключите развертывание раздела "Продажа".
21. В поле "Цена" введите число.
    * Эта цена используется, если не найдены подходящие коммерческие соглашения.  
22. Нажмите кнопку "Сохранить".
23. В области действий щелкните "Продажа".
24. Щелкните "Создать коммерческие соглашения".
25. Нажмите Создать.
26. В поле "Имя" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
27. В списке выберите "Розница".
    * В демонстрационных данных имя журнала "Розница" по умолчанию имеет связь "Цена (продажи)". Это означает, что все новые созданные строки по умолчанию будут иметь цены продажи из коммерческих соглашений.  
28. Щелкните "Строки".
29. В поле "Код счета" выберите "Группа".
30. В поле "Выбор счета" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
31. В списке найдите и выберите требуемую запись.
    * Это завершить связь от канала к ценовой группе и коммерческому соглашению.  
32. В поле "Связь номенклатуры" введите значение.
33. В поле "Сумма в валюте" введите число.
34. Установите или снимите флажок "Найди далее".
    * Если поле "Найти далее" имеет значение "Да" модуль ценообразования продолжает поиск применимых коммерческих соглашений с более низкой ценой продажи. Когда поле "Найти далее" имеет значение "Нет", модуль ценообразования прекращает поиск и использует коммерческое соглашение.  
35. Щелкните "Разнести".
36. Нажмите кнопку "OК".
37. Закройте страницу.
38. В области действий щелкните "Продажа".
39. Щелкните "Цена продажи".
