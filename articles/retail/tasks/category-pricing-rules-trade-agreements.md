# <a name="category-pricing-rules-to-create-trade-agreements"></a>Правила ценообразования категорий для создания коммерческих соглашений

[!include[task guide banner](../includes/task-guide-banner.md)]

Эта процедура демонстрирует способ создания коммерческих соглашения по ценам продажи с использованием правила ценообразования категории. В качестве компании с демонстрационными данными для создания этой задачи используется USRT. Эта задача предназначена для роли Директора по сбыту в розничной торговле.

1. Щелкните "Управление ценообразованием и скидками".
2. Щелкните "Создать".
3. Щелкните "Правило цены категории".
4. В списке пометьте выбранную строку.
5. В поле "Код счета" выберите значение.
    * Для настройки коммерческих соглашений по ценам продажи, специфичных для каналов, программ лояльности, каталогов и назначений, используется код счета типа "Группа".  
6. В поле "Выбор счета" введите или выберите значение.
7. В поле "Категория" введите или выберите значение.
8. В поле "Сумма/процент" введите число.
9. В поле "Версия округления" введите или выберите значение.
10. Щелкните "Создать коммерческие соглашения".
11. Щелкните Далее.
12. В поле "Дата начала" введите дату.
13. В поле "Дата окончания" введите дату.
14. Выберите значение "Да" в поле "Найти далее".
15. Щелкните Далее.
16. Нажмите кнопку Готово.
    * При этом создается журнал коммерческих соглашений и открывается для вашего просмотра.  
17. В списке найдите и выберите требуемую запись.
    * Журналы коммерческих соглашений, созданные из правил ценообразования категории, не разносятся. Вы можете просматривать и изменять созданные цены перед их разноской.  
18. Щелкните "Изменить".
19. В поле "Сумма в валюте" введите число.
20. Щелкните "Разнести".
21. Нажмите кнопку "OК".
22. Закройте страницу.
23. Закройте страницу.
24. Щелкните вкладку "Правила цены категории".
    * В этом списке будут отображаться правила ценообразования категории, относящиеся к каналу.
