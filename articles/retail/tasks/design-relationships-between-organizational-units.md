# <a name="design-the-relationships-between-organizational-units"></a>Определение отношений между подразделениями организации

[!include[task guide banner](../includes/task-guide-banner.md)]

В этой процедуре показано, как определить отношение между подразделениями. Необходимо создать новую цель организации перед определением отношения либо можно использовать существующую цель организации. В качестве компании с демонстрационными данными для выполнения этой процедуры используется USRT. Эта задача предназначена для роли администратора.

1. Перейдите в раздел "Управление организацией" > "Организации" > "Организационные иерархии".
2. Щелкните "Создать".
3. В поле "Имя" введите значение.
4. Щелкните "Назначить цель".
5. В списке найдите и выберите требуемую запись.
6. Нажмите кнопку Добавить.
7. В списке найдите и выберите требуемую запись.
8. Нажмите кнопку "OК".
    * Можно выбрать любое количество целей организации в зависимости от требований организации.  
9. В списке найдите и выберите требуемую запись.
10. Щелкните "Установить по умолчанию".
11. Закройте страницу.
12. Нажмите кнопку "Сохранить".
13. Щелкните "Вид".
14. Щелкните "Изменить".
15. Нажмите Вставить.
16. Щелкните "Бизнес-единица".
17. В списке найдите и выберите требуемую запись.
18. В списке перейдите по ссылке в выбранной строке.
19. Нажмите Вставить.
20. Щелкните "Канал розничной торговли".
21. В списке найдите и выберите требуемую запись.
22. В списке перейдите по ссылке в выбранной строке.
    * Можно добавить любое количество подразделений в зависимости от требований.  
23. Нажмите кнопку "Сохранить".
24. Щелкните "Закрыть".
25. Щелкните "Опубликовать", чтобы открыть раскрывающееся диалоговое окно.
26. В поле "Действует с" введите дату и время.
27. В поле "Действует с" введите дату и время.
28. В поле "Опишите изменения" введите значение.
29. Щелкните "Опубликовать".
30. Щелкните "Закрыть".
