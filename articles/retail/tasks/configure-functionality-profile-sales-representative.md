# <a name="configure-functionality-profiles-for-sales-representatives"></a>Настройка профилей функциональности для торговых представителей

[!include [task guide banner](../includes/task-guide-banner.md)]

Эта процедура демонстрирует порядок задания настроек профиля функциональности магазина, которые применяются к торговым представителям. В этой процедуре используется компания с демонстрационными данными USRT.

1. Перейдите в раздел > "Розничная торговля и коммерция" > "Настройка POS" > "Профили POS" > "Профили функциональности".
2. Щелкните "Изменить".
3. Разверните раздел "Функции".
    * Можно использовать настройки профиля функциональности для настройки POS на автоматическое добавление группы продаж по умолчанию для кассира, для вывода приглашения на ввод групп продаж и для запроса групп продаж.  
4. В поле "Запрашивать имя торгового представителя" выберите вариант.
5. Выберите значение "Да" в поле "Требовать указания торгового представителя".
