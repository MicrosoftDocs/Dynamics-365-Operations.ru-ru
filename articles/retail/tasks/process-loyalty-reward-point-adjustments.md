# <a name="process-loyalty-reward-point-adjustments"></a>Обработка корректировок баллов поощрения по программе лояльности

[!include[task guide banner](../includes/task-guide-banner.md)]

Эта процедура показывает, как искать данные карточки постоянного клиента и корректировать баллы поощрения лояльности. В качестве компании с демонстрационными данными для создания этой задачи используется USRT. Эта задача назначена для роли директора-распорядителя в розничной торговле или менеджера по обслуживанию клиентов.

1. Перейдите в раздел "Карты лояльности".
2. В списке найдите и выберите требуемую запись.
3. В списке перейдите по ссылке в выбранной строке.
4. Щелкните "Проводки по карте".
    * На этой странице можно просмотреть все проводки программы лояльности для выбранной карточки постоянного клиента.  
5. Закройте страницу.
6. Щелкните "Корректировки по карте".
7. Щелкните "Создать".
8. В поле "Точки вознаграждения" введите или выберите значение.
9. В поле "Сумма или количество" введите число.
    * Можно добавлять или удалять баллы из карточки постоянного клиента с помощью него положительных и отрицательных значений.  
10. В поле "Программа лояльности" введите или выберите значение.
11. В поле "Комментарий" введите значение.
12. Щелкните "Разноска корректировки".
13. Щелкните Да.
14. Закройте страницу.
    * Обычно на этом этапе можно обновить страницу, чтобы увидеть результаты корректировки баллов поощрения на вкладке "Сводка баллов поощрений". Но если вы выполняете это руководство по задаче, не обновляйте страницу сейчас, поскольку в этом случае выполнение руководства по задаче остановится.  
15. Щелкните "Проводки по карте".
16. Закройте страницу.
