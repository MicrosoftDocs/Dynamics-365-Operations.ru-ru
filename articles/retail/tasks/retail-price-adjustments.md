# <a name="retail-price-adjustments"></a>Корректировки розничных цен

[!INCLUDE [task guide banner](../includes/task-guide-banner.md)]

Эта процедура содержит инструкции по созданию корректировки розничной цены. Корректировка розничной цены может задавать цену продажи номенклатуры напрямую или изменят базовую цену продажи и цену продажи из коммерческого соглашения. В этой процедуре используется компания с демонстрационными данными USRT.

1. Щелкните "Управление ценообразованием и скидками".
2. Щелкните вкладку "Корректировки цены".
3. Щелкните "Создать".
    * Здесь можно создать все наиболее часто используемые правила цен и скидок, включая розничные скидки, корректировки цен, журналы коммерческих соглашений и правил ценообразования категории.  
4. Щелкните "Ценовая коррекция".
5. В поле "Имя" введите значение.
6. Разверните раздел "Строки".
7. Нажмите кнопку Добавить.
    * Например, введите в поле "Продукт" значение "81126".    Затем введите "10.0" в поле "Процент скидки".  
    * Корректировка цены типа процента скидки уменьшает базовую цену продажи или цену продажи из коммерческого соглашения.  
8. Нажмите кнопку Добавить.
    * Например, введите в поле "Продукт" значение "81125".    Затем выберите "Сумма скидки по оплате" в поле "Способ скидки".    Наконец, введите "5.0" в поле "Сумма скидки по оплате".  
    * Тип скидки "Сумма скидки по оплате" представляет собой сумму, на которую уменьшается базовая цена или цена из коммерческого соглашения.  
9. Щелкните "Ценовые группы".
    * В поле "Группа цен" введите "RP-Houston".  
    * Это свяжет корректировку цены с магазином в Хьюстоне.  
10. Нажмите кнопку "Сохранить".
11. Закройте страницу.
12. В поле "Состояние" выберите "Включено".
13. Нажмите кнопку "Сохранить".
14. Закройте страницу.
15. Обновите страницу.
