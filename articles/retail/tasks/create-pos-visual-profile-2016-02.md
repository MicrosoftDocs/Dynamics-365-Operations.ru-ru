# <a name="create-a-pos-visual-profile"></a>Создание визуального профиля POS 

[!INCLUDE [task guide banner](../includes/task-guide-banner.md)]

Эта процедура содержит инструкции по созданию нового визуального профиля ККМ. Визуальный профиль содержит основные сведения, которые определяют внешний вид ККМ. Можно создать несколько визуальных профилей и назначить их в зависимости от конкретных ККМ. В этой процедуре используется компания с демонстрационными данными USRT.

1. Перейдите в раздел Розничная торговля и коммерция > Настройка канала > Профили POS > Визуальные профили.
2. Щелкните "Создать".
3. В поле "Номер профиля" введите значение.
4. В поле "Описание" введите значение.
5. В поле "Тип приложения" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
6. В списке перейдите по ссылке в выбранной строке.
7. В поле "Тема" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
8. В списке перейдите по ссылке в выбранной строке.
9. В поле "Цвет выделения" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
10. В списке найдите и выберите требуемую запись.
11. В списке перейдите по ссылке в выбранной строке.
12. Переключите расширение раздела "Фон при входе в систему".
13. В поле "Идентификатор альбомного изображения" выберите или введите код изображения
14. В поле "Код портретного изображения" выберите или введите код изображения
15. Переключите развертывание раздела "Фон".
16. Запросите всплывающее окно кода изображения.
17. В списке перейдите по ссылке в выбранной строке.
18. Нажмите кнопку "Сохранить".
