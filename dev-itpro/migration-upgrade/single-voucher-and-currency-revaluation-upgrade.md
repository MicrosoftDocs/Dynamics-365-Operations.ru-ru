# <a name="single-voucher-and-currency-revaluation-upgrade-for-microsoft-dynamics-365-for-operations-version-1611"></a>Один ваучер переоценки в валюте обновления и для Microsoft Dynamics 365 для операций версии 1611

В некоторых организациях введите журналы, содержащие один ваучер, который имеет более одного клиента или поставщика, а также их выполнение расчетов с клиентами или расчетов с поставщиками в иностранной валюте Переоценка. В этом разделе описан процесс при их обновления до Microsoft Dynamics 365 для операций версии 1611 необходимо соблюдать эти организации.

При обновлении до Microsoft Dynamics 365 для операций версии 1611, выполните следующие действия.

1.  Перед обновлением до Dynamics 365 для операций, выполните процессы переоценки иностранной валюты для расчетов с клиентами и Расчеты с поставщиками. Настройка **метод** на **дату накладной**. Создается проводка переоценки, которая изменяет последней переоценки в иностранной валюте. Таким образом открытые проводки оцениваются по их исходной валюте по умолчанию.
2.  Обновлению Dynamics 365 для операций версии 1611.
3.  Снова запустите расчетов с клиентами и Расчеты с поставщиками в иностранной валюте переоценки процессов. Это время установлено **метод** на **стандартной**. Создается новая проводка переоценки на основании текущих валютных курсов. Эта проводка записывает нереализованные прибыли и убытков и на правильный итоговый счет ГК.
