### <a name="aeb-payment-formats-for-spain"></a>Форматы платежей AEB для Испании

Форматы платежей Consejo Superior Bancario используются для отправки файлов предъявления к оплате в банк для платежей клиентов или поставщикам. Содержание этих форматов определяется Asociación Española de Banca. Оно охватывает Cuaderno 19, 32, 58, 34.
