# <a name="new-or-updated-task-guides-may-2016"></a>Новые или обновленные задачи руководства (май 2016)

В этой статье перечислены новые или обновленные задачи руководств, которые были выпущены для Майкрософт может 2016 - все языки APQC библиотеки единой системе для Microsoft Dynamics AX. Новая библиотека начала работы не была выпущена. 

<a name="task-guides-in-the-may-2016---all-languages-apqc-unified-library-for-microsoft-dynamics-ax"></a>[]()Задача поможет в (май 2016 - все языки) APQC библиотеки единой системе для Microsoft Dynamics AX
------------------------------------------------------------------------------------------------

(Май 2016) APQC единой библиотека содержит задачи, руководства, которые предназначены для использования для получения справки по Microsoft Dynamics AX, май 2016 г и запуск в производство февраля 2016 Microsoft Dynamics AX. Локализованные версии руководства задачи, которые были выпущены в виде (февраля 2016) APQC единой библиотеки, доступны в данной библиотеке. Дополнительные сведения о подключении в справочной системе в библиотеку руководстве задач см. в разделе [приступить к работе с помощью](help-overview.md). **Примечание:** символ звездочки (**\***) рядом с заголовком бизнес-моделирования процесса (BPM) строка указывает, что руководство по задачам для этого бизнес-процесса была создана для Dynamics AX, май 2016 г и не поддерживается в более ранних версиях Microsoft Dynamics AX.
