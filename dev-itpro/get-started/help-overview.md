# <a name="help-overview"></a>Обзор справки

Эта статья содержит обзор компонентов справочной системы Microsoft Dynamics 365 for Operations. Он также описывает порядок, как можно предоставит пользовательскую документацию и обучение для вашей организации. 

Dynamics 365 for Operations включает систему справки, которая основана на двух главных компонентах:

-   Сайт документации
-   Руководства по задачам

Вы можете получить доступ к статьями и руководствам по задачам в области Справка в Dynamics 365 for Operations, как показано на следующем снимке экрана. [![Панель справки](./media/help-pane-ops-task-guides-1024x741.png)](./media/help-pane-ops-task-guides.png) В этой статье описывается система справки и объясняется, как можно создать пользовательскую документацию и ресурсы по обучению для организации.

## <a name="help-on-docsmicrosoftcom"></a>Справка на docs.microsoft.com
Сайт docs.microsoft.com ([docs.microsoft.com/dynamics365/operations](https://docs.microsoft.com/en-us/dynamics365/#pivot=solutions&panel=solutions_operations) является основным источником документации по продукту для Dynamics 365 for Operations. Сайт содержит следующие возможности.

-   **Доступ к самому последнему содержимому** — сайт предоставляет более быстрый и гибкий способ создания, доставки и обновления документации по продуктам. Поэтому она помогает гарантировать, что вы имеете доступ к самой последней технической информации.
-    **Содержимое, написанное специалистами** — сайт предоставляет более широкий набор документации по продуктам, который может расширяться членами сообщества как внутри, так и за пределами Майкрософт.
-   ** Доступ к различным типам содержимого** — сайт позволяет быстро получать доступ к различным типам содержимого о Dynamics 365 for Operations, такому как презентации Microsoft Office Mix, руководства по задачам, видео и статьи вики-сайта.
-    **Содержимое, которое поддерживает бизнес-процессы** — сайт включает содержимое с акцентом на бизнес-процессы, которое использует преимущества средства моделирования бизнес-процессов (BPM) в Microsoft Dynamics Lifecycle Services (LCS).

Мы перенесли все содержимое с нашего предыдущего вики-сайт справки в документы. Мы очень довольны нашим новым сайтом и надеемся, что он понравится и вам.

### <a name="when-can-we-use-it"></a>Когда можно его использовать?

Вы можете читать содержимое в документах уже сейчас. Этот сайт полностью доступен для всех, и на нем можно выполнять поиск без входа в систему. Вы можете использовать любой из ваших любимых механизмов поиска для поиска содержимого. Если хотите, можете оставлять комментарии к статьям на сайте, выполнив вход в систему с учетной записью GitHub.


## <a name="task-guides"></a>Руководства по задачам
Руководство по задаче — это контролируемое, управляемое и интерактивное взаимодействие с пользователем, при котором пользователь выполняет пошаговую процедуру выполнения задачи или бизнес-процесса. Вы можете открыть (воспроизвести) руководство по задаче в области Справка. При первом щелчке руководства по задаче в области Справка отобразятся пошаговые инструкции для задачи. Теперь доступны локализованные руководства по задачам. [![Представление чтения руководства по задаче](./media/task-guide-ops-1024x742.png)](./media/task-guide-ops.png) Чтобы приступить к управляемому, интерактивному взаимодействию с пользователем, щелкните **Запустить руководство по задаче** в нижней части области Справка. Откроется черный указатель, указывающий действие, которое необходимо выполнить. Следуйте инструкциям, которые отображаются в пользовательском интерфейс, и введите данные, когда это потребуется. [![Пошаговая инструкция к руководстве по задаче](./media/task-guide-step-1-ops.png)](./media/task-guide-step-1-ops.png) **Важно!** Данные, вводимые при воспроизведении руководства по задаче, являются реальными. Если вы находитесь в производственной среде, данные будут вводиться в компанию, которая используется в данный момент.

### <a name="it-all-begins-with-task-recorder"></a>Все начинается в регистраторе задач

Руководства по задачам создаются с помощью регистратора задач. При использовании регистратора задач записываются все ваши действия в пользовательском интерфейсе Dynamics 365 for Operations (например, щелчки меню, изменение настроек и ввод данных). Записываемые шаги вместе называются записью задачи. Как было сказано в предыдущем разделе, записи задач можно отобразить в области Справка и воспроизвести как руководства по задачам. Однако записи задач можно использовать по-другому:

-   **Сохранить записи задач в BPM** — можно сохранить запись задачи в строке иерархии в библиотеке BPM в LCS. При сохранении записи задачи в BPM создается и отображается диаграмма блок-схемы вместе с шагами записи. **Примечание.** Для отображения записи задачи в области Справка Dynamics 365 for Operations и ее воспроизведения как руководства по задаче следует сохранить запись в библиотеке BPM.
-   **Сохранить записи задач как документы Word** — сохраняя запись задач как документ Microsoft Word, вы можете легко создавать печатные руководства по задачам для организации.

Дополнительные сведения о регистраторе задач см. в разделе [Регистратор задач в Dynamics 365 for Operations](../user-interface/task-recorder.md).

### <a name="creating-customized-task-recordings"></a>Создание настраиваемых записей задач

Можно создать собственные записи задач или загрузить и настроить записи задач, предоставляемые Майкрософт. Таким образом, можно создать настраиваемую справку для организации, которая отражает определенную реализацию Dynamics 365 for Operations. Для отображения записи задачи в области Справка Dynamics 365 for Operations и ее воспроизведения как руководства по задаче следует сохранить запись в библиотеке BPM в LCS. Если вы являетесь партнером и обновляете библиотеку до корпоративной библиотеки и включаете ее в решение, она будет доступна вашим клиентам. Полные инструкции см. в разделе [Использование записей задач для создания документации или обучения](../user-interface/task-recorder.md).

## <a name="in-product-help"></a>Встроенная в продукт справка
Чтобы получить доступ к справке в Dynamics 365 for Operations, щелкните значок **Справка** (**?**), затем выберите Справку или нажмите сочетание клавиш CTRL+SHIFT+?. В обоих случаях откроется область Справка. В области Справка можно получить доступ к статьям или руководствам по задачам. [![](./media/help-pane-wiki-1024x684.png)](./media/help-pane-wiki.png)

### <a name="accessing-articles-from-the-help-pane"></a>Доступ к статьям из области "Справка"

В области Справка можно получить доступ к статьям, которые применяются к клиенту Dynamics 365 for Operations. При первом открытии области справки и выборе вкладки **Wiki** вы увидите статьи, которые применяются к странице, на которой вы в данный момент в Dynamics 365 for Operations. Если статьи не найдены, можно ввести ключевые слова для уточнения поиска. При выборе статьи в области Справка открывается новая вкладка в браузере и отображается статья. 

### <a name="accessing-task-guides-from-the-help-pane"></a>Доступ к руководствам по задачам из области "Справка"

Прежде чем вы сможете получить доступ к руководствам по задачам в области Справка, системный администратор должен перейти на страницу **Системные параметры** в Dynamics 365 for Operations и настроить некоторые параметры. **Примечания.**

-   Для настройки справки необходимо войти с помощью учетной записи в том же владельце, что и владелец, в котором развернуто приложение Dynamics 365 for Operations.
-   Невозможно подключиться к библиотеке LCS из экземпляра Dynamics 365 for Operations, выполняемого на локальном виртуальном жестком диске (VHD).

[![Форма "Системные параметры" с настройками системы справки](./media/system-parameters_ops-1024x437.png)](./media/system-parameters_ops.png) На странице **Системные параметры** выполните следующие действия:

1.  **Важно!** При первом открытии вкладки справки необходимо подключиться к службами Lifecycle Services. Необходимо нажать на ссылку в середине формы, дождаться подключения, закрыть диалоговое окно, затем нажать ОК, чтобы перейти к форме параметров.[![Подключение к LCS](./media/connect-to-lcs-crop-1024x365.png)](./media/connect-to-lcs-crop.png)
2.  Выберите проект Lifecycle Services для подключения.
3.  Выберите библиотеки BPM (в пределах выбранного проекта), из которых требуется извлечь записи задач.
4.  Задайте порядок отображения библиотек BPM. Это определяет порядок, в котором записи задач из библиотек будут отображаться в области Справка.

После того как системный администратор завершит эти шаги, вы сможете открыть область Справка и перейти на вкладку **Руководства по задачам**. Теперь вы увидите руководства по задачам, которые применяются к странице, на которой вы в данный момент в Dynamics 365 for Operations. Если руководства по задачам не найдены, можно ввести ключевые слова для уточнения поиска. После выбора руководства по задаче в области Справка в области Справка отобразятся пошаговые инструкции, и вы сможете воспроизвести руководство по задаче. [![Представление чтения руководства по задаче](./media/task-guide-ops-1024x742.png)](./media/task-guide-ops.png)

### <a name="where-are-the-translated-task-guides"></a>Где находятся переведенные руководства по задачам?

Переведенные руководства по задачам выпускаются в библиотеках с текстом "Все языки" в заголовке. В Dynamics 365 for Operations для просмотра локализованной справки руководств по задачам проверьте, что вы подключены к соответствующей библиотеке. Язык, на котором отображается руководство по задаче, контролируется для каждого пользователя на основании языковых параметров, заданных в разделе **Параметры** &gt; **Предпочтительные параметры**. 
-   Если руководство по задаче было переведено, при открытии этого руководства по задаче весь текст руководства по задаче будет отображаться на выбранном языке.
-   Если руководство по задаче еще не было переведено, при открытии этого руководства только некоторый текст (текст элементов управления) будет отображаться на выбранном языке.
