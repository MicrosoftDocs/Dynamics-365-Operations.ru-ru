# <a name="help-overview"></a>Обзор справки

[!include[banner](../includes/banner.md)]

Эта статья содержит обзор компонентов справочной системы Microsoft Dynamics 365 for Finance and Operations. Он также описывает порядок, как можно предоставит пользовательскую документацию и обучение для вашей организации. 

> [!NOTE] 
> Следующие продукты тесно связаны: Dynamics 365 for Finance and Operations, Enterprise edition; Dynamics 365 for Retail и Dynamics 365 for Talent. Одинаковые функциональные возможности могут быть доступны во всех 3 продуктах. Таким образом, в темах, которые в основном связаны с Retail, будет использоваться имя продукта Microsoft Dynamics 365 for Retail, а в темах, которые в основном связаны Talent, будет использоваться имя продукта Dynamics 365 for Talent, а в темах, которые в основном связаны основном продуктом, будет использоваться имя продукта Dynamics 365 for Finance and Operations. Темы, написанные для одного продукта, могут относиться и к аналогичным функциональным возможностям в связанном продукте.

Справочная система является общей для следующих продуктов:
- Dynamics 365 for Finance and Operations, выпуск Enterprise
- Dynamics 365 for Retail
- Dynamics 365 for Talent

Finance and Operations включает систему справки, которая основана на двух главных компонентах:

-   Сайт документации
-   Проводники по задачам

Можно получить доступ к справочным статьям и проводникам по задачам из области справки для любого средства. На следующем снимке экрана показано Dynamics 365 for Finance and Operations

![Область справки](./media/help-pane-ops-task-guides.png)

В этой теме описывается система справки и объясняется, как можно создать пользовательскую документацию и ресурсы по обучению для организации.

> [!IMPORTANT]
> Вкладка **Проводники по задачам** в настоящее время недоступна в Microsoft Dynamics 365 for Talent и Microsoft Dynamics 365 for Retail. Мы в настоящее время работаем для включения этой функции в будущих выпусках. Проводники по задачам в разделе начала работы в Talent остаются доступными для работы с основными функциями. Практические советы можно также получить на сайте docs.microsoft.com ([docs.microsoft.com/dynamics365/operations](/dynamics365/unified-operations/fin-and-ops/index)) для Retail и Talent.

## <a name="help-on-docsmicrosoftcom"></a>Справка на docs.microsoft.com

Сайт docs.microsoft.com ([docs.microsoft.com/dynamics365/operations](/dynamics365/unified-operations/fin-and-ops/index)) является основным источником документации по продукту для Finance and Operations. Сайт содержит следующие возможности.

-   **Доступ к самому последнему содержимому** — сайт предоставляет более быстрый и гибкий способ создания, доставки и обновления документации по продуктам. Поэтому она помогает гарантировать, что вы имеете доступ к самой последней технической информации.
-   **Содержимое, написанное специалистами** — сайт предоставляет более широкий набор документации по продуктам, который может расширяться членами сообщества как внутри, так и за пределами Майкрософт.
-   **Доступ к различным типам содержимого** — сайт позволяет быстро получать доступ к различным типам содержимого о Finance and Operations, такому как презентации Microsoft Office Mix, руководства по задачам, видео и разделам.
-   **Содержимое, которое поддерживает бизнес-процессы** — сайт включает содержимое с акцентом на бизнес-процессы, которое использует преимущества средства моделирования бизнес-процессов (BPM) в Microsoft Dynamics Lifecycle Services (LCS).

## <a name="task-guides"></a>Проводники по задачам
Руководство по задаче — это контролируемое, управляемое и интерактивное взаимодействие с пользователем, при котором пользователь выполняет пошаговую процедуру выполнения задачи или бизнес-процесса. Вы можете открыть (воспроизвести) руководство по задаче в области Справка. При первом щелчке руководства по задаче в области Справка отобразятся пошаговые инструкции для задачи. Теперь доступны локализованные руководства по задачам. 

Для каждого продукта, например Dynamics 365 for Retail, Dynamics 365 for Finance and Operations или Dynamics 365 for Talent, предоставляются свои наборы проводников по задачам. В разделе проводников по задачам доступа в области "Справка" содержатся сведения о поиске правильных проводников по задачам для продукта. 

![Представление чтения руководства по задаче](./media/task-guide-ops.png)

Чтобы приступить к управляемому, интерактивному взаимодействию с пользователем, щелкните **Запустить руководство по задаче** в нижней части области "Справка". Откроется черный указатель, указывающий действие, которое необходимо выполнить. Следуйте инструкциям, которые отображаются в пользовательском интерфейс, и введите данные, когда это потребуется. 
![Инструкция шага руководства по задаче](./media/task-guide-step-1-ops.png)

> [!IMPORTANT] 
> Данные, вводимые при воспроизведении руководства по задаче, являются реальными. Если вы находитесь в производственной среде, данные будут вводиться в компанию, которая используется в данный момент.

### <a name="it-all-begins-with-task-recorder"></a>Все начинается в регистраторе задач

Руководства по задачам создаются с помощью регистратора задач. При использовании регистратора задач записываются все ваши действия в пользовательском интерфейсе Finance and Operations (например, щелчки меню, изменение настроек и ввод данных). Записываемые шаги вместе называются записью задачи. Как было сказано в предыдущем разделе, записи задач можно отобразить в области Справка и воспроизвести как руководства по задачам. Однако записи задач можно использовать по-другому:

-   **Сохранить записи задач в BPM** — можно сохранить запись задачи в строке иерархии в библиотеке BPM в LCS. При сохранении записи задачи в BPM создается и отображается диаграмма блок-схемы вместе с шагами записи. **Примечание.** Для отображения записи задачи в области "Справка" Finance and Operations и ее воспроизведения как руководства по задаче следует сохранить запись в библиотеке BPM.
-   **Сохранить записи задач как документы Word** — сохраняя запись задач как документ Microsoft Word, вы можете легко создавать печатные руководства по задачам для организации.

Дополнительные сведения о регистраторе задач см. в разделе [Регистратор задач в Finance and Operations](../user-interface/task-recorder.md).

### <a name="creating-customized-task-recordings"></a>Создание настраиваемых записей задач

Можно создать собственные записи задач или загрузить и настроить записи задач, предоставляемые Майкрософт. Таким образом, можно создать настраиваемую справку для организации, которая отражает определенную реализацию Finance and Operations. Для отображения записи задачи в области "Справка" Finance and Operations и ее воспроизведения как проводника по задаче следует сохранить запись в библиотеке BPM в LCS. Если вы являетесь партнером и обновляете библиотеку до корпоративной библиотеки и включаете ее в решение, она будет доступна вашим клиентам. Полные инструкции см. в разделе [Использование записей задач для создания документации или обучения](../user-interface/task-recorder.md).

## <a name="in-product-help"></a>Встроенная в продукт справка
Чтобы получить доступ к справке в Finance and Operations, щелкните значок **Справка** (**?**), затем выберите Справку или нажмите сочетание клавиш CTRL+SHIFT+?. В обоих случаях откроется область Справка. В области Справка можно получить доступ к статьям или руководствам по задачам. 

![Область справки](./media/help-pane-wiki.png)

### <a name="accessing-help-topics-from-the-help-pane"></a>Доступ к проводникам по задачам из области "Справка"

В области Справка можно получить доступ к статьям, которые применяются к клиенту Finance and Operations. При первом открытии области справки и выборе вкладки **Wiki** вы увидите статьи, которые применяются к странице, на которой вы в данный момент в Finance and Operations. Если статьи не найдены, можно ввести ключевые слова для уточнения поиска. При выборе статьи в области Справка открывается новая вкладка в браузере и отображается статья. 

> [!IMPORTANT]
> Этот раздел неприменим к Dynamics 365 for Talent. Справочная система Talent автоматически подключается к проводникам по задачам для этого продукта. Кроме того, нельзя создавать пользовательские проводники по задаче для Talent. 

### <a name="accessing-task-guides-from-the-help-pane"></a>Доступ к руководствам по задачам из области "Справка"

Прежде чем вы сможете получить доступ к руководствам по задачам в области Справка, системный администратор должен перейти на страницу **Системные параметры** в Finance and Operations и настроить некоторые параметры. 

> [!NOTE]
> -   Для настройки справки необходимо войти с помощью учетной записи в том же владельце, что и владелец, в котором развернуто Finance and Operations.
> -   Невозможно подключиться к библиотеке LCS из экземпляра Finance and Operations, выполняемого на локальном виртуальном жестком диске (VHD).

![Форма "Системные параметры" с настройками системы справки](./media/system-parameters_ops-1024x437.png)

На странице **Системные параметры** выполните следующие действия:

1.  **Важно!** При первом открытии вкладки "Справка" необходимо подключиться к службами Lifecycle Services. Необходимо нажать на ссылку в середине формы, дождаться подключения, закрыть диалоговое окно, затем нажать **ОК**, чтобы перейти к форме параметров.
![Подключение к LCS](./media/connect-to-lcs-crop-1024x365.png)
2.  Выберите проект Lifecycle Services для подключения.
3.  Выберите библиотеки BPM (в пределах выбранного проекта), из которых требуется извлечь записи задач.
    - В Finance and Operations для содержимого Microsoft выберите февраль 2017 г. "Унифицированная библиотека QPC" для Microsoft Dynamics 365 for Finance and Operations. 
    - Для Retail мы выпустим библиотеку в июле. 
    - Нет необходимости выбирать библиотеку для Talent: правильная библиотека уже подключена. 

4.  Задайте порядок отображения библиотек BPM. Это определяет порядок, в котором записи задач из библиотек будут отображаться в области Справка.

После того как системный администратор завершит эти шаги, вы сможете открыть область Справка и перейти на вкладку **Руководства по задачам**. Теперь вы увидите проводники по задаче, которые применяются к странице, на которой вы в данный момент в Finance and Operations. Если руководства по задачам не найдены, можно ввести ключевые слова для уточнения поиска. После выбора руководства по задаче в области Справка в области Справка отобразятся пошаговые инструкции, и вы сможете воспроизвести руководство по задаче. 

![Представление чтения руководства по задаче](./media/task-guide-ops.png)

### <a name="where-are-the-translated-task-guides"></a>Где находятся переведенные руководства по задачам?

Переведенные руководства по задачам выпускаются в библиотеках с текстом "Все языки" в заголовке. В Finance and Operations для просмотра локализованной справки проводников по задачам проверьте, что вы подключены к правильной библиотеке. Язык, на котором отображается руководство по задаче, контролируется для каждого пользователя на основании языковых параметров, заданных в разделе **Параметры** &gt; **Предпочтительные параметры**. 
-   Если руководство по задаче было переведено, при открытии этого руководства по задаче весь текст руководства по задаче будет отображаться на выбранном языке.
-   Если руководство по задаче еще не было переведено, при открытии этого руководства только некоторый текст (текст элементов управления) будет отображаться на выбранном языке.
