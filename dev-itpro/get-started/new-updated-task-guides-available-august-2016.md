# <a name="new-or-updated-task-guides-august-2016"></a>Новые или обновленные задачи руководства (2016 августа)

В этом разделе перечислены новые или обновленные задачи инструкциям, выпущенные для Microsoft (2016 августа — все языки) APQC единой библиотеки для Microsoft Dynamics AX. Новая библиотека начала работы не была выпущена.

<a name="task-guides-in-the-august-2016---all-languages-apqc-unified-library-for-microsoft-dynamics-ax"></a>[]()Задача поможет в (август 2016 - все языки) APQC библиотеки единой системе для Microsoft Dynamics AX
---------------------------------------------------------------------------------------------------

(Август 2016 - все языки) APQC единой библиотека содержит руководства задач, которые предназначены для использования с обновления платформы Microsoft Dynamics AX (2016 августа), 2 Dynamics AX для справки может обновить приложения 2016 и высвобождать февраля 2016 Microsoft Dynamics AX. Дополнительные сведения об этих выпусках см. в разделе [того, что новые или измененные](whats-new-changed.md). В этой библиотеке доступны локализованной версии руководства задачи. Дополнительные сведения о подключении в справочной системе в библиотеку руководстве задач см. в разделе [приступить к работе с помощью](help-overview.md).
