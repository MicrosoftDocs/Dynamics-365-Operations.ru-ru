# <a name="new-or-updated-task-guides-november-2016"></a>Новые или обновленные задачи поможет (ноябрь 2016)

В этом разделе перечислены новые или обновленные задачи инструкциям, выпущенные для Microsoft (ноябрь 2016 - все языки) APQC единой библиотека для Microsoft Dynamics 365 для операций. Новая библиотека начала работы не была выпущена.

(Ноябрь 2016--все языки) APQC единой библиотека содержит руководства задачи, которые должны использоваться для получения справки по Microsoft Dynamics 365 для операций, Microsoft Dynamics 365 для операций обновления платформы Microsoft Dynamics AX платформы обновление 2, 3, Dynamics AX май 2016 обновления и февраля 2016 Microsoft Dynamics AX на запуск в производство. Дополнительные сведения об этих выпусках см. в разделе [того, что новые или измененные](whats-new-changed.md). В этой библиотеке доступны локализованной версии руководства задачи. Дополнительные сведения о подключении в справочной системе в библиотеку руководстве задач см. в разделе [приступить к работе с помощью](help-overview.md). **Примечание:** символ звездочки (**\***) рядом с заголовком бизнес-моделирования процесса (BPM) строка указывает, что руководство по задачам для этого бизнес-процесса была создана для обновления Dynamics 365 for Operations 2016 ноября и не поддерживается в более ранних версиях Microsoft Dynamics AX.
