# <a name="mobile-workspaces-recently-released"></a>Недавно выпущенные мобильные рабочие области

[!include[banner](../includes/banner.md)]



В этой теме перечислены мобильные рабочие области, которые недавно были выпущены для мобильного приложения Microsoft Dynamics 365 for Unified Operations.

> [!NOTE]
> Прежнее название мобильного приложения — *Microsoft Dynamics 365 for Finance and Operations*.


## <a name="new-mobile-workspaces-released-in-june-2017"></a>Новые мобильные рабочие области, выпущенные в июне 2017 г.

| Мобильная рабочая область     | описание   | Дополнительные сведения   |
|----------------------|---------------|--------------|
|Каталог компании| Позволяет пользователям просматривать и связываться с другими сотрудниками в своей организации.| [Мобильная рабочая область каталога компании](company-directory-mobile-workspace.md)|    
|Моя группа| Позволяет пользователям просматривать своих прямых подчиненных и расширенный персонал, а также отправлять благодарности для отдельных лиц в рамках их цепочки отчетности.| [Мобильная рабочая группа "Моя группа"](manager-self-service-mobile-workspace.md)|     
|Утверждения накладных| Содержит список накладных, которые были назначены вам через процесс workflow-процесса заголовка накладной поставщика.|[Мобильная рабочая область "Утверждения накладных"](invoice-approval-mobile-workspace.md)    |
