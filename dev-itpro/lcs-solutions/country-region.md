# <a name="localization-and-regulatory-features"></a>Локализации и нормативные функции

Microsoft Dynamics 356 для операций включает в себя функцию для некоторых страной/регионом правилами и процессов. Эта функция включена, на основе основного адреса активной юридического лица. В этом разделе содержатся списки ресурсов, которые помогут вам узнать больше о функциях страной/регионом.

Электронная отчетность позволяет управлять этими форматами в течение их жизненного цикла. Дополнительные сведения см. в следующих разделах:
