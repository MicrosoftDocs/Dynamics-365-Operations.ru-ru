# <a name="financial-performance-power-bi-content"></a>Финансовые содержимое производительности Power BI

В этой теме описывается Microsoft Dynamics 365 для содержимого пакета Microsoft Power BI финансовых результатов операций. Описание панели мониторинга и отчеты, которые включены в пакет содержимого и содержит сведения о модели данных и объекты, которые были использованы для построения содержимого пакета.

<a name="accessing-the-content-pack"></a>Доступ к содержимого пакета
--------------------------

Доступны две версии с пакетом обновления результатов финансовой деятельности. Одна версия доступна из Microsoft Dynamics Lifecycle Services (LCS) и других доступна в PowerBI.com.

-   **Версия, доступная из LCS:** пакет содержимого результаты финансовой деятельности, который доступен из LCS поддерживает Microsoft Dynamics 365 операций версии 1611. Пакет содержимого можно найти в библиотеке общих средств LCS. Дополнительные сведения о том, как загрузить пакет содержимого и подключиться к Microsoft Dynamics 365 для данных операций см. в разделе [содержимое Power BI в LCS корпорации Майкрософт и ее партнеры](power-bi-content-microsoft-partners.md).
-   **Версия, доступная из PowerBI.com:** пакет содержимого результаты финансовой деятельности, который доступен из PowerBI.com поддерживает версии Microsoft Dynamics AX, 7.0 и 7.0.1. Дополнительные сведения о том, как для подключения и загрузки данных Dynamics 365 for Operations см. в разделе [содержимого Power BI доступ из PowerBI.com](power-bi-home-page.md).

## <a name="main-account-setup"></a>Настройка счета ГК
Поскольку организаций желают задолженности и суммы выручки в виде положительных сумм в отчетах, важно определить параметры счетов ГК в Dynamics 365 для операций. Для эти счета ГК, чтобы оно отображалось как положительные суммы, тип счета ГК должен задаваться как **задолженностью** или **выручка**. При использовании этих типов счетов отчетов с помощью бизнес-Аналитики Microsoft питания будет обратные знаки и отражают как положительные суммы.

## <a name="dashboard-and-reports-that-are-included-in-the-content-pack"></a>Панели мониторинга и отчеты, которые включены в пакет содержимого
После подключения пакета содержимого к данным Dynamics 365 for Operations финансовые данные будут отображаться на панели мониторинга и в отчетах. Если вы никогда не использовали Power BI перед, дополнительные о нем на [страницу интерактивной обучения для бизнес-Аналитики питания](https://powerbi.microsoft.com/en-us/guided-learning/?WT.mc_id=PBIService_GetData). Панель мониторинга содержит сводные плитки данных, основанные на базовых отчетах. Каждая плитка содержит сводные сведения по текущему году для всех компаний в организации. Ниже приведены некоторые разделы.

-   Наличный расчет
-   Итоговая выручка за этот год
-   Итоговые расходы за этот год
-   Чистый доход за этот год
-   Коэффициент быстрой ликвидности
-   Общие расходы за этот год по категории счетов
-   Коэффициент текущей ликвидности
-   Отношение задолженности к общим активам
-   Фактическая и прогнозируемая выручка
-   Выручка с выставленными счетами за этот год
-   Коэффициент операционных расходов за этот год
-   Рентабельность продаж за этот год
-   Фактические и бюджетные расходы — все компании

## <a name="understanding-the-data-model-and-entities"></a>Понимание модели данных и объектов
Данные, которые вносит панели мониторинга и отчеты в пакет содержимого финансовой производительности является Dynamics 365 для данных операций. Следующие объекты были использованы в качестве основы для содержимого пакета: **объединять объекты данных**

-   **GeneralLedgerActivities** — этой сущности агрегирует сальдо ГК с категорией счета.
-   **BudgetActivities** — этой сущности агрегирует сальдо бюджета по категориям учетной записи.

**Data entities**

-   FiscalCalendars
-   MainAccounts
-   LegalEntities
-   Главные книги
-   ChartofAccounts

Эти сущности были использованы для создания вычисляемых мер в модели данных. Вычисляемые меры используются для вычисления, ключевые индикаторы производительности (KPI) и отчетов, которые используются в пакет содержимого. По умолчанию пакет содержимого отображает данные за последние три года и один будущий год. Чтобы включить дополнительные расчеты от отчеты и панель мониторинга, можно изменить [книгу Microsoft Excel](https://mbs.microsoft.com/customersource/global/AX/downloads/reports/msdaxfinpercontentpowerbi). Эта книга является моделью данных по умолчанию, которая использовалась для создания пакета содержимого. По завершении внесения изменений можно создать организационный пакет содержимого панель мониторинга, содержащие добавленные сведения.
