# <a name="sales-and-profitability-performance-power-bi-content"></a>Продажи и прибыльность производительности содержимое питания бизнес-Аналитики

В этой теме описывается содержание Dynamics 365 для операций - продажи и прибыльность пакет содержимого производительность для бизнес-Аналитики Майкрософт для Power. Он объясняет, как получить доступ к отчетам, включенных в пакет содержимого и содержит сведения о модели данных и объекты, которые используются для построения содержимого пакета.

<a name="overview"></a>Обзор
--------

Этот пакет содержимого создан для менеджеров по продажам для отслеживания продаж ключевые показатели выручки, валовой прибыли и прибыли. Использует данные о транзакциях продажи из Dynamics 365 для операций и предоставляет как статистического представления показателей продаж в масштабе всей компании, так и разбивка показателей продаж для клиентов и продуктов. Выделения изменений в роста доходов и прибыли по времени, отчеты могут использоваться для менеджеров оповещения о тенденции положительные и отрицательные для отдельных клиентов и продуктов. Категории и региональных менеджеров будет оказаться полезным анализа сведений прибыльности и рентабельности разных категорий продуктов и групп клиентов друг с другом в одной laggards и руководители. Полный отчет, который отображает отдельным клиентом соотносятся доходы и прибыль предложения менеджеров по резервным данных основу для attune свои усилия по продажам и маркетингу, соответствующих профилю каждого клиента. Продажи и прибыльность производительности содержимого, что пакет позволяет менеджеры по продажам для анализа результатов продаж по:

-   Выручка, чтобы года (по группы клиентов и отдельных клиентов, категории продаж и отдельные продукты и географии)
-   Изменение дохода, по годам (по категории регионов и продаж клиента)

Прибыльность можно проанализировать по:

-   Брутто-прибыль и чистая прибыль (по группам клиентов и категории продаж продуктов)
-   Изменение валовой прибыли, по годам
-   Рентабельность клиента (по выручке и Валовая прибыль)

## <a name="accessing-the-content-pack"></a>Доступ к содержимого пакета
Продажи и прибыльность производительности Power BI содержимого пакета публикуется в качестве реализации основного средства в жизненный цикл служб (LCS) и доступных Dynamics 365 для операций. Дополнительные сведения о том, как получить доступ к и запуска отчетов Power BI см. в разделе [содержимое Power BI в LCS корпорации Майкрософт и ее партнеры](power-bi-content-microsoft-partners.md).

## <a name="understanding-the-data-model-and-entities"></a>Понимание модели данных и объектов
Dynamics 365 для данных операций используется для заполнения отчета в продажи и прибыльность производительности содержимого пакета. Это представляется как общий измерений, которые помещаются на временное хранение в хранилище объекта — базе данных Microsoft SQL, оптимизированная для аналитики. Дополнительные сведения о его в блоге [Power BI интеграции с магазином объекта в Dynamics](https://blogs.msdn.microsoft.com/dynamicsaxbi/2016/06/09/power-bi-integration-with-entity-store-in-dynamics-ax-7-may-update/). Общий измерения в этот пакет содержимого являются подмножество статистические измерений, которые были доступны в кубе продажи в Dynamics AX 2012 и AX 2012 R3. Провести совокупного измерений куба в хранилище объекта необходимо сделать их развертывания. Для получения дополнительных сведений см. процедуру по стадии совокупного измерений в хранилище объекта в блоге [Power BI интеграции с магазином объекта в Dynamics](https://blogs.msdn.microsoft.com/dynamicsaxbi/2016/06/09/power-bi-integration-with-entity-store-in-dynamics-ax-7-may-update/). Следующие ключевые совокупного измерениям объекта строки накладной, используются в качестве основы содержимого пакета.

|               |                                              |                                                 |                                              |                                          |
|---------------|----------------------------------------------|-------------------------------------------------|----------------------------------------------|------------------------------------------|
| **Entity**    | **Ключевые совокупного измерений**               | **Источник данных для Dynamics 365 для операций** | **Field**                                    | **Description**                          |
| Строки накладной | Реализация                                      | CustInvoiceTrans                                | SUM(LineAmountMST)                           | Сумма в валюте учета            |
|               | Стоимость проданных товаров                           | InventTrans                                     | Сумма (CostAmountPosted + CostAmountAdjustment) | Себестоимость + корректировка                 |
|               | Сумма по строке комиссии-валюта учета | CustInvoiceTrans                                | SUM(CommissAmountMST)                        | Сумма комиссии в валюте по умолчанию |

В следующей таблице показаны ключевые объекта строки накладной совокупного измерений, которые используются для создания нескольких вычисляемых мер в пакет содержимого набора данных.

|                   |                                                                                                  |
|-------------------|--------------------------------------------------------------------------------------------------|
| **Measure**       | **Вычисляется как**                                                                                |
| Валовая прибыль      | Сумма (выручка – себестоимость реализованной продукции – комиссии — налог (включен в сумма по строке накладной клиента))          |
| Валовая прибыль      | Сумма (Валовая прибыль / (выручка - налог (включен в сумма по строке накладной клиента)))             |
| Доход за прошлый год | Выручка прошлый год = РАССЧИТАТЬ (сумма ("Строки накладной"\[выручка\]), SAMEPERIODLASTYEAR (даты\[Дата\]) |

Следующие основные измерений в **куба продаж**, используются в качестве фильтров, чтобы разделить статистические значения, чтобы добиться более детальное и более глубокое представление аналитической.

|                  |                                                      |
|------------------|------------------------------------------------------|
| **Entity**       | **Примеры атрибутов**                           |
| Клиенты        | Группы клиентов, клиентов области, адрес, отрасли |
| Товары         | Номер продукта, название продукта, имя группы номенклатуры       |
| Категории продаж | Имена категорий продаж                                 |
| Юридические лица   | Имена юридического лица                                   |
| Даты            | Даты                                                |

По умолчанию пакет содержимого отображает данные для текущего календарного года, но можно открывать раздел фильтры отчета и изменить фильтр даты. Можно также изменить фильтр компании.
