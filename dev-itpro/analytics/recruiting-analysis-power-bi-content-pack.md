# <a name="recruiting-power-bi-content"></a>Содержимое Power BI по набору сотрудников

В этой теме описывается Dynamics 365 для операций - содержимое Power BI по набору сотрудников. Объясняется, как получить доступ к отчетам, которые включены в пакет содержимого и содержит сведения о модели данных и объекты, которые были использованы для построения содержимого пакета.

<a name="accessing-the-content-pack"></a>Доступ к содержимого пакета
--------------------------

Пакет содержимого набор персонала можно найти в библиотеке активов Shared в Microsoft Dynamics Lifecycle Services (LCS). Дополнительные сведения о том, как загрузить пакет содержимого и подключиться к Microsoft Dynamics 365 для данных операций см. в разделе [содержимое Power BI в LCS корпорации Майкрософт и ее партнеры](power-bi-content-microsoft-partners.md).

## <a name="reports-that-are-included-in-the-content-pack"></a>Отчеты, включенные в пакет содержимого
После подключения пакет содержимого данных Dynamics 365 for Operations отчетов отображают данные вашей организации. Если вы никогда не использовали Microsoft Power BI перед, дополнительные о нем на [страницу интерактивной обучения для бизнес-Аналитики питания](https://powerbi.microsoft.com/en-us/guided-learning/?WT.mc_id=PBIService_GetData). Отчеты, включенные в пакет содержимого имеют диаграммы и таблицы, которые содержат дополнительные сведения. В следующей таблице приводится описание отчетов.

| Отчет                       | Содержание                                                                                               |
|------------------------------|--------------------------------------------------------------------------------------------------------|
| Анализ кандидата           | Кандидаты на задания, источники кандидата, кандидатов, местоположение и общее число кандидатов           |
| Статус соискателя             | Кандидаты, тип и состояние и статусе кандидата                                                    |
| Демографические данные кандидата       | Кандидаты, возраста и пола и кандидатов, уровень образования и статус                             |
| Анализ по набору сотрудников          | Чистые отношение приема на работу, среднее число дней для найма процент плохой найма сотрудников и затрат для набора сотрудников                    |
| Анализ проектов по набору сотрудников | Число проектов по набору сотрудников, вступление проекту по набору сотрудников или кандидатов проекту по набору сотрудников |

Можно отфильтровать карточки на эти отчеты и диаграммы и прикрепить диаграммы и фрагментов на панель мониторинга. Дополнительные сведения о фильтрации и ПИН-код в Power BI см. в разделе [создать и настроить панель мониторинга](https://powerbi.microsoft.com/en-us/guided-learning/powerbi-learning-4-2-create-configure-dashboards).

## <a name="understanding-the-data-model-and-entities"></a>Понимание модели данных и объектов
Dynamics 365 для данных операций используется для заполнения отчетов в пакет содержимого набор персонала. В следующей таблице показаны объекты, что пакет содержимого был основан на.

| Объект                          | Содержание                                                         | Связи с другими сущностями                                                                                                                                                                                                                 |
|---------------------------------|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Набор сотрудников\_кандидата           | Кандидаты, зачислен принятый кандидатов, отношение нетто найма и затраты          | Набор сотрудников\_ApplicantName о приеме на работу\_компании о приеме на работу\_CalendarOffset Recuriting\_Дата найма на работу\_GeographicLocation о приеме на работу\_демографические сведения о приеме на работу\_заданий по набору сотрудников\_набором носителей\_RecruitmentProject                                |
| Набор сотрудников\_ApplicantName       | Имя кандидата, полное имя и фамилия                   | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_CalendarOffset      | Календарь корреспондирует с отчетами секторов                                | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_компании             | Фильтровать отчеты по компаниям                                   | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_даты                | Дни, недели, месяцы и годы                                   | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_демографические данные        | Дата рождения, пол, семейное положение и этнического происхождения         | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_EmployedApplicant   | Кандидат, производительность, даты начала и тип кандидата           | О приеме на работу\_компании о приеме на работу\_CalendarOffset о приеме на работу\_Дата найма на работу\_GeographicLocation о приеме на работу\_ApplicantName о приеме на работу\_о приеме на работу по набору сотрудников\_производительности по набору сотрудников\_заданий по набору сотрудников\_набором носителей\_RecruitmentProject          |
| Набор сотрудников\_о приеме на работу          | Дата начала, Дата окончания и датой перемещения                        | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_GeographicLocation  | Город, район, почтовый индекс и региона или района                 | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_заданий                 | Функция, тип и заголовок                                        | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_СМИ               | Источник кандидатов                                             | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_производительности         | Оценка, описание и модели рейтинга                            | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_RecruitmentProject  | Описание проекта, статус проекта и отверстия                | Набор сотрудников\_о приеме на работу кандидата\_о приеме на работу EmployedApplicant\_TerminatedApplicant                                                                                                                                                               |
| Набор сотрудников\_TerminatedApplicant | Прекращено кандидатов, причины, производительность и Дата увольнения | О приеме на работу\_компании о приеме на работу\_CalendarOffset о приеме на работу\_Дата найма на работу\_GeographicLocation о приеме на работу\_производительности по набору сотрудников\_демографические сведения о приеме на работу\_о приеме на работу по набору сотрудников\_набором носителей\_RecruitmentProject о приеме на работу\_ApplicantName |

Эти сущности были использованы, чтобы создать вычисляемые меры. Они рассчитываются затем меры используются для вычисления, ключевые индикаторы производительности (KPI) и отчетов, которые используются в пакет содержимого. Если требуется включить дополнительные вычисления на отчеты и панели мониторинга, можно загрузить и изменить файл Recruiting.pbix из LCS. Этот файл находится в модели данных по умолчанию, который был использован для создания содержимого пакета. После внесения изменений можно создать пакет организации содержимого и панели мониторинга, содержащие информацию, который был добавлен.
