# <a name="consolidation-account-groups-and-additional-consolidation-accounts"></a>Группы счетов консолидации и дополнительные счета консолидации

В этом разделе представлены сведения о группы счетов консолидации и дополнительные счета консолидации и объясняется, как они используются в Microsoft Dynamics 365 для операций.

<a name="consolidation-account-groups"></a>Группы счетов консолидации
----------------------------

Группы счетов консолидации позволяют создавать группы счетов, которые требуется использовать для консолидации данных. Чаще всего группу счетов консолидации представляет органами план счетов или отображает учетные записи в группу, которая определяется центрального офиса компании. Консолидацию можно найти группы счетов в **установки** область **консолидации** модуля. При добавлении новой группы вводится уникальный идентификатор и имя для группы счетов.

## <a name="additional-consolidation-accounts"></a>Дополнительные счета консолидации
Дополнительные счета консолидации позволяют назначить группу счетов консолидации счета из существующего плана счетов. Затем можно указать значение счета консолидации и имя. 

Можно найти дополнительные счета консолидации в **установки** область **консолидации** модуля. При создании нового счета консолидации, необходимо указать следующие сведения:

-   **Счет ГК** — это поле является поиск, содержащий все счета ГК, основанные на план счетов, которые были выбраны на странице. При выборе учетной записи, имя автоматически вводится в **имя учетной записи основной** поле.
-   **Группы счетов консолидации** — это поле используется для указания группы для назначения учетной записи. При консолидации двумя способами, необходимо добавить ту же учетную запись во все четыре консолидации учетной записи группы.
-   **Счет консолидации** — ввод значения счета консолидации. Это значение не обязательно должен быть счет из плана счетов. Он может быть любое значение, которое требуется.
-   **Имя счета консолидации** — введите название счета, которое они должны появляться в запросах и отчетах.
-   **Суббота уровень** — это поле используется для отчета по выписок по счету Мексиканского налоговым органам. 

После завершения создания группы счетов консолидации и дополнительные счета консолидации, можно выбрать группу в процессе интерактивной консолидации.
