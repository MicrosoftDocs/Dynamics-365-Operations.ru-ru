# <a name="budget-planning-templates-for-excel"></a>Шаблоны планирования бюджета для Excel

В этом разделе описывается, как создавать шаблоны Microsoft Excel, которые могут использоваться с бюджетными планами.

В этом разделе показано, как создавать шаблоны Excel, которые будут использоваться с бюджетными планами с помощью набора стандартных демонстрационных данных и имя пользователя Admin. Дополнительные сведения о планировании бюджета см. [Обзор планирования бюджета.](budget-planning-overview-configuration.md) Можно также выполнить [101 планирования бюджета](budget-plan.md) учебника, принципы настройки и использования базовом модуле.

## <a name="generate-a-worksheet-using-budget-plan-document-layout"></a>Создание листа с помощью макет документа бюджетного плана
Документы бюджетного плана можно просматривать и редактировать, используя один или несколько макетов. Каждый макет может иметь шаблон документа связанный бюджетный план для просмотра и редактирования данных бюджетного плана в электронной таблице Excel. В этом разделе шаблон документа бюджетного плана будут создаваться с использованием существующей конфигурации макета. Открытие **бюджетные планы список** (**бюджетирования**&gt;**бюджетные планы**). Щелкните **создать** для создания нового документа бюджетного плана. [![bpt1](./media/bpt11-1024x552.png)](./media/bpt11.png) 

Используйте **добавить** параметр, чтобы добавить строки по строке. Щелкните **макеты** для просмотра конфигурации макет документа бюджетного плана. 
[![bpt2](./media/bpt2-1024x274.png)](./media/bpt2.png) 

Можно просмотреть настройки макета и при необходимости измените его. Последовательно выберите пункты **шаблона**&gt;**формирование** для создания файла Excel для данного макета. После создания шаблона, перейдите к **шаблона**&gt;**представление**, чтобы открыть и просмотреть шаблона документа бюджетного плана. Можно сохранить файл Excel на локальном диске. [![bpt3](./media/bpt3-1024x545.png)](./media/bpt3.png) 

> [!NOTE] 
> Макет документа бюджетного плана не может редактироваться после с ним связан шаблон Excel. Чтобы изменить макет, удалите связанный файл шаблона Microsoft Excel и создать его заново. Это необходимо, чтобы сохранить поля в макете и синхронизация листа. 

Шаблон Excel будет содержать все элементы из макета документа бюджетного плана, где **доступно на листе** столбца имеет значение True. Перекрытие элементов не разрешены в шаблоне Excel. Например если формат содержит запрос Q1, запрос Q2, Q3 запрос и 4 квартала запроса столбцы и общего запроса столбец, представляющий общую сумму по всем колонкам 4 квартал, только столбцы ежеквартально или общей столбец доступным для использования в шаблоне Excel. Файл Excel невозможно обновить пересекающихся столбцов во время обновления данных в таблице может стать устаревшей и неточные.

[![bpt4](./media/bpt4-1024x615.png)](./media/bpt4.png)

> [!NOTE] 
> Чтобы избежать возможных проблем с просмотра и редактирования данных бюджетного плана с помощью Excel, тот же пользователь должен войти в обоих Dynamics 365 операций и надстройки Microsoft Dynamics Office подключения к данным.

## <a name="add-a-header-to-budget-plan-document-template"></a>Добавить заголовок документа шаблон бюджетного плана
Чтобы добавить сведения заголовка, выберите верхнюю строку в файле Microsoft Excel и вставить пустые строки. Щелкните **разработки** в **подключения к данным** для добавления полей заголовка в файле Excel.

[![bpt5](./media/bpt5-1024x615.png)](./media/bpt5.png) 

В **разработки** вкладке ** ** щелкните **добавить** поля, а затем выберите **BudgetPlanHeader** в качестве источника данных объекта.

[![bpt6](./media/bpt6-1024x615.png)](./media/bpt6.png)

Установить курсор в нужное место в файле Excel. Щелкните **добавить метку** Чтобы добавить подпись в выбранном месте. Выберите **добавить значение** для добавления туда, выбранной в поле значение. Щелкните **сделать** закрыть конструктор.

## <a name="bpt7mediabpt7pngmediabpt7png"></a>[![bpt7](./media/bpt7.png)](./media/bpt7.png)

<a name="add-a-calculated-column-to-budget-plan-document-template-table"></a>Добавление вычисляемого столбца таблицы шаблон документа бюджетного плана
--------------------------------------------------------------

Далее, вычисляемые столбцы будут добавлены шаблон документа созданного бюджетного плана. A **общего запроса** столбец, в котором приводится общая запрос Q1: столбцы запроса 4 квартала и **корректировки** столбец, в котором пересчитывает **общего запроса** столбца с множителем предопределенным.

Щелкните **разработки** в **подключения к данным** для добавления столбцов в таблице. Щелкните **редактирование** рядом с **BudgetPlanWorksheet** источника данных, чтобы начать добавление столбцов.

[![bpt8](./media/bpt8-1024x301.png)](./media/bpt8.png) 

Группы выбранного поля отображаются столбцы, доступные в шаблоне. Щелкните **формула**, чтобы добавить новый столбец. Имя нового столбца, а затем вставить формулу в **формула** поле. Щелкните **обновление** для вставки столбца.

[![bpt12](./media/bpt12-1024x565.png)](./media/bpt12.png)

> [!NOTE] 
> Чтобы определить формулу, создать формулу в электронной таблице и скопируйте ее в **разработки** окно. Dynamics 365 для присоединенной таблицы операций обычно можно назвать «AXTable1». Например, чтобы просуммировать запрос Q1: запрос 4 квартала столбцов электронных таблиц, формула = AxTable1\[запрос Q1\]+ AxTable1\[запрос Q2\]+ AxTable1\[запрос Q3\]+ AxTable1\[запрос 4 квартала\].

Повторите эти шаги, чтобы вставить **корректировки** столбца. Можно использовать формулу = AxTable1\[общего запроса\]\*$I$ 1 для этого столбца. Будет иметь значение в ячейке I1 и умножение значений в **общего запроса** столбца для расчета суммы корректировки.

Сохраните и закройте файл Excel. Вернуться к Dynamics 365 для операций, а также в **макеты**, щелкните **шаблона &gt;загрузки** для загрузки сохраненного шаблона Excel для бюджетного плана. 

[![bpt10](./media/bpt10-1024x352.png)](./media/bpt10.png) 

Закройте **макеты** ползунка. В **бюджетный план** документ, нажмите кнопку **листа** для просмотра и редактирования его в Excel. Обратите внимание, что скорректированного шаблона Excel, использованного для создания этот лист бюджетного плана и вычисляемые столбцы обновляются с использованием формулы, которые были определены на предыдущих шагах. 

[![bpt11](./media/bpt111-1024x431.png)](./media/bpt111.png)

## <a name="tips--tricks-for-creating-budget-plan-templates"></a>Советы и рекомендации по созданию шаблонов бюджетного плана
### <a name="can-i-add-and-use-additional-data-sources-to-a-budget-plan-template"></a>Можно ли добавлять и использовать дополнительные источники данных в шаблон бюджетного плана

Да, можно использовать **разработки** меню, чтобы добавить дополнительные сущности к тому же или других листов шаблона Excel. Например, можно добавить **BudgetPlanProposedProject** источник данных для создания и ведения списка предложенных проектов в то же время при работе с бюджета плана данных в Excel. Обратите внимание, что включая источники большого объема данных может повлиять на производительность рабочей книги Excel. 

Можно использовать **фильтр** параметр в **подключения к данным** для добавления нужного фильтров к дополнительным источникам данных.

### <a name="can-i-hide-the-design-option-in-the-data-connector-for-other-users"></a>Можно скрыть параметр конструктора в connector данных для других пользователей

Да, открыть **подключения к данным** скрытие **разработки** параметр от других пользователей.

[![bpt13](./media/bpt13-1024x565.png)](./media/bpt13.png)

Разверните **параметры connector данных** и снимите **включить разработки** флажок. Таким образом будут скрыты **разработки** параметра из **подключения к данным**.

[![bpt14](./media/bpt14-1024x592.png)](./media/bpt14.png)

### <a name="can-i-prevent-users-from-accidently-closing-the-data-connector-while-working-with-data"></a>Можно запретить пользователям из случайно закрытия подключения к данным во время работы с данными

Рекомендуется блокировать шаблон таким образом, чтобы пользователи не смогли закрыть его. Чтобы включить блокировку, щелкните **подключения к данным**, в верхнем правом углу появится стрелка. 

[![bpt15](./media/bpt15-1024x285.png)](./media/bpt15.png) 

Щелкните стрелку для дополнительное меню. Выберите **блокировка**.

### <a name="bpt16mediabpt16-1024x614pngmediabpt16png"></a>[![bpt16](./media/bpt16-1024x614.png)](./media/bpt16.png)

### <a name="can-i-use-other-excel-features-like-cell-formatting-colors-conditional-formatting-and-charts-with-my-budget-plan-templates"></a>Можно использовать другие возможности Excel, такие как форматирование ячеек, цвета, условное форматирование и диаграмм с Мои шаблоны бюджетного плана

Да, большинство стандартных возможностей Excel будет работать в шаблоны бюджетного плана. Рекомендуется использовать цветовое кодирование для пользователей, чтобы различать столбцы только для чтения и может быть изменено. Условное форматирование используется для выделения проблемных областей бюджета. Итоги по столбцам может быть легко представлен с помощью стандартной формулы Excel над таблицей.

Можно также создать и использовать сводные таблицы и диаграммы для дополнительных группирование и визуализации данных бюджета. На **данные** вкладке **соединений** щелкните **обновить все**и нажмите кнопку **свойства подключения**. Щелкните **использование** вкладке. В разделе **обновление**, выберите **обновление данных при открытии файла** флажок. 

[![bpt17](./media/bpt17-1024x614.png)](./media/bpt17.png)
