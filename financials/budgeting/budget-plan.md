# <a name="budget-planning"></a>Бюджетное планирование

Цель данной работы – для создания управляемой представления Microsoft Dynamics 365 для операций обновления функций в области планирования бюджета. Смысл этой практической работы — проиллюстрировать быстрый пример настройки модуля бюджетного планирования и показать, как планирование бюджета может быть выполнено используя эта конфигурация.  Это лаборатория будет уделено специально следующие бизнес-процессов или задач: Создание организационной иерархии для бюджетного планирования и настройке безопасности пользователей - Определение сценариев бюджетного плана, план столбцов бюджета, макеты и шаблонов Excel — Создание и активация бюджетного планирования процесса - создание документ бюджетного плана путем извлечения в фактические данные из главной книги — использование распределений для настройки данных документа бюджетного плана - редактирование данных бюджетного плана документа в Excel 

<a name="prerequisites"></a>Необходимые условия 
------------------

В этом руководстве необходимо для доступа к Dynamics 365 для операций среды с Contoso демонстрационные данные и быть подготовлен с правами администратора на экземпляре. Не используйте в частный режим обозревателя для этой демонстрации - выход из любой другой учетной записи в веб-обозревателе при необходимости и войдите в систему с Dynamics 365 учетных данных администратора операций. При входе в Dynamics 365 для операций, вы **необходимо** установите флажок «Запомнить меня вошел в систему». В результате будет создан постоянный файл cookie, необходимый приложению Excel в данный момент. При входе Dynamics 365 для операций с помощью обозревателя, отличных от IE, затем вам будет предложено вход в приложение Excel. При щелчке "Вход" в приложении Excel откроется всплывающее окно IE, и при входе **НЕОБХОДИМО** установить флажок "Оставаться в системе". Если при щелчке "Вход" в приложении Excel ничего не происходит, очистите кэш файлов cookie в IE.

## <a name="scenario-overview"></a>**Scenario overview**
Julia работает менеджером по финансам в Contoso Entertainment Systems в Германии (DEMF). По мере приближения FY2016 ей нужно приступать к настройке бюджета компании на предстоящий год. Подготовка бюджета выглядит следующим образом:

1.  Julia использует фактические суммы предыдущего года как начальную точка для создания бюджета.
2.  На основании фактических сумм предыдущего года она создает оценки на 12 месяцев в предстоящем году.
3.  Julia пересматривает бюджет с финансовым директором. По завершении она вносит необходимые корректировки в бюджетный план и завершает подготовку бюджета.
