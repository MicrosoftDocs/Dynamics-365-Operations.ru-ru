# <a name="post-fixed-asset-transactions-to-posting-layers"></a>Разноска проводок ОС по слоям разноски

Эта статья содержит обзор функции слоя разноски для проводок основных средств.

Основные средства часто амортизируются различными способами для различных целей. Амортизация для целей налогообложения рассчитывается с использованием текущего принципа налогообложения, чтобы достигнуть самой высокой возможной амортизации перед выплатой налогов, но амортизация для целей отчетности вычисляется в соответствии с законами и стандартами учета. Различные виды амортизации вычисляются и записываются отдельно в слоях разноски.

Каждая книга, которая присоединяется к основным средствам, настраивается для определенного слоя разноски, имеющего общее задание на амортизацию. Предусмотрено десять слоев разноски: Текущий, Операции, Налоги и семь настраиваемых слоев. Можно также отключить разноску в главную книгу для этой книги, установив в поле Разносить в главную книгу значение Нет. В поле Слой разноски затем автоматически задается значение Нет. Журналы, которые не выполнять разноску в главную книгу, обычно используются для целей налоговой отчетности. Этот подход дает дополнительную гибкость для удаления архивных проводок из журнала активов, так как они еще не были подтверждены в главную книгу.

Журналы основных средств определяются с помощью страницы Наименования журналов по пути Главная книга > Настройка журнала > Наименования журналов. Каждый журнал, в котором можно разнести амортизацию, определяется своим наименованием журнала только для одного слоя разноски. Слой разноски в журнале не может быть изменен. Это ограничение помогает гарантировать разделение проводок для каждого слоя разноски. Для каждого слоя разноски должно быть создано по крайней мере одно имя журнала. Если вы используете журналы, которые не выполнять разноску в главную книгу, необходимо также создать журнал, в котором слой разноски задано значение Нет.

Можно назначить счета ГК для проводок основных средств на странице Профили разноски основных средств. Для каждого профиля разноски необходимо выбрать соответствующий тип проводки и книгу, а затем назначить счета учета. Настройте запись профиля разноски для каждой книги, которая будет разноситься в главную книгу.

> [!NOTE] 
> С помощью журналов производной, в то же время можно разнести проводки по разным слоям разноски. Вами создаются проводки исходной книги в журнал, в котором слой разноски соответствует слою разноски книги. В процессе разноски проводки производных книг разносятся по соответствующим слоям разноски.

Для получения дополнительных сведений см. в [журналах производной](derived-books.md) и [Разноска с журналами производной](post-derived-value-models.md).
