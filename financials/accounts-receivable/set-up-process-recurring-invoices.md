# <a name="set-up-and-process-recurring-invoices"></a>Настройка и обработка периодических накладных

[!include[banner](../includes/banner.md)]


В этой статье описывается, как настроить и обработать повторяющиеся накладные. Можно использовать повторяющийся накладные, если необходимо выставлять накладные для клиентов на то же самое количества на регулярной основе.

<a name="create-a-recurring-free-text-invoice-template"></a>Создание шаблона повторения накладной с произвольным текстом
---------------------------------------------

Чтобы выставлять накладные клиентам за те же услуги на регулярной основе, необходимо определить шаблон накладной с произвольным текстом, который можно повторно использовать для создания накладных. Шаблон содержит следующую информацию:

-   Сведения о заголовке, например налоговые группы, условия оплаты и способ оплаты
-   Сведения о строке, например описание услуги, счета выручки, цена за единицу и сумма накладной
-   Накладные расходы по отгрузке или обработке
-   Распределения по бухгалтерским счетам вместе со сведениями о финансовых аналитиках, например места возникновения затрат и бизнес-единицы

В сущности, вы создаете всю накладную и сохраняете ее как шаблон. Шаблоны можно настроить на странице **Периодические накладные**.

## <a name="assign-a-free-text-invoice-template-to-a-customer-and-enter-recurrence-details"></a>Назначьте шаблон накладной с произвольным текстом клиенту и укажите параметры повторения.
После создания шаблона необходимо назначить шаблон клиентам, которым необходимо выставить накладные. Кроме того, необходимо указать, когда и как часто будет использоваться накладная. Шаблоны можно назначить на вкладке **Накладная** на странице **Клиенты**. Добавьте шаблон в список и обновите следующую информацию:

-   Дата начала и по желанию дата окончания для выставления периодических накладных
-   Частота выставления периодических накладных (например, каждый день или раз в месяц)
-   Максимальная сумма выставляемых накладных (если эта информация необходима)

Клиент может иметь несколько шаблонов с различной частотой.

## <a name="generate-the-recurring-invoices"></a>Создание периодических накладных
На странице **Периодические накладные** имеется задача, которая обрабатывает шаблоны периодических накладных. Укажите дату накладной и шаблон для создания накладных. Накладные будут созданы, и им будет назначен один номер кода повторения для каждой обрабатываемой группы накладных.
Накладные с произвольным текстом повторения разноски
---------------------------------

После создания периодических накладных коды повторения накладных будут отображаться в задаче разноски на странице **Периодические накладные**. Можно просмотреть все накладные по коду повторения, перейдя по ссылке. Во время предварительного просмотра накладных по коду повторения можно удалить отдельные накладные. Параметры повторения клиента будут сброшены для этого шаблона, чтобы его можно было повторно создать позже. Можно разнести одну, несколько или все накладные по коду повторения. Если включены workflow-процессы, щелкните **Отправить** перед разноской накладных.
Печать периодических накладных с произвольным текстом
----------------------------------

После разноски периодических накладных можно напечатать накладные на странице списка накладных с произвольным текстом. Можно напечатать выбранные накладные или можно выбрать ряд накладных для печати.
