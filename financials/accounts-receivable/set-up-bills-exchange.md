# <a name="set-up-bills-of-exchange"></a>Настройка спецификаций

В этом разделе описываются шаги по настройке переводных векселей.

Переводной вексель — это письменное или электронное распоряжение клиента, который указывает, что другой сторона (обычно банк), которая должна выплатить указанную сумму компании. При использовании переводного векселя в качестве средства платежа по накладной заказа на продажу или накладной с произвольным текстом вы кредитуете счет клиента. Этот кредит гарантируется переводным векселем до того момента, когда клиент представит переводный вексель в банк. Обычно оплата накладной с переводного векселя в конкретную дату. После получения уведомления из банка об акцептовании переводного векселя последний можно закрыть. Можно выписать переводный вексель через свой банк в одном из следующих случаев:

-   На дату исполнения. Такой подход называется предъявление к оплате.
-   До даты исполнения, обычно на дату учета векселя, который указан в поле условия оплаты, которые были настроены для клиента. При разноске проводки, сумма скидки разносится на счет расходов. Остаток суммы является вашим обязательством до получения банком оплаты от клиента. Такой подход называется предъявления для дисконтирования.

## <a name="set-up-posting-profiles-for-bills-of-exchange"></a>Настройка профилей разноски для переводных векселей
Используйте **клиентов, профили разноски** страницу, чтобы настроить профили разноски, которые можно использовать с переводными векселями, опротестованных переводных векселей, предъявлений к оплате и предъявления для дисконтирования. В **итоговый счет** выберите сводный счет для разноски сумм переводных векселей. Этот счет дебетуется или кредитуется, в зависимости от типа проводки переводного векселя:
-   Для переводных векселей этот счет дебетуется при разноске переводного векселя, и он кредитуется при разноске предъявления для дисконтирования или предъявления для погашения.
-   Для разнесенных переводных векселей этот счет дебетуется при разноске опротестованного переводного векселя.
-   Для предъявления для погашения этот счет дебетуется при разноске предъявления для погашения.
-   Для предъявления для дисконтирования этот счет дебетуется при разноске предъявления для дисконтирования.

В **закрытие счета** выберите наличный счет для разноски сумм переводных векселей. Этот счет дебетуется при оплате переводного векселя. В **счет налога по предоплате** выберите сводный счет для разноски сумм налогов при использовании переводных векселей для предоплат. В **обязательства для счета скидок** выберите счет для разноски суммы скидки для предъявления для дисконтирования для. Этот счет кредитуется при разноске предъявления для дисконтирования.

## <a name="set-up-accounts-receivable-parameters-for-bills-of-exchange"></a>Настройка параметров расчетов с клиентами для переводных векселей
На **параметры расчетов с клиентами** страницей по умолчанию, профили разноски для переводных векселей, введенные в **Главная книга и налог** вкладке. Номерных серий определяются в **номерные серии** вкладке.
Настройка имен журналов для переводных векселей
------------------------------------------

На **имена журналов** страницу, создайте по крайней мере пять имен журналов для переводных векселей. Ниже приведены типы журналов:
-   **Выписка переводного векселя на клиента** — создание наименования журнала для журнала выписки переводных векселей.
-   **Протест переводного векселя на клиента** — создание наименования журнала для журнала опротестования переводного векселя.
-   **Перевыписки переводного векселя на клиента** — создание наименования журнала для журнала перевыписки переводных векселей.
-   **Банковский перевод клиента** — создание наименования журнала для журнала предъявлений к оплате.
-   **Сопоставление переводного векселя клиентом** – Создайте имя журнала для журнала сопоставления переводных векселей.

На странице журнала операций для каждого журнала переводных векселей введите информацию о переводном векселе на **переводной вексель** вкладке. После разноски строк журнала переводного векселя их можно просмотреть в **запрос журнала переводных векселей** страницы и **Статистика переводных векселей** страницы.
Настройка способов платежа для переводных векселей
-----------------------------------------------

На **способы оплаты** страницу, настроить по крайней мере одного метода платежа для переводных векселей. Если вы ведете дела через несколько банков, настройте способ платежа, соответствующий формату предъявления к оплате, требующий каждого из банков для переводных векселей.
Настройка сборов по платежам для переводных векселей
-----------------------------------------

Сбор по платежу - это сбор, связанный с процессом сбора платежей с клиентов. Несколько строк могут быть Настройка сборов по платежам, связанных каждый сбор по платежу. Можно помощью строк настройки можно управлять способами расчета сумм сборов по платежам по умолчанию. Например, можно создать строки настройки для методов платежа, спецификаций платежей, валют и периодов. Можно также создать строки настройки для процента или суммы на основе интервалов дней. Например можно настроить процентной ставки на основе указанного времени, когда платеж просрочен. Если банк взимает различные сборы для перевода различных типов, таких как **коллекции** или **скидка**, настройте строку сбора отдельный платеж для каждого типа предъявления к оплате.
Настройка сборов за предъявления к оплате для файлов банковских переводов.
------------------------------------------------

На **банковские счета** страницу, можно настроить сборы по платежам, банк взимает для каждого создаваемого файла предъявления к оплате. Сборы по платежам разносятся после того, как перевод подтвержден и стали известны суммы осуществленных сборов. Сборы за предъявление к оплате отличаются от сборов по платежам, которые собираются от клиентов и присоединенных к строкам журнала.
Настройка структуры документов для переводных векселей
---------------------------------------------

На **банковские счета** и щелкните **Настройка**и укажите шаблон документа, который необходим для каждого банковского счета, что которого будут создаваться печатные переводных векселей документов для.
Настройка клиентов для переводных векселей
--------------------------------------

На **клиентов** странице, для каждого клиента, который выразил согласие осуществлять оплату переводным векселем, можно настроить способ оплаты по умолчанию для переводных векселей на **значения оплаты по умолчанию** вкладке.
