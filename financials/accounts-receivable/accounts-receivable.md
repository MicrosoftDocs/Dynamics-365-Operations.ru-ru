# <a name="accounts-receivable-home-page"></a>Домашняя страница расчетов с клиентами

[!include[banner](../includes/banner.md)]


Используйте расчеты с клиентами для отслеживания накладных клиента и входящих платежей клиента. 

Можно создать накладные клиента, основанные на заказах на продажу или отборочных накладных. Можно также ввести накладные с произвольным текстом, которые не относятся к заказам на продажу. Платежи можно получить с помощью нескольких различных типов платежей. К ним относятся переводные векселя, наличные деньги, чеки, кредитные карты и электронные платежи. Если в организации имеется несколько юридические лиц, можно использовать централизованные платежи для записи платежей в одном юридическом лице от имени других юридических лиц.
