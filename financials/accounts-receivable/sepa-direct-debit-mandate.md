# <a name="set-up-sepa-direct-debit-mandate"></a>Настройка предписания прямого дебетования SEPA



Прямое дебетование единой зоны платежей в евро (SEPA) позволяет кредитору снимать средства с банковского счета клиента, при условии что клиент предоставил кредитору подписанное предписание. Клиент подписывает предписание, которое дает кредитору право собирать платежи и поручает банку клиента осуществлять оплату. В этом разделе представлены для отображения процесса настройки предписаний SEPA прямого дебетования.

## <a name="prerequisites"></a>Необходимые условия
В следующей таблице показаны обязательные компоненты, которые должны быть подготовлены до начала работы.

| Категория       | Необходимые условия                                                                                                                                              |
|----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| Страна/регион | Основной адрес для юридического лица должен находиться в следующих странах/зонах: Австралия, Бельгия, Германия, Испания, Франция, Италия или Нидерланды. |

1. Настройте номерную серию для прямого дебетования предписаний каждого прямого дебетования предписания должен иметь уникальный номер. Используйте страницу **Номерные серии**, чтобы создать номерную серию для предписаний прямого дебетования. Вы будете использовать этот идентификатор, чтобы назначать номерную серию системе предписаний прямого дебетования на странице **Параметры расчетов с клиентами**.

2. Настройка параметров расчетов с клиентами для прямого дебетования требует использования **параметры расчетов с клиентами** страницу, чтобы настроить параметры для прямого дебетования предписаний. Чтобы настроить параметры, на **прямого дебетования** измените параметры по умолчанию, сколько требуется. Затем на **номерные серии** вкладке, обновите **код прямого дебетования предписания** поле к номерной серии, которая настраивается ранее.

3. Настройка способа оплаты для прямого дебетования задает вы необходимо настроить метод платежа для прямого дебетования предписаний. Этот способ оплаты используется для запроса накладных для создания платежей прямого дебетования. Используйте страницу **Способ оплаты**, чтобы настроить способ оплаты. Чтобы настроить способ оплаты для предписаний прямого дебетования, выполните следующие дополнительные шаги для способа оплаты:

-   В поле **Тип платежа** выберите **Электронный платеж**.
-   Необязательно: Если ожидается, что каждый из ваших клиентов, чтобы иметь несколько задает, в **период** выберите **накладной**. Создается отдельный платеж для каждой накладной, и каждый платеж будет использовать требование, которое указано для накладной.
-   Выберите параметр **Требуется поручение**, чтобы создавать платежи с помощью предписаний прямого дебетования. Параметр **Требуется поручение** доступен, только если в поле **Тип платежа** выбрано значение **Электронный платеж**.
