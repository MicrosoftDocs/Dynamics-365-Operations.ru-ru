# <a name="settle-a-partial-vendor-payment-and-the-final-payment-in-full-before-the-discount-date"></a>Сопоставление частичного платежа поставщика и окончательного платежа полностью до даты скидки

[!include[banner](../includes/banner.md)]


В этой статье рассматривается сценарий, в котором производится частичные платежи по накладной поставщика и берется скидка по оплате.

Компания Fabrikam покупает товары у поставщика 3064. Поставщик предоставляет Fabrikam скидку на оплату в размере 1 %, если счет будет оплачен в течение 14 дней. Накладные должны быть оплачены в течение 30 дней. Поставщик также позволяет Fabrikam использовать скидки на оплату по частичным платежам. Параметры сопоставления расположены на странице **Параметры модуля расчетов с поставщиками**. 25 июня Эйприл вводит накладную на 1 000,00 для поставщика 3064.

## <a name="vendor-invoice-on-june-25"></a>Накладная поставщика от 25 июня
25 июня Эйприл вводит и разносит накладную на 1 000,00 для поставщика 3064. На странице **Проводки по поставщику** Эйприл может просмотреть эту проводку.

| Операция   | Дата      | Счет | Дебетовая сумма в валюте проводки | Сумма кредита в валюте проводки | Сальдо   | Валютное |
|-----------|-----------|---------|--------------------------------------|---------------------------------------|-----------|----------|
| Inv-10010 | 25.06.2015 | 10010   |                                      | 1 000,00                              | -1 000,00 | американский доллар      |

Со страницы **Поставщики** Эйприл открывает страницу **Сопоставление проводок**. Она может использовать форму **Сопоставление проводок** для просмотра дат и сумм скидок по оплате. Срок выполнения — 25 июля, и скидка по оплате 10,00 доступна, если накладная оплачена к 9 июля.

| Пометка | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
|      | Обычная            | Inv-10010 | 3064    | 25.06.2015 | 25.07.2015 | 10010   | 1 000,00                       | американский доллар      | 990,00           |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 09.07.2015 |
| Сумма скидки по оплате         | -10,00    |
| Использовать скидку по оплате            | Обычная    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | -10,00    |

Эйприл щелкает вкладку **Скидка по оплате**, чтобы просмотреть сумму скидки.

| Дата скидки по оплате | Сумма скидки по оплате | Сумма в валюте проводки |
|--------------------|----------------------|--------------------------------|
| 09.07.2015           | 10,00                | 990,00                         |
| 25.07.2015          | 0,00                 | 1 000,00                       |

## <a name="partial-payment-on-july-1-by-using-the-settle-transactions-page"></a>Частичный платеж на 1-го июля с помощью страницы "Сопоставление проводок"
Эйприл может создать журнал платежей для данного платежа, открыв форму **Журнал платежей** в модуле "Расчеты с поставщиками". Она создает новый журнал и вводит строку для поставщика 3064. Она после этого открывает страницу **Сопоставление проводок**, чтобы отметить накладную для сопоставления. Эйприл отмечает накладную и меняет значение в поле **Сумма сопоставления** на **-500,00**. Она видит, что значение в поле **Сумма скидки по оплате** равно **-10,00** для всей накладной, и что значение в поле **Сумма скидки по оплате** равно **-5,05**. Следовательно, Эйприл сопоставляет -505,05 из данной накладной.

| Пометка     | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
| Выбрано | Обычная            | FTI-10010 | 4028    | 25.06.2015 | 25.07.2015 | 10010   | 1 000,00                       | американский доллар      | -500,00          |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 09.07.2015 |
| Сумма скидки по оплате         | -10,00    |
| Использовать скидку по оплате            | Обычная    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | -5,05     |

Эйприл хочет сопоставить ровно половину накладной. Поэтому она изменяет значение в поле **Сумма сопоставления** на **-495,00**. Полная сопоставленная сумма теперь равна 500,00. Эта сумма включает скидку по оплате -5,00.

| Пометка     | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
| Выбрано | Обычная            | FTI-10010 | 4028    | 25.06.2015 | 25.07.2015 | 10010   | 1 000,00                       | американский доллар      | -495,00          |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 09.07.2015 |
| Сумма скидки по оплате         | -10,00    |
| Использовать скидку по оплате            | Обычная    |
| Скидка по оплате          | 0,00      |
| Сумма скидки по оплате | -5,00     |

Эйприл закрывает страницу **Сопоставление проводок**. Строка платежа для 495,00 создана в журнале, затем Эйприл разносит журнал. Эйприл может просмотреть проводки по поставщику на странице **Проводки по поставщику**. Она видит, что накладная имеет сальдо -500,00. Также она видит платеж на сумму 495,00 и скидку по оплате 5,00.

## <a name="remaining-amount-paid-on-july-8"></a>Остальная сумма уплачивается 8 июля
Эйприл платит остальную часть накладной поставщику 3064 8 июля, попадающее в период предоставления скидки. Она создает журнал платежей 8 июля и помечает проводку для сопоставления. Она видит, что необходимо сопоставить 495,00. Значение в поле **Оцененная скидка по оплате** — **-5,00**, потому что ранее была использована скидка 5,00.

|                         |        |
|-------------------------|--------|
| Итого по отмеченным строкам            | 495,00 |
| Оцененная скидка по оплате | -5,00  |

Информация о отмеченных проводах появляется в таблице на странице **Сопоставление открытых проводок**.

| Пометка     | Использовать скидку по оплате | Операция   | Учетная запись | Дата      | Срок выполнения  | Счет | Сумма в валюте проводки | Валютное | Сумма сопоставления |
|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------|----------|------------------|
| Выбрано | Обычная            | FTI-10010 | 4028    | 25.06.2015 | 25.07.2015 | 10010   | 1 000,00                       | американский доллар      | 495,00           |

Данные по скидкам появляются внизу страницы **Сопоставление открытых проводок**.

|                              |           |
|------------------------------|-----------|
| Дата скидки по оплате           | 09.07.2015 |
| Сумма скидки по оплате         | 10,00     |

| Скидка по оплате          | 5,00      |
