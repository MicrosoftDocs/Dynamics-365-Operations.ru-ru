# <a name="vendor-payments-for-a-partial-amount"></a>Платежи поставщику на частичную сумму

Иногда необходимо производить платеж поставщику, который меньше суммы накладной. Эта статья описывает различные варианты для обработки этой ситуации. Доступные варианты зависят от потребностей и конфигурации бизнеса. 

<a name="cash-discount-amounts"></a>Суммы скидки по оплате
---------------------

Поставщик может предложить скидку по оплате в случае оплаты по накладной до срока выполнения. Например, вы вводите накладную на 100,00, которая определяет скидку по оплате в размере 2 процентов, если накладная оплачена в течение 10 дней. Условия срока оплаты — 30 дней. Если предложение по оплате использует скидки по оплате как критерий выбора накладную, а предложение, запускается на или до даты скидки при оплате наличными, выбранной накладной для платежа, а платеж создается суммой 98,00. Скидка при оплате наличными, также может быть получено для одноразовых платежа, который был создан вручную.

## <a name="partial-payments-with-cash-discounts"></a>Частичные платежи со скидками по оплате наличными
При выполнении частичного платежа можно запланировать дополнительные частичные платежи для полной оплаты накладной. Чтобы получить скидку при оплате наличными для частичного платежа, необходимо задать ** вычислять скидки по оплате для частичных платежей ** параметр для **Да** на **с поставщиками параметры счета** страницы. 

Например, можно получить скидку по оплате 2%, если накладная оплачивается в течение 10 дней после выставления. Накладная разнесена для 100,00. Если сделан платеж на 49,00 в течение 10 дней, дебет 49,00 вводится в журнал платежей. При сопоставлении частичного платежа на **сопоставление открытых проводок** страницу, **1,00** появляется в **сумма скидки при оплате наличными по** поле. 

> [!NOTE] 
> Если ввести частичный платеж и оставьте полную сумму накладной в **сумма для сопоставления** поле, **сумма скидки при оплате наличными по** автоматически пересчитывается при разноске проводок.

## <a name="credit-notes-with-cash-discounts"></a>Кредит-ноты со скидками по оплате
Можно вернуть некоторые номенклатуры, указанные в накладной, и получить кредит-ноту. Если скидка по оплате уже была получена для исходной накладной, можно вычесть значение скидки и получить возмещение верной суммы. Если ** вычислять скидки по оплате для кредит-нот ** включен режим **Да** на **параметры расчетов с поставщиками** страницы, скидка рассчитывается автоматически для кредит-ноты. 

Например, можно получить скидку по оплате 2%, если накладная оплачивается в течение 10 дней после выставления. Накладная разнесена для 100,00. При возврате товаров и получении кредит-ноты, можно ввести кредит-ноты для полной суммы исходной накладной 100,00 вместе с 2 процента скидки при оплате наличными, также определенный в кредит ноте.  При просмотре кредит-ноты на **сопоставление проводок** страницу, **98,00** появляется в **сумма для сопоставления** поле, и **-2,00** появляется в **сумма скидки при оплате наличными** поле. Сумма скидки разнесена на счет скидки по оплате.

## <a name="overpaymentunderpayment-amounts"></a>Суммы по переплате/недоплате
Вы могли сделать частичную оплату, при которой оставшаяся для сопоставления сумма будет очень небольшой. Например, сумма по накладной поставщика равна 1 000,00, а вы оплачиваете 999,90. Если оставшаяся сумма меньше суммы, которая определена для переплат или недоплат на странице **Параметры модуля расчетов с поставщиками**, разница автоматически разносится на счет ГК для переплат и недоплат.
