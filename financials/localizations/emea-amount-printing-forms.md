# <a name="update-how-amounts-are-displayed-on-reports-and-documents"></a>Обновление отображения сумм в отчетах и документов

В этом разделе представлена информация о том, как обновить способ отображения сумм в отчетах и других документах для Эстонии, Латвии, Литве, Польше, Чехии, Венгрии и России.

Для компаний в Эстонии, Латвии, Литве, Польше, Чехии, Венгрии и России можно настроить полные имена и имена для денежных единиц и единиц. Эти имена можно использовать для преобразования представления сумм в документах и отчеты. Например: сумма **100.20 ЛИТОВ** может отображаться как **100 литов 20 центов**.

## <a name="set-up-full-and-short-names-for-currency-units-and-subunits"></a>Настройка полные и краткие имена для денежных единиц и единиц
Чтобы настроить полные и краткие имена, денежных единиц и единиц для языка, выполните следующие действия:

1.  Открытие **валют** страницы.
2.  Выбор валюты.
3.  В области действий щелкните **Склонение**.
4.  Чтобы добавить полное наименование и краткое имя для языка, щелкните **создать** и заполните следующие поля.
    |                                                           |                                                                                                                                                                                                                    |
    |-----------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
    | **Field**                                                 | **Description**                                                                                                                                                                                                    |
    | **Language**                                              | Выбрать язык для текущего текста.                                                                                                                                                                          |
    | **Именительный падеж единственное число (имя поля группы единиц)**       | Ввод валюты в единственном. Например в единственном литов является литов.                                                                                                                         |
    | **Именительный падеж множественное число (имя поля группы единиц)**         | Ввод валюты во множественном числе формы. Например введите Litai. **Примечание**: **родительный падеж единственное число** и **род.** поля доступны в зависимости от языка, выбранного в **язык** поле. |
    | **Единственное число nominative поле (имя части группа полей)** | Введите в единственном подгруппа валюты.                                                                                                                                                            |
    | **Именительный падеж множественное число (имя части группа полей)**         | Введите в форму множественного числа подгруппа валюты.                                                                                                                                                              |
    | **Сокращенное наименование единиц (короткое имя группа полей)**       | Введите код для идентификации валюты ISO. Например, введите LTL для определения литовских литов.                                                                                                                             |
    | **Сокращенное наименование частей (короткое имя группа полей)**      | Введите валюту номинала валюты подгруппу. Например введите центов.                                                                                                                                         |
    | **Conjunction 'and' between units and parts**             | Выберите для печати Союза «и» между валютами и частями единицы измерения. Например в счетах и отчетах, 100.20 ЛИТОВ суммы будет отображаться как 100 литов и 20 центов.                      |

5.  Click **Save**.
