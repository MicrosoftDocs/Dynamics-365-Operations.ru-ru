# <a name="tax-directives-for-europe"></a>Налоговые директивы для Европы
Директивы ЕС содержат требования, которым должны следовать все государства-участники относительно правил накладных и сведений, которые должны быть включены при создании накладных. Директивы ЕС содержит сведения, которые должны требовать поставщики от своих клиентов в ЕС. Например, накладная может включать ссылку, которая описывает юридические причины нулевого НДС или налогового освобождения. Если накладная изменяет ранее выпущенную накладную, например кредит-ноту, должна быть создана однозначная ссылка на исходную накладную.

- Для печати описания налоговых кодов на накладной по продаже, накладной с произвольным текстом или отчета по накладным проекта, настройте текст ссылки на налоговую директиву по налоговым кодам.
- Для кредит-нот номер и дата исходной накладной печатаются вместе с причиной исправления.
