# <a name="petty-cash-for-eastern-europe"></a>Наличные деньги для Восточной Европы

В этом разделе содержатся сведения о функциях наличные деньги, которое позволяет пользователям в Эстонии, Литве, Чехии, Венгрии, Латвии, Польше и России отражают кассовых операций в системе.

Для автоматизации операций приходов и расходов при оплате наличными, создание первичных документов и печать связанных отчетов можно использовать функцию наличные деньги. Наличные деньги функций позволяет выполнять следующие операции:

-   Счет приход и расход доступного наличного имущества.
-   Создание форм, типичные при оплате наличными: ордеры, расходные кассовые ордера и журнал регистрации кассовых ордеров по оплате.
-   Управлять максимальную сумму наличных денег, которые необходимы для операций с клиентами, поставщиками и т. д.
-   Отразить кассовых операций в различных валютах в системе.
-   Преобразуйте суммы операций с наличными в иностранной валюте в валюту по умолчанию для предоставления бухгалтерской отчетности.
-   Создание **кассовой книги** отчет и Отчет кассира.

## <a name="prerequisites"></a>Необходимые условия
Прежде чем использовать функцию наличные деньги, необходимо выполнить следующие предварительные условия:

-   Настройка кассовых счетов.
-   Настройка профилей разноски кассы.
-   Настройка номерных серий для нумерации документа при оплате наличными.
-   Настройка значений по умолчанию для кассы параметров управления банком и.
-   Настройка имен журналов денежных средств в главной книге.

### <a name="set-up-cash-accounts"></a>Настройка счетов денежных средств

| Поле                 | описание                                                                                                          |
|-----------------------|----------------------------------------------------------------------------------------------------------------------|
| Наличный расчет                  | Введите код, определяющий кассовый счет (наличные).                                                                    |
| Наименование                  | Введите описание кассового счета.                                                                             |
| Валютное              | Выбор кода валюты по умолчанию для проводок по кассе.                                                              |
| Группа номерных серий | Если Нумерация кассовых документов должны отличаться от нумерации, который указан в параметрах, введите код. |

| Отрицательная касса         | Установите этот флажок, чтобы разрешить отрицательных денежных сальдо в любой валюте.                                               |

## <a name="cash-transaction-approval-and-posting"></a>Утверждение проводки при оплате наличными и разноски
Для проводок по кассе, могут быть применены следующие статусы: **нет**, **подтверждено**, **утвержденных**, и **Отклонено**. A **использовать Подтверждение** параметра в **утверждения** экспресс- **денежных средств** tab в **управления банком и кассовыми**&gt;**установки**&gt;**параметров управления банком и кассовыми** позволяет активировать два дополнительных статуса: **подтверждено** и **Отклонено**. Подтверждение подходит, когда выдаются кассовых документов и денежных приходов или расходов являются общими для двух сотрудников: бухгалтер и удобства работы кассиров. **Сброс статуса** функция меняет текущий статус проводки. **Утвержденные** становится **подтверждено**, и **подтверждено** становится **нет**. Записи в журнале при оплате наличными могут редактироваться только в том случае, если статус- **нет**. Проводки денежных средств может быть отклонен только в том случае, если статус проводки имеет значение **подтверждено**. Отклонено документов, включаемых в денежных **журнала регистрации кассовых документов** отчета, но они не влияет на значения на **кассовой книги** отчета. Чтобы проверить транзакцию, выберите соответствующую строку журнала для отборочной накладной и нажмите кнопку **документы утверждения**&gt;**Подтверждение**. Номер заказа создается на основе указанной номерной серии. Статус проводки изменяется на **подтверждено**, и больше не можно изменить строки журнала. Баланс счета денежных средств не меняется. Чтобы отклонить документ при оплате наличными, щелкните **документы утверждения**&gt;**отклонение**. Этот параметр доступен только для документов, которые имеют **подтверждено** состояния. Чтобы утвердить проводку, выберите соответствующую строку журнала накладной и нажмите кнопку **документы утверждения**&gt;**утвердить**. **Утвержденных** статус указывает, что получены или увеличиваются денежных средств. Сальдо при оплате наличными изменяется. Можно разнести проводки по кассе. Чтобы отменить **утвержденных** статус и сброс статуса для **нет**, щелкните **документы утверждения**&gt;**сброс статуса**. Только утвержденные денежные средства, которые могут быть разнесены. Для учета журнала щелкните **Разноска**&gt;**Разноска**.

## <a name="print-a-cash-order"></a>Печать кассового ордера
Чтобы напечатать заказ на денежных средств, выберите строку накладной и в области действий нажмите кнопку **печать**&gt;**отчет заказа при оплате наличными**. Система генерирует форму печати кассового ордера или расходного кассового ордера, в зависимости от того, является ли сумма вводится в **Дебет** поле или **кредит** для выбранной строки:

-   Если есть сумму в **Дебет** поля: по оплате ПКО
-   Если есть сумму в **кредит** поля: РКО по оплате

Отборочной накладной для строки журнала, которые имеют **подтверждено**, **утвержденных**, или **Отклонено** состояния могут быть напечатаны. Можно также печатать документы заказа при оплате наличными в **управления банками и кассами**&gt;**Inquires и отчетов**&gt;**по оплате заказа**.
