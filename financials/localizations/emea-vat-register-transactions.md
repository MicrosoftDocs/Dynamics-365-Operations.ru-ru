# <a name="vat-register-transactions-for-eastern-europe"></a>Регистр НДС по проводок для Восточной Европы

В этом разделе представлена информация о транзакции в регистре НДС для Эстонии, Литве, Чехии, Венгрии, Латвии и Польше. 

Для юридических лиц с основным адресом в Эстонии, Литве, Чехии, Венгрии, Латвии и Польше, можно проверить налоговых проводок и обновить **дату зачета НДС** для них. Для просмотра проводок регистрации НДС, откройте **налог**&gt;**периодические задачи**&gt;**проводок регистр НДС по**. На этой странице отображаются налоговых проводок, которые были созданы с налоговыми кодами типа **стандартный НДС** для которого **дату зачета НДС** пуст. Можно заполнять **дату зачета НДС** для налоговых проводок с помощью **дату зачета НДС** кнопку на **главное меню**. Для юридических лиц с основным адресом в Чехии существует дополнительный столбец, **отложена НДС**. Если выбрано это поле указывает, налог на добавленную стоимость не было зарегистрировано для проводки. Обычно это относится к кредит-нотами, которые ожидают подтверждение от клиента. Этот флажок установлен, можно выбрать во время процесса разноски накладной по продаже. На **налоговые коды** страница (**налог**&gt;**налог с продаж**&gt;**налоговые коды**), можно просмотреть и обновить тип кода налога с продаж. <!---For more information about sales tax codes, refer to [Set up sales tax codes](http://ax.help.dynamics.com/en/wiki/set-up-sales-tax-codes/).-->
