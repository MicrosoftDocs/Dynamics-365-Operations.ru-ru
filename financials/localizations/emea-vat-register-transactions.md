# <a name="vat-register-transactions-for-eastern-europe"></a>Проводки в регистре НДС для Восточной Европы

[!include[banner](../includes/banner.md)]


В этом разделе представлена информация о проводках в регистре НДС для Эстонии, Литвы, Чехии, Венгрии, Латвии и Польши. 

Для юридических лиц с основным адресом в Эстонии, Литве, Чехии, Венгрии, Латвии и Польше можно просмотреть налоговые проводки и обновить параметр **Дата зачета НДС** для них. Для просмотра проводок регистрации НДС откройте **Налог** &gt; **Периодические задачи** &gt; **Проводки в регистре НДС**. На этой странице отображаются налоговые проводки, которые были созданы с налоговыми кодами типа **Стандартный НДС** и для которых поле **Дата зачета НДС** пусто. Можно заполнять поле **Дата зачета НДС** для налоговых проводок с помощью кнопки **Дата зачета НДС** в пункте **Главное меню**. Для юридических лиц с основным адресом в Чехии существует дополнительный столбец, **Отложенный НДС**. Если это поле выбрано, это означает, что налог на добавленную стоимость не был зарегистрирован для проводки. Обычно это относится к кредит-нотами по продажам, которые ожидают подтверждения от клиента. Можно установить этот флажок во время процесса разности накладной по продаже. На странице **Налоговые коды** (**Налог** &gt; **Налог** &gt; **Налоговые коды**) можно просмотреть или обновить тип кода налога с продаж. <!---For more information about sales tax codes, refer to [Set up sales tax codes](http://ax.help.dynamics.com/en/wiki/set-up-sales-tax-codes/).-->
