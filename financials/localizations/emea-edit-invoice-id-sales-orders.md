# <a name="edit-invoice-ids-on-sales-orders-for-eastern-europe"></a>Редактирование кодов накладных в заказах на продажу для Восточной Европы

[!include[banner](../includes/banner.md)]


В этом разделе представлена информация о параметрах модуля расчетов с клиентами, которые влияют на номера накладных на отборочных накладных и накладных для Чехии, Эстонии, Венгрии, Литвы, Латвии и Польши.

Для юридических лиц в Чехии, Эстонии, Венгрии, Литве, Латвии и Польше параметр **Редактирование номера накладной на заказах на продажу** позволяет вручную вводить номера отборочной накладной и накладной для отборочных накладных и накладных клиентов. Следующие параметры доступны на странице **Параметры модуля расчетов с клиентами** на экспресс-вкладке **Накладная** на вкладке **Обновления**.

<li>Изменять поле <strong>Накладная</strong> на странице <strong>Разноска накладной</strong>.</li>
<li>Выберите параметр <strong>Проверка использованного номера накладной</strong>.</li>
</ul></td>
</tr>
<tr class="odd">
