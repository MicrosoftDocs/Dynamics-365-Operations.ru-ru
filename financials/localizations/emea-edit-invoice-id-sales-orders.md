# <a name="edit-invoice-ids-on-sales-orders-for-eastern-europe"></a>Редактировать накладную идентификаторы заказов на продажу для Восточной Европы

В этом разделе представлена информация о параметры расчетов с клиентами, которые влияют на процесс номеров накладных на накладных и отборочных накладных для Чехии, Эстонии, Венгрии, Литве, Латвии и Польше.

Для компаний в Чехии, Эстонии, Венгрии, Литве, Латвии и Польше **редактирование номера накладной на заказах на продажу** позволяет вручную ввести отборочной накладной и номера накладной для клиента, в котором накладных и отборочных накладных. Следующие параметры доступны на **параметры расчетов с клиентами** на странице **накладной** экспресс-в **обновления** вкладке.

<td> <strong>Description</strong></td>
</tr>
<tr class="even">
<td>Редактирование номера накладной на заказах</td>
<td>Если выбран этот параметр, можно сделать следующее:
<ul>
<li>Изменение <strong>отборочной накладной</strong> на <strong>разноски отборочной накладной</strong> страницы.</li>
<li>Изменение <strong>накладной</strong> на <strong>Разноска накладной</strong> страницы.</li>
<li>Выберите <strong>Проверка использованного номера накладной</strong> параметр.</li>
</ul></td>
</tr>
<tr class="odd">
<td>Проверка использованного номера накладной</td>
<td>Если этот параметр выбран, во время разноски, накладных системы убедитесь, что номер счета не существует для учтенных счетов. Если число отсутствует, отображается сообщение об ошибке.</td>
