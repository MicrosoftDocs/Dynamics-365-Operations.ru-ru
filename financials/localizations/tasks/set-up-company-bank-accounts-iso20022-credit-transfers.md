# <a name="set-up-company-bank-accounts-for-iso20022-credit-transfers"></a>Настройка банковских счетов компании для кредитных переводов ISO20022

[!include[task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано, как настроить сведения о банковском счете компании, которые необходимы для создания файла платежа. Вы настраиваете информацию, необходимую для создания формата перемещения кредита ISO 20022, но в зависимости от выбранного формата также может требоваться другая информация, такая как код компании или код сортировки. 

В качестве компании с демонстрационными данными для создания этой процедуры используется DEMF.

Это вторая процедура из пяти, которые иллюстрируют процесс платежа поставщикам с помощью конфигурации электронной отчетности. Эта процедура предназначена для функции, которая была добавлена в версии 1611 Dynamics 365 for Operations.


## <a name="set-up-iban-and-swift-code"></a>Настройка номера IBAN и SWIFT-кода
1. Перейдите в раздел "Управление банком и кассовыми операциями" > "Банковские счета".
2. Используйте экспресс-фильтр для фильтрации поля "Банковский счет" по значению 'DEMF OPER".
3. Щелкните "DEMF OPER", чтобы открыть сведения о банковском счете.
4. Щелкните "Изменить".
5. Разверните раздел "Дополнительная идентификация".
6. В поле "Номер IBAN" введите ''DE89370400440532013000".
7. В поле "SWIFT-код" введите "DEUTDEFF".
    * Обратите внимание, что SWIFT\BIC не требуется для многих форматов платежей, однако рекомендуется, чтобы этот код был зарегистрирован для банковского счета.  
8. Нажмите кнопку "Сохранить".

## <a name="set-up-bank-account-for-the-legal-entity"></a>Настройка банковского счета для юридического лица
1. Перейдите в раздел "Управление организацией" > "Организации" > "Юридические лица".
2. Щелкните "Изменить".
3. Разверните раздел "Сведения о банковском счете".
4. В поле "Банковский счет" введите или выберите значение.
5. Нажмите кнопку "Сохранить".
