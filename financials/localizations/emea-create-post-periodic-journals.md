# <a name="split-periods-in-periodic-journals"></a>Разделение периодов в периодических журналах

[!include[banner](../includes/banner.md)]


Периодические журналы иногда называются повторяющимися журналами, потому что сумма, текст и другие сведения повторяются при каждой разноске журнала. При создании журнала следует указать периода повторения, например в днях или месяцах. Также можно определить число периодов разноски журнала.

Чтобы многократно извлекать и разносить строки проводок, можно использовать страницу **Периодические журналы**. Для юридических лиц в Чехии, Эстонии, Венгрии, Латвии, Литве, Польше и России страница **Периодические журналы** расширяется за счет разбиения для функции периодов. <!---For more information, see [Create and process a periodic journal](http://ax.help.dynamics.com/en/wiki/create-and-process-a-periodic-journal/).-->

### <a name="example-split-for-periods-in-periodic-journals"></a>Пример: разбиение для периодов в периодических журналах

Страховая компания предлагает вашей организации скидку при авансовой оплате страхового полиса за целый год. Оплата разносится на счет активов, например как предоплаченная страховка. После этого вы амортизируете ежемесячный расход на страховку в течение года, создавая периодический журнал, содержащий кредит на счете предоплаченной страховки и дебет на счете расходов на страхование. В таком случае можно использовать функцию распределения по периодам. Нажмите кнопку **Распределить по периодам** на панели действий на странице **Строки периодического** **журнала**, затем укажите следующие поля.

|                       |                                                                                                                                                                                                             |
|-----------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Поле**             | **Описание**                                                                                                                                                                                             |
| **Начальная дата**        | Выберите дату первой строки периодического журнала.                                                                                                                                                        |
| **Число периодов** | Введите количество периодов, между которыми должна быть разделена строка журнала. Это значение определяет, сколько новых проводок создается. Сумма проводки распределяется равномерно между новыми проводками. |
| **Единица измерения**              | Выберите единицу измерения для периода.                                                                                                                                                                  |
| **Интервал периода**   | Определите интервал между периодами разноски.                                                                                                                                                              |

Например, чтобы создать ежеквартальную разноску, введите **4** в поле **Число периодов**, выберите **Месяцы** в поле **Единица измерения** и введите **3** в поле **Интервал периода**. Система создает четыре строки журнала, каждую на четверть введенной суммы строки журнала с интервалом в 3 месяца. Аналогичная функция доступна также для общего журнала. При просмотре строк общего журнала выберите **Периодический журнал** &gt; **Сохранить журнал**.
