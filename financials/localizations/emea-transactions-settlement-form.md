# <a name="view-transactions-on-settlement-for-eastern-europe"></a>Просмотр проводок по сопоставлению для Восточной Европы

[!include[banner](../includes/banner.md)]


В этом разделе приводятся сведения о странице "Проводки по сопоставлению" для клиентов и поставщиков.

Используйте страницу **Проводки по сопоставлению**, чтобы просмотреть сведения о сложных проводках по сопоставлению для клиента или поставщика. Эта функция доступна только для юридических лиц с основным адресом в Литве, Латвии, Эстонии, Чехии, Венгрии или Польше. Страницу **Проводки по сопоставлению** можно найти в следующих местах:

-   **Расчеты с поставщиками** &gt; **Поставщики** &gt; **Все поставщики**. В области действий на вкладке **Поставщик** щелкните **Проводки** &gt; **Проводки**. На странице **Проводки по поставщику** выберите проводку, затем щелкните **Проводки по сопоставлению**.
-   **Расчеты с клиентами** &gt; **Клиенты** &gt; **Все клиенты**. В области действий на вкладке **Клиент** щелкните **Проводки**. На странице **Проводки по клиенту** выберите проводку, затем щелкните **Проводки по сопоставлению**.

Информация, относящаяся к сопоставлениям, фиксируется и может отображаться на странице **Проводки по сопоставлению** для проводок, которые были созданы в следующих сценариях:

-   **Курсовая разница** — сопоставление накладной и платежа создает реализационную или внереализационную разницу валютного курса.
-   **Изменение профиля разноски** — сопоставляются две записи, например накладная и кредит-нота, которые имеют разные профили разноски.
-   Предоплата преобразуется в платеж или платеж преобразуется в предоплату.
-   **Скидки по оплате** — накладная сопоставляется с платежом, из которого была вычтена сумма скидки по оплате.
-   **Допустимое расхождение** — накладная сопоставлена с платежом, сумма которого слегка отличается от суммы накладной.
-   **Разноска условного налога** — накладная сопоставляется с платежом, к которому применен условный налог.
-   **Сопоставления между компаниями** — внутрихолдинговая функция, используемая для сопоставления накладной с платежом от организации.
