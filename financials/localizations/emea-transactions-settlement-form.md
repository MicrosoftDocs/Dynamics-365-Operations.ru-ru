# <a name="view-transactions-on-settlement-for-eastern-europe"></a>Просмотр проводок в сопоставления для Восточной Европы

В этом разделе сведения о транзакциях на странице сопоставления для клиентов и поставщиков.

Используйте **проводки по сопоставлению** страницу, чтобы просмотреть сведения о проводках сложные сопоставления для клиента или поставщика. Эта возможность доступна только для юридических лиц, основной адреса в Литве, Латвии, Эстонии, Чехии, Венгрии или Польше. Можно найти **проводки по сопоставлению** страницы по следующим ссылкам:

-   **Расчеты с поставщиками**&gt;**поставщики**&gt;**все поставщики**. В области действий на **поставщика**, щелкните **проводок**&gt;**операции**. На **проводок поставщика** странице, выберите проводку и нажмите кнопку **проводки по сопоставлению**.
-   **Расчеты с клиентами**&gt;**клиентов**&gt;**всех клиентов**. В области действий на **клиента**, щелкните **операции**. На **проводок клиента** странице, выберите проводку и нажмите кнопку **проводки по сопоставлению**.

Информация, относящаяся к сопоставления фиксируется и может быть отображен на **проводки по сопоставлению** страница для проводок, которые были созданы в следующих сценариях:

-   **Курсовая разница** – сопоставление накладной и платежа создает реализованных или нереализованных Курсовая разница.
-   **Изменение профиля разноски** — сопоставляются две записи, например, счет и кредит-нота, которые используются разные профили разноски.
-   Предоплата преобразовывается в оплату или преобразуются платеж в предоплату.
-   **Скидка при оплате наличными** – накладная сопоставлена и вычитания суммы скидки из платежа.
-   **Допустимое расхождение** – накладная сопоставлена с платежом, имеющего суммы, слегка отличающиеся от накладной.
-   **Разноски условного налога** – накладная сопоставлена с платежом, применения условного налога.
-   **Сопоставления между компаниями** — внутрихолдинговые функции, используемый для сопоставления накладной с платежом в организации.
