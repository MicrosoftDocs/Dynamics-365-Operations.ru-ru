# <a name="set-up-signers-for-print-forms"></a>Настройка подписывающие лица для печати форм

Для компаний в Чехии, Эстонии, Венгрии, Литве, Латвии, Польше и России можно настроить подписывающие лица и должности для клиентов и поставщиков, печатаемых документов, таких как заказы на накладных и наличных денег.

<a name="set-up-default-values"></a>Настройка значений по умолчанию
---------------------

Чтобы настроить подписывающие лица для документов, которые компания распечатывает, используйте **должностных лиц** страницы. Как для компании, так и для клиентов или поставщиков, в зависимости от типа документа, можно настроить подписывающие лица и их заголовков. ## <a name="set-up-a-number-sequence-code-for-officials"></a>Настроить код номерной серии для должностных лиц
Можно назначить код номерной серии для должностных лиц в **номерные серии** раздел **компаниями** страницы. Выберите код номерной серии для **код сессии должностных лиц** ссылки.

## <a name="modify-signers-in-primary-documents"></a>Изменение подписывающие лица в первичных документов
Функции работы с должностных лиц указывает подписывающие лица заранее определенные по умолчанию из таблицы должностные лица. На **Разноска накладной** на странице **должностных лиц** вкладке, можно изменить имя подписавшего лица и заголовка в основной документ для следующих типов документов:

-   Накладная клиента
-   Накладная поставщика
-   Отгрузить заказ на перемещение
-   Кассовый ордер

**Примечание:** после разноски документа должностных лиц, не может быть изменено.
