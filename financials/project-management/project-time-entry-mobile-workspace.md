# <a name="project-time-entry-mobile-workspace"></a>Мобильная рабочая область регистрации времени по проектам

[!include[banner](../includes/banner.md)]



В этом разделе проводятся сведения о мобильной рабочей области регистрации времени по проекту для Dynamics 365 for Operations. Эта рабочая область позволяет пользователям вводить и сохранять время по проекту, используя мобильное устройство.

<a name="overview-of-the-project-time-entry-mobile-workspace"></a>Общие сведения о мобильной рабочей области регистрации времени по проекту
---------------------------------------------------

В рамках своей повседневной работы ресурсы проекта часто находятся на площадке или в командировке. Мобильная рабочая область **Регистрация времени по проекту** позволяет пользователям вводить оплачиваемое или неоплачиваемое время по проекту на мобильном устройстве по своему выбору. Таким образом, ресурсы проекта могут регистрировать записи времени в любое время и в любом месте. Они также могут просмотреть записи времени, которые уже были зарегистрированы. 

В частности, мобильная рабочая область **Регистрация времени по проекту** предоставляет следующие возможности:

-   Для любой выбранной даты введите число часов, затраченных на конкретную задачу.
-   Поиск по имени проекта или клиенту, чтобы найти проект для ввода времени.
-   Указание категория и мероприятия для затраченного времени.
-   Запись времени как оплачиваемого или неоплачиваемого по проекту.
-   При необходимости ввод любых внешних или внутренних комментариев.

Для реализации мобильной рабочей области **Регистрация времени по проекту**, см. ниже в этом разделе.

## <a name="download-and-install-the-dynamics-365-for-operations-mobile-app"></a>Загрузка и установка мобильного приложения Dynamics 365 for Operations
Загрузите и установите мобильное приложение Dynamics 365 for Operations из магазина мобильных приложений.

-   Для Android: [Dynamics 365 for Operations в магазине Google Play](https://play.google.com/store/apps/details?id=com.microsoft.dynamics365.operations.mobile)
-   Для iPhone: [Dynamics 365 for Operations в магазине приложений iTunes](https://itunes.apple.com/us/app/dynamics-365-for-operations/id1180836730?mt=8)

## <a name="sign-in-to-the-dynamics-365-for-operations-mobile-app"></a>Вход в мобильное приложение Dynamics 365 for Operations
1.  Запустите приложение на мобильном устройстве.
2.  Введите свой URL-адрес Dynamics 365 for Operations.
3.  Введите компанию для входа в систему. Например, введите **USMF**.
4.  При первом входе появится запрос ввести имя пользователя и пароль для учетной записи Dynamics 365 for Operations. Введите свои учетные данные.
5.  После входа вы увидите доступные рабочие области для вашей компании. Обратите внимание, что если позже системный администратор опубликует новую рабочую область, вы может потянуть список, чтобы обновить список мобильных рабочих областей.

[![Потянуть для обновления](./media/pull-to-refresh-list-of-workspaces-183x300.png)](./media/pull-to-refresh-list-of-workspaces.png)

## <a name="enter-time-by-using-the-project-time-entry-mobile-workspace"></a>Ввод времени с помощью мобильной рабочей области регистрации времени по проекту
1.  На мобильном устройстве выберите рабочую область **Регистрация времени по проекту**.
2.  Выберите **Регистрация времени**. Отображаются календарные даты для текущей недели.
3.  Для выбранной даты выберите **Действия** &gt; **Создать запись**.
4.  Введите число часов, которое требуется зарегистрировать.
5.  Выберите проект для регистрации времени. Отображается список проектов, который загружены в ваше приложение для работы в автономном режиме. По умолчанию загружаются 50 номенклатур, но разработчик может изменить это число. Для получения дополнительных сведений разработчики должны ознакомиться с разделом [Мобильная платформа Dynamics 365 for Operations](/dynamics365/operations/dev-itpro/mobile-apps/mobile-platform).
6.  Если проект отсутствует в списке, выберите **Поиск** для выполнения поиска по Интернету в Dynamics 365 for Operations. Выполните поиск по имени или переключитесь на поиск по имени проекта или клиенту.
7.  Выбирается категория. Отображается список категорий, который загружены в ваше приложение для работы в автономном режиме. По умолчанию загружаются 50 номенклатур, но разработчик может изменить это число. Для получения дополнительных сведений разработчики должны ознакомиться с разделом [Мобильная платформа Dynamics 365 for Operations](/dynamics365/operations/dev-itpro/mobile-apps/mobile-platform).
8.  Если категория отсутствует в списке, выберите **Поиск** для выполнения поиска по Интернету в Dynamics 365 for Operations. Выполните поиск по категории или переключитесь на поиск по названию категории.
9.  Выбор мероприятия. Отображается список мероприятий, который загружены в ваше приложение для работы в автономном режиме. По умолчанию загружаются 50 номенклатур, но разработчик может изменить это число. Для получения дополнительных сведений разработчики должны ознакомиться с разделом [Мобильная платформа Dynamics 365 for Operations](/dynamics365/operations/dev-itpro/mobile-apps/mobile-platform).
10. Если мероприятие отсутствует в списке, выберите **Поиск** для выполнения поиска по Интернету в Dynamics 365 for Operations. Выполните поиск по номеру мероприятия или переключитесь на поиск по назначению.
11. Выбор свойство строки.
12. Необязательно: введите любые внешние и внутренние комментарии.
13. Выберите **Готово**.
