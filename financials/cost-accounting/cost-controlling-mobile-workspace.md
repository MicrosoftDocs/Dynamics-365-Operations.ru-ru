# <a name="cost-controlling-mobile-workspace"></a>Мобильная рабочая область управления затратами

[!include[banner](../includes/banner.md)]


В этом разделе проводятся сведения о мобильной рабочей области управления затратами, которая доступна для мобильного приложения Microsoft Dynamics 365 for Operations. Эта рабочая область позволяет менеджерам центра затрат просматривать сведения о производительности центра затрат в любое время и в любом месте. 

<a name="overview-of-the-cost-controlling-mobile-workspace"></a>Общие сведения о мобильной рабочей области управления затратами
-------------------------------------------------

Мобильная рабочая область **Управление затратами** обеспечивает мгновенное отображение текущей производительности центров затрат путем сравнения фактических затрат с бюджетными затратами. Можно выполнить детализацию до статуса отдельных элементов затрат. Например, сотрудник получает приглашение на международную конференцию, но организация должна оплатить все командировочные расходы. Сотрудник запрашивает у своего менеджера разрешение на посещение этой конференции. Менеджер открывает мобильную рабочую область **Управление затратами** на своем мобильном устройстве и смотрит, имеет ли она бюджет для оплаты посещения конференции сотрудником.

### <a name="data-security"></a>Безопасность данных

Данные в мобильной рабочей области **Управление затратами** защищены учетными данными пользователя. Менеджеры центра затрат могут просматривать сведения только для своего собственного центра затрат. Управление безопасностью уровня доступа осуществляется в модуле **Учет затрат**. Бухгалтеры по учету затрат определяют конфигурацию мобильной рабочей области **Управление затратами** в модуле **Учет затрат**. После публикации рабочей области в мобильном приложении Microsoft Dynamics 365 for Operations она становится доступна в приложении. Таким образом, все менеджеры центров затрат в рамках организации могут просматривать данные в одинаковом формате.

### <a name="actions-views-and-links"></a>Действия, представления и ссылки

Мобильная рабочая область **Управление затратами** для приложения Dynamics 365 for Operations содержит следующие действия, представления и ссылки:

-   **Действия:**
    -   Используйте **Выбрать конфигурацию** для выбора макета.
    -   Используйте **Выбрать объект затрат** для выбора центров затрат, по которым требуется фильтровать данные. **Примечание:** Центры затрат, которые отображаются в списке, зависят от уровня доступа, предоставленного в модуле **Учет затрат**.
-   **Представления:** На основании выбранных действий и конфигурации в модуле **Учет затрат** можно просмотреть следующую информацию на карточках.
    -   Факт/бюджет (текущий период)
    -   Факт/пересмотренный бюджет (текущий период)
    -   Факт/бюджет (предыдущий период)
    -   Факт/пересмотренный бюджет (предыдущий период)
    -   Факт/бюджет (с начала года)
    -   Факс/пересмотренный бюджет (с начала года)

    В каждой карточке показаны следующие суммы: фактическая, бюджет, отклонение и % отклонения.
-   **Ссылки:**
    -   Сведения для текущего периода
    -   Сведения для предыдущего периода
    -   Сведения с начала года

    При выборе ссылки отображается карта для каждого элемента затрат. На каждой карточке отображаются следующее суммы: фактическая, бюджетная, отклонение от бюджета, отклонение от бюджета в процентах, пересмотренный бюджет, отклонение от пересмотренного бюджета и отклонение от пересмотренного бюджета в процентах.

## <a name="download-and-install-the-dynamics-365-for-operations-mobile-app"></a>Загрузка и установка мобильного приложения Dynamics 365 for Operations
Загрузите и установите мобильное приложение Dynamics 365 for Operations из магазина мобильных приложений.

-   Для Android: [Dynamics 365 for Operations в магазине Google Play](https://play.google.com/store/apps/details?id=com.microsoft.dynamics365.operations.mobile)
-   Для iPhone: [Dynamics 365 for Operations в магазине приложений iTunes](https://itunes.apple.com/us/app/dynamics-365-for-operations/id1180836730?mt=8)

## <a name="sign-in-to-the-dynamics-365-for-operations-mobile-app"></a>Вход в мобильное приложение Dynamics 365 for Operations
1.  Запустите приложение на мобильном устройстве.
2.  Введите свой URL-адрес Dynamics 365 for Operations.
3.  Введите компанию для входа в систему. Например, введите **USMF**.
4.  При первом входе появится запрос ввести имя пользователя и пароль для учетной записи Dynamics 365 for Operations. Введите свои учетные данные.
5.  После входа вы увидите доступные рабочие области для вашей компании. Обратите внимание, что если позже системный администратор опубликует новую рабочую область, вы может потянуть список, чтобы обновить список мобильных рабочих областей. 

    [![Потянуть для обновления](./media/pull-to-refresh-list-of-workspaces-183x300.png)](./media/pull-to-refresh-list-of-workspaces.png)

## <a name="view-the-performance-of-your-cost-center-by-using-the-cost-controlling-mobile-workspace"></a>Просмотр производительности центра затрат с помощью мобильной рабочей области управления затратами
1.  На мобильном устройстве выберите рабочую область **Управление затратами**.
2.  Выберите **Управление объектами затрат**.
3.  Выберите **Действия**.
4.  Выберите **Выбрать конфигурацию**, чтобы выбрать макет управления затратами.
5.  Выберите **Готово**.
6.  Выберите **Действия**.
7.  Выберите **Выбрать объект затрат** для выбора центров затрат, к которым вам предоставлен доступ.
8.  Выберите **Готово**.
9.  Просмотрите общую производительность вашего центра затрат.
10. Выберите ссылку **Сведений для текущего периода**.
11. Просмотрите производительность конкретного центра затрат.
12. Можно также выполнять поиск конкретных элементов затрат.
