# <a name="cost-controlling-mobile-workspace-for-microsoft-dynamics-365-for-operations-app"></a>Мобильная рабочая область управления затратами для приложения Microsoft Dynamics 365 for Operations

С помощью мобильной рабочей области управления затратами менеджеры центра затрат могут в любое время и в любом месте просматривать производительность центра затрат.

| Необходимые условия                                                         | описание                                                                                                                                                                   |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Сведения о мобильной платформе Microsoft Dynamics 365 for Operations | [Мобильная платформа Dynamics 365 for Operations](/dynamics365/operations/dev-itpro/mobile-apps/mobile-platform)                                                              |
| Dynamics 365 for Operations                                          | Убедитесь, что используется среда с установленными Microsoft Dynamics 365 for Operations версии 1611 и обновлением 3 платформы Microsoft Dynamics for Operations (ноябрь 2016). |

| Мобильное устройство, на которое установлено приложение Dynamics 365 for Operations | Загрузите приложение Dynamics 365 for Operations из магазина мобильных приложений.                                                                                                      |

## <a name="introduction"></a>Введение
Мобильная рабочая область управления затратами обеспечивает мгновенное отображение текущей производительности центров затрат путем сравнения фактических затрат с бюджетными затратами. Можно выполнить детализацию до статуса отдельных элементов затрат.

### <a name="example"></a>Пример

Сотрудник получает приглашение на международную конференцию. Организация должна покрыть все командировочные расходы. Сотрудник запрашивает у своего менеджера разрешение на посещение этой конференции. Менеджер быстро открывает мобильное рабочее пространство управления затратами на своем мобильном телефоне и смотрит, имеет ли он бюджет для плата посещения конференции сотрудником.

### <a name="data-security"></a>Безопасность данных

Данные в рабочей области управления затратами защищены учетными данными пользователя. Менеджер центра затрат может просматривать сведения только для своего собственного центра затрат. Управление безопасностью уровня доступа осуществляется в модуле "Учет затрат". Бухгалтеры по учету затрат определяют конфигурацию мобильной рабочей области управления затратами в модуле "Учет затрат". После публикации рабочей области в приложении Microsoft Dynamics 365 for Operations она становится доступна в мобильном приложении Dynamics 365 for Operations. Это гарантирует, что все менеджеры центров затрат в рамках организации просматривают данные в одинаковом формате.

### <a name="actions-views-and-links"></a>Действия, представления и ссылки

Мобильная рабочая область управления затратами для приложения Dynamics 365 for Operations содержит следующие действия, представления и ссылки:

-   Действия 
    -   Выберите **Конфигурации**, чтобы выбрать макет.
    -   Выберите **Объекты затрат** для выбора центров затрат, для которых необходимо фильтровать данные. **Примечание.** Список отображается в соответствии с доступом,, предоставляемым в модуле "Учет затрат".

<!-- -->

-   На основании того, что выбрано в поле **Действия** и настроено в модуле "Учет затрат", можно просмотреть следующую информацию в карточках. Обратите внимание, что эта сумма отображается в том же формате: фактическая, бюджетная, отклонение и процент отклонения. 
    -   Факт/бюджет (текущий период)
    -   Факт/пересмотренный бюджет (текущий период)
    -   Факт/бюджет (предыдущий период)
    -   Факт/пересмотренный бюджет (предыдущий период)
    -   Факт/бюджет (с начала года)
    -   Факс/пересмотренный бюджет (с начала года)

<!-- -->

-   Ссылки
    -   Сведения для текущего периода.
    -   Сведения для предыдущего периода.
    -   Сведения с начала года.

При выборе одной из ссылок отображается карта для каждого элемента затрат. Сумма в карточках отображается в следующем формате: фактическая, бюджетная, отклонение от бюджета, отклонение от бюджета в процентах, пересмотренный бюджет, отклонение от пересмотренного бюджета и отклонение от пересмотренного бюджета в процентах. ## <a name="get-started"></a>Начать
Выполните следующие действия, чтобы приступить к работе с помощью мобильного приложения управления затратами на мобильном устройстве.

1.  Загрузите приложение Microsoft Dynamics 365 for Operations из магазина мобильных приложений и установите его.
2.  Запустите приложение на устройстве.
3.  Введите URL-адрес Dynamics 365.
4.  Введите компанию для входа в систему. Например, введите **USMF**.
5.  При первом входе появится запрос ввести имя пользователя и пароль для учетной записи Microsoft Dynamics 365 for Operations. Введите свои учетные данные. После входа вы увидите доступные рабочие области для вашей компании.

Чтобы просмотреть рабочие области в мобильном приложении, необходимо сначала опубликовать нужные рабочие области для приложения Dynamics 365 for Operations.

1.  Запустите Dynamics 365 for Operations.
2.  Перейдите в раздел **Администрирование системы** &gt; **Настройка** &gt; **параметры системы**.
3.  Выберите **Управление мобильным приложением**.
4.  Выберите рабочую область **Управление затратами**, чтобы опубликовать ее для мобильной платформы.
5.  Выберите **Публикация рабочей области**.
6.  Обновите устройство для просмотра опубликованных рабочих областей.

## <a name="view-the-performance-of-your-cost-center"></a>Просмотр производительности вашего центра затрат
1.  На мобильном устройстве выберите рабочую область **Управление затратами**.
2.  Выберите **Управление объектами затрат**.
3.  Щелкните **Действия**.
4.  Щелкните **Выбрать конфигурацию**, чтобы выбрать макет управления затратами.
5.  Щелкните **Готово**.
6.  Щелкните **Действия**.
7.  Щелкните **Выбрать объект затрат** для выбора центров затрат, к которым вам предоставлен доступ.
8.  Щелкните **Готово**.
9.  Просмотрите общую производительность вашего центра затрат.
10. Щелкните **Сведения для текущего периода**.
11. Просмотрите производительность конкретного центра затрат.
12. Можно также выполнять поиск конкретных элементов затрат.
