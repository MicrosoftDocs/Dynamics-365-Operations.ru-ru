<span data-ttu-id="fb8fe-105">**Учет затрат**</span><span class="sxs-lookup"><span data-stu-id="fb8fe-105">**Cost accounting**</span></span>

<span data-ttu-id="fb8fe-106">Учет затрат позволяет вам собирать данные из различных источников, таких как ГК, вспомогательные книги, бюджеты и статистическая информация.</span><span information.</span></span> <span data-ttu-id="fb8fe-107">Затем вы можете анализировать, суммировать и оценивать данные по затратам, чтобы руководство могло принимать оптимальные решения для обновления <span data-ttu-id="fb8fe-139">**Элемент вторичных затрат**</span><span class="sxs-lookup"><span data-stu-id="fb8fe-139">**Secondary cost element**</span></span>

<span data-ttu-id="fb8fe-146">**Единица управления затратами**</span><span class="sxs-lookup"><span data-stu-id="fb8fe-146">**Cost control unit**</span></span>
