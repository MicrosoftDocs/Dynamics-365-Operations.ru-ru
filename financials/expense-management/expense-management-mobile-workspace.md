# <a name="expense-management-mobile-workspace"></a>Мобильная рабочая область управления расходами

[!include[banner](../includes/banner.md)]


В этом разделе проводятся сведения о мобильной рабочей области управления расходами, которая доступна для мобильного приложения Microsoft Dynamics 365 for Finance and Operations. Эта рабочая область позволяет пользователям снять и отправить чек, чтобы потом его можно было приложить к отчету о расходах. Мобильная рабочая область также позволяет быстро создавать строки расходов с помощью приложенного чека.

<a name="overview-of-the-expense-management-mobile-workspace"></a>Общие сведения о мобильной рабочей области управления расходами
---------------------------------------------------

Во многих организациях требуется, чтобы копия чека была приложена к отчету о командировочных или деловых расходах, который сотрудник предоставляет для получения возмещения. Мобильная рабочая область **Управление расходами** позволяет пользователям быстрее создавать новые строки расходов на мобильном устройстве, выбранном пользователем, с помощью приложенной фотографии чека. Также пользователи могут сфотографировать чек и приложить его к отчету о расходах позднее. В частности, мобильная рабочая область **Управление расходами** позволяет пользователю:

-   Снять фотографию чека и отправить ее в Microsoft Dynamics 365 for Finance and Operations. Затем пользователь может приложить эту фотографию к отчету о расходах.
-   Отправить файл как снятый чек. Затем пользователь может приложить этот файл к отчету о расходах.
-   Создать новую строку расходов с использованием приложенного чека. Пользователь затем может добавить элемент строки в отчет о расходах и отправить его для утверждения и возмещения.

В остальных пунктах в этом разделе объясняется, как реализовать и использовать мобильную рабочую область **Управление расходами**.

## <a name="download-and-install-the-finance-and-operations-mobile-app"></a>Загрузите и установите мобильное приложение Finance and Operations.
Загрузите и установите мобильное приложение Finance and Operations из магазина мобильных приложений.

-   Для Android: [Finance and Operations в магазине Google Play](https://play.google.com/store/apps/details?id=com.microsoft.dynamics365.operations.mobile)
-   Для iPhone: [Finance and Operations в магазине приложений iTunes](https://itunes.apple.com/us/app/dynamics-365-for-operations/id1180836730?mt=8)

## <a name="sign-in-to-the-finance-and-operations-mobile-app"></a>Вход в мобильное приложение Finance and Operations
1.  Запустите приложение на мобильном устройстве.
2.  Введите URL-адрес Finance and Operations.
3.  Введите компанию для входа в систему. Например, введите **USMF**.
4.  При первом входе появится запрос ввести имя пользователя и пароль для учетной записи Finance and Operations. Введите свои учетные данные.
5.  После входа вы увидите доступные рабочие области для вашей компании. Обратите внимание, что если позже системный администратор опубликует новую рабочую область, вы может потянуть список, чтобы обновить список мобильных рабочих областей. 

[![Потянуть для обновления](./media/pull-to-refresh-list-of-workspaces-183x300.png)](./media/pull-to-refresh-list-of-workspaces.png)

## <a name="capture-a-receipt-by-using-the-expense-management-mobile-workspace"></a>Фотографирование чека с помощью мобильной рабочей области управления расходами
1.  На мобильном устройстве выберите рабочую область **Управление расходами**.
2.  Выберите **Захват чека**.
3.  Выберите **Сфотографировать** или **Выбрать изображение**.
4.  Если выбран вариант **Сфотографировать**, выполните следующие действия:
    1.  Будет открыто приложение камеры на мобильном устройстве, чтобы можно было сфотографировать чек. После съемки фотографии щелкните **OK**, чтобы принять фотографию.
    2.  Необязательно: введите имя для фотографии и введите любые примечания.

     **или**, если выбран пункт **Выбрать изображение**, выполните следующие действия:
    1.  Выберите изображение в списке.
    2.  Необязательно: введите имя для изображения и введите любые примечания.

5.  Выберите **Готово**.

## <a name="quick-expense-entry-by-using-the-expense-management-mobile-workspace"></a>Быстрый ввод расходов с помощью мобильной рабочей области управления расходами
1.  На мобильном устройстве выберите рабочую область **Управление расходами**.
2.  Выберите **Быстрый ввод расходов**.
3.  Выберите категорию для расхода. Отображается список категорий расходов, который загружены в ваше приложение для работы в автономном режиме. По умолчанию загружаются до 50 элементов, но разработчик может изменить это число. Для получения дополнительных сведений разработчики должны ознакомиться с разделом [Мобильная платформа Finance and Operations](/dynamics365/unified-operations/dev-itpro/mobile-apps/mobile-platform). Если категория отсутствует в списке, выберите **Поиск** для выполнения поиска по Интернету в Finance and Operations. Выполните поиск по категории расходов или переключитесь на поиск по типу расходов.
4.  Введите дату проводки расхода.
5.  Необязательно: введите получателя платежа для расхода.
6.  Введите сумму расхода.
7.  Выберите валюту для расхода. Отображается список кодов валют, который загружен в ваше приложение для работы в автономном режиме. По умолчанию загружаются до 400 валют, но разработчик может изменить это число. Для получения дополнительных сведений разработчики должны ознакомиться с разделом [Мобильная платформа Finance and Operations](/dynamics365/unified-operations/dev-itpro/mobile-apps/mobile-platform). Если валюта отсутствует в списке, выберите **Поиск** для выполнения поиска по Интернету в Finance and Operations. Выполните поиск по валюте или переключитесь на поиск по названию.
8.  Выберите **Сфотографировать** или **Выбрать изображение**.
9.  Если выбран вариант **Сфотографировать**, будет открыто приложение камеры на мобильном устройстве, чтобы можно было сфотографировать чек. После съемки фотографии щелкните **OK**, чтобы принять фотографию.  или, если выбран пункт **Выбрать изображение**, выберите изображение в списке.
10. Выберите **Готово**.
