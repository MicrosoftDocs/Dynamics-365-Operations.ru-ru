# <a name="whole-amount-and-interval-calculation-options-for-sales-tax-codes"></a>Параметры расчета "Полная сумма" и "Интервал" для налоговых кодов

[!include[banner](../includes/banner.md)]

[!include[retail name](../includes/retail-name.md)]



В этой статье описываются параметры для поля "Метод расчета" в налоговых кодах, а также порядок расчета налога для интервалов и полных сумм.

Вы можете настроить налоговый код для расчета на основе полной суммы или суммы в интервале. Выберите способ расчета налогового кода на странице "Налоговые коды" с помощью поля "Метод расчета" на экспресс-вкладке "Расчет".
-   Полная сумма — ставка налога применяется ко всей налогооблагаемой сумме целиком.
-   Интервал — налогооблагаемая сумма делится на части, каждая из которых попадает в интервал, для которого установлена конкретная ставка налога. Та часть суммы, которая попадает в указанный интервал, облагается налогом по ставке, установленной для данного интервала. Налог представляет собой совокупность налоговых сумм, рассчитанных для каждого интервала сумм.
> [!NOTE]                                                                                                                              
> Параметр "Интервал" доступен только когда вы выбираете значение "Строка" в поле "Метод расчета" в области "Налог" на странице "Параметры главной книги". 

Интервалы настраиваются на странице "Значения налогового кода" путем ввода минимальной и максимальной предельных сумм для каждой налоговой ставки. Для того, чтобы налоги рассчитывались по всем налогооблагаемым суммам, вне зависимости от используемого метода должны соблюдаться следующие правила в отношении интервалов:
-   Минимальный предел первого интервала должен быть равен нулю.
-   Максимальный предел последнего интервала должен быть равен нулю, который в данном случае обозначает бесконечность.
-   Максимальный предел интервала должен быть минимальным пределом следующего интервала.

Если сумма является максимальным пределом одного из интервалов и минимальным пределом следующего за ним интервала, то к этой сумме будет применена ставка налога, установленная для первого интервала. Если сумма выходит за границы интервалов, которые определены верхним и нижним пределами, то применяется налоговая ставка равная 0.

## <a name="example-whole-amount-method-of-calculation"></a>Пример. Метод расчета "Сумма целиком"
На странице "Значения налогового кода" ставки налога устанавливаются в следующих интервалах.
|                   |                   |              |
|-------------------|-------------------|--------------|
| **Нижний предел** | **Максимальный лимит** | **Ставка налога** |
| 0,00              | 50,00             | 30%          |
| 50,00             | 100,00            | 20%          |
| 100,00            | 0,00              | 10%          |

Налог рассчитывается от всей налогооблагаемой суммы.

| Налогооблагаемая сумма (цена) | Расчет    | Налог (НСП) |
|------------------------|----------------|-----------|
| 35,00                  | 35,00 \* 0,30  | 10,50     |
| 50,00                  | 50,00 \* 0,30  | 15,00     |
| 85,00                  | 85,00 \* 0,20  | 17,00     |
| 305,00                 | 305,00 \* 0,10 | 30,50     |

## <a name="example-interval-method-of-calculation"></a>Пример. Метод расчета "Интервал"
На странице "Значения" ставки налога устанавливаются в следующих интервалах.

|                   |                   |              |
|-------------------|-------------------|--------------|
| **Нижний предел** | **Максимальный лимит** | **Ставка налога** |
| 0,00              | 50,00             | 30%          |
| 50,00             | 100,00            | 20%          |
| 100,00            | 0,00              | 10%          |

Налог представляет собой совокупность налоговых сумм, рассчитанных для каждого интервала сумм.

| Налогооблагаемая сумма (цена) | Расчет                                                               | Налог (НСП) |
|------------------------|---------------------------------------------------------------------------|-----------|
| 35,00                  | 35,00 \* 0,30                                                             | 10,50     |
| 50,00                  | 50,00 \* 0,30                                                             | 15,00     |
| 85,00                  | (50,00 \* 0,30 = 15,00) + (35,00 \* 0,20 = 7,00)                          | 22,00     |
| 305,00                 | (50,00 \* 0,30 = 15,00) + (50,00 \* 0,20 = 10,00) + (205 \* 0,10 = 20,50) | 45,50     |

 

Дополнительные сведения см. в разделе [Определение ставок налога на основе полей "База маржинальной прибыли" и "Метода расчета"](marginal-base-field.md).
