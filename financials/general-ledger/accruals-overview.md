# <a name="accruals-overview"></a>Обзор начислений

Эта статья описывает начисления и представляет информация о том, как настроить их и создать проводки.

Начисления используются в учете методом начисления для отслеживания выручки, утвержденной за период, в котором она была получена, а не в момент получения платежа, и для отслеживания расходов (затрат), которые были утверждены в момент возникновения, а не в момент осуществления платежа.

## <a name="accrual-schemes"></a>Схемы начисления
Схемы начисления используются для настройки отложенной выручки и затрат. Одну и ту же схему начисления можно использовать как для выручки, так и для затрат. Начисления ГК позволяют перераспределять затраты или выручку из строки журнала, чтобы затраты и выручки распознавались в соответствующих периодах. На странице **Схема начисления** укажите дебетовый и кредитный счета, которые будут использоваться при применении схемы начисления.

-   **Дебет** — определяемый счет ГК, который заменит дебетовый счет ГК в строке ваучера журнала. Этот счет также будет использоваться для реверсирования расхода будущего периода на основании проводок начисления ГК.
-   **Кредит** — определяемый счет ГК, который заменит кредитный счет ГК в строке ваучера журнала. Этот счет также будет использоваться для реверсирования расхода будущего периода на основании проводок начисления ГК.

После определения счета для использования можно указать способ создания кода ваучера при создании проводок по начислениям. Также можно указать, как часто должны выполняться проводки, число раз, которое должны создаваться проводки, и время разноски проводок. После создания схемы начисления ее можно использовать в некоторых журналах с помощью функции "Начисления ГК".

## <a name="ledger-accruals"></a>Начисления ГК
При создании журнала можно щелкнуть **Начисления ГК** в меню **Функции**. Затем при выборе схемы начисления будет отображаться базовая сумма из журнала, которая будет распределена по периоду, как определено схемой начисления. Например если оплата за весь год, в январе страховки для сотрудника, а сумма равна 12 000, должна распознавать, расходами каждый месяц. Можно выбрать дату начала. Вы также можете указать, будет ли начисляемая сумма основываться на счете или корр. счете. После внесения параметров, выбранных, щелкните **проводок** для просмотра всех проводок, которые были созданы на основании схемы начисления. Например если 12000 в страховании расходов в течение года, вы увидите 1 000 для каждого месяца. После учета журнала можно просмотреть проводки, используя **коды операций** запроса страницы. Если не применить схемы начисления, (например, когда участвует накладной заказа на продажу или накладной заказа на покупку), можно Кредит Сумма предоплаты и дебетовать сумму расходов. Затем можно выбрать **Корр. счет** при применении схемы начисления.
