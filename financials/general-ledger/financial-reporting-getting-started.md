# <a name="financial-reporting"></a>Финансовая отчетность

В этом разделе описывается доступ к финансовой отчетности в Microsoft Dynamics 365 для операций и использовании финансовой отчетности. Он включает в себя описание по умолчанию финансовые отчеты, которые предоставляются.

<a name="accessing-financial-reporting"></a>Доступ к финансовой отчетности
-----------------------------

Можно найти **финансовой отчетности** меню в следующем помещает в Dynamics 365 для операций:

-   **Главная книга**&gt;**запросы и отчеты**
-   **Бюджетирование**&gt;**Inquires и отчетов**&gt;**базового бюджетирования**
-   **Бюджетирование**&gt;**запросов и отчетов**&gt;**планирования бюджета**
-   **Бюджетирование**&gt;**запросов и отчетов**&gt;**бюджетного контроля**
-   Консолидации

Чтобы создать и сгенерировать финансовые отчеты для юридического лица, необходимо настроить следующую информацию для этого юридического лица:

-   Финансовый календарь
-   Книга учета
-   План счетов
-   Валютное

Функции финансовой отчетности доступны для пользователей, которые имеют соответствующие привилегии и полномочия, назначенные им в ролях безопасности. В следующих разделах перечисляются эти привилегии и полномочия вместе со связанными ролями.

## <a name="opening-financial-reports"></a>Открытие финансовых отчетов
При переходе в меню **Финансовая отчетность** отображается список финансовых отчетов по умолчанию для компании. После этого можно открыть или изменить отчет. Чтобы открыть один из отчетов по умолчанию, выберите имя отчета. При открытии отчета в первый раз он автоматически создается за предыдущий месяц. Например при открытии отчета в первый раз в августе 2016 31 июля 2016 формируется отчет. После открытия отчета можно начать его изучение, выполнив детализацию до определенных частей данных и изменив параметры отчета.

## <a name="creating-and-modifying-financial-reports"></a>Создание и изменение финансовых отчетов
В списке финансовых отчетов можно создать новый отчет или изменить существующий отчет. Если имеются соответствующие разрешения, можно создать новый финансовый отчет, нажав кнопку **Нью** в области действий. Отчет программа конструктора загружается на устройство. Затем после запуска конструктора отчетов можно создать новый отчет. После сохранения новый отчет отобразится в списке финансовых отчетов. В списке отображаются только те отчеты, которые были созданы для компании, которая используется в Dynamics 365 для операций. Дополнительные сведения о процессе создания и изменения финансовых отчетов в Dynamics 365 for Operations называть [в блогах](https://blogs.msdn.microsoft.com/dynamics_financial_reporting/tag/learning/) блог из Dynamics финансовых отчетов. **Примечание:**, чтобы загрузить отчет конструктора клиента на должны иметь версию 4.6.2 платформы Microsoft .NET Framework, установленную на его компьютере. Можно загрузить и установить с помощью этой версии Microsoft .NET Framework [здесь](https://www.microsoft.com/en-us/download/details.aspx?id=53345). При использовании хром, необходимо установить расширение ClickOnce, чтобы загрузить клиент конструктора отчетов. При выполнении в режиме incognito, убедитесь, что расширение ClickOnce включено для работы в incognito режиме. Можно изменить отчет, отображаемый в списке финансовых отчетов. Если область вокруг имени отчета выбрана, нажмите кнопку **Правка** в области действий. Запустится программа конструктора отчетов.
