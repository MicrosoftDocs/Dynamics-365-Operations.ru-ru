# <a name="set-up-sales-tax-codes"></a>Настройка кодов налогов

[!include[task guide banner](../../includes/task-guide-banner.md)]

Налоговые коды создаются для каждого косвенного налога или пошлины, которые юридическое лицо должно рассчитывать, собирать и платить налоговым органам.

В этой задаче используется демонстрационная компания USMF.



1. Перейдите в раздел "Налог" > "Косвенные налоги" > "Налог" > "Налоговые коды".
2. Щелкните "Создать".
3. В поле "Налоговый код" введите значение.
4. В поле "Имя" введите значение.
5. Выберите период сопоставления, чтобы указать, в какие налоговые органы и через какие интервалы времени следует отправлять налоговые отчеты и выплачивать налоги.
6. В списке перейдите по ссылке в выбранной строке.
7. Выберите группу разноски ГК, чтобы указать счета ГК для разноски налога в главную книгу.
8. Поиск и выбор требуемой записи в списке.
9. В списке перейдите по ссылке в выбранной строке.
10. Разверните экспресс-вкладку "Расчет".
    * На экспресс-вкладке "Расчет" имеется несколько полей, с помощью которых можно управлять способом расчета сумм налогов.  
11. В области действий щелкните "Налоговый код".
12. Щелкните "Значение".
13. В списке пометьте выбранную строку.
14. Введите значение для данного налогового кода.
    * На экспресс-вкладке "Расчет" в поле "Источник", если выбран параметр "Сумма на единицу", значение будет умножаться на количество в проводке для расчета суммы налога.  Если налоговый код не основан на единице, значение выражается в процентах, которые применяются к источнику для данного налогового кода для расчета суммы налога.     
15. Нажмите кнопку "Сохранить".
16. Закройте страницу.
17. Нажмите кнопку "Сохранить".
