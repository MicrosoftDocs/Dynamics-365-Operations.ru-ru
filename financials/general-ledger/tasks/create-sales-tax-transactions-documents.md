# <a name="create-sales-tax-transactions-on-documents"></a>Создание налоговых проводок по документам

[!include[task guide banner](../../includes/task-guide-banner.md)]

Налог в документах рассчитывается путем указания налоговой группы и налоговой группы номенклатур в строках документа. Эти значения берутся по умолчанию из справочника, но их можно изменить вручную при необходимости. Рассчитанный налог можно проверить на уровне строки и документа. В этой задаче используется демонстрационная компания USMF.

1. Перейдите в раздел "Расчеты с клиентами" > "Заказы" > "Все заказы на продажу".
2. Щелкните "Создать".
3. В поле "Счет клиента" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
4. В списке найдите и выберите требуемую запись.
5. В списке перейдите по ссылке в выбранной строке.
6. Нажмите кнопку OK.
7. В списке пометьте выбранную строку.
8. В поле "Код номенклатуры" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
9. В списке перейдите по ссылке в выбранной строке.
10. В поле "Цена за единицу" введите число.
11. Разверните или сверните раздел "Сведения о строке".
12. Перейдите на вкладку Настройка.
13. В поле "Налоговая группа номенклатур" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
14. Поиск и выбор требуемой записи в списке.
15. В списке перейдите по ссылке в выбранной строке.
16. Щелкните "Статистика".
17. Щелкните "Налог".
18. Нажмите кнопку "OК".
19. Щелкните "Добавить строку".
20. В списке пометьте выбранную строку.
21. В поле "Код номенклатуры" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
22. Поиск и выбор требуемой записи в списке.
23. В списке перейдите по ссылке в выбранной строке.
24. В поле "Цена за единицу" введите число.
25. В поле "Налоговая группа номенклатур" нажмите кнопку раскрывающегося списка, чтобы открыть поиск.
26. Поиск и выбор требуемой записи в списке.
27. В списке перейдите по ссылке в выбранной строке.
28. В области действий щелкните "Продажа".
29. Щелкните "Налог".
30. Нажмите кнопку "OК".
