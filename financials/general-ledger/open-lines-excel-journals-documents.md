# <a name="publish-journal-lines-and-documents-from-excel"></a>Публикация документов из Excel и строки журнала

В этом разделе объясняется, как ввести и опубликовать строк для финансовых журналов из Microsoft Excel. Сюда входят сведения о различных шаблонов, которые можно использовать, в зависимости от типа проводок, которые ввода.

Пользователи могут вводить и опубликовать строк для финансовых журналов из Microsoft Excel. После пользователь создает журнал, **открытые строки в Excel** при нажатии кнопки отображаются доступные шаблоны. Шаблоны разработаны для поддержки конкретных сценариев, однако не все комбинации типа счета поддерживается в журнале. В следующей таблице показаны доступных шаблонов и типов счетов, которые они поддерживают.
|                          |                                                                                                                         |                                                                                         |
|--------------------------|-------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| **Template**             | **Типы поддерживаемых учетных записей**                                                                                             | **Порядок доступа к шаблону**                                                          |
| Строки журнала ГК     | Счет: ГК, клиент, поставщик, Банк Корр. счет: главной книги, клиента, поставщика, банковского внутрихолдинговых поддерживается.       | Общий журнал                                                                         |
| Регистрация накладных         | Учетная запись: Поставщика корр. счет: внутрихолдинговых главной книги не поддерживается.                                                    | Регистрация накладных AP                                                                     |
| Журнал накладных          | Учетные записи: Поставщика корр. счет: поддерживается внутрихолдинговых главной книги.                                                      | Журнал накладных AP                                                                      |
| Накладная поставщика           |                                                                                                                         | Накладная поставщика                                                                          |
| Журнал накладных клиента | Учетная запись: Клиент корр. счет: поддерживается внутрихолдинговых главной книги.                                                     | Общий журнал                                                                         |
| Накладная с произвольным текстом        |                                                                                                                         | На **накладной с произвольным текстом** и щелкните **открыть в Excel** (значок Microsoft Office). |
| Журнал "Основные средства"     | Средства, книга, банк, клиент или поставщик. Внутрихолдинговый не поддерживается.                                               | Журнал основных средств                                                                     |
| Журнал платежей поставщикам   | Учетная запись: Поставщика корр. счет: внутрихолдинговых банковской книги, поддерживается.                                                 | Журнал платежей поставщикам                                                                  |
| Журнал платежей клиентов | Учетная запись: Клиент корр. счет: внутрихолдинговых банковской книги, поддерживается.                                               | Журнал платежей клиентов                                                                |
| Журнал расходов по проекту  | Учетная запись: Проект, главной книги, клиента, поставщика корр. счет: проект, главной книги, клиента, поставщика-внутрихолдингового поддерживается. | Общий журнал расходов (в области управления проектом и учета)                       |

При публикации строки, они проходят проверку, чтобы убедиться в том, что они соответствуют правила, которые определяются в финансовых журналов. После публикации по строкам, пользователи могут редактировать или выполнить разноску операций из Microsoft Dynamics 365 для операций. 

Чтобы добавить шаблон финансовых аналитик, требуются дополнительные изменения. Дополнительные сведения см. в разделе [Добавление аналитик шаблона Microsoft Excel](\dev-itpro\financial-dimensions\add-dimensions-excel-templates). После добавления объекта аналитики, они будут доступны в конструкторе Excel и могут быть добавлены в шаблон.
