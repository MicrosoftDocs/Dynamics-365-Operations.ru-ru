# <a name="navigation-search"></a>Навигационный поиск

В этой статье объясняется, как использовать функцию поиска для перехода к страницам в Microsoft Dynamics 365 для операций.

Dynamics 365 для операций предоставляет функциональные возможности для широкого спектра отраслей и вертикальных сегментов рынка. Приложение включает несколько областей и предназначены для выполнения различных задач. Чтобы быстро найти страницы, необходимые для выполнения задач, используйте функцию поиска переходов. Для использования этой функции щелкните значок **Поиск**, чтобы открыть окно **Поиск**. Затем введите одно или несколько слов в поле. Система немедленно выполнит поиск соответствующих страниц в приложении, которые соответствуют введенным словам. Например, если ввести "накладная поставщика", система отобразит результаты, которые соответствуют этим словам. **Примечание.** Поле **Поиск** помогает находить страницы и переходить к ним. С его помощью невозможно найти определенные данные или действия. 

[![поле поиска](./media/search-box.png)](./media/search-box.png) функцию поиска переходов также служит в качестве отличный способ быстрого перехода на определенную страницу. Например, если вы являетесь расчетов с поставщиками служащий, часто использует **журнал платежей** страницы, можно ввести в поле поиска «журнал оплат поставщикам». Поскольку ввод данных является точным соответствием в строке заголовка и страница отображается в верхней части результатов поиска можно быстро перейти к нему. 

[![Поиск для--журнал платежей](./media/searching-for-payment-journal.png)](./media/searching-for-payment-journal.png) 

Результаты поиска отображаются заголовок страницы и путь перехода. Это помогает определит расположение страницы в приложении. Также благодаря этому можно отличить две или более похожих страниц в результатах. Когда вы ищете страницу, вводимый запрос сопоставляется с заголовком страницы и ее навигационным путем. Например, если ввести «с клиентами» в ** ** поле поиска будут отображены результаты для страниц, доступные в расчеты с клиентами области — несмотря на то, что заголовки страниц не включают слова «клиентами». 

[![Поиск для word-с клиентами](./media/search-for-the-word-receivable.png)](./media/search-for-the-word-receivable.png) 

Администрирование и точки зрения безопасности Навигация поиск областей только функциональные возможности:

-   Страницы, которые включены в текущей конфигурации (с помощью конфигурационных ключей).
-   Страницы, к которым у пользователя есть доступ на основании его роли.

Список результатов поиска ограничен 10 позициями. Если в результатах отсутствует искомый элемент, уточните или обновите запрос поиска. С точки зрения разработки функцию навигационного поиска очень легко использовать, поскольку отсутствует задержка между развертыванием пунктов меню и их способностью отображаться в результатах поиска. Пока пункты меню связаны с областью перехода или панелью мониторинга, они автоматически становятся доступны для поиска. Функция навигационного поиска также включает востребованную функцию для опытных пользователей: способность быстро переходить к странице на основании имени технической формы. Множество пользователей настолько знакомы с системой, что они знают точные имена форм, с которыми они работают. Если вы принадлежите к их числу, вы можете ввести **форма:**, а затем ввести имя искомой формы. Например, если ввести **форма: vendinvoice**, в результатах поиска отобразится все страницы, на которых имя формы начинает с **vendinvoice**. 

[![Поиск для формы vendinvoice](./media/search-for-form-vendinvoice.png)](./media/search-for-form-vendinvoice.png)
