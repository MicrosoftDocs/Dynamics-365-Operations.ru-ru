# <a name="dynamics-365-for-operations-client-faq"></a>Dynamics 365 для операций клиента часто задаваемые вопросы

В этой статье содержатся ответы на часто задаваемые вопросы о Microsoft Dynamics 365 для операций клиента.

<a name="why-arent-symbols-loaded-when-i-use-dynamics-365-for-operations"></a>Почему не загружены символы при использовании Dynamics 365 для операций?
-----------------------------------------------------------------

Параметры безопасности в вашем браузере могут стать причиной неправильной загрузки символов. Чтобы решить эту проблему, выполните следующие шаги.

-   При возникновении этой проблемы в обозревателе Internet Explorer, щелкните **средств**, а затем щелкните **свойства обозревателя**.  В диалоговом окне Свойства обозревателя на **конфиденциальности**, щелкните **другой**и убедитесь, что **Загрузка шрифта** выбран параметр.
-   В противном случае необходимо добавить в список надежных узлов Dynamics 365 для операций узла.

## <a name="i-miss-the-ribbon-from-dynamics-ax-2012-can-i-keep-action-pane-tabs-open-all-the-time"></a>Пропущено ленты из Dynamics AX 2012. Можно сохранить вкладки открыть постоянно панели операций
Корпорация Майкрософт планирует скоро реализовать эту функцию. Пользователи затем смогут выбрать сохранение вкладок на панели операций откройте постоянно. В противном случае вкладки будут сворачиваться, когда не используются, чтобы предоставить больше пространства экрана для страницы.

## <a name="why-do-i-sometimes-see-different-shortcut-menus-when-i-rightclick"></a>Иногда осталась разные контекстные меню когда я rightclick?
При щелчке редактируемого поля правой кнопкой мыши (или при выборе текста) отображается контекстное меню браузера. Это меню предоставляет доступ к командам **Вырезать**, **Копировать** и **Вставить**. Корпорация Майкрософт не может внедрить эти команды в Dynamics 365 для операций контекстные меню, поскольку в целях безопасности обозреватель не позволяет программного доступа к в буфер обмена.

Если правой кнопкой мыши метку поля или значение элемента управления только для чтения, вы увидите Dynamics 365 для операций контекстном меню.

Чтобы упростить ввод с клавиатуры, корпорация Майкрософт планирует реализовать сочетания клавиш в будущем, будут открываться Dynamics 365 для операций контекстном меню.

## <a name="where-is-the-view-details-functionality-in-dynamics-365-for-operations"></a>Где находится представление сведений о функции в Dynamics 365 для операций?
Параметр **Просмотр сведений** доступен в нескольких местах:

-   Если элемент управления имеет возможности **Просмотр сведений **и если элемент управление имеет значение, то это значение отображается как гиперссылка. Можно перейти по гиперссылке, чтобы открыть страницу, которая содержит дополнительные сведения.
-   **Просмотр сведений о** может также использоваться в Dynamics 365 для операций контекстные меню. Дополнительные сведения о том, когда Dynamics 365 для операций контекстные меню отображается при щелчке правой кнопкой см. в предыдущем разделе.
