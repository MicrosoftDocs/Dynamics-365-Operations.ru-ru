# <a name="onboarding-home-page"></a>Домашняя страница адаптирования

[!include[banner](../includes/banner.md)]


Чтобы начать работать с Microsoft Dynamics 365 for Finance and Operations, можно выбрать уровень подписки, подходящий для вас, найти партнера с опытом в вашей области или отрасли и использовать программу Microsoft FastTrack, чтобы быстро приступить к работе.  

Следующие ресурсы содержат обзор служб Lifecycle Services и порядка работы с партнером, чтобы начать работать с Finance and Operations.
-   [Регистрация на подписку](/dynamics365/unified-operations/dev-itpro/dev-tools/sign-up-preview-subscription)
-   [Обзор Microsoft FastTrack for Dynamics 365](fasttrack-dynamics-365-overview.md)
-   [Введение для клиентов в службы Lifecycle Services](/dynamics365/unified-operations/dev-itpro/lifecycle-services/lcs-works-lcs)
