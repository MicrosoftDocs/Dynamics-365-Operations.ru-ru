# <a name="create-record-templates"></a>Создание шаблонов записей

Эта статья вводит понятие шаблонов записей и описывает порядок, как их можно использовать для создания записей, совместно использующих сведения.

Шаблоны записей позволяют создавать дополнительные быстро в Microsoft Dynamics 365 для операций записи. Шаблоны записей можно создать только для определенных типов записей в Microsoft Dynamics 365 для операций. Например *** представьте вводятся сведения об аренде для предприятия машине арендной платы находится в Сан-Франциско. В виду того что большинство клиентов, скорее всего, будут из региона Сан-Франциско, было бы славно, если вы могли автоматически заполнить значения для полей **Штат**, **Страна** и **Город** в форме аренды. **Примечание:** можно применять шаблоны только для тех частей Dynamics 365 для операций, которые у вас есть доступ. Все заголовки шаблонов видны пользователю при создании новой записи, а также другим пользователям, если пользователь создает шаблоны, которые будут доступны для всех пользователей. Имейте это в виду, присваивая названия шаблонам. Например, следует избегать использования названий, включающих такие слова, как "комиссия", если не все пользователи знают, что некоторые сотрудники компании получают заработную плату на комиссионной основе. Когда один или несколько шаблонов, к которым пользователь имеет доступ, создаются для определенной формы, а пользователь пытается создать в форме новую запись, отображается страница **Поиск шаблона для…**. При выборе шаблона из списка, на его базе создается новая запись, содержащая информацию по умолчанию. Если необходимо использовать шаблоны при создании новых записей, **не задавать вопрос** флажок в **Выбор шаблона для** страницы. Чтобы вернуться к отображению диалогового окна выбора шаблона, щелкните правой кнопкой мыши любую запись, выберите **сведения о записи**, а затем щелкните **Показать выбор шаблона**.
