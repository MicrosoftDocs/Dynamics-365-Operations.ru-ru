# <a name="approve-application-inbox-records"></a>Одобрение записей входящих заявлений

[!include[task guide banner](../../includes/task-guide-banner.md)]

В этой процедуре показано рассмотрение заявлений, полученных через портал самообслуживания сотрудников. Помимо рассмотрения заявлений, вы можете утверждать выбранные записи входящих заявлений. Записи входящих заявлений — это заявления о приеме на работу, отправленные в компанию на рассмотрение. После утверждения записи лица, отправившего заявление, будет создана запись кандидата. В качестве компании с демонстрационными данными для создания этой процедуры используется USMF.


## <a name="review-application-inbox-record"></a>Просмотр записи входящего заявления
1. Перейдите в раздел > "Управление персоналом" > "Набор сотрудников" > "Входящее заявление".
2. В списке найдите и выберите требуемую запись.
3. Разверните раздел "Адреса".
4. Разверните раздел "Контактная информация".
5. Разверните раздел "Вложения".

## <a name="approve-application-inbox-record"></a>Утверждение записи входящего заявления
1. В списке пометьте выбранную строку.
2. Запомните значение в поле "Имя"; в дальнейшем вам предстоит на него ссылаться
3. Нажмите кнопку Одобрить.
4. Нажмите кнопку "OК".
5. Закройте страницу.

## <a name="view-the-newly-created-application-record"></a>Просмотр созданной записи заявления
1. Перейдите в раздел "Управление персоналом" > "Набор сотрудников" > "Заявления" > "Заявления".
2. В списке перейдите по ссылке в выбранной строке.
3. Разверните раздел "Вложения".
