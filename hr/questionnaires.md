# <a name="questionnaires"></a>Анкеты

Этот раздел содержит общие сведения об анкетах и ссылки на более подробные сведения. Анкета — это набор вопросов, используемых для сбора сведений для различных целей. Анкеты можно разрабатывать, создавать, распространять и заполнять, а затем анализировать результаты. 

Можно настроить анкеты в соответствии с помощью различных функций, доступных в Microsoft Dynamics 365 для операций конкретным требованиям. Вот несколько примеров использования анкет:

-   Проверка профессиональных навыков и кандидатов.
-   Определение того, выучили ли участники курса материал курса.
-   Оценка курса (например, оснащение, содержимое и лектор).
-   Исследование степени удовлетворения сотрудников и клиентов.
-   Оценка производительности работы сотрудников.
